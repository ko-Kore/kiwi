<h2><span id="槪要(개요)"></span><span class="mw-headline" id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"><ruby class="hanja grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E5%BD%A2%E8%81%B2%E6%96%87%E5%AD%97&amp;action=edit&amp;section=1" title="부분 편집: 槪要(개요)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%E5%85%AD%E6%9B%B8" title="六書"><ruby class="hanja grade62"><rb>六書</rb><rp>(</rp><rt>육서</rt><rp>)</rp></ruby></a>의 <ruby class="hanja grade52"><rb>分類</rb><rp>(</rp><rt>분류</rt><rp>)</rp></ruby> <ruby class="hanja grade80"><rb>中</rb><rp>(</rp><rt>중</rt><rp>)</rp></ruby> 하나로 <ruby class="hanja grade42"><rb>意味</rb><rp>(</rp><rt>의미</rt><rp>)</rp></ruby>를 나타내는 <ruby class="hanja grade62"><rb>部分</rb><rp>(</rp><rt>부분</rt><rp>)</rp></ruby>과 <a href="/index.php?title=%EC%86%8C%EB%A6%AC&amp;action=edit&amp;redlink=1" class="new" title="소리 (없는 문서)">소리</a>를 나타내는 <ruby class="hanja grade62"><rb>部分</rb><rp>(</rp><rt>부분</rt><rp>)</rp></ruby>을 <ruby class="hanja grade60"><rb>合</rb><rp>(</rp><rt>합</rt><rp>)</rp></ruby>쳐서 만들어낸 글<ruby class="hanja grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>를 가리킨다.
</p>
<h2><span id="特徵(특징)"></span><span class="mw-headline" id=".E7.89.B9.E5.BE.B5.28.ED.8A.B9.EC.A7.95.29"><ruby class="hanja grade32"><rb>特徵</rb><rp>(</rp><rt>특징</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E5%BD%A2%E8%81%B2%E6%96%87%E5%AD%97&amp;action=edit&amp;section=2" title="부분 편집: 特徵(특징)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%E6%9C%83%E6%84%8F%E6%96%87%E5%AD%97" title="會意文字"><ruby class="hanja grade62"><rb>會意文字</rb><rp>(</rp><rt>회의문자</rt><rp>)</rp></ruby></a>와 마찬가지로 <ruby class="hanja grade72"><rb>上下左右</rb><rp>(</rp><rt>상하좌우</rt><rp>)</rp></ruby>나 안과 밖, 귀퉁이 <ruby class="hanja grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby> <ruby class="hanja grade40"><rb>多樣</rb><rp>(</rp><rt>다양</rt><rp>)</rp></ruby>한 <ruby class="hanja grade52"><rb>方法</rb><rp>(</rp><rt>방법</rt><rp>)</rp></ruby>으로 짜일 수 있으며, <a href="/index.php?title=%E5%BD%A2%E9%83%A8&amp;action=edit&amp;redlink=1" class="new" title="形部 (없는 문서)"><ruby class="hanja grade62 unknown"><rb>形部</rb><rp>(</rp><rt>형부</rt><rp>)</rp></ruby></a>(<ruby class="hanja grade42"><rb>意味</rb><rp>(</rp><rt>의미</rt><rp>)</rp></ruby>를 나타내는 <ruby class="hanja grade62"><rb>部分</rb><rp>(</rp><rt>부분</rt><rp>)</rp></ruby>)을 어디에 두고 <a href="/index.php?title=%E8%81%B2%E9%83%A8&amp;action=edit&amp;redlink=1" class="new" title="聲部 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>聲部</rb><rp>(</rp><rt>성부</rt><rp>)</rp></ruby></a>(소리를 나타내는 <ruby class="hanja grade62"><rb>部分</rb><rp>(</rp><rt>부분</rt><rp>)</rp></ruby>)을 어디에 둔다는 <ruby class="hanja grade50"><rb>規則</rb><rp>(</rp><rt>규칙</rt><rp>)</rp></ruby>이 따로 없다. 또, <ruby class="hanja grade62"><rb>音</rb><rp>(</rp><rt>음</rt><rp>)</rp></ruby>을 따올 때에는 <a href="/wiki/%E6%BC%A2%E5%AD%97" title="漢字"><ruby class="hanja grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby></a>가 만들어진 <a href="/index.php?title=%E5%8F%A4%E4%BB%A3%E4%B8%AD%E5%9C%8B%E8%AA%9E&amp;action=edit&amp;redlink=1" class="new" title="古代中國語 (없는 문서)"><ruby class="hanja grade60 unknown"><rb>古代中國語</rb><rp>(</rp><rt>고대중국어</rt><rp>)</rp></ruby></a>의 소리에서 비슷한 <a href="/index.php?title=%E9%9F%BB&amp;action=edit&amp;redlink=1" class="new" title="韻 (없는 문서)"><ruby class="hanja grade32 unknown"><rb>韻</rb><rp>(</rp><rt>운</rt><rp>)</rp></ruby></a>을 <ruby class="hanja grade32"><rb>基盤</rb><rp>(</rp><rt>기반</rt><rp>)</rp></ruby>으로 하기 때문에, <ruby class="hanja grade40"><rb>多樣</rb><rp>(</rp><rt>다양</rt><rp>)</rp></ruby>한 짜임과 더불어 처음 보는 글<ruby class="hanja grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>가 나왔을 때 <ruby class="hanja grade42"><rb>意味等</rb><rp>(</rp><rt>의미등</rt><rp>)</rp></ruby>을 <ruby class="hanja grade10"><rb>斟酌</rb><rp>(</rp><rt>짐작</rt><rp>)</rp></ruby>하기 어려운 <ruby class="hanja grade40"><rb>境遇</rb><rp>(</rp><rt>경우</rt><rp>)</rp></ruby>도 있다.
</p><p>새로운 글<ruby class="hanja grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>를 만드는 데에 있어서 가장 <ruby class="hanja grade32"><rb>效率</rb><rp>(</rp><rt>효율</rt><rp>)</rp></ruby>이 좋은 <ruby class="hanja grade52"><rb>方法</rb><rp>(</rp><rt>방법</rt><rp>)</rp></ruby>이기 때문에 <a href="/wiki/%E6%BC%A2%E5%AD%97" title="漢字"><ruby class="hanja grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby></a>의 80% <ruby class="hanja grade52"><rb>以上</rb><rp>(</rp><rt>이상</rt><rp>)</rp></ruby>이 여기에 <ruby class="hanja grade40"><rb>屬</rb><rp>(</rp><rt>속</rt><rp>)</rp></ruby>하며, <ruby class="hanja grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby>가 <a href="/index.php?title=%E8%A1%A8%E6%84%8F%E6%96%87%E5%AD%97&amp;action=edit&amp;redlink=1" class="new" title="表意文字 (없는 문서)"><ruby class="hanja grade62 unknown"><rb>表意文字</rb><rp>(</rp><rt>표의문자</rt><rp>)</rp></ruby></a>가 아닌 <a href="/index.php?title=%E8%A1%A8%E8%AA%9E%E6%96%87%E5%AD%97&amp;action=edit&amp;redlink=1" class="new" title="表語文字 (없는 문서)"><ruby class="hanja grade62 unknown"><rb>表語文字</rb><rp>(</rp><rt>표어문자</rt><rp>)</rp></ruby></a>로 <ruby class="hanja grade52"><rb>分類</rb><rp>(</rp><rt>분류</rt><rp>)</rp></ruby>되는 <ruby class="hanja grade60"><rb>理由</rb><rp>(</rp><rt>이유</rt><rp>)</rp></ruby>로 꼽힌다.
</p><p><a href="/wiki/%E9%9F%93" title="韓"><ruby class="hanja grade80"><rb>韓</rb><rp>(</rp><rt>한</rt><rp>)</rp></ruby></a>, <a href="/wiki/%E8%AA%9E" title="語"><ruby class="hanja grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby> <ruby class="hanja grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby>의 <ruby class="hanja href="/index.php?title=%E7%8F%BE%E4%BB%A3&amp;action=edit&amp;redlink=1" class="new" title="現代 (없는 문서)"><ruby class="hanja grade62 unknown"><rb>現代</rb><rp>(</rp><rt>현대</rt><rp>)</rp></ruby></a>에 들어 <a href="/wiki/%E4%B8%AD%E5%9C%8B" title="中國"><ruby class="hanja grade80"><rb>中國</rb><rp>(</rp><rt>중국</rt><rp>)</rp></ruby></a>에서 만들어진 <a href="/index.php?title=%E5%85%83%E7%B4%A0&amp;action=edit&amp;redlink=1" class="new" title="元素 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>元素</rb><rp>(</rp><rt>원소</rt><rp>)</rp></ruby></a>를 나타내는 글<ruby class="hanja grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>들(<a href="/index.php?title=%E6%B0%9F&amp;action=edit&amp;redlink=1" class="new" title="氟 (없는 문서)"><ruby class="hanja grade0
