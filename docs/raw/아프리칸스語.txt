<p>아프리칸스<ruby class="hanja api grade70"><rb>語</rb><rp>(</rp><rt>어</rt><rp>)</rp></ruby>(<span style="font-size: smaller;"><a class="mw-selflink selflink">아프리칸스語</a>: </span><span lang="af"><ruby><rb>Afrikaans</rb><rp>(</rp><rt>아프리칸스</rt><rp>)</rp></ruby></span>)는 <ruby class="hanja api grade20"><rb>南阿弗利加</rb><rp>(</rp><rt>남아불리가</rt><rp>)</rp></ruby>　<ruby class="hanja api grade60"><rb>地區</rb><rp>(</rp><rt>지구</rt><rp>)</rp></ruby> <ruby class="hanja api grade42"><rb>位置</rb><rp>(</rp><rt>위치</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade10"><rb>歐羅巴人</rb><rp>(</rp><rt>구라파인</rt><rp>)</rp></ruby> <ruby class="hanja api grade60"><rb>通用</rb><rp>(</rp><rt>통용</rt><rp>)</rp></ruby> <ruby class="hanja api grade60"><rb>言語</rb><rp>(</rp><rt>언어</rt><rp>)</rp></ruby>.
</p><p><a href="/wiki/%E5%8D%97%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4_%E5%85%B1%E5%92%8C%E5%9C%8B" class="mw-redirect" title="南아메리카 共和國"><ruby class="hanja api grade80"><rb>南</rb><rp>(</rp><rt>남</rt><rp>)</rp></ruby>아메리카 <ruby class="hanja api grade62"><rb>共和國</rb><rp>(</rp><rt>공화국</rt><rp>)</rp></ruby></a>의 <ruby class="hanja api grade62"><rb>公用語</rb><rp>(</rp><rt>공용어</rt><rp>)</rp></ruby>이며 <a href="/index.php?title=%EB%82%98%EB%AF%B8%EB%B9%84%EC%95%84&amp;action=edit&amp;redlink=1" class="new" title="나미비아 (없는 文書)">나미비아</a>에서 널리 쓰이는 서게르만<ruby class="hanja api grade70"><rb>語</rb><rp>(</rp><rt>어</rt><rp>)</rp></ruby><ruby class="hanja api grade80"><rb>軍</rb><rp>(</rp><rt>군</rt><rp>)</rp></ruby> <ruby class="hanja api grade60"><rb>言語</rb><rp>(</rp><rt>언어</rt><rp>)</rp></ruby>. 아프리카너들과 케이프 말레이인들 및 케이프 컬러드들이 <ruby class="hanja api grade70"><rb>主</rb><rp>(</rp><rt>주</rt><rp>)</rp></ruby>로 <ruby class="hanja api grade60"><rb>使用</rb><rp>(</rp><rt>사용</rt><rp>)</rp></ruby>한다.
