<p>그리고 세리에 B는 22팀으로 <ruby class="hanja api grade40"><rb>構成</rb><rp>(</rp><rt>구성</rt><rp>)</rp></ruby>되어 있다.

</p><p>세리에 B <ruby class="hanja api grade50"><rb>競技</rb><rp>(</rp><rt>경기</rt><rp>)</rp></ruby>는 <ruby class="hanja api grade40"><rb>普通</rb><rp>(</rp><rt>보통</rt><rp>)</rp></ruby> <ruby class="hanja api grade50"><rb>土曜日</rb><rp>(</rp><rt>토요일</rt><rp>)</rp></ruby>에 합니다. 하지만 <ruby class="hanja api grade52"><rb>例外的</rb><rp>(</rp><rt>예외적</rt><rp>)</rp></ruby>으로 <ruby class="hanja api grade72"><rb>每</rb><rp>(</rp><rt>매</rt><rp>)</rp></ruby> 라운드마다 한 <ruby class="hanja api grade50"><rb>競技</rb><rp>(</rp><rt>경기</rt><rp>)</rp></ruby>는 <ruby class="hanja api grade50"><rb>金曜日</rb><rp>(</rp><rt>금요일</rt><rp>)</rp></ruby>에 일찍 하고 한 <ruby class="hanja api grade50"><rb>競技</rb><rp>(</rp><rt>경기</rt><rp>)</rp></ruby>는 다음<ruby class="hanja api grade52"><rb>週</rb><rp>(</rp><rt>주</rt><rp>)</rp></ruby> <ruby class="hanja api grade50"><rb>月曜日</rb><rp>(</rp><rt>월요일</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade40"><rb>延期</rb><rp>(</rp><rt>연기</rt><rp>)</rp></ruby>됩니다. 세리에 B도 몇 <ruby class="hanja api grade52"><rb>週</rb><rp>(</rp><rt>주</rt><rp>)</rp></ruby> <ruby class="hanja api grade50"><rb>水曜日</rb><rp>(</rp><rt>수요일</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade50"><rb>競技</rb><rp>(</rp><rt>경기</rt><rp>)</rp></ruby>를 하여 42라운드 리그를 모두 <ruby class="hanja api grade32"><rb>計劃</rb><rp>(</rp><rt>계획</rt><rp>)</rp></ruby>하고 있습니다. 세리에 A가 그 <ruby class="hanja api grade52"><rb>州</rb><rp>(</rp><rt>주</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade62"><rb>休戰</rb><rp>(</rp><rt>휴전</rt><rp>)</rp></ruby>이라면 세리에 B는 <ruby class="hanja api grade50"><rb>日曜日</rb><rp>(</rp><rt>일요일</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade42"><rb>進行</rb><rp>(</rp><rt>진행</rt><rp>)</rp></ruby>됩니다.

<h2><span id=".E9.99.8D.E7.AD.89.28.EA.B0.95.EB.93.B1.29"></span><span class="mw-headline" id="降等(강등)"><ruby class="hanja api grade40"><rb>降等</rb><rp>(</rp><rt>강등</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EC%84%B8%EB%A6%AC%EC%97%90_B&amp;action=edit&amp;section=4" title="部分　編輯：降等(강등)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
