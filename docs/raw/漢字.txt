<h2><span id=".E7.89.B9.E5.BE.B5.28.ED.8A.B9.EC.A7.95.29"></span><span class="mw-headline" id="特徵(특징)"><ruby class="hanja grade32"><rb>特徵</rb><rp>(</rp><rt>특징</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%BC%A2%E5%AD%97&amp;action=edit&amp;section=1" title="부분 편집: 特徵(특징)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja grade42"><rb>個數</rb><rp>(</rp><rt>개수</rt><rp>)</rp></ruby>가 매우 많다! <a href="/wiki/%E6%AD%B7%E5%8F%B2" title="歷史"><ruby class="hanja grade52"><rb>歷史</rb><rp>(</rp><rt>역사</rt><rp>)</rp></ruby></a><ruby class="hanja grade72"><rb>上</rb><rp>(</rp><rt>상</rt><rp>)</rp></ruby> 있었던 글<ruby class="hanja grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>를 모두 세어 보면 <ruby class="hanja grade42"><rb>基準</rb><rp>(</rp><rt>기준</rt><rp>)</rp></ruby>에 따라 다르긴 하지만 8<ruby class="hanja grade70"><rb>萬字</rb><rp>(</rp><rt>만자</rt><rp>)</rp></ruby> <ruby class="hanja grade42"><rb>程度</rb><rp>(</rp><rt>정도</rt><rp>)</rp></ruby> 된다고 한다. <ruby class="hanja grade32"><rb>勿論</rb><rp>(</rp><rt>물론</rt><rp>)</rp></ruby> 이 8<ruby class="hanja grade70"><rb>萬字</rb><rp>(</rp><rt>만자</rt><rp>)</rp></ruby>를 모두 쓰는 <del>미친 짓을 하는 </del>나라는 어디에도 없고, <a href="/wiki/%E5%9C%8B%E5%AE%B6" title="國家"><ruby class="hanja grade72"><rb>國家</rb><rp>(</rp><rt>국가</rt><rp>)</rp></ruby></a><ruby class="hanja grade42"><rb>單位</rb><rp>(</rp><rt>단위</rt><rp>)</rp></ruby>로 <a href="/index.php?title=%E5%B8%B8%E7%94%A8%E6%BC%A2%E5%AD%97&amp;action=edit&amp;redlink=1" class="new" title="常用漢字 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>常用漢字</rb><rp>(</rp><rt>상용한자</rt><rp>)</rp></ruby></a> 같은 걸 <ruby class="hanja grade42"><rb>指定</rb><rp>(</rp><rt>지정</rt><rp>)</rp></ruby>해서 2~3<ruby class="hanja grade70"><rb>千字</rb><rp>(</rp><rt>천자</rt><rp>)</rp></ruby> <ruby class="hanja grade42"><rb>程度</rb><rp>(</rp><rt>정도</rt><rp>)</rp></ruby>로 <ruby class="hanja grade42"><rb>制限</rb><rp>(</rp><rt>제한</rt><rp>)</rp></ruby>해서 쓰고 있다.
</p><p>이렇게 글<ruby class="hanja grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby> <ruby class="hanja grade70"><rb>數</rb><rp>(</rp><rt>수</rt><rp>)</rp></ruby>가 많은 <ruby class="hanja grade60"><rb>理由</rb><rp>(</rp><rt>이유</rt><rp>)</rp></ruby>는 <ruby class="hanja grade62"><rb>各</rb><rp>(</rp><rt>각</rt><rp>)</rp></ruby> 글<ruby class="hanja grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>마다 뜻을 가지고 있기 때문이다. <ruby class="hanja grade60"><rb>古代</rb><rp>(</rp><rt>고대</rt><rp>)</rp></ruby> <a href="/wiki/%E8%B1%A1%E5%BD%A2%E6%96%87%E5%AD%97" title="象形文字"><ruby class="hanja grade40"><rb>象形文字</rb><rp>(</rp><rt>상형문자</rt><rp>)</rp></ruby></a>가 <a href="/index.php?title=%E8%A1%A8%E9%9F%B3%E6%96%87%E5%AD%97&amp;action=edit&amp;redlink=1" class="new" title="表音文字 (없는 문서)"><ruby class="hanja grade62 unknown"><rb>表音文字</rb><rp>(</rp><rt>표음문자</rt><rp>)</rp></ruby></a>로 바뀌지 않고 <a href="/index.php?title=%E8%A1%A8%E8%AA%9E%E6%96%87%E5%AD%97&amp;action=edit&amp;redlink=1" class="new" title="表語文字 (없는 문서)"><ruby class="hanja grade62 unknown"><rb>表語文字</rb><rp>(</rp><rt>표어문자</rt><rp>)</rp></ruby></a>로서 나름 <ruby class="hanja grade52"><rb>發展</rb><rp>(</rp><rt>발전</rt><rp>)</rp></ruby>해온 셈.

<p>그 <ruby class="hanja grade40"><rb>起源</rb><rp>(</rp><rt>기원</rt><rp>)</rp></ruby>은 <ruby class="hanja grade60"><rb>古代</rb><rp>(</rp><rt>고대</rt><rp>)</rp></ruby>에 <a href="/index.php?title=%E5%8D%A0&amp;action=edit&amp;redlink=1" class="new" title="占 (없는 문서)"><ruby class="hanja grade40 unknown"><rb>占</rb><rp>(</rp><rt>점</rt><rp>)</rp></ruby></a>의<ruby class="hanja grade52"><rb>結果</rb><rp>(</rp><rt>결과</rt><rp>)</rp></ruby>를 <ruby class="hanja grade42"><rb>記錄</rb><rp>(</rp><rt>기록</rt><rp>)</rp></ruby>하기<ruby class="hanja grade42"><rb>爲</rb><rp>(</rp><rt>위</rt><rp>)</rp></ruby>해 <a href="/index.php?title=%EA%B1%B0%EB%B6%81%EC%9D%B4&amp;action=edit&amp;redlink=1" class="new" title="거북이 (없는 문서)">거북이</a> 등껍질에 <ruby class="hanja grade42"><rb>直接</rb><rp>(</rp><rt>직접</rt><rp>)</rp></ruby>새긴 '<a href="/wiki/%E7%94%B2%E9%AA%A8%E6%96%87" class="mw-redirect" title="甲骨文"><ruby class="hanja grade40"><rb>甲骨文</rb><rp>(</rp><rt>갑골문</rt><rp>)</rp></ruby></a>'으로 여긴다. <a href="/wiki/%E5%80%89%E9%A0%A1" title="倉頡"><ruby class="hanja grade0"><rb>倉頡</rb><rp>(</rp><rt>창힐</rt><rp>)</rp></ruby></a>에<ruby class="hanja grade40"><rb>依</rb><rp>(</rp><rt>의</rt><rp>)</rp></ruby>해 <ruby class="hanja grade42"><rb>創製</rb><rp>(</rp><rt>창제</rt><rp>)</rp></ruby>되었다는 <ruby class="hanja grade52"><rb>傳說</rb><rp>(</rp><rt>전설</rt><rp>)</rp></ruby>도 있지만, <a href="/index.php?title=%E8%A8%80%E8%AA%9E%E5%AD%B8&amp;action=edit&amp;redlink=1" class="new" title="言語學 (없는 문서)"><ruby class="hanja grade60 unknown"><rb>言語學</rb><rp>(</rp><rt>언어학</rt><rp>)</rp></ruby></a>에서는 <ruby class="hanja grade40"><rb>甲骨文</rb><rp>(</rp><rt>갑골문</rt><rp>)</rp></ruby> <ruby class="hanja grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby> <ruby class="hanja grade50"><rb>順序</rb><rp>(</rp><rt>순서</rt><rp>)</rp></ruby>대로 <ruby class="hanja grade52"><rb>發展</rb><rp>(</rp><rt>발전</rt><rp>)</rp></ruby>했다고 본다.
