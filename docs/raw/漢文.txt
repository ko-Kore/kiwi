<h2><span id=".E7.8F.BE.E6.B3.81.28.ED.98.84.ED.99.A9.29"></span><span class="mw-headline" id="現況(현황)"><ruby class="hanja api grade40"><rb>現況</rb><rp>(</rp><rt>현황</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%BC%A2%E6%96%87&amp;action=edit&amp;section=1" title="부분 편집: 現況(현황)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja api grade20"><rb>臺灣</rb><rp>(</rp><rt>대만</rt><rp>)</rp></ruby>으로 옮겨간 <a href="/wiki/%E4%B8%AD%E8%8F%AF%E6%B0%91%E5%9C%8B" title="中華民國"><ruby class="hanja api grade40"><rb>中華民國</rb><rp>(</rp><rt>중화민국</rt><rp>)</rp></ruby></a><ruby class="hanja api grade42"><rb>政府</rb><rp>(</rp><rt>정부</rt><rp>)</rp></ruby>에서는 <ruby class="hanja api grade70"><rb>漢文</rb><rp>(</rp><rt>한문</rt><rp>)</rp></ruby>이 흔히 <a href="/index.php?title=%E5%85%AC%E6%96%87%E6%9B%B8&amp;action=edit&amp;redlink=1" class="new" title="公文書 (없는 문서)"><ruby class="hanja api grade62 unknown"><rb>公文書</rb><rp>(</rp><rt>공문서</rt><rp>)</rp></ruby></a>에 <ruby class="hanja api grade60"><rb>使用</rb><rp>(</rp><rt>사용</rt><rp>)</rp></ruby>된다.
</p><p><ruby class="hanja api grade0"><rb>東아시아</rb><rp>(</rp><rt>동아시아</rt><rp>)</rp></ruby>의 다른 나라에서는 <ruby class="hanja api grade80"><rb>學校</rb><rp>(</rp><rt>학교</rt><rp>)</rp></ruby>에서는 <a href="/index.php?title=%E6%BC%A2%E6%96%87_(%E6%95%8E%E7%A7%91)&amp;action=edit&amp;redlink=1" class="new" title="漢文 (敎科) (없는 문서)"><ruby class="hanja api grade70 unknown"><rb>漢文</rb><rp>(</rp><rt>한문</rt><rp>)</rp></ruby>을 가르치고 있지만</a> <ruby class="hanja api grade42"><rb>日常生活</rb><rp>(</rp><rt>일상생활</rt><rp>)</rp></ruby>에서는 더 <ruby class="hanja api grade52"><rb>以上</rb><rp>(</rp><rt>이상</rt><rp>)</rp></ruby> <ruby class="hanja api grade70"><rb>漢文</rb><rp>(</rp><rt>한문</rt><rp>)</rp></ruby>을 <ruby class="hanja api grade60"><rb>使用</rb><rp>(</rp><rt>사용</rt><rp>)</rp></ruby>하지 않는다.
</p>

<p><a href="/wiki/%E4%B8%AD%E5%9C%8B%E8%AA%9E" title="中國語"><ruby class="hanja api grade70"><rb>中國語</rb><rp>(</rp><rt>중국어</rt><rp>)</rp></ruby></a>와는 다르다! 흔히 말하는 <ruby class="hanja api
