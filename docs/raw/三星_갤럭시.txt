<h2><span id=".E6.AD.B7.E5.8F.B2.28.EC.97.AD.EC.82.AC.29"></span><span class="mw-headline" id="歷史(역사)"><ruby class="hanja api grade52"><rb>歷史</rb><rp>(</rp><rt>역사</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E4%B8%89%E6%98%9F_%EA%B0%A4%EB%9F%AD%EC%8B%9C&amp;action=edit&amp;section=1" title="部分 編輯: 歷史(역사)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja api grade70"><rb>出市</rb><rp>(</rp><rt>출시</rt><rp>)</rp></ruby> <ruby class="hanja api grade50"><rb>初期</rb><rp>(</rp><rt>초기</rt><rp>)</rp></ruby>에는 <a href="/index.php?title=%EC%95%A0%ED%94%8C&amp;action=edit&amp;redlink=1" class="new" title="애플 (없는 文書)">애플</a> <a href="/wiki/%EC%95%84%EC%9D%B4%ED%8F%B0" title="아이폰">아이폰</a>에 <ruby class="hanja api grade62"><rb>對</rb><rp>(</rp><rt>대</rt><rp>)</rp></ruby>한 <ruby class="hanja api grade10"><rb>剽竊</rb><rp>(</rp><rt>표절</rt><rp>)</rp></ruby> <ruby class="hanja api grade42"><rb>論難</rb><rp>(</rp><rt>논난</rt><rp>)</rp></ruby>과 함께 <ruby class="hanja api grade50"><rb>特許</rb><rp>(</rp><rt>특허</rt><rp>)</rp></ruby> <ruby class="hanja api grade32"><rb>紛爭</rb><rp>(</rp><rt>분쟁</rt><rp>)</rp></ruby>에 휘말렸다. 이 <ruby class="hanja api grade50"><rb>特許</rb><rp>(</rp><rt>특허</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>攻防</rb><rp>(</rp><rt>공방</rt><rp>)</rp></ruby>은 자그마치 7<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby>동안 <ruby class="hanja api grade40"><rb>繼續</rb><rp>(</rp><rt>계속</rt><rp>)</rp></ruby>되다 <a href="/index.php?title=2018%E5%B9%B4&amp;action=edit&amp;redlink=1" class="new" title="2018年 (없는 文書)">2018<ruby class="hanja api grade80 unknown"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby></a> <ruby class="hanja api grade60"><rb>合意</rb><rp>(</rp><rt>합의</rt><rp>)</rp></ruby>가 이뤄져 끝나게 되었다.
</p>
