<p>키보드<ruby class="hanja grade72"><rb>上</rb><rp>(</rp><rt>상</rt><rp>)</rp></ruby>에 <ruby class="hanja grade42"><rb>配置</rb><rp>(</rp><rt>배치</rt><rp>)</rp></ruby>된 <ruby class="hanja grade62"><rb>各</rb><rp>(</rp><rt>각</rt><rp>)</rp></ruby> 글<ruby class="hanja grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>는 <ruby class="hanja grade52"><rb>元來</rb><rp>(</rp><rt>원래</rt><rp>)</rp></ruby> <ruby class="hanja grade40"><rb>模樣</rb><rp>(</rp><rt>모양</rt><rp>)</rp></ruby>이 그대로 쓰이기도 하지만, <ruby class="hanja grade52"><rb>變形</rb><rp>(</rp><rt>변형</rt><rp>)</rp></ruby>되어 쓰이는 일도 많습니다. <a href="/wiki/%E4%BA%BA" title="人"><ruby class="hanja grade80"><rb>人</rb><rp>(</rp><rt>인</rt><rp>)</rp></ruby></a>-&gt;<ruby class="hanja grade0"><rb>亻</rb><rp>(</rp><rt>인</rt><rp>)</rp></ruby>이나 <a href="/wiki/%E6%B0%B4" title="水"><ruby class="hanja grade80"><rb>水</rb><rp>(</rp><rt>수</rt><rp>)</rp></ruby></a>-&gt;<ruby class="hanja grade0"><rb>氵</rb><rp>(</rp><rt>수</rt><rp>)</rp></ruby>같이 <a href="/wiki/%E6%BC%A2%E5%AD%97" title="漢字"><ruby class="hanja grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby></a>의 <ruby class="hanja grade50"><rb>原理</rb><rp>(</rp><rt>원리</rt><rp>)</rp></ruby>를 조금만 알아도 <ruby class="hanja grade40"><rb>推測</rb><rp>(</rp><rt>추측</rt><rp>)</rp></ruby>할 수 있는 것부터 <a href="/index.php?title=%E7%94%B0&amp;action=edit&amp;redlink=1" class="new" title="田 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>田</rb><rp>(</rp><rt>전</rt><rp>)</rp></ruby></a>-&gt;<ruby class="hanja grade0"><rb>囗</rb><rp>(</rp><rt>국</rt><rp>)</rp></ruby>, <a href="/index.php?title=%E5%BB%BF&amp;action=edit&amp;redlink=1" class="new" title="廿 (없는 문서)"><ruby class="hanja grade0 unknown"><rb>廿</rb><rp>(</rp><rt>입</rt><rp>)</rp></ruby></a>-&gt;<ruby class="hanja grade0"><rb>艹</rb><rp>(</rp><rt>초</rt><rp>)</rp></ruby>같이 조금 <ruby class="hanja grade42"><rb>難解</rb><rp>(</rp><rt>난해</rt><rp>)</rp></ruby>하지만 <ruby class="hanja grade30"><rb>首肯</rb><rp>(</rp><rt>수긍</rt><rp>)</rp></ruby>은 되는것, <a href="/index.php?title=%E5%8D%9C&amp;action=edit&amp;redlink=1" class="new" title="卜 (없는 문서)"><ruby class="hanja grade30 unknown"><rb>卜</rb><rp>(</rp><rt>복</rt><rp>)</rp></ruby></a>-&gt;<ruby class="hanja grade0"><rb>辶</rb><rp>(</rp><rt>착</rt><rp>)</rp></ruby><sup id="cite_ref-1" class="reference"><a href="#cite_note-1">[1]</a></sup>, <a href="/wiki/%E5%8D%81" title="十"><ruby class="hanja grade80"><rb>十</rb><rp>(</rp><rt>십</rt><rp>)</rp></ruby></a>-&gt;<ruby class="hanja grade0"><rb>宀</rb><rp>(</rp><rt>면</rt><rp>)</rp></ruby> <ruby class="hanja grade62"><rb>等等</rb><rp>(</rp><rt>등등</rt><rp>)</rp></ruby> 이건 어떻게 생각하면 저런 <ruby class="hanja grade50"><rb>規則</rb><rp>(</rp><rt>규칙</rt><rp>)</rp></ruby>이 나오는지 <ruby class="hanja grade42"><rb>理解</rb><rp>(</rp><rt>이해</rt><rp>)</rp></ruby>도 되지 않는 <ruby class="hanja grade52"><rb>部類</rb><rp>(</rp><rt>부류</rt><rp>)</rp></ruby>까지 <ruby class="hanja grade40"><rb>多樣</rb><rp>(</rp><rt>다양</rt><rp>)</rp></ruby>한 <ruby class="hanja grade42"><rb>形態</rb><rp>(</rp><rt>형태</rt><rp>)</rp></ruby>로 <ruby class="hanja grade52"><rb>變形</rb><rp>(</rp><rt>변형</rt><rp>)</rp></ruby>됩니다.
</p><p>여기서 <ruby class="hanja grade52"><rb>部品</rb><rp>(</rp><rt>부품</rt><rp>)</rp></ruby>의 <ruby class="hanja grade52"><rb>變形</rb><rp>(</rp><rt>변형</rt><rp>)</rp></ruby>은 <ruby class="hanja grade42"><rb>常識的</rb><rp>(</rp><rt>상식적</rt><rp>)</rp></ruby>으로 <ruby class="hanja grade52"><rb>變形</rb><rp>(</rp><rt>변형</rt><rp>)</rp></ruby>되는 <ruby class="hanja grade40"><rb>模樣</rb><rp>(</rp><rt>모양</rt><rp>)</rp></ruby> 뿐만이 아니라, 거기서 글<ruby class="hanja grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>를 돌리거나 <ruby class="hanja grade62"><rb>一部</rb><rp>(</rp><rt>일부</rt><rp>)</rp></ruby>를 <ruby class="hanja grade42"><rb>取</rb><rp>(</rp><rt>취</rt><rp>)</rp></ruby>한다고 생각하시면 조금이라도 <ruby class="hanja grade32"><rb>蓋然性</rb><rp>(</rp><rt>개연성</rt><rp>)</rp></ruby>을 <ruby class="hanja grade40"><rb>納得</rb><rp>(</rp><rt>납득</rt><rp>)</rp></ruby>하실수도 있겠습니다. <ruby class="hanja grade60"><rb>例</rb><rp>(</rp><rt>예</rt><rp>)</rp></ruby>를 들어 <ruby class="hanja grade80"><rb>山</rb><rp>(</rp><rt>산</rt><rp>)</rp></ruby>의 <ruby class="hanja grade72"><rb>中間</rb><rp>(</rp><rt>중간</rt><rp>)</rp></ruby> 세로<ruby class="hanja grade32"><rb>劃</rb><rp>(</rp><rt>획</rt><rp>)</rp></ruby>만을 따와 <ruby class="hanja grade80"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby>의 아래쪽을 막으면 <ruby class="hanja grade60"><rb>目</rb><rp>(</rp><rt>목</rt><rp>)</rp></ruby>(<ruby class="hanja grade80"><rb>月山</rb><rp>(</rp><rt>월산</rt><rp>)</rp></ruby>), 오른쪽 <ruby class="hanja grade62"><rb>半</rb><rp>(</rp><rt>반</rt><rp>)</rp></ruby>만 가져오면 <ruby class="hanja grade0"><rb>乚</rb><rp>(</rp><rt>은</rt><rp>)</rp></ruby>... 같은 <ruby class="hanja grade60"><rb>式</rb><rp>(</rp><rt>식</rt><rp>)</rp></ruby>의 <ruby class="hanja grade50"><rb>規則性</rb><rp>(</rp><rt>규칙성</rt><rp>)</rp></ruby>을 가지고 있어 통으로 외워야할 <ruby class="hanja grade42"><rb>程度</rb><rp>(</rp><rt>정도</rt><rp>)</rp></ruby>까지는 아닙니다.
</p><p>여기서 <ruby class="hanja grade20"><rb>戈</rb><rp>(</rp><rt>과</rt><rp>)</rp></ruby>는 <ruby class="hanja grade40"><rb>點</rb><rp>(</rp><rt>점</rt><rp>)</rp></ruby> 하나, <ruby class="hanja grade80"><rb>金</rb><rp>(</rp><rt>김</rt><rp>)</rp></ruby>은 가로로 된 <ruby class="hanja grade40"><rb>點</rb><rp>(</rp><rt>점</rt><rp>)</rp></ruby> 두<ruby class="hanja grade42"><rb>個</rb><rp>(</rp><rt>개</rt><rp>)</rp></ruby>, <ruby class="hanja grade30"><rb>卜</rb><rp>(</rp><rt>복</rt><rp>)</rp></ruby>은 세로<ruby class="hanja grade40"><rb>點</rb><rp>(</rp><rt>점</rt><rp>)</rp></ruby> 2<ruby class="hanja grade42"><rb>個</rb><rp>(</rp><rt>개</rt><rp>)</rp></ruby>, <ruby class="hanja grade80"><rb>水</rb><rp>(</rp><rt>수</rt><rp>)</rp></ruby>는 가로<ruby class="hanja grade40"><rb>點</rb><rp>(</rp><rt>점</rt><rp>)</rp></ruby> 3<ruby class="hanja grade42"><rb>個</rb><rp>(</rp><rt>개</rt><rp>)</rp></ruby>... 와 같이 많이 쓰이는 <ruby class="hanja grade42"><rb>形態</rb><rp>(</rp><rt>형태</rt><rp>)</rp></ruby>는 미리 외워놓으실 수도 있겠습니다.
</p>
