</p><p><a href="/index.php?title=%E9%96%8B%E5%8C%96%E6%9C%9F&amp;action=edit&amp;redlink=1" class="new" title="開化期 (없는 문서)"><ruby class="hanja grade50 unknown"><rb>開化期</rb><rp>(</rp><rt>개화기</rt><rp>)</rp></ruby></a>에 들어서 <a href="/index.php?title=%E8%A5%BF%E9%81%8A%E8%A6%8B%E8%81%9E&amp;action=edit&amp;redlink=1" class="new" title="西遊見聞 (없는 문서)"><ruby class="hanja grade40 unknown"><rb>西遊見聞</rb><rp>(</rp><rt>서유견문</rt><rp>)</rp></ruby></a>에서 <ruby class="hanja grade50"><rb>最初</rb><rp>(</rp><rt>최초</rt><rp>)</rp></ruby>로 <ruby class="hanja grade60"><rb>現代式</rb><rp>(</rp><rt>현대식</rt><rp>)</rp></ruby> <ruby class="hanja grade70"><rb>國漢文</rb><rp>(</rp><rt>국한문</rt><rp>)</rp></ruby>에 가까운 <ruby class="hanja grade62"><rb>文體</rb><rp>(</rp><rt>문체</rt><rp>)</rp></ruby>가 <ruby class="hanja grade60"><rb>使用</rb><rp>(</rp><rt>사용</rt><rp>)</rp></ruby>되었고, <a href="/index.php?title=%E7%94%B2%E5%8D%88%E6%94%B9%E9%9D%A9&amp;action=edit&amp;redlink=1" class="new" title="甲午改革 (없는 문서)"><ruby class="hanja grade40 unknown"><rb>甲午改革</rb><rp>(</rp><rt>갑오개혁</rt><rp>)</rp></ruby></a> <ruby class="hanja grade52"><rb>以後</rb><rp>(</rp><rt>이후</rt><rp>)</rp></ruby>로 <ruby class="hanja grade42"><rb>法律</rb><rp>(</rp><rt>법률</rt><rp>)</rp></ruby>과 <ruby class="hanja grade10"><rb>勅令</rb><rp>(</rp><rt>칙령</rt><rp>)</rp></ruby> <ruby class="hanja grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby>에서까지 널리 <ruby class="hanja grade60"><rb>使用</rb><rp>(</rp><rt>사용</rt><rp>)</rp></ruby>되게 되었다.
</p>
<h2><span id=".E8.A1.B0.E9.80.80.28.EC.87.A0.ED.87.B4.29"></span><span class="mw-headline" id="衰退(쇠퇴)"><ruby class="hanja grade32"><rb>衰退</rb><rp>(</rp><rt>쇠퇴</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E5%9C%8B%E6%BC%A2%E6%96%87%E6%B7%B7%E7%94%A8&amp;action=edit&amp;section=2" title="부분 편집: 衰退(쇠퇴)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/index.php?title=1969%E5%B9%B4&amp;action=edit&amp;redlink=1" class="new" title="1969年 (없는 문서)">1969<ruby class="hanja grade80 unknown"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby></a> <a href="/index.php?title=%ED%95%9C%EA%B8%80%E5%B0%88%E7%94%A8_%E4%BA%94%E5%80%8B%E5%B9%B4%E8%A8%88%E5%8A%83&amp;action=edit&amp;redlink=1" class="new" title="한글專用 五個年計劃 (없는 문서)">한글<ruby class="hanja grade40 unknown"><rb>專用</rb><rp>(</rp><rt>전용</rt><rp>)</rp></ruby> <ruby class="hanja grade32 unknown"><rb>五個年計劃</rb><rp>(</rp><rt>오개년계획</rt><rp>)</rp></ruby></a>이 <ruby class="hanja grade42"><rb>施行</rb><rp>(</rp><rt>시행</rt><rp>)</rp></ruby>되고 <ruby class="hanja grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby>에 <ruby class="hanja grade62"><rb>對</rb><rp>(</rp><rt>대</rt><rp>)</rp></ruby>한 <ruby class="hanja grade70"><rb>敎育</rb><rp>(</rp><rt>교육</rt><rp>)</rp></ruby>이 <ruby class="hanja grade42"><rb>中斷</rb><rp>(</rp><rt>중단</rt><rp>)</rp></ruby>되었지만, 그 <ruby class="hanja grade72"><rb>後</rb><rp>(</rp><rt>후</rt><rp>)</rp></ruby> 얼마<ruby class="hanja grade72"><rb>間</rb><rp>(</rp><rt>간</rt><rp>)</rp></ruby>은 <a href="/index.php?title=%E6%96%B0%E8%81%9E&amp;action=edit&amp;redlink=1" class="new" title="新聞 (없는 문서)"><ruby class="hanja grade62 unknown"><rb>新聞</rb><rp>(</rp><rt>신문</rt><rp>)</rp></ruby></a>이나 <ruby class="hanja grade32"><rb>媒體</rb><rp>(</rp><rt>매체</rt><rp>)</rp></ruby> <ruby class="hanja grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby> <ruby class="hanja grade42"><rb>日常</rb><rp>(</rp><rt>일상</rt><rp>)</rp></ruby>에서는 <ruby class="hanja grade40"><rb>國漢文混用體</rb><rp>(</rp><rt>국한문혼용체</rt><rp>)</rp></ruby>가 <ruby class="hanja grade42"><rb>如前</rb><rp>(</rp><rt>여전</rt><rp>)</rp></ruby>히 많이 쓰이고 있었다.
</p><p>그러나 <a href="/index.php?title=%ED%95%9C%EA%B8%80%E4%B8%96%E4%BB%A3&amp;action=edit&amp;redlink=1" class="new" title="한글世代 (없는 문서)">한글<ruby class="hanja grade62 unknown"><rb>世代</rb><rp>(</rp><rt>세대</rt><rp>)</rp></ruby></a>가 늘어나며 <a href="/wiki/%E6%BC%A2%E5%AD%97" title="漢字"><ruby class="hanja grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby></a>의 <ruby class="hanja grade40"><rb>識者層</rb><rp>(</rp><rt>식자층</rt><rp>)</rp></ruby>이 <ruby class="hanja grade32"><rb>漸次</rb><rp>(</rp><rt>점차</rt><rp>)</rp></ruby> 줄어들면서 <ruby class="hanja grade62"><rb>新聞</rb><rp>(</rp><rt>신문</rt><rp>)</rp></ruby>에서도 <ruby class="hanja grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby>는 <ruby class="hanja grade70"><rb>數字</rb><rp>(</rp><rt>숫자</rt><rp>)</rp></ruby> <ruby class="hanja grade42"><rb>程度</rb><rp>(</rp><rt>정도</rt><rp>)</rp></ruby>에나 쓰게 되다가, 그마저도 <a href="/wiki/%EA%B0%80%EB%A1%9C%EC%93%B0%EA%B8%B0" class="mw-redirect" title="가로쓰기">가로쓰기</a>와 함께 사라지며 <a href="/wiki/%ED%95%9C%EA%B8%80%E5%B0%88%E7%94%A8" title="한글專用">한글<ruby class="hanja grade40"><rb>專用</rb><rp>(</rp><rt>전용</rt><rp>)</rp></ruby></a>이 되었고, <ruby class="hanja grade70"><rb>國漢文</rb><rp>(</rp><rt>국한문</rt><rp>)</rp></ruby>으로 되어 있던 <a href="/index.php?title=%E6%B3%95%E5%BE%8B&amp;action=edit&amp;redlink=1" class="new" title="法律 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>法律</rb><rp>(</rp><rt>법률</rt><rp>)</rp></ruby></a>마저도 모두 한글<ruby class="hanja grade40"><rb>專用</rb><rp>(</rp><rt>전용</rt><rp>)</rp></ruby>으로 <ruby class="hanja grade32"><rb>改編</rb><rp>(</rp><rt>개편</rt><rp>)</rp></ruby>하는 <ruby class="hanja grade62"><rb>作業</rb><rp>(</rp><rt>작업</rt><rp>)</rp></ruby>을 하여 <ruby class="hanja grade60"><rb>現在</rb><rp>(</rp><rt>현재</rt><rp>)</rp></ruby>는 <ruby class="hanja grade70"><rb>國漢文</rb><rp>(</rp><rt>국한문</rt><rp>)</rp></ruby>으로 된 <ruby class="hanja grade62"><rb>文書</rb><rp>(</rp><rt>문서</rt><rp>)</rp></ruby>나 <ruby class="hanja grade32"><rb>媒體</rb><rp>(</rp><rt>매체</rt><rp>)</rp></ruby>를 거의 찾아볼 수 없게 되었다.
</p>
