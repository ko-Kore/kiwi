<p>세로로 길고 <ruby class="hanja api grade40"><rb>機器</rb><rp>(</rp><rt>기기</rt><rp>)</rp></ruby> 아래쪽을 <ruby class="hanja api grade42"><rb>兩</rb><rp>(</rp><rt>양</rt><rp>)</rp></ruby>손으로 잡고 플레이하는 <ruby class="hanja api grade50"><rb>初代</rb><rp>(</rp><rt>초대</rt><rp>)</rp></ruby> <a href="/wiki/%EA%B2%8C%EC%9E%84%EB%B3%B4%EC%9D%B4" title="게임보이">게임보이</a>부터 이어온 <ruby class="hanja api grade40"><rb>形象</rb><rp>(</rp><rt>형상</rt><rp>)</rp></ruby>이 가로<ruby class="hanja api grade62"><rb>形</rb><rp>(</rp><rt>형</rt><rp>)</rp></ruby>으로 바뀌었다. 이에 <del>더<ruby class="hanja api grade52"><rb>以上</rb><rp>(</rp><rt>이상</rt><rp>)</rp></ruby> 한손으로 플레이할수는 없게 되었지만 </del>트리거 버튼(L, R)이 <ruby class="hanja api grade32"><rb>追加</rb><rp>(</rp><rt>추가</rt><rp>)</rp></ruby>되었다. <ruby class="hanja api grade62"><rb>代表</rb><rp>(</rp><rt>대표</rt><rp>)</rp></ruby> 컬러는 인디고(<ruby class="hanja api grade20"><rb>藍色</rb><rp>(</rp><rt>람색</rt><rp>)</rp></ruby>).
