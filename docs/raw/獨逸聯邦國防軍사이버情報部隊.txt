<p><ruby class="hanja api grade30"><rb>旣存</rb><rp>(</rp><rt>기존</rt><rp>)</rp></ruby> <ruby class="hanja api grade52"><rb>陸海空軍</rb><rp>(</rp><rt>육해공군</rt><rp>)</rp></ruby>에 흩어져 있던 사이버<ruby class="hanja api grade42"><rb>情報</rb><rp>(</rp><rt>정보</rt><rp>)</rp></ruby> <ruby class="hanja api grade42"><rb>部隊</rb><rp>(</rp><rt>부대</rt><rp>)</rp></ruby>를 <ruby class="hanja api grade42"><rb>統合</rb><rp>(</rp><rt>통합</rt><rp>)</rp></ruby>하여 2017<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> <ruby class="hanja api grade42"><rb>創設</rb><rp>(</rp><rt>창설</rt><rp>)</rp></ruby>한 사이버<ruby class="hanja api grade42"><rb>部隊</rb><rp>(</rp><rt>부대</rt><rp>)</rp></ruby>. <ruby class="hanja api grade60"><rb>本部</rb><rp>(</rp><rt>본부</rt><rp>)</rp></ruby>는 <a href="/index.php?title=%E6%9C%AC%E5%B8%82&amp;action=edit&amp;redlink=1" class="new" title="本市 (없는 文書)"><ruby class="hanja api grade60 unknown"><rb>本</rb><rp>(</rp><rt>본</rt><rp>)</rp></ruby><ruby class="hanja api grade72 unknown"><rb>市</rb><rp>(</rp><rt>시</rt><rp>)</rp></ruby></a>에 있다. <a href="/index.php?title=%E8%BB%8D%E9%9C%80%E8%BB%8D&amp;action=edit&amp;redlink=1" class="new" title="軍需軍 (없는 文書)"><ruby class="hanja api grade32 unknown"><rb>軍需</rb><rp>(</rp><rt>군수</rt><rp>)</rp></ruby><ruby class="hanja api grade80 unknown"><rb>軍</rb><rp>(</rp><rt>군</rt><rp>)</rp></ruby></a>, <ruby class="hanja api grade42"><rb>義務</rb><rp>(</rp><rt>의무</rt><rp>)</rp></ruby><ruby class="hanja api grade80"><rb>軍</rb><rp>(</rp><rt>군</rt><rp>)</rp></ruby>과 마찬가지로 <ruby class="hanja api grade52"><rb>陸海空軍</rb><rp>(</rp><rt>육해공군</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade42"><rb>人員</rb><rp>(</rp><rt>인원</rt><rp>)</rp></ruby>들이 <ruby class="hanja api grade40"><rb>配屬</rb><rp>(</rp><rt>배속</rt><rp>)</rp></ruby>돼 <ruby class="hanja api grade40"><rb>勤務</rb><rp>(</rp><rt>근무</rt><rp>)</rp></ruby>한다.
