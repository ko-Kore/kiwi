<p><ruby class="hanja api grade50"><rb>氷島</rb><rp>(</rp><rt>빙도</rt><rp>)</rp></ruby><ruby class="hanja api grade62"><rb>共和國</rb><rp>(</rp><rt>공화국</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>略稱</rb><rp>(</rp><rt>약칭</rt><rp>)</rp></ruby> <ruby class="hanja api grade50"><rb>氷島</rb><rp>(</rp><rt>빙도</rt><rp>)</rp></ruby> <ruby class="hanja api grade50"><rb>首都</rb><rp>(</rp><rt>수도</rt><rp>)</rp></ruby> 레이캬비크는 <ruby class="hanja api grade60"><rb>北大西洋</rb><rp>(</rp><rt>북대서양</rt><rp>)</rp></ruby> <ruby class="hanja api grade62"><rb>中部</rb><rp>(</rp><rt>중부</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade42"><rb>位置</rb><rp>(</rp><rt>위치</rt><rp>)</rp></ruby>하며 <a href="/index.php?title=%E5%8C%97%E6%A5%B5&amp;action=edit&amp;redlink=1" class="new" title="北極 (없는 文書)"><ruby class="hanja api grade42 unknown"><rb>北極</rb><rp>(</rp><rt>북극</rt><rp>)</rp></ruby></a><ruby class="hanja api grade20"><rb>圈</rb><rp>(</rp><rt>권</rt><rp>)</rp></ruby>과 가깝고 유럽에서 두 <ruby class="hanja api grade60"><rb>番</rb><rp>(</rp><rt>번</rt><rp>)</rp></ruby>째로 큰 섬이다.
