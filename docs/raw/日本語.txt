<h2><span id="槪要(개요)"></span><span class="mw-headline" id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"><ruby class="hanja grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%97%A5%E6%9C%AC%E8%AA%9E&amp;action=edit&amp;section=1" title="부분 편집: 槪要(개요)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%E6%97%A5%E6%9C%AC" title="日本"><ruby class="hanja grade60"><rb>日本</rb><rp>(</rp><rt>일본</rt><rp>)</rp></ruby></a>에서 <ruby class="hanja grade70"><rb>主</rb><rp>(</rp><rt>주</rt><rp>)</rp></ruby>로 <ruby class="hanja grade60"><rb>使用</rb><rp>(</rp><rt>사용</rt><rp>)</rp></ruby>하는 <ruby class="hanja grade60"><rb>言語</rb><rp>(</rp><rt>언어</rt><rp>)</rp></ruby>. <a href="/index.php?title=%E6%97%A5%E6%9C%AC%E8%AA%9E%E6%97%8F&amp;action=edit&amp;redlink=1" class="new" title="日本語族 (없는 문서)"><ruby class="hanja grade60 unknown"><rb>日本語族</rb><rp>(</rp><rt>일본어족</rt><rp>)</rp></ruby></a>에 <ruby class="hanja grade40"><rb>屬</rb><rp>(</rp><rt>속</rt><rp>)</rp></ruby>한다.
</p>
<h2><span id="音韻(음운)"></span><span class="mw-headline" id=".E9.9F.B3.E9.9F.BB.28.EC.9D.8C.EC.9A.B4.29"><ruby class="hanja grade32"><rb>音韻</rb><rp>(</rp><rt>음운</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%97%A5%E6%9C%AC%E8%AA%9E&amp;action=edit&amp;section=2" title="부분 편집: 音韻(음운)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja grade52"><rb>基本</rb><rp>(</rp><rt>기본</rt><rp>)</rp></ruby><a href="/index.php?title=%E6%AF%8D%E9%9F%B3&amp;action=edit&amp;redlink=1" class="new" title="母音 (없는 문서)"><ruby class="hanja grade62 unknown"><rb>母音</rb><rp>(</rp><rt>모음</rt><rp>)</rp></ruby></a>은 <ruby class="hanja grade52"><rb>五種</rb><rp>(</rp><rt>오종</rt><rp>)</rp></ruby>에 <a href="/index.php?title=%E5%AD%90%E9%9F%B3&amp;action=edit&amp;redlink=1" class="new" title="子音 (없는 문서)"><ruby class="hanja grade62 unknown"><rb>子音</rb><rp>(</rp><rt>자음</rt><rp>)</rp></ruby></a>은 <ruby class="hanja grade52"><rb>十種</rb><rp>(</rp><rt>십종</rt><rp>)</rp></ruby>으로 <a href="/wiki/%E9%9F%93%E5%9C%8B%E8%AA%9E" title="韓國語"><ruby class="hanja grade70"><rb>韓國語</rb><rp>(</rp><rt>한국어</rt><rp>)</rp></ruby></a>에 <ruby class="hanja grade50"><rb>比</rb><rp>(</rp><rt>비</rt><rp>)</rp></ruby>해 <ruby class="hanja grade42"><rb>單純</rb><rp>(</rp><rt>단순</rt><rp>)</rp></ruby>하다. <ruby class="hanja grade32"><rb>但</rb><rp>(</rp><rt>단</rt><rp>)</rp></ruby>, <ruby class="hanja grade30"><rb>類似</rb><rp>(</rp><rt>유사</rt><rp>)</rp></ruby>한 <a href="/index.php?title=%E9%9F%B3%E9%9F%BB&amp;action=edit&amp;redlink=1" class="new" title="音韻 (없는 문서)"><ruby class="hanja grade32 unknown"><rb>音韻</rb><rp>(</rp><rt>음운</rt><rp>)</rp></ruby></a>이라도 <ruby class="hanja grade42"><rb>正確</rb><rp>(</rp><rt>정확</rt><rp>)</rp></ruby>한 <ruby class="hanja grade62"><rb>發音</rb><rp>(</rp><rt>발음</rt><rp>)</rp></ruby>은 <ruby class="hanja grade70"><rb>韓國語</rb><rp>(</rp><rt>한국어</rt><rp>)</rp></ruby>와 다른 <ruby class="hanja grade40"><rb>境遇</rb><rp>(</rp><rt>경우</rt><rp>)</rp></ruby>도 있다.
</p><p><a href="/index.php?title=%E8%81%B2%E8%AA%BF&amp;action=edit&amp;redlink=1" class="new" title="聲調 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>聲調</rb><rp>(</rp><rt>성조</rt><rp>)</rp></ruby></a>는 따로 없지만 <a href="/index.php?title=%E6%8A%91%E6%8F%9A&amp;action=edit&amp;redlink=1" class="new" title="抑揚 (없는 문서)"><ruby class="hanja grade32 unknown"><rb>抑揚</rb><rp>(</rp><rt>억양</rt><rp>)</rp></ruby></a><ruby class="hanja grade60"><rb>區分</rb><rp>(</rp><rt>구분</rt><rp>)</rp></ruby>이 <ruby class="hanja grade40"><rb>複雜</rb><rp>(</rp><rt>복잡</rt><rp>)</rp></ruby>해서 <ruby class="hanja grade52"><rb>中級以上</rb><rp>(</rp><rt>중급이상</rt><rp>)</rp></ruby>의 <ruby class="hanja grade60"><rb>學習者</rb><rp>(</rp><rt>학습자</rt><rp>)</rp></ruby>들에게 <ruby class="hanja grade42"><rb>難解</rb><rp>(</rp><rt>난해</rt><rp>)</rp></ruby>한 <ruby class="hanja grade42"><rb>要素中</rb><rp>(</rp><rt>요소중</rt><rp>)</rp></ruby> 하나로 꼽힌다.
</p>
<h2><span id="文字(문자)"></span><span class="mw-headline" id=".E6.96.87.E5.AD.97.28.EB.AC.B8.EC.9E.90.29"><ruby class="hanja grade70"><rb>文字</rb><rp>(</rp><rt>문자</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%97%A5%E6%9C%AC%E8%AA%9E&amp;action=edit&amp;section=3" title="부분 편집: 文字(문자)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%E5%B9%B3%E5%81%87%E5%90%8D" title="平假名"><ruby class="hanja grade42"><rb>平假名</rb><rp>(</rp><rt>평가명</rt><rp>)</rp></ruby></a>, <a href="/wiki/%E7%89%87%E5%81%87%E5%90%8D" title="片假名"><ruby class="hanja grade32"><rb>片假名</rb><rp>(</rp><rt>편가명</rt><rp>)</rp></ruby></a>, <a href="/wiki/%E6%BC%A2%E5%AD%97" title="漢字"><ruby class="hanja grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby></a> 세 <ruby class="hanja grade52"><rb>種類</rb><rp>(</rp><rt>종류</rt><rp>)</rp></ruby> <ruby class="hanja grade70"><rb>文字</rb><rp>(</rp><rt>문자</rt><rp>)</rp></ruby>를 <ruby class="hanja grade60"><rb>使用</rb><rp>(</rp><rt>사용</rt><rp>)</rp></ruby>해 <ruby class="hanja grade62"><rb>表記</rb><rp>(</rp><rt>표기</rt><rp>)</rp></ruby>한다. <a href="/index.php?title=%EB%9D%84%EC%96%B4%EC%93%B0%EA%B8%B0&amp;action=edit&amp;redlink=1" class="new" title="띄어쓰기 (없는 문서)">띄어쓰기</a>는 하지 않지만 세 가지 <a href="/wiki/%E6%96%87%E5%AD%97" title="文字"><ruby class="hanja grade70"><rb>文字</rb><rp>(</rp><rt>문자</rt><rp>)</rp></ruby></a><ruby class="hanja grade72"><rb>間</rb><rp>(</rp><rt>간</rt><rp>)</rp></ruby>의 <ruby class="hanja grade42"><rb>境界</rb><rp>(</rp><rt>경계</rt><rp>)</rp></ruby>로 <a href="/index.php?title=%E5%96%AE%E8%AA%9E&amp;action=edit&amp;redlink=1" class="new" title="單語 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>單語</rb><rp>(</rp><rt>단어</rt><rp>)</rp></ruby></a> 사이를 <ruby class="hanja grade40"><rb>簡單</rb><rp>(</rp><rt>간단</rt><rp>)</rp></ruby>하게 <ruby class="hanja grade60"><rb>區分</rb><rp>(</rp><rt>구분</rt><rp>)</rp></ruby>할 수 있다.
</p>
<h2><span id="語彙(어휘)"></span><span class="mw-headline" id=".E8.AA.9E.E5.BD.99.28.EC.96.B4.ED.9C.98.29"><ruby class="hanja grade10"><rb>語彙</rb><rp>(</rp><rt>어휘</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%97%A5%E6%9C%AC%E8%AA%9E&amp;action=edit&amp;section=4" title="부분 편집: 語彙(어휘)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja grade60"><rb>日本語</rb><rp>(</rp><rt>일본어</rt><rp>)</rp></ruby>의 <ruby class="hanja grade10"><rb>語彙</rb><rp>(</rp><rt>어휘</rt><rp>)</rp></ruby>를 <ruby class="hanja grade62"><rb>出身</rb><rp>(</rp><rt>출신</rt><rp>)</rp></ruby>으로 <ruby class="hanja grade52"><rb>分類</rb><rp>(</rp><rt>분류</rt><rp>)</rp></ruby>하면 <ruby class="hanja grade62"><rb>和語</rb><rp>(</rp><rt>화어</rt><rp>)</rp></ruby>·<ruby class="hanja grade70"><rb>漢字語</rb><rp>(</rp><rt>한자어</rt><rp>)</rp></ruby>·<ruby class="hanja grade70"><rb>外來語</rb><rp>(</rp><rt>외래어</rt><rp>)</rp></ruby> 및 그들이 섞인 <ruby class="hanja grade40"><rb>混種語</rb><rp>(</rp><rt>혼종어</rt><rp>)</rp></ruby>로 나뉜다.
</p><p><a href="/index.php?title=%E5%92%8C%E8%AA%9E&amp;action=edit&amp;redlink=1" class="new" title="和語 (없는 문서)"><ruby class="hanja grade62 unknown"><rb>和語</rb><rp>(</rp><rt>화어</rt><rp>)</rp></ruby></a>는 <ruby class="hanja grade60"><rb>日本</rb><rp>(</rp><rt>일본</rt><rp>)</rp></ruby> <ruby class="hanja grade60"><rb>古來</rb><rp>(</rp><rt>고래</rt><rp>)</rp></ruby>의 <ruby class="hanja grade50"><rb>固有語</rb><rp>(</rp><rt>고유어</rt><rp>)</rp></ruby> 또는 <ruby class="hanja grade50"><rb>固有語</rb><rp>(</rp><rt>고유어</rt><rp>)</rp></ruby>의 <ruby class="hanja grade40"><rb>組合</rb><rp>(</rp><rt>조합</rt><rp>)</rp></ruby>과 <ruby class="hanja grade52"><rb>變化</rb><rp>(</rp><rt>변화</rt><rp>)</rp></ruby> <ruby class="hanja grade42"><rb>形態</rb><rp>(</rp><rt>형태</rt><rp>)</rp></ruby>를 가리킨다.
</p><p><a href="/wiki/%E6%BC%A2%E5%AD%97%E8%AA%9E" title="漢字語"><ruby class="hanja grade70"><rb>漢字語</rb><rp>(</rp><rt>한자어</rt><rp>)</rp></ruby></a>는 <ruby class="hanja grade70"><rb>漢地</rb><rp>(</rp><rt>한지</rt><rp>)</rp></ruby> <ruby class="hanja grade32"><rb>渡來</rb><rp>(</rp><rt>도래</rt><rp>)</rp></ruby>의 <ruby class="hanja grade62"><rb>漢字音</rb><rp>(</rp><rt>한자음</rt><rp>)</rp></ruby>을 <ruby class="hanja grade60"><rb>使用</rb><rp>(</rp><rt>사용</rt><rp>)</rp></ruby>한 말을 가리킨다. <ruby class="hanja grade50"><rb>多量</rb><rp>(</rp><rt>다량</rt><rp>)</rp></ruby>은 <ruby class="hanja grade52"><rb>相對的</rb><rp>(</rp><rt>상대적</rt><rp>)</rp></ruby>으로 낡은 <ruby class="hanja grade62"><rb>時代</rb><rp>(</rp><rt>시대</rt><rp>)</rp></ruby>의 <ruby class="hanja grade60"><rb>漢民族</rb><rp>(</rp><rt>한민족</rt><rp>)</rp></ruby> <ruby class="hanja grade32"><rb>諸言語</rb><rp>(</rp><rt>제언어</rt><rp>)</rp></ruby>에서의 <ruby class="hanja grade32"><rb>借用語</rb><rp>(</rp><rt>차용어</rt><rp>)</rp></ruby>이지만, <ruby class="hanja grade60"><rb>漢文式</rb><rp>(</rp><rt>한문식</rt><rp>)</rp></ruby> <ruby class="hanja grade42"><rb>造語法</rb><rp>(</rp><rt>조어법</rt><rp>)</rp></ruby>에 <ruby class="hanja grade40"><rb>根據</rb><rp>(</rp><rt>근거</rt><rp>)</rp></ruby>하면서도, <ruby class="hanja grade60"><rb>日本</rb><rp>(</rp><rt>일본</rt><rp>)</rp></ruby>에서 <ruby class="hanja grade60"><rb>日本人</rb><rp>(</rp><rt>일본인</rt><rp>)</rp></ruby>에 <ruby class="hanja grade40"><rb>依</rb><rp>(</rp><rt>의</rt><rp>)</rp></ruby>해 만들어진 <ruby class="hanja grade42"><rb>造語</rb><rp>(</rp><rt>조어</rt><rp>)</rp></ruby>도 <ruby class="hanja grade32"><rb>包含</rb><rp>(</rp><rt>포함</rt><rp>)</rp></ruby>된다. <ruby class="hanja grade60"><rb>日本</rb><rp>(</rp><rt>일본</rt><rp>)</rp></ruby>에서는 그 <ruby class="hanja grade10"><rb>語彙</rb><rp>(</rp><rt>어휘</rt><rp>)</rp></ruby> <ruby class="hanja grade40"><rb>體系</rb><rp>(</rp><rt>체계</rt><rp>)</rp></ruby>를 “<a href="/wiki/%E6%BC%A2%E8%AA%9E" title="漢語"><ruby class="hanja grade70"><rb>漢語</rb><rp>(</rp><rt>한어</rt><rp>)</rp></ruby></a>”라고 부르는 <ruby class="hanja grade40"><rb>境遇</rb><rp>(</rp><rt>경우</rt><rp>)</rp></ruby>가 많다.
</p><p><a href="/index.php?title=%E5%A4%96%E4%BE%86%E8%AA%9E&amp;action=edit&amp;redlink=1" class="new" title="外來語 (없는 문서)"><ruby class="hanja grade70 unknown"><rb>外來語</rb><rp>(</rp><rt>외래어</rt><rp>)</rp></ruby></a>는 <ruby class="hanja grade32"><rb>一般的</rb><rp>(</rp><rt>일반적</rt><rp>)</rp></ruby>으로 <ruby class="hanja grade60"><rb>漢民族</rb><rp>(</rp><rt>한민족</rt><rp>)</rp></ruby> <ruby class="hanja grade32"><rb>諸言語</rb><rp>(</rp><rt>제언어</rt><rp>)</rp></ruby> <ruby class="hanja grade52"><rb>以外</rb><rp>(</rp><rt>이외</rt><rp>)</rp></ruby>의 다른 <ruby class="hanja grade60"><rb>言語</rb><rp>(</rp><rt>언어</rt><rp>)</rp></ruby>에 <ruby class="hanja grade60"><rb>由來</rb><rp>(</rp><rt>유래</rt><rp>)</rp></ruby>하는 <ruby class="hanja grade32"><rb>借用語</rb><rp>(</rp><rt>차용어</rt><rp>)</rp></ruby>를 가리킨다. 그 <ruby class="hanja grade50"><rb>多量</rb><rp>(</rp><rt>다량</rt><rp>)</rp></ruby>은 <a href="/index.php?title=%E8%A5%BF%E6%B4%8B&amp;action=edit&amp;redlink=1" class="new" title="西洋 (없는 문서)"><ruby class="hanja grade60 unknown"><rb>西洋</rb><rp>(</rp><rt>서양</rt><rp>)</rp></ruby></a>에서 <ruby class="hanja grade52"><rb>傳</rb><rp>(</rp><rt>전</rt><rp>)</rp></ruby>해진 <ruby class="hanja grade42"><rb>單語</rb><rp>(</rp><rt>단어</rt><rp>)</rp></ruby>이며, <ruby class="hanja grade80"><rb>中</rb><rp>(</rp><rt>중</rt><rp>)</rp></ruby>에서도 <a href="/wiki/%E8%8B%B1%E8%AA%9E" title="英語"><ruby class="hanja grade60"><rb>英語</rb><rp>(</rp><rt>영어</rt><rp>)</rp></ruby></a>에 <ruby class="hanja grade60"><rb>由來</rb><rp>(</rp><rt>유래</rt><rp>)</rp></ruby>하는 것이 많다.
</p><p><a href="/wiki/%E6%BC%A2%E5%AD%97%E8%AA%9E" title="漢字語"><ruby class="hanja grade70"><rb>漢字語</rb><rp>(</rp><rt>한자어</rt><rp>)</rp></ruby></a>의 <ruby class="hanja grade50"><rb>比重</rb><rp>(</rp><rt>비중</rt><rp>)</rp></ruby>이 <ruby class="hanja grade32"><rb>莫大</rb><rp>(</rp><rt>막대</rt><rp>)</rp></ruby>하나, <ruby class="hanja grade62"><rb>現代</rb><rp>(</rp><rt>현대</rt><rp>)</rp></ruby>에 들어서는 <a href="/wiki/%E8%8B%B1%E8%AA%9E" title="英語"><ruby class="hanja grade60"><rb>英語</rb><rp>(</rp><rt>영어</rt><rp>)</rp></ruby></a><ruby class="hanja grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby> <a href="/index.php?title=%E5%A4%96%E4%BE%86%E8%AA%9E&amp;action=edit&amp;redlink=1" class="new" title="外來語 (없는 문서)"><ruby class="hanja grade70 unknown"><rb>外來語</rb><rp>(</rp><rt>외래어</rt><rp>)</rp></ruby></a>가 많아지는 <ruby class="hanja grade20"><rb>趨勢</rb><rp>(</rp><rt>추세</rt><rp>)</rp></ruby>이다.
</p><p><a href="/wiki/%E5%9B%BA%E6%9C%89%E8%AA%9E" title="固有語"><ruby class="hanja grade50"><rb>固有語</rb><rp>(</rp><rt>고유어</rt><rp>)</rp></ruby></a>도 <a href="/wiki/%E6%BC%A2%E5%AD%97" title="漢字"><ruby class="hanja grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby></a>로 <ruby class="hanja grade62"><rb>表記</rb><rp>(</rp><rt>표기</rt><rp>)</rp></ruby>하고 <ruby class="hanja grade62"><rb>音</rb><rp>(</rp><rt>음</rt><rp>)</rp></ruby>이 아닌 <a href="/index.php?title=%E8%A8%93&amp;action=edit&amp;redlink=1" class="new" title="訓 (없는 문서)"><ruby class="hanja grade60 unknown"><rb>訓</rb><rp>(</rp><rt>훈</rt><rp>)</rp></ruby></a>으로만 읽는데, 이를 <a href="/index.php?title=%E8%A8%93%E8%AE%80&amp;action=edit&amp;redlink=1" class="new" title="訓讀 (없는 문서)"><ruby class="hanja grade60 unknown"><rb>訓讀</rb><rp>(</rp><rt>훈독</rt><rp>)</rp></ruby></a>이라 한다. <ruby class="hanja grade62"><rb>反對</rb><rp>(</rp><rt>반대</rt><rp>)</rp></ruby>로 <a href="/wiki/%E6%BC%A2%E6%96%87" title="漢文"><ruby class="hanja grade70"><rb>漢文</rb><rp>(</rp><rt>한문</rt><rp>)</rp></ruby></a><ruby class="hanja grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby>에서 온 <ruby class="hanja grade70"><rb>漢字語</rb><rp>(</rp><rt>한자어</rt><rp>)</rp></ruby>들은 <a href="/wiki/%E9%9F%93%E5%9C%8B%E8%AA%9E" title="韓國語"><ruby class="hanja grade70"><rb>韓國語</rb><rp>(</rp><rt>한국어</rt><rp>)</rp></ruby></a>에서 그러듯이 <ruby class="hanja grade62"><rb>漢字音</rb><rp>(</rp><rt>한자음</rt><rp>)</rp></ruby>으로 읽고, 이는 <a href="/index.php?title=%E9%9F%B3%E8%AE%80&amp;action=edit&amp;redlink=1" class="new" title="音讀 (없는 문서)"><ruby class="hanja grade62 unknown"><rb>音讀</rb><rp>(</rp><rt>음독</rt><rp>)</rp></ruby></a>이라고 한다.
</p><p>또, 한 <ruby class="hanja grade42"><rb>單語</rb><rp>(</rp><rt>단어</rt><rp>)</rp></ruby>에 <ruby class="hanja grade60"><rb>訓讀</rb><rp>(</rp><rt>훈독</rt><rp>)</rp></ruby>과 <ruby class="hanja grade62"><rb>音讀</rb><rp>(</rp><rt>음독</rt><rp>)</rp></ruby>이 <ruby class="hanja grade0"><rb>混淆</rb><rp>(</rp><rt>혼효</rt><rp>)</rp></ruby>되어 있거나 <ruby class="hanja grade52"><rb>讀法</rb><rp>(</rp><rt>독법</rt><rp>)</rp></ruby>을 <ruby class="hanja grade60"><rb>區別</rb><rp>(</rp><rt>구별</rt><rp>)</rp></ruby>하는 게 <ruby class="hanja grade60"><rb>不一定</rb><rp>(</rp><rt>부일정</rt><rp>)</rp></ruby>한 <a href="/index.php?title=%E7%86%9F%E5%AD%97%E8%A8%93&amp;action=edit&amp;redlink=1" class="new" title="熟字訓 (없는 문서)"><ruby class="hanja grade32 unknown"><rb>熟字訓</rb><rp>(</rp><rt>숙자훈</rt><rp>)</rp></ruby></a>이라는 것도 있다.
</p>
<h2><span id="文法(문법)"></span><span class="mw-headline" id=".E6.96.87.E6.B3.95.28.EB.AC.B8.EB.B2.95.29"><ruby class="hanja grade52"><rb>文法</rb><rp>(</rp><rt>문법</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%97%A5%E6%9C%AC%E8%AA%9E&amp;action=edit&amp;section=5" title="부분 편집: 文法(문법)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja grade52"><rb>基本語順</rb><rp>(</rp><rt>기본어순</rt><rp>)</rp></ruby>은 <a href="/index.php?title=%E4%B8%BB%E7%9B%AE%E8%BF%B0&amp;action=edit&amp;redlink=1" class="new" title="主目述 (없는 문서)"><ruby class="hanja grade32 unknown"><rb>主目述</rb><rp>(</rp><rt>주목술</rt><rp>)</rp></ruby></a>이며 <ruby class="hanja grade40"><rb>文章構成</rb><rp>(</rp><rt>문장구성</rt><rp>)</rp></ruby>이 <ruby class="hanja grade70"><rb>韓國語</rb><rp>(</rp><rt>한국어</rt><rp>)</rp></ruby>와 <ruby class="hanja grade10"><rb>宏壯</rb><rp>(</rp><rt>굉장</rt><rp>)</rp></ruby>히 <ruby class="hanja grade30"><rb>類似</rb><rp>(</rp><rt>유사</rt><rp>)</rp></ruby>하다. <ruby class="hanja grade80"><rb>韓國人</rb><rp>(</rp><rt>한국인</rt><rp>)</rp></ruby>이 <ruby class="hanja grade40"><rb>簡單</rb><rp>(</rp><rt>간단</rt><rp>)</rp></ruby>히 <ruby class="hanja grade42"><rb>習得可能</rb><rp>(</rp><rt>습득가능</rt><rp>)</rp></ruby>한 <a href="/index.php?title=%E5%A4%96%E5%9C%8B%E8%AA%9E&amp;action=edit&amp;redlink=1" class="new" title="外國語 (없는 문서)"><ruby class="hanja grade70 unknown"><rb>外國語</rb><rp>(</rp><rt>외국어</rt><rp>)</rp></ruby></a>로 <ruby class="hanja grade60"><rb>每番</rb><rp>(</rp><rt>매번</rt><rp>)</rp></ruby> 꼽힌다.
</p><p>그러나 <a href="/index.php?title=%E5%8A%A9%E8%A9%9E&amp;action=edit&amp;redlink=1" class="new" title="助詞 (없는 문서)"><ruby class="hanja grade32 unknown"><rb>助詞</rb><rp>(</rp><rt>조사</rt><rp>)</rp></ruby></a> <ruby class="hanja grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby>의 <ruby class="hanja grade52"><rb>用法</rb><rp>(</rp><rt>용법</rt><rp>)</rp></ruby>이 <ruby class="hanja grade32"><rb>恒常</rb><rp>(</rp><rt>항상</rt><rp>)</rp></ruby> <ruby class="hanja grade42"><rb>一對一對應</rb><rp>(</rp><rt>일대일대응</rt><rp>)</rp></ruby> 하는 것은 아니라 <ruby class="hanja grade50"><rb>初心者</rb><rp>(</rp><rt>초심자</rt><rp>)</rp></ruby>들이 <ruby class="hanja grade52"><rb>種種</rb><rp>(</rp><rt>종종</rt><rp>)</rp></ruby> <ruby class="hanja grade60"><rb>失手</rb><rp>(</rp><rt>실수</rt><rp>)</rp></ruby>하기도 한다.
</p>
