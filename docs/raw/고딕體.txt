<h2><span id="槪要(개요)"></span><span class="mw-headline" id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"><ruby class="hanja grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EA%B3%A0%EB%94%95%E9%AB%94&amp;action=edit&amp;section=1" title="부분 편집: 槪要(개요)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>고딕<ruby class="hanja grade62"><rb>體</rb><rp>(</rp><rt>체</rt><rp>)</rp></ruby>란 <a href="/index.php?title=%EC%84%B8%EB%A6%AC%ED%94%84&amp;action=edit&amp;redlink=1" class="new" title="세리프 (없는 문서)">세리프</a>가 없는 <a href="/wiki/%ED%95%9C%EA%B8%80" title="한글">한글</a>과 <a href="/wiki/%E6%BC%A2%E5%AD%97" title="漢字"><ruby class="hanja grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby></a> <ruby class="hanja grade62"><rb>書體</rb><rp>(</rp><rt>서체</rt><rp>)</rp></ruby>를 말한다. <a href="/wiki/%E6%98%8E%E6%9C%9D%E9%AB%94" title="明朝體"><ruby class="hanja grade60"><rb>明朝體</rb><rp>(</rp><rt>명조체</rt><rp>)</rp></ruby></a>와 <ruby class="hanja grade50"><rb>對比</rb><rp>(</rp><rt>대비</rt><rp>)</rp></ruby>되는 <ruby class="hanja grade32"><rb>槪念</rb><rp>(</rp><rt>개념</rt><rp>)</rp></ruby>이며, <a href="/wiki/%EB%A1%9C%EB%A7%88%E5%AD%97" title="로마字">로마<ruby class="hanja grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby></a> <ruby class="hanja grade62"><rb>書體</rb><rp>(</rp><rt>서체</rt><rp>)</rp></ruby>의 <a href="/index.php?title=%EC%82%B0%EC%84%B8%EB%A6%AC%ED%94%84%E9%AB%94&amp;action=edit&amp;redlink=1" class="new" title="산세리프體 (없는 문서)">산세리프<ruby class="hanja grade62 unknown"><rb>體</rb><rp>(</rp><rt>체</rt><rp>)</rp></ruby></a>에 <ruby class="hanja grade42"><rb>對應</rb><rp>(</rp><rt>대응</rt><rp>)</rp></ruby>된다.
</p>
<h2><span id="漢字(한자)_고딕體(체)"></span><span class="mw-headline" id=".E6.BC.A2.E5.AD.97.28.ED.95.9C.EC.9E.90.29_.EA.B3.A0.EB.94.95.E9.AB.94.28.EC.B2.B4.29"><ruby class="hanja grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby> 고딕<ruby class="hanja grade62"><rb>體</rb><rp>(</rp><rt>체</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EA%B3%A0%EB%94%95%E9%AB%94&amp;action=edit&amp;section=2" title="부분 편집: 漢字(한자) 고딕體(체)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%E4%B8%AD%E5%9C%8B%E8%AA%9E" title="中國語"><ruby class="hanja grade70"><rb>中國語</rb><rp>(</rp><rt>중국어</rt><rp>)</rp></ruby></a>로 <a href="/index.php?title=%E9%BB%91%E9%AB%94&amp;action=edit&amp;redlink=1" class="new" title="黑體 (없는 문서)"><ruby class="hanja grade50 unknown"><rb>黑體</rb><rp>(</rp><rt>흑체</rt><rp>)</rp></ruby></a>라고 한다. 그런데 고딕<ruby class="hanja grade62"><rb>體</rb><rp>(</rp><rt>체</rt><rp>)</rp></ruby>든 <ruby class="hanja grade50"><rb>黑體</rb><rp>(</rp><rt>흑체</rt><rp>)</rp></ruby>든 <ruby class="hanja grade60"><rb>英語</rb><rp>(</rp><rt>영어</rt><rp>)</rp></ruby>로 <ruby class="hanja grade32"><rb>直譯</rb><rp>(</rp><rt>직역</rt><rp>)</rp></ruby>하면 <a href="/index.php?title=%EB%B8%94%EB%9E%99%EB%A0%88%ED%84%B0&amp;action=edit&amp;redlink=1" class="new" title="블랙레터 (없는 문서)">블랙레터</a>가 되지만 이건 <ruby class="hanja grade72"><rb>全</rb><rp>(</rp><rt>전</rt><rp>)</rp></ruby>혀 다른 <ruby class="hanja grade62"><rb>書體</rb><rp>(</rp><rt>서체</rt><rp>)</rp></ruby>이므로 <ruby class="hanja grade32"><rb>誤譯</rb><rp>(</rp><rt>오역</rt><rp>)</rp></ruby>의 <ruby class="hanja grade30"><rb>輿地</rb><rp>(</rp><rt>여지</rt><rp>)</rp></ruby>가 있다.
</p>
