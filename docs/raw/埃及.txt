<h2><span id=".E6.A7.AA.E8.BF.B0.28.EA.B0.9C.EC.88.A0.29"></span><span class="mw-headline" id="槪述(개술)"><ruby class="hanja api grade32"><rb>槪述</rb><rp>(</rp><rt>개술</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E5%9F%83%E5%8F%8A&amp;action=edit&amp;section=1" title="部分 編輯: 槪述(개술)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>두 <ruby class="hanja api grade52"><rb>大陸</rb><rp>(</rp><rt>대륙</rt><rp>)</rp></ruby>에 걸쳐 있으며 <ruby class="hanja api grade62"><rb>大部分</rb><rp>(</rp><rt>대부분</rt><rp>)</rp></ruby>은 <a grade62"><rb>北東部</rb><rp>(</rp><rt>북동부</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade42"><rb>位置</rb><rp>(</rp><rt>위치</rt><rp>)</rp></ruby>하고 있으며 수에즈 <ruby class="hanja api grade50"><rb>運河</rb><rp>(</rp><rt>운하</rt><rp>)</rp></ruby> <ruby class="hanja api grade80"><rb>東</rb><rp>(</rp><rt>동</rt><rp>)</rp></ruby>쪽의 시나이 <ruby class="hanja api grade50"><rb>半島</rb><rp>(</rp><rt>반도</rt><rp>)</rp></ruby>만 <ruby class="hanja api grade80"><rb>西南</rb><rp>(</rp><rt>서남</rt><rp>)</rp></ruby>아시아에 있습니다. <ruby class="hanja api grade80"><rb>西</rb><rp>(</rp><rt>서</rt><rp>)</rp></ruby>쪽으로는 리비아, <ruby class="hanja api grade80"><rb>南</rb><rp>(</rp><rt>남</rt><rp>)</rp></ruby>쪽으로는 <ruby class="hanja api
