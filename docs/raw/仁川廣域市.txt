<p><b><ruby class="hanja api grade40"><rb>仁川廣域市</rb><rp>(</rp><rt>인천광역시</rt><rp>)</rp></ruby></b>는 <a href="/wiki/%E5%A4%A7%E9%9F%93%E6%B0%91%E5%9C%8B" title="大韓民國"><ruby class="hanja api grade80"><rb>大韓民國</rb><rp>(</rp><rt>대한민국</rt><rp>)</rp></ruby></a>의 <a href="/index.php?title=%E5%BB%A3%E5%9F%9F%E5%B8%82&amp;action=edit&amp;redlink=1" class="new" title="廣域市(없는　文書）"><ruby class="hanja api grade40 unknown"><rb>廣域市</rb><rp>(</rp><rt>광역시</rt><rp>)</rp></ruby></a>이다.
</p>
<h2><span id=".E5.9C.B0.E7.90.86.28.EC.A7.80.EB.A6.AC.29"></span><span class="mw-headline" id="地理(지리)"><ruby class="hanja api grade62"><rb>地理</rb><rp>(</rp><rt>지리</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E4%BB%81%E5%B7%9D%E5%BB%A3%E5%9F%9F%E5%B8%82&amp;action=edit&amp;section=1" title="部分　編輯：地理(지리)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/index.php?title=%E8%A5%BF%E6%B5%B7%E5%B2%B8&amp;action=edit&amp;redlink=1" class="new" title="西海岸(없는　文書）"><ruby class="hanja api grade32 unknown"><rb>西海岸</rb><rp>(</rp><rt>서해안</rt><rp>)</rp></ruby></a>과 맞닿아있으며, <ruby class="hanja api grade80"><rb>東</rb><rp>(</rp><rt>동</rt><rp>)</rp></ruby>쪽으로 <a href="/wiki/%EC%84%9C%EC%9A%B8%E7%89%B9%E5%88%A5%E5%B8%82" title="서울特別市">서울<ruby class="hanja api grade60"><rb>特別市</rb><rp>(</rp><rt>특별시</rt><rp>)</rp></ruby></a>, 나머지 <ruby class="hanja api grade62"><rb>部分</rb><rp>(</rp><rt>부분</rt><rp>)</rp></ruby>은 <a href="/wiki/%E4%BA%AC%E7%95%BF%E9%81%93" title="京畿道"><ruby class="hanja api grade32"><rb>京畿道</rb><rp>(</rp><rt>경기도</rt><rp>)</rp></ruby></a>와 <ruby class="hanja api grade42"><rb>接</rb><rp>(</rp><rt>접</rt><rp>)</rp></ruby>한다. <ruby class="hanja api grade60"><rb>特</rb><rp>(</rp><rt>특</rt><rp>)</rp></ruby>히 <ruby class="hanja api grade32"><rb>西海岸</rb><rp>(</rp><rt>서해안</rt><rp>)</rp></ruby> <ruby class="hanja api grade60"><rb>方向</rb><rp>(</rp><rt>방향</rt><rp>)</rp></ruby>으로 <a href="/index.php?title=%E5%B9%B2%E6%8B%93&amp;action=edit&amp;redlink=1" class="new" title="干拓(없는　文書）"><ruby class="hanja api grade32 unknown"><rb>干拓</rb><rp>(</rp><rt>간척</rt><rp>)</rp></ruby></a>이 <ruby class="hanja api grade40"><rb>大巨</rb><rp>(</rp><rt>대거</rt><rp>)</rp></ruby> 이루어져 <a href="/index.php?title=%E6%B5%B7%E5%B2%B8%E7%B7%9A&amp;action=edit&amp;redlink=1" class="new" title="海岸線(없는　文書）"><ruby class="hanja api grade32 unknown"><rb>海岸線</rb><rp>(</rp><rt>해안선</rt><rp>)</rp></ruby></a>이 곧게 펴 있다.
</p><p>또, 이때문인지 <a href="/index.php?title=%E8%A1%8C%E6%94%BF%E5%8D%80%E5%9F%9F&amp;action=edit&amp;redlink=1" class="new" title="行政區域(없는　文書）"><ruby class="hanja api grade40 unknown"><rb>行政區域</rb><rp>(</rp><rt>행정구역</rt><rp>)</rp></ruby></a><ruby class="hanja api grade72"><rb>名</rb><rp>(</rp><rt>명</rt><rp>)</rp></ruby>들이 <ruby class="hanja api grade42"><rb>實際</rb><rp>(</rp><rt>실제</rt><rp>)</rp></ruby><ruby class="hanja api grade42"><rb>位置</rb><rp>(</rp><rt>위치</rt><rp>)</rp></ruby>와 <ruby class="hanja api grade10"><rb>乖離</rb><rp>(</rp><rt>괴리</rt><rp>)</rp></ruby>가 있다. <ruby class="hanja api grade60"><rb>中區</rb><rp>(</rp><rt>중구</rt><rp>)</rp></ruby>가 <ruby class="hanja api grade60"><rb>西區</rb><rp>(</rp><rt>서구</rt><rp>)</rp></ruby>보다 <ruby class="hanja api grade80"><rb>西</rb><rp>(</rp><rt>서</rt><rp>)</rp></ruby>쪽에 있고, <ruby class="hanja api grade60"><rb>東區</rb><rp>(</rp><rt>동구</rt><rp>)</rp></ruby>는 <ruby class="hanja api grade60"><rb>中區</rb><rp>(</rp><rt>중구</rt><rp>)</rp></ruby>와 <ruby class="hanja api grade60"><rb>西區</rb><rp>(</rp><rt>서구</rt><rp>)</rp></ruby>의 사이에 있다(...).
</p>
