<p><ruby class="hanja api grade10"><rb>巴林</rb><rp>(</rp><rt>파림</rt><rp>)</rp></ruby><ruby class="hanja api grade80"><rb>王國</rb><rp>(</rp><rt>왕국</rt><rp>)</rp></ruby> 줄여서 <ruby class="hanja api grade10"><rb>巴林</rb><rp>(</rp><rt>파림</rt><rp>)</rp></ruby>. <a href="/index.php?title=%E6%B3%A2%E6%96%AF%E7%81%A3&amp;action=edit&amp;redlink=1" class="new" title="波斯灣 (없는 文書)"><ruby class="hanja api grade30 unknown"><rb>波斯</rb><rp>(</rp><rt>파사</rt><rp>)</rp></ruby><ruby class="hanja api grade20 unknown"><rb>灣</rb><rp>(</rp><rt>만</rt><rp>)</rp></ruby></a> <ruby class="hanja api grade62"><rb>南西部</rb><rp>(</rp><rt>남서부</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade42"><rb>位置</rb><rp>(</rp><rt>위치</rt><rp>)</rp></ruby>한 섬나라로 <a href="/index.php?title=%E8%8F%AF%E5%A4%AA%E7%91%A0&amp;action=edit&amp;redlink=1" class="new" title="華太瑠 (없는 文書)"><ruby class="hanja api grade0 unknown"><rb>華太瑠</rb><rp>(</rp><rt>화태류</rt><rp>)</rp></ruby></a>와 <a href="/index.php?title=%E6%B2%99%E5%9C%B0%E4%BA%9E%E5%89%8C%E6%AF%94%E4%BA%9E&amp;action=edit&amp;redlink=1" class="new" title="沙地亞剌比亞 (없는 文書)"><ruby class="hanja api grade10 unknown"><rb>沙地亞剌比亞</rb><rp>(</rp><rt>사지아랄비아</rt><rp>)</rp></ruby></a> 사이에 있으며 사우디와 바다를 가로지르는 다리로 <ruby class="hanja api grade42"><rb>連結</rb><rp>(</rp><rt>연결</rt><rp>)</rp></ruby>되어 있다.
</p><p><ruby class="hanja api grade42"><rb>熱帶</rb><rp>(</rp><rt>열대</rt><rp>)</rp></ruby><ruby class="hanja api grade32"><rb>沙漠氣候</rb><rp>(</rp><rt>사막기후</rt><rp>)</rp></ruby>로 여름은 덥고 <ruby class="hanja api grade32"><rb>濕</rb><rp>(</rp><rt>습</rt><rp>)</rp></ruby>하며 7~9<ruby class="hanja api grade80"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>平均</rb><rp>(</rp><rt>평균</rt><rp>)</rp></ruby> <ruby class="hanja api grade60"><rb>氣溫</rb><rp>(</rp><rt>기온</rt><rp>)</rp></ruby>은 36℃이다. 겨울은 쌀쌀할 때 비가 내리고 12<ruby class="hanja api grade80"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby>부터 이듬해 2<ruby class="hanja api grade80"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby>까지 <ruby class="hanja api grade40"><rb>平均</rb><rp>(</rp><rt>평균</rt><rp>)</rp></ruby> <ruby class="hanja api grade60"><rb>氣溫</rb><rp>(</rp><rt>기온</rt><rp>)</rp></ruby>은 10℃-20℃입니다. 다른 <ruby class="hanja api grade72"><rb>時間</rb><rp>(</rp><rt>시간</rt><rp>)</rp></ruby>(3~5<ruby class="hanja api grade80"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby>, 10~11<ruby class="hanja api grade80"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby>) <ruby class="hanja api grade60"><rb>氣溫</rb><rp>(</rp><rt>기온</rt><rp>)</rp></ruby>은 20℃-30℃입니다. <ruby class="hanja api grade40"><rb>年平均</rb><rp>(</rp><rt>연평균</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>降水量</rb><rp>(</rp><rt>강수량</rt><rp>)</rp></ruby>은 71밀리미터이고 <ruby class="hanja api grade80"><rb>國土</rb><rp>(</rp><rt>국토</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>面積</rb><rp>(</rp><rt>면적</rt><rp>)</rp></ruby>은 779.95평방킬로미터이다.
</p><p><ruby class="hanja api grade10"><rb>巴林</rb><rp>(</rp><rt>파림</rt><rp>)</rp></ruby>은 4<ruby class="hanja api grade42"><rb>個</rb><rp>(</rp><rt>개</rt><rp>)</rp></ruby> <ruby class="hanja api grade42"><rb>城</rb><rp>(</rp><rt>성</rt><rp>)</rp></ruby>으로 나뉘는데 <ruby class="hanja api grade50"><rb>首都</rb><rp>(</rp><rt>수도</rt><rp>)</rp></ruby> 무하라크 <ruby class="hanja api grade72"><rb>北方</rb><rp>(</rp><rt>북방</rt><rp>)</rp></ruby> <ruby class="hanja api grade72"><rb>南方</rb><rp>(</rp><rt>남방</rt><rp>)</rp></ruby>이며 <ruby class="hanja api grade50"><rb>首都</rb><rp>(</rp><rt>수도</rt><rp>)</rp></ruby>는 마나마이다. <ruby class="hanja api grade10"><rb>巴林</rb><rp>(</rp><rt>파림</rt><rp>)</rp></ruby><ruby class="hanja api grade80"><rb>王國</rb><rp>(</rp><rt>왕국</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade70"><rb>人口</rb><rp>(</rp><rt>인구</rt><rp>)</rp></ruby>는 157<ruby class="hanja api grade80"><rb>萬</rb><rp>(</rp><rt>만</rt><rp>)</rp></ruby> <ruby class="hanja api grade72"><rb>名</rb><rp>(</rp><rt>명</rt><rp>)</rp></ruby>이고 <ruby class="hanja api grade80"><rb>外國人</rb><rp>(</rp><rt>외국인</rt><rp>)</rp></ruby>은 55%이다.

<h2><span id=".E6.AD.B7.E5.8F.B2.28.EC.97.AD.EC.82.AC.29"></span><span class="mw-headline" id="歷史(역사)"><ruby class="hanja api grade52"><rb>歷史</rb><rp>(</rp><rt>역사</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E5%B7%B4%E6%9E%97&amp;action=edit&amp;section=1" title="部分 編輯: 歷史(역사)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja api grade40"><rb>西紀</rb><rp>(</rp><rt>서기</rt><rp>)</rp></ruby> 7<ruby class="hanja api grade40"><rb>世紀</rb><rp>(</rp><rt>세기</rt><rp>)</rp></ruby>에 아랍 <ruby class="hanja api grade40"><rb>帝國</rb><rp>(</rp><rt>제국</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade62"><rb>一部</rb><rp>(</rp><rt>일부</rt><rp>)</rp></ruby>가 되어 바스라 <ruby class="hanja api grade52"><rb>州</rb><rp>(</rp><rt>주</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade40"><rb>屬</rb><rp>(</rp><rt>속</rt><rp>)</rp></ruby>했다.

</p><p>1602<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby>부터 1782<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby>까지 <a href="/index.php?title=%E6%B3%A2%E6%96%AF%E5%B8%9D%E5%9C%8B&amp;action=edit&amp;redlink=1" class="new" title="波斯帝國 (없는 文書)"><ruby class="hanja api grade30 unknown"><rb>波斯</rb><rp>(</rp><rt>파사</rt><rp>)</rp></ruby><ruby class="hanja api grade40 unknown"><rb>帝國</rb><rp>(</rp><rt>제국</rt><rp>)</rp></ruby></a>의 <ruby class="hanja api grade42"><rb>支配</rb><rp>(</rp><rt>지배</rt><rp>)</rp></ruby><ruby class="hanja api grade72"><rb>下</rb><rp>(</rp><rt>하</rt><rp>)</rp></ruby>에 있었다.
