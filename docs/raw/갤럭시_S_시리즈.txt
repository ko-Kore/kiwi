<h2><span id="槪要(개요)"></span><span class="mw-headline" id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"><ruby class="hanja grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EA%B0%A4%EB%9F%AD%EC%8B%9C_S_%EC%8B%9C%EB%A6%AC%EC%A6%88&amp;action=edit&amp;section=1" title="부분 편집: 槪要(개요)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>갤럭시 S 시리즈는 <a href="/wiki/%E4%B8%89%E6%98%9F_%EA%B0%A4%EB%9F%AD%EC%8B%9C" title="三星 갤럭시"><ruby class="hanja grade42"><rb>三星</rb><rp>(</rp><rt>삼성</rt><rp>)</rp></ruby> 갤럭시</a> <ruby class="hanja grade40"><rb>製品群</rb><rp>(</rp><rt>제품군</rt><rp>)</rp></ruby>에서 <ruby class="hanja grade52"><rb>高性能</rb><rp>(</rp><rt>고성능</rt><rp>)</rp></ruby> <ruby class="hanja grade42"><rb>主力製品</rb><rp>(</rp><rt>주력제품</rt><rp>)</rp></ruby>을 <ruby class="hanja grade42"><rb>擔當</rb><rp>(</rp><rt>담당</rt><rp>)</rp></ruby>하는 시리즈이다.
</p>
<h2><span id="名稱(명칭)"></span><span class="mw-headline" id=".E5.90.8D.E7.A8.B1.28.EB.AA.85.EC.B9.AD.29"><ruby class="hanja grade40"><rb>名稱</rb><rp>(</rp><rt>명칭</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EA%B0%A4%EB%9F%AD%EC%8B%9C_S_%EC%8B%9C%EB%A6%AC%EC%A6%88&amp;action=edit&amp;section=2" title="부분 편집: 名稱(명칭)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>시리즈 <ruby class="hanja grade40"><rb>名稱</rb><rp>(</rp><rt>명칭</rt><rp>)</rp></ruby>에 들어가는 <a href="/index.php?title=%EC%95%8C%ED%8C%8C%EB%B2%B3&amp;action=edit&amp;redlink=1" class="new" title="알파벳 (없는 문서)">알파벳</a> <a href="/index.php?title=S&amp;action=edit&amp;redlink=1" class="new" title="S (없는 문서)">S</a>의 뜻에 <ruby class="hanja grade62"><rb>對</rb><rp>(</rp><rt>대</rt><rp>)</rp></ruby>한 <a href="/index.php?title=%E4%B8%89%E6%98%9F%E9%9B%BB%E5%AD%90&amp;action=edit&amp;redlink=1" class="new" title="三星電子 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>三星電子</rb><rp>(</rp><rt>삼성전자</rt><rp>)</rp></ruby></a>의 <ruby class="hanja grade60"><rb>公式</rb><rp>(</rp><rt>공식</rt><rp>)</rp></ruby> <ruby class="hanja grade72"><rb>立場</rb><rp>(</rp><rt>입장</rt><rp>)</rp></ruby>은 <a href="/wiki/%EC%8A%A4%EB%A7%88%ED%8A%B8%ED%8F%B0" title="스마트폰">Super Smart</a>, <a href="/index.php?title=%E4%B8%89%E6%98%9F%E9%9B%BB%E5%AD%90&amp;action=edit&amp;redlink=1" class="new" title="三星電子 (없는 문서)">Samsung</a> <ruby class="hanja grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby>이라 한다.
</p><p>하지만 <ruby class="hanja grade40"><rb>使用者層</rb><rp>(</rp><rt>사용자층</rt><rp>)</rp></ruby>에서는 1<ruby class="hanja grade62"><rb>世代</rb><rp>(</rp><rt>세대</rt><rp>)</rp></ruby> <ruby class="hanja grade42"><rb>製品</rb><rp>(</rp><rt>제품</rt><rp>)</rp></ruby>인 <a href="/index.php?title=%EA%B0%A4%EB%9F%AD%EC%8B%9C_S&amp;action=edit&amp;redlink=1" class="new" title="갤럭시 S (없는 문서)">갤럭시 S</a>가 <a href="/index.php?title=SKT&amp;action=edit&amp;redlink=1" class="new" title="SKT (없는 문서)">SKT</a> <ruby class="hanja grade40"><rb>專用</rb><rp>(</rp><rt>전용</rt><rp>)</rp></ruby> <ruby class="hanja grade40"><rb>機器</rb><rp>(</rp><rt>기기</rt><rp>)</rp></ruby>였던 <ruby class="hanja grade52"><rb>事實</rb><rp>(</rp><rt>사실</rt><rp>)</rp></ruby>과, <a href="/index.php?title=%EA%B0%A4%EB%9F%AD%EC%8B%9C_K&amp;action=edit&amp;redlink=1" class="new" title="갤럭시 K (없는 문서)">갤럭시 K</a>, <a href="/index.php?title=%EA%B0%A4%EB%9F%AD%EC%8B%9C_U&amp;action=edit&amp;redlink=1" class="new" title="갤럭시 U (없는 문서)">갤럭시 U</a> 모델 <ruby class="hanja grade32"><rb>亦是</rb><rp>(</rp><rt>역시</rt><rp>)</rp></ruby> <ruby class="hanja grade40"><rb>存在</rb><rp>(</rp><rt>존재</rt><rp>)</rp></ruby>했던 것으로 보아서 <ruby class="hanja grade52"><rb>元來</rb><rp>(</rp><rt>원래</rt><rp>)</rp></ruby>는 SKT를 <ruby class="hanja grade42"><rb>意味</rb><rp>(</rp><rt>의미</rt><rp>)</rp></ruby>하던 것을 <a href="/index.php?title=%E5%86%8D%E8%A7%A3%E9%87%8B&amp;action=edit&amp;redlink=1" class="new" title="再解釋 (없는 문서)"><ruby class="hanja grade32 unknown"><rb>再解釋</rb><rp>(</rp><rt>재해석</rt><rp>)</rp></ruby></a>한 것이라고 <ruby class="hanja grade32"><rb>解釋</rb><rp>(</rp><rt>해석</rt><rp>)</rp></ruby>하기도 한다.
</p>
<h2><span id="特徵(특징)"></span><span class="mw-headline" id=".E7.89.B9.E5.BE.B5.28.ED.8A.B9.EC.A7.95.29"><ruby class="hanja grade32"><rb>特徵</rb><rp>(</rp><rt>특징</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EA%B0%A4%EB%9F%AD%EC%8B%9C_S_%EC%8B%9C%EB%A6%AC%EC%A6%88&amp;action=edit&amp;section=3" title="부분 편집: 特徵(특징)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%E4%B8%89%E6%98%9F_%EA%B0%A4%EB%9F%AD%EC%8B%9C" title="三星 갤럭시"><ruby class="hanja grade42"><rb>三星</rb><rp>(</rp><rt>삼성</rt><rp>)</rp></ruby> 갤럭시</a>의 <ruby class="hanja grade70"><rb>主力</rb><rp>(</rp><rt>주력</rt><rp>)</rp></ruby> <ruby class="hanja grade40"><rb>製品群</rb><rp>(</rp><rt>제품군</rt><rp>)</rp></ruby>으로, <ruby class="hanja grade42"><rb>新製品</rb><rp>(</rp><rt>신제품</rt><rp>)</rp></ruby>은 <ruby class="hanja grade52"><rb>高性能</rb><rp>(</rp><rt>고성능</rt><rp>)</rp></ruby>과 함께 <ruby class="hanja grade52"><rb>各種</rb><rp>(</rp><rt>각종</rt><rp>)</rp></ruby> <ruby class="hanja grade50"><rb>新技術</rb><rp>(</rp><rt>신기술</rt><rp>)</rp></ruby>로 <ruby class="hanja grade40"><rb>武裝</rb><rp>(</rp><rt>무장</rt><rp>)</rp></ruby>하고 나온다.
</p><p><a href="/wiki/%EC%95%88%EB%93%9C%EB%A1%9C%EC%9D%B4%EB%93%9C_%E9%81%8B%E7%87%9F%E9%AB%94%E5%88%B6" title="안드로이드 運營體制">안드로이드 <ruby class="hanja grade40"><rb>運營體制</rb><rp>(</rp><rt>운영체제</rt><rp>)</rp></ruby></a>가 <ruby class="hanja grade10"><rb>搭載</rb><rp>(</rp><rt>탑재</rt><rp>)</rp></ruby>된다. 하지만 <ruby class="hanja grade42"><rb>三星</rb><rp>(</rp><rt>삼성</rt><rp>)</rp></ruby>의 커스텀이 들어가 <a href="/index.php?title=%EC%9D%B8%ED%84%B0%ED%8E%98%EC%9D%B4%EC%8A%A4&amp;action=edit&amp;redlink=1" class="new" title="인터페이스 (없는 문서)">인터페이스</a>가 <ruby class="hanja grade40"><rb>大巨</rb><rp>(</rp><rt>대거</rt><rp>)</rp></ruby> <ruby class="hanja grade40"><rb>修整</rb><rp>(</rp><rt>수정</rt><rp>)</rp></ruby>되며, <a href="/index.php?title=%E4%B8%89%E6%98%9F_%EB%85%B9%EC%8A%A4&amp;action=edit&amp;redlink=1" class="new" title="三星 녹스 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>三星</rb><rp>(</rp><rt>삼성</rt><rp>)</rp></ruby> 녹스</a>나 <a href="/index.php?title=%E4%B8%89%E6%98%9F%ED%8E%98%EC%9D%B4&amp;action=edit&amp;redlink=1" class="new" title="三星페이 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>三星</rb><rp>(</rp><rt>삼성</rt><rp>)</rp></ruby>페이</a> <ruby class="hanja grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby>을 <ruby class="hanja grade60"><rb>使用</rb><rp>(</rp><rt>사용</rt><rp>)</rp></ruby>할 수 있다.
</p>
