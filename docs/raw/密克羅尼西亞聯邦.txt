</p><p><a href="/index.php?title=%E7%86%B1%E5%B8%B6%E6%B5%B7%E6%B4%8B%E6%80%A7%E6%B0%A3%E5%80%99&amp;action=edit&amp;redlink=1" class="new" title="熱帶海洋性氣候 (없는 文書)"><ruby class="hanja api grade42 unknown"><rb>熱帶</rb><rp>(</rp><rt>열대</rt><rp>)</rp></ruby><ruby class="hanja api grade40 unknown"><rb>海洋性氣候</rb><rp>(</rp><rt>해양성기후</rt><rp>)</rp></ruby></a>에 <ruby class="hanja api grade40"><rb>屬</rb><rp>(</rp><rt>속</rt><rp>)</rp></ruby>한다. 12<ruby class="hanja api grade80"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby>부터 이듬해 3<ruby class="hanja api grade80"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby>까지는 <ruby class="hanja api grade32"><rb>乾期</rb><rp>(</rp><rt>건기</rt><rp>)</rp></ruby>이고 4<ruby class="hanja api grade80"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby>부터 11<ruby class="hanja api grade80"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby>까지는 장마철이다. <ruby class="hanja api grade40"><rb>年平均</rb><rp>(</rp><rt>연평균</rt><rp>)</rp></ruby> <ruby class="hanja api grade60"><rb>氣溫</rb><rp>(</rp><rt>기온</rt><rp>)</rp></ruby>은 27℃이고 <ruby class="hanja api grade72"><rb>年間</rb><rp>(</rp><rt>연간</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>降水量</rb><rp>(</rp><rt>강수량</rt><rp>)</rp></ruby>은 <ruby class="hanja api grade52"><rb>約</rb><rp>(</rp><rt>약</rt><rp>)</rp></ruby> 2000mm이며 그 <ruby class="hanja api grade80"><rb>中</rb><rp>(</rp><rt>중</rt><rp>)</rp></ruby> 포나페의 <ruby class="hanja api grade72"><rb>年間</rb><rp>(</rp><rt>연간</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>降水量</rb><rp>(</rp><rt>강수량</rt><rp>)</rp></ruby>은 3000mm <ruby class="hanja api grade52"><rb>以上</rb><rp>(</rp><rt>이상</rt><rp>)</rp></ruby>으로 <ruby class="hanja api grade62"><rb>世界</rb><rp>(</rp><rt>세계</rt><rp>)</rp></ruby>에서 <ruby class="hanja api grade40"><rb>降水量</rb><rp>(</rp><rt>강수량</rt><rp>)</rp></ruby>이 가장 많은 곳 <ruby class="hanja api grade80"><rb>中</rb><rp>(</rp><rt>중</rt><rp>)</rp></ruby> 하나이다.
</p><p><ruby class="hanja api grade70"><rb>人口</rb><rp>(</rp><rt>인구</rt><rp>)</rp></ruby>는 10.48<ruby class="hanja api grade80"><rb>萬</rb><rp>(</rp><rt>만</rt><rp>)</rp></ruby> <ruby class="hanja api grade72"><rb>名</rb><rp>(</rp><rt>명</rt><rp>)</rp></ruby>입니다. 그 <ruby class="hanja api grade80"><rb>中</rb><rp>(</rp><rt>중</rt><rp>)</rp></ruby> <ruby class="hanja api grade70"><rb>天主敎</rb><rp>(</rp><rt>천주교</rt><rp>)</rp></ruby>는 50%, <ruby class="hanja api grade50"><rb>改新敎</rb><rp>(</rp><rt>개신교</rt><rp>)</rp></ruby>는 47%이다.
