<h2><span id=".E4.BD.8D.E7.BD.AE.28.EC.9C.84.EC.B9.98.29"></span><span class="mw-headline" id="位置(위치)"><ruby class="hanja api grade42"><rb>位置</rb><rp>(</rp><rt>위치</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E8%AB%BE%E5%A8%81&amp;action=edit&amp;section=1" title="部分 編輯: 位置(위치)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja api grade32"><rb>諾威</rb><rp>(</rp><rt>낙위</rt><rp>)</rp></ruby>는 <ruby class="hanja api grade80"><rb>北</rb><rp>(</rp><rt>북</rt><rp>)</rp></ruby>유럽 스칸디나비아 <ruby class="hanja api grade50"><rb>半島</rb><rp>(</rp><rt>반도</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade80"><rb>西</rb><rp>(</rp><rt>서</rt><rp>)</rp></ruby>쪽에 <ruby class="hanja api grade42"><rb>位置</rb><rp>(</rp><rt>위치</rt><rp>)</rp></ruby>해 있다. <ruby class="hanja api grade80"><rb>東</rb><rp>(</rp><rt>동</rt><rp>)</rp></ruby>쪽은 <ruby class="hanja api grade20"><rb>瑞典</rb><rp>(</rp><rt>서전</rt><rp>)</rp></ruby>, <ruby class="hanja api grade80"><rb>北東</rb><rp>(</rp><rt>북동</rt><rp>)</rp></ruby>쪽은 핀란드와 러시아, <ruby class="hanja api grade80"><rb>南</rb><rp>(</rp><rt>남</rt><rp>)</rp></ruby>쪽은 덴마크, <ruby class="hanja api grade80"><rb>西</rb><rp>(</rp><rt>서</rt><rp>)</rp></ruby>쪽은 <ruby class="hanja api grade32"><rb>諾威</rb><rp>(</rp><rt>낙위</rt><rp>)</rp></ruby><ruby class="hanja api grade72"><rb>海</rb><rp>(</rp><rt>해</rt><rp>)</rp></ruby>와 <ruby class="hanja api grade42"><rb>接</rb><rp>(</rp><rt>접</rt><rp>)</rp></ruby>하고 있다. <ruby class="hanja api grade62"><rb>大部分</rb><rp>(</rp><rt>대부분</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade40"><rb>地域</rb><rp>(</rp><rt>지역</rt><rp>)</rp></ruby>은 <ruby class="hanja api grade42"><rb>溫帶</rb><rp>(</rp><rt>온대</rt><rp>)</rp></ruby> <ruby class="hanja api grade52"><rb>海洋性</rb><rp>(</rp><rt>해양성</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>氣候</rb><rp>(</rp><rt>기후</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade40"><rb>屬</rb><rp>(</rp><rt>속</rt><rp>)</rp></ruby>한다.
</p>
<h2><span id=".E6.AD.B7.E5.8F.B2.28.EC.97.AD.EC.82.AC.29"></span><span class="mw-headline" id="歷史(역사)"><ruby class="hanja api grade52"><rb>歷史</rb><rp>(</rp><rt>역사</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E8%AB%BE%E5%A8%81&amp;action=edit&amp;section=2" title="部分 編輯: 歷史(역사)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>9<ruby class="hanja api grade40"><rb>世紀</rb><rp>(</rp><rt>세기</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade42"><rb>統一</rb><rp>(</rp><rt>통일</rt><rp>)</rp></ruby> <ruby class="hanja api grade80"><rb>王國</rb><rp>(</rp><rt>왕국</rt><rp>)</rp></ruby>이 <ruby class="hanja api grade62"><rb>形成</rb><rp>(</rp><rt>형성</rt><rp>)</rp></ruby>되었다. 
</p><p>9-11<ruby class="hanja api grade40"><rb>世紀</rb><rp>(</rp><rt>세기</rt><rp>)</rp></ruby>가 <ruby class="hanja api grade42"><rb>全盛期</rb><rp>(</rp><rt>전성기</rt><rp>)</rp></ruby>에 접어들었다. 
</p><p>14<ruby class="hanja api grade40"><rb>世紀</rb><rp>(</rp><rt>세기</rt><rp>)</rp></ruby> <ruby class="hanja api grade32"><rb>中盤</rb><rp>(</rp><rt>중반</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade32"><rb>衰退</rb><rp>(</rp><rt>쇠퇴</rt><rp>)</rp></ruby>하기 <ruby class="hanja api grade62"><rb>始作</rb><rp>(</rp><rt>시작</rt><rp>)</rp></ruby>하여 1397<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> 덴마크와 <a href="/wiki/%E7%91%9E%E5%85%B8" title="瑞典"><ruby class="hanja api grade20"><rb>瑞典</rb><rp>(</rp><rt>서전</rt><rp>)</rp></ruby></a>과 칼마르 <ruby class="hanja api grade32"><rb>同盟</rb><rp>(</rp><rt>동맹</rt><rp>)</rp></ruby>을 맺고 <a href="/wiki/%E4%B8%81%E6%8A%B9" title="丁抹"><ruby class="hanja api grade10"><rb>丁抹</rb><rp>(</rp><rt>정말</rt><rp>)</rp></ruby></a>의 <ruby class="hanja api grade80"><rb>女王</rb><rp>(</rp><rt>여왕</rt><rp>)</rp></ruby> 마가렛 1<ruby class="hanja api grade52"><rb>歲</rb><rp>(</rp><rt>세</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade42"><rb>支配</rb><rp>(</rp><rt>지배</rt><rp>)</rp></ruby>를 받았다. 
</p><p>1814<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> <ruby class="hanja api grade10"><rb>丁抹</rb><rp>(</rp><rt>정말</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade40"><rb>依</rb><rp>(</rp><rt>의</rt><rp>)</rp></ruby>해 <ruby class="hanja api grade20"><rb>瑞典</rb><rp>(</rp><rt>서전</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade32"><rb>讓渡</rb><rp>(</rp><rt>양도</rt><rp>)</rp></ruby>되었다. 1905<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> 6<ruby class="hanja api grade80"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby> 7<ruby class="hanja api grade80"><rb>日</rb><rp>(</rp><rt>일</rt><rp>)</rp></ruby> 리노 <ruby class="hanja api grade32"><rb>聯合</rb><rp>(</rp><rt>연합</rt><rp>)</rp></ruby>에서 <ruby class="hanja api grade52"><rb>獨立</rb><rp>(</rp><rt>독립</rt><rp>)</rp></ruby>하여 덴마크 <ruby class="hanja api grade72"><rb>王子</rb><rp>(</rp><rt>왕자</rt><rp>)</rp></ruby>를 <ruby class="hanja api grade80"><rb>王</rb><rp>(</rp><rt>왕</rt><rp>)</rp></ruby>으로 <ruby class="hanja api grade50"><rb>選出</rb><rp>(</rp><rt>선출</rt><rp>)</rp></ruby>하여 하콘 7<ruby class="hanja api grade72"><rb>世</rb><rp>(</rp><rt>세</rt><rp>)</rp></ruby>라 하였다. 
</p><p><a href="/wiki/%E7%AC%AC%E4%B8%80%E6%AC%A1%E4%B8%96%E7%95%8C%E5%A4%A7%E6%88%B0" title="第一次世界大戰"><ruby class="hanja api grade42"><rb>第一次世界大戰</rb><rp>(</rp><rt>제일차세계대전</rt><rp>)</rp></ruby></a><ruby class="hanja api grade80"><rb>中</rb><rp>(</rp><rt>중</rt><rp>)</rp></ruby> <ruby class="hanja api grade72"><rb>中立</rb><rp>(</rp><rt>중립</rt><rp>)</rp></ruby>. <a href="/wiki/%E7%AC%AC2%E6%AC%A1_%E4%B8%96%E7%95%8C_%E5%A4%A7%E6%88%B0" class="mw-redirect" title="第2次 世界 大戰"><ruby class="hanja api grade62"><rb>第</rb><rp>(</rp><rt>제</rt><rp>)</rp></ruby>2<ruby class="hanja api grade42"><rb>次</rb><rp>(</rp><rt>차</rt><rp>)</rp></ruby> <ruby class="hanja api grade62"><rb>世界</rb><rp>(</rp><rt>세계</rt><rp>)</rp></ruby> <ruby class="hanja api grade62"><rb>大戰</rb><rp>(</rp><rt>대전</rt><rp>)</rp></ruby></a>에서 <a href="/wiki/%E7%8D%A8%E9%80%B8" title="獨逸"><ruby class="hanja api grade32"><rb>獨逸</rb><rp>(</rp><rt>독일</rt><rp>)</rp></ruby></a>에 <ruby class="hanja api grade40"><rb>占領</rb><rp>(</rp><rt>점령</rt><rp>)</rp></ruby>되었다. 
</p><p>1945<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> 5<ruby class="hanja api grade80"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby> <ruby class="hanja api grade32"><rb>獨逸</rb><rp>(</rp><rt>독일</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>占領軍</rb><rp>(</rp><rt>점령군</rt><rp>)</rp></ruby>이 <ruby class="hanja api grade40"><rb>降伏</rb><rp>(</rp><rt>항복</rt><rp>)</rp></ruby>을 <ruby class="hanja api grade40"><rb>宣言</rb><rp>(</rp><rt>선언</rt><rp>)</rp></ruby>하고 <ruby class="hanja api grade32"><rb>諾威</rb><rp>(</rp><rt>낙위</rt><rp>)</rp></ruby>가 <ruby class="hanja api grade42"><rb>回復</rb><rp>(</rp><rt>회복</rt><rp>)</rp></ruby>되었다. 
</p><p>1947<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> 마셜 플랜을 <ruby class="hanja api grade42"><rb>受容</rb><rp>(</rp><rt>수용</rt><rp>)</rp></ruby>하면서 <ruby class="hanja api grade42"><rb>經濟</rb><rp>(</rp><rt>경제</rt><rp>)</rp></ruby>가 <ruby class="hanja api grade32"><rb>漸次</rb><rp>(</rp><rt>점차</rt><rp>)</rp></ruby> <ruby class="hanja api grade42"><rb>回復</rb><rp>(</rp><rt>회복</rt><rp>)</rp></ruby>되었다.

<h2><span id=".E6.86.B2.E6.B3.95.28.ED.97.8C.EB.B2.95.29"></span><span class="mw-headline" id="憲法(헌법)"><ruby class="hanja api grade40"><rb>憲法</rb><rp>(</rp><rt>헌법</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E8%AB%BE%E5%A8%81&amp;action=edit&amp;section=3" title="部分 編輯: 憲法(헌법)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
