<h2><span id=".E6.AD.B7.E5.8F.B2.28.EC.97.AD.EC.82.AC.29"></span><span class="mw-headline" id="歷史(역사)"><ruby class="hanja api grade52"><rb>歷史</rb><rp>(</rp><rt>역사</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%96%AF%E6%B4%9B%E4%BC%90%E5%85%8B&amp;action=edit&amp;section=2" title="部分 編輯: 歷史(역사)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>5-6<ruby class="hanja api grade40"><rb>世紀</rb><rp>(</rp><rt>세기</rt><rp>)</rp></ruby>에 서슬라브족이 <ruby class="hanja api grade20"><rb>斯洛伐克</rb><rp>(</rp><rt>사락벌극</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade52"><rb>定着</rb><rp>(</rp><rt>정착</rt><rp>)</rp></ruby>했습니다.
</p><p>7<ruby class="hanja api grade40"><rb>世紀</rb><rp>(</rp><rt>세기</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade20"><rb>斯洛伐克</rb><rp>(</rp><rt>사락벌극</rt><rp>)</rp></ruby>는 <ruby class="hanja api grade30"><rb>私募</rb><rp>(</rp><rt>사모</rt><rp>)</rp></ruby> <ruby class="hanja api grade62"><rb>公國</rb><rp>(</rp><rt>공국</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade70"><rb>中心地</rb><rp>(</rp><rt>중심지</rt><rp>)</rp></ruby>였습니다.
</p><p>8<ruby class="hanja api grade40"><rb>世紀</rb><rp>(</rp><rt>세기</rt><rp>)</rp></ruby>에 니트라 <ruby class="hanja api grade62"><rb>公國</rb><rp>(</rp><rt>공국</rt><rp>)</rp></ruby>이 이곳에 세워졌고 <ruby class="hanja api grade62"><rb>書記</rb><rp>(</rp><rt>서기</rt><rp>)</rp></ruby> 830<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> <ruby class="hanja api grade52"><rb>以後</rb><rp>(</rp><rt>이후</rt><rp>)</rp></ruby>에는 <ruby class="hanja api grade62"><rb>對</rb><rp>(</rp><rt>대</rt><rp>)</rp></ruby> 모라비아 <ruby class="hanja api grade40"><rb>帝國</rb><rp>(</rp><rt>제국</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade62"><rb>一部</rb><rp>(</rp><rt>일부</rt><rp>)</rp></ruby>가 되었습니다. 833 <ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>연</rt><rp>)</rp></ruby>에 그것은 <ruby class="hanja api grade62"><rb>代</rb><rp>(</rp><rt>대</rt><rp>)</rp></ruby> 모라비아의 <ruby class="hanja api grade70"><rb>中心</rb><rp>(</rp><rt>중심</rt><rp>)</rp></ruby>이되었습니다.
