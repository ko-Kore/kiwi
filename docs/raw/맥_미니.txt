<h2><span id=".E7.89.B9.E5.BE.B5.28.ED.8A.B9.EC.A7.95.29"></span><span class="mw-headline" id="特徵(특징)"><ruby class="hanja grade32"><rb>特徵</rb><rp>(</rp><rt>특징</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EB%A7%A5_%EB%AF%B8%EB%8B%88&amp;action=edit&amp;section=1" title="부분 편집: 特徵(특징)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>패키지 <ruby class="hanja grade40"><rb>構成品</rb><rp>(</rp><rt>구성품</rt><rp>)</rp></ruby>은 <ruby class="hanja grade40"><rb>簡單</rb><rp>(</rp><rt>간단</rt><rp>)</rp></ruby>하게 <ruby class="hanja grade60"><rb>本體</rb><rp>(</rp><rt>본체</rt><rp>)</rp></ruby>와 <ruby class="hanja grade40"><rb>電源</rb><rp>(</rp><rt>전원</rt><rp>)</rp></ruby>코드가 끝. <a href="/index.php?title=%EB%AA%A8%EB%8B%88%ED%84%B0&amp;action=edit&amp;redlink=1" class="new" title="모니터 (없는 문서)">모니터</a>와 <a href="/wiki/%ED%82%A4%EB%B3%B4%EB%93%9C" title="키보드">키보드</a> 및 <a href="/index.php?title=%EB%A7%88%EC%9A%B0%EC%8A%A4&amp;action=edit&amp;redlink=1" class="new" title="마우스 (없는 문서)">마우스</a>는 따로 <ruby class="hanja grade42"><rb>求</rb><rp>(</rp><rt>구</rt><rp>)</rp></ruby>해야 <ruby class="hanja grade60"><rb>使用</rb><rp>(</rp><rt>사용</rt><rp>)</rp></ruby>할수 있다. 하지만 그런만큼 <ruby class="hanja grade52"><rb>價格</rb><rp>(</rp><rt>가격</rt><rp>)</rp></ruby>도 착해서 <a href="/wiki/%E9%9F%93%E8%B2%A8" class="mw-redirect" title="韓貨"><ruby class="hanja grade42"><rb>韓貨</rb><rp>(</rp><rt>한화</rt><rp>)</rp></ruby></a> 80<ruby class="hanja grade80"><rb>萬</rb><rp>(</rp><rt>만</rt><rp>)</rp></ruby>원<ruby class="hanja grade42"><rb>帶</rb><rp>(</rp><rt>대</rt><rp>)</rp></ruby>에 <ruby class="hanja grade30"><rb>販賣</rb><rp>(</rp><rt>판매</rt><rp>)</rp></ruby>된다. 가장 <ruby class="hanja grade30"><rb>低廉</rb><rp>(</rp><rt>저렴</rt><rp>)</rp></ruby>한 <ruby class="hanja grade50"><rb>費用</rb><rp>(</rp><rt>비용</rt><rp>)</rp></ruby>으로 <a href="/index.php?title=%EB%A7%A4%ED%82%A8%ED%86%A0%EC%8B%9C_(%EC%BB%B4%ED%93%A8%ED%84%B0)&amp;action=edit&amp;redlink=1" class="new" title="매킨토시 (컴퓨터) (없는 문서)">맥</a>을 <ruby class="hanja grade60"><rb>使用</rb><rp>(</rp><rt>사용</rt><rp>)</rp></ruby>해볼수있는 <ruby class="hanja grade42"><rb>製品</rb><rp>(</rp><rt>제품</rt><rp>)</rp></ruby>.
</p>

<h3><span id="2023_.28M2_.26_M2_.ED.94.84.EB.A1.9C.29"></span><span class="mw-headline" id="2023_(M2_&amp;_M2_프로)">2023 (M2 &amp; M2 프로)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EB%A7%A5_%EB%AF%B8%EB%8B%88&amp;action=edit&amp;section=4" title="부분 편집: 2023 (M2 &amp; M2 프로)">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/index.php?title=2023%E5%B9%B4&amp;action=edit&amp;redlink=1" class="new" title="2023年 (없는 문서)">2023<ruby class="hanja grade80 unknown"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby></a><a href="/index.php?title=1%E6%9C%8817%E6%97%A5&amp;action=edit&amp;redlink=1" class="new" title="1月17日 (없는 문서)">1<ruby class="hanja grade80 unknown"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby>17<ruby class="hanja grade80 unknown"><rb>日</rb><rp>(</rp><rt>일</rt><rp>)</rp></ruby></a> <a href="/index.php?title=M2&amp;action=edit&amp;redlink=1" class="new" title="M2 (없는 문서)">M2</a>와 <a href="/index.php?title=M2_%ED%94%84%EB%A1%9C&amp;action=edit&amp;redlink=1" class="new" title="M2 프로 (없는 문서)">M2 프로</a>칩을 <ruby class="hanja grade10"><rb>搭載</rb><rp>(</rp><rt>탑재</rt><rp>)</rp></ruby>한 모델이 <ruby class="hanja grade62"><rb>發表</rb><rp>(</rp><rt>발표</rt><rp>)</rp></ruby>되었다. M2 프로 모델은 <ruby class="hanja grade70"><rb>後面</rb><rp>(</rp><rt>후면</rt><rp>)</rp></ruby>의 <a href="/index.php?title=%EC%8D%AC%EB%8D%94%EB%B3%BC%ED%8A%B84&amp;action=edit&amp;redlink=1" class="new" title="썬더볼트4 (없는 문서)">썬더볼트4</a><ruby class="hanja grade42"><rb>端子</rb><rp>(</rp><rt>단자</rt><rp>)</rp></ruby>가 4<ruby class="hanja grade70"><rb>口</rb><rp>(</rp><rt>구</rt><rp>)</rp></ruby>로 늘었다. M2모델은 <ruby class="hanja grade52"><rb>基本價格</rb><rp>(</rp><rt>기본가격</rt><rp>)</rp></ruby>이 699<ruby class="hanja grade20"><rb>弗</rb><rp>(</rp><rt>불</rt><rp>)</rp></ruby>에서 599<a href="/index.php?title=%E5%BC%97&amp;action=edit&amp;redlink=1" class="new" title="弗 (없는 문서)"><ruby class="hanja grade20 unknown"><rb>弗</rb><rp>(</rp><rt>불</rt><rp>)</rp></ruby></a>로 <ruby class="hanja grade42"><rb>引下</rb><rp>(</rp><rt>인하</rt><rp>)</rp></ruby>되었다.
</p>
