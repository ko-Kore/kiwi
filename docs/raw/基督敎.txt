<p><ruby class="hanja grade42"><rb>基督敎</rb><rp>(</rp><rt>기독교</rt><rp>)</rp></ruby>는 <a href="/index.php?title=%E8%80%B6%E8%98%87&amp;action=edit&amp;redlink=1" class="new" title="耶蘇 (없는 문서)"><ruby class="hanja grade30 unknown"><rb>耶蘇</rb><rp>(</rp><rt>야소</rt><rp>)</rp></ruby></a>(예수)의 가르침을 따르는 <ruby class="hanja grade42"><rb>宗敎</rb><rp>(</rp><rt>종교</rt><rp>)</rp></ruby>이다. 아브라함<ruby class="hanja grade62"><rb>界</rb><rp>(</rp><rt>계</rt><rp>)</rp></ruby> <ruby class="hanja grade42"><rb>宗敎</rb><rp>(</rp><rt>종교</rt><rp>)</rp></ruby>의 하나로 <ruby class="hanja grade32"><rb>猶太敎</rb><rp>(</rp><rt>유태교</rt><rp>)</rp></ruby>에서 <ruby class="hanja grade12"><rb>分岐</rb><rp>(</rp><rt>분기</rt><rp>)</rp></ruby>되었고 르네상스때 <a href="/index.php?title=%E5%AE%97%E6%95%8E%E6%94%B9%E9%9D%A9&amp;action=edit&amp;redlink=1" class="new" title="宗敎改革 (없는 문서)"><ruby class="hanja grade40 unknown"><rb>宗敎改革</rb><rp>(</rp><rt>종교개혁</rt><rp>)</rp></ruby></a>으로 크게 <a href="/index.php?title=%E5%A4%A9%E4%B8%BB%E6%95%8E&amp;action=edit&amp;redlink=1" class="new" title="天主敎 (없는 문서)"><ruby class="hanja grade70 unknown"><rb>天主敎</rb><rp>(</rp><rt>천주교</rt><rp>)</rp></ruby></a>와 <a href="/index.php?title=%E6%94%B9%E6%96%B0%E6%95%8E&amp;action=edit&amp;redlink=1" class="new" title="改新敎 (없는 문서)"><ruby class="hanja grade50 unknown"><rb>改新敎</rb><rp>(</rp><rt>개신교</rt><rp>)</rp></ruby></a>로 나뉘어졌다.
</p><p><ruby class="hanja grade42"><rb>經典</rb><rp>(</rp><rt>경전</rt><rp>)</rp></ruby>으로는 <a href="/index.php?title=%E8%88%8A%E7%B4%84%E8%81%96%E7%B6%93&amp;action=edit&amp;redlink=1" class="new" title="舊約聖經 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>舊約聖經</rb><rp>(</rp><rt>구약성경</rt><rp>)</rp></ruby></a>과 <a href="/index.php?title=%E6%96%B0%E7%B4%84%E8%81%96%E7%B6%93&amp;action=edit&amp;redlink=1" class="new" title="新約聖經 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>新約聖經</rb><rp>(</rp><rt>신약성경</rt><rp>)</rp></ruby></a>이 있고 <ruby class="hanja grade52"><rb>舊約</rb><rp>(</rp><rt>구약</rt><rp>)</rp></ruby>은 <a href="/index.php?title=%E7%8C%B6%E5%A4%AA%E6%95%8E&amp;action=edit&amp;redlink=1" class="new" title="猶太敎 (없는 문서)"><ruby class="hanja grade32 unknown"><rb>猶太敎</rb><rp>(</rp><rt>유태교</rt><rp>)</rp></ruby></a> <ruby class="hanja grade42"><rb>經典</rb><rp>(</rp><rt>경전</rt><rp>)</rp></ruby>과 <ruby class="hanja grade40"><rb>大同小異</rb><rp>(</rp><rt>대동소이</rt><rp>)</rp></ruby>하다. <ruby class="hanja grade52"><rb>新約</rb><rp>(</rp><rt>신약</rt><rp>)</rp></ruby>부터는 <ruby class="hanja grade30"><rb>耶蘇</rb><rp>(</rp><rt>야소</rt><rp>)</rp></ruby>와 그 <ruby class="hanja grade52"><rb>以後</rb><rp>(</rp><rt>이후</rt><rp>)</rp></ruby>를 다룬다.
</p><p><ruby class="hanja grade70"><rb>天主敎</rb><rp>(</rp><rt>천주교</rt><rp>)</rp></ruby>와 <ruby class="hanja grade50"><rb>改新敎</rb><rp>(</rp><rt>개신교</rt><rp>)</rp></ruby>를 <ruby class="hanja grade70"><rb>同一</rb><rp>(</rp><rt>동일</rt><rp>)</rp></ruby>하게 본다면 <ruby class="hanja grade62"><rb>世界</rb><rp>(</rp><rt>세계</rt><rp>)</rp></ruby>에서 <ruby class="hanja grade60"><rb>信者數</rb><rp>(</rp><rt>신자수</rt><rp>)</rp></ruby>가 가장 많은 종교이다.
