<html><head></head><body><div class="mw-parser-output"><p><b>드보락(Dvorak)</b>은 <a href="/index.php?title=%EC%96%B4%EA%B1%B0%EC%8A%A4%ED%8A%B8_%EB%93%9C%EB%B3%B4%EB%9D%BD&amp;action=edit&amp;redlink=1" class="new" title="어거스트 드보락 (없는 문서)">어거스트 드보락</a> <ruby class="hanja grade42"><rb>博士</rb><rp>(</rp><rt>박사</rt><rp>)</rp></ruby>가 <ruby class="hanja grade50"><rb>考案</rb><rp>(</rp><rt>고안</rt><rp>)</rp></ruby>한 <a href="/wiki/%EB%A1%9C%EB%A7%88%E5%AD%97" title="로마字">로마<ruby class="hanja grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby></a> 키보드<ruby class="hanja grade42"><rb>配列</rb><rp>(</rp><rt>배열</rt><rp>)</rp></ruby>이다.
</p>
<h2><span id=".E9.95.B7.E7.9F.AD.E9.BB.9E.28.EC.9E.A5.EB.8B.A8.EC.A0.90.29"></span><span class="mw-headline" id="長短點(장단점)"><ruby class="hanja grade40"><rb>長短點</rb><rp>(</rp><rt>장단점</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EB%93%9C%EB%B3%B4%EB%9D%BD%E5%AD%97%E6%9D%BF&amp;action=edit&amp;section=1" title="부분 편집: 長短點(장단점)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul><li>왼쪽 <ruby class="hanja grade72"><rb>中間</rb><rp>(</rp><rt>중간</rt><rp>)</rp></ruby>에 <a href="/index.php?title=%E6%AF%8D%E9%9F%B3&amp;action=edit&amp;redlink=1" class="new" title="母音 (없는 문서)"><ruby class="hanja grade62 unknown"><rb>母音</rb><rp>(</rp><rt>모음</rt><rp>)</rp></ruby></a>을 모두 <ruby class="hanja grade42"><rb>配置</rb><rp>(</rp><rt>배치</rt><rp>)</rp></ruby>하여 <ruby class="hanja grade42"><rb>兩</rb><rp>(</rp><rt>양</rt><rp>)</rp></ruby>손을 번갈아가며 쓰게 된다.</li>
<li><a href="/index.php?title=%E5%AD%90%E9%9F%B3&amp;action=edit&amp;redlink=1" class="new" title="子音 (없는 문서)"><ruby class="hanja grade62 unknown"><rb>子音</rb><rp>(</rp><rt>자음</rt><rp>)</rp></ruby></a> <ruby class="hanja grade32"><rb>亦是</rb><rp>(</rp><rt>역시</rt><rp>)</rp></ruby> <ruby class="hanja grade30"><rb>頻度</rb><rp>(</rp><rt>빈도</rt><rp>)</rp></ruby>가 높은 글<ruby class="hanja grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>가 <ruby class="hanja grade42"><rb>中間列</rb><rp>(</rp><rt>중간렬</rt><rp>)</rp></ruby>을 차지한고 있어 <a href="/wiki/%EC%BF%BC%ED%8B%B0" class="mw-redirect" title="쿼티">쿼티</a>에 <ruby class="hanja grade50"><rb>比</rb><rp>(</rp><rt>비</rt><rp>)</rp></ruby>해 손을 덜 움직이게 된다.</li>
<li><ruby class="hanja grade60"><rb>文章</rb><rp>(</rp><rt>문장</rt><rp>)</rp></ruby> <ruby class="hanja grade50"><rb>打字時</rb><rp>(</rp><rt>타자시</rt><rp>)</rp></ruby>가 아닌 <ruby class="hanja grade62"><rb>作業中</rb><rp>(</rp><rt>작업중</rt><rp>)</rp></ruby>에도 <a href="/index.php?title=%E7%9F%AD%E7%B8%AE%ED%82%A4&amp;action=edit&amp;redlink=1" class="new" title="短縮키 (없는 문서)"><ruby class="hanja grade40 unknown"><rb>短縮</rb><rp>(</rp><rt>단축</rt><rp>)</rp></ruby>키</a>의 위치가 모두 바뀌게 되어 <ruby class="hanja grade40"><rb>適應</rb><rp>(</rp><rt>적응</rt><rp>)</rp></ruby>이 <ruby class="hanja grade52"><rb>必要</rb><rp>(</rp><rt>필요</rt><rp>)</rp></ruby>하다.
<dl><dd>게다가 <ruby class="hanja grade40"><rb>短縮</rb><rp>(</rp><rt>단축</rt><rp>)</rp></ruby>키들은 <a href="/wiki/%EC%BF%BC%ED%8B%B0" class="mw-redirect" title="쿼티">쿼티</a>를 <ruby class="hanja grade42"><rb>基準</rb><rp>(</rp><rt>기준</rt><rp>)</rp></ruby>으로 <ruby class="hanja grade60"><rb>定</rb><rp>(</rp><rt>정</rt><rp>)</rp></ruby>해진 것이 많아 <ruby class="hanja grade60"><rb>使用</rb><rp>(</rp><rt>사용</rt><rp>)</rp></ruby>이 더욱 <ruby class="hanja grade70"><rb>不便</rb><rp>(</rp><rt>불편</rt><rp>)</rp></ruby>해진다. <sup id="cite_ref-1" class="reference"><a href="#cite_note-1">[1]</a></sup></dd></dl></li>
<li><ruby class="hanja grade62"><rb>一部</rb><rp>(</rp><rt>일부</rt><rp>)</rp></ruby> <ruby class="hanja grade32"><rb>文章符號</rb><rp>(</rp><rt>문장부호</rt><rp>)</rp></ruby>와 <ruby class="hanja grade32"><rb>特殊文字</rb><rp>(</rp><rt>특수문자</rt><rp>)</rp></ruby>의 <ruby class="hanja grade42"><rb>配列</rb><rp>(</rp><rt>배열</rt><rp>)</rp></ruby>이 달라져 <ruby class="hanja grade40"><rb>適應</rb><rp>(</rp><rt>적응</rt><rp>)</rp></ruby>이 <ruby class="hanja grade52"><rb>必要</rb><rp>(</rp><rt>필요</rt><rp>)</rp></ruby>하며, 한글<ruby class="hanja grade50"><rb>字板</rb><rp>(</rp><rt>자판</rt><rp>)</rp></ruby>과 함께 쓸때도 <ruby class="hanja grade70"><rb>不便</rb><rp>(</rp><rt>불편</rt><rp>)</rp></ruby>함이 생긴다.</li></ul>
