<p>갤럭시 노트 시리즈는 <a href="/wiki/%E4%B8%89%E6%98%9F_%EA%B0%A4%EB%9F%AD%EC%8B%9C" title="三星 갤럭시"><ruby class="hanja grade42"><rb>三星</rb><rp>(</rp><rt>삼성</rt><rp>)</rp></ruby> 갤럭시</a>의 <a href="/index.php?title=%ED%8C%A8%EB%B8%94%EB%A6%BF&amp;action=edit&amp;redlink=1" class="new" title="패블릿 (없는 문서)">패블릿</a> <a href="/wiki/%EC%8A%A4%EB%A7%88%ED%8A%B8%ED%8F%B0" title="스마트폰">스마트폰</a> 시리즈이다.
</p>

<p><a href="/wiki/%EC%8A%A4%EB%A7%88%ED%8A%B8%ED%8F%B0" title="스마트폰">스마트폰</a>과 <a href="/wiki/%ED%83%9C%EB%B8%94%EB%A6%BFPC" title="태블릿PC">태블릿PC</a>의 <ruby class="hanja grade42"><rb>中間程度</rb><rp>(</rp><rt>중간정도</rt><rp>)</rp></ruby> 되는 <ruby class="hanja grade60"><rb>畵面</rb><rp>(</rp><rt>화면</rt><rp>)</rp></ruby>크기에 <a href="/index.php?title=S%ED%8E%9C&amp;action=edit&amp;redlink=1" class="new" title="S펜 (없는 문서)">S펜</a>을 <ruby class="hanja grade62"><rb>利用</rb><rp>(</rp><rt>이용</rt><rp>)</rp></ruby>한 <ruby class="hanja grade52"><rb>筆記感</rb><rp>(</rp><rt>필기감</rt><rp>)</rp></ruby>의 <ruby class="hanja grade50"><rb>再現</rb><rp>(</rp><rt>재현</rt><rp>)</rp></ruby> <ruby class="hanja grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby>을 내세우는 <a href="/wiki/%EA%B0%A4%EB%9F%AD%EC%8B%9C_S_%EC%8B%9C%EB%A6%AC%EC%A6%88" title="갤럭시 S 시리즈">갤럭시 S 시리즈</a>와 어깨를 나란히 하는 <a href="/index.php?title=%ED%94%8C%EB%9E%98%EA%B7%B8%EC%89%BD&amp;action=edit&amp;redlink=1" class="new" title="플래그쉽 (없는 문서)">플래그쉽</a> <ruby class="hanja grade40"><rb>製品群</rb><rp>(</rp><rt>제품군</rt><rp>)</rp></ruby>이다.
</p><p><a href="/index.php?title=%EA%B0%A4%EB%9F%AD%EC%8B%9C_%EB%85%B8%ED%8A%B8&amp;action=edit&amp;redlink=1" class="new" title="갤럭시 노트 (없는 문서)">갤럭시 노트</a>가 <a href="/index.php?title=%E5%A4%A7%E8%88%B6&amp;action=edit&amp;redlink=1" class="new" title="大舶 (없는 문서)"><ruby class="hanja grade20 unknown"><rb>大舶</rb><rp>(</rp><rt>대박</rt><rp>)</rp></ruby></a>을 친 <ruby class="hanja grade52"><rb>以後</rb><rp>(</rp><rt>이후</rt><rp>)</rp></ruby>로 <ruby class="hanja grade50"><rb>年初</rb><rp>(</rp><rt>연초</rt><rp>)</rp></ruby>에는 <a href="/wiki/%EA%B0%A4%EB%9F%AD%EC%8B%9C_S_%EC%8B%9C%EB%A6%AC%EC%A6%88" title="갤럭시 S 시리즈">갤럭시 S 시리즈</a>의 <ruby class="hanja grade42"><rb>新製品</rb><rp>(</rp><rt>신제품</rt><rp>)</rp></ruby>을, <ruby class="hanja grade50"><rb>年末</rb><rp>(</rp><rt>연말</rt><rp>)</rp></ruby>에는 이 시리즈의 <ruby class="hanja grade42"><rb>新製品</rb><rp>(</rp><rt>신제품</rt><rp>)</rp></ruby>을 내는 <ruby class="hanja grade60"><rb>式</rb><rp>(</rp><rt>식</rt><rp>)</rp></ruby>의 <ruby class="hanja grade40"><rb>出市戰略</rb><rp>(</rp><rt>출시전략</rt><rp>)</rp></ruby>이 이루어졌다.
