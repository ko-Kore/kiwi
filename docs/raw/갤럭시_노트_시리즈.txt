<h2><span id=".E7.89.B9.E5.BE.B5.28.ED.8A.B9.EC.A7.95.29"></span><span class="mw-headline" id="特徵(특징)"><ruby class="hanja api grade32"><rb>特徵</rb><rp>(</rp><rt>특징</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EA%B0%A4%EB%9F%AD%EC%8B%9C_%EB%85%B8%ED%8A%B8_%EC%8B%9C%EB%A6%AC%EC%A6%88&amp;action=edit&amp;section=1" title="部分 編輯: 特徵(특징)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%8A%A4%EB%A7%88%ED%8A%B8%ED%8F%B0" title="스마트폰">스마트폰</a>과 <a href="/wiki/%ED%83%9C%EB%B8%94%EB%A6%BFPC" title="태블릿PC">태블릿PC</a>의 <ruby class="hanja api grade72"><rb>中間</rb><rp>(</rp><rt>중간</rt><rp>)</rp></ruby><ruby class="hanja api grade42"><rb>程度</rb><rp>(</rp><rt>정도</rt><rp>)</rp></ruby> 되는 <ruby class="hanja api grade60"><rb>畵面</rb><rp>(</rp><rt>화면</rt><rp>)</rp></ruby>크기에 <a href="/index.php?title=S%ED%8E%9C&amp;action=edit&amp;redlink=1" class="new" title="S펜 (없는 文書)">S펜</a>을 <ruby class="hanja api grade62"><rb>利用</rb><rp>(</rp><rt>이용</rt><rp>)</rp></ruby>한 <ruby class="hanja api grade52"><rb>筆記</rb><rp>(</rp><rt>필기</rt><rp>)</rp></ruby><ruby class="hanja api grade60"><rb>感</rb><rp>(</rp><rt>감</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade50"><rb>再現</rb><rp>(</rp><rt>재현</rt><rp>)</rp></ruby> <ruby class="hanja api grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby>을 내세우는 <a href="/wiki/%EA%B0%A4%EB%9F%AD%EC%8B%9C_S_%EC%8B%9C%EB%A6%AC%EC%A6%88" title="갤럭시 S 시리즈">갤럭시 S 시리즈</a>와 어깨를 나란히 하는 <a href="/index.php?title=%ED%94%8C%EB%9E%98%EA%B7%B8%EC%89%BD&amp;action=edit&amp;redlink=1" class="new" title="플래그쉽 (없는 文書)">플래그쉽</a> <ruby class="hanja api grade40"><rb>製品群</rb><rp>(</rp><rt>제품군</rt><rp>)</rp></ruby>이다.
</p><p><a href="/index.php?title=%EA%B0%A4%EB%9F%AD%EC%8B%9C_%EB%85%B8%ED%8A%B8&amp;action=edit&amp;redlink=1" class="new" title="갤럭시 노트 (없는 文書)">갤럭시 노트</a>가 <a href="/index.php?title=%E5%A4%A7%E8%88%B6&amp;action=edit&amp;redlink=1" class="new" title="大舶 (없는 文書)"><ruby class="hanja api grade20 unknown"><rb>大舶</rb><rp>(</rp><rt>대박</rt><rp>)</rp></ruby></a>을 친 <ruby class="hanja api grade52"><rb>以後</rb><rp>(</rp><rt>이후</rt><rp>)</rp></ruby>로 <ruby class="hanja api grade50"><rb>年初</rb><rp>(</rp><rt>연초</rt><rp>)</rp></ruby>에는 <a href="/wiki/%EA%B0%A4%EB%9F%AD%EC%8B%9C_S_%EC%8B%9C%EB%A6%AC%EC%A6%88" title="갤럭시 S 시리즈">갤럭시 S 시리즈</a>의 <ruby class="hanja api grade42"><rb>新製品</rb><rp>(</rp><rt>신제품</rt><rp>)</rp></ruby>을, <ruby class="hanja api grade50"><rb>年末</rb><rp>(</rp><rt>연말</rt><rp>)</rp></ruby>에는 이 시리즈의 <ruby class="hanja api grade42"><rb>新製品</rb><rp>(</rp><rt>신제품</rt><rp>)</rp></ruby>을 내는 <ruby class="hanja api grade60"><rb>式</rb><rp>(</rp><rt>식</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade70"><rb>出市</rb><rp>(</rp><rt>출시</rt><rp>)</rp></ruby><ruby class="hanja api grade40"><rb>戰略</rb><rp>(</rp><rt>전략</rt><rp>)</rp></ruby>이 이루어졌다.
</p>
<h2><span id=".E6.9C.AA.E4.BE.86.28.EB.AF.B8.EB.9E.98.29"></span><span class="mw-headline" id="未來(미래)"><ruby class="hanja api grade42"><rb>未來</rb><rp>(</rp><rt>미래</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EA%B0%A4%EB%9F%AD%EC%8B%9C_%EB%85%B8%ED%8A%B8_%EC%8B%9C%EB%A6%AC%EC%A6%88&amp;action=edit&amp;section=2" title="部分 編輯: 未來(미래)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/index.php?title=%EA%B0%A4%EB%9F%AD%EC%8B%9C_Z_%ED%8F%B4%EB%93%9C_3&amp;action=edit&amp;redlink=1" class="new" title="갤럭시 Z 폴드 3 (없는 文書)">갤럭시 Z 폴드 3</a>가 <a href="/index.php?title=S%ED%8E%9C&amp;action=edit&amp;redlink=1" class="new" title="S펜 (없는 文書)">S펜</a>을 <ruby class="hanja api grade40"><rb>支援</rb><rp>(</rp><rt>지원</rt><rp>)</rp></ruby>하게 되면서 <ruby class="hanja api grade52"><rb>事實上</rb><rp>(</rp><rt>사실상</rt><rp>)</rp></ruby> <a href="/wiki/%EA%B0%A4%EB%9F%AD%EC%8B%9C_Z_%EC%8B%9C%EB%A6%AC%EC%A6%88" title="갤럭시 Z 시리즈">갤럭시 Z 시리즈</a>의 <a href="/index.php?title=%E4%B8%8B%E4%BD%8D%E4%BA%92%E6%8F%9B&amp;action=edit&amp;redlink=1" class="new" title="下位互換 (없는 文書)"><ruby class="hanja api grade50 unknown"><rb>下位</rb><rp>(</rp><rt>하위</rt><rp>)</rp></ruby><ruby class="hanja api grade30 unknown"><rb>互換</rb><rp>(</rp><rt>호환</rt><rp>)</rp></ruby></a> 신세가 되어 버렸다. <a href="/wiki/2021%E5%B9%B4" title="2021年">2021<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby></a> 노트 <ruby class="hanja api grade42"><rb>新製品</rb><rp>(</rp><rt>신제품</rt><rp>)</rp></ruby>이 없다는 <ruby class="hanja api grade62"><rb>發表</rb><rp>(</rp><rt>발표</rt><rp>)</rp></ruby>와 함께 이로 <ruby class="hanja api grade50"><rb>因</rb><rp>(</rp><rt>인</rt><rp>)</rp></ruby>해 시리즈 <ruby class="hanja api grade62"><rb>自體</rb><rp>(</rp><rt>자체</rt><rp>)</rp></ruby>가 사라질 것이라는 <ruby class="hanja api grade52"><rb>說</rb><rp>(</rp><rt>설</rt><rp>)</rp></ruby>과 함께 <ruby class="hanja api grade42"><rb>斷種</rb><rp>(</rp><rt>단종</rt><rp>)</rp></ruby>을 <ruby class="hanja api grade62"><rb>反對</rb><rp>(</rp><rt>반대</rt><rp>)</rp></ruby>하는 <ruby class="hanja api grade42"><rb>請願</rb><rp>(</rp><rt>청원</rt><rp>)</rp></ruby><a rel="nofollow" class="external autonumber" href="https://www.hankyung.com/it/article/202108037857g">[一]</a>까지 올라왔다.

</p>
