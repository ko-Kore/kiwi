<h3><span id=".EC.A0.81.EC.9D.80_.EC.89.AC.ED.94.84.ED.8A.B8_.E4.BD.BF.E7.94.A8.28.EC.82.AC.EC.9A.A9.29"></span><span class="mw-headline" id="적은_쉬프트_使用(사용)">적은 쉬프트 <ruby class="hanja api grade60"><rb>使用</rb><rp>(</rp><rt>사용</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E5%85%AC%E7%82%B3%E7%A6%B9%E6%89%93%E5%AD%97%E6%A9%9F&amp;action=edit&amp;section=4" title="部分 編輯: 적은 쉬프트 使用(사용)">編輯</a><span class="mw-editsection-bracket">]</span></span></h3>
