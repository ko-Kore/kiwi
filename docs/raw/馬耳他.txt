<h2><span id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"></span><span class="mw-headline" id="槪要(개요)"><ruby class="hanja api grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E9%A6%AC%E8%80%B3%E4%BB%96&amp;action=edit&amp;section=1" title="部分 編輯: 槪要(개요)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja api grade50"><rb>馬耳他</rb><rp>(</rp><rt>마이타</rt><rp>)</rp></ruby>는 <ruby class="hanja api grade70"><rb>地中海</rb><rp>(</rp><rt>지중해</rt><rp>)</rp></ruby> 한가운데에 <ruby class="hanja api grade42"><rb>位置</rb><rp>(</rp><rt>위치</rt><rp>)</rp></ruby>한 섬나라로 "<ruby class="hanja api grade70"><rb>地中海</rb><rp>(</rp><rt>지중해</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade32"><rb>心臟</rb><rp>(</rp><rt>심장</rt><rp>)</rp></ruby>"으로 알려져 있습니다. <ruby class="hanja api grade32"><rb>海岸線</rb><rp>(</rp><rt>해안선</rt><rp>)</rp></ruby>의 길이는 190km가 넘고 많은 <ruby class="hanja api grade70"><rb>自然</rb><rp>(</rp><rt>자연</rt><rp>)</rp></ruby> <ruby class="hanja api grade42"><rb>港口</rb><rp>(</rp><rt>항구</rt><rp>)</rp></ruby>가 있습니다. <ruby class="hanja api grade32"><rb>亞熱帶</rb><rp>(</rp><rt>아열대</rt><rp>)</rp></ruby> <ruby class="hanja api grade52"><rb>地中海性</rb><rp>(</rp><rt>지중해성</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>氣候</rb><rp>(</rp><rt>기후</rt><rp>)</rp></ruby>로 <ruby class="hanja api grade40"><rb>年平均</rb><rp>(</rp><rt>연평균</rt><rp>)</rp></ruby> <ruby class="hanja api grade60"><rb>氣溫</rb><rp>(</rp><rt>기온</rt><rp>)</rp></ruby>은 19.7°C, <ruby class="hanja api grade50"><rb>最高</rb><rp>(</rp><rt>최고</rt><rp>)</rp></ruby><ruby class="hanja api grade60"><rb>氣溫</rb><rp>(</rp><rt>기온</rt><rp>)</rp></ruby>은 40°C, <ruby class="hanja api grade42"><rb>最低</rb><rp>(</rp><rt>최저</rt><rp>)</rp></ruby><ruby class="hanja api grade60"><rb>氣溫</rb><rp>(</rp><rt>기온</rt><rp>)</rp></ruby>은 5°C이다. <ruby class="hanja api grade40"><rb>年平均</rb><rp>(</rp><rt>연평균</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>降水量</rb><rp>(</rp><rt>강수량</rt><rp>)</rp></ruby>은 560mm입니다.
</p>
<h2><span id=".E6.AD.B7.E5.8F.B2.28.EC.97.AD.EC.82.AC.29"></span><span class="mw-headline" id="歷史(역사)"><ruby class="hanja api grade52"><rb>歷史</rb><rp>(</rp><rt>역사</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E9%A6%AC%E8%80%B3%E4%BB%96&amp;action=edit&amp;section=2" title="部分 編輯: 歷史(역사)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja api grade40"><rb>紀元前</rb><rp>(</rp><rt>기원전</rt><rp>)</rp></ruby> 5000 <ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>연</rt><rp>)</rp></ruby> <ruby class="hanja api grade52"><rb>以來</rb><rp>(</rp><rt>이래</rt><rp>)</rp></ruby>로 섬에는 <ruby class="hanja api grade72"><rb>人間</rb><rp>(</rp><rt>인간</rt><rp>)</rp></ruby> <ruby class="hanja api grade72"><rb>活動</rb><rp>(</rp><rt>활동</rt><rp>)</rp></ruby>이있었습니다. 
</p><p><ruby class="hanja api grade40"><rb>紀元前</rb><rp>(</rp><rt>기원전</rt><rp>)</rp></ruby> 10<ruby class="hanja api grade40"><rb>世紀</rb><rp>(</rp><rt>세기</rt><rp>)</rp></ruby>부터 8<ruby class="hanja api grade40"><rb>世紀</rb><rp>(</rp><rt>세기</rt><rp>)</rp></ruby>까지 페니키아<ruby class="hanja api grade80"><rb>人</rb><rp>(</rp><rt>인</rt><rp>)</rp></ruby>들이 이곳에 <ruby class="hanja api grade52"><rb>定着</rb><rp>(</rp><rt>정착</rt><rp>)</rp></ruby>했습니다. 
</p><p><ruby class="hanja api grade40"><rb>紀元前</rb><rp>(</rp><rt>기원전</rt><rp>)</rp></ruby> 218<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby>부터 <a href="/index.php?title=%E7%BE%85%E9%A6%AC%E5%B8%9D%E5%9C%8B&amp;action=edit&amp;redlink=1" class="new" title="羅馬帝國 (없는 文書)"><ruby class="hanja api grade42 unknown"><rb>羅馬</rb><rp>(</rp><rt>나마</rt><rp>)</rp></ruby><ruby class="hanja api grade40 unknown"><rb>帝國</rb><rp>(</rp><rt>제국</rt><rp>)</rp></ruby></a>의 <ruby class="hanja api grade42"><rb>支配</rb><rp>(</rp><rt>지배</rt><rp>)</rp></ruby>를 받았습니다. 
</p><p>9<ruby class="hanja api grade40"><rb>世紀</rb><rp>(</rp><rt>세기</rt><rp>)</rp></ruby>부터 아랍<ruby class="hanja api grade80"><rb>人</rb><rp>(</rp><rt>인</rt><rp>)</rp></ruby>과 노르만<ruby class="hanja api grade60"><rb>族</rb><rp>(</rp><rt>족</rt><rp>)</rp></ruby>이 <ruby class="hanja api grade40"><rb>占領</rb><rp>(</rp><rt>점령</rt><rp>)</rp></ruby>했습니다. 
</p><p>1530 <ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>연</rt><rp>)</rp></ruby>에 예루살렘의 <a href="/index.php?title=%E8%81%96_%EC%9A%94%ED%95%9C_%E9%A8%8E%E5%A3%AB%E5%9C%98&amp;action=edit&amp;redlink=1" class="new" title="聖 요한 騎士團 (없는 文書)"><ruby class="hanja api grade42 unknown"><rb>聖</rb><rp>(</rp><rt>성</rt><rp>)</rp></ruby> 요한 <ruby class="hanja api grade32 unknown"><rb>騎士團</rb><rp>(</rp><rt>기사단</rt><rp>)</rp></ruby></a>은 로도스에서 이곳으로 <ruby class="hanja api grade10"><rb>移徙</rb><rp>(</rp><rt>이사</rt><rp>)</rp></ruby>했습니다. 
</p><p>1798<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby>, <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" class="mw-redirect" title="프랑스">프랑스</a> <ruby class="hanja api grade42"><rb>軍隊</rb><rp>(</rp><rt>군대</rt><rp>)</rp></ruby>는 <ruby class="hanja api grade32"><rb>騎士團</rb><rp>(</rp><rt>기사단</rt><rp>)</rp></ruby>을 <ruby class="hanja api grade32"><rb>追放</rb><rp>(</rp><rt>추방</rt><rp>)</rp></ruby>했다. 
</p><p>1800 <ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>연</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade60"><rb>英國</rb><rp>(</rp><rt>영국</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade40"><rb>依</rb><rp>(</rp><rt>의</rt><rp>)</rp></ruby>해 <ruby class="hanja api grade40"><rb>占領</rb><rp>(</rp><rt>점령</rt><rp>)</rp></ruby>되었고 1814 <ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>연</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade70"><rb>植民地</rb><rp>(</rp><rt>식민지</rt><rp>)</rp></ruby>가되어 <ruby class="hanja api grade52"><rb>重要</rb><rp>(</rp><rt>중요</rt><rp>)</rp></ruby>한 <a href="/wiki/%E8%8B%B1%E5%9C%8B" title="英國"><ruby class="hanja api grade60"><rb>英國</rb><rp>(</rp><rt>영국</rt><rp>)</rp></ruby></a> <ruby class="hanja api grade72"><rb>海軍</rb><rp>(</rp><rt>해군</rt><rp>)</rp></ruby> <ruby class="hanja api grade52"><rb>基地</rb><rp>(</rp><rt>기지</rt><rp>)</rp></ruby><ruby class="hanja api grade42"><rb>街</rb><rp>(</rp><rt>가</rt><rp>)</rp></ruby>되었습니다. 
</p><p><a href="/wiki/%E7%AC%AC2%E6%AC%A1_%E4%B8%96%E7%95%8C%E5%A4%A7%E6%88%B0" class="mw-redirect" title="第2次 世界大戰"><ruby class="hanja api grade62"><rb>第</rb><rp>(</rp><rt>제</rt><rp>)</rp></ruby>2<ruby class="hanja api grade42"><rb>次</rb><rp>(</rp><rt>차</rt><rp>)</rp></ruby> <ruby class="hanja api grade62"><rb>世界大戰</rb><rp>(</rp><rt>세계대전</rt><rp>)</rp></ruby></a> <ruby class="hanja api grade72"><rb>後</rb><rp>(</rp><rt>후</rt><rp>)</rp></ruby> <ruby class="hanja api grade60"><rb>英國</rb><rp>(</rp><rt>영국</rt><rp>)</rp></ruby>은 1947<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> <ruby class="hanja api grade50"><rb>馬耳他</rb><rp>(</rp><rt>마이타</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade42"><rb>自治</rb><rp>(</rp><rt>자치</rt><rp>)</rp></ruby><ruby class="hanja api grade42"><rb>政府</rb><rp>(</rp><rt>정부</rt><rp>)</rp></ruby> <ruby class="hanja api grade60"><rb>樹立</rb><rp>(</rp><rt>수립</rt><rp>)</rp></ruby>을 <ruby class="hanja api grade42"><rb>許容</rb><rp>(</rp><rt>허용</rt><rp>)</rp></ruby>했으나 1959<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> 1<ruby class="hanja api grade80"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby> 다시 <ruby class="hanja api grade42"><rb>直接</rb><rp>(</rp><rt>직접</rt><rp>)</rp></ruby> <ruby class="hanja api grade42"><rb>統治</rb><rp>(</rp><rt>통치</rt><rp>)</rp></ruby>를 <ruby class="hanja api grade50"><rb>再開</rb><rp>(</rp><rt>재개</rt><rp>)</rp></ruby>했다. 
</p><p>1964<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> 9<ruby class="hanja api grade80"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby> 21<ruby class="hanja api grade80"><rb>日</rb><rp>(</rp><rt>일</rt><rp>)</rp></ruby>, <ruby class="hanja api grade50"><rb>馬耳他</rb><rp>(</rp><rt>마이타</rt><rp>)</rp></ruby>는 <ruby class="hanja api grade52"><rb>公式的</rb><rp>(</rp><rt>공식적</rt><rp>)</rp></ruby>으로 <ruby class="hanja api grade52"><rb>獨立</rb><rp>(</rp><rt>독립</rt><rp>)</rp></ruby>을 <ruby class="hanja api grade40"><rb>宣言</rb><rp>(</rp><rt>선언</rt><rp>)</rp></ruby>하고 <ruby class="hanja api grade30"><rb>英聯邦</rb><rp>(</rp><rt>영연방</rt><rp>)</rp></ruby> <ruby class="hanja api grade42"><rb>會員國</rb><rp>(</rp><rt>회원국</rt><rp>)</rp></ruby>을 <ruby class="hanja api grade32"><rb>維持</rb><rp>(</rp><rt>유지</rt><rp>)</rp></ruby>했습니다.
</p><p>1979<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> 3<ruby class="hanja api grade80"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby> 31<ruby class="hanja api grade80"><rb>日</rb><rp>(</rp><rt>일</rt><rp>)</rp></ruby>, <ruby class="hanja api grade60"><rb>英國</rb><rp>(</rp><rt>영국</rt><rp>)</rp></ruby>은 <ruby class="hanja api grade50"><rb>馬耳他</rb><rp>(</rp><rt>마이타</rt><rp>)</rp></ruby>에 있는 <ruby class="hanja api grade72"><rb>軍事</rb><rp>(</rp><rt>군사</rt><rp>)</rp></ruby> <ruby class="hanja api grade52"><rb>基地</rb><rp>(</rp><rt>기지</rt><rp>)</rp></ruby>에서 <ruby class="hanja api grade20"><rb>撤收</rb><rp>(</rp><rt>철수</rt><rp>)</rp></ruby>했다.
</p>
<h2><span id=".E6.94.BF.E6.B2.BB.28.EC.A0.95.EC.B9.98.29"></span><span class="mw-headline" id="政治(정치)"><ruby class="hanja api grade42"><rb>政治</rb><rp>(</rp><rt>정치</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E9%A6%AC%E8%80%B3%E4%BB%96&amp;action=edit&amp;section=3" title="部分 編輯: 政治(정치)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span id=".E5.A4.A7.E7.B6.B1.28.EB.8C.80.EA.B0.95.29"></span><span class="mw-headline" id="大綱(대강)"><ruby class="hanja api grade32"><rb>大綱</rb><rp>(</rp><rt>대강</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E9%A6%AC%E8%80%B3%E4%BB%96&amp;action=edit&amp;section=4" title="部分 編輯: 大綱(대강)">編輯</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>1966<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> <ruby class="hanja api grade50"><rb>馬耳他</rb><rp>(</rp><rt>마이타</rt><rp>)</rp></ruby>는 <ruby class="hanja api grade52"><rb>獨立</rb><rp>(</rp><rt>독립</rt><rp>)</rp></ruby> <ruby class="hanja api grade52"><rb>以後</rb><rp>(</rp><rt>이후</rt><rp>)</rp></ruby> 첫 <ruby class="hanja api grade42"><rb>總選</rb><rp>(</rp><rt>총선</rt><rp>)</rp></ruby>에서 <ruby class="hanja api grade42"><rb>國民黨</rb><rp>(</rp><rt>국민당</rt><rp>)</rp></ruby>이 <ruby class="hanja api grade60"><rb>勝利</rb><rp>(</rp><rt>승리</rt><rp>)</rp></ruby>했다. 
</p><p>1971-1987<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> <ruby class="hanja api grade42"><rb>勞動黨</rb><rp>(</rp><rt>노동당</rt><rp>)</rp></ruby>이 <ruby class="hanja api grade42"><rb>連續</rb><rp>(</rp><rt>연속</rt><rp>)</rp></ruby> <ruby class="hanja api grade32"><rb>執權</rb><rp>(</rp><rt>집권</rt><rp>)</rp></ruby>했다. 그 <ruby class="hanja api grade72"><rb>後</rb><rp>(</rp><rt>후</rt><rp>)</rp></ruby> <ruby class="hanja api grade42"><rb>國民黨</rb><rp>(</rp><rt>국민당</rt><rp>)</rp></ruby>과 <ruby class="hanja api grade42"><rb>勞動黨</rb><rp>(</rp><rt>노동당</rt><rp>)</rp></ruby>은 오랫동안 번갈아 <ruby class="hanja api grade32"><rb>執權</rb><rp>(</rp><rt>집권</rt><rp>)</rp></ruby>했다.
</p>
<h3><span id=".E8.AD.B0.E6.9C.83.28.EC.9D.98.ED.9A.8C.29"></span><span class="mw-headline" id="議會(의회)"><ruby class="hanja api grade42"><rb>議會</rb><rp>(</rp><rt>의회</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E9%A6%AC%E8%80%B3%E4%BB%96&amp;action=edit&amp;section=5" title="部分 編輯: 議會(의회)">編輯</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><ruby class="hanja api grade50"><rb>下院</rb><rp>(</rp><rt>하원</rt><rp>)</rp></ruby>이라 불리며 <ruby class="hanja api grade40"><rb>普通選擧</rb><rp>(</rp><rt>보통선거</rt><rp>)</rp></ruby>로 <ruby class="hanja api grade50"><rb>選出</rb><rp>(</rp><rt>선출</rt><rp>)</rp></ruby>되며 <ruby class="hanja api grade50"><rb>任期</rb><rp>(</rp><rt>임기</rt><rp>)</rp></ruby>는 5<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby>이며 <ruby class="hanja api grade42"><rb>立法府</rb><rp>(</rp><rt>입법부</rt><rp>)</rp></ruby>이다. 
</p><p>이<ruby class="hanja api grade60"><rb>番</rb><rp>(</rp><rt>번</rt><rp>)</rp></ruby> <ruby class="hanja api grade42"><rb>議會</rb><rp>(</rp><rt>의회</rt><rp>)</rp></ruby>는 2022<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> 3<ruby class="hanja api grade80"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade50"><rb>選出</rb><rp>(</rp><rt>선출</rt><rp>)</rp></ruby>되어 <ruby class="hanja api grade60"><rb>現在</rb><rp>(</rp><rt>현재</rt><rp>)</rp></ruby> 79<ruby class="hanja api grade60"><rb>席</rb><rp>(</rp><rt>석</rt><rp>)</rp></ruby>으로 <ruby class="hanja api grade42"><rb>勞動黨</rb><rp>(</rp><rt>노동당</rt><rp>)</rp></ruby>(<ruby class="hanja api grade40"><rb>與黨</rb><rp>(</rp><rt>여당</rt><rp>)</rp></ruby>) 43<ruby class="hanja api grade60"><rb>席</rb><rp>(</rp><rt>석</rt><rp>)</rp></ruby>, <ruby class="hanja api grade42"><rb>國民黨</rb><rp>(</rp><rt>국민당</rt><rp>)</rp></ruby>(<ruby class="hanja api grade42"><rb>野黨</rb><rp>(</rp><rt>야당</rt><rp>)</rp></ruby>) 35<ruby class="hanja api grade60"><rb>席</rb><rp>(</rp><rt>석</rt><rp>)</rp></ruby>, <ruby class="hanja api grade52"><rb>獨立</rb><rp>(</rp><rt>독립</rt><rp>)</rp></ruby> <ruby class="hanja api grade42"><rb>議員</rb><rp>(</rp><rt>의원</rt><rp>)</rp></ruby> 1<ruby class="hanja api grade60"><rb>席</rb><rp>(</rp><rt>석</rt><rp>)</rp></ruby>이다. 안굴루 파루지아(Anġlu Farrugia)의 <ruby class="hanja api grade42"><rb>議長</rb><rp>(</rp><rt>의장</rt><rp>)</rp></ruby>.
</p>
<h3><span id=".E6.94.BF.E5.BA.9C.28.EC.A0.95.EB.B6.80.29"></span><span class="mw-headline" id="政府(정부)"><ruby class="hanja api grade42"><rb>政府</rb><rp>(</rp><rt>정부</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E9%A6%AC%E8%80%B3%E4%BB%96&amp;action=edit&amp;section=6" title="部分 編輯: 政府(정부)">編輯</a><span class="mw-editsection-bracket">]</span></span></h3>
