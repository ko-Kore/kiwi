<h2><span id="槪要(개요)"></span><span class="mw-headline" id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"><ruby class="hanja grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E5%8B%95%E7%89%A9%EC%9D%98_%EC%88%B2_%EC%8B%9C%EB%A6%AC%EC%A6%88&amp;action=edit&amp;section=1" title="부분 편집: 槪要(개요)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja grade72"><rb>動物</rb><rp>(</rp><rt>동물</rt><rp>)</rp></ruby>의 숲(どうぶつの<span class="noruby"><ruby><rb><ruby class="hanja grade32"><rb>森</rb><rp>(</rp><rt>삼</rt><rp>)</rp></ruby></rb><rp>(</rp><rt>もり</rt><rp>)</rp></ruby></span>)은 <a href="/index.php?title=%E4%BB%BB%E5%A4%A9%E5%A0%82&amp;action=edit&amp;redlink=1" class="new" title="任天堂 (없는 문서)"><ruby class="hanja grade52 unknown"><rb>任天堂</rb><rp>(</rp><rt>임천당</rt><rp>)</rp></ruby></a>에서 <ruby class="hanja grade42"><rb>製作</rb><rp>(</rp><rt>제작</rt><rp>)</rp></ruby>하는 게임 시리즈이다.
</p>
<h2><span id="特徵(특징)"></span><span class="mw-headline" id=".E7.89.B9.E5.BE.B5.28.ED.8A.B9.EC.A7.95.29"><ruby class="hanja grade32"><rb>特徵</rb><rp>(</rp><rt>특징</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E5%8B%95%E7%89%A9%EC%9D%98_%EC%88%B2_%EC%8B%9C%EB%A6%AC%EC%A6%88&amp;action=edit&amp;section=2" title="부분 편집: 特徵(특징)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja grade52"><rb>現實</rb><rp>(</rp><rt>현실</rt><rp>)</rp></ruby>과 같이 <ruby class="hanja grade72"><rb>時間</rb><rp>(</rp><rt>시간</rt><rp>)</rp></ruby>이 흘러가는 <a href="/index.php?title=%EC%88%B2&amp;action=edit&amp;redlink=1" class="new" title="숲 (없는 문서)">숲</a> 속 <a href="/index.php?title=%EB%A7%88%EC%9D%84&amp;action=edit&amp;redlink=1" class="new" title="마을 (없는 문서)">마을</a>에서 <ruby class="hanja grade72"><rb>動物</rb><rp>(</rp><rt>동물</rt><rp>)</rp></ruby> <ruby class="hanja grade70"><rb>住民</rb><rp>(</rp><rt>주민</rt><rp>)</rp></ruby>들과 <ruby class="hanja grade52"><rb>交流</rb><rp>(</rp><rt>교류</rt><rp>)</rp></ruby>하며 <a href="/index.php?title=%EB%82%9A%EC%8B%9C&amp;action=edit&amp;redlink=1" class="new" title="낚시 (없는 문서)">낚시</a>, <a href="/index.php?title=%E6%8E%A1%E9%9B%86&amp;action=edit&amp;redlink=1" class="new" title="採集 (없는 문서)"><ruby class="hanja grade40 unknown"><rb>採集</rb><rp>(</rp><rt>채집</rt><rp>)</rp></ruby></a><ruby class="hanja grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby> 여러 <ruby class="hanja grade72"><rb>活動</rb><rp>(</rp><rt>활동</rt><rp>)</rp></ruby>을 하며 <ruby class="hanja grade40"><rb>假象</rb><rp>(</rp><rt>가상</rt><rp>)</rp></ruby>의 <ruby class="hanja grade72"><rb>生活</rb><rp>(</rp><rt>생활</rt><rp>)</rp></ruby>을 느긋하게  즐기는 게임이다.
</p>
