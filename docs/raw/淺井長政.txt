<h2><span id=".E7.95.A5.E6.AD.B7.28.EC.95.BD.EB.A0.A5.29"></span><span class="mw-headline" id="略歷(약력)"><ruby class="hanja api grade40"><rb>略歷</rb><rp>(</rp><rt>약력</rt><rp>)</rp></ruby>&nbsp;</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%B7%BA%E4%BA%95%E9%95%B7%E6%94%BF&amp;action=edit&amp;section=1" title="부분 편집: 略歷(약력)&nbsp;">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja api grade32"><rb>淺井</rb><rp>(</rp><rt>천정</rt><rp>)</rp></ruby> <ruby class="hanja api grade72"><rb>家門</rb><rp>(</rp><rt>가문</rt><rp>)</rp></ruby>을 <ruby class="hanja api grade60"><rb>日本</rb><rp>(</rp><rt>일본</rt><rp>)</rp></ruby> <ruby class="hanja api grade60"><rb>近江國</rb><rp>(</rp><rt>근강국</rt><rp>)</rp></ruby> <ruby class="hanja api grade62"><rb>北部</rb><rp>(</rp><rt>북부</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade72"><rb>大名</rb><rp>(</rp><rt>대명</rt><rp>)</rp></ruby>로 <ruby class="hanja api grade62"><rb>成長</rb><rp>(</rp><rt>성장</rt><rp>)</rp></ruby>시켜 <ruby class="hanja api grade60"><rb>近江國</rb><rp>(</rp><rt>근강국</rt><rp>)</rp></ruby> <ruby class="hanja api grade62"><rb>北東部</rb><rp>(</rp><rt>북동부</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade42"><rb>勢力</rb><rp>(</rp><rt>세력</rt><rp>)</rp></ruby>을 갖고 있었다. 아내 오빠 <a href="/wiki/%E7%B9%94%E7%94%B0%E4%BF%A1%E9%95%B7" title="織田信長"><ruby class="hanja api grade40"><rb>織田信長</rb><rp>(</rp><rt>직전신장</rt><rp>)</rp></ruby></a>과 <ruby class="hanja api grade32"><rb>同盟</rb><rp>(</rp><rt>동맹</rt><rp>)</rp></ruby>을 맺는 <ruby class="hanja api grade32"><rb>淺井</rb><rp>(</rp><rt>천정</rt><rp>)</rp></ruby> <ruby class="hanja api grade72"><rb>家門</rb><rp>(</rp><rt>가문</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade42"><rb>全盛期</rb><rp>(</rp><rt>전성기</rt><rp>)</rp></ruby>를 쌓았지만 나중에 <ruby class="hanja api grade40"><rb>織田信長</rb><rp>(</rp><rt>직전신장</rt><rp>)</rp></ruby>과 <ruby class="hanja api grade32"><rb>決裂</rb><rp>(</rp><rt>결렬</rt><rp>)</rp></ruby>하여 <ruby class="hanja api grade40"><rb>織田軍</rb><rp>(</rp><rt>직전군</rt><rp>)</rp></ruby>과의 싸움에 <ruby class="hanja api grade50"><rb>敗</rb><rp>(</rp><rt>패</rt><rp>)</rp></ruby>해 <ruby class="hanja api grade52"><rb>自害</rb><rp>(</rp><rt>자해</rt><rp>)</rp></ruby>하고 <ruby class="hanja api grade32"><rb>淺井</rb><rp>(</rp><rt>천정</rt><rp>)</rp></ruby> <ruby class="hanja api grade72"><rb>家門</rb><rp>(</rp><rt>가문</rt><rp>)</rp></ruby>은 <ruby class="hanja api grade32"><rb>滅亡</rb><rp>(</rp><rt>멸망</rt><rp>)</rp></ruby>했다.
</p>
