<p><ruby class="hanja grade62"><rb>大體</rb><rp>(</rp><rt>대체</rt><rp>)</rp></ruby>로 <a href="/index.php?title=%ED%94%8C%EB%A0%88%EC%9D%B4%EC%96%B4&amp;action=edit&amp;redlink=1" class="new" title="플레이어 (없는 문서)">플레이어</a>는 <a href="/index.php?title=%E5%8B%87%E8%80%85&amp;action=edit&amp;redlink=1" class="new" title="勇者 (없는 문서)"><ruby class="hanja grade60 unknown"><rb>勇者</rb><rp>(</rp><rt>용자</rt><rp>)</rp></ruby></a> ‘<a href="/index.php?title=%EB%A7%81%ED%81%AC&amp;action=edit&amp;redlink=1" class="new" title="링크 (없는 문서)">링크</a>’가 되어 <a href="/index.php?title=%ED%95%98%EC%9D%B4%EB%9E%84%E7%8E%8B%E5%9C%8B&amp;action=edit&amp;redlink=1" class="new" title="하이랄王國 (없는 문서)">하이랄<ruby class="hanja grade80 unknown"><rb>王國</rb><rp>(</rp><rt>왕국</rt><rp>)</rp></ruby></a> <ruby class="hanja grade62"><rb>等地</rb><rp>(</rp><rt>등지</rt><rp>)</rp></ruby>를 돌아다니며 <a href="/index.php?title=%EB%AA%AC%EC%8A%A4%ED%84%B0&amp;action=edit&amp;redlink=1" class="new" title="몬스터 (없는 문서)">몬스터</a>를 쓰러뜨리고 <ruby class="hanja grade42"><rb>寶物</rb><rp>(</rp><rt>보물</rt><rp>)</rp></ruby>을 찾으며 <a href="/index.php?title=%E9%AD%94%E7%8E%8B&amp;action=edit&amp;redlink=1" class="new" title="魔王 (없는 문서)"><ruby class="hanja grade20 unknown"><rb>魔王</rb><rp>(</rp><rt>마왕</rt><rp>)</rp></ruby></a> '가논' <ruby class="hanja grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby> <ruby class="hanja grade52"><rb>惡</rb><rp>(</rp><rt>악</rt><rp>)</rp></ruby>으로부터 <a href="/wiki/%E4%B8%96%E7%95%8C" title="世界"><ruby class="hanja grade62"><rb>世界</rb><rp>(</rp><rt>세계</rt><rp>)</rp></ruby></a>를 <ruby class="hanja grade50"><rb>救</rb><rp>(</rp><rt>구</rt><rp>)</rp></ruby>한다는 <a href="/index.php?title=%E5%8B%87%E5%A3%AB%E7%89%A9&amp;action=edit&amp;redlink=1" class="new" title="勇士物 (없는 문서)"><ruby class="hanja grade52 unknown"><rb>勇士物</rb><rp>(</rp><rt>용사물</rt><rp>)</rp></ruby></a> 스토리를 갖춘다.
