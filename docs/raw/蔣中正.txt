<p><b><ruby class="hanja api grade12"><rb>蔣</rb><rp>(</rp><rt>장</rt><rp>)</rp></ruby><ruby class="hanja api grade72"><rb>中正</rb><rp>(</rp><rt>중정</rt><rp>)</rp></ruby></b>은 <a href="/wiki/%E4%B8%AD%E8%8F%AF%E6%B0%91%E5%9C%8B" title="中華民國"><ruby class="hanja api grade40"><rb>中華民國</rb><rp>(</rp><rt>중화민국</rt><rp>)</rp></ruby></a>의 <a href="/index.php?title=%E6%94%BF%E6%B2%BB%E5%AE%B6&amp;action=edit&amp;redlink=1" class="new" title="政治家(없는　文書）"><ruby class="hanja api grade42 unknown"><rb>政治家</rb><rp>(</rp><rt>정치가</rt><rp>)</rp></ruby></a>, <a href="/index.php?title=%E8%BB%8D%E4%BA%BA&amp;action=edit&amp;redlink=1" class="new" title="軍人(없는　文書）"><ruby class="hanja api grade80 unknown"><rb>軍人</rb><rp>(</rp><rt>군인</rt><rp>)</rp></ruby></a>이다. 그는 <ruby class="hanja api grade40"><rb>中華民國</rb><rp>(</rp><rt>중화민국</rt><rp>)</rp></ruby>과 <a href="/index.php?title=%E4%B8%AD%E5%9C%8B%E5%9C%8B%E6%B0%91%E9%BB%A8&amp;action=edit&amp;redlink=1" class="new" title="中國國民黨(없는　文書）"><ruby class="hanja api grade42 unknown"><rb>中國國民黨</rb><rp>(</rp><rt>중국국민당</rt><rp>)</rp></ruby></a>의 <a href="/wiki/%E6%AD%B7%E5%8F%B2" title="歷史"><ruby class="hanja api grade52"><rb>歷史</rb><rp>(</rp><rt>역사</rt><rp>)</rp></ruby></a><ruby class="hanja api grade72"><rb>上</rb><rp>(</rp><rt>상</rt><rp>)</rp></ruby> 가장 <ruby class="hanja api grade52"><rb>重要</rb><rp>(</rp><rt>중요</rt><rp>)</rp></ruby>한 <ruby class="hanja api grade42"><rb>政治人</rb><rp>(</rp><rt>정치인</rt><rp>)</rp></ruby>으로 여겨진다.
</p><p><a href="/wiki/%E5%AD%AB%E6%96%87" title="孫文"><ruby class="hanja api grade60"><rb>孫文</rb><rp>(</rp><rt>손문</rt><rp>)</rp></ruby></a>의 <ruby class="hanja api grade40"><rb>後繼者</rb><rp>(</rp><rt>후계자</rt><rp>)</rp></ruby>로서 <a href="/index.php?title=%E5%8C%97%E4%BC%90&amp;action=edit&amp;redlink=1" class="new" title="北伐(없는　文書）"><ruby class="hanja api grade42 unknown"><rb>北伐</rb><rp>(</rp><rt>북벌</rt><rp>)</rp></ruby></a>을 <ruby class="hanja api grade30"><rb>完遂</rb><rp>(</rp><rt>완수</rt><rp>)</rp></ruby>하고, <ruby class="hanja api grade60"><rb>北洋</rb><rp>(</rp><rt>북양</rt><rp>)</rp></ruby><ruby class="hanja api grade40"><rb>系</rb><rp>(</rp><rt>계</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade42"><rb>勢力</rb><rp>(</rp><rt>세력</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade32"><rb>壞滅的</rb><rp>(</rp><rt>괴멸적</rt><rp>)</rp></ruby> <ruby class="hanja api grade50"><rb>最高</rb><rp>(</rp><rt>최고</rt><rp>)</rp></ruby><ruby class="hanja api grade42"><rb>指導者</rb><rp>(</rp><rt>지도자</rt><rp>)</rp></ruby>가 된다. <a href="/wiki/%E7%AC%AC%E4%BA%8C%E6%AC%A1%E4%B8%96%E7%95%8C%E5%A4%A7%E6%88%B0" title="第二次世界大戰"><ruby class="hanja api grade42"><rb>第二次世界大戰</rb><rp>(</rp><rt>제이차세계대전</rt><rp>)</rp></ruby></a>에서는 <ruby grade80"><rb>國</rb><rp>(</rp><rt>국</rt><rp>)</rp></ruby><sup id="cite_ref-1" class="reference"><a href="#cite_note-1">[<ruby class="hanja api href="/index.php?title=%E5%9C%8B%E5%85%B1%E5%85%A7%E6%88%B0&amp;action=edit&amp;redlink=1" class="new" title="國共內戰(없는　文書）"><ruby class="hanja api grade62 grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> <ruby class="hanja api grade50"><rb>死亡</rb><rp>(</rp><rt>사망</rt><rp>)</rp></ruby>할 때까지 <ruby class="hanja api grade42"><rb>回復</rb><rp>(</rp><rt>회복</rt><rp>)</rp></ruby>하지 않고 <ruby class="hanja api grade40"><rb>中華民國</rb><rp>(</rp><rt>중화민국</rt><rp>)</rp></ruby>의 <ruby class="hanja api
