<html><head></head><body><div class="mw-parser-output"><h2><span id="槪要(개요)"></span><span class="mw-headline" id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"><ruby class="hanja grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%96%87%E5%8C%96%E8%AA%9E&amp;action=edit&amp;section=1" title="부분 편집: 槪要(개요)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%E6%9C%9D%E9%AE%AE%E6%B0%91%E4%B8%BB%E4%B8%BB%E7%BE%A9%E4%BA%BA%E6%B0%91%E5%85%B1%E5%92%8C%E5%9C%8B" title="朝鮮民主主義人民共和國"><ruby class="hanja grade42"><rb>朝鮮民主主義人民共和國</rb><rp>(</rp><rt>조선민주주의인민공화국</rt><rp>)</rp></ruby></a>에서 <a href="/index.php?title=1966%E5%B9%B4&amp;action=edit&amp;redlink=1" class="new" title="1966年 (없는 문서)">1966<ruby class="hanja grade80 unknown"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby></a>부터 <ruby class="hanja grade60"><rb>使用</rb><rp>(</rp><rt>사용</rt><rp>)</rp></ruby>하는 <a href="/index.php?title=%E6%A8%99%E6%BA%96%E8%AA%9E&amp;action=edit&amp;redlink=1" class="new" title="標準語 (없는 문서)"><ruby class="hanja grade40 unknown"><rb>標準語</rb><rp>(</rp><rt>표준어</rt><rp>)</rp></ruby></a>를 말한다.
</p>
<h2><span id="特徵(특징)"></span><span class="mw-headline" id=".E7.89.B9.E5.BE.B5.28.ED.8A.B9.EC.A7.95.29"><ruby class="hanja grade32"><rb>特徵</rb><rp>(</rp><rt>특징</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%96%87%E5%8C%96%E8%AA%9E&amp;action=edit&amp;section=2" title="부분 편집: 特徵(특징)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>서울말을 <ruby class="hanja grade42"><rb>基準</rb><rp>(</rp><rt>기준</rt><rp>)</rp></ruby>으로 한 <ruby class="hanja grade80"><rb>南韓</rb><rp>(</rp><rt>남한</rt><rp>)</rp></ruby>의 <a href="/index.php?title=%E6%A8%99%E6%BA%96%E8%AA%9E&amp;action=edit&amp;redlink=1" class="new" title="標準語 (없는 문서)"><ruby class="hanja grade40 unknown"><rb>標準語</rb><rp>(</rp><rt>표준어</rt><rp>)</rp></ruby></a>에 <ruby class="hanja grade42"><rb>對應</rb><rp>(</rp><rt>대응</rt><rp>)</rp></ruby>하는 <a href="/wiki/%E5%8C%97%E9%9F%93" title="北韓"><ruby class="hanja grade80"><rb>北韓</rb><rp>(</rp><rt>북한</rt><rp>)</rp></ruby></a>의 <ruby class="hanja grade40"><rb>標準語</rb><rp>(</rp><rt>표준어</rt><rp>)</rp></ruby>이다. <ruby class="hanja grade52"><rb>文化語</rb><rp>(</rp><rt>문화어</rt><rp>)</rp></ruby>의 <ruby class="hanja grade32"><rb>音韻</rb><rp>(</rp><rt>음운</rt><rp>)</rp></ruby> <ruby class="hanja grade40"><rb>體系</rb><rp>(</rp><rt>체계</rt><rp>)</rp></ruby>는 <a href="/index.php?title=%E5%B9%B3%E5%A3%A4%E6%96%B9%E8%A8%80&amp;action=edit&amp;redlink=1" class="new" title="平壤方言 (없는 문서)"><ruby class="hanja grade32 unknown"><rb>平壤方言</rb><rp>(</rp><rt>평양방언</rt><rp>)</rp></ruby></a>보다 <a href="/index.php?title=%EC%84%9C%EC%9A%B8%E6%96%B9%E8%A8%80&amp;action=edit&amp;redlink=1" class="new" title="서울方言 (없는 문서)">서울<ruby class="hanja grade60 unknown"><rb>方言</rb><rp>(</rp><rt>방언</rt><rp>)</rp></ruby></a>과 더 비슷하지만, <a href="/wiki/%E5%8C%97%E9%9F%93" title="北韓"><ruby class="hanja grade80"><rb>北韓</rb><rp>(</rp><rt>북한</rt><rp>)</rp></ruby></a> <ruby class="hanja grade42"><rb>政府</rb><rp>(</rp><rt>정부</rt><rp>)</rp></ruby>는 <a href="/index.php?title=%E5%B9%B3%E5%A3%A4&amp;action=edit&amp;redlink=1" class="new" title="平壤 (없는 문서)"><ruby class="hanja grade32 unknown"><rb>平壤</rb><rp>(</rp><rt>평양</rt><rp>)</rp></ruby></a>말을 <ruby class="hanja grade42"><rb>基準</rb><rp>(</rp><rt>기준</rt><rp>)</rp></ruby>으로 한다고 <ruby class="hanja grade40"><rb>主張</rb><rp>(</rp><rt>주장</rt><rp>)</rp></ruby>한다.
</p><p><ruby class="hanja grade70"><rb>主</rb><rp>(</rp><rt>주</rt><rp>)</rp></ruby>된 <ruby class="hanja grade32"><rb>特徵</rb><rp>(</rp><rt>특징</rt><rp>)</rp></ruby>으로는 <a href="/index.php?title=%E9%A0%AD%E9%9F%B3%E6%B3%95%E5%89%87&amp;action=edit&amp;redlink=1" class="new" title="頭音法則 (없는 문서)"><ruby class="hanja grade50 unknown"><rb>頭音法則</rb><rp>(</rp><rt>두음법칙</rt><rp>)</rp></ruby></a>을 <ruby class="hanja grade40"><rb>適用</rb><rp>(</rp><rt>적용</rt><rp>)</rp></ruby>하지 않는 것과 <a href="/index.php?title=%E7%A1%AC%E9%9F%B3%E5%8C%96&amp;action=edit&amp;redlink=1" class="new" title="硬音化 (없는 문서)"><ruby class="hanja grade32 unknown"><rb>硬音化</rb><rp>(</rp><rt>경음화</rt><rp>)</rp></ruby></a>가 더 잘 일어난다는 것 <ruby class="hanja grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby>이 있다.
</p>
