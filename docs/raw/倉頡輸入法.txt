<h2><span id="槪要(개요)"></span><span class="mw-headline" id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"><ruby class="hanja grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E5%80%89%E9%A0%A1%E8%BC%B8%E5%85%A5%E6%B3%95&amp;action=edit&amp;section=1" title="부분 편집: 槪要(개요)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja grade0"><rb>倉頡輸入法</rb><rp>(</rp><rt>창힐수입법</rt><rp>)</rp></ruby>은 <a href="/index.php?title=%E5%AD%97%E5%BD%A2&amp;action=edit&amp;redlink=1" class="new" title="字形 (없는 문서)"><ruby class="hanja grade62 unknown"><rb>字形</rb><rp>(</rp><rt>자형</rt><rp>)</rp></ruby></a>에 <ruby class="hanja grade32"><rb>基盤</rb><rp>(</rp><rt>기반</rt><rp>)</rp></ruby>해서 <ruby class="hanja grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby>를 수 있는 <a href="/wiki/%E4%B8%AD%E5%9C%8B%E8%AA%9E_%E5%85%A5%E5%8A%9B%E6%96%B9%E5%BC%8F" title="中國語 入力方式"><ruby class="hanja grade70"><rb>中國語</rb><rp>(</rp><rt>중국어</rt><rp>)</rp></ruby> <ruby class="hanja grade60"><rb>入力方式</rb><rp>(</rp><rt>입력방식</rt><rp>)</rp></ruby></a>이다.
</p>
<h2><span id="名稱(명칭)"></span><span class="mw-headline" id=".E5.90.8D.E7.A8.B1.28.EB.AA.85.EC.B9.AD.29"><ruby class="hanja grade40"><rb>名稱</rb><rp>(</rp><rt>명칭</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E5%80%89%E9%A0%A1%E8%BC%B8%E5%85%A5%E6%B3%95&amp;action=edit&amp;section=2" title="부분 편집: 名稱(명칭)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>이름의 ‘<ruby class="hanja grade0"><rb>倉頡</rb><rp>(</rp><rt>창힐</rt><rp>)</rp></ruby>’은 <ruby class="hanja grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby>를 만들었다는 <ruby class="hanja grade52"><rb>傳說</rb><rp>(</rp><rt>전설</rt><rp>)</rp></ruby> 속 <ruby class="hanja grade72"><rb>人物</rb><rp>(</rp><rt>인물</rt><rp>)</rp></ruby> <a href="/wiki/%E5%80%89%E9%A0%A1" title="倉頡"><ruby class="hanja grade0"><rb>倉頡</rb><rp>(</rp><rt>창힐</rt><rp>)</rp></ruby></a>에서 따 왔으며, ‘<ruby class="hanja grade32"><rb>輸入法</rb><rp>(</rp><rt>수입법</rt><rp>)</rp></ruby>’은 'Input Method(<a href="/wiki/%E5%85%A5%E5%8A%9B%E6%96%B9%E5%BC%8F" title="入力方式"><ruby class="hanja grade60"><rb>入力方式</rb><rp>(</rp><rt>입력방식</rt><rp>)</rp></ruby></a>, <ruby class="hanja grade52"><rb>入力法</rb><rp>(</rp><rt>입력법</rt><rp>)</rp></ruby>)'의 <a href="/wiki/%E4%B8%AD%E5%9C%8B%E8%AA%9E" title="中國語"><ruby class="hanja grade70"><rb>中國語</rb><rp>(</rp><rt>중국어</rt><rp>)</rp></ruby></a> <ruby class="hanja grade32"><rb>譯語</rb><rp>(</rp><rt>역어</rt><rp>)</rp></ruby>이다. <sup id="cite_ref-1" class="reference"><a href="#cite_note-1">[1]</a></sup>
</p>
<h2><span id="特徵(특징)"></span><span class="mw-headline" id=".E7.89.B9.E5.BE.B5.28.ED.8A.B9.EC.A7.95.29"><ruby class="hanja grade32"><rb>特徵</rb><rp>(</rp><rt>특징</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E5%80%89%E9%A0%A1%E8%BC%B8%E5%85%A5%E6%B3%95&amp;action=edit&amp;section=3" title="부분 편집: 特徵(특징)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby>의 <ruby class="hanja grade62"><rb>字形</rb><rp>(</rp><rt>자형</rt><rp>)</rp></ruby>을 <ruby class="hanja grade32"><rb>基盤</rb><rp>(</rp><rt>기반</rt><rp>)</rp></ruby>으로 <ruby class="hanja grade62"><rb>各</rb><rp>(</rp><rt>각</rt><rp>)</rp></ruby> 글<ruby class="hanja grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>의 <ruby class="hanja grade52"><rb>入力法</rb><rp>(</rp><rt>입력법</rt><rp>)</rp></ruby>이 <ruby class="hanja grade60"><rb>定</rb><rp>(</rp><rt>정</rt><rp>)</rp></ruby>해지므로 <ruby class="hanja grade60"><rb>特定</rb><rp>(</rp><rt>특정</rt><rp>)</rp></ruby> <a href="/wiki/%E8%A8%80%E8%AA%9E" title="言語"><ruby class="hanja grade60"><rb>言語</rb><rp>(</rp><rt>언어</rt><rp>)</rp></ruby></a>의 <a href="/index.php?title=%E6%BC%A2%E5%AD%97%E9%9F%B3&amp;action=edit&amp;redlink=1" class="new" title="漢字音 (없는 문서)"><ruby class="hanja grade62 unknown"><rb>漢字音</rb><rp>(</rp><rt>한자음</rt><rp>)</rp></ruby></a>을 모르더라도 <ruby class="hanja grade40"><rb>模樣</rb><rp>(</rp><rt>모양</rt><rp>)</rp></ruby>만 알고 있다면 <ruby class="hanja grade40"><rb>支援</rb><rp>(</rp><rt>지원</rt><rp>)</rp></ruby>하는 모든 글<ruby class="hanja grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>를 <ruby class="hanja grade70"><rb>入力</rb><rp>(</rp><rt>입력</rt><rp>)</rp></ruby>할 수 있다. <ruby class="hanja grade60"><rb>例</rb><rp>(</rp><rt>예</rt><rp>)</rp></ruby>를 들어 <a href="/wiki/%E4%B8%80" title="一"><ruby class="hanja grade80"><rb>一</rb><rp>(</rp><rt>일</rt><rp>)</rp></ruby></a>을 세<ruby class="hanja grade60"><rb>番</rb><rp>(</rp><rt>번</rt><rp>)</rp></ruby> 누르면 <a href="/wiki/%E4%B8%89" title="三"><ruby class="hanja grade80"><rb>三</rb><rp>(</rp><rt>삼</rt><rp>)</rp></ruby></a>이, <a href="/wiki/%E6%97%A5" title="日"><ruby class="hanja grade80"><rb>日</rb><rp>(</rp><rt>일</rt><rp>)</rp></ruby></a>과 <a href="/wiki/%E6%9C%88" title="月"><ruby class="hanja grade80"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby></a>을 <ruby class="hanja grade42"><rb>次例</rb><rp>(</rp><rt>차례</rt><rp>)</rp></ruby>로 누르면 <a href="/index.php?title=%E6%98%8E&amp;action=edit&amp;redlink=1" class="new" title="明 (없는 문서)"><ruby class="hanja grade62 unknown"><rb>明</rb><rp>(</rp><rt>명</rt><rp>)</rp></ruby></a>이 <ruby class="hanja grade70"><rb>入力</rb><rp>(</rp><rt>입력</rt><rp>)</rp></ruby>된다.
</p><p>모든 글<ruby class="hanja grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>는 5<ruby class="hanja grade50"><rb>打內</rb><rp>(</rp><rt>타내</rt><rp>)</rp></ruby>에 <ruby class="hanja grade50"><rb>完成</rb><rp>(</rp><rt>완성</rt><rp>)</rp></ruby>된다. 위의 <ruby class="hanja grade80"><rb>三</rb><rp>(</rp><rt>삼</rt><rp>)</rp></ruby>이나 <ruby class="hanja grade62"><rb>明</rb><rp>(</rp><rt>명</rt><rp>)</rp></ruby>보다 <ruby class="hanja grade40"><rb>複雜</rb><rp>(</rp><rt>복잡</rt><rp>)</rp></ruby>하고 <ruby class="hanja grade32"><rb>劃</rb><rp>(</rp><rt>획</rt><rp>)</rp></ruby>이 많은 글<ruby class="hanja grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>는 <ruby class="hanja grade50"><rb>規則</rb><rp>(</rp><rt>규칙</rt><rp>)</rp></ruby>에 따라 <ruby class="hanja grade72"><rb>中間</rb><rp>(</rp><rt>중간</rt><rp>)</rp></ruby>의 <ruby class="hanja grade42"><rb>要素</rb><rp>(</rp><rt>요소</rt><rp>)</rp></ruby>를 <ruby class="hanja grade40"><rb>省略</rb><rp>(</rp><rt>생략</rt><rp>)</rp></ruby>해서 어떻게든 한 글<ruby class="hanja grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>가 5<ruby class="hanja grade50"><rb>打內</rb><rp>(</rp><rt>타내</rt><rp>)</rp></ruby>로 <ruby class="hanja grade40"><rb>構成</rb><rp>(</rp><rt>구성</rt><rp>)</rp></ruby>되게 된다.
</p>
<h3><span id="破字法(파자법)"></span><span class="mw-headline" id=".E7.A0.B4.E5.AD.97.E6.B3.95.28.ED.8C.8C.EC.9E.90.EB.B2.95.29"><ruby class="hanja grade42"><rb>破字法</rb><rp>(</rp><rt>파자법</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E5%80%89%E9%A0%A1%E8%BC%B8%E5%85%A5%E6%B3%95&amp;action=edit&amp;section=4" title="부분 편집: 破字法(파자법)">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><ruby class="hanja grade50"><rb>英文字板</rb><rp>(</rp><rt>영문자판</rt><rp>)</rp></ruby> 26키 <ruby class="hanja grade52"><rb>以內</rb><rp>(</rp><rt>이내</rt><rp>)</rp></ruby>에 모든 <ruby class="hanja grade40"><rb>構成要素</rb><rp>(</rp><rt>구성요소</rt><rp>)</rp></ruby>를 집어넣다 보니, 글<ruby class="hanja grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>에 <ruby class="hanja grade62"><rb>對</rb><rp>(</rp><rt>대</rt><rp>)</rp></ruby>해 갖고 있는 이미지에 <ruby class="hanja grade62"><rb>反</rb><rp>(</rp><rt>반</rt><rp>)</rp></ruby>하거나 억지스러운 <ruby class="hanja grade42"><rb>破字法</rb><rp>(</rp><rt>파자법</rt><rp>)</rp></ruby>이 <ruby class="hanja grade70"><rb>登場</rb><rp>(</rp><rt>등장</rt><rp>)</rp></ruby>하기도 한다. <ruby class="hanja grade52"><rb>代表的</rb><rp>(</rp><rt>대표적</rt><rp>)</rp></ruby>인 <ruby class="hanja grade50"><rb>例示</rb><rp>(</rp><rt>예시</rt><rp>)</rp></ruby>로 ‘<a href="/index.php?title=%E8%BB%8A&amp;action=edit&amp;redlink=1" class="new" title="車 (없는 문서)"><ruby class="hanja grade72 unknown"><rb>車</rb><rp>(</rp><rt>차</rt><rp>)</rp></ruby></a>’는 <ruby class="hanja grade42"><rb>十田十</rb><rp>(</rp><rt>십전십</rt><rp>)</rp></ruby>으로 <ruby class="hanja grade70"><rb>入力</rb><rp>(</rp><rt>입력</rt><rp>)</rp></ruby>한다(...).
</p><p>또, <ruby class="hanja grade52"><rb>基本的</rb><rp>(</rp><rt>기본적</rt><rp>)</rp></ruby>인 <ruby class="hanja grade40"><rb>構成要素</rb><rp>(</rp><rt>구성요소</rt><rp>)</rp></ruby>조차도 다른 글<ruby class="hanja grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>와 <ruby class="hanja grade40"><rb>組合</rb><rp>(</rp><rt>조합</rt><rp>)</rp></ruby>할 때에는 <ruby class="hanja grade32"><rb>劃</rb><rp>(</rp><rt>획</rt><rp>)</rp></ruby>의 <ruby class="hanja grade62"><rb>一部</rb><rp>(</rp><rt>일부</rt><rp>)</rp></ruby>만 따서 가져오거나 그걸 다시 <ruby class="hanja grade30"><rb>擴大解釋</rb><rp>(</rp><rt>확대해석</rt><rp>)</rp></ruby>하는 <ruby class="hanja grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby> 마냥 쉽게만 배울 수 있는 <ruby class="hanja grade42"><rb>水準</rb><rp>(</rp><rt>수준</rt><rp>)</rp></ruby>은 아니다.
</p>
<h2><span id="用例(용례)"></span><span class="mw-headline" id=".E7.94.A8.E4.BE.8B.28.EC.9A.A9.EB.A1.80.29"><ruby class="hanja grade60"><rb>用例</rb><rp>(</rp><rt>용례</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E5%80%89%E9%A0%A1%E8%BC%B8%E5%85%A5%E6%B3%95&amp;action=edit&amp;section=5" title="부분 편집: 用例(용례)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja grade32"><rb>字形基盤</rb><rp>(</rp><rt>자형기반</rt><rp>)</rp></ruby>이며 1<ruby class="hanja grade52"><rb>字當</rb><rp>(</rp><rt>자당</rt><rp>)</rp></ruby> <ruby class="hanja grade50"><rb>最大</rb><rp>(</rp><rt>최대</rt><rp>)</rp></ruby> 5<ruby class="hanja grade50"><rb>打</rb><rp>(</rp><rt>타</rt><rp>)</rp></ruby>라는 빠른 <ruby class="hanja grade60"><rb>速度</rb><rp>(</rp><rt>속도</rt><rp>)</rp></ruby>를 갖췄기 때문에, <a href="/wiki/%E4%B8%AD%E5%9C%8B%E8%AA%9E" title="中國語"><ruby class="hanja grade70"><rb>中國語</rb><rp>(</rp><rt>중국어</rt><rp>)</rp></ruby></a>를 모르거나 <ruby class="hanja grade60"><rb>使用</rb><rp>(</rp><rt>사용</rt><rp>)</rp></ruby>하지 않더라도 이걸 익혀서 <a href="/wiki/%E6%BC%A2%E6%96%87" title="漢文"><ruby class="hanja grade70"><rb>漢文</rb><rp>(</rp><rt>한문</rt><rp>)</rp></ruby></a> <ruby class="hanja grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby>을 <ruby class="hanja grade70"><rb>入力</rb><rp>(</rp><rt>입력</rt><rp>)</rp></ruby>할 때 <ruby class="hanja grade60"><rb>使用</rb><rp>(</rp><rt>사용</rt><rp>)</rp></ruby>하는 <ruby class="hanja grade40"><rb>境遇</rb><rp>(</rp><rt>경우</rt><rp>)</rp></ruby>도 있다.
</p><p>또, <ruby class="hanja grade60"><rb>方言</rb><rp>(</rp><rt>방언</rt><rp>)</rp></ruby>이 <ruby class="hanja grade40"><rb>多樣</rb><rp>(</rp><rt>다양</rt><rp>)</rp></ruby>한 <a href="/wiki/%E4%B8%AD%E5%9C%8B%E8%AA%9E" title="中國語"><ruby class="hanja grade70"><rb>中國語</rb><rp>(</rp><rt>중국어</rt><rp>)</rp></ruby></a>의 <ruby class="hanja grade40"><rb>境遇</rb><rp>(</rp><rt>경우</rt><rp>)</rp></ruby> <ruby class="hanja grade62"><rb>各</rb><rp>(</rp><rt>각</rt><rp>)</rp></ruby> <a href="/index.php?title=%E6%96%B9%E8%A8%80&amp;action=edit&amp;redlink=1" class="new" title="方言 (없는 문서)"><ruby class="hanja grade60 unknown"><rb>方言</rb><rp>(</rp><rt>방언</rt><rp>)</rp></ruby></a><ruby class="hanja grade60"><rb>別</rb><rp>(</rp><rt>별</rt><rp>)</rp></ruby>로 <ruby class="hanja grade62"><rb>漢字音</rb><rp>(</rp><rt>한자음</rt><rp>)</rp></ruby>이 <ruby class="hanja grade40"><rb>差異</rb><rp>(</rp><rt>차이</rt><rp>)</rp></ruby>가 많이 나는데, 이와 같은 <ruby class="hanja grade32"><rb>字形基盤</rb><rp>(</rp><rt>자형기반</rt><rp>)</rp></ruby> <ruby class="hanja grade60"><rb>入力方式</rb><rp>(</rp><rt>입력방식</rt><rp>)</rp></ruby>을 <ruby class="hanja grade60"><rb>使用</rb><rp>(</rp><rt>사용</rt><rp>)</rp></ruby>할 경우, <ruby class="hanja grade42"><rb>基準</rb><rp>(</rp><rt>기준</rt><rp>)</rp></ruby>이 되는 <ruby class="hanja grade62"><rb>發音</rb><rp>(</rp><rt>발음</rt><rp>)</rp></ruby>을 잘 모르는 <ruby class="hanja grade60"><rb>方言</rb><rp>(</rp><rt>방언</rt><rp>)</rp></ruby> <ruby class="hanja grade60"><rb>話者</rb><rp>(</rp><rt>화자</rt><rp>)</rp></ruby>도 손쉽게 글<ruby class="hanja grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>를 <ruby class="hanja grade70"><rb>入力</rb><rp>(</rp><rt>입력</rt><rp>)</rp></ruby>할 수 있다는 <ruby class="hanja grade40"><rb>長點</rb><rp>(</rp><rt>장점</rt><rp>)</rp></ruby>이 있다. 이러한 <ruby class="hanja grade52"><rb>特性</rb><rp>(</rp><rt>특성</rt><rp>)</rp></ruby> 때문에 <a href="/index.php?title=%E9%A6%99%E6%B8%AF&amp;action=edit&amp;redlink=1" class="new" title="香港 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>香港</rb><rp>(</rp><rt>향항</rt><rp>)</rp></ruby></a>에서 <a
