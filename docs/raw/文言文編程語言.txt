<h2><span id=".E7.89.B9.E5.BE.B5.28.ED.8A.B9.EC.A7.95.29"></span><span class="mw-headline" id="特徵(특징)"><ruby class="hanja grade32"><rb>特徵</rb><rp>(</rp><rt>특징</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%96%87%E8%A8%80%E6%96%87%E7%B7%A8%E7%A8%8B%E8%AA%9E%E8%A8%80&amp;action=edit&amp;section=1" title="부분 편집: 特徵(특징)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja grade52"><rb>基本的</rb><rp>(</rp><rt>기본적</rt><rp>)</rp></ruby>으로는 <ruby class="hanja grade52"><rb>古典</rb><rp>(</rp><rt>고전</rt><rp>)</rp></ruby><a href="/wiki/%E6%BC%A2%E6%96%87" title="漢文"><ruby class="hanja grade70"><rb>漢文</rb><rp>(</rp><rt>한문</rt><rp>)</rp></ruby></a><ruby class="hanja grade62"><rb>風</rb><rp>(</rp><rt>풍</rt><rp>)</rp></ruby>으로 보이는 프로그래밍 <ruby class="hanja grade60"><rb>言語</rb><rp>(</rp><rt>언어</rt><rp>)</rp></ruby>. <ruby class="hanja grade42"><rb>實際</rb><rp>(</rp><rt>실제</rt><rp>)</rp></ruby> <ruby class="hanja grade70"><rb>漢文</rb><rp>(</rp><rt>한문</rt><rp>)</rp></ruby>을 <ruby class="hanja grade32"><rb>基盤</rb><rp>(</rp><rt>기반</rt><rp>)</rp></ruby>으로 <ruby class="hanja grade40"><rb>文法體系</rb><rp>(</rp><rt>문법체계</rt><rp>)</rp></ruby>가 짜여 있으며, <ruby class="hanja grade10"><rb>仔細</rb><rp>(</rp><rt>자세</rt><rp>)</rp></ruby>히 읽어보지 않으면 <ruby class="hanja grade32"><rb>大綱</rb><rp>(</rp><rt>대강</rt><rp>)</rp></ruby> <a href="/index.php?title=%E5%84%92%E6%95%8E&amp;action=edit&amp;redlink=1" class="new" title="儒敎 (없는 문서)"><ruby class="hanja grade40 unknown"><rb>儒敎</rb><rp>(</rp><rt>유교</rt><rp>)</rp></ruby></a><ruby class="hanja grade42"><rb>經典</rb><rp>(</rp><rt>경전</rt><rp>)</rp></ruby>같이 보이는 것이 <ruby class="hanja grade32"><rb>特徵</rb><rp>(</rp><rt>특징</rt><rp>)</rp></ruby>.
</p><p>가장 <ruby class="hanja grade52"><rb>基本的</rb><rp>(</rp><rt>기본적</rt><rp>)</rp></ruby>인 <a href="/index.php?title=%E8%AE%8A%E6%95%B8&amp;action=edit&amp;redlink=1" class="new" title="變數 (없는 문서)"><ruby class="hanja grade52 unknown"><rb>變數</rb><rp>(</rp><rt>변수</rt><rp>)</rp></ruby></a><ruby class="hanja grade40"><rb>宣言</rb><rp>(</rp><rt>선언</rt><rp>)</rp></ruby>을 <ruby class="hanja grade60"><rb>例</rb><rp>(</rp><rt>예</rt><rp>)</rp></ruby>로들자면 <code>吾有一數。曰三。名之曰「甲」。</code>과 같이 <ruby class="hanja grade62"><rb>作成</rb><rp>(</rp><rt>작성</rt><rp>)</rp></ruby>한다. <ruby class="hanja grade70"><rb>韓國語</rb><rp>(</rp><rt>한국어</rt><rp>)</rp></ruby>로 <ruby class="hanja grade32"><rb>直譯</rb><rp>(</rp><rt>직역</rt><rp>)</rp></ruby>(...)하면 "나는 <ruby class="hanja grade70"><rb>數</rb><rp>(</rp><rt>수</rt><rp>)</rp></ruby>가 하나 있다. 그<ruby class="hanja grade70"><rb>數</rb><rp>(</rp><rt>수</rt><rp>)</rp></ruby>는 3이고 이름은 「<ruby class="hanja grade40"><rb>甲</rb><rp>(</rp><rt>갑</rt><rp>)</rp></ruby>」이라한다."쯤 되는데, <a href="/index.php?title=%EC%9E%90%EB%B0%94%EC%8A%A4%ED%81%AC%EB%A6%BD%ED%8A%B8&amp;action=edit&amp;redlink=1" class="new" title="자바스크립트 (없는 문서)">자바스크립트</a> 코드 <code>var a = 3;</code>에 <ruby class="hanja grade30"><rb>該當</rb><rp>(</rp><rt>해당</rt><rp>)</rp></ruby>한다.
</p><p>또, <ruby class="hanja grade50"><rb>原則上</rb><rp>(</rp><rt>원칙상</rt><rp>)</rp></ruby> <a href="/index.php?title=%E5%8F%A5%E8%AE%80%E9%BB%9E&amp;action=edit&amp;redlink=1" class="new" title="句讀點 (없는 문서)"><ruby class="hanja grade40 unknown"><rb>句讀點</rb><rp>(</rp><rt>구두점</rt><rp>)</rp></ruby></a>을 찍지않는 <ruby class="hanja grade70"><rb>漢文</rb><rp>(</rp><rt>한문</rt><rp>)</rp></ruby>의 <ruby class="hanja grade52"><rb>特性</rb><rp>(</rp><rt>특성</rt><rp>)</rp></ruby>에 맞춰, <ruby class="hanja grade40"><rb>句讀點</rb><rp>(</rp><rt>구두점</rt><rp>)</rp></ruby>은 <ruby class="hanja grade32"><rb>選擇事項</rb><rp>(</rp><rt>선택사항</rt><rp>)</rp></ruby>이다. <ruby class="hanja grade32"><rb>卽</rb><rp>(</rp><rt>즉</rt><rp>)</rp></ruby>, <ruby class="hanja grade40"><rb>普通</rb><rp>(</rp><rt>보통</rt><rp>)</rp></ruby>은 <ruby class="hanja grade42"><rb>語句</rb><rp>(</rp><rt>어구</rt><rp>)</rp></ruby>마다 <ruby class="hanja grade40"><rb>句讀點</rb><rp>(</rp><rt>구두점</rt><rp>)</rp></ruby>을 붙이고 줄바꿈과 <a href="/index.php?title=%EB%93%A4%EC%97%AC%EC%93%B0%EA%B8%B0&amp;action=edit&amp;redlink=1" class="new" title="들여쓰기 (없는 문서)">들여쓰기</a>를 <ruby class="hanja grade40"><rb>適切</rb><rp>(</rp><rt>적절</rt><rp>)</rp></ruby>히 하여 <a href="/index.php?title=%E5%8F%AF%E8%AE%80%E6%80%A7&amp;action=edit&amp;redlink=1" class="new" title="可讀性 (없는 문서)"><ruby class="hanja grade50 unknown"><rb>可讀性</rb><rp>(</rp><rt>가독성</rt><rp>)</rp></ruby></a>을 높이겠지만, 하고만 싶다면 <ruby class="hanja grade50"><rb>改行文字</rb><rp>(</rp><rt>개행문자</rt><rp>)</rp></ruby>를 <ruby class="hanja grade32"><rb>包含</rb><rp>(</rp><rt>포함</rt><rp>)</rp></ruby>한 <ruby class="hanja grade72"><rb>空白</rb><rp>(</rp><rt>공백</rt><rp>)</rp></ruby>을 모두 없애고 <ruby class="hanja grade72"><rb>中間中間</rb><rp>(</rp><rt>중간중간</rt><rp>)</rp></ruby> 찍힌 <a href="/index.php?title=%EA%B3%A0%EB%A6%AC%E9%BB%9E&amp;action=edit&amp;redlink=1" class="new" title="고리點 (없는 문서)">고리<ruby class="hanja grade40 unknown"><rb>點</rb><rp>(</rp><rt>점</rt><rp>)</rp></ruby></a>까지 다 날려서 주르륵 <a href="/wiki/%E6%BC%A2%E5%AD%97" title="漢字"><ruby class="hanja grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby></a>만 가득하게 만들어놨더라도 <ruby class="hanja grade52"><rb>實行</rb><rp>(</rp><rt>실행</rt><rp>)</rp></ruby>이 <ruby class="hanja grade50"><rb>可能</rb><rp>(</rp><rt>가능</rt><rp>)</rp></ruby>하다는 것이다.
</p>

<pre>吾有一數。曰三。名之曰「甲」。
