<h2><span id=".E5.90.8D.E7.A8.B1.28.EB.AA.85.EC.B9.AD.29"></span><span class="mw-headline" id="名稱(명칭)"><ruby class="hanja api grade40"><rb>名稱</rb><rp>(</rp><rt>명칭</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E5%A4%A7%E9%98%AA%E5%B8%82&amp;action=edit&amp;section=1" title="部分　編輯：名稱(명칭)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja api grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby><ruby class="hanja api grade62"><rb>表記</rb><rp>(</rp><rt>표기</rt><rp>)</rp></ruby>는 <ruby class="hanja api grade50"><rb>當初</rb><rp>(</rp><rt>당초</rt><rp>)</rp></ruby> “<ruby class="hanja api grade80"><rb>大</rb><rp>(</rp><rt>대</rt><rp>)</rp></ruby><ruby class="hanja api grade0"><rb>坂</rb><rp>(</rp><rt>판</rt><rp>)</rp></ruby>”이 <ruby class="hanja api grade32"><rb>一般的</rb><rp>(</rp><rt>일반적</rt><rp>)</rp></ruby>이었지만, “<ruby class="hanja api grade0"><rb>坂</rb><rp>(</rp><rt>판</rt><rp>)</rp></ruby>”의 글<ruby class="hanja api grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>를 <ruby class="hanja api grade42"><rb>分解</rb><rp>(</rp><rt>분해</rt><rp>)</rp></ruby>하면 “땅(<ruby class="hanja api grade80"><rb>土</rb><rp>(</rp><rt>토</rt><rp>)</rp></ruby>)에 휜다(<ruby class="hanja api grade62"><rb>反</rb><rp>(</rp><rt>반</rt><rp>)</rp></ruby>)”라고 읽혀 <ruby class="hanja api grade52"><rb>財數</rb><rp>(</rp><rt>재수</rt><rp>)</rp></ruby>가 없다고 하여, <a href="/index.php?title=%E6%B1%9F%E6%88%B6%E6%99%82%E4%BB%A3&amp;action=edit&amp;redlink=1" class="new" title="江戶時代(없는　文書）"><ruby class="hanja api grade42 unknown"><rb>江戶</rb><rp>(</rp><rt>강호</rt><rp>)</rp></ruby><ruby class="hanja api grade62 unknown"><rb>時代</rb><rp>(</rp><rt>시대</rt><rp>)</rp></ruby></a> 무렵부터 “<ruby class="hanja api grade12"><rb>大阪</rb><rp>(</rp><rt>대판</rt><rp>)</rp></ruby>”이라고도 쓰게 되어, <a href="/index.php?title=%E6%98%8E%E6%B2%BB%E6%99%82%E4%BB%A3&amp;action=edit&amp;redlink=1" class="new" title="明治時代(없는　文書）"><ruby class="hanja api grade42 unknown"><rb>明治</rb><rp>(</rp><rt>명치</rt><rp>)</rp></ruby><ruby class="hanja api grade62 unknown"><rb>時代</rb><rp>(</rp><rt>시대</rt><rp>)</rp></ruby></a>에는 <ruby class="hanja api grade12"><rb>大阪</rb><rp>(</rp><rt>대판</rt><rp>)</rp></ruby>이라는 <ruby class="hanja api grade62"><rb>表記</rb><rp>(</rp><rt>표기</rt><rp>)</rp></ruby>가 <ruby class="hanja api grade52"><rb>定着</rb><rp>(</rp><rt>정착</rt><rp>)</rp></ruby>한다. <ruby class="hanja api grade52"><rb>一說</rb><rp>(</rp><rt>일설</rt><rp>)</rp></ruby>에 “<ruby class="hanja api grade0"><rb>坂</rb><rp>(</rp><rt>판</rt><rp>)</rp></ruby>”에서 “<ruby class="hanja api grade12"><rb>阪</rb><rp>(</rp><rt>판</rt><rp>)</rp></ruby>”으로 <ruby class="hanja api grade40"><rb>變更</rb><rp>(</rp><rt>변경</rt><rp>)</rp></ruby>된 것은 <a href="/index.php?title=%E6%98%8E%E6%B2%BB%E6%96%B0%E6%94%BF%E5%BA%9C&amp;action=edit&amp;redlink=1" class="new" title="明治新政府(없는　文書）"><ruby class="hanja api grade42 unknown"><rb>明治</rb><rp>(</rp><rt>명치</rt><rp>)</rp></ruby><ruby class="hanja api grade42 unknown"><rb>新政府</rb><rp>(</rp><rt>신정부</rt><rp>)</rp></ruby></a>가 “<ruby class="hanja api grade0"><rb>坂</rb><rp>(</rp><rt>판</rt><rp>)</rp></ruby>”이 “<ruby class="hanja api grade52"><rb>士</rb><rp>(</rp><rt>사</rt><rp>)</rp></ruby>가 <ruby class="hanja api grade62"><rb>反</rb><rp>(</rp><rt>반</rt><rp>)</rp></ruby>한다”, <ruby class="hanja api grade32"><rb>卽</rb><rp>(</rp><rt>즉</rt><rp>)</rp></ruby> “<a href="/index.php?title=%E6%AD%A6%E5%A3%AB&amp;action=edit&amp;redlink=1" class="new" title="武士(없는　文書）"><ruby class="hanja api grade42 unknown"><rb>武士</rb><rp>(</rp><rt>무사</rt><rp>)</rp></ruby></a>가 <ruby class="hanja api grade30"><rb>叛亂</rb><rp>(</rp><rt>반란</rt><rp>)</rp></ruby>하다”라고 읽을 수 있기 때문에 “<ruby class="hanja api grade0"><rb>坂</rb><rp>(</rp><rt>판</rt><rp>)</rp></ruby>”이라는 글<ruby class="hanja api grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>를 싫어했다는 것도 있다.
</p>
