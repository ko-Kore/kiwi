<p>QWERTY는 가장 널리 쓰이는 <a href="/wiki/%EC%BB%B4%ED%93%A8%ED%84%B0_%ED%82%A4%EB%B3%B4%EB%93%9C" title="컴퓨터 키보드">컴퓨터 키보드</a> <a href="/wiki/%EB%A1%9C%EB%A7%88%E5%AD%97" title="로마字">로마<ruby class="hanja grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby></a> <ruby class="hanja grade42"><rb>配列</rb><rp>(</rp><rt>배열</rt><rp>)</rp></ruby>이다. 이름은 키보드 <ruby class="hanja grade32"><rb>左側上段</rb><rp>(</rp><rt>좌측상단</rt><rp>)</rp></ruby>부터 <ruby class="hanja grade42"><rb>配定</rb><rp>(</rp><rt>배정</rt><rp>)</rp></ruby>된 알파벳을 <ruby class="hanja grade50"><rb>順序</rb><rp>(</rp><rt>순서</rt><rp>)</rp></ruby>대로 읽으면 QWERTY가 된다는 것에서 <ruby class="hanja grade60"><rb>由來</rb><rp>(</rp><rt>유래</rt><rp>)</rp></ruby>했다.
</p>
<h2><span id=".E7.89.B9.E5.BE.B5.28.ED.8A.B9.EC.A7.95.29"></span><span class="mw-headline" id="特徵(특징)"><ruby class="hanja grade32"><rb>特徵</rb><rp>(</rp><rt>특징</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=QWERTY&amp;action=edit&amp;section=1" title="부분 편집: 特徵(특징)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/index.php?title=%E8%8B%B1%E6%96%87&amp;action=edit&amp;redlink=1" class="new" title="英文 (없는 문서)"><ruby class="hanja grade60 unknown"><rb>英文</rb><rp>(</rp><rt>영문</rt><rp>)</rp></ruby></a>을 <ruby class="hanja grade50"><rb>打字</rb><rp>(</rp><rt>타자</rt><rp>)</rp></ruby>하는 데에 있어서는 매우 <ruby class="hanja grade32"><rb>非效率的</rb><rp>(</rp><rt>비효율적</rt><rp>)</rp></ruby>이다. 자주 쓰이는 글<ruby class="hanja grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>가 <ruby class="hanja grade52"><rb>基本</rb><rp>(</rp><rt>기본</rt><rp>)</rp></ruby>자리로부터 멀리 떨어져 있는게 많은데다 <ruby class="hanja grade72"><rb>左右</rb><rp>(</rp><rt>좌우</rt><rp>)</rp></ruby> 손의 <ruby class="hanja grade42"><rb>分配</rb><rp>(</rp><rt>분배</rt><rp>)</rp></ruby>, 같은 손가락의 <ruby class="hanja grade42"><rb>連</rb><rp>(</rp><rt>연</rt><rp>)</rp></ruby>이은 <ruby class="hanja grade60"><rb>使用</rb><rp>(</rp><rt>사용</rt><rp>)</rp></ruby> <ruby class="hanja grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby> <ruby class="hanja grade70"><rb>不便</rb><rp>(</rp><rt>불편</rt><rp>)</rp></ruby>한<ruby class="hanja grade40"><rb>點</rb><rp>(</rp><rt>점</rt><rp>)</rp></ruby>이 매우 많다. <ruby class="hanja grade30"><rb>甚至於</rb><rp>(</rp><rt>심지어</rt><rp>)</rp></ruby> ‘stewardess’같은 <ruby class="hanja grade62"><rb>一部</rb><rp>(</rp><rt>일부</rt><rp>)</rp></ruby> <ruby class="hanja grade42"><rb>單語</rb><rp>(</rp><rt>단어</rt><rp>)</rp></ruby>는 왼손으로만 쳐야 한다!
</p><p>이 <ruby class="hanja grade32"><rb>非效率的</rb><rp>(</rp><rt>비효율적</rt><rp>)</rp></ruby>인 <ruby class="hanja grade42"><rb>配列</rb><rp>(</rp><rt>배열</rt><rp>)</rp></ruby>의 <ruby class="hanja grade40"><rb>起源</rb><rp>(</rp><rt>기원</rt><rp>)</rp></ruby>에 <ruby class="hanja grade62"><rb>對</rb><rp>(</rp><rt>대</rt><rp>)</rp></ruby>해서는 <ruby class="hanja grade32"><rb>諸說</rb><rp>(</rp><rt>제설</rt><rp>)</rp></ruby>이 있지만, 그<ruby class="hanja grade80"><rb>中</rb><rp>(</rp><rt>중</rt><rp>)</rp></ruby> <ruby class="hanja grade32"><rb>機械式</rb><rp>(</rp><rt>기계식</rt><rp>)</rp></ruby> <a href="/wiki/%E6%89%93%E5%AD%97%E6%A9%9F" title="打字機"><ruby class="hanja grade40"><rb>打字機</rb><rp>(</rp><rt>타자기</rt><rp>)</rp></ruby></a><ruby class="hanja grade52"><rb>時節</rb><rp>(</rp><rt>시절</rt><rp>)</rp></ruby> <a href="/wiki/%E6%B4%BB%E5%AD%97" title="活字"><ruby class="hanja grade70"><rb>活字</rb><rp>(</rp><rt>활자</rt><rp>)</rp></ruby></a>가 꼬이지 않게 하기 <ruby class="hanja grade42"><rb>爲</rb><rp>(</rp><rt>위</rt><rp>)</rp></ruby>해 <ruby class="hanja grade52"><rb>意圖的</rb><rp>(</rp><rt>의도적</rt><rp>)</rp></ruby>으로 <ruby class="hanja grade30"><rb>頻度</rb><rp>(</rp><rt>빈도</rt><rp>)</rp></ruby>가 높은 글<ruby class="hanja grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>를 떨어뜨려 놓는 <ruby class="hanja grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby>의 <ruby class="hanja grade52"><rb>方法</rb><rp>(</rp><rt>방법</rt><rp>)</rp></ruby>으로 <ruby class="hanja grade42"><rb>設計</rb><rp>(</rp><rt>설계</rt><rp>)</rp></ruby>했다는 <ruby class="hanja grade40"><rb>主張</rb><rp>(</rp><rt>주장</rt><rp>)</rp></ruby>도 있다.
</p><p>‘TYPEWRITER’에 들어가는 모든 글<ruby class="hanja grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>가 <ruby class="hanja grade40"><rb>最上段</rb><rp>(</rp><rt>최상단</rt><rp>)</rp></ruby>에 <ruby class="hanja grade42"><rb>配定</rb><rp>(</rp><rt>배정</rt><rp>)</rp></ruby>되어 있다는 걸로 보아 <ruby class="hanja grade42"><rb>草創期</rb><rp>(</rp><rt>초창기</rt><rp>)</rp></ruby> <a href="/wiki/%E6%89%93%E5%AD%97%E6%A9%9F" title="打字機"><ruby class="hanja grade40"><rb>打字機</rb><rp>(</rp><rt>타자기</rt><rp>)</rp></ruby></a>를 <ruby class="hanja grade42"><rb>試演</rb><rp>(</rp><rt>시연</rt><rp>)</rp></ruby>할때 쉽게 칠 수 있도록 만든 것이라는 <ruby class="hanja grade52"><rb>說</rb><rp>(</rp><rt>설</rt><rp>)</rp></ruby>도 있다.
</p>
<h2><span id=".E8.AE.8A.E7.A8.AE.28.EB.B3.80.EC.A2.85.29"></span><span class="mw-headline" id="變種(변종)"><ruby class="hanja grade52"><rb>變種</rb><rp>(</rp><rt>변종</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=QWERTY&amp;action=edit&amp;section=2" title="부분 편집: 變種(변종)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja grade60"><rb>使用</rb><rp>(</rp><rt>사용</rt><rp>)</rp></ruby>되는 <ruby class="hanja grade40"><rb>地域</rb><rp>(</rp><rt>지역</rt><rp>)</rp></ruby>의 <ruby class="hanja grade52"><rb>言語實情</rb><rp>(</rp><rt>언어실정</rt><rp>)</rp></ruby>에 맞게 <ruby class="hanja grade62"><rb>一部</rb><rp>(</rp><rt>일부</rt><rp>)</rp></ruby> 글<ruby class="hanja grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>의 <ruby class="hanja grade42"><rb>位置</rb><rp>(</rp><rt>위치</rt><rp>)</rp></ruby>가 <ruby class="hanja grade40"><rb>調整</rb><rp>(</rp><rt>조정</rt><rp>)</rp></ruby>되기도 한다. <a href="/index.php?title=AZERTY&amp;action=edit&amp;redlink=1" class="new" title="AZERTY (없는 문서)">AZERTY</a>, <a href="/index.php?title=QWERTZ&amp;action=edit&amp;redlink=1" class="new" title="QWERTZ (없는 문서)">QWERTZ</a> <ruby class="hanja grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby>이 그 <ruby class="hanja grade60"><rb>例</rb><rp>(</rp><rt>예</rt><rp>)</rp></ruby>. 이런 <ruby class="hanja grade50"><rb>字板</rb><rp>(</rp><rt>자판</rt><rp>)</rp></ruby>들은 <ruby class="hanja grade60"><rb>英語</rb><rp>(</rp><rt>영어</rt><rp>)</rp></ruby>에서 쓰는 <a href="/wiki/%EB%A1%9C%EB%A7%88%E5%AD%97" title="로마字">로마<ruby class="hanja grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby></a> 알파벳 <ruby class="hanja grade52"><rb>以外</rb><rp>(</rp><rt>이외</rt><rp>)</rp></ruby>에 <ruby class="hanja grade60"><rb>各言語別</rb><rp>(</rp><rt>각언어별</rt><rp>)</rp></ruby>로 <ruby class="hanja grade32"><rb>追加</rb><rp>(</rp><rt>추가</rt><rp>)</rp></ruby>된 글<ruby class="hanja grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>는 맨 위 <ruby class="hanja grade42"><rb>數字列</rb><rp>(</rp><rt>숫자렬</rt><rp>)</rp></ruby>이나 오른쪽 <ruby class="hanja grade60"><rb>記號</rb><rp>(</rp><rt>기호</rt><rp>)</rp></ruby>키에 밀려나 <ruby class="hanja grade42"><rb>配置</rb><rp>(</rp><rt>배치</rt><rp>)</rp></ruby>된다는 <ruby class="hanja grade40"><rb>共通點</rb><rp>(</rp><rt>공통점</rt><rp>)</rp></ruby>도 가진다.
</p>
