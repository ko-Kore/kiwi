<h2><span id="槪要(개요)"></span><span class="mw-headline" id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"><ruby class="hanja grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%ED%94%84%EB%A1%9C%EA%B7%B8%EB%A0%88%EC%8B%9C%EB%B8%8C_%EB%A1%9D&amp;action=edit&amp;section=1" title="부분 편집: 槪要(개요)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>프로그레시브 록은 <a href="/index.php?title=%EC%82%AC%EC%9D%B4%ED%82%A4%EB%8D%B8%EB%A6%AD_%EB%A1%9D&amp;action=edit&amp;redlink=1" class="new" title="사이키델릭 록 (없는 문서)">사이키델릭 록</a>, <a href="/index.php?title=%EC%95%84%ED%8A%B8_%EB%A1%9D&amp;action=edit&amp;redlink=1" class="new" title="아트 록 (없는 문서)">아트 록</a>에서 <ruby class="hanja grade40"><rb>派生</rb><rp>(</rp><rt>파생</rt><rp>)</rp></ruby>한 <a href="/wiki/%EB%A1%9D_%E9%9F%B3%E6%A8%82" title="록 音樂">록 <ruby class="hanja grade62"><rb>音樂</rb><rp>(</rp><rt>음악</rt><rp>)</rp></ruby></a> 장르이다.
</p>
<h2><span id="歷史(역사)"></span><span class="mw-headline" id=".E6.AD.B7.E5.8F.B2.28.EC.97.AD.EC.82.AC.29"><ruby class="hanja grade52"><rb>歷史</rb><rp>(</rp><rt>역사</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%ED%94%84%EB%A1%9C%EA%B7%B8%EB%A0%88%EC%8B%9C%EB%B8%8C_%EB%A1%9D&amp;action=edit&amp;section=2" title="부분 편집: 歷史(역사)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>그 <ruby class="hanja grade60"><rb>由來</rb><rp>(</rp><rt>유래</rt><rp>)</rp></ruby>는 <a href="/index.php?title=%EC%A7%80%EB%AF%B8_%ED%97%A8%EB%93%9C%EB%A6%AD%EC%8A%A4&amp;action=edit&amp;redlink=1" class="new" title="지미 헨드릭스 (없는 문서)">지미 헨드릭스</a>를 비롯한 <a href="/index.php?title=%EC%82%AC%EC%9D%B4%ED%82%A4%EB%8D%B8%EB%A6%AD_%EB%A1%9D&amp;action=edit&amp;redlink=1" class="new" title="사이키델릭 록 (없는 문서)">사이키델릭 록</a> 씬에서 록<ruby class="hanja grade62"><rb>音樂</rb><rp>(</rp><rt>음악</rt><rp>)</rp></ruby>의 <ruby class="hanja grade32"><rb>演奏</rb><rp>(</rp><rt>연주</rt><rp>)</rp></ruby>의<ruby class="hanja grade30"><rb>幅</rb><rp>(</rp><rt>폭</rt><rp>)</rp></ruby>을 넓히는 움직임이 <ruby class="hanja grade32"><rb>補足</rb><rp>(</rp><rt>보족</rt><rp>)</rp></ruby>된 것에서 <ruby class="hanja grade62"><rb>始作</rb><rp>(</rp><rt>시작</rt><rp>)</rp></ruby>한다. 그 <ruby class="hanja grade80"><rb>中</rb><rp>(</rp><rt>중</rt><rp>)</rp></ruby>에서, <ruby class="hanja grade40"><rb>過激</rb><rp>(</rp><rt>과격</rt><rp>)</rp></ruby>한 <ruby class="hanja grade32"><rb>演奏</rb><rp>(</rp><rt>연주</rt><rp>)</rp></ruby>를 <ruby class="hanja grade32"><rb>追求</rb><rp>(</rp><rt>추구</rt><rp>)</rp></ruby>한 "<a href="/index.php?title=%ED%95%98%EB%93%9C_%EB%A1%9D&amp;action=edit&amp;redlink=1" class="new" title="하드 록 (없는 문서)">하드 록</a>"과, 더욱 <ruby class="hanja grade32"><rb>演奏</rb><rp>(</rp><rt>연주</rt><rp>)</rp></ruby>의 <ruby class="hanja grade30"><rb>幅</rb><rp>(</rp><rt>폭</rt><rp>)</rp></ruby>을 넓혀 <a href="/index.php?title=%ED%81%B4%EB%9E%98%EC%8B%9D_%E9%9F%B3%E6%A8%82&amp;action=edit&amp;redlink=1" class="new" title="클래식 音樂 (없는 문서)">클래식 <ruby class="hanja grade62 unknown"><rb>音樂</rb><rp>(</rp><rt>음악</rt><rp>)</rp></ruby></a>이나 <a href="/index.php?title=%EC%9E%AC%EC%A6%88&amp;action=edit&amp;redlink=1" class="new" title="재즈 (없는 문서)">재즈</a>로부터 <ruby class="hanja grade32"><rb>影響</rb><rp>(</rp><rt>영향</rt><rp>)</rp></ruby>을 받은 <b>"프로그레시브 록"</b>이 태어났다.
</p><p><a href="/index.php?title=%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C&amp;action=edit&amp;redlink=1" class="new" title="잉글랜드 (없는 문서)">잉글랜드</a>의 "<a href="/index.php?title=%ED%82%B9_%ED%81%AC%EB%A6%BC%EC%8A%A8&amp;action=edit&amp;redlink=1" class="new" title="킹 크림슨 (없는 문서)">킹 크림슨</a>"이 장르를 <ruby class="hanja grade62"><rb>始作</rb><rp>(</rp><rt>시작</rt><rp>)</rp></ruby>하고, <a href="/index.php?title=%ED%95%91%ED%81%AC_%ED%94%8C%EB%A1%9C%EC%9D%B4%EB%93%9C&amp;action=edit&amp;redlink=1" class="new" title="핑크 플로이드 (없는 문서)">핑크 플로이드</a>가 장르의 <ruby class="hanja grade40"><rb>存在</rb><rp>(</rp><rt>존재</rt><rp>)</rp></ruby>를 <ruby class="hanja grade42"><rb>確立</rb><rp>(</rp><rt>확립</rt><rp>)</rp></ruby>했다. ‘프로그레시브 록’이라는 이름의 <ruby class="hanja grade60"><rb>由來</rb><rp>(</rp><rt>유래</rt><rp>)</rp></ruby>는 <ruby class="hanja grade32"><rb>諸說</rb><rp>(</rp><rt>제설</rt><rp>)</rp></ruby>이 있지만 핑크 플로이드의 앨범 <a href="/index.php?title=%EC%95%84%ED%86%B0_%ED%95%98%ED%8A%B8_%EB%A7%88%EB%8D%94&amp;action=edit&amp;redlink=1" class="new" title="아톰 하트 마더 (없는 문서)">아톰 하트 마더</a> <ruby class="hanja grade32"><rb>日本盤</rb><rp>(</rp><rt>일본반</rt><rp>)</rp></ruby>의 <ruby class="hanja grade40"><rb>宣傳</rb><rp>(</rp><rt>선전</rt><rp>)</rp></ruby>에 <ruby class="hanja grade60"><rb>使用</rb><rp>(</rp><rt>사용</rt><rp>)</rp></ruby>된 "핑크 플로이드의 길은 프로그레시브 록의 길이다!(ピンク・フロイドの<ruby class="hanja grade72"><rb>道</rb><rp>(</rp><rt>도</rt><rp>)</rp></ruby>はプログレッシヴ・ロックの<ruby class="hanja grade72"><rb>道</rb><rp>(</rp><rt>도</rt><rp>)</rp></ruby>なり!)"라는 <ruby class="hanja grade60"><rb>文章</rb><rp>(</rp><rt>문장</rt><rp>)</rp></ruby>이 <ruby class="hanja grade60"><rb>由來</rb><rp>(</rp><rt>유래</rt><rp>)</rp></ruby>라는 <ruby class="hanja grade52"><rb>說</rb><rp>(</rp><rt>설</rt><rp>)</rp></ruby>이 <ruby class="hanja grade70"><rb>有力</rb><rp>(</rp><rt>유력</rt><rp>)</rp></ruby>하다.
</p>
