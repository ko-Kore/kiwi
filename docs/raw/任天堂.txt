<p><ruby class="hanja api grade42"><rb>草創期</rb><rp>(</rp><rt>초창기</rt><rp>)</rp></ruby>에는 <a href="/index.php?title=%E8%8A%B1%E9%AC%AA%E5%BC%B5&amp;action=edit&amp;redlink=1" class="new" title="花鬪張 (없는 文書)"><ruby class="hanja api grade40 unknown"><rb>花鬪張</rb><rp>(</rp><rt>화투장</rt><rp>)</rp></ruby></a>이나 <a href="/index.php?title=%ED%8A%B8%EB%9F%BC%ED%94%84_%EC%B9%B4%EB%93%9C&amp;action=edit&amp;redlink=1" class="new" title="트럼프 카드 (없는 文書)">트럼프 카드</a><sup id="cite_ref-1" class="reference"><a href="#cite_note-1">[1]</a></sup>를 만들던 <ruby class="hanja api grade62"><rb>會社</rb><rp>(</rp><rt>회사</rt><rp>)</rp></ruby>였다가 <a href="/index.php?title=%EA%B2%8C%EC%9E%84%E6%A9%9F&amp;action=edit&amp;redlink=1" class="new" title="게임機 (없는 文書)">게임<ruby class="hanja api grade40 unknown"><rb>機</rb><rp>(</rp><rt>기</rt><rp>)</rp></ruby></a> <ruby class="hanja api grade60"><rb>開發</rb><rp>(</rp><rt>개발</rt><rp>)</rp></ruby>을 <ruby class="hanja api grade62"><rb>始作</rb><rp>(</rp><rt>시작</rt><rp>)</rp></ruby>하고, <a href="/index.php?title=%EA%B2%8C%EC%9E%84%26%EC%9B%8C%EC%B9%98&amp;action=edit&amp;redlink=1" class="new" title="게임&amp;워치 (없는 文書)">게임&amp;워치</a>와 <a href="/index.php?title=%ED%8C%A8%EB%B0%80%EB%A6%AC%EC%BB%B4%ED%93%A8%ED%84%B0&amp;action=edit&amp;redlink=1" class="new" title="패밀리컴퓨터 (없는 文書)">패밀리컴퓨터</a>로 <ruby class="hanja api grade52"><rb>本格的</rb><rp>(</rp><rt>본격적</rt><rp>)</rp></ruby>으로 게임<ruby class="hanja api grade40"><rb>機</rb><rp>(</rp><rt>기</rt><rp>)</rp></ruby> <ruby class="hanja api grade60"><rb>開發</rb><rp>(</rp><rt>개발</rt><rp>)</rp></ruby>에 힘쓰게 된다.

<h3><span id=".EA.B2.8C.EC.9E.84_.EC.86.8C.ED.94.84.ED.8A.B8.EC.9B.A8.EC.96.B4"></span><span class="mw-headline" id="게임_소프트웨어">게임 소프트웨어</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E4%BB%BB%E5%A4%A9%E5%A0%82&amp;action=edit&amp;section=6" title="部分 編輯: 게임 소프트웨어">編輯</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>게임<ruby class="hanja api grade40"><rb>機</rb><rp>(</rp><rt>기</rt><rp>)</rp></ruby>(하드웨어)와 게임(소프트웨어)를 <a href="/index.php?title=%EC%95%A0%ED%94%8C&amp;action=edit&amp;redlink=1" class="new" title="애플 (없는 文書)">모두 만든다.</a> 이를 <ruby class="hanja api grade62"><rb>利用</rb><rp>(</rp><rt>이용</rt><rp>)</rp></ruby>해 Wii/DS <ruby class="hanja api grade52"><rb>以後</rb><rp>(</rp><rt>이후</rt><rp>)</rp></ruby>로 게임 하드웨어 <ruby class="hanja api grade62"><rb>自體</rb><rp>(</rp><rt>자체</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade40"><rb>多樣</rb><rp>(</rp><rt>다양</rt><rp>)</rp></ruby>한 <ruby class="hanja api grade42"><rb>試圖</rb><rp>(</rp><rt>시도</rt><rp>)</rp></ruby>를 하면서 그에 알맞는 게임 시스템이나 <ruby class="hanja api grade50"><rb>技法</rb><rp>(</rp><rt>기법</rt><rp>)</rp></ruby>을 <ruby class="hanja api grade42"><rb>試驗</rb><rp>(</rp><rt>시험</rt><rp>)</rp></ruby>하는것이 보인다. 또, <ruby class="hanja api grade32"><rb>後述</rb><rp>(</rp><rt>후술</rt><rp>)</rp></ruby>할 <ruby class="hanja api grade62"><rb>自社</rb><rp>(</rp><rt>자사</rt><rp>)</rp></ruby>IP로 나온 이른바 '퍼스트 파티' 게임 시리즈들은 이 <ruby class="hanja api grade62"><rb>會社</rb><rp>(</rp><rt>회사</rt><rp>)</rp></ruby>나 게임을 잘 알지 못하더라도 <a href="/index.php?title=%EB%A7%88%EB%A6%AC%EC%98%A4_%EC%8B%9C%EB%A6%AC%EC%A6%88&amp;action=edit&amp;redlink=1" class="new" title="마리오 시리즈 (없는 文書)">이름<ruby class="hanja api grade42 unknown"><rb>程度</rb><rp>(</rp><rt>정도</rt><rp>)</rp></ruby>는</a> <a href="/index.php?title=%ED%8F%AC%EC%BC%93%EB%AA%AC%EC%8A%A4%ED%84%B0_%EC%8B%9C%EB%A6%AC%EC%A6%88&amp;action=edit&amp;redlink=1" class="new" title="포켓몬스터 시리즈 (없는 文書)">들어본 적 있거나</a> <a href="/wiki/%EC%A0%A4%EB%8B%A4%EC%9D%98%E5%82%B3%E8%AA%AA_%EC%8B%9C%EB%A6%AC%EC%A6%88" title="젤다의傳說 시리즈">나왔다 하면 온갖 <ruby class="hanja api grade32"><rb>媒體</rb><rp>(</rp><rt>매체</rt><rp>)</rp></ruby>에서 <ruby class="hanja api grade40"><rb>滿點</rb><rp>(</rp><rt>만점</rt><rp>)</rp></ruby>을 쓸어버리는</a> <ruby class="hanja api grade42"><rb>認知度</rb><rp>(</rp><rt>인지도</rt><rp>)</rp></ruby>와 퀄리티를 자랑한다.
</p>
