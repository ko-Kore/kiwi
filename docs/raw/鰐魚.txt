<h2><span id=".E9.87.8B.E7.BE.A9.28.EC.84.9D.EC.9D.98.29"></span><span class="mw-headline" id="釋義(석의)"><ruby class="hanja api grade32"><rb>釋義</rb><rp>(</rp><rt>석의</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E9%B0%90%E9%AD%9A&amp;action=edit&amp;section=1" title="部分 編輯: 釋義(석의)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja api grade10"><rb>爬蟲類</rb><rp>(</rp><rt>파충류</rt><rp>)</rp></ruby> <ruby class="hanja api grade72"><rb>動物</rb><rp>(</rp><rt>동물</rt><rp>)</rp></ruby> 몸길이는 몇 미터이고 머리와 몸통은 <ruby class="hanja api grade72"><rb>平平</rb><rp>(</rp><rt>평평</rt><rp>)</rp></ruby>하고 꼬리는 길다. 온몸에 딱딱한 <ruby class="hanja api grade20"><rb>皮膚</rb><rp>(</rp><rt>피부</rt><rp>)</rp></ruby>와 <ruby class="hanja api grade52"><rb>角質</rb><rp>(</rp><rt>각질</rt><rp>)</rp></ruby> 비늘이 있다. 팔다리가 짧고 기어다니는 것도 잘하고 <ruby class="hanja api grade30"><rb>水泳</rb><rp>(</rp><rt>수영</rt><rp>)</rp></ruby>도 할 수 있다. <ruby class="hanja api grade40"><rb>層積</rb><rp>(</rp><rt>층적</rt><rp>)</rp></ruby>으로 사나워서 <ruby class="hanja api grade72"><rb>動物</rb><rp>(</rp><rt>동물</rt><rp>)</rp></ruby>을 잡아 먹는다. <a href="/index.php?title=%E7%86%B1%E5%B8%B6&amp;action=edit&amp;redlink=1" class="new" title="熱帶 (없는 文書)"><ruby class="hanja api grade42 unknown"><rb>熱帶</rb><rp>(</rp><rt>열대</rt><rp>)</rp></ruby></a>와 <a href="/index.php?title=%E4%BA%9E%E7%86%B1%E5%B8%B6&amp;action=edit&amp;redlink=1" class="new" title="亞熱帶 (없는 文書)"><ruby class="hanja api grade32 unknown"><rb>亞熱帶</rb><rp>(</rp><rt>아열대</rt><rp>)</rp></ruby></a><a href="/index.php?title=%E6%B5%B7%E5%B2%B8&amp;action=edit&amp;redlink=1" class="new" title="海岸 (없는 文書)"><ruby class="hanja api grade32 unknown"><rb>海岸</rb><rp>(</rp><rt>해안</rt><rp>)</rp></ruby></a>과 <ruby class="hanja api grade72"><rb>江</rb><rp>(</rp><rt>강</rt><rp>)</rp></ruby>과 <ruby class="hanja api grade50"><rb>湖水</rb><rp>(</rp><rt>호수</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade42"><rb>分布</rb><rp>(</rp><rt>분포</rt><rp>)</rp></ruby>한다.
