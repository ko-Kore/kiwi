</p><p>1147<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> <ruby class="hanja api grade30"><rb>莫斯科</rb><rp>(</rp><rt>막사과</rt><rp>)</rp></ruby><ruby class="hanja api grade62"><rb>大公</rb><rp>(</rp><rt>대공</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade62"><rb>時代</rb><rp>(</rp><rt>시대</rt><rp>)</rp></ruby>부터 차르 <ruby class="hanja api grade62"><rb>時代</rb><rp>(</rp><rt>시대</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade32"><rb>露西亞</rb><rp>(</rp><rt>노서아</rt><rp>)</rp></ruby>, 소비에트<ruby class="hanja api grade30"><rb>聯邦</rb><rp>(</rp><rt>연방</rt><rp>)</rp></ruby>, 그리고 <ruby class="hanja api grade60"><rb>現在</rb><rp>(</rp><rt>현재</rt><rp>)</rp></ruby> <ruby class="hanja api grade32"><rb>露西亞</rb><rp>(</rp><rt>노서아</rt><rp>)</rp></ruby><ruby class="hanja api grade30"><rb>聯邦</rb><rp>(</rp><rt>연방</rt><rp>)</rp></ruby>의 나라들의 <ruby class="hanja api grade50"><rb>首都</rb><rp>(</rp><rt>수도</rt><rp>)</rp></ruby> <ruby class="hanja api grade32"><rb>役割</rb><rp>(</rp><rt>역할</rt><rp>)</rp></ruby>을 해왔다. 오랜 <ruby class="hanja api grade52"><rb>歷史</rb><rp>(</rp><rt>역사</rt><rp>)</rp></ruby>를 가진 <a href="/index.php?title=%ED%81%AC%EB%A0%98%EB%A6%B0&amp;action=edit&amp;redlink=1" class="new" title="크렘린 (없는 文書)">크렘린</a>이 있었고, 이들 <ruby class="hanja api grade42"><rb>政權</rb><rp>(</rp><rt>정권</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade60"><rb>本部</rb><rp>(</rp><rt>본부</rt><rp>)</rp></ruby>로 <ruby class="hanja api grade60"><rb>使用</rb><rp>(</rp><rt>사용</rt><rp>)</rp></ruby>되었습니다.
