<h3><span id=".E6.BC.A2.E8.AA.9E.E6.8B.BC.E9.9F.B3.28.ED.95.9C.EC.96.B4.ED.8F.89.EC.9D.8C.29"></span><span class="mw-headline" id="漢語拼音(한어평음)"><ruby class="hanja api grade0"><rb>漢語拼音</rb><rp>(</rp><rt>한어평음</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E4%B8%AD%E5%9C%8B%E8%AA%9E_%E5%85%A5%E5%8A%9B%E6%96%B9%E5%BC%8F&amp;action=edit&amp;section=2" title="부분 편집: 漢語拼音(한어평음)">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><span class="mw-default-size" typeof="mw:File"><a href="/wiki/%ED%8C%8C%EC%9D%BC:%EC%A4%91%EA%B5%AD%EC%96%B4_%ED%83%80%EC%9E%90%EA%B8%B0-_%ED%95%9C%EC%96%B4%EB%B3%91%EC%9D%8C.png" class="mw-file-description"><img src="/images/8/86/%EC%A4%91%EA%B5%AD%EC%96%B4_%ED%83%80%EC%9E%90%EA%B8%B0-_%ED%95%9C%EC%96%B4%EB%B3%91%EC%9D%8C.png" decoding="async" width="656" height="78"></a></span>
</p><p><a href="/wiki/%E6%BC%A2%E8%AA%9E%E6%8B%BC%E9%9F%B3" title="漢語拼音"><ruby class="hanja api grade0"><rb>漢語拼音</rb><rp>(</rp><rt>한어평음</rt><rp>)</rp></ruby></a> <ruby class="hanja api grade32"><rb>基盤</rb><rp>(</rp><rt>기반</rt><rp>)</rp></ruby>의 <a href="/wiki/%EB%A1%9C%EB%A7%88%E5%AD%97" title="로마字">로마<ruby class="hanja api grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby></a>에서 <a href="/wiki/%E6%BC%A2%E5%AD%97" title="漢字"><ruby class="hanja api grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby></a>로 <ruby class="hanja api grade32"><rb>變換</rb><rp>(</rp><rt>변환</rt><rp>)</rp></ruby> 하는 <ruby class="hanja api grade60"><rb>方式</rb><rp>(</rp><rt>방식</rt><rp>)</rp></ruby>이다. <a href="/wiki/%E4%B8%AD%E5%9C%8B" title="中國"><ruby class="hanja api grade80"><rb>中國</rb><rp>(</rp><rt>중국</rt><rp>)</rp></ruby></a><ruby class="hanja api grade60"><rb>本土</rb><rp>(</rp><rt>본토</rt><rp>)</rp></ruby>에서 많이 쓰이는 <ruby class="hanja api grade52"><rb>方法</rb><rp>(</rp><rt>방법</rt><rp>)</rp></ruby>이다. <ruby class="hanja api grade50"><rb>例示</rb><rp>(</rp><rt>예시</rt><rp>)</rp></ruby>로 ‘zhong’ 이라고 쓰면 ‘<ruby class="hanja api grade80"><rb>中</rb><rp>(</rp><rt>중</rt><rp>)</rp></ruby>’, ‘<ruby class="hanja api grade70"><rb>重</rb><rp>(</rp><rt>중</rt><rp>)</rp></ruby>’ <ruby class="hanja api grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby> zhong으로 <ruby class="hanja api grade62"><rb>發音</rb><rp>(</rp><rt>발음</rt><rp>)</rp></ruby>되는 <a href="/wiki/%E6%BC%A2%E5%AD%97" title="漢字"><ruby class="hanja api grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby></a>가 나온다. <ruby class="hanja api grade42"><rb>聲調</rb><rp>(</rp><rt>성조</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>普通</rb><rp>(</rp><rt>보통</rt><rp>)</rp></ruby> <ruby class="hanja api grade70"><rb>入力</rb><rp>(</rp><rt>입력</rt><rp>)</rp></ruby>하지 않아도 알아서 <ruby class="hanja api grade40"><rb>推測</rb><rp>(</rp><rt>추측</rt><rp>)</rp></ruby>된다.
</p><p><ruby class="hanja api grade50"><rb>最近</rb><rp>(</rp><rt>최근</rt><rp>)</rp></ruby>에 나오는 IME는 ‘zg’ 라고만 쳐도 <a href="/wiki/%E4%B8%AD%E5%9C%8B" title="中國"><ruby class="hanja api grade80"><rb>中國</rb><rp>(</rp><rt>중국</rt><rp>)</rp></ruby></a>이 나오는 <ruby class="hanja api grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby> <ruby class="hanja api grade60"><rb>强力</rb><rp>(</rp><rt>강력</rt><rp>)</rp></ruby>한 <ruby class="hanja api grade40"><rb>豫測</rb><rp>(</rp><rt>예측</rt><rp>)</rp></ruby>·<ruby class="hanja api grade50"><rb>自動完成</rb><rp>(</rp><rt>자동완성</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>機能</rb><rp>(</rp><rt>기능</rt><rp>)</rp></ruby>을 갖고 나온다.
</p>
<h3><span id=".E9.9B.99.E6.8B.BC.28.EC.8C.8D.ED.8F.89.29"></span><span class="mw-headline" id="雙拼(쌍평)"><ruby class="hanja api grade0"><rb>雙拼</rb><rp>(</rp><rt>쌍평</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E4%B8%AD%E5%9C%8B%E8%AA%9E_%E5%85%A5%E5%8A%9B%E6%96%B9%E5%BC%8F&amp;action=edit&amp;section=3" title="부분 편집: 雙拼(쌍평)">편집</a><span class="mw-editsection-bracket">]</span></span></h3>

이 <ruby class="hanja api grade62"><rb>部分</rb><rp>(</rp><rt>부분</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade60"><rb>本文</rb><rp>(</rp><rt>본문</rt><rp>)</rp></ruby>은 <a href="/wiki/%E9%9B%99%E6%8B%BC" title="雙拼"><ruby class="hanja api grade0"><rb>雙拼</rb><rp>(</rp><rt>쌍평</rt><rp>)</rp></ruby></a>입니다.</div></div>
<p><ruby class="hanja api grade62"><rb>各</rb><rp>(</rp><rt>각</rt><rp>)</rp></ruby> 키<ruby class="hanja api grade0"><rb>別로</rb><rp>(</rp><rt>별로</rt><rp>)</rp></ruby> <a href="/index.php?title=%E8%81%B2%E6%AF%8D&amp;action=edit&amp;redlink=1" class="new" title="聲母 (없는 문서)"><ruby class="hanja api grade42 unknown"><rb>聲母</rb><rp>(</rp><rt>성모</rt><rp>)</rp></ruby></a>와 <a href="/index.php?title=%E9%9F%BB%E6%AF%8D&amp;action=edit&amp;redlink=1" class="new" title="韻母 (없는 문서)"><ruby class="hanja api grade32 unknown"><rb>韻母</rb><rp>(</rp><rt>운모</rt><rp>)</rp></ruby></a>가 <ruby class="hanja api grade40"><rb>組合</rb><rp>(</rp><rt>조합</rt><rp>)</rp></ruby>된 <ruby class="hanja api grade42"><rb>狀態</rb><rp>(</rp><rt>상태</rt><rp>)</rp></ruby>로 <ruby class="hanja api grade42"><rb>配列</rb><rp>(</rp><rt>배열</rt><rp>)</rp></ruby>되어있어 <ruby class="hanja api grade70"><rb>中國語</rb><rp>(</rp><rt>중국어</rt><rp>)</rp></ruby>의 모든 <a href="/index.php?title=%E9%9F%B3%E7%AF%80&amp;action=edit&amp;redlink=1" class="new" title="音節 (없는 문서)"><ruby class="hanja api grade52 unknown"><rb>音節</rb><rp>(</rp><rt>음절</rt><rp>)</rp></ruby></a>을 2<ruby class="hanja api grade50"><rb>打</rb><rp>(</rp><rt>타</rt><rp>)</rp></ruby>로 칠수있는 <ruby class="hanja api grade60"><rb>方式</rb><rp>(</rp><rt>방식</rt><rp>)</rp></ruby>이다.
</p>
