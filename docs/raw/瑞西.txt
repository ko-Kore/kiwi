<h2><span id=".E6.A7.AA.E8.BF.B0.28.EA.B0.9C.EC.88.A0.29"></span><span class="mw-headline" id="槪述(개술)"><ruby class="hanja api grade32"><rb>槪述</rb><rp>(</rp><rt>개술</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E7%91%9E%E8%A5%BF&amp;action=edit&amp;section=1" title="部分 編輯: 槪述(개술)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><b><ruby class="hanja api grade20"><rb>瑞西</rb><rp>(</rp><rt>서서</rt><rp>)</rp></ruby><ruby class="hanja api grade30"><rb>聯邦</rb><rp>(</rp><rt>연방</rt><rp>)</rp></ruby></b>(<span style="font-size: grade62"><rb>中部</rb><rp>(</rp><rt>중부</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade42"><rb>位置</rb><rp>(</rp><rt>위치</rt><rp>)</rp></ruby>하며 <ruby class="hanja api grade30"><rb>列支敦士登</rb><rp>(</rp><rt>열지돈사등</rt><rp>)</rp></ruby></a>, <ruby class="hanja api grade80"><rb>南</rb><rp>(</rp><rt>남</rt><rp>)</rp></ruby>쪽은 <a href="/wiki/%E4%BC%8A%E5%A4%AA%E5%88%A9" grade42"><rb>接</rb><rp>(</rp><rt>접</rt><rp>)</rp></ruby>하고 있다. <ruby class="hanja api grade42"><rb>北溫帶</rb><rp>(</rp><rt>북온대</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade42"><rb>位置</rb><rp>(</rp><rt>위치</rt><rp>)</rp></ruby>하며 <ruby class="hanja api grade40"><rb>四季節</rb><rp>(</rp><rt>사계절</rt><rp>)</rp></ruby>이 뚜렷하고 <ruby class="hanja api
