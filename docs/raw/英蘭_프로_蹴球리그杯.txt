<h2><span id=".EA.B0.9C.EC.9A.94"></span><span class="mw-headline" id="개요">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E8%8B%B1%E8%98%AD_%ED%94%84%EB%A1%9C_%E8%B9%B4%E7%90%83%EB%A6%AC%EA%B7%B8%E6%9D%AF&amp;action=edit&amp;section=1" title="部分 編輯: 개요">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1960<ruby class="hanja api grade60"><rb>年度</rb><rp>(</rp><rt>연도</rt><rp>)</rp></ruby>터 진행중인 <a href="/wiki/%E8%8B%B1%E8%98%AD" title="英蘭"><ruby class="hanja api grade32"><rb>英蘭</rb><rp>(</rp><rt>영란</rt><rp>)</rp></ruby></a>의 <ruby class="hanja api grade20"><rb>蹴球</rb><rp>(</rp><rt>축구</rt><rp>)</rp></ruby> 대회. 흔히 '리그<ruby class="hanja api grade30"><rb>杯</rb><rp>(</rp><rt>배</rt><rp>)</rp></ruby>'이라고 줄여서 부르거나 스폰서명을 붙여 부른다.
</p>
<h2><span id=".EC.97.AD.EC.82.AC"></span><span class="mw-headline" id="역사">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E8%8B%B1%E8%98%AD_%ED%94%84%EB%A1%9C_%E8%B9%B4%E7%90%83%EB%A6%AC%EA%B7%B8%E6%9D%AF&amp;action=edit&amp;section=2" title="部分 編輯: 역사">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1950<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby>대 <a href="/wiki/%E8%8B%B1%E8%98%AD" title="英蘭"><ruby class="hanja api grade32"><rb>英蘭</rb><rp>(</rp><rt>영란</rt><rp>)</rp></ruby></a> <ruby class="hanja api grade20"><rb>蹴球</rb><rp>(</rp><rt>축구</rt><rp>)</rp></ruby> 경기장에 야간 조명을 설치하면서 최초로 실시했다. 하지만 초기에는 참가를 거부하는 팀이 생기기도 했다. 현재 1961<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> 대회를 1회 대회로 여기고 있는데 첫 결승전은 로더럼 유나이티드와 아스톤 빌라가 맞붙었고 홈 앤드 어웨이 방식으로 경기를 치렀다. 결국 아스톤 빌라가 총합 3-2로 승리하며 <a href="/wiki/EFL%E6%9D%AF" class="mw-redirect" title="EFL杯">EFL<ruby class="hanja api grade30"><rb>杯</rb><rp>(</rp><rt>배</rt><rp>)</rp></ruby></a> 최초의 우승 트로피를 차지했다. 그러다 1967<ruby class="hanja api grade60"><rb>年度</rb><rp>(</rp><rt>연도</rt><rp>)</rp></ruby>터 결승전 룰을 단판경기 방식으로 변경하였고 현재까지 이어져오고 있다. 1981<ruby class="hanja api grade60"><rb>年度</rb><rp>(</rp><rt>연도</rt><rp>)</rp></ruby>터는 대회 스폰서의 명칭을 붙여 이름을 부르고 있는데, 첫 이름은 밀크<ruby class="hanja api grade30"><rb>杯</rb><rp>(</rp><rt>배</rt><rp>)</rp></ruby>이었다.
</p><p><ruby class="hanja api grade50"><rb>下位</rb><rp>(</rp><rt>하위</rt><rp>)</rp></ruby> <ruby class="hanja api grade62"><rb>大會</rb><rp>(</rp><rt>대회</rt><rp>)</rp></ruby>로, 1~2<ruby class="hanja api grade62"><rb>部</rb><rp>(</rp><rt>부</rt><rp>)</rp></ruby> 리그 팀의 2<ruby class="hanja api grade80"><rb>軍</rb><rp>(</rp><rt>군</rt><rp>)</rp></ruby> (U-23) <ruby class="hanja api grade80"><rb>中</rb><rp>(</rp><rt>중</rt><rp>)</rp></ruby> 18팀과 EFL 리그 1 및 EFL 리그 2에 <ruby class="hanja api grade40"><rb>所屬</rb><rp>(</rp><rt>소속</rt><rp>)</rp></ruby>된 64팀이 함께 <ruby class="hanja api grade50"><rb>競技</rb><rp>(</rp><rt>경기</rt><rp>)</rp></ruby>를 치르는 EFL 트로피도 있다. 이 쪽도 스폰서가 바뀌어가며 존스톤스 페인트 트로피, 체커트레이드 트로피 <ruby class="hanja api grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>名稱</rb><rp>(</rp><rt>명칭</rt><rp>)</rp></ruby>이 바뀌다가 2021-21 시즌 <ruby class="hanja api grade42"><rb>基準</rb><rp>(</rp><rt>기준</rt><rp>)</rp></ruby>으로는 파파존스 트로피가 되었다.
</p><p>2016-17 시즌은 스폰서 없이 <ruby class="hanja api grade32"><rb>開催</rb><rp>(</rp><rt>개최</rt><rp>)</rp></ruby>되었으나, <a href="/index.php?title=%E6%B3%B0%E5%9C%8B&amp;action=edit&amp;redlink=1" class="new" title="泰國 (없는 文書)"><ruby class="hanja api grade32 unknown"><rb>泰國</rb><rp>(</rp><rt>태국</rt><rp>)</rp></ruby></a>의 에너지 드링크 <ruby class="hanja api grade62"><rb>會社</rb><rp>(</rp><rt>회사</rt><rp>)</rp></ruby> 카라바오와 새로운 스폰서 <ruby class="hanja api grade32"><rb>契約</rb><rp>(</rp><rt>계약</rt><rp>)</rp></ruby>을 맺고 2017-18 시즌부터 카라바오 <ruby class="hanja api grade30"><rb>杯</rb><rp>(</rp><rt>배</rt><rp>)</rp></ruby>이란 <ruby class="hanja api grade40"><rb>名稱</rb><rp>(</rp><rt>명칭</rt><rp>)</rp></ruby>으로 <ruby class="hanja api grade62"><rb>大會</rb><rp>(</rp><rt>대회</rt><rp>)</rp></ruby>를 <ruby class="hanja api grade42"><rb>進行</rb><rp>(</rp><rt>진행</rt><rp>)</rp></ruby>한다.
</p>
