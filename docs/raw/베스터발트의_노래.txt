<p>베스터발트의 노래 이 <ruby class="hanja api grade42"><rb>民謠</rb><rp>(</rp><rt>민요</rt><rp>)</rp></ruby>는 빌리 뮌커(<ruby class="hanja api grade32"><rb>詞</rb><rp>(</rp><rt>사</rt><rp>)</rp></ruby>)과 요제프 노이호이저(<ruby class="hanja api grade50"><rb>曲</rb><rp>(</rp><rt>곡</rt><rp>)</rp></ruby>)의 노래입니다. 이 노래는 <ruby class="hanja api grade52"><rb>種種</rb><rp>(</rp><rt>종종</rt><rp>)</rp></ruby> <ruby class="hanja api grade52"><rb>兵士</rb><rp>(</rp><rt>병사</rt><rp>)</rp></ruby>들이 불렀기 때문에 <a href="/index.php?title=%E7%8D%A8%E9%80%B8%E5%9C%8B%E9%98%B2%E8%BB%8D&amp;action=edit&amp;redlink=1" class="new" title="獨逸國防軍 (없는 文書)"><ruby class="hanja api grade32 unknown"><rb>獨逸</rb><rp>(</rp><rt>독일</rt><rp>)</rp></ruby><ruby class="hanja api grade42 unknown"><rb>國防軍</rb><rp>(</rp><rt>국방군</rt><rp>)</rp></ruby></a>이 <ruby class="hanja api grade42"><rb>如前</rb><rp>(</rp><rt>여전</rt><rp>)</rp></ruby>히 부르고 있습니다.
</p><p>하지만 <a href="/wiki/%E7%AC%AC%E4%BA%8C%E6%AC%A1%E4%B8%96%E7%95%8C%E5%A4%A7%E6%88%B0" title="第二次世界大戰"><ruby class="hanja api grade42"><rb>第二次世界大戰</rb><rp>(</rp><rt>제이차세계대전</rt><rp>)</rp></ruby></a> <ruby class="hanja api grade80"><rb>中</rb><rp>(</rp><rt>중</rt><rp>)</rp></ruby>에 이 노래는 <ruby class="hanja api grade42"><rb>輕快</rb><rp>(</rp><rt>경쾌</rt><rp>)</rp></ruby>한 <ruby class="hanja api grade50"><rb>曲調</rb><rp>(</rp><rt>곡조</rt><rp>)</rp></ruby>와 <ruby class="hanja api grade70"><rb>便安</rb><rp>(</rp><rt>편안</rt><rp>)</rp></ruby>한 <ruby class="hanja api grade32"><rb>歌詞</rb><rp>(</rp><rt>가사</rt><rp>)</rp></ruby> 때문에 <ruby class="hanja api grade32"><rb>獨逸</rb><rp>(</rp><rt>독일</rt><rp>)</rp></ruby> <ruby class="hanja api grade80"><rb>軍人</rb><rp>(</rp><rt>군인</rt><rp>)</rp></ruby>들이 불렀습니다. 그러나 이 그림자는 노래의 <ruby class="hanja api grade32"><rb>光彩</rb><rp>(</rp><rt>광채</rt><rp>)</rp></ruby>를 숨기지 않았고, 이 노래는 <ruby class="hanja api grade42"><rb>設立</rb><rp>(</rp><rt>설립</rt><rp>)</rp></ruby> <ruby class="hanja api grade72"><rb>後</rb><rp>(</rp><rt>후</rt><rp>)</rp></ruby> <a href="/index.php?title=%E8%81%AF%E9%82%A6%E6%94%BF%E5%BA%9C&amp;action=edit&amp;redlink=1" class="new" title="聯邦政府 (없는 文書)"><ruby class="hanja api grade30 unknown"><rb>聯邦</rb><rp>(</rp><rt>연방</rt><rp>)</rp></ruby><ruby class="hanja api grade42 unknown"><rb>政府</rb><rp>(</rp><rt>정부</rt><rp>)</rp></ruby></a>에 <ruby class="hanja api grade40"><rb>依</rb><rp>(</rp><rt>의</rt><rp>)</rp></ruby>해 기꺼이 받아들여졌다.
</p><p>이 노래는 <ruby class="hanja api grade62"><rb>第</rb><rp>(</rp><rt>제</rt><rp>)</rp></ruby>2<ruby class="hanja api grade42"><rb>次</rb><rp>(</rp><rt>차</rt><rp>)</rp></ruby> <ruby class="hanja api grade62"><rb>世界</rb><rp>(</rp><rt>세계</rt><rp>)</rp></ruby> <ruby class="hanja api grade62"><rb>大戰</rb><rp>(</rp><rt>대전</rt><rp>)</rp></ruby>이 끝나고 <ruby class="hanja api grade52"><rb>約</rb><rp>(</rp><rt>약</rt><rp>)</rp></ruby> 10<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> <ruby class="hanja api grade72"><rb>後</rb><rp>(</rp><rt>후</rt><rp>)</rp></ruby>에 다시 <ruby class="hanja api grade72"><rb>人氣</rb><rp>(</rp><rt>인기</rt><rp>)</rp></ruby>를 얻었다. <a href="/wiki/%E7%8D%A8%E9%80%B8" title="獨逸"><ruby class="hanja api grade32"><rb>獨逸</rb><rp>(</rp><rt>독일</rt><rp>)</rp></ruby></a>, <a href="/wiki/%E5%A2%BA%E5%9C%B0%E5%88%A9" title="墺地利"><ruby class="hanja api grade12"><rb>墺地利</rb><rp>(</rp><rt>오지리</rt><rp>)</rp></ruby></a>, <a href="/wiki/%E7%91%9E%E8%A5%BF" title="瑞西"><ruby class="hanja api grade20"><rb>瑞西</rb><rp>(</rp><rt>서서</rt><rp>)</rp></ruby></a>뿐만 아니라 <a href="/wiki/%E6%AD%90%E6%B4%B2" class="mw-redirect" title="歐洲"><ruby class="hanja api grade20"><rb>歐洲</rb><rp>(</rp><rt>구주</rt><rp>)</rp></ruby></a>과 <ruby class="hanja api grade72"><rb>海外</rb><rp>(</rp><rt>해외</rt><rp>)</rp></ruby>의 다른 많은 <ruby class="hanja api grade72"><rb>國家</rb><rp>(</rp><rt>국가</rt><rp>)</rp></ruby>에서도 잘 알려져 있습니다. 이 노래는 <ruby class="hanja api grade42"><rb>如前</rb><rp>(</rp><rt>여전</rt><rp>)</rp></ruby>히 <ruby class="hanja api grade32"><rb>獨逸</rb><rp>(</rp><rt>독일</rt><rp>)</rp></ruby> <ruby class="hanja api grade42"><rb>民俗</rb><rp>(</rp><rt>민속</rt><rp>)</rp></ruby>과 <ruby class="hanja api grade70"><rb>歌手</rb><rp>(</rp><rt>가수</rt><rp>)</rp></ruby>들이 부르고 있습니다.
</p>
