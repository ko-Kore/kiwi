<h2><span id="槪要(개요)"></span><span class="mw-headline" id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"><ruby class="hanja grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EB%AA%A8%EC%97%90_%E6%93%AC%E4%BA%BA%E5%8C%96&amp;action=edit&amp;section=1" title="부분 편집: 槪要(개요)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>모에 <ruby class="hanja grade10"><rb>擬人化</rb><rp>(</rp><rt>의인화</rt><rp>)</rp></ruby>란 <a href="/wiki/%E6%93%AC%E4%BA%BA%E5%8C%96" title="擬人化"><ruby class="hanja grade10"><rb>擬人化</rb><rp>(</rp><rt>의인화</rt><rp>)</rp></ruby></a>를 모에<ruby class="hanja grade42"><rb>要素</rb><rp>(</rp><rt>요소</rt><rp>)</rp></ruby>로서 <ruby class="hanja grade40"><rb>適用</rb><rp>(</rp><rt>적용</rt><rp>)</rp></ruby>한 것을 말한다.
</p>
<h2><span id="모에化(화)와의_差異點(차이점)"></span><span class="mw-headline" id=".EB.AA.A8.EC.97.90.E5.8C.96.28.ED.99.94.29.EC.99.80.EC.9D.98_.E5.B7.AE.E7.95.B0.E9.BB.9E.28.EC.B0.A8.EC.9D.B4.EC.A0.90.29">모에<ruby class="hanja grade52"><rb>化</rb><rp>(</rp><rt>화</rt><rp>)</rp></ruby>와의 <ruby class="hanja grade40"><rb>差異點</rb><rp>(</rp><rt>차이점</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EB%AA%A8%EC%97%90_%E6%93%AC%E4%BA%BA%E5%8C%96&amp;action=edit&amp;section=2" title="부분 편집: 모에化(화)와의 差異點(차이점)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/index.php?title=%EB%AA%A8%EC%97%90%E5%8C%96&amp;action=edit&amp;redlink=1" class="new" title="모에化 (없는 문서)">모에<ruby class="hanja grade52 unknown"><rb>化</rb><rp>(</rp><rt>화</rt><rp>)</rp></ruby></a>라는 <a href="/index.php?title=%E5%96%AE%E8%AA%9E&amp;action=edit&amp;redlink=1" class="new" title="單語 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>單語</rb><rp>(</rp><rt>단어</rt><rp>)</rp></ruby></a>는 <a href="/wiki/%E9%9F%93%E5%9C%8B" title="韓國"><ruby class="hanja grade80"><rb>韓國</rb><rp>(</rp><rt>한국</rt><rp>)</rp></ruby></a>에서만 <a href="/wiki/%E4%BD%BF%E7%94%A8" title="使用"><ruby class="hanja grade60"><rb>使用</rb><rp>(</rp><rt>사용</rt><rp>)</rp></ruby></a>되며 <a href="/wiki/%E6%97%A5%E6%9C%AC" title="日本"><ruby class="hanja grade60"><rb>日本</rb><rp>(</rp><rt>일본</rt><rp>)</rp></ruby></a>에서는 <a class="mw-selflink selflink">모에 擬人化</a>라고도한다.
또한 <ruby class="hanja grade10"><rb>擬人化</rb><rp>(</rp><rt>의인화</rt><rp>)</rp></ruby>는 <a href="/index.php?title=%EC%82%AC%EB%9E%8C&amp;action=edit&amp;redlink=1" class="new" title="사람 (없는 문서)">사람</a>이 아닌 것을 <a href="/index.php?title=%E4%BA%BA%E9%96%93%E5%8C%96&amp;action=edit&amp;redlink=1" class="new" title="人間化 (없는 문서)"><ruby class="hanja grade52 unknown"><rb>人間化</rb><rp>(</rp><rt>인간화</rt><rp>)</rp></ruby></a> 하는 것이지만 <a href="/index.php?title=%EB%AA%A8%EC%97%90%E5%8C%96&amp;action=edit&amp;redlink=1" class="new" title="모에化 (없는 문서)">모에<ruby class="hanja grade52 unknown"><rb>化</rb><rp>(</rp><rt>화</rt><rp>)</rp></ruby></a>는 사람이 아닌 것은 <a href="/index.php?title=%E5%8B%BF%E8%AB%96&amp;action=edit&amp;redlink=1" class="new" title="勿論 (없는 문서)"><ruby class="hanja grade32 unknown"><rb>勿論</rb><rp>(</rp><rt>물론</rt><rp>)</rp></ruby></a> 사람에 <a href="/index.php?title=%EB%AA%A8%EC%97%90%E8%A6%81%E7%B4%A0&amp;action=edit&amp;redlink=1" class="new" title="모에要素 (없는 문서)">모에<ruby class="hanja grade42 unknown"><rb>要素</rb><rp>(</rp><rt>요소</rt><rp>)</rp></ruby></a>를 <a href="/index.php?title=%E8%BF%BD%E5%8A%A0&amp;action=edit&amp;redlink=1" class="new" title="追加 (없는 문서)"><ruby class="hanja grade32 unknown"><rb>追加</rb><rp>(</rp><rt>추가</rt><rp>)</rp></ruby></a>하는 것도 <a href="/index.php?title=%E6%8C%87%E7%A8%B1&amp;action=edit&amp;redlink=1" class="new" title="指稱 (없는 문서)"><ruby class="hanja grade40 unknown"><rb>指稱</rb><rp>(</rp><rt>지칭</rt><rp>)</rp></ruby></a>한다.
</p>
