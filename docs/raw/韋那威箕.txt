<h2><span id="槪要(개요)"></span><span class="mw-headline" id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"><ruby class="hanja grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E9%9F%8B%E9%82%A3%E5%A8%81%E7%AE%95&amp;action=edit&amp;section=1" title="부분 편집: 槪要(개요)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja grade12"><rb>韋那威箕</rb><rp>(</rp><rt>위나위기</rt><rp>)</rp></ruby>(Vi-na Uy-ki, 비나위키)는 <a href="/wiki/%E5%A7%94%E7%8F%AD%E5%BE%A9%E7%94%9F%E6%BC%A2%E5%96%83%E8%B6%8A%E5%8D%97" title="委班復生漢喃越南"><ruby class="hanja grade0"><rb>委班復生漢喃越南</rb><rp>(</rp><rt>위반복생한남월남</rt><rp>)</rp></ruby></a>에서 <ruby class="hanja grade40"><rb>運營</rb><rp>(</rp><rt>운영</rt><rp>)</rp></ruby>하는 <a href="/wiki/%E8%B6%8A%E5%8D%97%E8%AA%9E" title="越南語"><ruby class="hanja grade32"><rb>越南語</rb><rp>(</rp><rt>월남어</rt><rp>)</rp></ruby></a> <a href="/wiki/%EC%9C%84%ED%82%A4%EC%9C%84%ED%82%A4" title="위키위키">위키위키</a> 사이트이다. <ruby class="hanja grade32"><rb>越南語</rb><rp>(</rp><rt>월남어</rt><rp>)</rp></ruby>를 <a href="/wiki/%E6%BC%A2%E5%AD%97" title="漢字"><ruby class="hanja grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby></a>와 <a href="/wiki/%E5%96%83%E5%AD%97" class="mw-redirect" title="喃字"><ruby class="hanja grade0"><rb>喃字</rb><rp>(</rp><rt>남자</rt><rp>)</rp></ruby></a>로 <ruby class="hanja grade62"><rb>表記</rb><rp>(</rp><rt>표기</rt><rp>)</rp></ruby>하며, 이를 <ruby class="hanja grade42"><rb>爲</rb><rp>(</rp><rt>위</rt><rp>)</rp></ruby>한 <a href="/index.php?title=%E6%BC%A2%E5%96%83&amp;action=edit&amp;redlink=1" class="new" title="漢喃 (없는 문서)"><ruby class="hanja grade0 unknown"><rb>漢喃</rb><rp>(</rp><rt>한남</rt><rp>)</rp></ruby></a>의 <ruby class="hanja grade40"><rb>標準化</rb><rp>(</rp><rt>표준화</rt><rp>)</rp></ruby>도 <ruby class="hanja grade40"><rb>推進</rb><rp>(</rp><rt>추진</rt><rp>)</rp></ruby>하고 있다.
</p>
