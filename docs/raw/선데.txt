<html><head></head><body><div class="mw-parser-output"><h2><span id="槪要(개요)"></span><span class="mw-headline" id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"><ruby class="hanja grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EC%84%A0%EB%8D%B0&amp;action=edit&amp;section=1" title="부분 편집: 槪要(개요)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>선데(sundae)는 <ruby class="hanja grade60"><rb>美國</rb><rp>(</rp><rt>미국</rt><rp>)</rp></ruby>에서 태어난 <a href="/index.php?title=%EC%95%84%EC%9D%B4%EC%8A%A4%ED%81%AC%EB%A6%BC&amp;action=edit&amp;redlink=1" class="new" title="아이스크림 (없는 문서)">아이스크림</a>을 <ruby class="hanja grade62"><rb>主體</rb><rp>(</rp><rt>주체</rt><rp>)</rp></ruby>로 한 <a href="/index.php?title=%EB%94%94%EC%A0%80%ED%8A%B8&amp;action=edit&amp;redlink=1" class="new" title="디저트 (없는 문서)">디저트</a>이다. 한 컵 <ruby class="hanja grade42"><rb>程度</rb><rp>(</rp><rt>정도</rt><rp>)</rp></ruby>의 아이스크림 위에 <a href="/index.php?title=%EC%8B%9C%EB%9F%BD&amp;action=edit&amp;redlink=1" class="new" title="시럽 (없는 문서)">시럽</a> <ruby class="hanja grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby>을 뿌려먹는 <a href="/index.php?title=%E6%B0%B7%E8%8F%93&amp;action=edit&amp;redlink=1" class="new" title="氷菓 (없는 문서)"><ruby class="hanja grade20 unknown"><rb>氷菓</rb><rp>(</rp><rt>빙과</rt><rp>)</rp></ruby></a>의 한 <ruby class="hanja grade52"><rb>種類</rb><rp>(</rp><rt>종류</rt><rp>)</rp></ruby>로, <ruby class="hanja grade40"><rb>趣向</rb><rp>(</rp><rt>취향</rt><rp>)</rp></ruby>에 따라 <a href="/index.php?title=%EB%95%85%EC%BD%A9&amp;action=edit&amp;redlink=1" class="new" title="땅콩 (없는 문서)">땅콩</a>가루나 <a href="/index.php?title=%EB%A7%88%EB%9D%BC%EC%8A%A4%ED%82%A4%EB%85%B8_%EC%B2%B4%EB%A6%AC&amp;action=edit&amp;redlink=1" class="new" title="마라스키노 체리 (없는 문서)">마라스키노 체리</a>, <a href="/index.php?title=%ED%9C%A9%ED%8A%B8_%ED%81%AC%EB%A6%BC&amp;action=edit&amp;redlink=1" class="new" title="휩트 크림 (없는 문서)">휩트 크림</a> <ruby class="hanja grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby>을 <ruby class="hanja grade32"><rb>追加</rb><rp>(</rp><rt>추가</rt><rp>)</rp></ruby>한다.
</p>

<p><ruby class="hanja grade70"><rb>中國語</rb><rp>(</rp><rt>중국어</rt><rp>)</rp></ruby>에서 sundae은 “<ruby class="hanja grade42"><rb>聖代</rb><rp>(</rp><rt>성대</rt><rp>)</rp></ruby>”<s><ruby class="hanja grade42"><rb>聖世</rb><rp>(</rp><rt>성세</rt><rp>)</rp></ruby></s>으로 <ruby class="hanja grade32"><rb>音譯</rb><rp>(</rp><rt>음역</rt><rp>)</rp></ruby>됩니다.
