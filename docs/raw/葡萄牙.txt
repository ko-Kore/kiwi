<p><b><ruby class="hanja api grade10"><rb>葡萄牙</rb><rp>(</rp><rt>포도아</rt><rp>)</rp></ruby></b>의 <ruby class="hanja api grade60"><rb>正式</rb><rp>(</rp><rt>정식</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>名稱</rb><rp>(</rp><rt>명칭</rt><rp>)</rp></ruby>은 <ruby class="hanja api grade10"><rb>葡萄牙</rb><rp>(</rp><rt>포도아</rt><rp>)</rp></ruby><ruby class="hanja api grade62"><rb>共和國</rb><rp>(</rp><rt>공화국</rt><rp>)</rp></ruby>으로 유럽 <ruby class="hanja api grade62"><rb>南部</rb><rp>(</rp><rt>남부</rt><rp>)</rp></ruby>의 한 나라로 1143<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade42"><rb>設立</rb><rp>(</rp><rt>설립</rt><rp>)</rp></ruby>되었으며 <ruby class="hanja api grade40"><rb>總面積</rb><rp>(</rp><rt>총면적</rt><rp>)</rp></ruby>은 92,212<ruby class="hanja api grade72"><rb>平方</rb><rp>(</rp><rt>평방</rt><rp>)</rp></ruby><ruby class="hanja api grade0"><rb>粁</rb><rp>(</rp><rt>천</rt><rp>)</rp></ruby>이다. <ruby class="hanja api grade52"><rb>大陸</rb><rp>(</rp><rt>대륙</rt><rp>)</rp></ruby>은 <a href="/wiki/%E4%BC%8A%E6%AF%94%E5%88%A9%E4%BA%9E%E5%8D%8A%E5%B3%B6" title="伊比利亞半島"><ruby class="hanja api grade12"><rb>伊</rb><rp>(</rp><rt>이</rt><rp>)</rp></ruby><ruby class="hanja api grade50"><rb>比</rb><rp>(</rp><rt>비</rt><rp>)</rp></ruby><ruby class="hanja api grade62"><rb>利</rb><rp>(</rp><rt>리</rt><rp>)</rp></ruby><ruby class="hanja api grade32"><rb>亞</rb><rp>(</rp><rt>아</rt><rp>)</rp></ruby><ruby class="hanja api grade50"><rb>半島</rb><rp>(</rp><rt>반도</rt><rp>)</rp></ruby></a>의 <ruby class="hanja api grade80"><rb>南西</rb><rp>(</rp><rt>남서</rt><rp>)</rp></ruby>쪽에 <ruby class="hanja api grade42"><rb>位置</rb><rp>(</rp><rt>위치</rt><rp>)</rp></ruby>하며 <ruby class="hanja api <h2><span id=".E5.9C.8B.E5.AE.B6.28.EA.B5.AD.EA.B0.80.29.E8.B1.A1.E5.BE.B5.28.EC.83.81.EC.A7.95.29"></span><span class="mw-headline" id="國家(국가)象徵(상징)"><ruby class="hanja api grade72"><rb>國家</rb><rp>(</rp><rt>국가</rt><rp>)</rp></ruby><ruby class="hanja api grade32"><rb>象徵</rb><rp>(</rp><rt>상징</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E8%91%A1%E8%90%84%E7%89%99&amp;action=edit&amp;section=3" title="部分 編輯: 國家(국가)象徵(상징)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span id=".E5.9C.8B.E6.97.97.28.EA.B5.AD.EA.B8.B0.29"></span><span class="mw-headline" id="國旗(국기)"><ruby class="hanja api grade70"><rb>國旗</rb><rp>(</rp><rt>국기</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E8%91%A1%E8%90%84%E7%89%99&amp;action=edit&amp;section=4" title="部分 編輯: 國旗(국기)">編輯</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><ruby class="hanja api grade62"><rb>共和國</rb><rp>(</rp><rt>공화국</rt><rp>)</rp></ruby>이 <ruby class="hanja api grade60"><rb>樹立</rb><rp>(</rp><rt>수립</rt><rp>)</rp></ruby>된 <ruby class="hanja api grade72"><rb>後</rb><rp>(</rp><rt>후</rt><rp>)</rp></ruby> 1911<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> 6<ruby class="hanja api grade80"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby> 19<ruby class="hanja api grade80"><rb>日</rb><rp>(</rp><rt>일</rt><rp>)</rp></ruby> <ruby class="hanja api grade72"><rb>全國</rb><rp>(</rp><rt>전국</rt><rp>)</rp></ruby><ruby class="hanja api grade40"><rb>制憲</rb><rp>(</rp><rt>제헌</rt><rp>)</rp></ruby><ruby class="hanja api grade42"><rb>會議</rb><rp>(</rp><rt>회의</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade50"><rb>法令</rb><rp>(</rp><rt>법령</rt><rp>)</rp></ruby>으로 새로운 <ruby class="hanja api grade70"><rb>國旗</rb><rp>(</rp><rt>국기</rt><rp>)</rp></ruby>가 <ruby class="hanja api grade42"><rb>承認</rb><rp>(</rp><rt>승인</rt><rp>)</rp></ruby>되었다.
</p><p><ruby class="hanja api grade70"><rb>國旗</rb><rp>(</rp><rt>국기</rt><rp>)</rp></ruby>는 <ruby class="hanja api grade32"><rb>垂直</rb><rp>(</rp><rt>수직</rt><rp>)</rp></ruby>으로 두 가지 <ruby class="hanja api grade70"><rb>色</rb><rp>(</rp><rt>색</rt><rp>)</rp></ruby>으로 나뉜다 - 짙은 <ruby class="hanja api grade60"><rb>綠色</rb><rp>(</rp><rt>녹색</rt><rp>)</rp></ruby>과 빨간<ruby class="hanja api grade70"><rb>色</rb><rp>(</rp><rt>색</rt><rp>)</rp></ruby> - <ruby class="hanja api grade60"><rb>綠色</rb><rp>(</rp><rt>녹색</rt><rp>)</rp></ruby>은 <ruby class="hanja api grade70"><rb>旗</rb><rp>(</rp><rt>기</rt><rp>)</rp></ruby>대 쪽에 있다. <ruby class="hanja api grade32"><rb>中央</rb><rp>(</rp><rt>중앙</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade70"><rb>色</rb><rp>(</rp><rt>색</rt><rp>)</rp></ruby>깔의 <ruby class="hanja api grade40"><rb>警戒</rb><rp>(</rp><rt>경계</rt><rp>)</rp></ruby>에 겹쳐져 있는 것은 흰<ruby class="hanja api grade70"><rb>色</rb><rp>(</rp><rt>색</rt><rp>)</rp></ruby>으로 <ruby class="hanja api grade32"><rb>裝飾</rb><rp>(</rp><rt>장식</rt><rp>)</rp></ruby>된 <ruby class="hanja api grade52"><rb>局長</rb><rp>(</rp><rt>국장</rt><rp>)</rp></ruby>이며, 노란<ruby class="hanja api grade70"><rb>色</rb><rp>(</rp><rt>색</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade20"><rb>勳</rb><rp>(</rp><rt>훈</rt><rp>)</rp></ruby><ruby class="hanja api grade62"><rb>天意</rb><rp>(</rp><rt>천의</rt><rp>)</rp></ruby>에는 검은<ruby class="hanja api grade70"><rb>色</rb><rp>(</rp><rt>색</rt><rp>)</rp></ruby>으로 <ruby class="hanja api grade32"><rb>突出</rb><rp>(</rp><rt>돌출</rt><rp>)</rp></ruby>되어 있다.
</p><p><ruby class="hanja api grade0"><rb>旗발</rb><rp>(</rp><rt>깃발</rt><rp>)</rp></ruby>의 길이는 <ruby class="hanja api grade70"><rb>旗</rb><rp>(</rp><rt>기</rt><rp>)</rp></ruby>대 높이의 1.5<ruby class="hanja api grade50"><rb>倍</rb><rp>(</rp><rt>배</rt><rp>)</rp></ruby>이다. <ruby class="hanja api grade62"><rb>全體</rb><rp>(</rp><rt>전체</rt><rp>)</rp></ruby> 길이의 5<ruby class="hanja api grade62"><rb>分</rb><rp>(</rp><rt>분</rt><rp>)</rp></ruby>의 2는 <ruby class="hanja api grade60"><rb>綠色</rb><rp>(</rp><rt>녹색</rt><rp>)</rp></ruby>이고 나머지 5<ruby class="hanja api grade62"><rb>分</rb><rp>(</rp><rt>분</rt><rp>)</rp></ruby>의 3은 빨간<ruby class="hanja api grade70"><rb>色</rb><rp>(</rp><rt>색</rt><rp>)</rp></ruby>이다.
</p>
<h3><span id=".E5.9C.8B.E7.AB.A0.28.EA.B5.AD.EC.9E.A5.29"></span><span class="mw-headline" id="國章(국장)"><ruby class="hanja api grade60"><rb>國章</rb><rp>(</rp><rt>국장</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E8%91%A1%E8%90%84%E7%89%99&amp;action=edit&amp;section=5" title="部分 編輯: 國章(국장)">編輯</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><ruby class="hanja api grade60"><rb>國章</rb><rp>(</rp><rt>국장</rt><rp>)</rp></ruby>은 높이의 <ruby class="hanja api grade40"><rb>折半</rb><rp>(</rp><rt>절반</rt><rp>)</rp></ruby>을 차지하며 <ruby class="hanja api grade72"><rb>上下</rb><rp>(</rp><rt>상하</rt><rp>)</rp></ruby> 테두리와 같은 거리에 <ruby class="hanja api grade42"><rb>配置</rb><rp>(</rp><rt>배치</rt><rp>)</rp></ruby>되어 있다.
