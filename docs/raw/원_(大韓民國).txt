<h2><span id="槪說(개설)"></span><span class="mw-headline" id=".E6.A7.AA.E8.AA.AA.28.EA.B0.9C.EC.84.A4.29"><ruby class="hanja grade32"><rb>槪說</rb><rp>(</rp><rt>개설</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EC%9B%90_(%E5%A4%A7%E9%9F%93%E6%B0%91%E5%9C%8B)&amp;action=edit&amp;section=1" title="부분 편집: 槪說(개설)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>원은 <a href="/wiki/%E5%A4%A7%E9%9F%93%E6%B0%91%E5%9C%8B" title="大韓民國"><ruby class="hanja grade80"><rb>大韓民國</rb><rp>(</rp><rt>대한민국</rt><rp>)</rp></ruby></a>의<ruby class="hanja grade42"><rb>通貨</rb><rp>(</rp><rt>통화</rt><rp>)</rp></ruby>이다. <a href="/index.php?title=1962%E5%B9%B4&amp;action=edit&amp;redlink=1" class="new" title="1962年 (없는 문서)">1962<ruby class="hanja grade80 unknown"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby></a> <a href="/index.php?title=%E7%B7%8A%E6%80%A5%E9%80%9A%E8%B2%A8%E6%8E%AA%E7%BD%AE%E6%B3%95&amp;action=edit&amp;redlink=1" class="new" title="緊急通貨措置法 (없는 문서)"><ruby class="hanja grade20 unknown"><rb>緊急通貨措置法</rb><rp>(</rp><rt>긴급통화조치법</rt><rp>)</rp></ruby></a>에따라 <ruby class="hanja grade30"><rb>旣存</rb><rp>(</rp><rt>기존</rt><rp>)</rp></ruby>의 <ruby class="hanja grade0"><rb><span class="noruby"><a href="/index.php?title=%E5%9C%9C&amp;action=edit&amp;redlink=1" class="new" title="圜 (없는 문서)"><ruby class="hanja grade0 unknown"><rb>圜</rb><rp>(</rp><rt>원</rt><rp>)</rp></ruby></a></span></rb><rp>(</rp><rt>환</rt><rp>)</rp></ruby>을 <ruby class="hanja grade30"><rb>代替</rb><rp>(</rp><rt>대체</rt><rp>)</rp></ruby>할<ruby class="hanja grade42"><rb>通貨</rb><rp>(</rp><rt>통화</rt><rp>)</rp></ruby>로서 <ruby class="hanja grade42"><rb>導入</rb><rp>(</rp><rt>도입</rt><rp>)</rp></ruby>되었다. ISO4217에<ruby class="hanja grade40"><rb>依</rb><rp>(</rp><rt>의</rt><rp>)</rp></ruby>해 <ruby class="hanja grade42"><rb>配定</rb><rp>(</rp><rt>배정</rt><rp>)</rp></ruby>된 <ruby class="hanja grade42"><rb>通貨</rb><rp>(</rp><rt>통화</rt><rp>)</rp></ruby>코드는 KRW이며, <ruby class="hanja grade72"><rb>韓國內</rb><rp>(</rp><rt>한국내</rt><rp>)</rp></ruby>에서는 <ruby class="hanja grade70"><rb>主</rb><rp>(</rp><rt>주</rt><rp>)</rp></ruby>로 <ruby class="hanja grade70"><rb>文字</rb><rp>(</rp><rt>문자</rt><rp>)</rp></ruby>인원 <ruby class="hanja grade40"><rb>或</rb><rp>(</rp><rt>혹</rt><rp>)</rp></ruby>은 <ruby class="hanja grade60"><rb>記號</rb><rp>(</rp><rt>기호</rt><rp>)</rp></ruby>인￦을 <ruby class="hanja grade60"><rb>使用</rb><rp>(</rp><rt>사용</rt><rp>)</rp></ruby>해 <ruby class="hanja grade62"><rb>表記</rb><rp>(</rp><rt>표기</rt><rp>)</rp></ruby>한다. <a href="/index.php?title=2016%E5%B9%B4&amp;action=edit&amp;redlink=1" class="new" title="2016年 (없는 문서)">2016<ruby class="hanja grade80 unknown"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby></a><ruby class="hanja grade42"><rb>基準</rb><rp>(</rp><rt>기준</rt><rp>)</rp></ruby> <ruby class="hanja grade10"><rb>通貨膨脹率</rb><rp>(</rp><rt>통화팽창솔</rt><rp>)</rp></ruby>은 1.3％.
</p>
<h2><span id="發行(발행)"></span><span class="mw-headline" id=".E7.99.BC.E8.A1.8C.28.EB.B0.9C.ED.96.89.29"><ruby class="hanja grade60"><rb>發行</rb><rp>(</rp><rt>발행</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EC%9B%90_(%E5%A4%A7%E9%9F%93%E6%B0%91%E5%9C%8B)&amp;action=edit&amp;section=2" title="부분 편집: 發行(발행)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja grade42"><rb>發行處</rb><rp>(</rp><rt>발행처</rt><rp>)</rp></ruby>는 <a href="/wiki/%E5%A4%A7%E9%9F%93%E6%B0%91%E5%9C%8B" title="大韓民國"><ruby class="hanja grade80"><rb>大韓民國</rb><rp>(</rp><rt>대한민국</rt><rp>)</rp></ruby></a>의<ruby class="hanja grade32"><rb>中央銀行</rb><rp>(</rp><rt>중앙은행</rt><rp>)</rp></ruby>인 <a href="/index.php?title=%E9%9F%93%E5%9C%8B%E9%8A%80%E8%A1%8C&amp;action=edit&amp;redlink=1" class="new" title="韓國銀行 (없는 문서)"><ruby class="hanja grade60 unknown"><rb>韓國銀行</rb><rp>(</rp><rt>한국은행</rt><rp>)</rp></ruby></a>이며, <ruby class="hanja grade42"><rb>製造</rb><rp>(</rp><rt>제조</rt><rp>)</rp></ruby>는 <a href="/index.php?title=%E9%9F%93%E5%9C%8B%E9%80%A0%E5%B9%A3%E5%85%AC%E7%A4%BE&amp;action=edit&amp;redlink=1" class="new" title="韓國造幣公社 (없는 문서)"><ruby class="hanja grade30 unknown"><rb>韓國造幣公社</rb><rp>(</rp><rt>한국조폐공사</rt><rp>)</rp></ruby></a>가 <ruby class="hanja grade60"><rb>行</rb><rp>(</rp><rt>행</rt><rp>)</rp></ruby>하고있다.
</p>
