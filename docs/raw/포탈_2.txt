<h2><span id="槪要(개요)"></span><span class="mw-headline" id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"><ruby class="hanja grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%ED%8F%AC%ED%83%88_2&amp;action=edit&amp;section=1" title="부분 편집: 槪要(개요)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>포탈 2는 <a href="/index.php?title=2011%E5%B9%B4&amp;action=edit&amp;redlink=1" class="new" title="2011年 (없는 문서)">2011<ruby class="hanja grade80 unknown"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby></a> <ruby class="hanja grade50"><rb>發賣</rb><rp>(</rp><rt>발매</rt><rp>)</rp></ruby>된 <a href="/index.php?title=%EB%B0%B8%EB%B8%8C_%EC%BD%94%ED%8D%BC%EB%A0%88%EC%9D%B4%EC%85%98&amp;action=edit&amp;redlink=1" class="new" title="밸브 코퍼레이션 (없는 문서)">밸브<ruby class="hanja grade62 unknown"><rb>社</rb><rp>(</rp><rt>사</rt><rp>)</rp></ruby></a>의 게임으로, <a href="/index.php?title=%ED%8F%AC%ED%83%88_(%EA%B2%8C%EC%9E%84)&amp;action=edit&amp;redlink=1" class="new" title="포탈 (게임) (없는 문서)">포탈</a>의 <ruby class="hanja grade42"><rb>後續作</rb><rp>(</rp><rt>후속작</rt><rp>)</rp></ruby>이다.
</p>
<h2><span id="줄거리"></span><span class="mw-headline" id=".EC.A4.84.EA.B1.B0.EB.A6.AC">줄거리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%ED%8F%AC%ED%83%88_2&amp;action=edit&amp;section=2" title="부분 편집: 줄거리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/index.php?title=%ED%8F%AC%ED%83%88_(%EA%B2%8C%EC%9E%84)&amp;action=edit&amp;redlink=1" class="new" title="포탈 (게임) (없는 문서)">포탈</a>로부터 먼 <ruby class="hanja grade42"><rb>未來</rb><rp>(</rp><rt>미래</rt><rp>)</rp></ruby>의 이야기를 그린다. 포탈의 엔딩 <ruby class="hanja grade70"><rb>場面</rb><rp>(</rp><rt>장면</rt><rp>)</rp></ruby> <ruby class="hanja grade72"><rb>後</rb><rp>(</rp><rt>후</rt><rp>)</rp></ruby>로 긴 <ruby class="hanja grade72"><rb>時間</rb><rp>(</rp><rt>시간</rt><rp>)</rp></ruby>동안 <ruby class="hanja grade32"><rb>冬眠狀態</rb><rp>(</rp><rt>동면상태</rt><rp>)</rp></ruby>에 있던 <a href="/index.php?title=%EC%B2%BC&amp;action=edit&amp;redlink=1" class="new" title="첼 (없는 문서)">첼</a>을 <a href="/index.php?title=%ED%9C%98%ED%8B%80%EB%A6%AC&amp;action=edit&amp;redlink=1" class="new" title="휘틀리 (없는 문서)">휘틀리</a>가 깨워, <a href="/index.php?title=%EC%95%A0%ED%8D%BC%EC%B3%90_%EC%82%AC%EC%9D%B4%EC%96%B8%EC%8A%A4&amp;action=edit&amp;redlink=1" class="new" title="애퍼쳐 사이언스 (없는 문서)">애퍼쳐 사이언스</a>로부터 <ruby class="hanja grade40"><rb>脫出</rb><rp>(</rp><rt>탈출</rt><rp>)</rp></ruby>하려고 하지만, <a href="/index.php?title=GLaDOS&amp;action=edit&amp;redlink=1" class="new" title="GLaDOS (없는 문서)">GLaDOS</a>의 <ruby class="hanja grade42"><rb>房</rb><rp>(</rp><rt>방</rt><rp>)</rp></ruby>을 지나려다 <del>멍청한 휘틀리가</del> GLaDOS를 깨우고 말아 다시 <ruby class="hanja grade42"><rb>實驗</rb><rp>(</rp><rt>실험</rt><rp>)</rp></ruby>을 <ruby class="hanja grade42"><rb>進行</rb><rp>(</rp><rt>진행</rt><rp>)</rp></ruby>하게 된다.
</p>
<h2><span id="게임_플레이"></span><span class="mw-headline" id=".EA.B2.8C.EC.9E.84_.ED.94.8C.EB.A0.88.EC.9D.B4">게임 플레이</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%ED%8F%AC%ED%83%88_2&amp;action=edit&amp;section=3" title="부분 편집: 게임 플레이">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span id="實驗用(실험용)_裝置(장치)"></span><span class="mw-headline" id=".E5.AF.A6.E9.A9.97.E7.94.A8.28.EC.8B.A4.ED.97.98.EC.9A.A9.29_.E8.A3.9D.E7.BD.AE.28.EC.9E.A5.EC.B9.98.29"><ruby class="hanja grade42"><rb>實驗用</rb><rp>(</rp><rt>실험용</rt><rp>)</rp></ruby> <ruby class="hanja grade40"><rb>裝置</rb><rp>(</rp><rt>장치</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%ED%8F%AC%ED%83%88_2&amp;action=edit&amp;section=4" title="부분 편집: 實驗用(실험용) 裝置(장치)">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul><li><ruby class="hanja grade50"><rb>高熱</rb><rp>(</rp><rt>고열</rt><rp>)</rp></ruby> <ruby class="hanja grade10"><rb>挫折</rb><rp>(</rp><rt>좌절</rt><rp>)</rp></ruby> 빔&nbsp;: <ruby class="hanja grade62"><rb>全作</rb><rp>(</rp><rt>전작</rt><rp>)</rp></ruby>의 ‘에너지 볼’이 <ruby class="hanja grade70"><rb>登場</rb><rp>(</rp><rt>등장</rt><rp>)</rp></ruby>하지 않는 <ruby class="hanja grade62"><rb>代身</rb><rp>(</rp><rt>대신</rt><rp>)</rp></ruby> <ruby class="hanja grade32"><rb>追加</rb><rp>(</rp><rt>추가</rt><rp>)</rp></ruby>되었다. ‘<ruby class="hanja grade10"><rb>挫折</rb><rp>(</rp><rt>좌절</rt><rp>)</rp></ruby> <ruby class="hanja grade42"><rb>再傳導</rb><rp>(</rp><rt>재전도</rt><rp>)</rp></ruby> 큐브’로 <ruby class="hanja grade40"><rb>屈折</rb><rp>(</rp><rt>굴절</rt><rp>)</rp></ruby>시켜 <ruby class="hanja grade0"><rb>琉璃窓</rb><rp>(</rp><rt>유리창</rt><rp>)</rp></ruby>을 <ruby class="hanja grade52"><rb>通過</rb><rp>(</rp><rt>통과</rt><rp>)</rp></ruby>시켜 터릿을 <ruby class="hanja grade32"><rb>破壞</rb><rp>(</rp><rt>파괴</rt><rp>)</rp></ruby>하거나 레이저 <ruby class="hanja grade42"><rb>感知器</rb><rp>(</rp><rt>감지기</rt><rp>)</rp></ruby>에 <ruby class="hanja grade32"><rb>誘導</rb><rp>(</rp><rt>유도</rt><rp>)</rp></ruby>시켜 <ruby class="hanja grade40"><rb>裝置</rb><rp>(</rp><rt>장치</rt><rp>)</rp></ruby>를 <ruby class="hanja grade62"><rb>作動</rb><rp>(</rp><rt>작동</rt><rp>)</rp></ruby>시킬 수 있다.</li></ul>
<ul><li>‘하드라이트 브리지&nbsp;: <ruby class="hanja grade62"><rb>光線</rb><rp>(</rp><rt>광선</rt><rp>)</rp></ruby>으로 된 <a href="/index.php?title=%EB%8B%A4%EB%A6%AC&amp;action=edit&amp;redlink=1" class="new" title="다리 (없는 문서)">다리</a>(<ruby class="hanja grade50"><rb>橋</rb><rp>(</rp><rt>교</rt><rp>)</rp></ruby>). 발로 밟고 <ruby class="hanja grade42"><rb>移動</rb><rp>(</rp><rt>이동</rt><rp>)</rp></ruby>하거나 터릿의 <ruby class="hanja grade40"><rb>攻擊</rb><rp>(</rp><rt>공격</rt><rp>)</rp></ruby>을 막을 수 있으며, <ruby class="hanja grade42"><rb>壁</rb><rp>(</rp><rt>벽</rt><rp>)</rp></ruby>에 닿는 <ruby class="hanja grade62"><rb>部分</rb><rp>(</rp><rt>부분</rt><rp>)</rp></ruby>에 포탈을 열어 다른 <ruby class="hanja grade42"><rb>位置</rb><rp>(</rp><rt>위치</rt><rp>)</rp></ruby>에서 나타나게 할 수도 있다.</li></ul>
