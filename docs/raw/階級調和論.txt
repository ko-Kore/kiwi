<h3><span id=".E4.B8.80.28.EC.9D.BC.29:.E5.90.8C.E6.84.8F.28.EB.8F.99.EC.9D.98.29.E9.A1.9E.E5.9E.8B.28.EC.9C.A0.ED.98.95.29"></span><span class="mw-headline" id="一(일):同意(동의)類型(유형)"><ruby class="hanja api grade80"><rb>一</rb><rp>(</rp><rt>일</rt><rp>)</rp></ruby>:<ruby class="hanja api grade62"><rb>同意</rb><rp>(</rp><rt>동의</rt><rp>)</rp></ruby><ruby class="hanja api grade20"><rb>類型</rb><rp>(</rp><rt>유형</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E9%9A%8E%E7%B4%9A%E8%AA%BF%E5%92%8C%E8%AB%96&amp;action=edit&amp;section=2" title="部分 編輯: 一(일):同意(동의)類型(유형)">編輯</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>1. <ruby class="hanja api grade72"><rb>國家</rb><rp>(</rp><rt>국가</rt><rp>)</rp></ruby>는 <ruby class="hanja api grade42"><rb>利益</rb><rp>(</rp><rt>이익</rt><rp>)</rp></ruby><ruby class="hanja api grade32"><rb>仲介</rb><rp>(</rp><rt>중개</rt><rp>)</rp></ruby><ruby class="hanja api grade40"><rb>組織</rb><rp>(</rp><rt>조직</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade40"><rb>存在</rb><rp>(</rp><rt>존재</rt><rp>)</rp></ruby>를 <ruby class="hanja api grade42"><rb>許容</rb><rp>(</rp><rt>허용</rt><rp>)</rp></ruby>하고, <ruby class="hanja api grade62"><rb>反對</rb><rp>(</rp><rt>반대</rt><rp>)</rp></ruby>로 그들은 <ruby class="hanja api grade72"><rb>國家</rb><rp>(</rp><rt>국가</rt><rp>)</rp></ruby>가 <ruby class="hanja api grade42"><rb>經濟</rb><rp>(</rp><rt>경제</rt><rp>)</rp></ruby><ruby class="hanja api grade62"><rb>社會</rb><rp>(</rp><rt>사회</rt><rp>)</rp></ruby><ruby class="hanja api grade42"><rb>事務</rb><rp>(</rp><rt>사무</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade62"><rb>對</rb><rp>(</rp><rt>대</rt><rp>)</rp></ruby>한 <ruby class="hanja api grade42"><rb>行動權</rb><rp>(</rp><rt>행동권</rt><rp>)</rp></ruby>을 <ruby class="hanja api grade30"><rb>享有</rb><rp>(</rp><rt>향유</rt><rp>)</rp></ruby>하도록 <ruby class="hanja api grade42"><rb>許容</rb><rp>(</rp><rt>허용</rt><rp>)</rp></ruby>한다.
</p><p>2. <ruby class="hanja api grade42"><rb>協力</rb><rp>(</rp><rt>협력</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>體系</rb><rp>(</rp><rt>체계</rt><rp>)</rp></ruby><ruby class="hanja api grade72"><rb>內</rb><rp>(</rp><rt>내</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade62"><rb>各</rb><rp>(</rp><rt>각</rt><rp>)</rp></ruby><ruby class="hanja api grade40"><rb>組織</rb><rp>(</rp><rt>조직</rt><rp>)</rp></ruby>은 <ruby class="hanja api grade32"><rb>價値</rb><rp>(</rp><rt>가치</rt><rp>)</rp></ruby>와 <ruby class="hanja api grade40"><rb>目標</rb><rp>(</rp><rt>목표</rt><rp>)</rp></ruby><ruby class="hanja api grade70"><rb>面</rb><rp>(</rp><rt>면</rt><rp>)</rp></ruby>에서 높은 <ruby class="hanja api grade42"><rb>水準</rb><rp>(</rp><rt>수준</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade32"><rb>一貫性</rb><rp>(</rp><rt>일관성</rt><rp>)</rp></ruby>을 가지고 있기 때문에 <ruby class="hanja api grade72"><rb>國家</rb><rp>(</rp><rt>국가</rt><rp>)</rp></ruby>는 <ruby class="hanja api grade42"><rb>制限</rb><rp>(</rp><rt>제한</rt><rp>)</rp></ruby>된 <ruby class="hanja api grade42"><rb>統制</rb><rp>(</rp><rt>통제</rt><rp>)</rp></ruby>만 <ruby class="hanja api grade42"><rb>施行</rb><rp>(</rp><rt>시행</rt><rp>)</rp></ruby>하면 되고, <ruby class="hanja api grade32"><rb>仲介</rb><rp>(</rp><rt>중개</rt><rp>)</rp></ruby><ruby class="hanja api grade40"><rb>組織</rb><rp>(</rp><rt>조직</rt><rp>)</rp></ruby>과 <ruby class="hanja api grade62"><rb>社會</rb><rp>(</rp><rt>사회</rt><rp>)</rp></ruby><ruby class="hanja api grade42"><rb>行爲者</rb><rp>(</rp><rt>행위자</rt><rp>)</rp></ruby>는 뚜렷한 <ruby class="hanja api grade42"><rb>自律性</rb><rp>(</rp><rt>자율성</rt><rp>)</rp></ruby>을 가지고 있다.
</p><p>3. <ruby class="hanja api grade42"><rb>協力</rb><rp>(</rp><rt>협력</rt><rp>)</rp></ruby> <ruby class="hanja api grade42"><rb>救助</rb><rp>(</rp><rt>구조</rt><rp>)</rp></ruby>는 <ruby class="hanja api grade52"><rb>正體性</rb><rp>(</rp><rt>정체성</rt><rp>)</rp></ruby>을 <ruby class="hanja api grade32"><rb>促進</rb><rp>(</rp><rt>촉진</rt><rp>)</rp></ruby>하기 위해 <ruby class="hanja api grade40"><rb>構築</rb><rp>(</rp><rt>구축</rt><rp>)</rp></ruby>되었다.
</p>
<h3><span id=".E4.BA.8C.28.EC.9D.B4.29:.E6.AC.8A.E5.A8.81.28.EA.B6.8C.EC.9C.84.29.E9.A1.9E.E5.9E.8B.28.EC.9C.A0.ED.98.95.29"></span><span class="mw-headline" id="二(이):權威(권위)類型(유형)"><ruby class="hanja api grade80"><rb>二</rb><rp>(</rp><rt>이</rt><rp>)</rp></ruby>:<ruby class="hanja api grade40"><rb>權威</rb><rp>(</rp><rt>권위</rt><rp>)</rp></ruby><ruby class="hanja api grade20"><rb>類型</rb><rp>(</rp><rt>유형</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E9%9A%8E%E7%B4%9A%E8%AA%BF%E5%92%8C%E8%AB%96&amp;action=edit&amp;section=3" title="部分 編輯: 二(이):權威(권위)類型(유형)">編輯</a><span class="mw-editsection-bracket">]</span></span></h3>
