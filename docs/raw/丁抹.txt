</p><p>그는 1949<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> <a href="/wiki/%E5%8C%97%E5%A4%A7%E8%A5%BF%E6%B4%8B%E6%A2%9D%E7%B4%84%E6%A9%9F%E6%A7%8B" title="北大西洋條約機構"><ruby class="hanja api grade40"><rb>北大西洋條約機構</rb><rp>(</rp><rt>북대서양조약기구</rt><rp>)</rp></ruby></a>에 <ruby class="hanja api grade50"><rb>加入</rb><rp>(</rp><rt>가입</rt><rp>)</rp></ruby>했고, 1973<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> <ruby class="hanja api grade10"><rb>歐羅巴</rb><rp>(</rp><rt>구라파</rt><rp>)</rp></ruby><ruby class="hanja api grade62"><rb>共同體</rb><rp>(</rp><rt>공동체</rt><rp>)</rp></ruby>(<ruby class="hanja api grade62"><rb>今</rb><rp>(</rp><rt>금</rt><rp>)</rp></ruby> <a href="/index.php?title=%E6%AD%90%E7%BE%85%E5%B7%B4_%E8%81%AF%E5%90%88&amp;action=edit&amp;redlink=1" class="new" title="歐羅巴 聯合 (없는 文書)"><ruby class="hanja api grade10 unknown"><rb>歐羅巴</rb><rp>(</rp><rt>구라파</rt><rp>)</rp></ruby> <ruby class="hanja api grade32 unknown"><rb>聯合</rb><rp>(</rp><rt>연합</rt><rp>)</rp></ruby></a>)에 <ruby class="hanja api grade50"><rb>加入</rb><rp>(</rp><rt>가입</rt><rp>)</rp></ruby>했다. <a href="/wiki/%E6%A0%BC%E9%99%B5%E8%98%AD" title="格陵蘭"><ruby class="hanja api grade52"><rb>格</rb><rp>(</rp><rt>격</rt><rp>)</rp></ruby><ruby class="hanja api grade32"><rb>陵</rb><rp>(</rp><rt>릉</rt><rp>)</rp></ruby><ruby class="hanja api grade32"><rb>蘭</rb><rp>(</rp><rt>란</rt><rp>)</rp></ruby></a>와 <a href="/wiki/%E6%B3%95%E7%BE%85%E8%AB%B8%E5%B3%B6" title="法羅諸島"><ruby class="hanja api grade52"><rb>法</rb><rp>(</rp><rt>법</rt><rp>)</rp></ruby><ruby class="hanja api grade42"><rb>羅</rb><rp>(</rp><rt>라</rt><rp>)</rp></ruby><ruby class="hanja api grade32"><rb>諸島</rb><rp>(</rp><rt>제도</rt><rp>)</rp></ruby></a>에 <ruby class="hanja api grade62"><rb>對</rb><rp>(</rp><rt>대</rt><rp>)</rp></ruby>한 <ruby class="hanja api grade42"><rb>主權</rb><rp>(</rp><rt>주권</rt><rp>)</rp></ruby>.

<h2><span id=".E6.86.B2.E6.B3.95.28.ED.97.8C.EB.B2.95.29"></span><span class="mw-headline" id="憲法(헌법)"><ruby class="hanja api grade40"><rb>憲法</rb><rp>(</rp><rt>헌법</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E4%B8%81%E6%8A%B9&amp;action=edit&amp;section=4" title="部分 編輯: 憲法(헌법)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja api grade60"><rb>現行</rb><rp>(</rp><rt>현행</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>憲法</rb><rp>(</rp><rt>헌법</rt><rp>)</rp></ruby>은 1849<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade42"><rb>制定</rb><rp>(</rp><rt>제정</rt><rp>)</rp></ruby>되어 1866<ruby class="hanja api grade42"><rb>次例</rb><rp>(</rp><rt>차례</rt><rp>)</rp></ruby> <ruby class="hanja api grade50"><rb>改正</rb><rp>(</rp><rt>개정</rt><rp>)</rp></ruby>되었다. <ruby class="hanja api grade40"><rb>憲法</rb><rp>(</rp><rt>헌법</rt><rp>)</rp></ruby>은 <ruby class="hanja api grade10"><rb>丁抹</rb><rp>(</rp><rt>정말</rt><rp>)</rp></ruby>가 <ruby class="hanja api grade40"><rb>立憲</rb><rp>(</rp><rt>입헌</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>君主國</rb><rp>(</rp><rt>군주국</rt><rp>)</rp></ruby>을 가지고 있다고 <ruby class="hanja api grade50"><rb>規定</rb><rp>(</rp><rt>규정</rt><rp>)</rp></ruby>합니다. <ruby class="hanja api grade42"><rb>議會</rb><rp>(</rp><rt>의회</rt><rp>)</rp></ruby>에서 6<ruby class="hanja api grade32"><rb>分之</rb><rp>(</rp><rt>분지</rt><rp>)</rp></ruby>5의 <ruby class="hanja api grade60"><rb>多數</rb><rp>(</rp><rt>다수</rt><rp>)</rp></ruby>를 차지한 <ruby class="hanja api grade42"><rb>政府</rb><rp>(</rp><rt>정부</rt><rp>)</rp></ruby>는 <ruby class="hanja api grade60"><rb>一定</rb><rp>(</rp><rt>일정</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>範圍</rb><rp>(</rp><rt>범위</rt><rp>)</rp></ruby> <ruby class="hanja api grade72"><rb>內</rb><rp>(</rp><rt>내</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade42"><rb>主權</rb><rp>(</rp><rt>주권</rt><rp>)</rp></ruby>을 어떤 "<ruby class="hanja api grade40"><rb>國際機構</rb><rp>(</rp><rt>국제기구</rt><rp>)</rp></ruby>"에 <ruby class="hanja api grade40"><rb>委任</rb><rp>(</rp><rt>위임</rt><rp>)</rp></ruby>할 수 있다.
</p>
