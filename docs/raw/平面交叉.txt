<p>레벨 크로싱은 <ruby class="hanja api grade52"><rb>合流</rb><rp>(</rp><rt>합류</rt><rp>)</rp></ruby>하거나 <ruby class="hanja api grade10"><rb>交叉</rb><rp>(</rp><rt>교차</rt><rp>)</rp></ruby>하는 <ruby class="hanja api grade50"><rb>鐵道線路</rb><rp>(</rp><rt>철도선로</rt><rp>)</rp></ruby>가 <ruby class="hanja api grade62"><rb>反對便</rb><rp>(</rp><rt>반대편</rt><rp>)</rp></ruby><ruby class="hanja api grade50"><rb>鐵道</rb><rp>(</rp><rt>철도</rt><rp>)</rp></ruby><ruby class="hanja api grade60"><rb>路線</rb><rp>(</rp><rt>로선</rt><rp>)</rp></ruby> 앞에서 <ruby class="hanja api grade10"><rb>交叉</rb><rp>(</rp><rt>교차</rt><rp>)</rp></ruby>하는 <ruby class="hanja api grade40"><rb>構成</rb><rp>(</rp><rt>구성</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade50"><rb>鐵道</rb><rp>(</rp><rt>철도</rt><rp>)</rp></ruby><ruby class="hanja api grade72"><rb>間</rb><rp>(</rp><rt>간</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade10"><rb>交叉點</rb><rp>(</rp><rt>교차점</rt><rp>)</rp></ruby>이다.
