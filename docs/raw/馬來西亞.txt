<p><ruby class="hanja api grade52"><rb>主要</rb><rp>(</rp><rt>주요</rt><rp>)</rp></ruby> <ruby class="hanja api grade60"><rb>石油</rb><rp>(</rp><rt>석유</rt><rp>)</rp></ruby> 가스 <ruby class="hanja api grade20"><rb>採掘</rb><rp>(</rp><rt>채굴</rt><rp>)</rp></ruby> <ruby class="hanja api grade42"><rb>爲主</rb><rp>(</rp><rt>위주</rt><rp>)</rp></ruby>로. 2021<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> BP <ruby class="hanja api grade62"><rb>世界</rb><rp>(</rp><rt>세계</rt><rp>)</rp></ruby> 에너지 <ruby class="hanja api grade42"><rb>統計</rb><rp>(</rp><rt>통계</rt><rp>)</rp></ruby> <ruby class="hanja api grade32"><rb>年鑑</rb><rp>(</rp><rt>연감</rt><rp>)</rp></ruby>에 따르면 2021<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> <ruby class="hanja api grade50"><rb>末</rb><rp>(</rp><rt>말</rt><rp>)</rp></ruby> <ruby class="hanja api grade50"><rb>原油</rb><rp>(</rp><rt>원유</rt><rp>)</rp></ruby> <ruby class="hanja api grade30"><rb>埋藏量</rb><rp>(</rp><rt>매장량</rt><rp>)</rp></ruby>은 27<ruby class="hanja api grade50"><rb>億</rb><rp>(</rp><rt>억</rt><rp>)</rp></ruby> 배럴, <ruby class="hanja api grade70"><rb>天然</rb><rp>(</rp><rt>천연</rt><rp>)</rp></ruby> 가스 <ruby class="hanja api grade30"><rb>埋藏量</rb><rp>(</rp><rt>매장량</rt><rp>)</rp></ruby>은 9000<ruby class="hanja api grade50"><rb>億</rb><rp>(</rp><rt>억</rt><rp>)</rp></ruby> 입방미터이다. 2021<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> <ruby class="hanja api grade60"><rb>石油</rb><rp>(</rp><rt>석유</rt><rp>)</rp></ruby> 일일 <ruby class="hanja api grade50"><rb>生産量</rb><rp>(</rp><rt>생산량</rt><rp>)</rp></ruby>은 73.7배럴이고 <ruby class="hanja api grade70"><rb>天然</rb><rp>(</rp><rt>천연</rt><rp>)</rp></ruby>가스 일일 <ruby class="hanja api grade50"><rb>生産量</rb><rp>(</rp><rt>생산량</rt><rp>)</rp></ruby>은 2<ruby class="hanja api grade50"><rb>億</rb><rp>(</rp><rt>억</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>標準</rb><rp>(</rp><rt>표준</rt><rp>)</rp></ruby> 입방미터이다.

<h5><span id=".E8.A7.80.E5.85.89.E6.A5.AD.28.EA.B4.80.EA.B4.91.EC.97.85.29"></span><span class="mw-headline" id="觀光業(관광업)"><ruby class="hanja api grade52"><rb>觀光業</rb><rp>(</rp><rt>관광업</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E9%A6%AC%E4%BE%86%E8%A5%BF%E4%BA%9E&amp;action=edit&amp;section=29" title="部分 編輯: 觀光業(관광업)">編輯</a><span class="mw-editsection-bracket">]</span></span></h5>
