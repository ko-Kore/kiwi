<p>1939-1940<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby>의 '<a href="/wiki/%EA%B2%A8%EC%9A%B8_%E6%88%B0%E7%88%AD" class="mw-redirect" title="겨울 戰爭">겨울 <ruby class="hanja api grade50"><rb>戰爭</rb><rp>(</rp><rt>전쟁</rt><rp>)</rp></ruby></a>'은 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" class="mw-redirect" title="러시아">러시아</a>-<a href="/wiki/%ED%95%80%EB%9E%80%EB%93%9C" class="mw-redirect" title="핀란드">핀란드</a> <ruby class="hanja api grade50"><rb>戰爭</rb><rp>(</rp><rt>전쟁</rt><rp>)</rp></ruby>으로도 알려져 있으며, 이 작은 <a href="/wiki/%ED%95%80%EB%9E%80%EB%93%9C" class="mw-redirect" title="핀란드">핀란드</a> <ruby class="hanja api grade42"><rb>軍隊</rb><rp>(</rp><rt>군대</rt><rp>)</rp></ruby>는 <a href="/wiki/%E8%98%87%E8%81%AF" title="蘇聯"><ruby class="hanja api grade32"><rb>蘇聯</rb><rp>(</rp><rt>소련</rt><rp>)</rp></ruby></a>의 <ruby class="hanja api grade40"><rb>巨大</rb><rp>(</rp><rt>거대</rt><rp>)</rp></ruby>한 붉은 <ruby class="hanja api grade42"><rb>軍隊</rb><rp>(</rp><rt>군대</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade40"><rb>威力</rb><rp>(</rp><rt>위력</rt><rp>)</rp></ruby>에 맞섰습니다.
</p><p>두 나라 사이에는 <ruby class="hanja api grade62"><rb>不信</rb><rp>(</rp><rt>불신</rt><rp>)</rp></ruby>이 있었다. 핀란드는 <ruby class="hanja api grade32"><rb>蘇聯</rb><rp>(</rp><rt>소련</rt><rp>)</rp></ruby>이 <ruby class="hanja api grade72"><rb>自國</rb><rp>(</rp><rt>자국</rt><rp>)</rp></ruby> <ruby class="hanja api grade50"><rb>領土</rb><rp>(</rp><rt>영토</rt><rp>)</rp></ruby>를 <ruby class="hanja api grade30"><rb>擴張</rb><rp>(</rp><rt>확장</rt><rp>)</rp></ruby>하려 한다고 믿었고, <ruby class="hanja api grade32"><rb>蘇聯</rb><rp>(</rp><rt>소련</rt><rp>)</rp></ruby>은 핀란드가 <ruby class="hanja api grade42"><rb>敵軍</rb><rp>(</rp><rt>적군</rt><rp>)</rp></ruby>이 <ruby class="hanja api grade40"><rb>攻擊</rb><rp>(</rp><rt>공격</rt><rp>)</rp></ruby>할 수 있는 <ruby class="hanja api grade52"><rb>基地</rb><rp>(</rp><rt>기지</rt><rp>)</rp></ruby>로 <ruby class="hanja api grade62"><rb>利用</rb><rp>(</rp><rt>이용</rt><rp>)</rp></ruby>될 것을 두려워했다.
</p><p>핀란드는 <a href="/wiki/%E7%AC%AC2%E6%AC%A1_%E4%B8%96%E7%95%8C%E5%A4%A7%E6%88%B0" class="mw-redirect" title="第2次 世界大戰"><ruby class="hanja api grade62"><rb>第</rb><rp>(</rp><rt>제</rt><rp>)</rp></ruby>2<ruby class="hanja api grade42"><rb>次</rb><rp>(</rp><rt>차</rt><rp>)</rp></ruby> <ruby class="hanja api grade62"><rb>世界大戰</rb><rp>(</rp><rt>세계대전</rt><rp>)</rp></ruby></a>이 <ruby class="hanja api grade10"><rb>勃發</rb><rp>(</rp><rt>발발</rt><rp>)</rp></ruby>하자 <ruby class="hanja api grade72"><rb>中立國</rb><rp>(</rp><rt>중립국</rt><rp>)</rp></ruby><sup id="cite_ref-1" class="reference"><a href="#cite_note-1">[1]</a></sup>을 <ruby class="hanja api grade40"><rb>宣言</rb><rp>(</rp><rt>선언</rt><rp>)</rp></ruby>했으나 <ruby class="hanja api grade32"><rb>蘇聯</rb><rp>(</rp><rt>소련</rt><rp>)</rp></ruby>은 <ruby class="hanja api grade32"><rb>讓步</rb><rp>(</rp><rt>양보</rt><rp>)</rp></ruby>를 <ruby class="hanja api grade42"><rb>要求</rb><rp>(</rp><rt>요구</rt><rp>)</rp></ruby>했다. 핀란드는 <ruby class="hanja api grade72"><rb>時間</rb><rp>(</rp><rt>시간</rt><rp>)</rp></ruby>을 틈타 <ruby class="hanja api grade42"><rb>軍隊</rb><rp>(</rp><rt>군대</rt><rp>)</rp></ruby>를 <ruby class="hanja api grade42"><rb>動員</rb><rp>(</rp><rt>동원</rt><rp>)</rp></ruby>하고 <a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4" class="mw-redirect" title="스웨덴">스웨덴</a>과 <ruby class="hanja api grade60"><rb>英國</rb><rp>(</rp><rt>영국</rt><rp>)</rp></ruby>, <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" class="mw-redirect" title="프랑스">프랑스</a>를 <ruby class="hanja api href="/index.php?title=%E8%81%AF%E5%90%88%E5%9C%8B&amp;action=edit&amp;redlink=1" class="new" title="聯合國 (없는 文書)"><ruby class="hanja api grade32 unknown"><rb>聯合國</rb><rp>(</rp><rt>연합국</rt><rp>)</rp></ruby></a>에 도움을 <ruby class="hanja api grade42"><rb>要請</rb><rp>(</rp><rt>요청</rt><rp>)</rp></ruby>했지만 거의 <ruby class="hanja api </p><p><ruby class="hanja api grade42"><rb>假</rb><rp>(</rp><rt>가</rt><rp>)</rp></ruby>짜 <ruby class="hanja api grade42"><rb>國境</rb><rp>(</rp><rt>국경</rt><rp>)</rp></ruby> <ruby class="hanja api grade50"><rb>事件</rb><rp>(</rp><rt>사건</rt><rp>)</rp></ruby>은 1939<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> 11<ruby class="hanja api grade80"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby> 30<ruby class="hanja api grade80"><rb>日</rb><rp>(</rp><rt>일</rt><rp>)</rp></ruby> <ruby class="hanja api grade32"><rb>蘇聯</rb><rp>(</rp><rt>소련</rt><rp>)</rp></ruby>이 <ruby class="hanja api grade40"><rb>侵攻</rb><rp>(</rp><rt>침공</rt><rp>)</rp></ruby>할 <ruby class="hanja api grade52"><rb>口實</rb><rp>(</rp><rt>구실</rt><rp>)</rp></ruby>을 <ruby class="hanja api grade32"><rb>提供</rb><rp>(</rp><rt>제공</rt><rp>)</rp></ruby>했다. 붉은 <ruby class="hanja api grade42"><rb>軍隊</rb><rp>(</rp><rt>군대</rt><rp>)</rp></ruby>는 <ruby class="hanja api grade40"><rb>裝備</rb><rp>(</rp><rt>장비</rt><rp>)</rp></ruby>가 <ruby class="hanja api grade62"><rb>地形</rb><rp>(</rp><rt>지형</rt><rp>)</rp></ruby>과 겨울 날씨에 <ruby class="hanja api grade42"><rb>對處</rb><rp>(</rp><rt>대처</rt><rp>)</rp></ruby>할 수 없었다. 핀란드 <ruby class="hanja api
