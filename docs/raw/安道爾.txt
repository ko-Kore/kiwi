<p><b><ruby class="hanja api grade10"><rb>安道爾</rb><rp>(</rp><rt>안도이</rt><rp>)</rp></ruby><ruby class="hanja api grade62"><rb>公國</rb><rp>(</rp><rt>공국</rt><rp>)</rp></ruby></b>(<span style="font-size: smaller;"><a href="/index.php?title=%EC%B9%B4%ED%83%88%EB%A3%A8%EB%83%90%E8%AA%9E&amp;action=edit&amp;redlink=1" class="new" title="카탈루냐語 (없는 文書)">카탈루냐語</a>: </span><span lang="ca">Principat d'Andorra</span>)은 <ruby class="hanja api grade80"><rb>南西</rb><rp>(</rp><rt>남서</rt><rp>)</rp></ruby><a href="/wiki/%E6%AD%90%E6%B4%B2" class="mw-redirect" title="歐洲"><ruby class="hanja api grade20"><rb>歐洲</rb><rp>(</rp><rt>구주</rt><rp>)</rp></ruby></a>에 <ruby class="hanja api grade42"><rb>位置</rb><rp>(</rp><rt>위치</rt><rp>)</rp></ruby>하며 <a href="/wiki/%E8%A5%BF%E7%8F%AD%E7%89%99" title="西班牙"><ruby class="hanja api grade32"><rb>西班牙</rb><rp>(</rp><rt>서반아</rt><rp>)</rp></ruby></a>과 <a href="/wiki/%E4%BD%9B%E8%98%AD%E8%A5%BF" title="佛蘭西"><ruby class="hanja api grade32"><rb>佛蘭西</rb><rp>(</rp><rt>불란서</rt><rp>)</rp></ruby></a> 사이에 <ruby class="hanja api grade42"><rb>位置</rb><rp>(</rp><rt>위치</rt><rp>)</rp></ruby>하며 <a href="/index.php?title=%E6%AF%94%E5%88%A9%E7%89%9B%E6%96%AF%E5%B1%B1%E8%84%88&amp;action=edit&amp;redlink=1" class="new" title="比利牛斯山脈 (없는 文書)"><ruby class="hanja api grade50 unknown"><rb>比</rb><rp>(</rp><rt>비</rt><rp>)</rp></ruby><ruby class="hanja api grade62 unknown"><rb>利</rb><rp>(</rp><rt>리</rt><rp>)</rp></ruby><ruby class="hanja api grade50 unknown"><rb>牛</rb><rp>(</rp><rt>우</rt><rp>)</rp></ruby><ruby class="hanja api grade30 unknown"><rb>斯</rb><rp>(</rp><rt>사</rt><rp>)</rp></ruby><ruby class="hanja api grade42 unknown"><rb>山脈</rb><rp>(</rp><rt>산맥</rt><rp>)</rp></ruby></a>의 한가운데에 <ruby class="hanja api grade42"><rb>位置</rb><rp>(</rp><rt>위치</rt><rp>)</rp></ruby>하며 <ruby class="hanja api grade52"><rb>內陸國</rb><rp>(</rp><rt>내륙국</rt><rp>)</rp></ruby>이다. 
</p><p><ruby class="hanja api grade80"><rb>北</rb><rp>(</rp><rt>북</rt><rp>)</rp></ruby>쪽은 <a href="/wiki/%E4%BD%9B%E8%98%AD%E8%A5%BF" title="佛蘭西"><ruby class="hanja api grade32"><rb>佛蘭西</rb><rp>(</rp><rt>불란서</rt><rp>)</rp></ruby></a>와 57<ruby class="hanja api grade0"><rb>粁</rb><rp>(</rp><rt>천</rt><rp>)</rp></ruby> <ruby class="hanja api grade80"><rb>南</rb><rp>(</rp><rt>남</rt><rp>)</rp></ruby>쪽은 <a href="/wiki/%E8%A5%BF%E7%8F%AD%E7%89%99" title="西班牙"><ruby class="hanja api grade32"><rb>西班牙</rb><rp>(</rp><rt>서반아</rt><rp>)</rp></ruby></a>과 64<ruby class="hanja api grade0"><rb>粁</rb><rp>(</rp><rt>천</rt><rp>)</rp></ruby> <ruby class="hanja api grade80"><rb>國土</rb><rp>(</rp><rt>국토</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>面積</rb><rp>(</rp><rt>면적</rt><rp>)</rp></ruby>은 468<ruby class="hanja api grade72"><rb>平方</rb><rp>(</rp><rt>평방</rt><rp>)</rp></ruby><ruby class="hanja api grade0"><rb>粁</rb><rp>(</rp><rt>천</rt><rp>)</rp></ruby>이다. 모두 <ruby class="hanja api grade52"><rb>産地</rb><rp>(</rp><rt>산지</rt><rp>)</rp></ruby>이며 <ruby class="hanja api grade40"><rb>平均</rb><rp>(</rp><rt>평균</rt><rp>)</rp></ruby> 높이는 1996<ruby class="hanja api grade60"><rb>米</rb><rp>(</rp><rt>미</rt><rp>)</rp></ruby>이다. <ruby class="hanja api grade30"><rb>山岳</rb><rp>(</rp><rt>산악</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>氣候</rb><rp>(</rp><rt>기후</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade40"><rb>屬</rb><rp>(</rp><rt>속</rt><rp>)</rp></ruby>하며 <ruby class="hanja api grade40"><rb>年平均</rb><rp>(</rp><rt>연평균</rt><rp>)</rp></ruby> <ruby class="hanja api grade60"><rb>氣溫</rb><rp>(</rp><rt>기온</rt><rp>)</rp></ruby>은 9.9℃이다. <ruby class="hanja api grade10"><rb>安道爾</rb><rp>(</rp><rt>안도이</rt><rp>)</rp></ruby>는 <ruby class="hanja api grade52"><rb>全國的</rb><rp>(</rp><rt>전국적</rt><rp>)</rp></ruby>으로 일곱 <ruby class="hanja api grade42"><rb>個</rb><rp>(</rp><rt>개</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade42"><rb>行政</rb><rp>(</rp><rt>행정</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>區域</rb><rp>(</rp><rt>구역</rt><rp>)</rp></ruby>으로 되어 있고 <ruby class="hanja api grade50"><rb>首都</rb><rp>(</rp><rt>수도</rt><rp>)</rp></ruby>는 <ruby class="hanja api grade10"><rb>安道爾</rb><rp>(</rp><rt>안도이</rt><rp>)</rp></ruby>시티이다. 2023<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> <ruby class="hanja api grade60"><rb>現在</rb><rp>(</rp><rt>현재</rt><rp>)</rp></ruby> <ruby class="hanja api grade10"><rb>安道爾</rb><rp>(</rp><rt>안도이</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade70"><rb>人口</rb><rp>(</rp><rt>인구</rt><rp>)</rp></ruby>는 85,530<ruby class="hanja api grade72"><rb>名</rb><rp>(</rp><rt>명</rt><rp>)</rp></ruby>이다.
</p><p>그 <ruby class="hanja api grade80"><rb>中</rb><rp>(</rp><rt>중</rt><rp>)</rp></ruby> <ruby class="hanja api grade10"><rb>安道爾</rb><rp>(</rp><rt>안도이</rt><rp>)</rp></ruby><ruby class="hanja api grade80"><rb>人</rb><rp>(</rp><rt>인</rt><rp>)</rp></ruby>은 48.7%로 카탈루냐<ruby class="hanja api grade80"><rb>人</rb><rp>(</rp><rt>인</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade40"><rb>屬</rb><rp>(</rp><rt>속</rt><rp>)</rp></ruby>한다. <ruby class="hanja api grade80"><rb>外國人</rb><rp>(</rp><rt>외국인</rt><rp>)</rp></ruby> <ruby class="hanja api grade42"><rb>移民</rb><rp>(</rp><rt>이민</rt><rp>)</rp></ruby><ruby class="hanja api grade60"><rb>者</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby> <ruby class="hanja api grade80"><rb>中</rb><rp>(</rp><rt>중</rt><rp>)</rp></ruby> <ruby class="hanja api grade32"><rb>西班牙</rb><rp>(</rp><rt>서반아</rt><rp>)</rp></ruby><ruby class="hanja api grade80"><rb>人</rb><rp>(</rp><rt>인</rt><rp>)</rp></ruby>이 24.6%를 차지하고 <ruby class="hanja api grade10"><rb>葡萄牙</rb><rp>(</rp><rt>포도아</rt><rp>)</rp></ruby><ruby class="hanja api grade80"><rb>人</rb><rp>(</rp><rt>인</rt><rp>)</rp></ruby>(11.6%)과 <ruby class="hanja api grade32"><rb>佛蘭西</rb><rp>(</rp><rt>불란서</rt><rp>)</rp></ruby><ruby class="hanja api grade80"><rb>人</rb><rp>(</rp><rt>인</rt><rp>)</rp></ruby>(4.4%)이 그 뒤를 이었다.
</p>
