<h2><span id="槪要(개요)"></span><span class="mw-headline" id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"><ruby class="hanja grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E4%B8%8B%E9%97%9C%E6%A2%9D%E7%B4%84&amp;action=edit&amp;section=1" title="부분 편집: 槪要(개요)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><b><ruby class="hanja grade40"><rb>下關條約</rb><rp>(</rp><rt>하관조약</rt><rp>)</rp></ruby></b>은 <a href="/index.php?title=1895%E5%B9%B4&amp;action=edit&amp;redlink=1" class="new" title="1895年 (없는 문서)">1895<ruby class="hanja grade80 unknown"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby></a><a href="/index.php?title=4%E6%9C%8817%E6%97%A5&amp;action=edit&amp;redlink=1" class="new" title="4月17日 (없는 문서)">4<ruby class="hanja grade80 unknown"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby>17<ruby class="hanja grade80 unknown"><rb>日</rb><rp>(</rp><rt>일</rt><rp>)</rp></ruby></a>에 <a href="/wiki/%E6%97%A5%E6%9C%AC" title="日本"><ruby class="hanja grade60"><rb>日本</rb><rp>(</rp><rt>일본</rt><rp>)</rp></ruby></a> <a href="/index.php?title=%E5%B1%B1%E5%8F%A3%E7%B8%A3&amp;action=edit&amp;redlink=1" class="new" title="山口縣 (없는 문서)"><ruby class="hanja grade30 unknown"><rb>山口縣</rb><rp>(</rp><rt>산구현</rt><rp>)</rp></ruby></a> <ruby class="hanja grade50"><rb>赤間關市</rb><rp>(</rp><rt>적간관시</rt><rp>)</rp></ruby>(<ruby class="hanja grade62"><rb>現</rb><rp>(</rp><rt>현</rt><rp>)</rp></ruby> <a href="/index.php?title=%E4%B8%8B%E9%97%9C%E5%B8%82&amp;action=edit&amp;redlink=1" class="new" title="下關市 (없는 문서)"><ruby class="hanja grade52 unknown"><rb>下關市</rb><rp>(</rp><rt>하관시</rt><rp>)</rp></ruby></a>)에서 <a href="/wiki/%E6%97%A5%E6%9C%AC" title="日本"><ruby class="hanja grade60"><rb>日本</rb><rp>(</rp><rt>일본</rt><rp>)</rp></ruby></a>과 <a href="/index.php?title=%E6%B7%B8%EB%82%98%EB%9D%BC&amp;action=edit&amp;redlink=1" class="new" title="淸나라 (없는 문서)"><ruby class="hanja grade62 unknown"><rb>淸</rb><rp>(</rp><rt>청</rt><rp>)</rp></ruby>나라</a> 사이에 열린 <a href="/index.php?title=%E6%B7%B8%E6%97%A5%E6%88%B0%E7%88%AD&amp;action=edit&amp;redlink=1" class="new" title="淸日戰爭 (없는 문서)"><ruby class="hanja grade50 unknown"><rb>淸日戰爭</rb><rp>(</rp><rt>청일전쟁</rt><rp>)</rp></ruby></a>의 <ruby class="hanja grade42"><rb>講和會議</rb><rp>(</rp><rt>강화회의</rt><rp>)</rp></ruby>로 <ruby class="hanja grade20"><rb>締結</rb><rp>(</rp><rt>체결</rt><rp>)</rp></ruby>된 <ruby class="hanja grade40"><rb>條約</rb><rp>(</rp><rt>조약</rt><rp>)</rp></ruby>으로서 <ruby class="hanja grade40"><rb>正式名稱</rb><rp>(</rp><rt>정식명칭</rt><rp>)</rp></ruby>은 <b><ruby class="hanja grade0"><rb>日淸兩國媾和條約</rb><rp>(</rp><rt>일청량국구화조약</rt><rp>)</rp></ruby></b>(<ruby class="hanja grade60"><rb>日本</rb><rp>(</rp><rt>일본</rt><rp>)</rp></ruby> <ruby class="hanja grade32"><rb>側</rb><rp>(</rp><rt>측</rt><rp>)</rp></ruby>), <b><ruby class="hanja grade50"><rb>馬關新約</rb><rp>(</rp><rt>마관신약</rt><rp>)</rp></ruby></b>(<ruby class="hanja grade62"><rb>淸</rb><rp>(</rp><rt>청</rt><rp>)</rp></ruby>나라 <ruby class="hanja grade32"><rb>側</rb><rp>(</rp><rt>측</rt><rp>)</rp></ruby>)이다.
</p><p>예전 <ruby class="hanja grade42"><rb>會議</rb><rp>(</rp><rt>회의</rt><rp>)</rp></ruby>가 열린 <ruby class="hanja grade50"><rb>赤間關市</rb><rp>(</rp><rt>적간관시</rt><rp>)</rp></ruby>의 <ruby class="hanja grade40"><rb>通稱</rb><rp>(</rp><rt>통칭</rt><rp>)</rp></ruby>이었다 “<ruby class="hanja grade50"><rb>馬關</rb><rp>(</rp><rt>마관</rt><rp>)</rp></ruby>”을 <ruby class="hanja grade42"><rb>取</rb><rp>(</rp><rt>취</rt><rp>)</rp></ruby>하고 <ruby class="hanja grade32"><rb>一般的</rb><rp>(</rp><rt>일반적</rt><rp>)</rp></ruby>으로 <ruby class="hanja grade40"><rb>馬關條約</rb><rp>(</rp><rt>마관조약</rt><rp>)</rp></ruby>이라고 불렸다. “<ruby class="hanja grade40"><rb>下關條約</rb><rp>(</rp><rt>하관조약</rt><rp>)</rp></ruby>”은 <ruby class="hanja grade60"><rb>日本</rb><rp>(</rp><rt>일본</rt><rp>)</rp></ruby>에서 <a href="/wiki/%E7%AC%AC%E4%BA%8C%E6%AC%A1%E4%B8%96%E7%95%8C%E5%A4%A7%E6%88%B0" title="第二次世界大戰"><ruby class="hanja grade42"><rb>第二次世界大戰</rb><rp>(</rp><rt>제이차세계대전</rt><rp>)</rp></ruby></a> <ruby class="hanja grade72"><rb>後</rb><rp>(</rp><rt>후</rt><rp>)</rp></ruby> <ruby class="hanja grade52"><rb>定着</rb><rp>(</rp><rt>정착</rt><rp>)</rp></ruby>한 <ruby class="hanja grade40"><rb>呼稱</rb><rp>(</rp><rt>호칭</rt><rp>)</rp></ruby>이며 <a href="/wiki/%E4%B8%AD%E5%9C%8B" title="中國"><ruby class="hanja grade80"><rb>中國</rb><rp>(</rp><rt>중국</rt><rp>)</rp></ruby></a>에서는 <ruby class="hanja grade30"><rb>只今</rb><rp>(</rp><rt>지금</rt><rp>)</rp></ruby>도 “<ruby class="hanja grade40"><rb>馬關條約</rb><rp>(</rp><rt>마관조약</rt><rp>)</rp></ruby>”라고 부르고있다.
</p>
