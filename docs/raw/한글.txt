<h2><span id=".E5.90.8D.E7.A8.B1.28.EB.AA.85.EC.B9.AD.29"></span><span class="mw-headline" id="名稱(명칭)"><ruby class="hanja grade40"><rb>名稱</rb><rp>(</rp><rt>명칭</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%ED%95%9C%EA%B8%80&amp;action=edit&amp;section=1" title="부분 편집: 名稱(명칭)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%E9%9F%93" title="韓"><ruby class="hanja grade80"><rb>韓</rb><rp>(</rp><rt>한</rt><rp>)</rp></ruby></a><a href="/wiki/%E3%90%8E" title="㐎"><ruby class="hanja grade0"><rb>㐎</rb><rp>(</rp><rt>글</rt><rp>)</rp></ruby></a>과 같이 <ruby class="hanja grade32"><rb>音借</rb><rp>(</rp><rt>음차</rt><rp>)</rp></ruby>되는 <ruby class="hanja grade40"><rb>境遇</rb><rp>(</rp><rt>경우</rt><rp>)</rp></ruby>도 있으며, <ruby class="hanja grade52"><rb>元來</rb><rp>(</rp><rt>원래</rt><rp>)</rp></ruby> 크다는 <ruby class="hanja grade42"><rb>意味</rb><rp>(</rp><rt>의미</rt><rp>)</rp></ruby>의 ‘하다’와 ‘글’이 <ruby class="hanja grade60"><rb>合</rb><rp>(</rp><rt>합</rt><rp>)</rp></ruby>쳐진 말이라는<ruby class="hanja grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby> 여러 <ruby class="hanja grade52"><rb>說</rb><rp>(</rp><rt>설</rt><rp>)</rp></ruby>이 있다. <ruby class="hanja grade60"><rb>本來</rb><rp>(</rp><rt>본래</rt><rp>)</rp></ruby> <a href="/index.php?title=%E8%A8%93%E6%B0%91%E6%AD%A3%E9%9F%B3&amp;action=edit&amp;redlink=1" class="new" title="訓民正音 (없는 문서)"><ruby class="hanja grade60 unknown"><rb>訓民正音</rb><rp>(</rp><rt>훈민정음</rt><rp>)</rp></ruby></a>에서 <ruby class="hanja grade62"><rb>發表</rb><rp>(</rp><rt>발표</rt><rp>)</rp></ruby>된 이 <ruby class="hanja grade40"><rb>文字體系</rb><rp>(</rp><rt>문자체계</rt><rp>)</rp></ruby>에 <ruby class="hanja grade62"><rb>對</rb><rp>(</rp><rt>대</rt><rp>)</rp></ruby>한 이름을 ‘한글’로 부르기 <ruby class="hanja grade62"><rb>始作</rb><rp>(</rp><rt>시작</rt><rp>)</rp></ruby>한 것도 <a href="/index.php?title=%E5%91%A8%E6%99%82%E7%B6%93&amp;action=edit&amp;redlink=1" class="new" title="周時經 (없는 문서)"><ruby class="hanja grade40 unknown"><rb>周時經</rb><rp>(</rp><rt>주시경</rt><rp>)</rp></ruby></a>이라고 알려져 있기도 하지만, 이 <ruby class="hanja grade32"><rb>亦是</rb><rp>(</rp><rt>역시</rt><rp>)</rp></ruby> <ruby class="hanja grade32"><rb>諸說</rb><rp>(</rp><rt>제설</rt><rp>)</rp></ruby>이 <ruby class="hanja grade40"><rb>存在</rb><rp>(</rp><rt>존재</rt><rp>)</rp></ruby>한다.
</p>
<h2><span id=".E7.89.B9.E6.80.A7.28.ED.8A.B9.EC.84.B1.29"></span><span class="mw-headline" id="特性(특성)"><ruby class="hanja grade52"><rb>特性</rb><rp>(</rp><rt>특성</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%ED%95%9C%EA%B8%80&amp;action=edit&amp;section=2" title="부분 편집: 特性(특성)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/index.php?title=%E9%9F%B3%E7%AF%80&amp;action=edit&amp;redlink=1" class="new" title="音節 (없는 문서)"><ruby class="hanja grade52 unknown"><rb>音節</rb><rp>(</rp><rt>음절</rt><rp>)</rp></ruby></a><ruby class="hanja grade42"><rb>單位</rb><rp>(</rp><rt>단위</rt><rp>)</rp></ruby>로 모아 쓰기 때문에 <ruby class="hanja grade52"><rb>音節間</rb><rp>(</rp><rt>음절간</rt><rp>)</rp></ruby>의 <ruby class="hanja grade60"><rb>區分</rb><rp>(</rp><rt>구분</rt><rp>)</rp></ruby>이 <ruby class="hanja grade42"><rb>明確</rb><rp>(</rp><rt>명확</rt><rp>)</rp></ruby>해 <a href="/index.php?title=%E9%9F%B3%E7%B4%A0%E6%96%87%E5%AD%97&amp;action=edit&amp;redlink=1" class="new" title="音素文字 (없는 문서)"><ruby title="音節文字 (없는 문서)"><ruby class="hanja grade52 unknown"><rb>音節文字</rb><rp>(</rp><rt>음절문자</rt><rp>)</rp></ruby></a>의 <ruby class="hanja grade52"><rb>性格</rb><rp>(</rp><rt>성격</rt><rp>)</rp></ruby>도 <ruby class="hanja grade70"><rb>同時</rb><rp>(</rp><rt>동시</rt><rp>)</rp></ruby>에 지닌다. <a href="/index.php?title=%EB%AA%A8%EC%95%84%EC%93%B0%EA%B8%B0&amp;action=edit&amp;redlink=1" class="new" title="모아쓰기 (없는 문서)">모아쓰기</a>를 하는 <ruby class="hanja grade60"><rb>理由</rb><rp>(</rp><rt>이유</rt><rp>)</rp></ruby>는 <a href="/wiki/%E6%BC%A2%E5%AD%97" title="漢字"><ruby class="hanja grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby></a>와의 <a href="/index.php?title=%E4%BA%92%E6%8F%9B%E6%80%A7&amp;action=edit&amp;redlink=1" class="new" title="互換性 (없는 문서)"><ruby class="hanja </p><p>각 <ruby class="hanja grade42"><rb>字素</rb><rp>(</rp><rt>자소</rt><rp>)</rp></ruby>는 <ruby class="hanja grade50"><rb>科學原理</rb><rp>(</rp><rt>과학원리</rt><rp>)</rp></ruby>를 <ruby class="hanja grade40"><rb>適用</rb><rp>(</rp><rt>적용</rt><rp>)</rp></ruby>하여 만들어졌다. <ruby class="hanja grade62"><rb>子音</rb><rp>(</rp><rt>자음</rt><rp>)</rp></ruby>은 <ruby class="hanja grade42"><rb>發音器官</rb><rp>(</rp><rt>발음기관</rt><rp>)</rp></ruby>의 <ruby class="hanja grade40"><rb>形象</rb><rp>(</rp><rt>형상</rt><rp>)</rp></ruby>을 본따서 <ruby class="hanja grade52"><rb>基本</rb><rp>(</rp><rt>기본</rt><rp>)</rp></ruby>글자를 만들고 <a href="/index.php?title=%E5%8A%83&amp;action=edit&amp;redlink=1" class="new" title="劃 (없는 문서)"><ruby class="hanja grade32 unknown"><rb>劃</rb><rp>(</rp><rt>획</rt><rp>)</rp></ruby></a>을 <ruby class="hanja grade30"><rb>添加</rb><rp>(</rp><rt>첨가</rt><rp>)</rp></ruby>하여 <ruby class="hanja grade42"><rb>細部發音</rb><rp>(</rp><rt>세부발음</rt><rp>)</rp></ruby>을 <ruby class="hanja grade60"><rb>區分</rb><rp>(</rp><rt>구분</rt><rp>)</rp></ruby>했으며, <ruby class="hanja grade62"><rb>母音</rb><rp>(</rp><rt>모음</rt><rp>)</rp></ruby>은 <a href="/index.php?title=%E5%A4%A9%E5%9C%B0%E4%BA%BA&amp;action=edit&amp;redlink=1" class="new" title="天地人 (없는 문서)"><ruby class="hanja grade70 unknown"><rb>天地人</rb><rp>(</rp><rt>천지인</rt><rp>)</rp></ruby></a>을 <ruby class="hanja grade52"><rb>基本</rb><rp>(</rp><rt>기본</rt><rp>)</rp></ruby>으로 <ruby class="hanja grade40"><rb>組合</rb><rp>(</rp><rt>조합</rt><rp>)</rp></ruby>하여<ruby class="hanja grade40"><rb>構成</rb><rp>(</rp><rt>구성</rt><rp>)</rp></ruby>된다.
