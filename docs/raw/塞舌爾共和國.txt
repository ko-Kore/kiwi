<h3><span id=".E5.9C.8B.E6.97.97.28.EA.B5.AD.EA.B8.B0.29"></span><span class="mw-headline" id="國旗(국기)"><ruby class="hanja api grade70"><rb>國旗</rb><rp>(</rp><rt>국기</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E5%A1%9E%E8%88%8C%E7%88%BE%E5%85%B1%E5%92%8C%E5%9C%8B&amp;action=edit&amp;section=2" title="部分 編輯: 國旗(국기)">編輯</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><span class="mw-default-size" typeof="mw:File"><a href="/wiki/%ED%8C%8C%EC%9D%BC:Flag_of_Seychelles.svg" class="mw-file-description"><img src="https://upload.wikimedia.org/wikipedia/commons/thumb/f/fc/Flag_of_Seychelles.svg/langko-kore-1200px-Flag_of_Seychelles.svg.png" decoding="async" width="1200" height="600" srcset="https://upload.wikimedia.org/wikipedia/commons/thumb/f/fc/Flag_of_Seychelles.svg/langko-kore-1800px-Flag_of_Seychelles.svg.png 1.5x, https://upload.wikimedia.org/wikipedia/commons/thumb/f/fc/Flag_of_Seychelles.svg/langko-kore-2400px-Flag_of_Seychelles.svg.png 2x"></a></span>
<ruby class="hanja api grade10"><rb>塞舌爾</rb><rp>(</rp><rt>새설이</rt><rp>)</rp></ruby><ruby class="hanja api grade62"><rb>共和國</rb><rp>(</rp><rt>공화국</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade70"><rb>國旗</rb><rp>(</rp><rt>국기</rt><rp>)</rp></ruby>는 위에서 아래로 빨간<ruby class="hanja api grade70"><rb>色</rb><rp>(</rp><rt>색</rt><rp>)</rp></ruby>, 흰<ruby class="hanja api grade70"><rb>色</rb><rp>(</rp><rt>색</rt><rp>)</rp></ruby> 및 <ruby class="hanja api grade60"><rb>綠色</rb><rp>(</rp><rt>녹색</rt><rp>)</rp></ruby>의 세 가지 물결로 <ruby class="hanja api grade40"><rb>構成</rb><rp>(</rp><rt>구성</rt><rp>)</rp></ruby>됩니다. <ruby class="hanja api grade60"><rb>綠色</rb><rp>(</rp><rt>녹색</rt><rp>)</rp></ruby>은 <ruby class="hanja api grade10"><rb>塞舌爾</rb><rp>(</rp><rt>새설이</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade70"><rb>土地</rb><rp>(</rp><rt>토지</rt><rp>)</rp></ruby>, <ruby class="hanja api grade70"><rb>植物</rb><rp>(</rp><rt>식물</rt><rp>)</rp></ruby>, <ruby class="hanja api grade70"><rb>土地</rb><rp>(</rp><rt>토지</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>資源</rb><rp>(</rp><rt>자원</rt><rp>)</rp></ruby> 및 <ruby class="hanja api grade62"><rb>農業</rb><rp>(</rp><rt>농업</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade52"><rb>發展</rb><rp>(</rp><rt>발전</rt><rp>)</rp></ruby>을 <ruby class="hanja api grade32"><rb>象徵</rb><rp>(</rp><rt>상징</rt><rp>)</rp></ruby>합니다. 흰<ruby class="hanja api grade70"><rb>色</rb><rp>(</rp><rt>색</rt><rp>)</rp></ruby>은 <ruby class="hanja api grade40"><rb>海底</rb><rp>(</rp><rt>해저</rt><rp>)</rp></ruby>의 바다, <ruby class="hanja api grade60"><rb>海洋</rb><rp>(</rp><rt>해양</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>資源</rb><rp>(</rp><rt>자원</rt><rp>)</rp></ruby>, <ruby class="hanja api grade60"><rb>石油</rb><rp>(</rp><rt>석유</rt><rp>)</rp></ruby> 및 <ruby class="hanja api grade32"><rb>其他</rb><rp>(</rp><rt>기타</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>鑛物</rb><rp>(</rp><rt>광물</rt><rp>)</rp></ruby> <ruby class="hanja api grade30"><rb>埋葬地</rb><rp>(</rp><rt>매장지</rt><rp>)</rp></ruby>를 <ruby class="hanja api grade32"><rb>象徵</rb><rp>(</rp><rt>상징</rt><rp>)</rp></ruby>합니다. 붉은<ruby class="hanja api grade70"><rb>色</rb><rp>(</rp><rt>색</rt><rp>)</rp></ruby>은 <ruby class="hanja api grade40"><rb>革命</rb><rp>(</rp><rt>혁명</rt><rp>)</rp></ruby>과 <ruby class="hanja api grade42"><rb>進步</rb><rp>(</rp><rt>진보</rt><rp>)</rp></ruby>를 <ruby class="hanja api grade32"><rb>象徵</rb><rp>(</rp><rt>상징</rt><rp>)</rp></ruby>하며 <ruby class="hanja api grade80"><rb>人民</rb><rp>(</rp><rt>인민</rt><rp>)</rp></ruby>이 새로운 <ruby class="hanja api grade42"><rb>進步</rb><rp>(</rp><rt>진보</rt><rp>)</rp></ruby><ruby class="hanja api grade62"><rb>社會</rb><rp>(</rp><rt>사회</rt><rp>)</rp></ruby>를 <ruby class="hanja api grade42"><rb>建設</rb><rp>(</rp><rt>건설</rt><rp>)</rp></ruby>하기를 <ruby class="hanja api grade50"><rb>願</rb><rp>(</rp><rt>원</rt><rp>)</rp></ruby>한다는 것을 나타낸다. 세 가지 <ruby class="hanja api grade52"><rb>色相</rb><rp>(</rp><rt>색상</rt><rp>)</rp></ruby>은 <ruby class="hanja api grade62"><rb>平等</rb><rp>(</rp><rt>평등</rt><rp>)</rp></ruby>, <ruby class="hanja api grade60"><rb>自由</rb><rp>(</rp><rt>자유</rt><rp>)</rp></ruby>, <ruby class="hanja api grade42"><rb>博愛</rb><rp>(</rp><rt>박애</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade32"><rb>基盤</rb><rp>(</rp><rt>기반</rt><rp>)</rp></ruby>한 <ruby class="hanja api grade72"><rb>國家</rb><rp>(</rp><rt>국가</rt><rp>)</rp></ruby> <ruby class="hanja api grade60"><rb>和合</rb><rp>(</rp><rt>화합</rt><rp>)</rp></ruby>을 <ruby class="hanja api grade32"><rb>象徵</rb><rp>(</rp><rt>상징</rt><rp>)</rp></ruby>합니다.

</p><p><ruby class="hanja api grade10"><rb>塞舌爾</rb><rp>(</rp><rt>새설이</rt><rp>)</rp></ruby><ruby class="hanja api grade62"><rb>共和國</rb><rp>(</rp><rt>공화국</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade60"><rb>文章</rb><rp>(</rp><rt>문장</rt><rp>)</rp></ruby>은 1976 <ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>연</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade42"><rb>設立</rb><rp>(</rp><rt>설립</rt><rp>)</rp></ruby> 된 <ruby class="hanja api grade10"><rb>楕圓形</rb><rp>(</rp><rt>타원형</rt><rp>)</rp></ruby> <ruby class="hanja api grade60"><rb>文章</rb><rp>(</rp><rt>문장</rt><rp>)</rp></ruby>입니다. <ruby class="hanja api grade10"><rb>防牌</rb><rp>(</rp><rt>방패</rt><rp>)</rp></ruby>는 푸른 하늘, 흰 구름, 깊고 푸른 바다로 그려져 있습니다. 멀리 푸른 <ruby class="hanja api grade80"><rb>山</rb><rp>(</rp><rt>산</rt><rp>)</rp></ruby>이 있고 멀리 푸른 땅이 있어 <ruby class="hanja api grade10"><rb>塞舌爾</rb><rp>(</rp><rt>새설이</rt><rp>)</rp></ruby>이 두 그룹의 섬으로 이루어진 섬나라임을 <ruby class="hanja api grade32"><rb>象徵</rb><rp>(</rp><rt>상징</rt><rp>)</rp></ruby>합니다. 흰 <ruby class="hanja api grade10"><rb>帆船</rb><rp>(</rp><rt>범선</rt><rp>)</rp></ruby>이 바다를 거슬러 올라가고 있어 <ruby class="hanja api grade50"><rb>漁村</rb><rp>(</rp><rt>어촌</rt><rp>)</rp></ruby>임을 알 수 있습니다.
