<h2><span id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"></span><span class="mw-headline" id="槪要(개요)"><ruby class="hanja api grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E4%BA%9C%E7%88%BE%E7%84%B6%E4%B8%81&amp;action=edit&amp;section=1" title="部分 編輯: 槪要(개요)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja api grade60"><rb>南美</rb><rp>(</rp><rt>남미</rt><rp>)</rp></ruby> <ruby class="hanja api grade62"><rb>南東部</rb><rp>(</rp><rt>남동부</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade42"><rb>位置</rb><rp>(</rp><rt>위치</rt><rp>)</rp></ruby>하여 <ruby class="hanja api grade80"><rb>東</rb><rp>(</rp><rt>동</rt><rp>)</rp></ruby>쪽은 <ruby class="hanja api grade52"><rb>大陸</rb><rp>(</rp><rt>대륙</rt><rp>)</rp></ruby>, <ruby class="hanja api grade80"><rb>西</rb><rp>(</rp><rt>서</rt><rp>)</rp></ruby>쪽은 칠레, <ruby class="hanja api grade80"><rb>東西</rb><rp>(</rp><rt>동서</rt><rp>)</rp></ruby> 너비 1,423km입니다. <ruby class="hanja api grade52"><rb>陸地</rb><rp>(</rp><rt>육지</rt><rp>)</rp></ruby> <ruby class="hanja api <h2><span id=".E6.AD.B7.E5.8F.B2.28.EC.97.AD.EC.82.AC.29"></span><span class="mw-headline" id="歷史(역사)"><ruby class="hanja api grade52"><rb>歷史</rb><rp>(</rp><rt>역사</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E4%BA%9C%E7%88%BE%E7%84%B6%E4%B8%81&amp;action=edit&amp;section=2" title="部分 編輯: 歷史(역사)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>16<ruby class="hanja api grade40"><rb>世紀</rb><rp>(</rp><rt>세기</rt><rp>)</rp></ruby> <ruby class="hanja api grade52"><rb>以前</rb><rp>(</rp><rt>이전</rt><rp>)</rp></ruby>에는 <ruby class="hanja api grade52"><rb>土着</rb><rp>(</rp><rt>토착</rt><rp>)</rp></ruby> 인디언들이 살았다. 
</p><p>16<ruby class="hanja api grade40"><rb>世紀</rb><rp>(</rp><rt>세기</rt><rp>)</rp></ruby> <ruby class="hanja api grade32"><rb>中盤</rb><rp>(</rp><rt>중반</rt><rp>)</rp></ruby>에 스페인 <ruby class="hanja api grade70"><rb>植民地</rb><rp>(</rp><rt>식민지</rt><rp>)</rp></ruby>가 되었습니다. 
</p><p>1810<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> 5<ruby class="hanja api grade80"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby> 25<ruby class="hanja api grade80"><rb>日</rb><rp>(</rp><rt>일</rt><rp>)</rp></ruby>, 스페인 <ruby class="hanja api grade70"><rb>植民</rb><rp>(</rp><rt>식민</rt><rp>)</rp></ruby> <ruby class="hanja api grade42"><rb>統治</rb><rp>(</rp><rt>통치</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade40"><rb>對抗</rb><rp>(</rp><rt>대항</rt><rp>)</rp></ruby>하는 5<ruby class="hanja api grade80"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>革命</rb><rp>(</rp><rt>혁명</rt><rp>)</rp></ruby>이 일어나 <ruby class="hanja api grade50"><rb>最初</rb><rp>(</rp><rt>최초</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade42"><rb>政府</rb><rp>(</rp><rt>정부</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>委員會</rb><rp>(</rp><rt>위원회</rt><rp>)</rp></ruby>가 <ruby class="hanja api grade40"><rb>構成</rb><rp>(</rp><rt>구성</rt><rp>)</rp></ruby>되었다. 
</p><p>1812 <ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>연</rt><rp>)</rp></ruby> <ruby class="hanja api grade72"><rb>國家</rb><rp>(</rp><rt>국가</rt><rp>)</rp></ruby> <ruby class="hanja api grade50"><rb>英雄</rb><rp>(</rp><rt>영웅</rt><rp>)</rp></ruby> San Martin은 스페인 <ruby class="hanja api grade70"><rb>植民地</rb><rp>(</rp><rt>식민지</rt><rp>)</rp></ruby> <ruby class="hanja api grade42"><rb>軍隊</rb><rp>(</rp><rt>군대</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade40"><rb>對抗</rb><rp>(</rp><rt>대항</rt><rp>)</rp></ruby>하여 사람들을 이끌고 1816 <ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>연</rt><rp>)</rp></ruby> 7 <ruby class="hanja api grade80"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby> 9 <ruby class="hanja api grade80"><rb>日</rb><rp>(</rp><rt>일</rt><rp>)</rp></ruby> <ruby class="hanja api grade52"><rb>獨立</rb><rp>(</rp><rt>독립</rt><rp>)</rp></ruby>을 <ruby class="hanja api grade40"><rb>宣言</rb><rp>(</rp><rt>선언</rt><rp>)</rp></ruby>했습니다. 
</p><p>그 <ruby class="hanja api grade52"><rb>以後</rb><rp>(</rp><rt>이후</rt><rp>)</rp></ruby>로 아프가니스탄은 오랫동안 <ruby class="hanja api grade40"><rb>混亂</rb><rp>(</rp><rt>혼란</rt><rp>)</rp></ruby>과 <ruby class="hanja api grade32"><rb>分裂</rb><rp>(</rp><rt>분열</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade42"><rb>狀態</rb><rp>(</rp><rt>상태</rt><rp>)</rp></ruby>에 있었습니다. 
</p><p>1853<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> 우르키<ruby class="hanja api grade72"><rb>子</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby> <ruby class="hanja api grade42"><rb>將軍</rb><rp>(</rp><rt>장군</rt><rp>)</rp></ruby>은 <ruby class="hanja api grade50"><rb>最初</rb><rp>(</rp><rt>최초</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade40"><rb>憲法</rb><rp>(</rp><rt>헌법</rt><rp>)</rp></ruby>을 <ruby class="hanja api grade42"><rb>制定</rb><rp>(</rp><rt>제정</rt><rp>)</rp></ruby>하고 <ruby class="hanja api grade30"><rb>聯邦</rb><rp>(</rp><rt>연방</rt><rp>)</rp></ruby> <ruby class="hanja api grade62"><rb>共和國</rb><rp>(</rp><rt>공화국</rt><rp>)</rp></ruby>을 <ruby class="hanja api grade60"><rb>樹立</rb><rp>(</rp><rt>수립</rt><rp>)</rp></ruby>했으며, 우르키<ruby class="hanja api grade60"><rb>者</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>는 <ruby class="hanja api grade40"><rb>憲法</rb><rp>(</rp><rt>헌법</rt><rp>)</rp></ruby> <ruby class="hanja api grade42"><rb>制定</rb><rp>(</rp><rt>제정</rt><rp>)</rp></ruby> <ruby class="hanja api grade72"><rb>後</rb><rp>(</rp><rt>후</rt><rp>)</rp></ruby> 아프가니스탄의 첫 <ruby class="hanja api grade60"><rb>番</rb><rp>(</rp><rt>번</rt><rp>)</rp></ruby>째 <ruby class="hanja api grade42"><rb>大統領</rb><rp>(</rp><rt>대통령</rt><rp>)</rp></ruby>이 되었다. 
</p><p>1860 <ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>연</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade62"><rb>共和國</rb><rp>(</rp><rt>공화국</rt><rp>)</rp></ruby>으로 <ruby class="hanja api grade40"><rb>變更</rb><rp>(</rp><rt>변경</rt><rp>)</rp></ruby>되었습니다. 
</p><p>20<ruby class="hanja api grade40"><rb>世紀</rb><rp>(</rp><rt>세기</rt><rp>)</rp></ruby> 30<ruby class="hanja api grade62"><rb>年代</rb><rp>(</rp><rt>연대</rt><rp>)</rp></ruby> <ruby class="hanja api grade52"><rb>以後</rb><rp>(</rp><rt>이후</rt><rp>)</rp></ruby> <ruby class="hanja api grade62"><rb>軍部</rb><rp>(</rp><rt>군부</rt><rp>)</rp></ruby>와 <ruby class="hanja api grade72"><rb>民間</rb><rp>(</rp><rt>민간</rt><rp>)</rp></ruby>이 번갈아 <ruby class="hanja api grade42"><rb>權力</rb><rp>(</rp><rt>권력</rt><rp>)</rp></ruby>을 휘두르는 <ruby class="hanja api
