<h2><span id="槪說(개설)"></span><span class="mw-headline" id=".E6.A7.AA.E8.AA.AA.28.EA.B0.9C.EC.84.A4.29"><ruby class="hanja grade32"><rb>槪說</rb><rp>(</rp><rt>개설</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E7%BE%8E%E5%9C%8B%EB%8B%AC%EB%9F%AC&amp;action=edit&amp;section=1" title="부분 편집: 槪說(개설)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja grade60"><rb>美國</rb><rp>(</rp><rt>미국</rt><rp>)</rp></ruby>달러(<ruby class="hanja grade60"><rb>英</rb><rp>(</rp><rt>영</rt><rp>)</rp></ruby>&nbsp;: United States Dollar)는 <a href="/wiki/%E7%BE%8E%E5%90%88%E8%A1%86%E5%9C%8B" title="美合衆國"><ruby class="hanja grade42"><rb>美合衆國</rb><rp>(</rp><rt>미합중국</rt><rp>)</rp></ruby></a>의<a href="/index.php?title=%E6%B3%95%E5%AE%9A%E9%80%9A%E8%B2%A8&amp;action=edit&amp;redlink=1" class="new" title="法定通貨 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>法定通貨</rb><rp>(</rp><rt>법정통화</rt><rp>)</rp></ruby></a>이다. <ruby class="hanja grade40"><rb>別稱</rb><rp>(</rp><rt>별칭</rt><rp>)</rp></ruby>으로는 <ruby class="hanja grade42"><rb>美合衆國</rb><rp>(</rp><rt>미합중국</rt><rp>)</rp></ruby><a href="/index.php?title=%E5%BC%97&amp;action=edit&amp;redlink=1" class="new" title="弗 (없는 문서)"><ruby class="hanja grade20 unknown"><rb>弗</rb><rp>(</rp><rt>불</rt><rp>)</rp></ruby></a>, <ruby class="hanja grade20"><rb>美弗</rb><rp>(</rp><rt>미불</rt><rp>)</rp></ruby>, 딸라, 달러<ruby class="hanja grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby>이있다. ISO4217에<ruby class="hanja grade40"><rb>依</rb><rp>(</rp><rt>의</rt><rp>)</rp></ruby>해<ruby class="hanja grade42"><rb>配定</rb><rp>(</rp><rt>배정</rt><rp>)</rp></ruby>된 <a href="/index.php?title=%E9%80%9A%E8%B2%A8%EC%BD%94%EB%93%9C&amp;action=edit&amp;redlink=1" class="new" title="通貨코드 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>通貨</rb><rp>(</rp><rt>통화</rt><rp>)</rp></ruby>코드</a>는 USD이며, <a href="/wiki/%E7%BE%8E%E5%9C%8B" class="mw-redirect" title="美國"><ruby class="hanja grade60"><rb>美國</rb><rp>(</rp><rt>미국</rt><rp>)</rp></ruby></a><ruby class="hanja grade72"><rb>內</rb><rp>(</rp><rt>내</rt><rp>)</rp></ruby>에서는 <ruby class="hanja grade70"><rb>主</rb><rp>(</rp><rt>주</rt><rp>)</rp></ruby>로 <ruby class="hanja grade60"><rb>記號</rb><rp>(</rp><rt>기호</rt><rp>)</rp></ruby>인＄로<ruby class="hanja grade62"><rb>表記</rb><rp>(</rp><rt>표기</rt><rp>)</rp></ruby>되고, <ruby class="hanja grade70"><rb>口語</rb><rp>(</rp><rt>구어</rt><rp>)</rp></ruby>로는'dollar(s)'<ruby class="hanja grade40"><rb>或</rb><rp>(</rp><rt>혹</rt><rp>)</rp></ruby>은 <ruby class="hanja grade40"><rb>俗稱</rb><rp>(</rp><rt>속칭</rt><rp>)</rp></ruby>인'bucks'로불린다. <a href="/index.php?title=%E8%A3%9C%E5%8A%A9%E5%96%AE%E4%BD%8D&amp;action=edit&amp;redlink=1" class="new" title="補助單位 (없는 문서)"><ruby class="hanja grade32 unknown"><rb>補助單位</rb><rp>(</rp><rt>보조단위</rt><rp>)</rp></ruby></a>로는 <a href="/index.php?title=%EC%84%BC%ED%8A%B8&amp;action=edit&amp;redlink=1" class="new" title="센트 (없는 문서)">센트</a>(￠)가있다. 1<ruby class="hanja grade20"><rb>弗</rb><rp>(</rp><rt>불</rt><rp>)</rp></ruby> = ￠100.
</p><p><ruby class="hanja grade20"><rb>弗</rb><rp>(</rp><rt>불</rt><rp>)</rp></ruby>이라는<ruby class="hanja grade40"><rb>呼稱</rb><rp>(</rp><rt>호칭</rt><rp>)</rp></ruby>은 <a grade20"><rb>美合衆國弗</rb><rp>(</rp><rt>미합중국불</rt><rp>)</rp></ruby>'을<a href="/index.php?title=%E6%8C%87%E7%A8%B1&amp;action=edit&amp;redlink=1" class="new" title="指稱 (없는 문서)"><ruby class="hanja grade40 unknown"><rb>指稱</rb><rp>(</rp><rt>지칭</rt><rp>)</rp></ruby></a>한다. <ruby class="hanja grade52"><rb>知名度</rb><rp>(</rp><rt>지명도</rt><rp>)</rp></ruby>와<ruby class="hanja grade32"><rb>信賴性</rb><rp>(</rp><rt>신뢰성</rt><rp>)</rp></ruby>, <ruby class="hanja grade50"><rb>去來量等</rb><rp>(</rp><rt>거래량등</rt><rp>)</rp></ruby>을<ruby class="hanja grade62"><rb>問題</rb><rp>(</rp><rt>문제</rt><rp>)</rp></ruby>가없을<ruby class="hanja grade42"><rb>程度</rb><rp>(</rp><rt>정도</rt><rp>)</rp></ruby>의 <ruby class="hanja
