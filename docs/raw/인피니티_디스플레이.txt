<h2><span id="槪要(개요)"></span><span class="mw-headline" id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"><ruby class="hanja grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EC%9D%B8%ED%94%BC%EB%8B%88%ED%8B%B0_%EB%94%94%EC%8A%A4%ED%94%8C%EB%A0%88%EC%9D%B4&amp;action=edit&amp;section=1" title="부분 편집: 槪要(개요)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>인피니티 디스플레이는 <a href="/index.php?title=%E4%B8%89%E6%98%9F%E9%9B%BB%E5%AD%90&amp;action=edit&amp;redlink=1" class="new" title="三星電子 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>三星電子</rb><rp>(</rp><rt>삼성전자</rt><rp>)</rp></ruby></a>가 <a href="/index.php?title=2017%E5%B9%B4&amp;action=edit&amp;redlink=1" class="new" title="2017年 (없는 문서)">2017<ruby class="hanja grade80 unknown"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby></a> <a href="/index.php?title=%EA%B0%A4%EB%9F%AD%EC%8B%9C_S8&amp;action=edit&amp;redlink=1" class="new" title="갤럭시 S8 (없는 문서)">갤럭시 S8</a>부터 <ruby class="hanja grade40"><rb>適用</rb><rp>(</rp><rt>적용</rt><rp>)</rp></ruby>한 디스플레이 <ruby class="hanja grade50"><rb>技術</rb><rp>(</rp><rt>기술</rt><rp>)</rp></ruby>이다.
</p>
<h2><span id="詳細(상세)_技術(기술)"></span><span class="mw-headline" id=".E8.A9.B3.E7.B4.B0.28.EC.83.81.EC.84.B8.29_.E6.8A.80.E8.A1.93.28.EA.B8.B0.EC.88.A0.29"><ruby class="hanja grade32"><rb>詳細</rb><rp>(</rp><rt>상세</rt><rp>)</rp></ruby> <ruby class="hanja grade50"><rb>技術</rb><rp>(</rp><rt>기술</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EC%9D%B8%ED%94%BC%EB%8B%88%ED%8B%B0_%EB%94%94%EC%8A%A4%ED%94%8C%EB%A0%88%EC%9D%B4&amp;action=edit&amp;section=2" title="부분 편집: 詳細(상세) 技術(기술)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja grade52"><rb>以前</rb><rp>(</rp><rt>이전</rt><rp>)</rp></ruby>까지 <ruby class="hanja grade42"><rb>三星</rb><rp>(</rp><rt>삼성</rt><rp>)</rp></ruby>은 <ruby class="hanja grade62"><rb>自社</rb><rp>(</rp><rt>자사</rt><rp>)</rp></ruby>의 <a href="/index.php?title=%E7%95%B5%E9%9D%A2&amp;action=edit&amp;redlink=1" class="new" title="畵面 (없는 문서)"><ruby class="hanja grade60 unknown"><rb>畵面</rb><rp>(</rp><rt>화면</rt><rp>)</rp></ruby></a> <ruby class="hanja grade50"><rb>技術</rb><rp>(</rp><rt>기술</rt><rp>)</rp></ruby>을 <a href="/index.php?title=%EC%8A%88%ED%8D%BC_%EC%95%84%EB%AA%B0%EB%A0%88%EB%93%9C&amp;action=edit&amp;redlink=1" class="new" title="슈퍼 아몰레드 (없는 문서)">슈퍼 아몰레드</a>라고 불렀으나, <a href="/index.php?title=%EA%B0%A4%EB%9F%AD%EC%8B%9C_S8&amp;action=edit&amp;redlink=1" class="new" title="갤럭시 S8 (없는 문서)">갤럭시 S8</a>부터 <ruby class="hanja grade10"><rb>搭載</rb><rp>(</rp><rt>탑재</rt><rp>)</rp></ruby>된 베젤을 <ruby class="hanja grade50"><rb>最小化</rb><rp>(</rp><rt>최소화</rt><rp>)</rp></ruby>한 <a href="/index.php?title=%EC%95%84%EB%AA%B0%EB%A0%88%EB%93%9C&amp;action=edit&amp;redlink=1" class="new" title="아몰레드 (없는 문서)">아몰레드</a> <ruby class="hanja grade60"><rb>畵面</rb><rp>(</rp><rt>화면</rt><rp>)</rp></ruby>을 인피니티 디스플레이라고 <ruby class="hanja grade40"><rb>名稱</rb><rp>(</rp><rt>명칭</rt><rp>)</rp></ruby>하기 <ruby class="hanja grade62"><rb>始作</rb><rp>(</rp><rt>시작</rt><rp>)</rp></ruby>했다.
</p><p>그 <ruby class="hanja grade72"><rb>後</rb><rp>(</rp><rt>후</rt><rp>)</rp></ruby> <a href="/index.php?title=2018%E5%B9%B4&amp;action=edit&amp;redlink=1" class="new" title="2018年 (없는 문서)">2018<ruby class="hanja grade80 unknown"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby></a><a href="/index.php?title=11%E6%9C%88&amp;action=edit&amp;redlink=1" class="new" title="11月 (없는 문서)">11<ruby class="hanja grade80 unknown"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby></a>에 위 <ruby class="hanja grade42"><rb>寫眞</rb><rp>(</rp><rt>사진</rt><rp>)</rp></ruby>과 같이 새 인피니티 디스플레이를 4<ruby class="hanja grade52"><rb>種類</rb><rp>(</rp><rt>종류</rt><rp>)</rp></ruby> <ruby class="hanja grade60"><rb>公開</rb><rp>(</rp><rt>공개</rt><rp>)</rp></ruby>하며 <a href="/index.php?title=%EB%85%B8%EC%B9%98_%EB%94%94%EC%8A%A4%ED%94%8C%EB%A0%88%EC%9D%B4&amp;action=edit&amp;redlink=1" class="new" title="노치 디스플레이 (없는 문서)">노치 디스플레이</a>를 <ruby class="hanja grade42"><rb>導入</rb><rp>(</rp><rt>도입</rt><rp>)</rp></ruby>할 것이라 <ruby class="hanja grade62"><rb>發表</rb><rp>(</rp><rt>발표</rt><rp>)</rp></ruby>했다.
</p>
<h2><span id="種類(종류)"></span><span class="mw-headline" id=".E7.A8.AE.E9.A1.9E.28.EC.A2.85.EB.A5.98.29"><ruby class="hanja grade52"><rb>種類</rb><rp>(</rp><rt>종류</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EC%9D%B8%ED%94%BC%EB%8B%88%ED%8B%B0_%EB%94%94%EC%8A%A4%ED%94%8C%EB%A0%88%EC%9D%B4&amp;action=edit&amp;section=3" title="부분 편집: 種類(종류)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span id="인피니티_U"></span><span class="mw-headline" id=".EC.9D.B8.ED.94.BC.EB.8B.88.ED.8B.B0_U">인피니티 U</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EC%9D%B8%ED%94%BC%EB%8B%88%ED%8B%B0_%EB%94%94%EC%8A%A4%ED%94%8C%EB%A0%88%EC%9D%B4&amp;action=edit&amp;section=4" title="부분 편집: 인피니티 U">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><ruby class="hanja grade60"><rb>畵面</rb><rp>(</rp><rt>화면</rt><rp>)</rp></ruby> <ruby class="hanja grade42"><rb>上端</rb><rp>(</rp><rt>상단</rt><rp>)</rp></ruby>을 U<ruby class="hanja grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby> <ruby class="hanja grade40"><rb>模樣</rb><rp>(</rp><rt>모양</rt><rp>)</rp></ruby>으로 잘라내 카메라를 넣은 <ruby class="hanja grade42"><rb>形態</rb><rp>(</rp><rt>형태</rt><rp>)</rp></ruby>이다.
</p>
<h3><span id="인피니티_V"></span><span class="mw-headline" id=".EC.9D.B8.ED.94.BC.EB.8B.88.ED.8B.B0_V">인피니티 V</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EC%9D%B8%ED%94%BC%EB%8B%88%ED%8B%B0_%EB%94%94%EC%8A%A4%ED%94%8C%EB%A0%88%EC%9D%B4&amp;action=edit&amp;section=5" title="부분 편집: 인피니티 V">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><ruby class="hanja grade60"><rb>畵面</rb><rp>(</rp><rt>화면</rt><rp>)</rp></ruby> <ruby class="hanja grade42"><rb>上端</rb><rp>(</rp><rt>상단</rt><rp>)</rp></ruby>을 V<ruby class="hanja grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby> <ruby class="hanja grade40"><rb>模樣</rb><rp>(</rp><rt>모양</rt><rp>)</rp></ruby>으로 잘라내 카메라를 넣은 <ruby class="hanja grade42"><rb>形態</rb><rp>(</rp><rt>형태</rt><rp>)</rp></ruby>이다.
</p>
<h3><span id="인피니티_O"></span><span class="mw-headline" id=".EC.9D.B8.ED.94.BC.EB.8B.88.ED.8B.B0_O">인피니티 O</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EC%9D%B8%ED%94%BC%EB%8B%88%ED%8B%B0_%EB%94%94%EC%8A%A4%ED%94%8C%EB%A0%88%EC%9D%B4&amp;action=edit&amp;section=6" title="부분 편집: 인피니티 O">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>숨기는 것이 가장 어려운 <a href="/wiki/%EC%B9%B4%EB%A9%94%EB%9D%BC" class="mw-redirect" title="카메라">카메라</a>를 <ruby class="hanja grade42"><rb>除外</rb><rp>(</rp><rt>제외</rt><rp>)</rp></ruby>한 모든 센서를 <ruby class="hanja grade60"><rb>畵面</rb><rp>(</rp><rt>화면</rt><rp>)</rp></ruby> 패널 밑으로 숨긴 <ruby class="hanja grade42"><rb>形態</rb><rp>(</rp><rt>형태</rt><rp>)</rp></ruby>로, <ruby class="hanja grade60"><rb>畵面</rb><rp>(</rp><rt>화면</rt><rp>)</rp></ruby>에는 카메라를 <ruby class="hanja grade42"><rb>爲</rb><rp>(</rp><rt>위</rt><rp>)</rp></ruby>한 구멍만이 하나 뚫려 있는 <ruby class="hanja grade40"><rb>模樣</rb><rp>(</rp><rt>모양</rt><rp>)</rp></ruby>새라 '펀치홀 디스플레이'라고도 한다.
</p><p><a href="/index.php?title=%EA%B0%A4%EB%9F%AD%EC%8B%9C_A9_%ED%94%84%EB%A1%9C&amp;action=edit&amp;redlink=1" class="new" title="갤럭시 A9 프로 (없는 문서)">갤럭시 A9 프로</a>에서 처음으로 <ruby class="hanja grade40"><rb>適用</rb><rp>(</rp><rt>적용</rt><rp>)</rp></ruby>되었고, <ruby class="hanja grade52"><rb>以後</rb><rp>(</rp><rt>이후</rt><rp>)</rp></ruby> <a href="/wiki/%EA%B0%A4%EB%9F%AD%EC%8B%9C_S10" title="갤럭시 S10">갤럭시 S10</a>과 <a href="/index.php?title=%EA%B0%A4%EB%9F%AD%EC%8B%9C_%EB%85%B8%ED%8A%B8_10&amp;action=edit&amp;redlink=1" class="new" title="갤럭시 노트 10 (없는 문서)">갤럭시 노트 10</a>에 이어 <a href="/index.php?title=%EA%B0%A4%EB%9F%AD%EC%8B%9C_S20&amp;action=edit&amp;redlink=1" class="new" title="갤럭시 S20 (없는 문서)">갤럭시 S20</a> <ruby class="hanja grade62"><rb>等等</rb><rp>(</rp><rt>등등</rt><rp>)</rp></ruby> <a href="/index.php?title=%E4%B8%BB%E5%8A%9B%E8%A3%BD%E5%93%81&amp;action=edit&amp;redlink=1" class="new" title="主力製品 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>主力製品</rb><rp>(</rp><rt>주력제품</rt><rp>)</rp></ruby></a>들에 <ruby class="hanja grade42"><rb>連</rb><rp>(</rp><rt>연</rt><rp>)</rp></ruby>달아 <ruby class="hanja grade40"><rb>適用</rb><rp>(</rp><rt>적용</rt><rp>)</rp></ruby>되고 있다. <a href="/index.php?title=%EB%89%B4_%EC%9D%B8%ED%94%BC%EB%8B%88%ED%8B%B0&amp;action=edit&amp;redlink=1" class="new" title="뉴 인피니티 (없는 문서)">뉴 인피니티</a>가 <ruby class="hanja grade30"><rb>普遍化</rb><rp>(</rp><rt>보편화</rt><rp>)</rp></ruby>되기 <ruby class="hanja grade72"><rb>前</rb><rp>(</rp><rt>전</rt><rp>)</rp></ruby>까지는 <ruby class="hanja grade42"><rb>主力製品</rb><rp>(</rp><rt>주력제품</rt><rp>)</rp></ruby>의 <ruby class="hanja grade32"><rb>特徵</rb><rp>(</rp><rt>특징</rt><rp>)</rp></ruby>으로 <ruby class="hanja grade32"><rb>維持</rb><rp>(</rp><rt>유지</rt><rp>)</rp></ruby>될 것으로 보인다.
</p>
<h3><span id="뉴_인피니티"></span><span class="mw-headline" id=".EB.89.B4_.EC.9D.B8.ED.94.BC.EB.8B.88.ED.8B.B0">뉴 인피니티</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EC%9D%B8%ED%94%BC%EB%8B%88%ED%8B%B0_%EB%94%94%EC%8A%A4%ED%94%8C%EB%A0%88%EC%9D%B4&amp;action=edit&amp;section=7" title="부분 편집: 뉴 인피니티">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>이 <ruby class="hanja grade50"><rb>技術</rb><rp>(</rp><rt>기술</rt><rp>)</rp></ruby>의 <ruby class="hanja grade40"><rb>終着點</rb><rp>(</rp><rt>종착점</rt><rp>)</rp></ruby>으로, <a href="/wiki/%EC%B9%B4%EB%A9%94%EB%9D%BC" class="mw-redirect" title="카메라">카메라</a> <ruby class="hanja grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby>의 것을 모두 <ruby class="hanja grade60"><rb>畵面</rb><rp>(</rp><rt>화면</rt><rp>)</rp></ruby> 밑으로 숨기거나 없애서 <a href="/wiki/%EC%8A%A4%EB%A7%88%ED%8A%B8%ED%8F%B0" title="스마트폰">스마트폰</a>의 <ruby class="hanja grade70"><rb>前面</rb><rp>(</rp><rt>전면</rt><rp>)</rp></ruby>에는 <ruby class="hanja grade60"><rb>畵面</rb><rp>(</rp><rt>화면</rt><rp>)</rp></ruby>만이 보이게 하는 것이다.
</p><p><a href="/index.php?title=%EA%B0%A4%EB%9F%AD%EC%8B%9C_A80&amp;action=edit&amp;redlink=1" class="new" title="갤럭시 A80 (없는 문서)">갤럭시 A80</a>에서 처음으로 <ruby class="hanja grade42"><rb>實際</rb><rp>(</rp><rt>실제</rt><rp>)</rp></ruby>로 <ruby class="hanja grade40"><rb>適用</rb><rp>(</rp><rt>적용</rt><rp>)</rp></ruby>되었다. <a href="/wiki/%EC%B9%B4%EB%A9%94%EB%9D%BC" class="mw-redirect" title="카메라">카메라</a>를 <a href="/index.php?title=%E7%95%B5%E9%9D%A2&amp;action=edit&amp;redlink=1" class="new" title="畵面 (없는 문서)"><ruby class="hanja grade60 unknown"><rb>畵面</rb><rp>(</rp><rt>화면</rt><rp>)</rp></ruby></a> 밑에 숨긴 건 아니고, <ruby class="hanja grade70"><rb>後面</rb><rp>(</rp><rt>후면</rt><rp>)</rp></ruby> 카메라가 튀어나와 뒤집어져서 <ruby class="hanja grade70"><rb>前面</rb><rp>(</rp><rt>전면</rt><rp>)</rp></ruby> 카메라의 <ruby class="hanja grade32"><rb>役割</rb><rp>(</rp><rt>역할</rt><rp>)</rp></ruby>도 하는 <ruby class="hanja grade60"><rb>式</rb><rp>(</rp><rt>식</rt><rp>)</rp></ruby>으로 <ruby class="hanja grade52"><rb>具現</rb><rp>(</rp><rt>구현</rt><rp>)</rp></ruby>된 것이다.
