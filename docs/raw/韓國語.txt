<p><ruby class="hanja grade70"><rb>韓國語</rb><rp>(</rp><rt>한국어</rt><rp>)</rp></ruby>는 <a href="/wiki/%E5%A4%A7%E9%9F%93%E6%B0%91%E5%9C%8B" title="大韓民國"><ruby class="hanja grade80"><rb>大韓民國</rb><rp>(</rp><rt>대한민국</rt><rp>)</rp></ruby></a>과 <a href="/wiki/%E6%9C%9D%E9%AE%AE%E6%B0%91%E4%B8%BB%E4%B8%BB%E7%BE%A9%E4%BA%BA%E6%B0%91%E5%85%B1%E5%92%8C%E5%9C%8B" title="朝鮮民主主義人民共和國"><ruby class="hanja grade42"><rb>朝鮮民主主義人民共和國</rb><rp>(</rp><rt>조선민주주의인민공화국</rt><rp>)</rp></ruby></a> <ruby class="hanja grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby>에서 <ruby class="hanja grade70"><rb>主</rb><rp>(</rp><rt>주</rt><rp>)</rp></ruby>로 <ruby class="hanja grade60"><rb>使用</rb><rp>(</rp><rt>사용</rt><rp>)</rp></ruby>하는 <a href="/wiki/%E8%A8%80%E8%AA%9E" title="言語"><ruby class="hanja grade60"><rb>言語</rb><rp>(</rp><rt>언어</rt><rp>)</rp></ruby></a>이다.
</p>
<h2><span id="名稱(명칭)"></span><span class="mw-headline" id=".E5.90.8D.E7.A8.B1.28.EB.AA.85.EC.B9.AD.29"><ruby class="hanja grade40"><rb>名稱</rb><rp>(</rp><rt>명칭</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E9%9F%93%E5%9C%8B%E8%AA%9E&amp;action=edit&amp;section=2" title="부분 편집: 名稱(명칭)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%E5%8C%97%E9%9F%93" title="北韓"><ruby class="hanja grade80"><rb>北韓</rb><rp>(</rp><rt>북한</rt><rp>)</rp></ruby></a>에서는 <ruby class="hanja grade52"><rb>朝鮮語</rb><rp>(</rp><rt>조선어</rt><rp>)</rp></ruby>라고 부르며, <a href="/wiki/%E4%B8%AD%E8%8F%AF%E4%BA%BA%E6%B0%91%E5%85%B1%E5%92%8C%E5%9C%8B" title="中華人民共和國"><ruby class="hanja grade40"><rb>中華人民共和國</rb><rp>(</rp><rt>중화인민공화국</rt><rp>)</rp></ruby></a>에서는 <ruby class="hanja grade52"><rb>朝鮮語</rb><rp>(</rp><rt>조선어</rt><rp>)</rp></ruby>, <ruby class="hanja grade70"><rb>韓語</rb><rp>(</rp><rt>한어</rt><rp>)</rp></ruby>라는 <ruby class="hanja grade40"><rb>名稱</rb><rp>(</rp><rt>명칭</rt><rp>)</rp></ruby>을 둘&nbsp;다 쓴다. <a href="/wiki/%E6%97%A5%E6%9C%AC" title="日本"><ruby class="hanja grade60"><rb>日本</rb><rp>(</rp><rt>일본</rt><rp>)</rp></ruby></a>에서는 <ruby class="hanja grade62"><rb>大部分</rb><rp>(</rp><rt>대부분</rt><rp>)</rp></ruby> <ruby class="hanja grade52"><rb>朝鮮語</rb><rp>(</rp><rt>조선어</rt><rp>)</rp></ruby>라고 하지만 <ruby class="hanja grade50"><rb>最近</rb><rp>(</rp><rt>최근</rt><rp>)</rp></ruby> 들어서는 <ruby class="hanja grade70"><rb>韓國語</rb><rp>(</rp><rt>한국어</rt><rp>)</rp></ruby>라고 하는 <ruby class="hanja grade40"><rb>境遇</rb><rp>(</rp><rt>경우</rt><rp>)</rp></ruby>도 많이 보인다.
</p><p><a href="/wiki/%E8%8B%B1%E8%AA%9E" title="英語"><ruby class="hanja grade60"><rb>英語</rb><rp>(</rp><rt>영어</rt><rp>)</rp></ruby></a>에서는 <a href="/index.php?title=%E9%AB%98%E9%BA%97&amp;action=edit&amp;redlink=1" class="new" title="高麗 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>高麗</rb><rp>(</rp><rt>고려</rt><rp>)</rp></ruby></a>에서<ruby class="hanja grade60"><rb>由來</rb><rp>(</rp><rt>유래</rt><rp>)</rp></ruby>한 <a href="/index.php?title=Korea&amp;action=edit&amp;redlink=1" class="new" title="Korea (없는 문서)">Korea</a>에서나온 Korean이라는 <ruby class="hanja grade40"><rb>名稱</rb><rp>(</rp><rt>명칭</rt><rp>)</rp></ruby>을 쓴다. <ruby class="hanja grade52"><rb>當然</rb><rp>(</rp><rt>당연</rt><rp>)</rp></ruby>히 ‘<a href="/index.php?title=%E9%9F%93%E5%9C%8B%E4%BA%BA&amp;action=edit&amp;redlink=1" class="new" title="韓國人 (없는 문서)"><ruby class="hanja grade80 unknown"><rb>韓國人</rb><rp>(</rp><rt>한국인</rt><rp>)</rp></ruby></a>’을 뜻하는 <ruby class="hanja grade42"><rb>單語</rb><rp>(</rp><rt>단어</rt><rp>)</rp></ruby>와 겹친다...
</p>
