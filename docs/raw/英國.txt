<p><ruby class="hanja api grade50"><rb>原語</rb><rp>(</rp><rt>원어</rt><rp>)</rp></ruby>인 <a href="/wiki/%E8%8B%B1%E8%AA%9E" title="英語"><ruby class="hanja api grade60"><rb>英語</rb><rp>(</rp><rt>영어</rt><rp>)</rp></ruby></a>로 <ruby class="hanja api grade60"><rb>公式</rb><rp>(</rp><rt>공식</rt><rp>)</rp></ruby> <ruby class="hanja api grade60"><rb>國號</rb><rp>(</rp><rt>국호</rt><rp>)</rp></ruby>는 United Kingdom of Great Britain and Northern Ireland로 <ruby class="hanja api grade40"><rb>現存</rb><rp>(</rp><rt>현존</rt><rp>)</rp></ruby> <a href="/wiki/%E5%9C%8B%E5%AE%B6" title="國家"><ruby class="hanja api grade72"><rb>國家</rb><rp>(</rp><rt>국가</rt><rp>)</rp></ruby></a>의 <ruby class="hanja api grade60"><rb>英語</rb><rp>(</rp><rt>영어</rt><rp>)</rp></ruby> <a href="/index.php?title=%E5%9C%8B%E8%99%9F&amp;action=edit&amp;redlink=1" class="new" title="國號 (없는 文書)"><ruby class="hanja api grade60 unknown"><rb>國號</rb><rp>(</rp><rt>국호</rt><rp>)</rp></ruby></a> <ruby class="hanja api grade80"><rb>中</rb><rp>(</rp><rt>중</rt><rp>)</rp></ruby>에서 가장 길다. <ruby class="hanja api grade60"><rb>國號</rb><rp>(</rp><rt>국호</rt><rp>)</rp></ruby>의 앞<ruby class="hanja api grade62"><rb>部分</rb><rp>(</rp><rt>부분</rt><rp>)</rp></ruby>을 따서 United Kingdom, <a href="/index.php?title=%E9%A0%AD%E6%96%87%E5%AD%97&amp;action=edit&amp;redlink=1" class="new" title="頭文字 (없는 文書)"><ruby class="hanja api grade60 unknown"><rb>頭文字</rb><rp>(</rp><rt>두문자</rt><rp>)</rp></ruby></a>로는 UK라고 부르는 <ruby class="hanja api grade40"><rb>境遇</rb><rp>(</rp><rt>경우</rt><rp>)</rp></ruby>가 많다.
</p><p><ruby class="hanja api grade42"><rb>統治</rb><rp>(</rp><rt>통치</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>地域</rb><rp>(</rp><rt>지역</rt><rp>)</rp></ruby>이 <ruby class="hanja api grade70"><rb>主</rb><rp>(</rp><rt>주</rt><rp>)</rp></ruby>로 <a href="/index.php?title=%E5%A4%A7%EB%B8%8C%EB%A6%AC%ED%8A%BC%EC%84%AC&amp;action=edit&amp;redlink=1" class="new" title="大브리튼섬 (없는 文書)"><ruby class="hanja api grade80 unknown"><rb>大</rb><rp>(</rp><rt>대</rt><rp>)</rp></ruby>브리튼섬</a>이기 때문에 Great Britain 또는 <ruby class="hanja api grade40"><rb>簡單</rb><rp>(</rp><rt>간단</rt><rp>)</rp></ruby>히 Britain이라고도 합니다.
</p><p><ruby class="hanja api grade50"><rb>過去</rb><rp>(</rp><rt>과거</rt><rp>)</rp></ruby> <ruby class="hanja api grade60"><rb>英國</rb><rp>(</rp><rt>영국</rt><rp>)</rp></ruby>은 <ruby class="hanja api grade60"><rb>本國</rb><rp>(</rp><rt>본국</rt><rp>)</rp></ruby>인 <ruby class="hanja api grade32"><rb>聯合</rb><rp>(</rp><rt>연합</rt><rp>)</rp></ruby><ruby class="hanja api grade80"><rb>王國</rb><rp>(</rp><rt>왕국</rt><rp>)</rp></ruby>과 <ruby class="hanja api grade40"><rb>屬領</rb><rp>(</rp><rt>속령</rt><rp>)</rp></ruby>들을 한꺼번에 다 일컫는 <ruby class="hanja api grade42"><rb>非公式的</rb><rp>(</rp><rt>비공식적</rt><rp>)</rp></ruby>인 <ruby class="hanja api grade62"><rb>用語</rb><rp>(</rp><rt>용어</rt><rp>)</rp></ruby>가 따로 있었는데 바로 <a href="/wiki/%E5%A4%A7%E8%8B%B1%E5%B8%9D%E5%9C%8B" class="mw-redirect" title="大英帝國"><ruby class="hanja api grade40"><rb>大英帝國</rb><rp>(</rp><rt>대영제국</rt><rp>)</rp></ruby></a>(British Empire)이다.
</p><p><ruby class="hanja api grade60"><rb>公式</rb><rp>(</rp><rt>공식</rt><rp>)</rp></ruby> <ruby class="hanja api grade60"><rb>國號</rb><rp>(</rp><rt>국호</rt><rp>)</rp></ruby>의 <a href="/wiki/%E9%9F%93%E5%9C%8B%E8%AA%9E" title="韓國語"><ruby class="hanja api grade70"><rb>韓國語</rb><rp>(</rp><rt>한국어</rt><rp>)</rp></ruby></a> <ruby class="hanja api grade30"><rb>飜譯</rb><rp>(</rp><rt>번역</rt><rp>)</rp></ruby>은 그레이트브리튼 및 <ruby class="hanja api grade80"><rb>北</rb><rp>(</rp><rt>북</rt><rp>)</rp></ruby>아일랜드<ruby class="hanja api grade32"><rb>聯合</rb><rp>(</rp><rt>연합</rt><rp>)</rp></ruby><ruby class="hanja api grade80"><rb>王國</rb><rp>(</rp><rt>왕국</rt><rp>)</rp></ruby>, <ruby class="hanja api grade40"><rb>通稱</rb><rp>(</rp><rt>통칭</rt><rp>)</rp></ruby>은 <a href="/wiki/%E5%9C%8B%E5%90%8D%EC%9D%98_%E6%BC%A2%E5%AD%97%E8%A1%A8%E8%A8%98" title="國名의 漢字表記"><ruby class="hanja api grade60"><rb>英國</rb><rp>(</rp><rt>영국</rt><rp>)</rp></ruby></a>이다. <a href="/wiki/%E5%9C%8B%E5%90%8D%EC%9D%98_%E6%BC%A2%E5%AD%97_%E7%95%A5%E7%A8%B1" title="國名의 漢字 略稱"><ruby class="hanja api grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby> 한 글<ruby class="hanja api grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby></a>로는 <ruby class="hanja api grade60"><rb>英</rb><rp>(</rp><rt>영</rt><rp>)</rp></ruby>으로 <ruby class="hanja api grade40"><rb>略稱</rb><rp>(</rp><rt>약칭</rt><rp>)</rp></ruby>한다.
</p>
