<h2><span id=".E7.94.A8.E8.AA.9E.28.EC.9A.A9.EC.96.B4.29"></span><span class="mw-headline" id="用語(용어)"><ruby class="hanja api grade62"><rb>用語</rb><rp>(</rp><rt>용어</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%B4%BB%E7%89%88%E5%8D%B0%E5%88%B7/%E9%91%84%E9%80%A0&amp;action=edit&amp;section=1" title="部分 編輯: 用語(용어)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<dl><dt><ruby class="hanja api grade20"><rb>母型</rb><rp>(</rp><rt>모형</rt><rp>)</rp></ruby></dt>
<dd>글<ruby class="hanja api grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>가 올바른 <ruby class="hanja api grade60"><rb>方向</rb><rp>(</rp><rt>방향</rt><rp>)</rp></ruby>으로 <ruby class="hanja api grade40"><rb>陰刻</rb><rp>(</rp><rt>음각</rt><rp>)</rp></ruby>되어있는 <ruby class="hanja api grade40"><rb>金屬</rb><rp>(</rp><rt>금속</rt><rp>)</rp></ruby>막대.</dd>
<dt><ruby class="hanja api grade20"><rb>父型</rb><rp>(</rp><rt>부형</rt><rp>)</rp></ruby></dt>
<dd>글<ruby class="hanja api grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>가 <ruby class="hanja api grade72"><rb>左右</rb><rp>(</rp><rt>좌우</rt><rp>)</rp></ruby><ruby class="hanja api grade40"><rb>反轉</rb><rp>(</rp><rt>반전</rt><rp>)</rp></ruby>으로 <ruby class="hanja api grade40"><rb>陽刻</rb><rp>(</rp><rt>양각</rt><rp>)</rp></ruby>된 <ruby class="hanja api grade70"><rb>活字</rb><rp>(</rp><rt>활자</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade40"><rb>標本</rb><rp>(</rp><rt>표본</rt><rp>)</rp></ruby>.</dd></dl>
<h2><span id=".E7.88.B6.E5.9E.8B.28.EB.B6.80.ED.98.95.29.EA.B3.BC_.E6.AF.8D.E5.9E.8B.28.EB.AA.A8.ED.98.95.29"></span><span class="mw-headline" id="父型(부형)과_母型(모형)"><ruby class="hanja api grade20"><rb>父型</rb><rp>(</rp><rt>부형</rt><rp>)</rp></ruby>과 <ruby class="hanja api grade20"><rb>母型</rb><rp>(</rp><rt>모형</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%B4%BB%E7%89%88%E5%8D%B0%E5%88%B7/%E9%91%84%E9%80%A0&amp;action=edit&amp;section=2" title="部分 編輯: 父型(부형)과 母型(모형)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja api grade70"><rb>數</rb><rp>(</rp><rt>수</rt><rp>)</rp></ruby>많은 <a href="/wiki/%E6%B4%BB%E5%AD%97" title="活字"><ruby class="hanja api grade70"><rb>活字</rb><rp>(</rp><rt>활자</rt><rp>)</rp></ruby></a>에 글<ruby class="hanja api grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>를 작은 크기로 하나하나 파낼수는 없으므로 먼저 <ruby class="hanja api grade70"><rb>活字</rb><rp>(</rp><rt>활자</rt><rp>)</rp></ruby>의 <a href="/index.php?title=%E6%A8%99%E6%9C%AC&amp;action=edit&amp;redlink=1" class="new" title="標本 (없는 文書)"><ruby class="hanja api grade40 unknown"><rb>標本</rb><rp>(</rp><rt>표본</rt><rp>)</rp></ruby></a>, <a href="/wiki/%EC%BB%B4%ED%93%A8%ED%84%B0" title="컴퓨터">컴퓨터</a>로치면 <a href="/index.php?title=%ED%8F%B0%ED%8A%B8&amp;action=edit&amp;redlink=1" class="new" title="폰트 (없는 文書)">폰트</a>에 <ruby class="hanja api grade30"><rb>該當</rb><rp>(</rp><rt>해당</rt><rp>)</rp></ruby>하는 <ruby class="hanja api grade20"><rb>父型</rb><rp>(</rp><rt>부형</rt><rp>)</rp></ruby>을 만든다. <ruby class="hanja api grade20"><rb>父型</rb><rp>(</rp><rt>부형</rt><rp>)</rp></ruby>은 <ruby class="hanja api grade50"><rb>完成</rb><rp>(</rp><rt>완성</rt><rp>)</rp></ruby>될 <ruby class="hanja api grade70"><rb>活字</rb><rp>(</rp><rt>활자</rt><rp>)</rp></ruby>와 같이 <ruby class="hanja api grade72"><rb>左右</rb><rp>(</rp><rt>좌우</rt><rp>)</rp></ruby>가 <ruby class="hanja api grade62"><rb>反對</rb><rp>(</rp><rt>반대</rt><rp>)</rp></ruby>인 글<ruby class="hanja api grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>가 <ruby class="hanja api grade40"><rb>陽刻</rb><rp>(</rp><rt>양각</rt><rp>)</rp></ruby>되어있다. 이것을 <ruby class="hanja api grade40"><rb>金屬</rb><rp>(</rp><rt>금속</rt><rp>)</rp></ruby>막대에 눌러찍거나 하여 글<ruby class="hanja api grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>가 올바른 <ruby class="hanja api grade60"><rb>方向</rb><rp>(</rp><rt>방향</rt><rp>)</rp></ruby>으로 새겨진 <ruby class="hanja api grade20"><rb>母型</rb><rp>(</rp><rt>모형</rt><rp>)</rp></ruby>을 만든다. 만들어진 <ruby class="hanja api grade20"><rb>母型</rb><rp>(</rp><rt>모형</rt><rp>)</rp></ruby>을 거푸집에 끼우고 납물을 부어 <ruby class="hanja api grade32"><rb>鑄造</rb><rp>(</rp><rt>주조</rt><rp>)</rp></ruby>하면 <ruby class="hanja api grade70"><rb>活字</rb><rp>(</rp><rt>활자</rt><rp>)</rp></ruby> 하나가 <ruby class="hanja api grade50"><rb>完成</rb><rp>(</rp><rt>완성</rt><rp>)</rp></ruby>된다.
</p><p><ruby class="hanja api grade20"><rb>父型</rb><rp>(</rp><rt>부형</rt><rp>)</rp></ruby>과 <ruby class="hanja api grade20"><rb>母型</rb><rp>(</rp><rt>모형</rt><rp>)</rp></ruby>을 거쳐 <ruby class="hanja api grade42"><rb>實際</rb><rp>(</rp><rt>실제</rt><rp>)</rp></ruby> <ruby class="hanja api grade70"><rb>活字</rb><rp>(</rp><rt>활자</rt><rp>)</rp></ruby>가 나오는 <ruby class="hanja api grade42"><rb>工程</rb><rp>(</rp><rt>공정</rt><rp>)</rp></ruby>은 <ruby class="hanja api grade62"><rb>各</rb><rp>(</rp><rt>각</rt><rp>)</rp></ruby> <ruby class="hanja api grade20"><rb>言語圈</rb><rp>(</rp><rt>언어권</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade70"><rb>文字</rb><rp>(</rp><rt>문자</rt><rp>)</rp></ruby><ruby class="hanja api grade60"><rb>使用</rb><rp>(</rp><rt>사용</rt><rp>)</rp></ruby><ruby class="hanja api grade42"><rb>實態</rb><rp>(</rp><rt>실태</rt><rp>)</rp></ruby>나 <ruby class="hanja api grade50"><rb>技術</rb><rp>(</rp><rt>기술</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade42"><rb>發達</rb><rp>(</rp><rt>발달</rt><rp>)</rp></ruby>를 <ruby class="hanja api grade40"><rb>反映</rb><rp>(</rp><rt>반영</rt><rp>)</rp></ruby>했으며, 몇가지 <ruby class="hanja api grade52"><rb>方法</rb><rp>(</rp><rt>방법</rt><rp>)</rp></ruby>이 <ruby class="hanja api grade40"><rb>存在</rb><rp>(</rp><rt>존재</rt><rp>)</rp></ruby>한다.
</p>
