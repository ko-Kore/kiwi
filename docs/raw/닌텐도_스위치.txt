<h2><span id="槪要(개요)"></span><span class="mw-headline" id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"><ruby class="hanja grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EB%8B%8C%ED%85%90%EB%8F%84_%EC%8A%A4%EC%9C%84%EC%B9%98&amp;action=edit&amp;section=1" title="부분 편집: 槪要(개요)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>닌텐도 스위치는 <a href="/index.php?title=%E4%BB%BB%E5%A4%A9%E5%A0%82&amp;action=edit&amp;redlink=1" class="new" title="任天堂 (없는 문서)"><ruby class="hanja grade52 unknown"><rb>任天堂</rb><rp>(</rp><rt>임천당</rt><rp>)</rp></ruby></a>에서 <a href="/index.php?title=2017%E5%B9%B4&amp;action=edit&amp;redlink=1" class="new" title="2017年 (없는 문서)">2017<ruby class="hanja grade80 unknown"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby></a><a href="/index.php?title=3%E6%9C%883%E6%97%A5&amp;action=edit&amp;redlink=1" class="new" title="3月3日 (없는 문서)">3<ruby class="hanja grade80 unknown"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby>3<ruby class="hanja grade80 unknown"><rb>日</rb><rp>(</rp><rt>일</rt><rp>)</rp></ruby></a>에 <ruby class="hanja grade50"><rb>發賣</rb><rp>(</rp><rt>발매</rt><rp>)</rp></ruby>한 <a href="/index.php?title=%EA%B2%8C%EC%9E%84%E6%A9%9F&amp;action=edit&amp;redlink=1" class="new" title="게임機 (없는 문서)">게임<ruby class="hanja grade40 unknown"><rb>機</rb><rp>(</rp><rt>기</rt><rp>)</rp></ruby></a>이다. <ruby class="hanja grade52"><rb>開發當時</rb><rp>(</rp><rt>개발당시</rt><rp>)</rp></ruby>에는 '닌텐도 NX'라는 <ruby class="hanja grade40"><rb>名稱</rb><rp>(</rp><rt>명칭</rt><rp>)</rp></ruby>으로 알려져있었다.
</p>
<h2><span id="特徵(특징)"></span><span class="mw-headline" id=".E7.89.B9.E5.BE.B5.28.ED.8A.B9.EC.A7.95.29"><ruby class="hanja grade32"><rb>特徵</rb><rp>(</rp><rt>특징</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EB%8B%8C%ED%85%90%EB%8F%84_%EC%8A%A4%EC%9C%84%EC%B9%98&amp;action=edit&amp;section=2" title="부분 편집: 特徵(특징)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/index.php?title=%E6%8D%AE%E7%BD%AE%E5%9E%8B%EA%B2%8C%EC%9E%84%E6%A9%9F&amp;action=edit&amp;redlink=1" class="new" title="据置型게임機 (없는 문서)"><ruby class="hanja grade0 href="/index.php?title=%E6%90%BA%E5%B8%B6%E7%94%A8%EA%B2%8C%EC%9E%84%E6%A9%9F&amp;action=edit&amp;redlink=1" class="new" title="携帶用게임機 (없는 문서)"><ruby class="hanja grade30 grade42"><rb>平素</rb><rp>(</rp><rt>평소</rt><rp>)</rp></ruby>에는 <ruby class="hanja grade30"><rb>携帶</rb><rp>(</rp><rt>휴대</rt><rp>)</rp></ruby>하며 플레이하다가, <ruby class="hanja <h2><span id="하드웨어"></span><span class="mw-headline" id=".ED.95.98.EB.93.9C.EC.9B.A8.EC.96.B4">하드웨어</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EB%8B%8C%ED%85%90%EB%8F%84_%EC%8A%A4%EC%9C%84%EC%B9%98&amp;action=edit&amp;section=3" title="부분 편집: 하드웨어">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/index.php?title=%EC%97%94%EB%B9%84%EB%94%94%EC%95%84&amp;action=edit&amp;redlink=1" class="new" title="엔비디아 (없는 문서)">엔비디아</a>의 <a href="/index.php?title=%ED%85%8C%EA%B7%B8%EB%9D%BC&amp;action=edit&amp;redlink=1" class="new" title="테그라 (없는 문서)">테그라</a> 프로세서를 <ruby class="hanja grade10"><rb>搭載</rb><rp>(</rp><rt>탑재</rt><rp>)</rp></ruby>했다. <ruby class="hanja grade30"><rb>携帶</rb><rp>(</rp><rt>휴대</rt><rp>)</rp></ruby>모드로 플레이할<ruby class="hanja grade72"><rb>時</rb><rp>(</rp><rt>시</rt><rp>)</rp></ruby> 720p <a href="/index.php?title=HD&amp;action=edit&amp;redlink=1" class="new" title="HD (없는 문서)">HD</a><ruby class="hanja grade32"><rb>解像度</rb><rp>(</rp><rt>해상도</rt><rp>)</rp></ruby>로 7인치 <ruby class="hanja grade32"><rb>內藏畵面</rb><rp>(</rp><rt>내장화면</rt><rp>)</rp></ruby>에<ruby class="hanja grade70"><rb>出力</rb><rp>(</rp><rt>출력</rt><rp>)</rp></ruby>되며, <ruby class="hanja grade0"><rb>据置</rb><rp>(</rp><rt>거치</rt><rp>)</rp></ruby>모드로는 텔레비전<ruby class="hanja grade60"><rb>畵面</rb><rp>(</rp><rt>화면</rt><rp>)</rp></ruby>으로 1080p <a href="/index.php?title=%ED%92%80_HD&amp;action=edit&amp;redlink=1" class="new" title="풀 HD (없는 문서)">풀 HD</a><ruby class="hanja grade32"><rb>映像</rb><rp>(</rp><rt>영상</rt><rp>)</rp></ruby>이 <ruby class="hanja grade70"><rb>出力</rb><rp>(</rp><rt>출력</rt><rp>)</rp></ruby>된다.
</p><p><ruby class="hanja grade40"><rb>器機下端</rb><rp>(</rp><rt>기기하단</rt><rp>)</rp></ruby>에는 <a href="/index.php?title=USB_C&amp;action=edit&amp;redlink=1" class="new" title="USB C (없는 문서)">USB C</a><ruby class="hanja grade42"><rb>端子</rb><rp>(</rp><rt>단자</rt><rp>)</rp></ruby>로 <a href="/index.php?title=%E5%85%85%E9%9B%BB%E6%A9%9F&amp;action=edit&amp;redlink=1" class="new" title="充電機 (없는 문서)"><ruby class="hanja grade40 unknown"><rb>充電機</rb><rp>(</rp><rt>충전기</rt><rp>)</rp></ruby></a> <ruby class="hanja grade40"><rb>或</rb><rp>(</rp><rt>혹</rt><rp>)</rp></ruby> <a href="/index.php?title=%EB%8F%84%ED%82%B9_%EC%8A%A4%ED%85%8C%EC%9D%B4%EC%85%98&amp;action=edit&amp;redlink=1" class="new" title="도킹 스테이션 (없는 문서)">도킹 스테이션</a>에<ruby class="hanja grade42"><rb>連結</rb><rp>(</rp><rt>연결</rt><rp>)</rp></ruby>된다. <ruby class="hanja grade72"><rb>左右</rb><rp>(</rp><rt>좌우</rt><rp>)</rp></ruby>로는 <a href="/index.php?title=%EC%A1%B0%EC%9D%B4%EC%BD%98&amp;action=edit&amp;redlink=1" class="new" title="조이콘 (없는 문서)">조이콘</a> <ruby class="hanja grade32"><rb>附着用</rb><rp>(</rp><rt>부착용</rt><rp>)</rp></ruby> 레일이 있다. <ruby class="hanja grade70"><rb>後面</rb><rp>(</rp><rt>후면</rt><rp>)</rp></ruby>에는 <ruby class="hanja grade0"><rb>据置用</rb><rp>(</rp><rt>거치용</rt><rp>)</rp></ruby> <ruby class="hanja grade32"><rb>支持臺</rb><rp>(</rp><rt>지지대</rt><rp>)</rp></ruby>가 <ruby class="hanja grade32"><rb>內藏</rb><rp>(</rp><rt>내장</rt><rp>)</rp></ruby>되어 있으며, <ruby class="hanja grade62"><rb>內部</rb><rp>(</rp><rt>내부</rt><rp>)</rp></ruby>에는 <a href="/index.php?title=%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C_SD&amp;action=edit&amp;redlink=1" class="new" title="마이크로 SD (없는 문서)">마이크로 SD</a> 리더가있다.
</p>
<h3><span id="도킹_스테이션"></span><span class="mw-headline" id=".EB.8F.84.ED.82.B9_.EC.8A.A4.ED.85.8C.EC.9D.B4.EC.85.98">도킹 스테이션</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EB%8B%8C%ED%85%90%EB%8F%84_%EC%8A%A4%EC%9C%84%EC%B9%98&amp;action=edit&amp;section=4" title="부분 편집: 도킹 스테이션">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/index.php?title=%ED%94%8C%EB%9D%BC%EC%8A%A4%ED%8B%B1&amp;action=edit&amp;redlink=1" class="new" title="플라스틱 (없는 문서)">플라스틱</a><ruby class="hanja grade52"><rb>材質</rb><rp>(</rp><rt>재질</rt><rp>)</rp></ruby>로 <ruby class="hanja grade72"><rb>中間</rb><rp>(</rp><rt>중간</rt><rp>)</rp></ruby>의홈에 <ruby class="hanja grade60"><rb>本體</rb><rp>(</rp><rt>본체</rt><rp>)</rp></ruby>를 <ruby class="hanja grade0"><rb>据置</rb><rp>(</rp><rt>거치</rt><rp>)</rp></ruby>할수있다. <ruby class="hanja grade0"><rb>据置時</rb><rp>(</rp><rt>거치시</rt><rp>)</rp></ruby>에는 <a href="/index.php?title=USB_C&amp;action=edit&amp;redlink=1" class="new" title="USB C (없는 문서)">USB C</a>포트로 <ruby class="hanja grade60"><rb>本體</rb><rp>(</rp><rt>본체</rt><rp>)</rp></ruby>가 <ruby class="hanja grade52"><rb>充電</rb><rp>(</rp><rt>충전</rt><rp>)</rp></ruby>되며, <ruby class="hanja grade70"><rb>同時</rb><rp>(</rp><rt>동시</rt><rp>)</rp></ruby>에 <a href="/index.php?title=HDMI&amp;action=edit&amp;redlink=1" class="new" title="HDMI (없는 문서)">HDMI</a>로 <ruby class="hanja grade60"><rb>畵面</rb><rp>(</rp><rt>화면</rt><rp>)</rp></ruby>이 <ruby class="hanja grade70"><rb>出力</rb><rp>(</rp><rt>출력</rt><rp>)</rp></ruby>되고 <a href="/index.php?title=USB&amp;action=edit&amp;redlink=1" class="new" title="USB (없는 문서)">USB</a>포트 2<ruby class="hanja grade42"><rb>個</rb><rp>(</rp><rt>개</rt><rp>)</rp></ruby>를 <ruby class="hanja grade62"><rb>利用</rb><rp>(</rp><rt>이용</rt><rp>)</rp></ruby>할수 있다.
</p><p><ruby class="hanja grade60"><rb>本體</rb><rp>(</rp><rt>본체</rt><rp>)</rp></ruby> <ruby class="hanja grade40"><rb>支持用</rb><rp>(</rp><rt>지지용</rt><rp>)</rp></ruby> <ruby class="hanja grade40"><rb>構造物</rb><rp>(</rp><rt>구조물</rt><rp>)</rp></ruby>을<ruby class="hanja grade42"><rb>除外</rb><rp>(</rp><rt>제외</rt><rp>)</rp></ruby>하면 그냥 <a href="/index.php?title=USB_C&amp;action=edit&amp;redlink=1" class="new" title="USB C (없는 문서)">USB C</a> <a href="/index.php?title=%ED%97%88%EB%B8%8C&amp;action=edit&amp;redlink=1" class="new" title="허브 (없는 문서)">허브</a>에 <ruby class="hanja grade52"><rb>不過</rb><rp>(</rp><rt>불과</rt><rp>)</rp></ruby>하여, <ruby class="hanja grade32"><rb>電源供給</rb><rp>(</rp><rt>전원공급</rt><rp>)</rp></ruby>이되는 <ruby class="hanja grade30"><rb>類似</rb><rp>(</rp><rt>유사</rt><rp>)</rp></ruby>한<ruby class="hanja grade40"><rb>構成</rb><rp>(</rp><rt>구성</rt><rp>)</rp></ruby>의 <ruby class="hanja grade42"><rb>製品</rb><rp>(</rp><rt>제품</rt><rp>)</rp></ruby>으로도 독의<ruby class="hanja grade40"><rb>機能</rb><rp>(</rp><rt>기능</rt><rp>)</rp></ruby>을 <ruby class="hanja grade62"><rb>代身</rb><rp>(</rp><rt>대신</rt><rp>)</rp></ruby>할수있다.
</p>
