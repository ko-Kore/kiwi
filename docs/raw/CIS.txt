<h2><span id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"></span><span class="mw-headline" id="槪要(개요)"><ruby class="hanja api grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=CIS&amp;action=edit&amp;section=1" title="部分 編輯: 槪要(개요)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja api grade52"><rb>獨立國</rb><rp>(</rp><rt>독립국</rt><rp>)</rp></ruby><ruby class="hanja api grade72"><rb>家</rb><rp>(</rp><rt>가</rt><rp>)</rp></ruby><ruby class="hanja api grade32"><rb>聯合</rb><rp>(</rp><rt>연합</rt><rp>)</rp></ruby>(<span style="font-size: smaller;"><a href="/wiki/%E8%8B%B1%E8%AA%9E" title="英語">英語</a>: </span><span lang="en">Commonwealth of Independent States</span>)은 <ruby class="hanja api grade32"><rb>蘇聯</rb><rp>(</rp><rt>소련</rt><rp>)</rp></ruby> <ruby class="hanja api grade30"><rb>崩壞</rb><rp>(</rp><rt>붕괴</rt><rp>)</rp></ruby>로 인해 <ruby class="hanja api grade52"><rb>獨立國</rb><rp>(</rp><rt>독립국</rt><rp>)</rp></ruby><ruby class="hanja api grade72"><rb>家</rb><rp>(</rp><rt>가</rt><rp>)</rp></ruby>가 된 <ruby class="hanja api grade52"><rb>舊</rb><rp>(</rp><rt>구</rt><rp>)</rp></ruby> <ruby class="hanja api grade32"><rb>蘇聯</rb><rp>(</rp><rt>소련</rt><rp>)</rp></ruby> <ruby class="hanja api grade62"><rb>共和國</rb><rp>(</rp><rt>공화국</rt><rp>)</rp></ruby>들의 <ruby class="hanja api grade32"><rb>聯合體</rb><rp>(</rp><rt>연합체</rt><rp>)</rp></ruby>로서 <ruby class="hanja api grade52"><rb>結成</rb><rp>(</rp><rt>결성</rt><rp>)</rp></ruby>된 <ruby class="hanja api grade40"><rb>國際機構</rb><rp>(</rp><rt>국제기구</rt><rp>)</rp></ruby>이자 <ruby class="hanja api grade72"><rb>國家</rb><rp>(</rp><rt>국가</rt><rp>)</rp></ruby> <ruby class="hanja api grade32"><rb>聯合</rb><rp>(</rp><rt>연합</rt><rp>)</rp></ruby>(Confederation)이다. <ruby class="hanja api grade32"><rb>蘇聯</rb><rp>(</rp><rt>소련</rt><rp>)</rp></ruby> <ruby class="hanja api grade30"><rb>崩壞</rb><rp>(</rp><rt>붕괴</rt><rp>)</rp></ruby> <ruby class="hanja api grade72"><rb>直前</rb><rp>(</rp><rt>직전</rt><rp>)</rp></ruby>인 1991<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> 12<ruby class="hanja api grade80"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby> 8<ruby class="hanja api grade80"><rb>日</rb><rp>(</rp><rt>일</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade42"><rb>設立</rb><rp>(</rp><rt>설립</rt><rp>)</rp></ruby>되었다.
</p><p><ruby class="hanja api grade40"><rb>稱</rb><rp>(</rp><rt>칭</rt><rp>)</rp></ruby><ruby class="hanja api grade42"><rb>呼</rb><rp>(</rp><rt>호</rt><rp>)</rp></ruby>은 <a href="/wiki/%E9%9C%B2%E8%A5%BF%E4%BA%9E" title="露西亞"><ruby class="hanja api grade32"><rb>露西亞</rb><rp>(</rp><rt>노서아</rt><rp>)</rp></ruby></a><ruby class="hanja api grade70"><rb>語</rb><rp>(</rp><rt>어</rt><rp>)</rp></ruby>로 СНГ(SNG), <a href="/wiki/%E8%8B%B1%E8%AA%9E" title="英語"><ruby class="hanja api grade60"><rb>英語</rb><rp>(</rp><rt>영어</rt><rp>)</rp></ruby></a>로 CIS(Commonwealth of Independent States). <a href="/wiki/%E5%8C%97%E9%9F%93" title="北韓"><ruby class="hanja api grade80"><rb>北韓</rb><rp>(</rp><rt>북한</rt><rp>)</rp></ruby></a>에서는 <ruby class="hanja api grade52"><rb>獨立國</rb><rp>(</rp><rt>독립국</rt><rp>)</rp></ruby><ruby class="hanja api grade72"><rb>家</rb><rp>(</rp><rt>가</rt><rp>)</rp></ruby><ruby class="hanja api grade42"><rb>協同體</rb><rp>(</rp><rt>협동체</rt><rp>)</rp></ruby>, <a href="/wiki/%E6%97%A5%E6%9C%AC" title="日本"><ruby class="hanja api grade60"><rb>日本</rb><rp>(</rp><rt>일본</rt><rp>)</rp></ruby></a>에서는 <ruby class="hanja api grade52"><rb>獨立國</rb><rp>(</rp><rt>독립국</rt><rp>)</rp></ruby><ruby class="hanja api grade72"><rb>家</rb><rp>(</rp><rt>가</rt><rp>)</rp></ruby><ruby class="hanja api grade62"><rb>共同體</rb><rp>(</rp><rt>공동체</rt><rp>)</rp></ruby>(<span style="font-size: smaller;"><a href="/wiki/%E6%97%A5%E6%9C%AC%E8%AA%9E" title="日本語">日本語</a>: </span><span lang="ja"><ruby><rb>獨立</rb><rp>(</rp><rt>どくりつ</rt><rp>)</rp></ruby><ruby><rb>國</rb><rp>(</rp><rt>こっ</rt><rp>)</rp></ruby><ruby><rb>家</rb><rp>(</rp><rt>か</rt><rp>)</rp></ruby><ruby><rb>共</rb><rp>(</rp><rt>きょう</rt><rp>)</rp></ruby><ruby><rb>同體</rb><rp>(</rp><rt>どうたい</rt><rp>)</rp></ruby></span>), <ruby class="hanja api grade80"><rb>中國</rb><rp>(</rp><rt>중국</rt><rp>)</rp></ruby>에서는 <ruby class="hanja api grade52"><rb>獨立國</rb><rp>(</rp><rt>독립국</rt><rp>)</rp></ruby><ruby class="hanja api grade72"><rb>家</rb><rp>(</rp><rt>가</rt><rp>)</rp></ruby><ruby class="hanja api grade32"><rb>聯合體</rb><rp>(</rp><rt>연합체</rt><rp>)</rp></ruby>(<span style="font-size: smaller;"><a href="/wiki/%E4%B8%AD%E5%9C%8B%E8%AA%9E" title="中國語">中國語</a>: </span><span lang="zh">独立国家联合体</span>)라고 부른다.
</p><p>구 소련 구성국가 15개국 중 8개국이 정회원국이다.
</p>
