<p><ruby class="hanja api grade52"><rb>獨立國</rb><rp>(</rp><rt>독립국</rt><rp>)</rp></ruby><ruby class="hanja api grade72"><rb>家</rb><rp>(</rp><rt>가</rt><rp>)</rp></ruby><ruby class="hanja api
