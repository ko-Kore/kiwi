<h2><span id="槪要(개요)"></span><span class="mw-headline" id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"><ruby class="hanja grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E9%87%9C%E7%94%B0%E9%A9%9B_(%E6%9D%B1%E6%B5%B7%E7%B7%9A)&amp;action=edit&amp;section=1" title="부분 편집: 槪要(개요)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja grade12"><rb>釜田驛</rb><rp>(</rp><rt>부전역</rt><rp>)</rp></ruby>은 <a href="/wiki/%E9%87%9C%E5%B1%B1%E5%BB%A3%E5%9F%9F%E5%B8%82" title="釜山廣域市"><ruby class="hanja grade12"><rb>釜山廣域市</rb><rp>(</rp><rt>부산광역시</rt><rp>)</rp></ruby></a><a href="/index.php?title=%E9%87%9C%E5%B1%B1%E9%8E%AD%E5%8D%80&amp;action=edit&amp;redlink=1" class="new" title="釜山鎭區 (없는 문서)"><ruby class="hanja grade12 unknown"><rb>釜山鎭區</rb><rp>(</rp><rt>부산진구</rt><rp>)</rp></ruby></a><a href="/index.php?title=%E9%87%9C%E7%94%B0%E6%B4%9E&amp;action=edit&amp;redlink=1" class="new" title="釜田洞 (없는 문서)"><ruby class="hanja grade12 unknown"><rb>釜田洞</rb><rp>(</rp><rt>부전동</rt><rp>)</rp></ruby></a>에 <ruby class="hanja grade42"><rb>位置</rb><rp>(</rp><rt>위치</rt><rp>)</rp></ruby>한 <ruby class="hanja grade62"><rb>東海線</rb><rp>(</rp><rt>동해선</rt><rp>)</rp></ruby>의 <ruby class="hanja grade32"><rb>鐵道驛</rb><rp>(</rp><rt>철도역</rt><rp>)</rp></ruby>이다.
</p>
<h2><span id="運行列車(운행렬차)"></span><span class="mw-headline" id=".E9.81.8B.E8.A1.8C.E5.88.97.E8.BB.8A.28.EC.9A.B4.ED.96.89.EB.A0.AC.EC.B0.A8.29"><ruby class="hanja grade42"><rb>運行列車</rb><rp>(</rp><rt>운행렬차</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E9%87%9C%E7%94%B0%E9%A9%9B_(%E6%9D%B1%E6%B5%B7%E7%B7%9A)&amp;action=edit&amp;section=2" title="부분 편집: 運行列車(운행렬차)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/index.php?title=%E6%B5%A6%E9%A0%85&amp;action=edit&amp;redlink=1" class="new" title="浦項 (없는 문서)"><ruby class="hanja grade32 unknown"><rb>浦項</rb><rp>(</rp><rt>포항</rt><rp>)</rp></ruby></a>이나 <a href="/index.php?title=%E6%9D%B1%E5%A4%A7%E9%82%B1&amp;action=edit&amp;redlink=1" class="new" title="東大邱 (없는 문서)"><ruby class="hanja grade12 unknown"><rb>東大邱</rb><rp>(</rp><rt>동대구</rt><rp>)</rp></ruby></a>, <a href="/index.php?title=%E9%A0%86%E5%A4%A9&amp;action=edit&amp;redlink=1" class="new" title="順天 (없는 문서)"><ruby class="hanja grade52 unknown"><rb>順天</rb><rp>(</rp><rt>순천</rt><rp>)</rp></ruby></a> <ruby class="hanja grade60"><rb>方向</rb><rp>(</rp><rt>방향</rt><rp>)</rp></ruby> <a href="/index.php?title=%E7%84%A1%E7%AA%AE%E8%8A%B1%E8%99%9F&amp;action=edit&amp;redlink=1" class="new" title="無窮花號 (없는 문서)"><ruby class="hanja grade40 unknown"><rb>無窮花號</rb><rp>(</rp><rt>무궁화호</rt><rp>)</rp></ruby></a> <ruby class="hanja grade42"><rb>列車</rb><rp>(</rp><rt>열차</rt><rp>)</rp></ruby>를 <ruby class="hanja grade62"><rb>利用</rb><rp>(</rp><rt>이용</rt><rp>)</rp></ruby>할수있다. 서울<ruby class="hanja grade60"><rb>方向</rb><rp>(</rp><rt>방향</rt><rp>)</rp></ruby> <a href="/index.php?title=ITX-%EC%83%88%EB%A7%88%EC%9D%84&amp;action=edit&amp;redlink=1" class="new" title="ITX-새마을 (없는 문서)">ITX-새마을</a>도 하루 2<ruby class="hanja grade42"><rb>回</rb><rp>(</rp><rt>회</rt><rp>)</rp></ruby> <ruby class="hanja grade50"><rb>停車</rb><rp>(</rp><rt>정차</rt><rp>)</rp></ruby>한다.
</p><p>또한, <ruby class="hanja grade12"><rb>東萊</rb><rp>(</rp><rt>동래</rt><rp>)</rp></ruby>, <ruby class="hanja grade32"><rb>海雲臺</rb><rp>(</rp><rt>해운대</rt><rp>)</rp></ruby>, <ruby class="hanja grade40"><rb>機長</rb><rp>(</rp><rt>기장</rt><rp>)</rp></ruby>을거쳐 <a href="/wiki/%E8%94%9A%E5%B1%B1%E5%BB%A3%E5%9F%9F%E5%B8%82" title="蔚山廣域市"><ruby class="hanja grade12"><rb>蔚山</rb><rp>(</rp><rt>울산</rt><rp>)</rp></ruby></a> <a href="/index.php?title=%E5%A4%AA%E5%92%8C%E6%B1%9F%E9%A9%9B&amp;action=edit&amp;redlink=1" class="new" title="太和江驛 (없는 문서)"><ruby class="hanja grade32 unknown"><rb>太和江驛</rb><rp>(</rp><rt>태화강역</rt><rp>)</rp></ruby></a>까지 <ruby class="hanja grade60"><rb>運行</rb><rp>(</rp><rt>운행</rt><rp>)</rp></ruby>하는 <a href="/index.php?title=%E6%9D%B1%E6%B5%B7%E7%B7%9A%E5%BB%A3%E5%9F%9F%E9%9B%BB%E9%90%B5&amp;action=edit&amp;redlink=1" class="new" title="東海線廣域電鐵 (없는 문서)"><ruby class="hanja grade40 unknown"><rb>東海線廣域電鐵</rb><rp>(</rp><rt>동해선광역전철</rt><rp>)</rp></ruby></a>의 <ruby class="hanja grade32"><rb>始發驛</rb><rp>(</rp><rt>시발역</rt><rp>)</rp></ruby>이다.
</p>
<h2><span id="周邊情報(주변정보)"></span><span class="mw-headline" id=".E5.91.A8.E9.82.8A.E6.83.85.E5.A0.B1.28.EC.A3.BC.EB.B3.80.EC.A0.95.EB.B3.B4.29"><ruby class="hanja grade40"><rb>周邊情報</rb><rp>(</rp><rt>주변정보</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E9%87%9C%E7%94%B0%E9%A9%9B_(%E6%9D%B1%E6%B5%B7%E7%B7%9A)&amp;action=edit&amp;section=3" title="부분 편집: 周邊情報(주변정보)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1<ruby class="hanja grade60"><rb>番出口</rb><rp>(</rp><rt>번출구</rt><rp>)</rp></ruby> 바로 앞에는 <a href="/index.php?title=%E9%87%9C%E7%94%B0%E5%9C%B0%E5%8D%80%E9%9A%8A&amp;action=edit&amp;redlink=1" class="new" title="釜田地區隊 (없는 문서)"><ruby class="hanja grade12 unknown"><rb>釜田地區隊</rb><rp>(</rp><rt>부전지구대</rt><rp>)</rp></ruby></a>와 <a href="/index.php?title=%E9%87%9C%E7%94%B0%E5%B8%82%E5%A0%B4&amp;action=edit&amp;redlink=1" class="new" title="釜田市場 (없는 문서)"><ruby class="hanja grade12 unknown"><rb>釜田市場</rb><rp>(</rp><rt>부전시장</rt><rp>)</rp></ruby></a>이 있다. 2<ruby class="hanja grade60"><rb>番出口</rb><rp>(</rp><rt>번출구</rt><rp>)</rp></ruby>로 나가서는 <ruby class="hanja grade32"><rb>徒步距離</rb><rp>(</rp><rt>도보거리</rt><rp>)</rp></ruby>에 <a href="/index.php?title=%E9%87%9C%E5%B1%B1%E5%B8%82%E6%B0%91%E5%85%AC%E5%9C%92&amp;action=edit&amp;redlink=1" class="new" title="釜山市民公園 (없는 문서)"><ruby class="hanja grade12 unknown"><rb>釜山市民公園</rb><rp>(</rp><rt>부산시민공원</rt><rp>)</rp></ruby></a>이 있다.
</p><p><ruby class="hanja grade42"><rb>中心街</rb><rp>(</rp><rt>중심가</rt><rp>)</rp></ruby>인 <a href="/index.php?title=%E8%A5%BF%E9%9D%A2&amp;action=edit&amp;redlink=1" class="new" title="西面 (없는 문서)"><ruby class="hanja grade70 unknown"><rb>西面</rb><rp>(</rp><rt>서면</rt><rp>)</rp></ruby></a>과는 <ruby class="hanja grade50"><rb>地下鐵</rb><rp>(</rp><rt>지하철</rt><rp>)</rp></ruby> 한 <ruby class="hanja grade50"><rb>停車場</rb><rp>(</rp><rt>정거장</rt><rp>)</rp></ruby> <ruby class="hanja grade32"><rb>距離</rb><rp>(</rp><rt>거리</rt><rp>)</rp></ruby>로 <ruby class="hanja grade10"><rb>曖昧</rb><rp>(</rp><rt>애매</rt><rp>)</rp></ruby>하게 떨어진 <ruby class="hanja grade42"><rb>位置</rb><rp>(</rp><rt>위치</rt><rp>)</rp></ruby>에 있다.
</p>
