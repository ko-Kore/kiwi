<h2><span id=".E5.91.BC.E7.A8.B1.28.ED.98.B8.EC.B9.AD.29"></span><span class="mw-headline" id="呼稱(호칭)"><ruby class="hanja api grade40"><rb>呼稱</rb><rp>(</rp><rt>호칭</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E4%B8%AD%E5%9C%8B%E8%AA%9E&amp;action=edit&amp;section=1" title="部分 編輯: 呼稱(호칭)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja api grade80"><rb>中國</rb><rp>(</rp><rt>중국</rt><rp>)</rp></ruby>·<ruby class="hanja api grade20"><rb>臺灣</rb><rp>(</rp><rt>대만</rt><rp>)</rp></ruby>에서는 쓰여진 <ruby class="hanja api grade70"><rb>中國語</rb><rp>(</rp><rt>중국어</rt><rp>)</rp></ruby>를 “<b><ruby class="hanja api grade70"><rb>中文</rb><rp>(</rp><rt>중문</rt><rp>)</rp></ruby></b>”, <ruby class="hanja api grade62"><rb>對話</rb><rp>(</rp><rt>대화</rt><rp>)</rp></ruby>할 때의 <ruby class="hanja api grade70"><rb>中國語</rb><rp>(</rp><rt>중국어</rt><rp>)</rp></ruby>를 “<b><ruby class="hanja api grade70"><rb>漢語</rb><rp>(</rp><rt>한어</rt><rp>)</rp></ruby></b>”라고 부르고 있다. <ruby class="hanja api grade60"><rb>例</rb><rp>(</rp><rt>예</rt><rp>)</rp></ruby>를 들어, “<ruby class="hanja api grade70"><rb>漢語</rb><rp>(</rp><rt>한어</rt><rp>)</rp></ruby>를 말한다”, “<ruby class="hanja api grade70"><rb>中文</rb><rp>(</rp><rt>중문</rt><rp>)</rp></ruby>을 쓴다”<ruby class="hanja api grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby>. 그러나, <ruby class="hanja api grade70"><rb>口語</rb><rp>(</rp><rt>구어</rt><rp>)</rp></ruby>에서는 “<ruby class="hanja api grade70"><rb>漢語</rb><rp>(</rp><rt>한어</rt><rp>)</rp></ruby>를 말한다”, “<ruby class="hanja api grade70"><rb>中文</rb><rp>(</rp><rt>중문</rt><rp>)</rp></ruby>을 말한다”, “<ruby class="hanja api grade70"><rb>中文</rb><rp>(</rp><rt>중문</rt><rp>)</rp></ruby>을 쓴다”, “<ruby class="hanja api grade70"><rb>漢語</rb><rp>(</rp><rt>한어</rt><rp>)</rp></ruby>를 쓴다”라고 하는 것처럼 <ruby class="hanja api grade60"><rb>區別</rb><rp>(</rp><rt>구별</rt><rp>)</rp></ruby>없이 <ruby class="hanja api grade60"><rb>使用</rb><rp>(</rp><rt>사용</rt><rp>)</rp></ruby>되기도 한다.
</p>
