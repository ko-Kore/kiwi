<h2><span id="槪要(개요)"></span><span class="mw-headline" id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"><ruby class="hanja grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%AD%B7%E5%8F%B2%E7%9A%84%E5%AD%97%E9%9F%B3%E5%81%87%E5%90%8D%E8%A1%A8%E8%A8%98%E6%B3%95&amp;action=edit&amp;section=1" title="부분 편집: 槪要(개요)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja grade42"><rb>歷史的字音假名表記法</rb><rp>(</rp><rt>역사적자음가명표기법</rt><rp>)</rp></ruby>(<ruby class="hanja grade30"><rb>歷史的字音假名遣</rb><rp>(</rp><rt>역사적자음가명견</rt><rp>)</rp></ruby>)은 <ruby class="hanja grade60"><rb>日本語</rb><rp>(</rp><rt>일본어</rt><rp>)</rp></ruby>에서 <ruby class="hanja grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby>의<a href="/index.php?title=%E9%9F%B3%E8%AE%80&amp;action=edit&amp;redlink=1" class="new" title="音讀 (없는 문서)"><ruby class="hanja grade62 unknown"><rb>音讀</rb><rp>(</rp><rt>음독</rt><rp>)</rp></ruby></a>을<ruby class="hanja grade62"><rb>表記</rb><rp>(</rp><rt>표기</rt><rp>)</rp></ruby>하는 <a href="/index.php?title=%E5%AD%97%E9%9F%B3%E5%81%87%E5%90%8D%E8%A1%A8%E8%A8%98%E6%B3%95&amp;action=edit&amp;redlink=1" class="new" title="字音假名表記法 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>字音假名表記法</rb><rp>(</rp><rt>자음가명표기법</rt><rp>)</rp></ruby></a><ruby class="hanja grade80"><rb>中</rb><rp>(</rp><rt>중</rt><rp>)</rp></ruby>, <a href="/index.php?title=%E6%B1%9F%E6%88%B6%E6%99%82%E4%BB%A3&amp;action=edit&amp;redlink=1" class="new" title="江戶時代 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>江戶時代</rb><rp>(</rp><rt>강호시대</rt><rp>)</rp></ruby></a>의 <a href="/index.php?title=%E6%9C%AC%E5%B1%85%E5%AE%A3%E9%95%B7&amp;action=edit&amp;redlink=1" class="new" title="本居宣長 (없는 문서)"><ruby class="hanja grade40 unknown"><rb>本居宣長</rb><rp>(</rp><rt>본거선장</rt><rp>)</rp></ruby></a>이<ruby class="hanja grade60"><rb>定</rb><rp>(</rp><rt>정</rt><rp>)</rp></ruby>한 <a href="/wiki/%E6%AD%B7%E5%8F%B2%E7%9A%84%E5%81%87%E5%90%8D%E8%A1%A8%E8%A8%98%E6%B3%95" title="歷史的假名表記法"><ruby class="hanja grade42"><rb>歷史的假名表記法</rb><rp>(</rp><rt>역사적가명표기법</rt><rp>)</rp></ruby></a>에 <ruby class="hanja grade30"><rb>該當</rb><rp>(</rp><rt>해당</rt><rp>)</rp></ruby>하는 것을 말한다.
</p>
<h2><span id="特徵(특징)"></span><span class="mw-headline" id=".E7.89.B9.E5.BE.B5.28.ED.8A.B9.EC.A7.95.29"><ruby class="hanja grade32"><rb>特徵</rb><rp>(</rp><rt>특징</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%AD%B7%E5%8F%B2%E7%9A%84%E5%AD%97%E9%9F%B3%E5%81%87%E5%90%8D%E8%A1%A8%E8%A8%98%E6%B3%95&amp;action=edit&amp;section=2" title="부분 편집: 特徵(특징)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja grade62"><rb>現代語</rb><rp>(</rp><rt>현대어</rt><rp>)</rp></ruby>에서는 <ruby class="hanja grade60"><rb>區別</rb><rp>(</rp><rt>구별</rt><rp>)</rp></ruby>되지않는 <ruby class="hanja grade62"><rb>漢字音</rb><rp>(</rp><rt>한자음</rt><rp>)</rp></ruby>의 <ruby class="hanja grade40"><rb>差異</rb><rp>(</rp><rt>차이</rt><rp>)</rp></ruby>도 <a href="/index.php?title=%E8%90%AC%E8%91%89%E9%9B%86&amp;action=edit&amp;redlink=1" class="new" title="萬葉集 (없는 문서)"><ruby class="hanja grade50 unknown"><rb>萬葉集</rb><rp>(</rp><rt>만엽집</rt><rp>)</rp></ruby></a>과 <ruby class="hanja grade80"><rb>中國</rb><rp>(</rp><rt>중국</rt><rp>)</rp></ruby><a href="/index.php?title=%E9%9F%BB%E6%9B%B8&amp;action=edit&amp;redlink=1" class="new" title="韻書 (없는 문서)"><ruby class="hanja grade32 unknown"><rb>韻書</rb><rp>(</rp><rt>운서</rt><rp>)</rp></ruby></a>의 <a href="/index.php?title=%E5%8F%8D%E5%88%87&amp;action=edit&amp;redlink=1" class="new" title="反切 (없는 문서)"><ruby class="hanja grade52 unknown"><rb>反切</rb><rp>(</rp><rt>반절</rt><rp>)</rp></ruby></a>에 <ruby class="hanja grade32"><rb>基盤</rb><rp>(</rp><rt>기반</rt><rp>)</rp></ruby>하여 <ruby class="hanja grade60"><rb>區別</rb><rp>(</rp><rt>구별</rt><rp>)</rp></ruby>한다.
</p>
<h3><span id="韓國漢字音(한국한자음)과_比較(비교)"></span><span class="mw-headline" id=".E9.9F.93.E5.9C.8B.E6.BC.A2.E5.AD.97.E9.9F.B3.28.ED.95.9C.EA.B5.AD.ED.95.9C.EC.9E.90.EC.9D.8C.29.EA.B3.BC_.E6.AF.94.E8.BC.83.28.EB.B9.84.EA.B5.90.29"><ruby class="hanja grade62"><rb>韓國漢字音</rb><rp>(</rp><rt>한국한자음</rt><rp>)</rp></ruby>과 <ruby class="hanja grade32"><rb>比較</rb><rp>(</rp><rt>비교</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%AD%B7%E5%8F%B2%E7%9A%84%E5%AD%97%E9%9F%B3%E5%81%87%E5%90%8D%E8%A1%A8%E8%A8%98%E6%B3%95&amp;action=edit&amp;section=3" title="부분 편집: 韓國漢字音(한국한자음)과 比較(비교)">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><ruby class="hanja grade52"><rb>歷史的</rb><rp>(</rp><rt>역사적</rt><rp>)</rp></ruby>인 <ruby class="hanja grade52"><rb>音價</rb><rp>(</rp><rt>음가</rt><rp>)</rp></ruby>의 <ruby class="hanja grade40"><rb>差異</rb><rp>(</rp><rt>차이</rt><rp>)</rp></ruby>를 <ruby class="hanja grade40"><rb>反映</rb><rp>(</rp><rt>반영</rt><rp>)</rp></ruby>하기 때문에 <a href="/wiki/%E9%9F%93%E5%9C%8B%E8%AA%9E" title="韓國語"><ruby class="hanja grade70"><rb>韓國語</rb><rp>(</rp><rt>한국어</rt><rp>)</rp></ruby></a> <ruby class="hanja grade62"><rb>漢字音</rb><rp>(</rp><rt>한자음</rt><rp>)</rp></ruby>과 <ruby class="hanja grade30"><rb>互換</rb><rp>(</rp><rt>호환</rt><rp>)</rp></ruby>(?)이 잘 되는 <ruby class="hanja grade40"><rb>境遇</rb><rp>(</rp><rt>경우</rt><rp>)</rp></ruby>가 많다.
