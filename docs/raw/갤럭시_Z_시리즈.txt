<h2><span id=".E5.90.8D.E7.A8.B1.28.EB.AA.85.EC.B9.AD.29"></span><span class="mw-headline" id="名稱(명칭)"><ruby class="hanja api grade40"><rb>名稱</rb><rp>(</rp><rt>명칭</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EA%B0%A4%EB%9F%AD%EC%8B%9C_Z_%EC%8B%9C%EB%A6%AC%EC%A6%88&amp;action=edit&amp;section=1" title="部分 編輯: 名稱(명칭)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>시리즈의 첫 <ruby class="hanja api grade42"><rb>製品</rb><rp>(</rp><rt>제품</rt><rp>)</rp></ruby>인 <a href="/index.php?title=%EA%B0%A4%EB%9F%AD%EC%8B%9C_%ED%8F%B4%EB%93%9C&amp;action=edit&amp;redlink=1" class="new" title="갤럭시 폴드 (없는 文書)">갤럭시 폴드</a>는 ‘Z’라는 시리즈<ruby class="hanja api grade72"><rb>名</rb><rp>(</rp><rt>명</rt><rp>)</rp></ruby>이 붙지 않았지만 <ruby class="hanja api grade72"><rb>後</rb><rp>(</rp><rt>후</rt><rp>)</rp></ruby>에 <a href="/index.php?title=%EA%B0%A4%EB%9F%AD%EC%8B%9C_Z_%ED%94%8C%EB%A6%BD&amp;action=edit&amp;redlink=1" class="new" title="갤럭시 Z 플립 (없는 文書)">갤럭시 Z 플립</a>이 <ruby class="hanja api grade70"><rb>出市</rb><rp>(</rp><rt>출시</rt><rp>)</rp></ruby>되고 나서 시리즈<ruby class="hanja api grade52"><rb>化</rb><rp>(</rp><rt>화</rt><rp>)</rp></ruby>되어 갤럭시 Z 시리즈가 <ruby class="hanja api grade30"><rb>誕生</rb><rp>(</rp><rt>탄생</rt><rp>)</rp></ruby>하게 되었다.
</p>
