<h2><span id="槪要(개요)"></span><span class="mw-headline" id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"><ruby class="hanja grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%A8%99%E6%BA%96%E4%B8%AD%E5%9C%8B%E8%AA%9E&amp;action=edit&amp;section=1" title="부분 편집: 槪要(개요)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja grade40"><rb>標準中國語</rb><rp>(</rp><rt>표준중국어</rt><rp>)</rp></ruby>는 <a href="/wiki/%E4%B8%AD%E5%9C%8B%E8%AA%9E" title="中國語"><ruby class="hanja grade70"><rb>中國語</rb><rp>(</rp><rt>중국어</rt><rp>)</rp></ruby></a> <a href="/index.php?title=%E5%AE%98%E8%A9%B1&amp;action=edit&amp;redlink=1" class="new" title="官話 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>官話</rb><rp>(</rp><rt>관화</rt><rp>)</rp></ruby></a><a href="/index.php?title=%E6%96%B9%E8%A8%80&amp;action=edit&amp;redlink=1" class="new" title="方言 (없는 문서)"><ruby class="hanja grade60 unknown"><rb>方言</rb><rp>(</rp><rt>방언</rt><rp>)</rp></ruby></a>의 <ruby class="hanja grade52"><rb>一種</rb><rp>(</rp><rt>일종</rt><rp>)</rp></ruby>이다.
</p>
<h2><span id="名稱(명칭)"></span><span class="mw-headline" id=".E5.90.8D.E7.A8.B1.28.EB.AA.85.EC.B9.AD.29"><ruby class="hanja grade40"><rb>名稱</rb><rp>(</rp><rt>명칭</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%A8%99%E6%BA%96%E4%B8%AD%E5%9C%8B%E8%AA%9E&amp;action=edit&amp;section=2" title="부분 편집: 名稱(명칭)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>부르는 <ruby class="hanja grade40"><rb>名稱</rb><rp>(</rp><rt>명칭</rt><rp>)</rp></ruby>은 나라마다 다르다. <a href="/wiki/%E4%B8%AD%E8%8F%AF%E4%BA%BA%E6%B0%91%E5%85%B1%E5%92%8C%E5%9C%8B" title="中華人民共和國"><ruby class="hanja grade40"><rb>中華人民共和國</rb><rp>(</rp><rt>중화인민공화국</rt><rp>)</rp></ruby></a>에서는 <span class="noruby"><ruby><rb><a href="/wiki/%E6%99%AE%E9%80%9A%E8%A9%B1" title="普通話"><ruby class="hanja grade40"><rb>普通話</rb><rp>(</rp><rt>보통화</rt><rp>)</rp></ruby></a></rb><rp>(</rp><rt>Pǔtōnghuà</rt><rp>)</rp></ruby></span>, <a href="/wiki/%E4%B8%AD%E8%8F%AF%E6%B0%91%E5%9C%8B" title="中華民國"><ruby class="hanja grade40"><rb>中華民國</rb><rp>(</rp><rt>중화민국</rt><rp>)</rp></ruby></a>에서는 <span class="noruby"><ruby><rb><a href="/wiki/%E5%9C%8B%E8%AA%9E" title="國語"><ruby class="hanja grade70"><rb>國語</rb><rp>(</rp><rt>국어</rt><rp>)</rp></ruby></a></rb><rp>(</rp><rt>ㄍㄨㄛˊ ㄩˇ</rt><rp>)</rp></ruby></span> 또는 <span class="noruby"><ruby><rb><a href="/index.php?title=%E8%8F%AF%E8%AA%9E&amp;action=edit&amp;redlink=1" class="new" title="華語 (없는 문서)"><ruby class="hanja grade40 unknown"><rb>華語</rb><rp>(</rp><rt>화어</rt><rp>)</rp></ruby></a></rb><rp>(</rp><rt>ㄏㄨㄚˊ ㄩˇ</rt><rp>)</rp></ruby></span>, <ruby class="hanja grade42"><rb>香港</rb><rp>(</rp><rt>향항</rt><rp>)</rp></ruby>이나 말레이시아에서는 <span class="noruby"><ruby><rb><ruby class="hanja grade40"><rb>華語</rb><rp>(</rp><rt>화어</rt><rp>)</rp></ruby></rb><rp>(</rp><rt>Huáyǔ</rt><rp>)</rp></ruby></span>라고 <ruby class="hanja grade40"><rb>稱</rb><rp>(</rp><rt>칭</rt><rp>)</rp></ruby>한다.
</p>
<h2><span id="特徵(특징)"></span><span class="mw-headline" id=".E7.89.B9.E5.BE.B5.28.ED.8A.B9.EC.A7.95.29"><ruby class="hanja grade32"><rb>特徵</rb><rp>(</rp><rt>특징</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%A8%99%E6%BA%96%E4%B8%AD%E5%9C%8B%E8%AA%9E&amp;action=edit&amp;section=3" title="부분 편집: 特徵(특징)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span id="聲調(성조)"></span><span class="mw-headline" id=".E8.81.B2.E8.AA.BF.28.EC.84.B1.EC.A1.B0.29"><ruby class="hanja grade42"><rb>聲調</rb><rp>(</rp><rt>성조</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%A8%99%E6%BA%96%E4%B8%AD%E5%9C%8B%E8%AA%9E&amp;action=edit&amp;section=4" title="부분 편집: 聲調(성조)">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
