<h2><span id=".E5.9C.8B.E5.AE.B6.28.EA.B5.AD.EA.B0.80.29"></span><span class="mw-headline" id="國家(국가)"><ruby class="hanja grade72"><rb>國家</rb><rp>(</rp><rt>국가</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%ED%99%80%EB%A1%9C%EB%9D%BC%EC%9D%B4%EB%B8%8C&amp;action=edit&amp;section=1" title="부분 편집: 國家(국가)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>홀로라이브는 <a href="/wiki/%E6%97%A5%E6%9C%AC" title="日本"><ruby class="hanja grade60"><rb>日本</rb><rp>(</rp><rt>일본</rt><rp>)</rp></ruby></a>, <a href="/index.php?title=%EC%9D%B8%EB%8F%84%EB%84%A4%EC%8B%9C%EC%95%84&amp;action=edit&amp;redlink=1" class="new" title="인도네시아 (없는 문서)">인도네시아</a>, <a href="/wiki/%E8%8B%B1%E8%AA%9E" title="英語">EN</a><ruby class="hanja grade20"><rb>圈</rb><rp>(</rp><rt>권</rt><rp>)</rp></ruby> <ruby class="hanja grade42"><rb>總</rb><rp>(</rp><rt>총</rt><rp>)</rp></ruby> 3<ruby class="hanja grade42"><rb>個</rb><rp>(</rp><rt>개</rt><rp>)</rp></ruby>가 <ruby class="hanja grade40"><rb>存在</rb><rp>(</rp><rt>존재</rt><rp>)</rp></ruby>한다. <ruby class="hanja grade52"><rb>元來</rb><rp>(</rp><rt>원래</rt><rp>)</rp></ruby>는 <a href="/wiki/%E4%B8%AD%E5%9C%8B" title="中國"><ruby class="hanja grade80"><rb>中國</rb><rp>(</rp><rt>중국</rt><rp>)</rp></ruby></a>도 <ruby class="hanja grade40"><rb>存在</rb><rp>(</rp><rt>존재</rt><rp>)</rp></ruby>했으나 <a href="/index.php?title=%E6%A1%90%E7%94%9F%EC%BD%94%EC%BD%94&amp;action=edit&amp;redlink=1" class="new" title="桐生코코 (없는 문서)"><ruby class="hanja grade20 unknown"><rb>桐生</rb><rp>(</rp><rt>동생</rt><rp>)</rp></ruby>코코</a>, <a href="/index.php?title=%E8%B5%A4%E4%BA%95%ED%95%98%EC%95%84%ED%86%A0&amp;action=edit&amp;redlink=1" class="new" title="赤井하아토 (없는 문서)"><ruby class="hanja grade32 unknown"><rb>赤井</rb><rp>(</rp><rt>적정</rt><rp>)</rp></ruby>하아토</a> <ruby class="hanja grade20"><rb>臺灣言及事件</rb><rp>(</rp><rt>대만언급사건</rt><rp>)</rp></ruby>으로 <ruby class="hanja grade30"><rb>契約破棄</rb><rp>(</rp><rt>계약파기</rt><rp>)</rp></ruby>가 됐다.
</p>
