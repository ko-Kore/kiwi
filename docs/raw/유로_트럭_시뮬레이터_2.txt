</p>

<p>처음에는 <ruby class="hanja api grade62"><rb>會社</rb><rp>(</rp><rt>회사</rt><rp>)</rp></ruby>는 있지만 <a href="/index.php?title=%ED%8A%B8%EB%9F%AD&amp;action=edit&amp;redlink=1" class="new" title="트럭 (없는 文書)">트럭</a>이 없어 ‘빠른 일거리’를 받아 <ruby class="hanja api grade72"><rb>每</rb><rp>(</rp><rt>매</rt><rp>)</rp></ruby> <ruby class="hanja api grade50"><rb>件</rb><rp>(</rp><rt>건</rt><rp>)</rp></ruby>마다 <ruby class="hanja api grade32"><rb>提供</rb><rp>(</rp><rt>제공</rt><rp>)</rp></ruby>해주는 트럭을 빌려 <ruby class="hanja api grade60"><rb>使用</rb><rp>(</rp><rt>사용</rt><rp>)</rp></ruby>하게 된다. 이 때는 <ruby class="hanja api grade42"><rb>訪問</rb><rp>(</rp><rt>방문</rt><rp>)</rp></ruby>한 적이 있는 <ruby class="hanja api grade50"><rb>都市</rb><rp>(</rp><rt>도시</rt><rp>)</rp></ruby> 어디서든 <ruby class="hanja api grade62"><rb>始作</rb><rp>(</rp><rt>시작</rt><rp>)</rp></ruby>할 수 있지만, 첫 트럭을 <ruby class="hanja api grade20"><rb>購買</rb><rp>(</rp><rt>구매</rt><rp>)</rp></ruby>한 <ruby class="hanja api grade72"><rb>後</rb><rp>(</rp><rt>후</rt><rp>)</rp></ruby>에는 먼저 <ruby class="hanja api grade62"><rb>出發地</rb><rp>(</rp><rt>출발지</rt><rp>)</rp></ruby><ruby class="hanja api grade40"><rb>點</rb><rp>(</rp><rt>점</rt><rp>)</rp></ruby>까지 <ruby class="hanja api grade42"><rb>移動</rb><rp>(</rp><rt>이동</rt><rp>)</rp></ruby>해서 <ruby class="hanja api grade42"><rb>貨物</rb><rp>(</rp><rt>화물</rt><rp>)</rp></ruby>을 받아야 한다.
