<p><ruby class="hanja api grade80"><rb>韓中</rb><rp>(</rp><rt>한중</rt><rp>)</rp></ruby><ruby class="hanja api grade80"><rb>日</rb><rp>(</rp><rt>일</rt><rp>)</rp></ruby>에서 <a href="/wiki/%E6%BC%A2%E5%AD%97" title="漢字"><ruby class="hanja api grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby></a> <ruby class="hanja api grade0"><rb>丶</rb><rp>(</rp><rt>주</rt><rp>)</rp></ruby>에서 <ruby class="hanja api grade60"><rb>由來</rb><rp>(</rp><rt>유래</rt><rp>)</rp></ruby>한 <ruby class="hanja api grade32"><rb>文章符號</rb><rp>(</rp><rt>문장부호</rt><rp>)</rp></ruby> [、]를 <ruby class="hanja api grade60"><rb>使用</rb><rp>(</rp><rt>사용</rt><rp>)</rp></ruby>하지만 <ruby class="hanja api grade62"><rb>各</rb><rp>(</rp><rt>각</rt><rp>)</rp></ruby> 쓰임이 다르다.

<h3><span id=".E9.9F.93.E5.9C.8B.E8.AA.9E.28.ED.95.9C.EA.B5.AD.EC.96.B4.29"></span><span class="mw-headline" id="韓國語(한국어)"><ruby class="hanja api grade70"><rb>韓國語</rb><rp>(</rp><rt>한국어</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E4%B8%B6&amp;action=edit&amp;section=2" title="部分　編輯：韓國語(한국어)">編輯</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><ruby class="hanja api grade32"><rb>縱書</rb><rp>(</rp><rt>종서</rt><rp>)</rp></ruby>일 때 <a href="/index.php?title=%EC%89%BC%ED%91%9C&amp;action=edit&amp;redlink=1" class="new" title="쉼표(없는　文書）">쉼표</a>로 쓰였다. 그러나 <ruby class="hanja api grade30"><rb>該當</rb><rp>(</rp><rt>해당</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>規範</rb><rp>(</rp><rt>규범</rt><rp>)</rp></ruby>은 2015<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> <ruby class="hanja api grade52"><rb>以後</rb><rp>(</rp><rt>이후</rt><rp>)</rp></ruby> <a href="/wiki/%E6%A9%AB%E6%9B%B8" title="橫書"><ruby class="hanja api grade32"><rb>橫書</rb><rp>(</rp><rt>횡서</rt><rp>)</rp></ruby></a>가 <ruby class="hanja api grade30"><rb>普遍化</rb><rp>(</rp><rt>보편화</rt><rp>)</rp></ruby>됨에 따라 <ruby class="hanja api grade32"><rb>削除</rb><rp>(</rp><rt>삭제</rt><rp>)</rp></ruby>되었다.
</p>
<h3><span id=".E4.B8.AD.E5.9C.8B.E8.AA.9E.28.EC.A4.91.EA.B5.AD.EC.96.B4.29"></span><span class="mw-headline" id="中國語(중국어)"><ruby class="hanja api grade70"><rb>中國語</rb><rp>(</rp><rt>중국어</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E4%B8%B6&amp;action=edit&amp;section=3" title="部分　編輯：中國語(중국어)">編輯</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><ruby class="hanja api grade52"><rb>同類</rb><rp>(</rp><rt>동류</rt><rp>)</rp></ruby>의 것들을 <ruby class="hanja api grade42"><rb>羅列</rb><rp>(</rp><rt>나렬</rt><rp>)</rp></ruby>할 때 쓰인다. <ruby class="hanja api grade70"><rb>韓國語</rb><rp>(</rp><rt>한국어</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade40"><rb>規範</rb><rp>(</rp><rt>규범</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade32"><rb>中央</rb><rp>(</rp><rt>중앙</rt><rp>)</rp></ruby><ruby class="hanja api grade40"><rb>點</rb><rp>(</rp><rt>점</rt><rp>)</rp></ruby>[·]과 <ruby class="hanja api grade30"><rb>類似</rb><rp>(</rp><rt>유사</rt><rp>)</rp></ruby>하지만 더 넓은 <ruby class="hanja api grade42"><rb>意味</rb><rp>(</rp><rt>의미</rt><rp>)</rp></ruby>로 쓰인다.
<a href="/wiki/%E9%A6%99%E6%B8%AF" title="香港"><ruby class="hanja api grade42"><rb>香港</rb><rp>(</rp><rt>향항</rt><rp>)</rp></ruby></a> 및 <a href="/wiki/%E8%87%BA%E7%81%A3" class="mw-redirect" title="臺灣"><ruby class="hanja api grade20"><rb>臺灣</rb><rp>(</rp><rt>대만</rt><rp>)</rp></ruby></a><ruby class="hanja api grade60"><rb>式</rb><rp>(</rp><rt>식</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>規範</rb><rp>(</rp><rt>규범</rt><rp>)</rp></ruby>에서도 마찬가지다.
