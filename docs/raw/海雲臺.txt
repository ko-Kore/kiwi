<h2><span id="槪要(개요)"></span><span class="mw-headline" id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"><ruby class="hanja grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%B5%B7%E9%9B%B2%E8%87%BA&amp;action=edit&amp;section=1" title="부분 편집: 槪要(개요)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja grade32"><rb>海雲臺</rb><rp>(</rp><rt>해운대</rt><rp>)</rp></ruby>는 <a href="/wiki/%E5%A4%A7%E9%9F%93%E6%B0%91%E5%9C%8B" title="大韓民國"><ruby class="hanja grade80"><rb>大韓民國</rb><rp>(</rp><rt>대한민국</rt><rp>)</rp></ruby></a> <a href="/wiki/%E9%87%9C%E5%B1%B1%E5%BB%A3%E5%9F%9F%E5%B8%82" title="釜山廣域市"><ruby class="hanja grade12"><rb>釜山廣域市</rb><rp>(</rp><rt>부산광역시</rt><rp>)</rp></ruby></a> <a href="/wiki/%E6%B5%B7%E9%9B%B2%E8%87%BA%E5%8D%80" title="海雲臺區"><ruby class="hanja grade32"><rb>海雲臺區</rb><rp>(</rp><rt>해운대구</rt><rp>)</rp></ruby></a> <ruby class="hanja grade42"><rb>一帶</rb><rp>(</rp><rt>일대</rt><rp>)</rp></ruby>와 <ruby class="hanja grade32"><rb>關聯</rb><rp>(</rp><rt>관련</rt><rp>)</rp></ruby>된 <ruby class="hanja grade70"><rb>地名</rb><rp>(</rp><rt>지명</rt><rp>)</rp></ruby>이다.
</p>
<h2><span id="由來(유래)"></span><span class="mw-headline" id=".E7.94.B1.E4.BE.86.28.EC.9C.A0.EB.9E.98.29"><ruby class="hanja grade60"><rb>由來</rb><rp>(</rp><rt>유래</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%B5%B7%E9%9B%B2%E8%87%BA&amp;action=edit&amp;section=2" title="부분 편집: 由來(유래)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/index.php?title=%E6%96%B0%E7%BE%85%E6%99%82%E4%BB%A3&amp;action=edit&amp;redlink=1" class="new" title="新羅時代 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>新羅時代</rb><rp>(</rp><rt>신라시대</rt><rp>)</rp></ruby></a> <a href="/index.php?title=%E5%B4%94%E8%87%B4%E9%81%A0&amp;action=edit&amp;redlink=1" class="new" title="崔致遠 (없는 문서)"><ruby class="hanja grade12 unknown"><rb>崔致遠</rb><rp>(</rp><rt>최치원</rt><rp>)</rp></ruby></a>이 <ruby class="hanja grade62"><rb>自身</rb><rp>(</rp><rt>자신</rt><rp>)</rp></ruby>의 <ruby class="hanja grade60"><rb>號</rb><rp>(</rp><rt>호</rt><rp>)</rp></ruby>인 ‘<ruby class="hanja grade52"><rb>海雲</rb><rp>(</rp><rt>해운</rt><rp>)</rp></ruby>’을 따서 붙인 이름이라고 <ruby class="hanja grade52"><rb>傳</rb><rp>(</rp><rt>전</rt><rp>)</rp></ruby>해진다.
</p>
<h2><span id="範圍(범위)"></span><span class="mw-headline" id=".E7.AF.84.E5.9C.8D.28.EB.B2.94.EC.9C.84.29"><ruby class="hanja grade40"><rb>範圍</rb><rp>(</rp><rt>범위</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%B5%B7%E9%9B%B2%E8%87%BA&amp;action=edit&amp;section=3" title="부분 편집: 範圍(범위)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>‘<ruby class="hanja grade32"><rb>海雲臺</rb><rp>(</rp><rt>해운대</rt><rp>)</rp></ruby>’라하면 넓은<ruby class="hanja grade42"><rb>意味</rb><rp>(</rp><rt>의미</rt><rp>)</rp></ruby>로는 <a href="/wiki/%E6%B5%B7%E9%9B%B2%E8%87%BA%E5%8D%80" title="海雲臺區"><ruby class="hanja grade32"><rb>海雲臺區</rb><rp>(</rp><rt>해운대구</rt><rp>)</rp></ruby></a> <ruby class="hanja grade62"><rb>全體</rb><rp>(</rp><rt>전체</rt><rp>)</rp></ruby>, <ruby class="hanja grade40"><rb>或</rb><rp>(</rp><rt>혹</rt><rp>)</rp></ruby>은 <ruby class="hanja grade32"><rb>海雲臺區</rb><rp>(</rp><rt>해운대구</rt><rp>)</rp></ruby>를 넘어서 <a href="/index.php?title=%E6%B0%B4%E7%87%9F%E5%8D%80&amp;action=edit&amp;redlink=1" class="new" title="水營區 (없는 문서)"><ruby class="hanja grade40 unknown"><rb>水營區</rb><rp>(</rp><rt>수영구</rt><rp>)</rp></ruby></a>, <a href="/index.php?title=%E5%8D%97%E5%8D%80_(%E9%87%9C%E5%B1%B1%E5%BB%A3%E5%9F%9F%E5%B8%82)&amp;action=edit&amp;redlink=1" class="new" title="南區 (釜山廣域市) (없는 문서)"><ruby class="hanja grade60 unknown"><rb>南區</rb><rp>(</rp><rt>남구</rt><rp>)</rp></ruby></a>까지 가리키기도 하지만, <ruby class="hanja grade62"><rb>大體</rb><rp>(</rp><rt>대체</rt><rp>)</rp></ruby>로는 <a href="/index.php?title=%E6%B5%B7%E9%9B%B2%E8%87%BA%E6%B5%B7%E6%B0%B4%E6%B5%B4%E5%A0%B4&amp;action=edit&amp;redlink=1" class="new" title="海雲臺海水浴場 (없는 문서)"><ruby class="hanja grade32 unknown"><rb>海雲臺海水浴場</rb><rp>(</rp><rt>해운대해수욕장</rt><rp>)</rp></ruby></a>과 그 <ruby class="hanja grade42"><rb>一帶</rb><rp>(</rp><rt>일대</rt><rp>)</rp></ruby>를 <ruby class="hanja grade40"><rb>指稱</rb><rp>(</rp><rt>지칭</rt><rp>)</rp></ruby>한다. <ruby class="hanja grade50"><rb>他地人</rb><rp>(</rp><rt>타지인</rt><rp>)</rp></ruby>이 말하는 ‘<ruby class="hanja grade32"><rb>海雲臺</rb><rp>(</rp><rt>해운대</rt><rp>)</rp></ruby>’가 바로 <ruby class="hanja grade60"><rb>後者</rb><rp>(</rp><rt>후자</rt><rp>)</rp></ruby>의 <ruby class="hanja grade40"><rb>境遇</rb><rp>(</rp><rt>경우</rt><rp>)</rp></ruby>.
</p>
