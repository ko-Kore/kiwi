<html><head></head><body><div class="mw-parser-output"><p><b><ruby class="hanja grade42"><rb>現代假名表記法</rb><rp>(</rp><rt>현대가명표기법</rt><rp>)</rp></ruby></b>은 <a href="/wiki/%E7%AC%AC%E4%BA%8C%E6%AC%A1%E4%B8%96%E7%95%8C%E5%A4%A7%E6%88%B0" title="第二次世界大戰"><ruby class="hanja grade42"><rb>第二次世界大戰</rb><rp>(</rp><rt>제이차세계대전</rt><rp>)</rp></ruby></a><ruby class="hanja grade72"><rb>後</rb><rp>(</rp><rt>후</rt><rp>)</rp></ruby>에 널리 쓰이기 <ruby class="hanja grade62"><rb>始作</rb><rp>(</rp><rt>시작</rt><rp>)</rp></ruby>한 <a href="/wiki/%E6%97%A5%E6%9C%AC%E8%AA%9E" title="日本語"><ruby class="hanja grade60"><rb>日本語</rb><rp>(</rp><rt>일본어</rt><rp>)</rp></ruby></a>의 <a href="/wiki/%E5%81%87%E5%90%8D%E6%96%87%E5%AD%97" title="假名文字"><ruby class="hanja grade42"><rb>假名</rb><rp>(</rp><rt>가명</rt><rp>)</rp></ruby></a><ruby class="hanja grade52"><rb>表記法</rb><rp>(</rp><rt>표기법</rt><rp>)</rp></ruby>이다.
</p>
<h2><span id="特徵(특징)"></span><span class="mw-headline" id=".E7.89.B9.E5.BE.B5.28.ED.8A.B9.EC.A7.95.29"><ruby class="hanja grade32"><rb>特徵</rb><rp>(</rp><rt>특징</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E7%8F%BE%E4%BB%A3%E5%81%87%E5%90%8D%E8%A1%A8%E8%A8%98%E6%B3%95&amp;action=edit&amp;section=1" title="부분 편집: 特徵(특징)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja grade42"><rb>單語</rb><rp>(</rp><rt>단어</rt><rp>)</rp></ruby>의 <ruby class="hanja grade40"><rb>語源</rb><rp>(</rp><rt>어원</rt><rp>)</rp></ruby>을 살려 <ruby class="hanja grade62"><rb>表記</rb><rp>(</rp><rt>표기</rt><rp>)</rp></ruby>하는 <a href="/wiki/%E6%AD%B7%E5%8F%B2%E7%9A%84%E5%81%87%E5%90%8D%E8%A1%A8%E8%A8%98%E6%B3%95" title="歷史的假名表記法"><ruby class="hanja grade42"><rb>歷史的假名表記法</rb><rp>(</rp><rt>역사적가명표기법</rt><rp>)</rp></ruby></a>과는 달리 <ruby class="hanja grade62"><rb>現代語</rb><rp>(</rp><rt>현대어</rt><rp>)</rp></ruby>에 가까운 <a href="/index.php?title=%E8%A1%A8%E9%9F%B3%E4%B8%BB%E7%BE%A9&amp;action=edit&amp;redlink=1" class="new" title="表音主義 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>表音主義</rb><rp>(</rp><rt>표음주의</rt><rp>)</rp></ruby></a>를 <ruby class="hanja grade32"><rb>基盤</rb><rp>(</rp><rt>기반</rt><rp>)</rp></ruby>으로 하는 <ruby class="hanja grade52"><rb>表記法</rb><rp>(</rp><rt>표기법</rt><rp>)</rp></ruby>이다. <ruby class="hanja grade32"><rb>助詞</rb><rp>(</rp><rt>조사</rt><rp>)</rp></ruby>로 쓰이는 <ruby class="hanja grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby> 몇몇 <ruby class="hanja grade40"><rb>境遇</rb><rp>(</rp><rt>경우</rt><rp>)</rp></ruby>를 <ruby class="hanja grade42"><rb>除外</rb><rp>(</rp><rt>제외</rt><rp>)</rp></ruby>하면 <ruby class="hanja grade70"><rb>口語</rb><rp>(</rp><rt>구어</rt><rp>)</rp></ruby> <ruby class="hanja grade62"><rb>發音</rb><rp>(</rp><rt>발음</rt><rp>)</rp></ruby>을 그대로 <ruby class="hanja grade62"><rb>表記</rb><rp>(</rp><rt>표기</rt><rp>)</rp></ruby>한다.
</p><p>이에 따라 같은 <ruby class="hanja grade62"><rb>發音</rb><rp>(</rp><rt>발음</rt><rp>)</rp></ruby>이 나지만 <a href="/wiki/%E6%AD%B7%E5%8F%B2%E7%9A%84%E5%81%87%E5%90%8D%E8%A1%A8%E8%A8%98%E6%B3%95" title="歷史的假名表記法"><ruby class="hanja grade42"><rb>歷史的假名表記法</rb><rp>(</rp><rt>역사적가명표기법</rt><rp>)</rp></ruby></a>에서는 <ruby class="hanja grade52"><rb>表記法</rb><rp>(</rp><rt>표기법</rt><rp>)</rp></ruby>이 <ruby class="hanja grade60"><rb>區別</rb><rp>(</rp><rt>구별</rt><rp>)</rp></ruby>되던 글<ruby class="hanja grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>들이 <ruby class="hanja grade42"><rb>統合</rb><rp>(</rp><rt>통합</rt><rp>)</rp></ruby>되었다.
</p>
<ul><li>い, ひ, ゐ → い</li>
<li>う, ふ → う</li>
<li>え, へ, ゑ → え</li>
<li>お, ほ, を → お</li></ul>
<p>또, <ruby class="hanja grade52"><rb>表記法</rb><rp>(</rp><rt>표기법</rt><rp>)</rp></ruby>이 <ruby class="hanja grade42"><rb>統合</rb><rp>(</rp><rt>통합</rt><rp>)</rp></ruby>됨에 따라 ゐ와 ゑ는 쓰이지 않게 되었다.
</p>

<h2><span id="其他(기타)"></span><span class="mw-headline" id=".E5.85.B6.E4.BB.96.28.EA.B8.B0.ED.83.80.29"><ruby class="hanja grade32"><rb>其他</rb><rp>(</rp><rt>기타</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E7%8F%BE%E4%BB%A3%E5%81%87%E5%90%8D%E8%A1%A8%E8%A8%98%E6%B3%95&amp;action=edit&amp;section=3" title="부분 편집: 其他(기타)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
