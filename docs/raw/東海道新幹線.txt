<h2><span id="槪要(개요)"></span><span class="mw-headline" id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"><ruby class="hanja grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%9D%B1%E6%B5%B7%E9%81%93%E6%96%B0%E5%B9%B9%E7%B7%9A&amp;action=edit&amp;section=1" title="부분 편집: 槪要(개요)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja grade32"><rb>東海道新幹線</rb><rp>(</rp><rt>동해도신간선</rt><rp>)</rp></ruby>(とうかいどうしんかんせん)은 <a href="/index.php?title=%E6%9D%B1%E4%BA%AC%E9%A9%9B&amp;action=edit&amp;redlink=1" class="new" title="東京驛 (없는 문서)"><ruby class="hanja grade32 unknown"><rb>東京驛</rb><rp>(</rp><rt>동경역</rt><rp>)</rp></ruby></a>에서 <a href="/index.php?title=%E6%96%B0%E5%A4%A7%E9%98%AA%E9%A9%9B&amp;action=edit&amp;redlink=1" class="new" title="新大阪驛 (없는 문서)"><ruby class="hanja grade12 unknown"><rb>新大阪驛</rb><rp>(</rp><rt>신대판역</rt><rp>)</rp></ruby></a>까지를 잇는 <a href="/index.php?title=JR%E6%9D%B1%E6%B5%B7&amp;action=edit&amp;redlink=1" class="new" title="JR東海 (없는 문서)">JR<ruby class="hanja grade72 unknown"><rb>東海</rb><rp>(</rp><rt>동해</rt><rp>)</rp></ruby></a>의 <ruby class="hanja grade50"><rb>高速鐵道</rb><rp>(</rp><rt>고속철도</rt><rp>)</rp></ruby> <ruby class="hanja grade60"><rb>路線</rb><rp>(</rp><rt>노선</rt><rp>)</rp></ruby>이다. <a href="/wiki/%E6%96%B0%E5%B9%B9%E7%B7%9A" title="新幹線"><ruby class="hanja grade32"><rb>新幹線</rb><rp>(</rp><rt>신간선</rt><rp>)</rp></ruby></a>의 첫 <ruby class="hanja grade60"><rb>路線</rb><rp>(</rp><rt>노선</rt><rp>)</rp></ruby>이자 <ruby class="hanja grade50"><rb>世界最初</rb><rp>(</rp><rt>세계최초</rt><rp>)</rp></ruby>의 <ruby class="hanja grade50"><rb>高速鐵道</rb><rp>(</rp><rt>고속철도</rt><rp>)</rp></ruby> <ruby class="hanja grade60"><rb>路線</rb><rp>(</rp><rt>노선</rt><rp>)</rp></ruby>이기도 하다.
</p>
<h2><span id="特徵(특징)"></span><span class="mw-headline" id=".E7.89.B9.E5.BE.B5.28.ED.8A.B9.EC.A7.95.29"><ruby class="hanja grade32"><rb>特徵</rb><rp>(</rp><rt>특징</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%9D%B1%E6%B5%B7%E9%81%93%E6%96%B0%E5%B9%B9%E7%B7%9A&amp;action=edit&amp;section=2" title="부분 편집: 特徵(특징)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%E9%AB%98%E9%80%9F%E9%90%B5%E9%81%93" title="高速鐵道"><ruby class="hanja grade50"><rb>高速鐵道</rb><rp>(</rp><rt>고속철도</rt><rp>)</rp></ruby></a>라고는 해도 꽤 옛날에 만들어진 <ruby class="hanja grade60"><rb>路線</rb><rp>(</rp><rt>노선</rt><rp>)</rp></ruby>이라 <ruby class="hanja grade60"><rb>線路</rb><rp>(</rp><rt>선로</rt><rp>)</rp></ruby> <ruby class="hanja grade50"><rb>改量</rb><rp>(</rp><rt>개량</rt><rp>)</rp></ruby>에 <ruby class="hanja grade50"><rb>改量</rb><rp>(</rp><rt>개량</rt><rp>)</rp></ruby>을 거듭하여 <ruby class="hanja grade50"><rb>最高速度</rb><rp>(</rp><rt>최고속도</rt><rp>)</rp></ruby>는 285km/h이다. 처음부터 300km/h <ruby class="hanja grade52"><rb>以上</rb><rp>(</rp><rt>이상</rt><rp>)</rp></ruby>으로 달릴 수 있게 <ruby class="hanja grade42"><rb>設計</rb><rp>(</rp><rt>설계</rt><rp>)</rp></ruby>된 <a href="/index.php?title=%E4%BA%AC%E9%87%9C%E9%AB%98%E9%80%9F%E7%B7%9A&amp;action=edit&amp;redlink=1" class="new" title="京釜高速線 (없는 문서)"><ruby class="hanja grade12 unknown"><rb>京釜高速線</rb><rp>(</rp><rt>경부고속선</rt><rp>)</rp></ruby></a>과는 <ruby class="hanja grade40"><rb>差異</rb><rp>(</rp><rt>차이</rt><rp>)</rp></ruby>가 나는 <ruby class="hanja grade62"><rb>部分</rb><rp>(</rp><rt>부분</rt><rp>)</rp></ruby>.
</p>
<h2><span id="歷史(역사)"></span><span class="mw-headline" id=".E6.AD.B7.E5.8F.B2.28.EC.97.AD.EC.82.AC.29"><ruby class="hanja grade52"><rb>歷史</rb><rp>(</rp><rt>역사</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%9D%B1%E6%B5%B7%E9%81%93%E6%96%B0%E5%B9%B9%E7%B7%9A&amp;action=edit&amp;section=3" title="부분 편집: 歷史(역사)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>그 <ruby class="hanja grade52"><rb>歷史</rb><rp>(</rp><rt>역사</rt><rp>)</rp></ruby>는 <a href="/wiki/%E7%AC%AC%E4%BA%8C%E6%AC%A1%E4%B8%96%E7%95%8C%E5%A4%A7%E6%88%B0" title="第二次世界大戰"><ruby class="hanja grade42"><rb>第二次世界大戰</rb><rp>(</rp><rt>제이차세계대전</rt><rp>)</rp></ruby></a>때의 <a href="/index.php?title=%E5%BD%88%E4%B8%B8%E5%88%97%E8%BB%8A%E8%A8%88%E5%8A%83&amp;action=edit&amp;redlink=1" class="new" grade52"><rb>現實化</rb><rp>(</rp><rt>현실화</rt><rp>)</rp></ruby>된 것은 <a href="/index.php?title=1950%E5%B9%B4%E4%BB%A3&amp;action=edit&amp;redlink=1" class="new" title="1950年代 (없는 문서)">1950<ruby grade42"><rb>限界</rb><rp>(</rp><rt>한계</rt><rp>)</rp></ruby>에 <ruby class="hanja grade42"><rb>達</rb><rp>(</rp><rt>달</rt><rp>)</rp></ruby>한 <a href="/index.php?title=%E6%9D%B1%E6%B5%B7%E9%81%93%E6%9C%AC%E7%B7%9A&amp;action=edit&amp;redlink=1" class="new" title="東海道本線 (없는 문서)"><ruby class="hanja grade60 unknown"><rb>東海道本線</rb><rp>(</rp><rt>동해도본선</rt><rp>)</rp></ruby></a>을 <a href="/wiki/%E8%A4%87%E8%A4%87%E7%B7%9A" title="複複線"><ruby class="hanja grade30"><rb>彈丸列車計劃</rb><rp>(</rp><rt>탄환열차계획</rt><rp>)</rp></ruby>을 되살려 <a href="/index.php?title=%E6%A8%99%E6%BA%96%E8%BB%8C&amp;action=edit&amp;redlink=1" class="new" title="標準軌 (없는 문서)"><ruby grade52"><rb>決定</rb><rp>(</rp><rt>결정</rt><rp>)</rp></ruby>되고, <a href="/index.php?title=1959%E5%B9%B4&amp;action=edit&amp;redlink=1" class="new" title="1959年 (없는 문서)">1959<ruby class="hanja grade80
