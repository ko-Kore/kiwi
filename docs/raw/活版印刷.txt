<h2><span id=".E5.90.8D.E7.A8.B1.28.EB.AA.85.EC.B9.AD.29"></span><span class="mw-headline" id="名稱(명칭)"><ruby class="hanja api grade40"><rb>名稱</rb><rp>(</rp><rt>명칭</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%B4%BB%E7%89%88%E5%8D%B0%E5%88%B7&amp;action=edit&amp;section=1" title="部分 編輯: 名稱(명칭)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja api grade60"><rb>英語</rb><rp>(</rp><rt>영어</rt><rp>)</rp></ruby>로는 ‘letterpress’라고 하며, <a href="/wiki/%E6%97%A5%E6%9C%AC%E8%AA%9E" title="日本語"><ruby class="hanja api grade60"><rb>日本語</rb><rp>(</rp><rt>일본어</rt><rp>)</rp></ruby></a>나 그로부터 <ruby class="hanja api grade32"><rb>影響</rb><rp>(</rp><rt>영향</rt><rp>)</rp></ruby>을 받은 <a href="/wiki/%E9%9F%93%E5%9C%8B%E8%AA%9E" title="韓國語"><ruby class="hanja api grade70"><rb>韓國語</rb><rp>(</rp><rt>한국어</rt><rp>)</rp></ruby></a>에서는 ‘<ruby class="hanja api grade32"><rb>活版印刷</rb><rp>(</rp><rt>활판인쇄</rt><rp>)</rp></ruby>’라고 한다. <ruby class="hanja api grade80"><rb>韓國</rb><rp>(</rp><rt>한국</rt><rp>)</rp></ruby>에서 <ruby class="hanja api grade60"><rb>英語</rb><rp>(</rp><rt>영어</rt><rp>)</rp></ruby><ruby class="hanja api grade40"><rb>名稱</rb><rp>(</rp><rt>명칭</rt><rp>)</rp></ruby>인 ‘레터프레스’는 <ruby class="hanja api grade70"><rb>活字</rb><rp>(</rp><rt>활자</rt><rp>)</rp></ruby>로 <ruby class="hanja api grade32"><rb>版</rb><rp>(</rp><rt>판</rt><rp>)</rp></ruby>을 짜서 <ruby class="hanja api grade32"><rb>印刷</rb><rp>(</rp><rt>인쇄</rt><rp>)</rp></ruby>하는 <ruby class="hanja api grade42"><rb>傳統的</rb><rp>(</rp><rt>전통적</rt><rp>)</rp></ruby>인 <ruby class="hanja api grade32"><rb>活版印刷</rb><rp>(</rp><rt>활판인쇄</rt><rp>)</rp></ruby>이 아닌 <a href="/index.php?title=%E9%8A%85%E7%89%88&amp;action=edit&amp;redlink=1" class="new" title="銅版 (없는 文書)"><ruby class="hanja api grade32 unknown"><rb>銅版</rb><rp>(</rp><rt>동판</rt><rp>)</rp></ruby></a>이나 <a href="/index.php?title=%E6%A8%B9%E8%84%82%E7%89%88&amp;action=edit&amp;redlink=1" class="new" title="樹脂版 (없는 文書)"><ruby class="hanja api grade20 grade32"><rb>印刷</rb><rp>(</rp><rt>인쇄</rt><rp>)</rp></ruby>를 <ruby class="hanja api grade40"><rb>指稱</rb><rp>(</rp><rt>지칭</rt><rp>)</rp></ruby>하는 <ruby class="hanja api grade40"><rb>境遇</rb><rp>(</rp><rt>경우</rt><rp>)</rp></ruby>가 많고, <a href="/wiki/%E6%B4%BB%E5%AD%97" title="活字"><ruby class="hanja api grade70"><rb>活字</rb><rp>(</rp><rt>활자</rt><rp>)</rp></ruby></a>를 <ruby class="hanja api grade62"><rb>利用</rb><rp>(</rp><rt>이용</rt><rp>)</rp></ruby>한 <ruby class="hanja api grade42"><rb>傳統的</rb><rp>(</rp><rt>전통적</rt><rp>)</rp></ruby>인 <ruby class="hanja api grade32"><rb>印刷</rb><rp>(</rp><rt>인쇄</rt><rp>)</rp></ruby><ruby class="hanja api grade60"><rb>方式</rb><rp>(</rp><rt>방식</rt><rp>)</rp></ruby>만 <ruby class="hanja api grade32"><rb>活版印刷</rb><rp>(</rp><rt>활판인쇄</rt><rp>)</rp></ruby>라고 한다. 그런데 <ruby class="hanja api grade20"><rb>英語圈</rb><rp>(</rp><rt>영어권</rt><rp>)</rp></ruby>에서는 <ruby class="hanja api grade70"><rb>活字</rb><rp>(</rp><rt>활자</rt><rp>)</rp></ruby>로 <ruby class="hanja api grade32"><rb>版</rb><rp>(</rp><rt>판</rt><rp>)</rp></ruby>을 짜든, <a href="/wiki/%EC%BB%B4%ED%93%A8%ED%84%B0" </p><p>그<ruby class="hanja api grade80"><rb>外</rb><rp>(</rp><rt>외</rt><rp>)</rp></ruby> <ruby class="hanja api grade32"><rb>活版印刷</rb><rp>(</rp><rt>활판인쇄</rt><rp>)</rp></ruby>와 <ruby class="hanja api grade32"><rb>關聯</rb><rp>(</rp><rt>관련</rt><rp>)</rp></ruby>된 <ruby class="hanja api grade62"><rb>用語</rb><rp>(</rp><rt>용어</rt><rp>)</rp></ruby>나 <ruby class="hanja api grade30"><rb>飜譯語</rb><rp>(</rp><rt>번역어</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade62"><rb>對</rb><rp>(</rp><rt>대</rt><rp>)</rp></ruby>해서는 <a href="/wiki/%E6%B4%BB%E7%89%88%E5%8D%B0%E5%88%B7/%E7%94%A8%E8%AA%9E" title="活版印刷/用語"><ruby class="hanja api grade32"><rb>活版印刷</rb><rp>(</rp><rt>활판인쇄</rt><rp>)</rp></ruby>/<ruby class="hanja api grade62"><rb>用語</rb><rp>(</rp><rt>용어</rt><rp>)</rp></ruby></a>를 <ruby class="hanja api grade50"><rb>參考</rb><rp>(</rp><rt>참고</rt><rp>)</rp></ruby>.
</p>
