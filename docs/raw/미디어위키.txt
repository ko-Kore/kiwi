<h3><span id="技術仕樣(기술사양)"></span><span class="mw-headline" id=".E6.8A.80.E8.A1.93.E4.BB.95.E6.A8.A3.28.EA.B8.B0.EC.88.A0.EC.82.AC.EC.96.91.29"><ruby class="hanja grade40"><rb>技術仕樣</rb><rp>(</rp><rt>기술사양</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EB%AF%B8%EB%94%94%EC%96%B4%EC%9C%84%ED%82%A4&amp;action=edit&amp;section=2" title="부분 편집: 技術仕樣(기술사양)">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>메인 프로그램은 <a href="/index.php?title=PHP&amp;action=edit&amp;redlink=1" class="new" title="PHP (없는 문서)">PHP</a>로 <ruby class="hanja grade62"><rb>作成</rb><rp>(</rp><rt>작성</rt><rp>)</rp></ruby>되었고, <a href="/index.php?title=%EB%8D%B0%EC%9D%B4%ED%84%B0%EB%B2%A0%EC%9D%B4%EC%8A%A4&amp;action=edit&amp;redlink=1" class="new" title="데이터베이스 (없는 문서)">데이터베이스</a> 백엔드로 <a href="/index.php?title=SQLite&amp;action=edit&amp;redlink=1" class="new" title="SQLite (없는 문서)">SQLite</a> 및 <a href="/index.php?title=MySQL&amp;action=edit&amp;redlink=1" class="new" title="MySQL (없는 문서)">MySQL</a>, <a href="/index.php?title=PostgreSQL&amp;action=edit&amp;redlink=1" class="new" title="PostgreSQL (없는 문서)">PostgreSQL</a> <ruby class="hanja grade80"><rb>中</rb><rp>(</rp><rt>중</rt><rp>)</rp></ruby> 하나를 <ruby class="hanja grade40"><rb>選擇</rb><rp>(</rp><rt>선택</rt><rp>)</rp></ruby>하여 <ruby class="hanja grade42"><rb>設置</rb><rp>(</rp><rt>설치</rt><rp>)</rp></ruby>할 수 있다.
</p>
<h2><span id="使用(사용)"></span><span class="mw-headline" id=".E4.BD.BF.E7.94.A8.28.EC.82.AC.EC.9A.A9.29"><ruby class="hanja grade60"><rb>使用</rb><rp>(</rp><rt>사용</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EB%AF%B8%EB%94%94%EC%96%B4%EC%9C%84%ED%82%A4&amp;action=edit&amp;section=3" title="부분 편집: 使用(사용)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja grade40"><rb>機能</rb><rp>(</rp><rt>기능</rt><rp>)</rp></ruby>에 매우 <ruby class="hanja grade52"><rb>充實</rb><rp>(</rp><rt>충실</rt><rp>)</rp></ruby>하며 <ruby class="hanja grade30"><rb>擴張性</rb><rp>(</rp><rt>확장성</rt><rp>)</rp></ruby>도 높은 데다가, <a href="/index.php?title=%EC%9C%84%ED%82%A4%E7%99%BE%E7%A7%91&amp;action=edit&amp;redlink=1" class="new" title="위키百科 (없는 문서)">위키<ruby class="hanja grade62 unknown"><rb>百科</rb><rp>(</rp><rt>백과</rt><rp>)</rp></ruby></a>를 <ruby class="hanja grade40"><rb>運營</rb><rp>(</rp><rt>운영</rt><rp>)</rp></ruby>하는 <a href="/index.php?title=%EC%9C%84%ED%82%A4%EB%AF%B8%EB%94%94%EC%96%B4_%EC%9E%AC%EB%8B%A8&amp;action=edit&amp;redlink=1" class="new" title="위키미디어 재단 (없는 문서)">위키미디어 재단</a>의 백도 있어서 가장 넓은 <ruby class="hanja grade40"><rb>使用者層</rb><rp>(</rp><rt>사용자층</rt><rp>)</rp></ruby>을 <ruby class="hanja grade42"><rb>保有</rb><rp>(</rp><rt>보유</rt><rp>)</rp></ruby>한 <a href="/index.php?title=%EC%9C%84%ED%82%A4_%EC%97%94%EC%A7%84&amp;action=edit&amp;redlink=1" class="new" title="위키 엔진 (없는 문서)">위키 엔진</a>이다. <ruby class="hanja grade40"><rb>大規模</rb><rp>(</rp><rt>대규모</rt><rp>)</rp></ruby> 위키의 <ruby class="hanja grade60"><rb>大多數</rb><rp>(</rp><rt>대다수</rt><rp>)</rp></ruby>가 이걸 <ruby class="hanja grade60"><rb>使用</rb><rp>(</rp><rt>사용</rt><rp>)</rp></ruby>한다.
</p>
