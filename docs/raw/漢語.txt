<h2><span id="日本語(일본어)"></span><span class="mw-headline" id=".E6.97.A5.E6.9C.AC.E8.AA.9E.28.EC.9D.BC.EB.B3.B8.EC.96.B4.29"><ruby class="hanja grade60"><rb>日本語</rb><rp>(</rp><rt>일본어</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%BC%A2%E8%AA%9E&amp;action=edit&amp;section=3" title="부분 편집: 日本語(일본어)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%E6%97%A5%E6%9C%AC%E8%AA%9E" title="日本語"><ruby class="hanja grade60"><rb>日本語</rb><rp>(</rp><rt>일본어</rt><rp>)</rp></ruby></a>의 낱말<ruby class="hanja grade80"><rb>中</rb><rp>(</rp><rt>중</rt><rp>)</rp></ruby> <a href="/index.php?title=%E5%8F%A4%E4%BB%A3%E4%B8%AD%E5%9C%8B%E8%AA%9E&amp;action=edit&amp;redlink=1" class="new" title="古代中國語 (없는 문서)"><ruby class="hanja grade60 unknown"><rb>古代中國語</rb><rp>(</rp><rt>고대중국어</rt><rp>)</rp></ruby></a>에서 <ruby class="hanja grade40"><rb>採用</rb><rp>(</rp><rt>채용</rt><rp>)</rp></ruby>된 것으로, <ruby class="hanja grade70"><rb>主</rb><rp>(</rp><rt>주</rt><rp>)</rp></ruby>로 <a href="/index.php?title=%E9%9F%B3%E8%AE%80&amp;action=edit&amp;redlink=1" class="new" title="音讀 (없는 문서)"><ruby class="hanja grade62 unknown"><rb>音讀</rb><rp>(</rp><rt>음독</rt><rp>)</rp></ruby></a>하는 <a href="/index.php?title=%E5%96%AE%E8%AA%9E&amp;action=edit&amp;redlink=1" class="new" title="單語 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>單語</rb><rp>(</rp><rt>단어</rt><rp>)</rp></ruby></a>를 일컫는다.
</p><p><ruby class="hanja grade60"><rb>日本語</rb><rp>(</rp><rt>일본어</rt><rp>)</rp></ruby>의 <a href="/wiki/%E5%9B%BA%E6%9C%89%E8%AA%9E" title="固有語"><ruby class="hanja grade50"><rb>固有語</rb><rp>(</rp><rt>고유어</rt><rp>)</rp></ruby></a>를 뜻하는 <a href="/index.php?title=%E5%92%8C%E8%AA%9E&amp;action=edit&amp;redlink=1" class="new" title="和語 (없는 문서)"><ruby class="hanja grade62 unknown"><rb>和語</rb><rp>(</rp><rt>화어</rt><rp>)</rp></ruby></a>나, <a href="/wiki/%E6%BC%A2%E5%AD%97" title="漢字"><ruby class="hanja grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby></a>가 아닌 <a href="/index.php?title=%E5%80%9F%E7%94%A8%E8%AA%9E&amp;action=edit&amp;redlink=1" class="new" title="借用語 (없는 문서)"><ruby class="hanja grade32 unknown"><rb>借用語</rb><rp>(</rp><rt>차용어</rt><rp>)</rp></ruby></a>를 뜻하는 <a href="/index.php?title=%E5%A4%96%E4%BE%86%E8%AA%9E&amp;action=edit&amp;redlink=1" class="new" title="外來語 (없는 문서)"><ruby class="hanja grade70 unknown"><rb>外來語</rb><rp>(</rp><rt>외래어</rt><rp>)</rp></ruby></a>와는 <ruby class="hanja grade62"><rb>對立</rb><rp>(</rp><rt>대립</rt><rp>)</rp></ruby>되는 <ruby class="hanja grade32"><rb>槪念</rb><rp>(</rp><rt>개념</rt><rp>)</rp></ruby>이다. 또한 <a href="/wiki/%E9%9F%93%E5%9C%8B%E8%AA%9E" title="韓國語"><ruby class="hanja grade70"><rb>韓國語</rb><rp>(</rp><rt>한국어</rt><rp>)</rp></ruby></a>의 <a href="/wiki/%E6%BC%A2%E5%AD%97%E8%AA%9E" title="漢字語"><ruby class="hanja grade70"><rb>漢字語</rb><rp>(</rp><rt>한자어</rt><rp>)</rp></ruby></a>에 <ruby class="hanja grade42"><rb>對應</rb><rp>(</rp><rt>대응</rt><rp>)</rp></ruby>되는 <ruby class="hanja grade32"><rb>槪念</rb><rp>(</rp><rt>개념</rt><rp>)</rp></ruby>이기도 하다.
