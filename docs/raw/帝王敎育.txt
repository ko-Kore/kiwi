<p><ruby class="hanja grade40"><rb>帝王敎育</rb><rp>(</rp><rt>제왕교육</rt><rp>)</rp></ruby>은 <a href="/index.php?title=Omoi&amp;action=edit&amp;redlink=1" class="new" title="Omoi (없는 문서)">Omoi</a>가 <ruby class="hanja grade32"><rb>作詞</rb><rp>(</rp><rt>작사</rt><rp>)</rp></ruby>·<ruby class="hanja grade50"><rb>作曲</rb><rp>(</rp><rt>작곡</rt><rp>)</rp></ruby>해 <a href="/index.php?title=%EC%9C%A0%ED%8A%9C%EB%B8%8C&amp;action=edit&amp;redlink=1" class="new" title="유튜브 (없는 문서)">유튜브</a>와 <a href="/wiki/%EB%8B%88%EC%BD%94%EB%8B%88%EC%BD%94%E5%8B%95%E7%95%B5" title="니코니코動畵">니코니코<ruby class="hanja grade60"><rb>動畵</rb><rp>(</rp><rt>동화</rt><rp>)</rp></ruby></a>에 <ruby class="hanja grade32"><rb>投稿</rb><rp>(</rp><rt>투고</rt><rp>)</rp></ruby>한 <a href="/index.php?title=%EB%B3%B4%EC%BB%AC%EB%A1%9C%EC%9D%B4%EB%93%9C_%EC%98%A4%EB%A6%AC%EC%A7%80%EB%84%90_%E6%9B%B2&amp;action=edit&amp;redlink=1" class="new" title="보컬로이드 오리지널 曲 (없는 문서)">보컬로이드 오리지널 <ruby class="hanja grade50 unknown"><rb>曲</rb><rp>(</rp><rt>곡</rt><rp>)</rp></ruby></a>으로, Omoi의 21<ruby class="hanja grade60"><rb>番</rb><rp>(</rp><rt>번</rt><rp>)</rp></ruby>째 <ruby class="hanja grade52"><rb>作品</rb><rp>(</rp><rt>작품</rt><rp>)</rp></ruby>이다.
</p>
<h2><span id="詳細情報(상세정보)"></span><span class="mw-headline" id=".E8.A9.B3.E7.B4.B0.E6.83.85.E5.A0.B1.28.EC.83.81.EC.84.B8.EC.A0.95.EB.B3.B4.29"><ruby class="hanja grade32"><rb>詳細情報</rb><rp>(</rp><rt>상세정보</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E5%B8%9D%E7%8E%8B%E6%95%8E%E8%82%B2&amp;action=edit&amp;section=1" title="부분 편집: 詳細情報(상세정보)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>보컬은 <a href="/index.php?title=%E5%88%9D%E9%9F%B3%EB%AF%B8%EC%BF%A0&amp;action=edit&amp;redlink=1" class="new" title="初音미쿠 (없는 문서)"><ruby class="hanja grade50 unknown"><rb>初音</rb><rp>(</rp><rt>초음</rt><rp>)</rp></ruby>미쿠</a>, 코러스는 <a href="/index.php?title=%E9%8F%A1%E9%9F%B3%EB%A6%B0&amp;action=edit&amp;redlink=1" class="new" title="鏡音린 (없는 문서)"><ruby class="hanja grade40 unknown"><rb>鏡音</rb><rp>(</rp><rt>경음</rt><rp>)</rp></ruby>린</a>을 <ruby class="hanja grade60"><rb>使用</rb><rp>(</rp><rt>사용</rt><rp>)</rp></ruby>하였다.
</p><p>노래 <ruby class="hanja grade72"><rb>中間</rb><rp>(</rp><rt>중간</rt><rp>)</rp></ruby>에 <ruby class="hanja grade32"><rb>映像</rb><rp>(</rp><rt>영상</rt><rp>)</rp></ruby>과 함께 <ruby class="hanja grade50"><rb>曲調</rb><rp>(</rp><rt>곡조</rt><rp>)</rp></ruby>가 <ruby class="hanja grade32"><rb>突變</rb><rp>(</rp><rt>돌변</rt><rp>)</rp></ruby>하는 <ruby class="hanja grade62"><rb>部分</rb><rp>(</rp><rt>부분</rt><rp>)</rp></ruby>이 두곳 있다.<del>한<ruby class="hanja grade50"><rb>曲</rb><rp>(</rp><rt>곡</rt><rp>)</rp></ruby>안에 세<ruby class="hanja grade50"><rb>曲</rb><rp>(</rp><rt>곡</rt><rp>)</rp></ruby>이들어있는 느낌</del> <ruby class="hanja grade52"><rb>音調</rb><rp>(</rp><rt>음조</rt><rp>)</rp></ruby>와 <ruby class="hanja grade40"><rb>拍子</rb><rp>(</rp><rt>박자</rt><rp>)</rp></ruby>를 <ruby class="hanja grade20"><rb>歪曲</rb><rp>(</rp><rt>왜곡</rt><rp>)</rp></ruby>하는 <ruby class="hanja grade52"><rb>效果</rb><rp>(</rp><rt>효과</rt><rp>)</rp></ruby>를 넣은 <ruby class="hanja grade62"><rb>部分</rb><rp>(</rp><rt>부분</rt><rp>)</rp></ruby>도 여럿 들어있다.
</p><p>MV에 <a href="/index.php?title=%E6%B6%88%E5%A4%B1%E9%BB%9E&amp;action=edit&amp;redlink=1" class="new" title="消失點 (없는 문서)"><ruby class="hanja grade40 unknown"><rb>消失點</rb><rp>(</rp><rt>소실점</rt><rp>)</rp></ruby></a>을 <ruby class="hanja grade32"><rb>中央</rb><rp>(</rp><rt>중앙</rt><rp>)</rp></ruby>에 둬서 <ruby class="hanja grade60"><rb>遠近感</rb><rp>(</rp><rt>원근감</rt><rp>)</rp></ruby>이 두드러지도록 찍은 <ruby class="hanja grade42"><rb>寫眞</rb><rp>(</rp><rt>사진</rt><rp>)</rp></ruby>이 <ruby class="hanja grade42"><rb>背景</rb><rp>(</rp><rt>배경</rt><rp>)</rp></ruby>으로 여럿 나오는데, 그<ruby class="hanja grade80"><rb>中</rb><rp>(</rp><rt>중</rt><rp>)</rp></ruby> <a href="/wiki/%ED%85%8C%EC%98%A4_(Omoi)" title="테오 (Omoi)">테오</a>에 쓰인 이미지도 있다!
</p>
