<p><b>CJK<ruby class="hanja grade42"><rb>統合漢字</rb><rp>(</rp><rt>통합한자</rt><rp>)</rp></ruby></b>는 <a href="/wiki/CJK" title="CJK">CJK</a>로 알려진 <a href="/wiki/%E4%B8%AD%E5%9C%8B%E8%AA%9E" title="中國語"><ruby class="hanja grade70"><rb>中國語</rb><rp>(</rp><rt>중국어</rt><rp>)</rp></ruby></a>·<a href="/wiki/%E6%97%A5%E6%9C%AC%E8%AA%9E" title="日本語"><ruby class="hanja grade60"><rb>日本語</rb><rp>(</rp><rt>일본어</rt><rp>)</rp></ruby></a>·<a href="/wiki/%E9%9F%93%E5%9C%8B%E8%AA%9E" title="韓國語"><ruby class="hanja grade70"><rb>韓國語</rb><rp>(</rp><rt>한국어</rt><rp>)</rp></ruby></a> <ruby class="hanja grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby> <ruby class="hanja grade60"><rb>言語</rb><rp>(</rp><rt>언어</rt><rp>)</rp></ruby>에서 <ruby class="hanja grade60"><rb>使用</rb><rp>(</rp><rt>사용</rt><rp>)</rp></ruby>하는 <a href="/wiki/%E6%BC%A2%E5%AD%97" title="漢字"><ruby class="hanja grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby></a>가 <ruby class="hanja grade42"><rb>收錄</rb><rp>(</rp><rt>수록</rt><rp>)</rp></ruby>된 <a href="/index.php?title=%EC%9C%A0%EB%8B%88%EC%BD%94%EB%93%9C&amp;action=edit&amp;redlink=1" class="new" title="유니코드 (없는 문서)">유니코드</a><ruby class="hanja grade40"><rb>領域</rb><rp>(</rp><rt>영역</rt><rp>)</rp></ruby>이다. <ruby class="hanja grade60"><rb>言語</rb><rp>(</rp><rt>언어</rt><rp>)</rp></ruby>·<ruby class="hanja grade60"><rb>國家別</rb><rp>(</rp><rt>국가별</rt><rp>)</rp></ruby> <ruby class="hanja grade52"><rb>獨自</rb><rp>(</rp><rt>독자</rt><rp>)</rp></ruby>코드<ruby class="hanja grade40"><rb>體系</rb><rp>(</rp><rt>체계</rt><rp>)</rp></ruby>로 다르게 인코딩되어있던 <ruby class="hanja grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby>코드들을 <a href="/index.php?title=Unihan&amp;action=edit&amp;redlink=1" class="new" title="Unihan (없는 문서)">Unihan</a> 프로젝트를 <ruby class="hanja grade60"><rb>通</rb><rp>(</rp><rt>통</rt><rp>)</rp></ruby>해 <ruby class="hanja grade42"><rb>統合</rb><rp>(</rp><rt>통합</rt><rp>)</rp></ruby>하여 붙은 이름이다.
