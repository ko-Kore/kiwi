<h2><span id="槪要(개요)"></span><span class="mw-headline" id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"><ruby class="hanja grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%B5%B7%E9%9B%B2%E8%87%BA%EA%B7%B8%EB%A6%B0%EC%8B%9C%ED%8B%B0&amp;action=edit&amp;section=1" title="부분 편집: 槪要(개요)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja grade32"><rb>海雲臺</rb><rp>(</rp><rt>해운대</rt><rp>)</rp></ruby>그린시티는 <a href="/wiki/%E9%87%9C%E5%B1%B1%E5%BB%A3%E5%9F%9F%E5%B8%82" title="釜山廣域市"><ruby class="hanja grade12"><rb>釜山廣域市</rb><rp>(</rp><rt>부산광역시</rt><rp>)</rp></ruby></a><a href="/wiki/%E6%B5%B7%E9%9B%B2%E8%87%BA%E5%8D%80" title="海雲臺區"><ruby class="hanja grade32"><rb>海雲臺區</rb><rp>(</rp><rt>해운대구</rt><rp>)</rp></ruby></a> <a href="/index.php?title=%E4%BD%90%E6%B4%9E&amp;action=edit&amp;redlink=1" class="new" title="佐洞 (없는 문서)"><ruby class="hanja grade30 unknown"><rb>佐洞</rb><rp>(</rp><rt>좌동</rt><rp>)</rp></ruby></a>에서 <a href="/index.php?title=%E4%B8%AD%E6%B4%9E&amp;action=edit&amp;redlink=1" class="new" title="中洞 (없는 문서)"><ruby class="hanja grade70 unknown"><rb>中洞</rb><rp>(</rp><rt>중동</rt><rp>)</rp></ruby></a>에걸쳐있는 <a href="/index.php?title=1980%E5%B9%B4%E4%BB%A3&amp;action=edit&amp;redlink=1" class="new" title="1980年代 (없는 문서)">1980<ruby class="hanja grade62 unknown"><rb>年代</rb><rp>(</rp><rt>년대</rt><rp>)</rp></ruby></a> <ruby class="hanja grade60"><rb>開發</rb><rp>(</rp><rt>개발</rt><rp>)</rp></ruby>된 <ruby class="hanja grade12"><rb>釜山最初</rb><rp>(</rp><rt>부산최초</rt><rp>)</rp></ruby>의 <a href="/index.php?title=%E6%96%B0%E9%83%BD%E5%B8%82&amp;action=edit&amp;redlink=1" class="new" title="新都市 (없는 문서)"><ruby class="hanja grade50 unknown"><rb>新都市</rb><rp>(</rp><rt>신도시</rt><rp>)</rp></ruby></a>이다.
</p>
<h2><span id="名稱(명칭)"></span><span class="mw-headline" id=".E5.90.8D.E7.A8.B1.28.EB.AA.85.EC.B9.AD.29"><ruby class="hanja grade40"><rb>名稱</rb><rp>(</rp><rt>명칭</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%B5%B7%E9%9B%B2%E8%87%BA%EA%B7%B8%EB%A6%B0%EC%8B%9C%ED%8B%B0&amp;action=edit&amp;section=2" title="부분 편집: 名稱(명칭)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja grade52"><rb>元來</rb><rp>(</rp><rt>원래</rt><rp>)</rp></ruby> ‘<ruby class="hanja grade32"><rb>海雲臺新市街地</rb><rp>(</rp><rt>해운대신시가지</rt><rp>)</rp></ruby>’라고 불렸으나 <a href="/wiki/2020%E5%B9%B4" title="2020年">2020<ruby class="hanja grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby></a> <ruby class="hanja grade30"><rb>公募</rb><rp>(</rp><rt>공모</rt><rp>)</rp></ruby>를<ruby class="hanja grade60"><rb>通</rb><rp>(</rp><rt>통</rt><rp>)</rp></ruby>해뽑힌 ‘<ruby class="hanja grade32"><rb>海雲臺</rb><rp>(</rp><rt>해운대</rt><rp>)</rp></ruby>그린시티’가 <ruby class="hanja grade40"><rb>正式名稱</rb><rp>(</rp><rt>정식명칭</rt><rp>)</rp></ruby>이 되었다. <a rel="nofollow" class="external autonumber" href="https://www.yna.co.kr/view/AKR20201224039300051">[1]</a> <del><a href="/index.php?title=%EC%84%BC%ED%85%80%EC%8B%9C%ED%8B%B0&amp;action=edit&amp;redlink=1" class="new" title="센텀시티 (없는 문서)">센텀시티</a>, <a href="/index.php?title=%EB%A7%88%EB%A6%B0%EC%8B%9C%ED%8B%B0&amp;action=edit&amp;redlink=1" class="new" title="마린시티 (없는 문서)">마린시티</a>와함께 <ruby class="hanja grade32"><rb>海雲臺</rb><rp>(</rp><rt>해운대</rt><rp>)</rp></ruby>3<ruby class="hanja grade80"><rb>大</rb><rp>(</rp><rt>대</rt><rp>)</rp></ruby>시티가되었다</del>
</p>
