<h2><span id="槪要(개요)"></span><span class="mw-headline" id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"><ruby class="hanja grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%97%A5%E6%9C%AC%E5%B8%9D%E5%9C%8B&amp;action=edit&amp;section=1" title="부분 편집: 槪要(개요)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja grade40"><rb>大日本帝國</rb><rp>(</rp><rt>대일본제국</rt><rp>)</rp></ruby>은 <a href="/wiki/%E6%97%A5%E6%9C%AC" title="日本"><ruby class="hanja grade60"><rb>日本</rb><rp>(</rp><rt>일본</rt><rp>)</rp></ruby></a>이 <ruby class="hanja grade60"><rb>使用</rb><rp>(</rp><rt>사용</rt><rp>)</rp></ruby>했던 <a href="/index.php?title=%E5%9C%8B%E8%99%9F&amp;action=edit&amp;redlink=1" class="new" title="國號 (없는 문서)"><ruby class="hanja grade60 unknown"><rb>國號</rb><rp>(</rp><rt>국호</rt><rp>)</rp></ruby></a><ruby class="hanja grade80"><rb>中</rb><rp>(</rp><rt>중</rt><rp>)</rp></ruby> 하나이다. <ruby class="hanja grade40"><rb>大日本帝國</rb><rp>(</rp><rt>대일본제국</rt><rp>)</rp></ruby>이라는 <ruby class="hanja grade40"><rb>名稱</rb><rp>(</rp><rt>명칭</rt><rp>)</rp></ruby>은 <a href="/index.php?title=%E6%B1%9F%E6%88%B6%E6%99%82%E4%BB%A3&amp;action=edit&amp;redlink=1" class="new" title="江戶時代 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>江戶時代</rb><rp>(</rp><rt>강호시대</rt><rp>)</rp></ruby></a> <ruby class="hanja grade50"><rb>末期</rb><rp>(</rp><rt>말기</rt><rp>)</rp></ruby>부터 <ruby class="hanja grade50"><rb>敗戰時</rb><rp>(</rp><rt>패전시</rt><rp>)</rp></ruby>까지 <ruby class="hanja grade60"><rb>使用</rb><rp>(</rp><rt>사용</rt><rp>)</rp></ruby>되었지만, <ruby class="hanja grade60"><rb>國號</rb><rp>(</rp><rt>국호</rt><rp>)</rp></ruby>를 「<ruby class="hanja grade40"><rb>大日本帝國</rb><rp>(</rp><rt>대일본제국</rt><rp>)</rp></ruby>」으로 <ruby class="hanja grade60"><rb>定</rb><rp>(</rp><rt>정</rt><rp>)</rp></ruby>한 <a href="/index.php?title=%E6%B3%95%E5%BE%8B&amp;action=edit&amp;redlink=1" class="new" title="法律 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>法律</rb><rp>(</rp><rt>법률</rt><rp>)</rp></ruby></a>은 <ruby class="hanja grade40"><rb>存在</rb><rp>(</rp><rt>존재</rt><rp>)</rp></ruby>하지 않았던 <ruby class="hanja grade42"><rb>故</rb><rp>(</rp><rt>고</rt><rp>)</rp></ruby>로 <a href="/index.php?title=%E5%A4%A7%E6%97%A5%E6%9C%AC%E5%B8%9D%E5%9C%8B%E6%86%B2%E6%B3%95&amp;action=edit&amp;redlink=1" class="new" title="大日本帝國憲法 (없는 문서)"><ruby class="hanja grade40 unknown"><rb>大日本帝國憲法</rb><rp>(</rp><rt>대일본제국헌법</rt><rp>)</rp></ruby></a><ruby class="hanja grade72"><rb>下</rb><rp>(</rp><rt>하</rt><rp>)</rp></ruby>의 <a href="/wiki/%E6%97%A5%E6%9C%AC" title="日本"><ruby class="hanja grade60"><rb>日本</rb><rp>(</rp><rt>일본</rt><rp>)</rp></ruby></a>과 그의 <ruby class="hanja grade50"><rb>海外領土</rb><rp>(</rp><rt>해외령토</rt><rp>)</rp></ruby> <ruby class="hanja grade32"><rb>及</rb><rp>(</rp><rt>급</rt><rp>)</rp></ruby> <a href="/index.php?title=%E7%A7%9F%E5%80%9F%E5%9C%B0&amp;action=edit&amp;redlink=1" class="new" title="租借地 (없는 문서)"><ruby class="hanja grade32 unknown"><rb>租借地</rb><rp>(</rp><rt>조차지</rt><rp>)</rp></ruby></a>의 <ruby class="hanja grade40"><rb>總稱</rb><rp>(</rp><rt>총칭</rt><rp>)</rp></ruby>으로 봐야 맞다.
