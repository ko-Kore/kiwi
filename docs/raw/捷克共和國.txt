<p><a class="mw-selflink selflink">捷克共和國</a>는 <ruby class="hanja api grade20"><rb>歐洲</rb><rp>(</rp><rt>구주</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade62"><rb>中部</rb><rp>(</rp><rt>중부</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade42"><rb>位置</rb><rp>(</rp><rt>위치</rt><rp>)</rp></ruby>하고 있습니다. <ruby class="hanja api grade80"><rb>東</rb><rp>(</rp><rt>동</rt><rp>)</rp></ruby>쪽으로는 <a href="/wiki/%E6%96%AF%E6%B4%9B%E4%BC%90%E5%85%8B" title="斯洛伐克"><ruby class="hanja api grade20"><rb>斯洛伐克</rb><rp>(</rp><rt>사락벌극</rt><rp>)</rp></ruby></a>, <ruby class="hanja api grade80"><rb>南</rb><rp>(</rp><rt>남</rt><rp>)</rp></ruby>쪽으로는 <a href="/wiki/%E5%A2%BA%E5%9C%B0%E5%88%A9" title="墺地利"><ruby class="hanja api grade12"><rb>墺地利</rb><rp>(</rp><rt>오지리</rt><rp>)</rp></ruby></a>, <ruby class="hanja api grade80"><rb>西</rb><rp>(</rp><rt>서</rt><rp>)</rp></ruby>쪽으로는 <a href="/wiki/%E7%8D%A8%E9%80%B8" title="獨逸"><ruby class="hanja api grade32"><rb>獨逸</rb><rp>(</rp><rt>독일</rt><rp>)</rp></ruby></a>, <ruby class="hanja api grade80"><rb>北</rb><rp>(</rp><rt>북</rt><rp>)</rp></ruby>쪽으로는 <a href="/index.php?title=%E6%B3%A2%E8%98%AD&amp;action=edit&amp;redlink=1" class="new" title="波蘭 (없는 文書)"><ruby class="hanja api grade32 unknown"><rb>波蘭</rb><rp>(</rp><rt>파란</rt><rp>)</rp></ruby></a>와 <ruby class="hanja api grade42"><rb>國境</rb><rp>(</rp><rt>국경</rt><rp>)</rp></ruby>을 <ruby class="hanja api grade42"><rb>接</rb><rp>(</rp><rt>접</rt><rp>)</rp></ruby>하고 있습니다. 
</p><p>그것은 <ruby class="hanja api grade80"><rb>北</rb><rp>(</rp><rt>북</rt><rp>)</rp></ruby>쪽 <ruby class="hanja api grade42"><rb>溫帶</rb><rp>(</rp><rt>온대</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>地域</rb><rp>(</rp><rt>지역</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade40"><rb>屬</rb><rp>(</rp><rt>속</rt><rp>)</rp></ruby>하며 <ruby class="hanja api grade20"><rb>典型的</rb><rp>(</rp><rt>전형적</rt><rp>)</rp></ruby>인 <ruby class="hanja api grade42"><rb>溫帶</rb><rp>(</rp><rt>온대</rt><rp>)</rp></ruby> <ruby class="hanja api grade52"><rb>大陸性</rb><rp>(</rp><rt>대륙성</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>氣候</rb><rp>(</rp><rt>기후</rt><rp>)</rp></ruby>를 가지고 있습니다. <ruby class="hanja api grade40"><rb>四季節</rb><rp>(</rp><rt>사계절</rt><rp>)</rp></ruby>이 뚜렷하고 여름 <ruby class="hanja api grade40"><rb>平均氣溫</rb><rp>(</rp><rt>평균기온</rt><rp>)</rp></ruby>은 <ruby class="hanja api grade52"><rb>約</rb><rp>(</rp><rt>약</rt><rp>)</rp></ruby> 18.5<ruby class="hanja api grade60"><rb>度</rb><rp>(</rp><rt>도</rt><rp>)</rp></ruby>, 겨울 <ruby class="hanja api grade40"><rb>平均氣溫</rb><rp>(</rp><rt>평균기온</rt><rp>)</rp></ruby>은 마이너스 3<ruby class="hanja api grade60"><rb>度</rb><rp>(</rp><rt>도</rt><rp>)</rp></ruby> <ruby class="hanja api grade42"><rb>程度</rb><rp>(</rp><rt>정도</rt><rp>)</rp></ruby>, <ruby class="hanja api grade40"><rb>氣候</rb><rp>(</rp><rt>기후</rt><rp>)</rp></ruby>는 <ruby class="hanja api grade32"><rb>濕</rb><rp>(</rp><rt>습</rt><rp>)</rp></ruby>한 <ruby class="hanja api grade40"><rb>氣候</rb><rp>(</rp><rt>기후</rt><rp>)</rp></ruby>, <ruby class="hanja api grade40"><rb>年平均</rb><rp>(</rp><rt>연평균</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>降水量</rb><rp>(</rp><rt>강수량</rt><rp>)</rp></ruby>은 683mm입니다. 
</p><p><ruby class="hanja api grade70"><rb>人口</rb><rp>(</rp><rt>인구</rt><rp>)</rp></ruby>는 1,053<ruby class="hanja api grade80"><rb>萬</rb><rp>(</rp><rt>만</rt><rp>)</rp></ruby> <ruby class="hanja api grade72"><rb>名</rb><rp>(</rp><rt>명</rt><rp>)</rp></ruby>(2022<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> <ruby class="hanja api grade42"><rb>基準</rb><rp>(</rp><rt>기준</rt><rp>)</rp></ruby>)입니다. 그들 <ruby class="hanja api grade80"><rb>中</rb><rp>(</rp><rt>중</rt><rp>)</rp></ruby> 90% <ruby class="hanja api grade52"><rb>以上</rb><rp>(</rp><rt>이상</rt><rp>)</rp></ruby>이 체코<ruby class="hanja api grade80"><rb>人</rb><rp>(</rp><rt>인</rt><rp>)</rp></ruby>, 슬로바키아<ruby class="hanja api grade80"><rb>人</rb><rp>(</rp><rt>인</rt><rp>)</rp></ruby> 2.9%, <ruby class="hanja api grade32"><rb>獨逸</rb><rp>(</rp><rt>독일</rt><rp>)</rp></ruby><ruby class="hanja api grade80"><rb>人</rb><rp>(</rp><rt>인</rt><rp>)</rp></ruby> 1%, 그리고 <ruby class="hanja api grade70"><rb>少數</rb><rp>(</rp><rt>소수</rt><rp>)</rp></ruby>의 폴란드<ruby class="hanja api grade80"><rb>人</rb><rp>(</rp><rt>인</rt><rp>)</rp></ruby>과 로마니(집시)이다. <ruby class="hanja api grade60"><rb>公式</rb><rp>(</rp><rt>공식</rt><rp>)</rp></ruby> <ruby class="hanja api grade60"><rb>言語</rb><rp>(</rp><rt>언어</rt><rp>)</rp></ruby>는 체코<ruby class="hanja api grade70"><rb>語</rb><rp>(</rp><rt>어</rt><rp>)</rp></ruby>입니다. <ruby class="hanja api grade52"><rb>主要</rb><rp>(</rp><rt>주요</rt><rp>)</rp></ruby> <ruby class="hanja api grade42"><rb>宗敎</rb><rp>(</rp><rt>종교</rt><rp>)</rp></ruby>는 로마 카톨릭입니다.

<p>5<ruby class="hanja api grade40"><rb>世紀</rb><rp>(</rp><rt>세기</rt><rp>)</rp></ruby>~6<ruby class="hanja api grade40"><rb>世紀</rb><rp>(</rp><rt>세기</rt><rp>)</rp></ruby>에 슬라브족은 <ruby class="hanja api grade80"><rb>西</rb><rp>(</rp><rt>서</rt><rp>)</rp></ruby>쪽으로 <ruby class="hanja api grade42"><rb>移動</rb><rp>(</rp><rt>이동</rt><rp>)</rp></ruby>하여 <ruby class="hanja api grade60"><rb>現在</rb><rp>(</rp><rt>현재</rt><rp>)</rp></ruby>의 체코와 슬로바키아에 이르렀고, <ruby class="hanja api grade62"><rb>書記</rb><rp>(</rp><rt>서기</rt><rp>)</rp></ruby> 830<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby>에는 이 <ruby class="hanja api grade40"><rb>地域</rb><rp>(</rp><rt>지역</rt><rp>)</rp></ruby>에 모라비아 <ruby class="hanja api grade40"><rb>帝國</rb><rp>(</rp><rt>제국</rt><rp>)</rp></ruby>이 세워졌습니다. 
</p><p>9 <ruby class="hanja api grade40"><rb>世紀</rb><rp>(</rp><rt>세기</rt><rp>)</rp></ruby> 말과 10 <ruby class="hanja api grade40"><rb>世紀</rb><rp>(</rp><rt>세기</rt><rp>)</rp></ruby> <ruby class="hanja api grade62"><rb>前半部</rb><rp>(</rp><rt>전반부</rt><rp>)</rp></ruby>에 체코 <ruby class="hanja api grade62"><rb>公國</rb><rp>(</rp><rt>공국</rt><rp>)</rp></ruby>이 <ruby class="hanja api grade60"><rb>現在</rb><rp>(</rp><rt>현재</rt><rp>)</rp></ruby>의 체코 <ruby class="hanja api grade40"><rb>地域</rb><rp>(</rp><rt>지역</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade42"><rb>設立</rb><rp>(</rp><rt>설립</rt><rp>)</rp></ruby>되었습니다.

grade50"><rb>貴族</rb><rp>(</rp><rt>귀족</rt><rp>)</rp></ruby>, <ruby class="hanja api grade32"><rb>封建</rb><rp>(</rp><rt>봉건</rt><rp>)</rp></ruby> <ruby class="hanja api grade42"><rb>統治</rb><rp>(</rp><rt>통치</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade40"><rb>對抗</rb><rp>(</rp><rt>대항</rt><rp>)</rp></ruby>하는 <a
