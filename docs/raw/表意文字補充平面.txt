<h2><span id=".E5.90.8D.E7.A8.B1.28.EB.AA.85.EC.B9.AD.29"></span><span class="mw-headline" id="名稱(명칭)"><ruby class="hanja api grade40"><rb>名稱</rb><rp>(</rp><rt>명칭</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E8%A1%A8%E6%84%8F%E6%96%87%E5%AD%97%E8%A3%9C%E5%85%85%E5%B9%B3%E9%9D%A2&amp;action=edit&amp;section=1" title="부분 편집: 名稱(명칭)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%E9%9F%93%E5%9C%8B%E8%AA%9E" title="韓國語"><ruby class="hanja api grade70"><rb>韓國語</rb><rp>(</rp><rt>한국어</rt><rp>)</rp></ruby></a>로는 <ruby class="hanja api grade32"><rb>表意文字補充平面</rb><rp>(</rp><rt>표의문자보충평면</rt><rp>)</rp></ruby>, <ruby class="hanja api grade32"><rb>象形文字補充平面</rb><rp>(</rp><rt>상형문자보충평면</rt><rp>)</rp></ruby> <ruby class="hanja api grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby>으로 불린다. 여기서 <a href="/index.php?title=%E8%A1%A8%E6%84%8F%E6%96%87%E5%AD%97&amp;action=edit&amp;redlink=1" class="new" title="表意文字 (없는 문서)"><ruby class="hanja api grade62 unknown"><rb>表意文字</rb><rp>(</rp><rt>표의문자</rt><rp>)</rp></ruby></a>나 <a href="/wiki/%E8%B1%A1%E5%BD%A2%E6%96%87%E5%AD%97" title="象形文字"><ruby class="hanja api grade40"><rb>象形文字</rb><rp>(</rp><rt>상형문자</rt><rp>)</rp></ruby></a>는 다름아닌 <a href="/wiki/%E6%BC%A2%E5%AD%97" title="漢字"><ruby class="hanja api grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby></a>를 <ruby class="hanja api grade42"><rb>意味</rb><rp>(</rp><rt>의미</rt><rp>)</rp></ruby>하는데, 그 이름으로써 드러나듯이 <a href="/wiki/%E6%BC%A2%E5%AD%97" title="漢字"><ruby class="hanja api grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby></a>만을 <ruby class="hanja api grade42"><rb>爲</rb><rp>(</rp><rt>위</rt><rp>)</rp></ruby>한 <ruby class="hanja api grade70"><rb>平面</rb><rp>(</rp><rt>평면</rt><rp>)</rp></ruby>이다.
</p><p>그런데 그마저도 거의 다 채워져가기 <ruby class="hanja api grade62"><rb>始作</rb><rp>(</rp><rt>시작</rt><rp>)</rp></ruby>하여, <a href="/index.php?title=%E7%AC%AC%E4%B8%89%E8%A1%A8%E6%84%8F%E6%96%87%E5%AD%97%E5%B9%B3%E9%9D%A2&amp;action=edit&amp;redlink=1" class="new" title="第三表意文字平面 (없는 문서)"><ruby class="hanja api grade70 unknown"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby>만을 <ruby class="hanja api grade42 unknown"><rb>爲</rb><rp>(</rp><rt>위</rt><rp>)</rp></ruby>한 <ruby class="hanja api grade70 unknown"><rb>平面</rb><rp>(</rp><rt>평면</rt><rp>)</rp></ruby>을 하나 더 만들기에</a> 이른다.
</p>
