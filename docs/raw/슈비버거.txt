<h2><span id=".E5.90.8D.E7.A8.B1.28.EB.AA.85.EC.B9.AD.29"></span><span class="mw-headline" id="名稱(명칭)"><ruby class="hanja api grade40"><rb>名稱</rb><rp>(</rp><rt>명칭</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EC%8A%88%EB%B9%84%EB%B2%84%EA%B1%B0&amp;action=edit&amp;section=1" title="부분 편집: 名稱(명칭)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>‘슈비’는 <b>슈</b>프림 <b>비</b>프(Supreme Beef)를 <ruby class="hanja api grade42"><rb>意味</rb><rp>(</rp><rt>의미</rt><rp>)</rp></ruby>한다.
</p>
<h2><span id=".E6.A7.8B.E6.88.90.28.EA.B5.AC.EC.84.B1.29"></span><span class="mw-headline" id="構成(구성)"><ruby class="hanja api grade40"><rb>構成</rb><rp>(</rp><rt>구성</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EC%8A%88%EB%B9%84%EB%B2%84%EA%B1%B0&amp;action=edit&amp;section=2" title="부분 편집: 構成(구성)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>빵, 소스, 슈프림 패티, 순쇠고기 패티, 양상추, 토마토, 소스로 구성되어 있으며 <ruby class="hanja api grade80"><rb>兄弟</rb><rp>(</rp><rt>형제</rt><rp>)</rp></ruby> 뻘인 슈슈버거와는 순쇠고기 패티가 들어가 있는 <ruby class="hanja api grade40"><rb>差異</rb><rp>(</rp><rt>차이</rt><rp>)</rp></ruby>가 있다.
</p>
