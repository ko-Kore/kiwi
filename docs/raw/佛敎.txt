<p><b><ruby class="hanja api grade42"><rb>佛敎</rb><rp>(</rp><rt>불교</rt><rp>)</rp></ruby></b>(बौद्धधर्म)는 <ruby class="hanja api grade60"><rb>古代</rb><rp>(</rp><rt>고대</rt><rp>)</rp></ruby> <ruby class="hanja api grade42"><rb>印度</rb><rp>(</rp><rt>인도</rt><rp>)</rp></ruby>의 <a href="/index.php?title=%E9%87%8B%E8%BF%A6%E7%89%9F%E5%B0%BC&amp;action=edit&amp;redlink=1" class="new" title="釋迦牟尼(없는　文書）"><ruby class="hanja api grade12 unknown"><rb>釋迦牟尼</rb><rp>(</rp><rt>석가모니</rt><rp>)</rp></ruby></a>에 <ruby class="hanja api grade40"><rb>依</rb><rp>(</rp><rt>의</rt><rp>)</rp></ruby>해 <ruby class="hanja api grade42"><rb>創始</rb><rp>(</rp><rt>창시</rt><rp>)</rp></ruby>된 브라흐미<ruby class="hanja api grade62"><rb>界</rb><rp>(</rp><rt>계</rt><rp>)</rp></ruby> <ruby class="hanja api grade42"><rb>宗敎</rb><rp>(</rp><rt>종교</rt><rp>)</rp></ruby>이다. <ruby class="hanja api grade42"><rb>佛敎</rb><rp>(</rp><rt>불교</rt><rp>)</rp></ruby>는 그가 펼친 가르침이자 또한 <ruby class="hanja api grade42"><rb>眞理</rb><rp>(</rp><rt>진리</rt><rp>)</rp></ruby>를 깨달아 <ruby class="hanja api grade10"><rb>佛陀</rb><rp>(</rp><rt>불타</rt><rp>)</rp></ruby>(깨우친 사람)가 될 것을 가르치는 <ruby class="hanja api grade42"><rb>宗敎</rb><rp>(</rp><rt>종교</rt><rp>)</rp></ruby>이다. 다만 <ruby class="hanja api grade40"><rb>多樣</rb><rp>(</rp><rt>다양</rt><rp>)</rp></ruby>한 <ruby class="hanja api grade40"><rb>地域</rb><rp>(</rp><rt>지역</rt><rp>)</rp></ruby>으로 <ruby class="hanja api grade30"><rb>傳播</rb><rp>(</rp><rt>전파</rt><rp>)</rp></ruby>되며 <ruby class="hanja api grade10"><rb>土蕃</rb><rp>(</rp><rt>토번</rt><rp>)</rp></ruby>의 본<ruby class="hanja api grade80"><rb>敎</rb><rp>(</rp><rt>교</rt><rp>)</rp></ruby>나 <ruby class="hanja api grade80"><rb>韓國</rb><rp>(</rp><rt>한국</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade10"><rb>巫俗</rb><rp>(</rp><rt>무속</rt><rp>)</rp></ruby>처럼 <ruby class="hanja api grade32"><rb>土俗信仰</rb><rp>(</rp><rt>토속신앙</rt><rp>)</rp></ruby>과 <ruby class="hanja api grade40"><rb>共存</rb><rp>(</rp><rt>공존</rt><rp>)</rp></ruby>하기도 한다.
</p><p><ruby class="hanja api grade52"><rb>世界的</rb><rp>(</rp><rt>세계적</rt><rp>)</rp></ruby>으로도 <a href="/wiki/%E5%9F%BA%E7%9D%A3%E6%95%8E" title="基督敎"><ruby class="hanja api grade42"><rb>基督敎</rb><rp>(</rp><rt>기독교</rt><rp>)</rp></ruby></a>·<a href="/wiki/%EC%9D%B4%EC%8A%AC%EB%9E%8C" class="mw-redirect" title="이슬람">이슬람<ruby class="hanja api grade80"><rb>敎</rb><rp>(</rp><rt>교</rt><rp>)</rp></ruby></a>에 이어 폭넓은 나라로 퍼지고 있어 <ruby class="hanja api grade42"><rb>世界宗敎</rb><rp>(</rp><rt>세계종교</rt><rp>)</rp></ruby>의 하나로 <ruby class="hanja api grade10"><rb>看做</rb><rp>(</rp><rt>간주</rt><rp>)</rp></ruby>되고 있다.
