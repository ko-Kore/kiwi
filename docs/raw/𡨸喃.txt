<p><ruby class="hanja api grade0"><rb>𡨸</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby><ruby class="hanja api grade0"><rb>喃</rb><rp>(</rp><rt>남</rt><rp>)</rp></ruby>(Chữ Nôm, 쯔놈)은 <a href="/wiki/%E8%B6%8A%E5%8D%97%E8%AA%9E" title="越南語"><ruby class="hanja api grade32"><rb>越南語</rb><rp>(</rp><rt>월남어</rt><rp>)</rp></ruby></a>에서 쓰이는 <a href="/wiki/%E6%96%87%E5%AD%97" title="文字"><ruby class="hanja api grade70"><rb>文字</rb><rp>(</rp><rt>문자</rt><rp>)</rp></ruby></a>이다. <a href="/index.php?title=14%E4%B8%96%E7%B4%80&amp;action=edit&amp;redlink=1" class="new" title="14世紀 (없는 文書)">14<ruby class="hanja api grade40 unknown"><rb>世紀</rb><rp>(</rp><rt>세기</rt><rp>)</rp></ruby></a>에 <a href="/index.php?title=%E5%8F%A3%E8%AA%9E%E9%AB%94&amp;action=edit&amp;redlink=1" class="new" title="口語體 (없는 文書)"><ruby class="hanja api grade62 unknown"><rb>口語體</rb><rp>(</rp><rt>구어체</rt><rp>)</rp></ruby></a> <ruby class="hanja api grade32"><rb>越南語</rb><rp>(</rp><rt>월남어</rt><rp>)</rp></ruby>를 <ruby class="hanja api grade62"><rb>表記</rb><rp>(</rp><rt>표기</rt><rp>)</rp></ruby>하기 <ruby class="hanja api grade42"><rb>爲</rb><rp>(</rp><rt>위</rt><rp>)</rp></ruby>해 만들어졌다.
</p><p><ruby class="hanja api grade50"><rb>完全</rb><rp>(</rp><rt>완전</rt><rp>)</rp></ruby>히 <ruby class="hanja api grade52"><rb>獨自</rb><rp>(</rp><rt>독자</rt><rp>)</rp></ruby>로 만들어진 <ruby class="hanja api grade70"><rb>文字</rb><rp>(</rp><rt>문자</rt><rp>)</rp></ruby>는 아니고, <a href="/wiki/%E6%BC%A2%E5%AD%97" title="漢字"><ruby class="hanja api grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby></a>를 <ruby class="hanja api grade32"><rb>越南語</rb><rp>(</rp><rt>월남어</rt><rp>)</rp></ruby>에 맞도록 <ruby class="hanja api grade30"><rb>擴張</rb><rp>(</rp><rt>확장</rt><rp>)</rp></ruby>하여 새로 만들어낸 글<ruby class="hanja api grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>들을 뜻한다.
</p>

<h2><span id=".E7.89.B9.E5.BE.B5.28.ED.8A.B9.EC.A7.95.29"></span><span class="mw-headline" id="特徵(특징)"><ruby class="hanja api grade32"><rb>特徵</rb><rp>(</rp><rt>특징</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%F0%A1%A8%B8%E5%96%83&amp;action=edit&amp;section=1" title="部分 編輯: 特徵(특징)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<figure class="mw-default-size mw-halign-right" typeof="mw:File/Thumb"><a href="/wiki/%ED%8C%8C%EC%9D%BC:Ph%E1%BB%9F.png" class="mw-file-description"><img src="https://upload.wikimedia.org/wikipedia/commons/thumb/5/58/Ph%E1%BB%9F.png/300px-Ph%E1%BB%9F.png" decoding="async" width="300" height="300" srcset="https://upload.wikimedia.org/wikipedia/commons/thumb/5/58/Ph%E1%BB%9F.png/450px-Ph%E1%BB%9F.png 1.5x, https://upload.wikimedia.org/wikipedia/commons/thumb/5/58/Ph%E1%BB%9F.png/600px-Ph%E1%BB%9F.png 2x"></a><figcaption>形聲字 原理로 만들어진 <a href="/wiki/%F0%AC%96%BE" title="𬖾">𬖾</a>(phở)</figcaption></figure>
<p><a href="/wiki/%E4%B8%AD%E5%9C%8B" title="中國"><ruby class="hanja api grade80"><rb>中國</rb><rp>(</rp><rt>중국</rt><rp>)</rp></ruby></a>에서 들어온 <a href="/wiki/%E6%BC%A2%E5%AD%97" title="漢字"><ruby class="hanja api grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby></a> 中에서 소리만을 빌려와 그대로 쓰는 <a href="/index.php?title=%E5%81%87%E5%80%9F%E6%96%87%E5%AD%97&amp;action=edit&amp;redlink=1" class="new" title="假借文字 (없는 文書)"><ruby class="hanja api grade32 unknown"><rb>假借</rb><rp>(</rp><rt>가차</rt><rp>)</rp></ruby><ruby class="hanja api grade70 unknown"><rb>文字</rb><rp>(</rp><rt>문자</rt><rp>)</rp></ruby></a>와 <a href="/wiki/%E5%85%AD%E6%9B%B8" title="六書"><ruby class="hanja api grade62"><rb>六書</rb><rp>(</rp><rt>육서</rt><rp>)</rp></ruby></a>의 <a href="/index.php?title=%E8%A3%BD%E5%AD%97%E5%8E%9F%E7%90%86&amp;action=edit&amp;redlink=1" class="new" title="製字原理 (없는 文書)"><ruby class="hanja api grade42 unknown"><rb>製</rb><rp>(</rp><rt>제</rt><rp>)</rp></ruby><ruby class="hanja api grade70 unknown"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby><ruby class="hanja api grade50 unknown"><rb>原理</rb><rp>(</rp><rt>원리</rt><rp>)</rp></ruby></a>에 따라 새로 만든 <a href="/wiki/%E6%9C%83%E6%84%8F%E6%96%87%E5%AD%97" title="會意文字"><ruby class="hanja api grade62"><rb>會意</rb><rp>(</rp><rt>회의</rt><rp>)</rp></ruby><ruby class="hanja api grade70"><rb>文字</rb><rp>(</rp><rt>문자</rt><rp>)</rp></ruby></a>나 <a href="/wiki/%E5%BD%A2%E8%81%B2%E6%96%87%E5%AD%97" title="形聲文字"><ruby class="hanja api grade42"><rb>形聲</rb><rp>(</rp><rt>형성</rt><rp>)</rp></ruby><ruby class="hanja api grade70"><rb>文字</rb><rp>(</rp><rt>문자</rt><rp>)</rp></ruby></a>가 있고, 漢字의 一部 <a href="/index.php?title=%E5%8A%83&amp;action=edit&amp;redlink=1" class="new" title="劃 (없는 文書)"><ruby class="hanja api grade32 unknown"><rb>劃</rb><rp>(</rp><rt>획</rt><rp>)</rp></ruby></a>을 따 와서 만든 文字도 있다.
</p><p><a href="/wiki/%E6%BC%A2%E5%AD%97" title="漢字"><ruby class="hanja api grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby></a>는 ‘𡨸漢’이나 ‘𡨸儒’ 또는 ‘漢字’라고 불리고 𡨸喃과는 區別되며, 𡨸喃과 漢字를 함께 使用한 表記를 <a href="/index.php?title=%E6%BC%A2%E5%96%83&amp;action=edit&amp;redlink=1" class="new" title="漢喃 (없는 文書)"><ruby class="hanja api grade72 unknown"><rb>漢</rb><rp>(</rp><rt>한</rt><rp>)</rp></ruby><ruby class="hanja api grade0 unknown"><rb>喃</rb><rp>(</rp><rt>남</rt><rp>)</rp></ruby></a>이라 한다.
</p>
<h3><span id=".E8.A3.BD.E5.AD.97.E6.96.B9.E5.BC.8F"></span><span class="mw-headline" id="製字方式">製字方式</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%F0%A1%A8%B8%E5%96%83&amp;action=edit&amp;section=2" title="部分 編輯: 製字方式">編輯</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/index.php?title=%E5%BD%A2%E8%81%B2&amp;action=edit&amp;redlink=1" class="new" title="形聲 (없는 文書)"><ruby class="hanja api grade42 unknown"><rb>形聲</rb><rp>(</rp><rt>형성</rt><rp>)</rp></ruby></a>의原理로 만들어진 글字가 가장 많다.
</p>
<ul><li><a href="/index.php?title=%F0%A0%80%A7&amp;action=edit&amp;redlink=1" class="new" title="𠀧 (없는 文書)"><ruby><rb>𠀧</rb><rp>(</rp><rt>ba</rt><rp>)</rp></ruby></a>: 三에서 뜻을, 巴에서 소리를 가져와서 固有語<a href="/index.php?title=%E6%95%B8%E8%A9%9E&amp;action=edit&amp;redlink=1" class="new" title="數詞 (없는 文書)"><ruby class="hanja api grade32 unknown"><rb>數詞</rb><rp>(</rp><rt>수사</rt><rp>)</rp></ruby></a> 3을 나타낸다.</li>
<li><a href="/index.php?title=%E5%AA%84&amp;action=edit&amp;redlink=1" class="new" title="媄 (없는 文書)"><ruby><rb>媄</rb><rp>(</rp><rt>mẹ</rt><rp>)</rp></ruby></a>: 女에서 뜻을, 美에서 소리를 가져와서 <a href="/index.php?title=%EC%96%B4%EB%A8%B8%EB%8B%88&amp;action=edit&amp;redlink=1" class="new" title="어머니 (없는 文書)">어머니</a>를 意味한다.</li></ul>
<p><a href="/index.php?title=%E6%9C%83%E6%84%8F&amp;action=edit&amp;redlink=1" class="new" title="會意 (없는 文書)"><ruby class="hanja api grade62 unknown"><rb>會意</rb><rp>(</rp><rt>회의</rt><rp>)</rp></ruby></a>의 原理로 만들어진 글字도 몇 存在한다.
</p>
<ul><li><a href="/index.php?title=%F0%A1%97%B6&amp;action=edit&amp;redlink=1" class="new" title="𡗶 (없는 文書)"><ruby><rb>𡗶</rb><rp>(</rp><rt>trời</rt><rp>)</rp></ruby></a>: 天과 上을 합쳐서 <a href="/index.php?title=%ED%95%98%EB%8A%98&amp;action=edit&amp;redlink=1" class="new" title="하늘 (없는 文書)">하늘</a>을 뜻하는 giời 或은 trời를 表記한다.</li></ul>
<h2><span id=".E5.BB.A2.E6.AD.A2"></span><span class="mw-headline" id="廢止">廢止</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%F0%A1%A8%B8%E5%96%83&amp;action=edit&amp;section=3" title="部分 編輯: 廢止">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%E8%B6%8A%E5%8D%97%E8%AA%9E" title="越南語"><ruby class="hanja api grade32"><rb>越南語</rb><rp>(</rp><rt>월남어</rt><rp>)</rp></ruby></a>를 말하는 그대로 表記할 수 있는 唯一한 方法이었으나, <a href="/wiki/%E6%BC%A2%E5%AD%97" title="漢字"><ruby class="hanja api grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby></a>를 익힌 上流層만이 使用할 수 있었다. 그러던 中 外國 宣敎師들이 越南語 學習用으로 <a href="/wiki/%EB%A1%9C%EB%A7%88%E5%AD%97" title="로마字">로마<ruby class="hanja api grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby></a> 表記를 始作하였는데, <a href="/wiki/%E8%B6%8A%E5%8D%97" class="mw-redirect" title="越南"><ruby class="hanja api grade32"><rb>越南</rb><rp>(</rp><rt>월남</rt><rp>)</rp></ruby></a> 支配를 始作한 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" class="mw-redirect" title="프랑스">프랑스</a>가 <a href="/wiki/%F0%A1%A8%B8%E5%9C%8B%E8%AA%9E" title="𡨸國語"><ruby class="hanja api grade0"><rb>𡨸</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby><ruby class="hanja api grade70"><rb>國語</rb><rp>(</rp><rt>국어</rt><rp>)</rp></ruby></a>를 普及하였고<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">[1]</a></sup>, 結局 <a href="/index.php?title=1945%E5%B9%B4&amp;action=edit&amp;redlink=1" class="new" title="1945年 (없는 文書)">1945<ruby class="hanja api grade80 unknown"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby></a> <a href="/index.php?title=%E8%B6%8A%E7%9B%9F&amp;action=edit&amp;redlink=1" class="new" title="越盟 (없는 文書)"><ruby class="hanja api grade32 unknown"><rb>越盟</rb><rp>(</rp><rt>월맹</rt><rp>)</rp></ruby></a>이 𡨸國語를 公式 <a href="/wiki/%E6%96%87%E5%AD%97" title="文字"><ruby class="hanja api grade70"><rb>文字</rb><rp>(</rp><rt>문자</rt><rp>)</rp></ruby></a>로 指定하면서 <a href="/wiki/%E8%B6%8A%E5%8D%97" class="mw-redirect" title="越南"><ruby class="hanja api grade32"><rb>越南</rb><rp>(</rp><rt>월남</rt><rp>)</rp></ruby></a>에서 𡨸喃은 거의 쓰이지 않게 되었다.<sup id="cite_ref-2" class="reference"><a href="#cite_note-2">[2]</a></sup>
</p>
<h2><span id=".E7.8F.BE.E5.9C.A8.EC.9D.98_.E4.BD.BF.E7.94.A8"></span><span class="mw-headline" id="現在의_使用">現在의 使用</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%F0%A1%A8%B8%E5%96%83&amp;action=edit&amp;section=4" title="部分 編輯: 現在의 使用">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<figure class="mw-default-size mw-halign-right" typeof="mw:File/Thumb"><a href="/wiki/%ED%8C%8C%EC%9D%BC:%E7%B0%A1%E5%8C%96%E5%96%83%E5%AD%97.webp" class="mw-file-description"><img src="/images/thumb/b/bb/%E7%B0%A1%E5%8C%96%E5%96%83%E5%AD%97.webp/300px-%E7%B0%A1%E5%8C%96%E5%96%83%E5%AD%97.webp.png" decoding="async" width="300" height="225" srcset="/images/thumb/b/bb/%E7%B0%A1%E5%8C%96%E5%96%83%E5%AD%97.webp/450px-%E7%B0%A1%E5%8C%96%E5%96%83%E5%AD%97.webp.png 1.5x, /images/thumb/b/bb/%E7%B0%A1%E5%8C%96%E5%96%83%E5%AD%97.webp/600px-%E7%B0%A1%E5%8C%96%E5%96%83%E5%AD%97.webp.png 2x"></a><figcaption>在中京族에 對한 <a href="/wiki/%E4%B8%AD%E8%8F%AF%E4%BA%BA%E6%B0%91%E5%85%B1%E5%92%8C%E5%9C%8B" title="中華人民共和國"><ruby class="hanja api grade40"><rb>中華人民共和國</rb><rp>(</rp><rt>중화인민공화국</rt><rp>)</rp></ruby></a>의 弘報資料. 𡨸喃이 <a href="/wiki/%E7%B0%A1%E5%8C%96%E5%AD%97" title="簡化字"><ruby class="hanja api grade40"><rb>簡</rb><rp>(</rp><rt>간</rt><rp>)</rp></ruby><ruby class="hanja api grade52"><rb>化</rb><rp>(</rp><rt>화</rt><rp>)</rp></ruby><ruby class="hanja api grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby></a>와 함께 쓰여있다.</figcaption></figure>
<p>𡨸喃이 廢止된지 100年이 지난 只今은 <a href="/wiki/%E6%BC%A2%E5%AD%97" title="漢字"><ruby class="hanja api grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby></a>와 더불어 <a href="/index.php?title=%E6%AD%B7%E5%8F%B2%E5%AD%B8%E8%80%85&amp;action=edit&amp;redlink=1" class="new" title="歷史學者 (없는 文書)"><ruby class="hanja api grade52 unknown"><rb>歷史學者</rb><rp>(</rp><rt>역사학자</rt><rp>)</rp></ruby></a>와 같은 關聯職種의從事員外에는, 읽을 수 있는 사람조차 찾기 힘들다.
</p><p>그러나 <a href="/wiki/%E5%9C%A8%E4%B8%AD%E4%BA%AC%E6%97%8F" class="mw-redirect" title="在中京族"><ruby class="hanja api grade60"><rb>在中</rb><rp>(</rp><rt>재중</rt><rp>)</rp></ruby><ruby class="hanja api grade60"><rb>京</rb><rp>(</rp><rt>경</rt><rp>)</rp></ruby><ruby class="hanja api grade60"><rb>族</rb><rp>(</rp><rt>족</rt><rp>)</rp></ruby></a><sup id="cite_ref-3" class="reference"><a href="#cite_note-3">[3]</a></sup>은 아직도 使用하고 있다고 한다.
</p>
<h3><span id=".E5.BE.A9.E6.B4.BB_.E5.8F.AF.E8.83.BD.E6.80.A7"></span><span class="mw-headline" id="復活_可能性">復活 可能性</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%F0%A1%A8%B8%E5%96%83&amp;action=edit&amp;section=5" title="部分 編輯: 復活 可能性">編輯</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/index.php?title=%E6%BC%A2%E5%96%83&amp;action=edit&amp;redlink=1" class="new" title="漢喃 (없는 文書)"><ruby class="hanja api grade72 unknown"><rb>漢</rb><rp>(</rp><rt>한</rt><rp>)</rp></ruby><ruby class="hanja api grade0 unknown"><rb>喃</rb><rp>(</rp><rt>남</rt><rp>)</rp></ruby></a>을 廢止한지 100年이 지난 只今 𡨸喃의復活은 <a href="/wiki/%E9%9F%93%E5%9C%8B" title="韓國"><ruby class="hanja api grade80"><rb>韓國</rb><rp>(</rp><rt>한국</rt><rp>)</rp></ruby></a>에서 <a href="/wiki/%E5%9C%8B%E6%BC%A2%E6%96%87%E6%B7%B7%E7%94%A8%E9%AB%94" class="mw-redirect" title="國漢文混用體"><ruby class="hanja api grade70"><rb>國漢文</rb><rp>(</rp><rt>국한문</rt><rp>)</rp></ruby><ruby class="hanja api grade40"><rb>混用體</rb><rp>(</rp><rt>혼용체</rt><rp>)</rp></ruby></a>를 復活시키는 것보다 더한 難題라 할 수 있다. 이뿐만아니라, 比較的 漢字와의互換이 圓滑한 <a href="/wiki/%ED%95%9C%EA%B8%80%E5%B0%88%E7%94%A8" class="mw-redirect" title="한글專用">한글<ruby class="hanja api grade40"><rb>專用</rb><rp>(</rp><rt>전용</rt><rp>)</rp></ruby></a>이 主要 表記方式이 된 <a href="/wiki/%E9%9F%93%E5%9C%8B%E8%AA%9E" title="韓國語"><ruby class="hanja api grade70"><rb>韓國語</rb><rp>(</rp><rt>한국어</rt><rp>)</rp></ruby></a>와는 달리, 現代의越南語는 漢字와 어떠한 關聯도 없는 <a href="/wiki/%EB%A1%9C%EB%A7%88%E5%AD%97" title="로마字">로마<ruby class="hanja api grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby></a> 基盤의 <a href="/wiki/%F0%A1%A8%B8%E5%9C%8B%E8%AA%9E" title="𡨸國語"><ruby class="hanja api grade0"><rb>𡨸</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby><ruby class="hanja api grade70"><rb>國語</rb><rp>(</rp><rt>국어</rt><rp>)</rp></ruby></a>를 使用하니, 現今의 越南社會에서는 漢字의 痕迹조차 찾기 힘들다.<sup id="cite_ref-4" class="reference"><a href="#cite_note-4">[4]</a></sup>
</p><p>그러나 一部는 <a href="/index.php?title=%E5%82%B3%E7%B5%B1%E6%96%87%E5%8C%96&amp;action=edit&amp;redlink=1" class="new" title="傳統文化 (없는 文書)"><ruby class="hanja api grade42 unknown"><rb>傳統文化</rb><rp>(</rp><rt>전통문화</rt><rp>)</rp></ruby></a>의 保全과 國語生活의 質 等을 理由로 漢喃 復活을 主張하기도 한다. 漢喃 復活을 主張하는 <a href="/index.php?title=%E6%B0%91%E9%96%93%E5%9C%98%E9%AB%94&amp;action=edit&amp;redlink=1" class="new" title="民間團體 (없는 文書)"><ruby class="hanja api grade52 unknown"><rb>民間團體</rb><rp>(</rp><rt>민간단체</rt><rp>)</rp></ruby></a>로 <a href="/wiki/%E5%A7%94%E7%8F%AD%E5%BE%A9%E7%94%9F%E6%BC%A2%E5%96%83%E8%B6%8A%E5%8D%97" title="委班復生漢喃越南"><ruby class="hanja api grade40"><rb>委</rb><rp>(</rp><rt>위</rt><rp>)</rp></ruby><ruby class="hanja api grade62"><rb>班</rb><rp>(</rp><rt>반</rt><rp>)</rp></ruby><ruby class="hanja api grade42"><rb>復生</rb><rp>(</rp><rt>부생</rt><rp>)</rp></ruby><ruby class="hanja api grade72"><rb>漢</rb><rp>(</rp><rt>한</rt><rp>)</rp></ruby><ruby class="hanja api grade0"><rb>喃</rb><rp>(</rp><rt>남</rt><rp>)</rp></ruby><ruby class="hanja api grade32"><rb>越南</rb><rp>(</rp><rt>월남</rt><rp>)</rp></ruby></a>이 있다.
