<h2><span id="槪要(개요)"></span><span class="mw-headline" id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"><ruby class="hanja grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EA%B0%A4%EB%9F%AD%EC%8B%9C_S10e&amp;action=edit&amp;section=1" title="부분 편집: 槪要(개요)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>갤럭시 S10e는 <a href="/wiki/2019%E5%B9%B4" title="2019年">2019<ruby class="hanja grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby></a><a href="/index.php?title=2%E6%9C%8820%E6%97%A5&amp;action=edit&amp;redlink=1" class="new" title="2月20日 (없는 문서)">2<ruby class="hanja grade80 unknown"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby>20<ruby class="hanja grade80 unknown"><rb>日</rb><rp>(</rp><rt>일</rt><rp>)</rp></ruby></a>에 <ruby class="hanja grade60"><rb>公開</rb><rp>(</rp><rt>공개</rt><rp>)</rp></ruby>되어 <ruby class="hanja grade70"><rb>同年</rb><rp>(</rp><rt>동년</rt><rp>)</rp></ruby> <a href="/index.php?title=3%E6%9C%888%E6%97%A5&amp;action=edit&amp;redlink=1" class="new" title="3月8日 (없는 문서)">3<ruby class="hanja grade80 unknown"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby>8<ruby class="hanja grade80 unknown"><rb>日</rb><rp>(</rp><rt>일</rt><rp>)</rp></ruby></a>에 <ruby class="hanja grade70"><rb>出市</rb><rp>(</rp><rt>출시</rt><rp>)</rp></ruby>된 <a href="/index.php?title=%E4%B8%89%E6%98%9F%E9%9B%BB%E5%AD%90&amp;action=edit&amp;redlink=1" class="new" title="三星電子 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>三星電子</rb><rp>(</rp><rt>삼성전자</rt><rp>)</rp></ruby></a>의 스마트폰이며, <a href="/wiki/%EA%B0%A4%EB%9F%AD%EC%8B%9C_S_%EC%8B%9C%EB%A6%AC%EC%A6%88" title="갤럭시 S 시리즈">갤럭시 S 시리즈</a>의 열<ruby class="hanja grade60"><rb>番</rb><rp>(</rp><rt>번</rt><rp>)</rp></ruby>째 <ruby class="hanja grade42"><rb>製品</rb><rp>(</rp><rt>제품</rt><rp>)</rp></ruby>인 <a href="/wiki/%EA%B0%A4%EB%9F%AD%EC%8B%9C_S10" title="갤럭시 S10">갤럭시 S10</a>의 <ruby class="hanja grade40"><rb>派生</rb><rp>(</rp><rt>파생</rt><rp>)</rp></ruby> 모델이다.
</p>
<h2><span id="特徵(특징)"></span><span class="mw-headline" id=".E7.89.B9.E5.BE.B5.28.ED.8A.B9.EC.A7.95.29"><ruby class="hanja grade32"><rb>特徵</rb><rp>(</rp><rt>특징</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EA%B0%A4%EB%9F%AD%EC%8B%9C_S10e&amp;action=edit&amp;section=2" title="부분 편집: 特徵(특징)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>S 시리즈에서 처음으로 생긴 <ruby class="hanja grade20"><rb>入門型</rb><rp>(</rp><rt>입문형</rt><rp>)</rp></ruby> 모델이다. 이름의 알파벳 'e'는 essential을 <ruby class="hanja grade42"><rb>意味</rb><rp>(</rp><rt>의미</rt><rp>)</rp></ruby>한다고 한다.<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">[1]</a></sup>
</p>
<h2><span id="仕樣(사양)"></span><span class="mw-headline" id=".E4.BB.95.E6.A8.A3.28.EC.82.AC.EC.96.91.29"><ruby class="hanja grade40"><rb>仕樣</rb><rp>(</rp><rt>사양</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EA%B0%A4%EB%9F%AD%EC%8B%9C_S10e&amp;action=edit&amp;section=3" title="부분 편집: 仕樣(사양)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>5.8인치로 S10 시리즈 <ruby class="hanja grade80"><rb>中</rb><rp>(</rp><rt>중</rt><rp>)</rp></ruby> 가장 작은 크기의 <ruby class="hanja grade12"><rb>扁平</rb><rp>(</rp><rt>편평</rt><rp>)</rp></ruby>한<sup id="cite_ref-2" class="reference"><a href="#cite_note-2">[2]</a></sup> <a href="/wiki/%EC%9D%B8%ED%94%BC%EB%8B%88%ED%8B%B0_O_%EB%94%94%EC%8A%A4%ED%94%8C%EB%A0%88%EC%9D%B4" class="mw-redirect" title="인피니티 O 디스플레이">인피니티 O 디스플레이</a>를 <ruby class="hanja grade10"><rb>搭載</rb><rp>(</rp><rt>탑재</rt><rp>)</rp></ruby>했다.
</p><p><a href="/index.php?title=%E6%8C%87%E7%B4%8B%E8%AA%8D%E8%AD%98&amp;action=edit&amp;redlink=1" class="new" title="指紋認識 (없는 문서)"><ruby class="hanja grade32 unknown"><rb>指紋認識</rb><rp>(</rp><rt>지문인식</rt><rp>)</rp></ruby></a> 센서는 <ruby class="hanja grade70"><rb>主</rb><rp>(</rp><rt>주</rt><rp>)</rp></ruby>모델과 다르게 <ruby class="hanja grade60"><rb>畵面內</rb><rp>(</rp><rt>화면내</rt><rp>)</rp></ruby>가 아닌 <ruby class="hanja grade32"><rb>側面</rb><rp>(</rp><rt>측면</rt><rp>)</rp></ruby> <a href="/index.php?title=%E9%9B%BB%E6%BA%90%EB%B2%84%ED%8A%BC&amp;action=edit&amp;redlink=1" class="new" title="電源버튼 (없는 문서)"><ruby class="hanja grade40 unknown"><rb>電源</rb><rp>(</rp><rt>전원</rt><rp>)</rp></ruby>버튼</a>에 <ruby class="hanja grade32"><rb>內藏</rb><rp>(</rp><rt>내장</rt><rp>)</rp></ruby>되었고, <ruby class="hanja grade32"><rb>左側面</rb><rp>(</rp><rt>좌측면</rt><rp>)</rp></ruby>의 <ruby class="hanja grade50"><rb>音量</rb><rp>(</rp><rt>음량</rt><rp>)</rp></ruby>버튼의 밑에는 <a href="/index.php?title=%EB%B9%85%EC%8A%A4%EB%B9%84&amp;action=edit&amp;redlink=1" class="new" title="빅스비 (없는 문서)">빅스비</a> <ruby class="hanja grade42"><rb>呼出</rb><rp>(</rp><rt>호출</rt><rp>)</rp></ruby> 버튼이 <ruby class="hanja grade42"><rb>位置</rb><rp>(</rp><rt>위치</rt><rp>)</rp></ruby>해 있다. <ruby class="hanja grade52"><rb>全體的</rb><rp>(</rp><rt>전체적</rt><rp>)</rp></ruby>으로 봤을 때 <ruby class="hanja grade50"><rb>音量</rb><rp>(</rp><rt>음량</rt><rp>)</rp></ruby>버튼보다 <ruby class="hanja grade40"><rb>電源</rb><rp>(</rp><rt>전원</rt><rp>)</rp></ruby>버튼이 <ruby class="hanja grade42"><rb>上端</rb><rp>(</rp><rt>상단</rt><rp>)</rp></ruby>으로 올라가 있는데, <ruby class="hanja grade40"><rb>機器</rb><rp>(</rp><rt>기기</rt><rp>)</rp></ruby>를 잡을 <ruby class="hanja grade72"><rb>時</rb><rp>(</rp><rt>시</rt><rp>)</rp></ruby> 손의 <ruby class="hanja grade42"><rb>位置</rb><rp>(</rp><rt>위치</rt><rp>)</rp></ruby>를 <ruby class="hanja grade40"><rb>考慮</rb><rp>(</rp><rt>고려</rt><rp>)</rp></ruby>했을 때 <a href="/index.php?title=%EB%B9%85%EC%8A%A4%EB%B9%84&amp;action=edit&amp;redlink=1" class="new" title="빅스비 (없는 문서)">빅스비</a>를 많이 쓰도록 <ruby class="hanja grade62"><rb>意圖</rb><rp>(</rp><rt>의도</rt><rp>)</rp></ruby>한 것이 아니냐는 <ruby class="hanja grade52"><rb>意見</rb><rp>(</rp><rt>의견</rt><rp>)</rp></ruby>도 있었다.
</p><p><ruby class="hanja grade70"><rb>後面</rb><rp>(</rp><rt>후면</rt><rp>)</rp></ruby> <a href="/wiki/%EC%B9%B4%EB%A9%94%EB%9D%BC" class="mw-redirect" title="카메라">카메라</a>는 <a href="/index.php?title=%E5%BB%A3%E8%A7%92&amp;action=edit&amp;redlink=1" class="new" title="廣角 (없는 문서)"><ruby class="hanja grade52 unknown"><rb>廣角</rb><rp>(</rp><rt>광각</rt><rp>)</rp></ruby></a>과 <a href="/index.php?title=%E6%A8%99%E6%BA%96%E7%95%B5%E8%A7%92&amp;action=edit&amp;redlink=1" class="new" title="標準畵角 (없는 문서)"><ruby class="hanja grade40 unknown"><rb>標準畵角</rb><rp>(</rp><rt>표준화각</rt><rp>)</rp></ruby></a>으로 두 가지가, <ruby class="hanja grade70"><rb>前面</rb><rp>(</rp><rt>전면</rt><rp>)</rp></ruby>은 하나가 <ruby class="hanja grade10"><rb>搭載</rb><rp>(</rp><rt>탑재</rt><rp>)</rp></ruby>되어 있다.
</p><p><ruby class="hanja grade52"><rb>色相</rb><rp>(</rp><rt>색상</rt><rp>)</rp></ruby>은 '프리즘 블랙', '프리즘 화이트', '카나리아 옐로우' 세 가지로 <ruby class="hanja grade70"><rb>出市</rb><rp>(</rp><rt>출시</rt><rp>)</rp></ruby>되었다.
</p>
