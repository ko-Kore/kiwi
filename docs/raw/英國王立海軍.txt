grade52"><rb>地球的</rb><rp>(</rp><rt>지구적</rt><rp>)</rp></ruby><ruby class="hanja api grade62"><rb>作戰</rb><rp>(</rp><rt>작전</rt><rp>)</rp></ruby>이 <ruby class="hanja api grade62"><rb>時代</rb><rp>(</rp><rt>시대</rt><rp>)</rp></ruby>부터 쭉 내려온 어마어마한 노하우와 <ruby class="hanja api grade52"><rb>實戰</rb><rp>(</rp><rt>실전</rt><rp>)</rp></ruby><ruby class="hanja api
