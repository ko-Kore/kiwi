<h2><span id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"></span><span class="mw-headline" id="槪要(개요)"><ruby class="hanja api grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%9A%AE%E5%9C%8B&amp;action=edit&amp;section=1" title="部分 編輯: 槪要(개요)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja api grade30"><rb>暮國</rb><rp>(</rp><rt>모국</rt><rp>)</rp></ruby>(<span style="font-size: smaller;"><a href="/index.php?title=%EC%8A%A4%ED%8E%98%EC%9D%B8%E8%AA%9E&amp;action=edit&amp;redlink=1" class="new" title="스페인語 (없는 文書)">스페인語</a>: </span><span lang="es">Estado Plurinacional de Bolivia</span>)는 <a href="/index.php?title=%E5%8D%97%E7%BE%8E&amp;action=edit&amp;redlink=1" class="new" title="南美 (없는 文書)"><ruby class="hanja api grade60 unknown"><rb>南美</rb><rp>(</rp><rt>남미</rt><rp>)</rp></ruby></a> <ruby class="hanja api grade62"><rb>中部</rb><rp>(</rp><rt>중부</rt><rp>)</rp></ruby> <ruby class="hanja api grade52"><rb>內陸國</rb><rp>(</rp><rt>내륙국</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade42"><rb>位置</rb><rp>(</rp><rt>위치</rt><rp>)</rp></ruby>해 있다. <ruby class="hanja api grade80"><rb>北東</rb><rp>(</rp><rt>북동</rt><rp>)</rp></ruby>쪽은 <a href="/wiki/%E4%BC%AF%E5%89%8C%E8%A5%BF%E7%88%BE" title="伯剌西爾"><ruby class="hanja api grade10"><rb>伯剌西爾</rb><rp>(</rp><rt>백랄서이</rt><rp>)</rp></ruby></a>과 <ruby class="hanja api grade42"><rb>國境</rb><rp>(</rp><rt>국경</rt><rp>)</rp></ruby>을 <ruby class="hanja api grade42"><rb>接</rb><rp>(</rp><rt>접</rt><rp>)</rp></ruby>하고 <ruby class="hanja api grade80"><rb>南東</rb><rp>(</rp><rt>남동</rt><rp>)</rp></ruby>쪽은 <a href="/index.php?title=%E5%B7%B4%E6%8B%89%E5%9C%AD&amp;action=edit&amp;redlink=1" class="new" title="巴拉圭 (없는 文書)"><ruby class="hanja api grade10 unknown"><rb>巴拉圭</rb><rp>(</rp><rt>파랍규</rt><rp>)</rp></ruby></a> <ruby class="hanja api grade80"><rb>南</rb><rp>(</rp><rt>남</rt><rp>)</rp></ruby>쪽은 <a href="/wiki/%E4%BA%9E%E7%88%BE%E7%84%B6%E4%B8%81" title="亞爾然丁"><ruby class="hanja api grade32"><rb>亞</rb><rp>(</rp><rt>아</rt><rp>)</rp></ruby><ruby class="hanja api grade10"><rb>爾</rb><rp>(</rp><rt>이</rt><rp>)</rp></ruby><ruby class="hanja api grade70"><rb>然</rb><rp>(</rp><rt>연</rt><rp>)</rp></ruby><ruby class="hanja api grade40"><rb>丁</rb><rp>(</rp><rt>정</rt><rp>)</rp></ruby></a> <ruby class="hanja api grade80"><rb>南西</rb><rp>(</rp><rt>남서</rt><rp>)</rp></ruby>쪽은 <a href="/index.php?title=%E6%99%BA%E5%88%A9&amp;action=edit&amp;redlink=1" class="new" title="智利 (없는 文書)"><ruby class="hanja api grade40 <ruby class="hanja api grade42"><rb>接</rb><rp>(</rp><rt>접</rt><rp>)</rp></ruby>한다. <a href="/index.php?title=%E6%BA%AB%E5%B8%B6%E6%B0%A3%E5%80%99&amp;action=edit&amp;redlink=1" class="new" title="溫帶氣候 </p><p><ruby class="hanja api grade70"><rb>人口</rb><rp>(</rp><rt>인구</rt><rp>)</rp></ruby>는 1183<ruby class="hanja api grade80"><rb>萬</rb><rp>(</rp><rt>만</rt><rp>)</rp></ruby> 2<ruby class="hanja api grade70"><rb>千</rb><rp>(</rp><rt>천</rt><rp>)</rp></ruby> <ruby class="hanja api grade72"><rb>名</rb><rp>(</rp><rt>명</rt><rp>)</rp></ruby>입니다. <ruby class="hanja api grade32"><rb>美洲</rb><rp>(</rp><rt>미주</rt><rp>)</rp></ruby> <ruby class="hanja api grade50"><rb>原住民</rb><rp>(</rp><rt>원주민</rt><rp>)</rp></ruby>은 54%, <ruby class="hanja api grade50"><rb>原住民</rb><rp>(</rp><rt>원주민</rt><rp>)</rp></ruby>-<ruby class="hanja api grade10"><rb>歐羅巴</rb><rp>(</rp><rt>구라파</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>混血</rb><rp>(</rp><rt>혼혈</rt><rp>)</rp></ruby>은 31%, <ruby class="hanja api grade80"><rb>白人</rb><rp>(</rp><rt>백인</rt><rp>)</rp></ruby>은 15%이다. <ruby class="hanja api grade62"><rb>公用語</rb><rp>(</rp><rt>공용어</rt><rp>)</rp></ruby>는 <ruby class="hanja api grade32"><rb>西班牙語</rb><rp>(</rp><rt>서반아어</rt><rp>)</rp></ruby>와 케추아<ruby class="hanja api grade70"><rb>語</rb><rp>(</rp><rt>어</rt><rp>)</rp></ruby> 아이말라<ruby class="hanja api grade70"><rb>語</rb><rp>(</rp><rt>어</rt><rp>)</rp></ruby> <ruby class="hanja api grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby> 36<ruby class="hanja api grade42"><rb>個</rb><rp>(</rp><rt>개</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade60"><rb>本土</rb><rp>(</rp><rt>본토</rt><rp>)</rp></ruby> <ruby class="hanja api grade60"><rb>民族</rb><rp>(</rp><rt>민족</rt><rp>)</rp></ruby> <ruby class="hanja api grade60"><rb>言語</rb><rp>(</rp><rt>언어</rt><rp>)</rp></ruby>이다. <ruby class="hanja api grade62"><rb>大部分</rb><rp>(</rp><rt>대부분</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade70"><rb>住民</rb><rp>(</rp><rt>주민</rt><rp>)</rp></ruby>들은 <a href="/wiki/%E5%A4%A9%E4%B8%BB%E6%95%8E" class="mw-redirect" title="天主敎"><ruby class="hanja api grade70"><rb>天主敎</rb><rp>(</rp><rt>천주교</rt><rp>)</rp></ruby></a>를 믿는다.
