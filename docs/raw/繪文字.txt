<h2><span id="槪要(개요)"></span><span class="mw-headline" id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"><ruby class="hanja grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E7%B9%AA%E6%96%87%E5%AD%97&amp;action=edit&amp;section=1" title="부분 편집: 槪要(개요)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja grade10"><rb>繪文字</rb><rp>(</rp><rt>회문자</rt><rp>)</rp></ruby>(<ruby class="hanja grade70"><rb>日語</rb><rp>(</rp><rt>일어</rt><rp>)</rp></ruby>: <ruby class="hanja grade0"><rb>絵文字</rb><rp>(</rp><rt>회문자</rt><rp>)</rp></ruby>; えもじ, 에모지)는 <ruby class="hanja grade60"><rb>日本</rb><rp>(</rp><rt>일본</rt><rp>)</rp></ruby>에서 <ruby class="hanja grade60"><rb>開發</rb><rp>(</rp><rt>개발</rt><rp>)</rp></ruby>된 <a href="/index.php?title=%EB%94%94%EC%A7%80%ED%84%B8&amp;action=edit&amp;redlink=1" class="new" title="디지털 (없는 문서)">디지털</a> <a href="/index.php?title=%EA%B7%B8%EB%A6%BC%E6%96%87%E5%AD%97&amp;action=edit&amp;redlink=1" class="new" title="그림文字 (없는 문서)">그림<ruby class="hanja grade70 unknown"><rb>文字</rb><rp>(</rp><rt>문자</rt><rp>)</rp></ruby></a> <ruby class="hanja grade50"><rb>技術</rb><rp>(</rp><rt>기술</rt><rp>)</rp></ruby>이다. <ruby class="hanja grade32"><rb>直譯</rb><rp>(</rp><rt>직역</rt><rp>)</rp></ruby>하면 그림(<a href="/index.php?title=%E7%B9%AA&amp;action=edit&amp;redlink=1" class="new" title="繪 (없는 문서)"><ruby class="hanja grade10 unknown"><rb>繪</rb><rp>(</rp><rt>회</rt><rp>)</rp></ruby></a>) <ruby class="hanja grade70"><rb>文字</rb><rp>(</rp><rt>문자</rt><rp>)</rp></ruby>가 된다.
</p>
<h2><span id="名稱(명칭)"></span><span class="mw-headline" id=".E5.90.8D.E7.A8.B1.28.EB.AA.85.EC.B9.AD.29"><ruby class="hanja grade40"><rb>名稱</rb><rp>(</rp><rt>명칭</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E7%B9%AA%E6%96%87%E5%AD%97&amp;action=edit&amp;section=2" title="부분 편집: 名稱(명칭)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/index.php?title=%E8%8B%B1%E6%96%87&amp;action=edit&amp;redlink=1" class="new" title="英文 (없는 문서)"><ruby class="hanja grade60 unknown"><rb>英文</rb><rp>(</rp><rt>영문</rt><rp>)</rp></ruby></a><ruby class="hanja grade62"><rb>表記</rb><rp>(</rp><rt>표기</rt><rp>)</rp></ruby>인 ‘Emoji’에 <ruby class="hanja grade40"><rb>依</rb><rp>(</rp><rt>의</rt><rp>)</rp></ruby>해 <del>꽤 그럴듯하게</del> <ruby class="hanja grade32"><rb>錯覺</rb><rp>(</rp><rt>착각</rt><rp>)</rp></ruby>되어 ‘이모지’라고 불리기도 하지만, <ruby class="hanja grade50"><rb>原語</rb><rp>(</rp><rt>원어</rt><rp>)</rp></ruby>가 <a href="/wiki/%E6%97%A5%E6%9C%AC%E8%AA%9E" title="日本語"><ruby class="hanja grade60"><rb>日本語</rb><rp>(</rp><rt>일본어</rt><rp>)</rp></ruby></a>의 ‘<ruby class="hanja grade0"><rb>絵文字</rb><rp>(</rp><rt>회문자</rt><rp>)</rp></ruby>’이기에 이 위키에서는 <ruby class="hanja grade10"><rb>繪文字</rb><rp>(</rp><rt>회문자</rt><rp>)</rp></ruby>로 <ruby class="hanja grade40"><rb>呼稱</rb><rp>(</rp><rt>호칭</rt><rp>)</rp></ruby>한다.
</p>
<h2><span id="이모티콘과의_差異(차이)"></span><span class="mw-headline" id=".EC.9D.B4.EB.AA.A8.ED.8B.B0.EC.BD.98.EA.B3.BC.EC.9D.98_.E5.B7.AE.E7.95.B0.28.EC.B0.A8.EC.9D.B4.29">이모티콘과의 <ruby class="hanja grade40"><rb>差異</rb><rp>(</rp><rt>차이</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E7%B9%AA%E6%96%87%E5%AD%97&amp;action=edit&amp;section=3" title="부분 편집: 이모티콘과의 差異(차이)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja grade60"><rb>英語表記</rb><rp>(</rp><rt>영어표기</rt><rp>)</rp></ruby>의 <ruby class="hanja grade62"><rb>發音</rb><rp>(</rp><rt>발음</rt><rp>)</rp></ruby>인 ‘이모지’로 <ruby class="hanja grade50"><rb>因</rb><rp>(</rp><rt>인</rt><rp>)</rp></ruby>해 <a href="/index.php?title=%EC%9D%B4%EB%AA%A8%ED%8B%B0%EC%BD%98&amp;action=edit&amp;redlink=1" class="new" title="이모티콘 (없는 문서)">이모티콘</a>과 <ruby class="hanja grade40"><rb>混同</rb><rp>(</rp><rt>혼동</rt><rp>)</rp></ruby>되기도 한다. 하지만 <ruby class="hanja grade50"><rb>完全</rb><rp>(</rp><rt>완전</rt><rp>)</rp></ruby>히 다른 <ruby class="hanja grade32"><rb>槪念</rb><rp>(</rp><rt>개념</rt><rp>)</rp></ruby>인데, 이모티콘은 <a href="/index.php?title=%E9%A1%94%E6%96%87%E5%AD%97&amp;action=edit&amp;redlink=1" class="new" title="顔文字 (없는 문서)"><ruby class="hanja grade32 unknown"><rb>顔文字</rb><rp>(</rp><rt>안문자</rt><rp>)</rp></ruby></a>나 <a href="/index.php?title=%EC%95%84%EC%8A%A4%ED%82%A4_%EC%95%84%ED%8A%B8&amp;action=edit&amp;redlink=1" class="new" title="아스키 아트 (없는 문서)">아스키 아트</a>와 같이 이미 <ruby class="hanja grade40"><rb>存在</rb><rp>(</rp><rt>존재</rt><rp>)</rp></ruby>하는 <a href="/wiki/%E6%96%87%E5%AD%97" title="文字"><ruby class="hanja grade70"><rb>文字</rb><rp>(</rp><rt>문자</rt><rp>)</rp></ruby></a>의 <ruby class="hanja grade40"><rb>組合</rb><rp>(</rp><rt>조합</rt><rp>)</rp></ruby>으로 <a href="/index.php?title=%EA%B7%B8%EB%A6%BC&amp;action=edit&amp;redlink=1" class="new" title="그림 (없는 문서)">그림</a> 같이 나타내는 <ruby class="hanja grade60"><rb>方式</rb><rp>(</rp><rt>방식</rt><rp>)</rp></ruby>이지만 <ruby class="hanja grade10"><rb>繪文字</rb><rp>(</rp><rt>회문자</rt><rp>)</rp></ruby>는 그림 하나하나가 하나의 <ruby class="hanja grade70"><rb>文字</rb><rp>(</rp><rt>문자</rt><rp>)</rp></ruby>로 <ruby class="hanja grade52"><rb>分類</rb><rp>(</rp><rt>분류</rt><rp>)</rp></ruby>되는 <ruby class="hanja grade70"><rb>文字</rb><rp>(</rp><rt>문자</rt><rp>)</rp></ruby> <ruby class="hanja grade40"><rb>體系</rb><rp>(</rp><rt>체계</rt><rp>)</rp></ruby>라고 볼 수 있다.
</p>
