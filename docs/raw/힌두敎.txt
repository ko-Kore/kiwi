<p>힌두<ruby class="hanja api grade80"><rb>敎</rb><rp>(</rp><rt>교</rt><rp>)</rp></ruby>는 <ruby class="hanja api grade42"><rb>印度</rb><rp>(</rp><rt>인도</rt><rp>)</rp></ruby>를 <ruby class="hanja api grade70"><rb>中心</rb><rp>(</rp><rt>중심</rt><rp>)</rp></ruby>으로 <ruby class="hanja api grade52"><rb>發展</rb><rp>(</rp><rt>발전</rt><rp>)</rp></ruby>한 <ruby class="hanja api grade42"><rb>宗敎</rb><rp>(</rp><rt>종교</rt><rp>)</rp></ruby>로 브라흐미<ruby class="hanja api grade62"><rb>界</rb><rp>(</rp><rt>계</rt><rp>)</rp></ruby> <ruby class="hanja api grade42"><rb>宗敎</rb><rp>(</rp><rt>종교</rt><rp>)</rp></ruby>의 하나이다. "힌두"는 <a href="/index.php?title=%EC%82%B0%EC%8A%A4%ED%81%AC%EB%A6%AC%ED%8A%B8%E8%AA%9E&amp;action=edit&amp;redlink=1" class="new" title="산스크리트語(없는　文書）">산스크리트<ruby class="hanja api grade70 unknown"><rb>語</rb><rp>(</rp><rt>어</rt><rp>)</rp></ruby></a>에서 <ruby class="hanja api grade60"><rb>由來</rb><rp>(</rp><rt>유래</rt><rp>)</rp></ruby>한 <ruby class="hanja api grade42"><rb>單語</rb><rp>(</rp><rt>단어</rt><rp>)</rp></ruby>로 <ruby class="hanja api grade60"><rb>本來</rb><rp>(</rp><rt>본래</rt><rp>)</rp></ruby> "sindhu"이며 "큰 <ruby class="hanja api grade72"><rb>江</rb><rp>(</rp><rt>강</rt><rp>)</rp></ruby>" <ruby class="hanja api grade32"><rb>卽</rb><rp>(</rp><rt>즉</rt><rp>)</rp></ruby> 인더스<ruby class="hanja api grade72"><rb>江</rb><rp>(</rp><rt>강</rt><rp>)</rp></ruby>을 <ruby class="hanja api grade42"><rb>意味</rb><rp>(</rp><rt>의미</rt><rp>)</rp></ruby>한다. 힌두<ruby class="hanja api grade80"><rb>敎</rb><rp>(</rp><rt>교</rt><rp>)</rp></ruby>(Hinduism)는 <ruby class="hanja api grade60"><rb>英語</rb><rp>(</rp><rt>영어</rt><rp>)</rp></ruby>로 불리는 이름으로, <ruby class="hanja api grade60"><rb>信者</rb><rp>(</rp><rt>신자</rt><rp>)</rp></ruby>들은 <ruby class="hanja api grade62"><rb>自身</rb><rp>(</rp><rt>자신</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade32"><rb>信仰</rb><rp>(</rp><rt>신앙</rt><rp>)</rp></ruby>을 <ruby class="hanja api grade0"><rb>恆</rb><rp>(</rp><rt>항</rt><rp>)</rp></ruby><ruby class="hanja api grade52"><rb>法</rb><rp>(</rp><rt>법</rt><rp>)</rp></ruby><ruby class="hanja api grade80"><rb>敎</rb><rp>(</rp><rt>교</rt><rp>)</rp></ruby>(सनातन धर्म, 사나타나 다르마)라고 부른다. <ruby class="hanja api grade80"><rb>中國</rb><rp>(</rp><rt>중국</rt><rp>)</rp></ruby>에서는 <ruby class="hanja api grade42"><rb>印度敎</rb><rp>(</rp><rt>인도교</rt><rp>)</rp></ruby>라 <ruby class="hanja api grade40"><rb>稱</rb><rp>(</rp><rt>칭</rt><rp>)</rp></ruby>한다.
</p><p><ruby class="hanja api grade60"><rb>古代</rb><rp>(</rp><rt>고대</rt><rp>)</rp></ruby> 아리아<ruby class="hanja api grade80"><rb>人</rb><rp>(</rp><rt>인</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade32"><rb>信仰</rb><rp>(</rp><rt>신앙</rt><rp>)</rp></ruby><ruby class="hanja api grade40"><rb>體系</rb><rp>(</rp><rt>체계</rt><rp>)</rp></ruby>에는 데바(deva)와 아수라(asura)의 두 <ruby class="hanja api grade20"><rb>軸</rb><rp>(</rp><rt>축</rt><rp>)</rp></ruby>이 <ruby class="hanja api grade40"><rb>存在</rb><rp>(</rp><rt>존재</rt><rp>)</rp></ruby>했는데 이 <ruby class="hanja api grade80"><rb>中</rb><rp>(</rp><rt>중</rt><rp>)</rp></ruby> 데바를 <ruby class="hanja api grade42"><rb>神聖化</rb><rp>(</rp><rt>신성화</rt><rp>)</rp></ruby>하고 아수라를 <ruby class="hanja api grade20"><rb>惡魔</rb><rp>(</rp><rt>악마</rt><rp>)</rp></ruby><ruby class="hanja api grade52"><rb>化</rb><rp>(</rp><rt>화</rt><rp>)</rp></ruby>한 것이 힌두<ruby class="hanja api grade80"><rb>敎</rb><rp>(</rp><rt>교</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade20"><rb>母胎</rb><rp>(</rp><rt>모태</rt><rp>)</rp></ruby>가 되는 <a href="/index.php?title=%E5%A9%86%E7%BE%85%E9%96%80%E6%95%8E&amp;action=edit&amp;redlink=1" class="new" title="婆羅門敎(없는　文書）"><ruby class="hanja api grade10 unknown"><rb>婆羅門敎</rb><rp>(</rp><rt>파라문교</rt><rp>)</rp></ruby></a>(브라만교)이며 반대로 데바가 <ruby class="hanja api grade52"><rb>惡</rb><rp>(</rp><rt>악</rt><rp>)</rp></ruby>, 아수라가 <ruby class="hanja api grade50"><rb>善</rb><rp>(</rp><rt>선</rt><rp>)</rp></ruby>의 포지션으로 <ruby class="hanja api grade42"><rb>設定</rb><rp>(</rp><rt>설정</rt><rp>)</rp></ruby>한 것이 <a href="/index.php?title=%E6%8B%9C%E7%81%AB%E6%95%8E&amp;action=edit&amp;redlink=1" class="new" title="拜火敎(없는　文書）"><ruby class="hanja api grade42 unknown"><rb>拜火敎</rb><rp>(</rp><rt>배화교</rt><rp>)</rp></ruby></a>(조로아스터<ruby class="hanja api grade80"><rb>敎</rb><rp>(</rp><rt>교</rt><rp>)</rp></ruby>)이다.<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">[<ruby class="hanja api grade80"><rb>一</rb><rp>(</rp><rt>일</rt><rp>)</rp></ruby>]</a></sup>
</p><p>힌두<ruby class="hanja api grade80"><rb>敎</rb><rp>(</rp><rt>교</rt><rp>)</rp></ruby>에는 3<ruby class="hanja api grade50"><rb>位</rb><rp>(</rp><rt>위</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade62"><rb>主神</rb><rp>(</rp><rt>주신</rt><rp>)</rp></ruby>이 있으며 각각 브라흐마, 비슈누, 시바이다. 이를 트리무르티라 <ruby class="hanja api grade40"><rb>稱</rb><rp>(</rp><rt>칭</rt><rp>)</rp></ruby>하며 그 <ruby class="hanja api grade80"><rb>外</rb><rp>(</rp><rt>외</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade62"><rb>神</rb><rp>(</rp><rt>신</rt><rp>)</rp></ruby>들은 이 셋의 <ruby class="hanja api grade52"><rb>化身</rb><rp>(</rp><rt>화신</rt><rp>)</rp></ruby>으로 <ruby class="hanja api grade10"><rb>取扱</rb><rp>(</rp><rt>취급</rt><rp>)</rp></ruby>한다.
</p><p><ruby class="hanja api grade42"><rb>經典</rb><rp>(</rp><rt>경전</rt><rp>)</rp></ruby>으로는 <a href="/index.php?title=%EB%A6%AC%EA%B7%B8%EB%B2%A0%EB%8B%A4&amp;action=edit&amp;redlink=1" class="new" title="리그베다(없는　文書）">리그베다</a>, <a href="/index.php?title=%EC%9A%B0%ED%8C%8C%EB%8B%88%EC%83%A4%EB%93%9C&amp;action=edit&amp;redlink=1" class="new" title="우파니샤드(없는　文書）">우파니샤드</a> <ruby class="hanja api grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby> 있다.
