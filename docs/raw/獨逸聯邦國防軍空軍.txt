<p><a href="/wiki/%E7%8D%A8%E9%80%B8%E8%AA%9E" title="獨逸語"><ruby class="hanja api grade32"><rb>獨逸語</rb><rp>(</rp><rt>독일어</rt><rp>)</rp></ruby></a> 이름도 둘 다 <b>루프트바페</b>(Luftwaffe)라는 같은 이름을 가졌는데, 이것은 이 <ruby class="hanja api grade42"><rb>單語</rb><rp>(</rp><rt>단어</rt><rp>)</rp></ruby><ruby class="hanja api grade62"><rb>自體</rb><rp>(</rp><rt>자체</rt><rp>)</rp></ruby>가 <ruby class="hanja api grade32"><rb>直譯</rb><rp>(</rp><rt>직역</rt><rp>)</rp></ruby>하면 “<ruby class="hanja api grade42"><rb>航空兵</rb><rp>(</rp><rt>항공병</rt><rp>)</rp></ruby><ruby class="hanja api grade42"><rb>器</rb><rp>(</rp><rt>기</rt><rp>)</rp></ruby>”, “하늘의 <ruby class="hanja api grade42"><rb>武器</rb><rp>(</rp><rt>무기</rt><rp>)</rp></ruby>” <ruby class="hanja api grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby>의 뜻을 가지는, <ruby class="hanja api grade32"><rb>卽</rb><rp>(</rp><rt>즉</rt><rp>)</rp></ruby> <ruby class="hanja api grade72"><rb>空軍</rb><rp>(</rp><rt>공군</rt><rp>)</rp></ruby>을 뜻하는 <ruby class="hanja api grade32"><rb>獨逸語</rb><rp>(</rp><rt>독일어</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade32"><rb>普通名詞</rb><rp>(</rp><rt>보통명사</rt><rp>)</rp></ruby>이기 때문이다.
