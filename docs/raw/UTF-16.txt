<h3><span id=".EC.84.9C.EB.9F.AC.EA.B2.8C.EC.9D.B4.ED.8A.B8_.E9.9B.99.28.EC.8C.8D.29"></span><span class="mw-headline" id="서러게이트_雙(쌍)">서러게이트 <ruby class="hanja api grade32"><rb>雙</rb><rp>(</rp><rt>쌍</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=UTF-16&amp;action=edit&amp;section=2" title="部分 編輯: 서러게이트 雙(쌍)">編輯</a><span class="mw-editsection-bracket">]</span></span></h3>

<p>"<a href="/wiki/%EC%9C%A0%EB%8B%88%EC%BD%94%EB%93%9C" title="유니코드">유니코드</a>는 모든 <ruby class="hanja api grade70"><rb>文字</rb><rp>(</rp><rt>문자</rt><rp>)</rp></ruby>가 2<a href="/index.php?title=%EB%B0%94%EC%9D%B4%ED%8A%B8&amp;action=edit&amp;redlink=1" class="new" title="바이트 (없는 文書)">바이트</a>"라는 <ruby class="hanja api grade42"><rb>認識</rb><rp>(</rp><rt>인식</rt><rp>)</rp></ruby>이 있지만, 이미 옛날 얘기가 되었다 볼 수 있다. <a href="/index.php?title=UCS-2&amp;action=edit&amp;redlink=1" class="new" title="UCS-2 (없는 文書)">2바이트(16비트) <ruby class="hanja api grade50 unknown"><rb>固定</rb><rp>(</rp><rt>고정</rt><rp>)</rp></ruby>길이</a>로는 <ruby class="hanja api grade62"><rb>全世界</rb><rp>(</rp><rt>전세계</rt><rp>)</rp></ruby>에서 쓰였거나 쓰이고 있는 <a href="/wiki/%E6%96%87%E5%AD%97" title="文字"><ruby class="hanja api grade70"><rb>文字</rb><rp>(</rp><rt>문자</rt><rp>)</rp></ruby></a><ruby class="hanja api grade40"><rb>體系</rb><rp>(</rp><rt>체계</rt><rp>)</rp></ruby>를 모두 <ruby class="hanja api grade42"><rb>收錄</rb><rp>(</rp><rt>수록</rt><rp>)</rp></ruby>하기에는 택도 없이 <ruby class="hanja api grade72"><rb>不足</rb><rp>(</rp><rt>부족</rt><rp>)</rp></ruby>했기 때문이다. <sup id="cite_ref-1" class="reference"><a href="#cite_note-1">[1]</a></sup> 이에 얼마 안 남은 <ruby class="hanja api grade72"><rb>空間</rb><rp>(</rp><rt>공간</rt><rp>)</rp></ruby>을 '서러게이트 <ruby class="hanja api grade32"><rb>雙</rb><rp>(</rp><rt>쌍</rt><rp>)</rp></ruby>'이라는 이름으로 <ruby class="hanja api grade40"><rb>豫約</rb><rp>(</rp><rt>예약</rt><rp>)</rp></ruby>하여, 이 <ruby class="hanja api grade40"><rb>領域</rb><rp>(</rp><rt>영역</rt><rp>)</rp></ruby>의 코드를 두<ruby class="hanja api grade42"><rb>個</rb><rp>(</rp><rt>개</rt><rp>)</rp></ruby> 늘어놓는 <ruby class="hanja api grade40"><rb>組合</rb><rp>(</rp><rt>조합</rt><rp>)</rp></ruby>으로 한 글<ruby class="hanja api grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>를 <ruby class="hanja api grade62"><rb>表現</rb><rp>(</rp><rt>표현</rt><rp>)</rp></ruby>하게 하였다.

href="/index.php?title=%E8%A1%A8%E6%84%8F%E6%96%87%E5%AD%97%E7%AC%AC%E4%B8%89%E5%B9%B3%E9%9D%A2&amp;action=edit&amp;redlink=1" class="new" title="表意文字第三平面 (없는 文書)"><ruby class="hanja api grade70
