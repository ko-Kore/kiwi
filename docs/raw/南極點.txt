<p><a href="/index.php?title=%E5%A4%A9%E9%AB%94&amp;action=edit&amp;redlink=1" class="new" title="天體 (없는 文書)"><ruby class="hanja api grade62 unknown"><rb>天體</rb><rp>(</rp><rt>천체</rt><rp>)</rp></ruby></a>의 回轉軸이 天體의 表面과 交叉하는 點 中 하나입니다. 地球의 南極, <a href="/index.php?title=%E7%81%AB%E6%98%9F&amp;action=edit&amp;redlink=1" class="new" title="火星 (없는 文書)"><ruby class="hanja api grade42 unknown"><rb>火星</rb><rp>(</rp><rt>화성</rt><rp>)</rp></ruby></a>의 南極, <a href="/index.php?title=%E6%9C%A8%E6%98%9F&amp;action=edit&amp;redlink=1" class="new" title="木星 (없는 文書)"><ruby class="hanja api grade42 unknown"><rb>木星</rb><rp>(</rp><rt>목성</rt><rp>)</rp></ruby></a>의 南極 等으로 불립니다. 天體의 表面과 回轉軸이 交叉하는 地點이 2個 있는데, 그 地點에서 極地方별의 一周運動을 보면 時計 方向으로 나타나는 것을 南極, 時計反對方向으로 나타나는 것을 <a href="/index.php?title=%E5%8C%97%E6%A5%B5&amp;action=edit&amp;redlink=1" class="new" title="北極 (없는 文書)"><ruby class="hanja api grade42 unknown"><rb>北極</rb><rp>(</rp><rt>북극</rt><rp>)</rp></ruby></a>이라고 합니다.
