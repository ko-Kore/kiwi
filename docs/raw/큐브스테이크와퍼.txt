<h2><span id=".E6.A7.8B.E6.88.90.28.EA.B5.AC.EC.84.B1.29"></span><span class="mw-headline" id="構成(구성)"><ruby class="hanja api grade40"><rb>構成</rb><rp>(</rp><rt>구성</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%ED%81%90%EB%B8%8C%EC%8A%A4%ED%85%8C%EC%9D%B4%ED%81%AC%EC%99%80%ED%8D%BC&amp;action=edit&amp;section=1" title="부분 편집: 構成(구성)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja api grade32"><rb>一般</rb><rp>(</rp><rt>일반</rt><rp>)</rp></ruby> 와퍼패티 위에 양념된 <a href="/index.php?title=%ED%81%90%EB%B8%8C%EC%8A%A4%ED%85%8C%EC%9D%B4%ED%81%AC&amp;action=edit&amp;redlink=1" class="new" title="큐브스테이크 (없는 문서)">큐브스테이크</a>가 올라간다. <a href="/index.php?title=%EC%B9%98%EC%A6%88&amp;action=edit&amp;redlink=1" class="new" title="치즈 (없는 문서)">치즈</a>는 들어가지 않는다. <ruby class="hanja api grade32"><rb>野菜</rb><rp>(</rp><rt>야채</rt><rp>)</rp></ruby>는 <a href="/index.php?title=%E6%B4%8B%EC%83%81%EC%B6%94&amp;action=edit&amp;redlink=1" class="new" title="洋상추 (없는 문서)"><ruby class="hanja api grade0 unknown"><rb>洋상추</rb><rp>(</rp><rt>양상추</rt><rp>)</rp></ruby></a>와 함께 <a href="/index.php?title=%EA%B0%88%EB%A6%AD%EB%B6%88%EA%B3%A0%EA%B8%B0%EC%99%80%ED%8D%BC&amp;action=edit&amp;redlink=1" class="new" title="갈릭불고기와퍼 (없는 문서)">갈릭불고기와퍼</a>에 들어간 <a href="/index.php?title=%EB%A7%88%EB%8A%98&amp;action=edit&amp;redlink=1" class="new" title="마늘 (없는 문서)">마늘</a>칩과 <a href="/index.php?title=%E6%B4%8B%ED%8C%8C&amp;action=edit&amp;redlink=1" class="new" title="洋파 (없는 문서)"><ruby class="hanja api grade0 unknown"><rb>洋파</rb><rp>(</rp><rt>양파</rt><rp>)</rp></ruby></a>가 들어간다.
</p><p>고기에 고기가 더해진 <ruby class="hanja api grade40"><rb>構成</rb><rp>(</rp><rt>구성</rt><rp>)</rp></ruby> <ruby class="hanja api grade52"><rb>德</rb><rp>(</rp><rt>덕</rt><rp>)</rp></ruby>에 <a href="/index.php?title=%EC%8A%A4%ED%83%9C%EC%BB%A4%EC%99%80%ED%8D%BC&amp;action=edit&amp;redlink=1" class="new" title="스태커와퍼 (없는 문서)">스태커</a>2와퍼에 버금가는 <ruby class="hanja api grade30"><rb>飽滿度</rb><rp>(</rp><rt>포만도</rt><rp>)</rp></ruby><del>와<ruby class="hanja api grade52"><rb>價格</rb><rp>(</rp><rt>가격</rt><rp>)</rp></ruby>을</del>를 <ruby class="hanja api grade42"><rb>經驗</rb><rp>(</rp><rt>경험</rt><rp>)</rp></ruby>할수있다. 다만 <ruby class="hanja api grade32"><rb>野菜</rb><rp>(</rp><rt>야채</rt><rp>)</rp></ruby>와 소스에도 <ruby class="hanja api grade52"><rb>充實</rb><rp>(</rp><rt>충실</rt><rp>)</rp></ruby>했기 때문에 퍽퍽한 고기맛만이 아닌 <ruby class="hanja api grade32"><rb>所謂</rb><rp>(</rp><rt>소위</rt><rp>)</rp></ruby> <a href="/index.php?title=%EB%8B%A8%EC%A7%A0&amp;action=edit&amp;redlink=1" class="new" title="단짠 (없는 문서)">단짠</a>에 가깝다.
</p>
