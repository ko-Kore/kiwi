<h2><span id="槪要(개요)"></span><span class="mw-headline" id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"><ruby class="hanja grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E5%81%87%E5%90%8D%E6%96%87%E5%AD%97&amp;action=edit&amp;section=1" title="부분 편집: 槪要(개요)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja grade42"><rb>假名文字</rb><rp>(</rp><rt>가명문자</rt><rp>)</rp></ruby>는 <a href="/wiki/%E6%97%A5%E6%9C%AC%E8%AA%9E" title="日本語"><ruby class="hanja grade60"><rb>日本語</rb><rp>(</rp><rt>일본어</rt><rp>)</rp></ruby></a>를 <ruby class="hanja grade62"><rb>表記</rb><rp>(</rp><rt>표기</rt><rp>)</rp></ruby>하는 데 <ruby class="hanja grade60"><rb>使用</rb><rp>(</rp><rt>사용</rt><rp>)</rp></ruby>하는 <a href="/wiki/%E6%96%87%E5%AD%97" title="文字"><ruby class="hanja grade70"><rb>文字</rb><rp>(</rp><rt>문자</rt><rp>)</rp></ruby></a>이다. <ruby class="hanja grade42"><rb>眞名</rb><rp>(</rp><rt>진명</rt><rp>)</rp></ruby>(<ruby class="hanja grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby>)에 <ruby class="hanja grade50"><rb>對比</rb><rp>(</rp><rt>대비</rt><rp>)</rp></ruby>되는 말이다.
</p>
<h2><span id="歷史(역사)"></span><span class="mw-headline" id=".E6.AD.B7.E5.8F.B2.28.EC.97.AD.EC.82.AC.29"><ruby class="hanja grade52"><rb>歷史</rb><rp>(</rp><rt>역사</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E5%81%87%E5%90%8D%E6%96%87%E5%AD%97&amp;action=edit&amp;section=2" title="부분 편집: 歷史(역사)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%E6%BC%A2%E5%AD%97" title="漢字"><ruby class="hanja grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby></a>를 빌려 <a href="/wiki/%E6%97%A5%E6%9C%AC%E8%AA%9E" title="日本語"><ruby class="hanja grade60"><rb>日本語</rb><rp>(</rp><rt>일본어</rt><rp>)</rp></ruby></a>를<ruby class="hanja grade62"><rb>表記</rb><rp>(</rp><rt>표기</rt><rp>)</rp></ruby>한 <a href="/index.php?title=%E8%90%AC%E8%91%89%E5%81%87%E5%90%8D&amp;action=edit&amp;redlink=1" class="new" title="萬葉假名 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>萬葉假名</rb><rp>(</rp><rt>만엽가명</rt><rp>)</rp></ruby></a>이 가장 먼저 <ruby class="hanja grade70"><rb>登場</rb><rp>(</rp><rt>등장</rt><rp>)</rp></ruby>하였으며, <ruby class="hanja grade72"><rb>後</rb><rp>(</rp><rt>후</rt><rp>)</rp></ruby>에 이를 <ruby class="hanja grade40"><rb>簡略化</rb><rp>(</rp><rt>간략화</rt><rp>)</rp></ruby>한 <a href="/wiki/%E5%B9%B3%E5%81%87%E5%90%8D" title="平假名"><ruby class="hanja grade42"><rb>平假名</rb><rp>(</rp><rt>평가명</rt><rp>)</rp></ruby></a>과 <a href="/wiki/%E7%89%87%E5%81%87%E5%90%8D" title="片假名"><ruby class="hanja grade32"><rb>片假名</rb><rp>(</rp><rt>편가명</rt><rp>)</rp></ruby></a>이 생겨났고, 그 <ruby class="hanja grade42"><rb>過程</rb><rp>(</rp><rt>과정</rt><rp>)</rp></ruby>에서 생겨났으나, <a href="/index.php?title=%E6%98%8E%E6%B2%BB%E6%99%82%E4%BB%A3&amp;action=edit&amp;redlink=1" class="new" title="明治時代 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>明治時代</rb><rp>(</rp><rt>명치시대</rt><rp>)</rp></ruby></a>에 <ruby class="hanja grade40"><rb>整理</rb><rp>(</rp><rt>정리</rt><rp>)</rp></ruby>되어 더 <ruby class="hanja grade52"><rb>以上</rb><rp>(</rp><rt>이상</rt><rp>)</rp></ruby> 쓰이지 않는 <ruby class="hanja grade62"><rb>字體</rb><rp>(</rp><rt>자체</rt><rp>)</rp></ruby>의 글<ruby class="hanja grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>들을 <a href="/index.php?title=%E8%AE%8A%E9%AB%94%E5%81%87%E5%90%8D&amp;action=edit&amp;redlink=1" class="new" title="變體假名 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>變體假名</rb><rp>(</rp><rt>변체가명</rt><rp>)</rp></ruby></a>이라 한다.
</p>
