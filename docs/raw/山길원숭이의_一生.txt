<h2><span id=".E7.89.B9.E5.BE.B5.28.ED.8A.B9.EC.A7.95.29"></span><span class="mw-headline" id="特徵(특징)"><ruby class="hanja api grade32"><rb>特徵</rb><rp>(</rp><rt>특징</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E5%B1%B1%EA%B8%B8%EC%9B%90%EC%88%AD%EC%9D%B4%EC%9D%98_%E4%B8%80%E7%94%9F&amp;action=edit&amp;section=1" title="部分 編輯: 特徵(특징)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>이 <ruby class="hanja api grade32"><rb>映像</rb><rp>(</rp><rt>영상</rt><rp>)</rp></ruby> 속 캐릭터는 모두 <a href="/index.php?title=Wojak&amp;action=edit&amp;redlink=1" class="new" title="Wojak (없는 文書)">Wojak</a> 밈 <ruby class="hanja api grade30"><rb>漫畵</rb><rp>(</rp><rt>만화</rt><rp>)</rp></ruby> 캐릭터로 <ruby class="hanja api grade62"><rb>表現</rb><rp>(</rp><rt>표현</rt><rp>)</rp></ruby>된다.

<p><a href="/wiki/%E4%BE%BF%E5%AE%9C%E5%BA%97" title="便宜店"><ruby class="hanja api grade30"><rb>便宜店</rb><rp>(</rp><rt>편의점</rt><rp>)</rp></ruby></a>에서 <ruby class="hanja api grade32"><rb>逸話</rb><rp>(</rp><rt>일화</rt><rp>)</rp></ruby>는 <ruby class="hanja api grade80"><rb>靑年</rb><rp>(</rp><rt>청년</rt><rp>)</rp></ruby>이 부풀려진 <ruby class="hanja api grade42"><rb>虛榮心</rb><rp>(</rp><rt>허영심</rt><rp>)</rp></ruby>으로 <ruby class="hanja api grade50"><rb>因</rb><rp>(</rp><rt>인</rt><rp>)</rp></ruby>해 <ruby class="hanja api grade42"><rb>關係</rb><rp>(</rp><rt>관계</rt><rp>)</rp></ruby>에서 <ruby class="hanja api grade62"><rb>自信</rb><rp>(</rp><rt>자신</rt><rp>)</rp></ruby>을 잃어버리고 <ruby class="hanja api grade52"><rb>結局</rb><rp>(</rp><rt>결국</rt><rp>)</rp></ruby> <ruby class="hanja api grade60"><rb>家族</rb><rp>(</rp><rt>가족</rt><rp>)</rp></ruby>을 <ruby class="hanja api grade42"><rb>背信</rb><rp>(</rp><rt>배신</rt><rp>)</rp></ruby>하고 <ruby class="hanja api grade80"><rb>山</rb><rp>(</rp><rt>산</rt><rp>)</rp></ruby>길에서 <ruby class="hanja api grade42"><rb>競走</rb><rp>(</rp><rt>경주</rt><rp>)</rp></ruby>를 하던 <ruby class="hanja api grade80"><rb>中</rb><rp>(</rp><rt>중</rt><rp>)</rp></ruby> <ruby class="hanja api grade42"><rb>事故</rb><rp>(</rp><rt>사고</rt><rp>)</rp></ruby>로 <ruby class="hanja api grade50"><rb>死亡</rb><rp>(</rp><rt>사망</rt><rp>)</rp></ruby>하는 이야기다.
