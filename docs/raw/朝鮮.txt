<h2><span id="槪要(개요)"></span><span class="mw-headline" id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"><ruby class="hanja grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%9C%9D%E9%AE%AE&amp;action=edit&amp;section=1" title="부분 편집: 槪要(개요)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja grade52"><rb>朝鮮</rb><rp>(</rp><rt>조선</rt><rp>)</rp></ruby>은 <a href="/index.php?title=1392%E5%B9%B4&amp;action=edit&amp;redlink=1" class="new" title="1392年 (없는 문서)">1392<ruby class="hanja grade80 unknown"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby></a> <a href="/index.php?title=%E6%9D%8E%E6%88%90%E6%A1%82&amp;action=edit&amp;redlink=1" class="new" title="李成桂 (없는 문서)"><ruby class="hanja grade32 unknown"><rb>李成桂</rb><rp>(</rp><rt>이성계</rt><rp>)</rp></ruby></a>에 <ruby class="hanja grade40"><rb>依</rb><rp>(</rp><rt>의</rt><rp>)</rp></ruby>해 <ruby class="hanja grade50"><rb>建國</rb><rp>(</rp><rt>건국</rt><rp>)</rp></ruby>되어 <a href="/index.php?title=1910%E5%B9%B4&amp;action=edit&amp;redlink=1" class="new" title="1910年 (없는 문서)">1910<ruby class="hanja grade80 unknown"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby></a> <a href="/wiki/%E6%97%A5%E6%9C%AC" title="日本"><ruby class="hanja grade60"><rb>日本</rb><rp>(</rp><rt>일본</rt><rp>)</rp></ruby></a>에 <ruby class="hanja grade40"><rb>依</rb><rp>(</rp><rt>의</rt><rp>)</rp></ruby>해 <ruby class="hanja grade20"><rb>强制倂合</rb><rp>(</rp><rt>강제병합</rt><rp>)</rp></ruby>되어 <ruby class="hanja grade32"><rb>滅亡</rb><rp>(</rp><rt>멸망</rt><rp>)</rp></ruby>하기까지 <a href="/index.php?title=%E9%9F%93%E5%8D%8A%E5%B3%B6&amp;action=edit&amp;redlink=1" class="new" title="韓半島 (없는 문서)"><ruby class="hanja grade50 unknown"><rb>韓半島</rb><rp>(</rp><rt>한반도</rt><rp>)</rp></ruby></a>에 <ruby class="hanja grade40"><rb>存在</rb><rp>(</rp><rt>존재</rt><rp>)</rp></ruby>하였던 <a href="/index.php?title=%E5%B0%88%E5%88%B6%E5%90%9B%E4%B8%BB%E5%9C%8B&amp;action=edit&amp;redlink=1" class="new" title="專制君主國 (없는 문서)"><ruby class="hanja grade40 unknown"><rb>專制君主國</rb><rp>(</rp><rt>전제군주국</rt><rp>)</rp></ruby></a>이다.
</p>
<h2><span id="歷史(역사)"></span><span class="mw-headline" id=".E6.AD.B7.E5.8F.B2.28.EC.97.AD.EC.82.AC.29"><ruby class="hanja grade52"><rb>歷史</rb><rp>(</rp><rt>역사</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%9C%9D%E9%AE%AE&amp;action=edit&amp;section=2" title="부분 편집: 歷史(역사)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>크게 <a href="/index.php?title=%E5%A3%AC%E8%BE%B0%E5%80%AD%E4%BA%82&amp;action=edit&amp;redlink=1" class="new" title="壬辰倭亂 (없는 문서)"><ruby class="hanja grade12 unknown"><rb>壬辰倭亂</rb><rp>(</rp><rt>임진왜란</rt><rp>)</rp></ruby></a>을 두고 <ruby class="hanja grade50"><rb>前期</rb><rp>(</rp><rt>전기</rt><rp>)</rp></ruby>와 <ruby class="hanja grade50"><rb>後期</rb><rp>(</rp><rt>후기</rt><rp>)</rp></ruby>로 <ruby class="hanja grade60"><rb>區分</rb><rp>(</rp><rt>구분</rt><rp>)</rp></ruby>된다.
