<html><head></head><body><div class="mw-parser-output"><p><b>나비 키보드(버터플라이 키보드)</b>는 다음을 <ruby class="hanja grade42"><rb>意味</rb><rp>(</rp><rt>의미</rt><rp>)</rp></ruby>한다.
</p>
<h2><span id="IBM_.EC.94.BD.ED.81.AC.ED.8C.A8.EB.93.9C_701"></span><span class="mw-headline" id="IBM_씽크패드_701">IBM 씽크패드 701</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EB%82%98%EB%B9%84_%ED%82%A4%EB%B3%B4%EB%93%9C&amp;action=edit&amp;section=1" title="부분 편집: IBM 씽크패드 701">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright"><div class="thumbinner" style="width:302px;"><a href="/wiki/%ED%8C%8C%EC%9D%BC:Thinkpad_701.jpg" class="image"><img alt="Thinkpad 701.jpg" src="https://upload.wikimedia.org/wikipedia/commons/thumb/e/e8/Thinkpad_701.jpg/300px-Thinkpad_701.jpg" decoding="async" width="300" height="255" class="thumbimage" srcset="https://upload.wikimedia.org/wikipedia/commons/thumb/e/e8/Thinkpad_701.jpg/450px-Thinkpad_701.jpg 1.5x, https://upload.wikimedia.org/wikipedia/commons/thumb/e/e8/Thinkpad_701.jpg/600px-Thinkpad_701.jpg 2x"></a>  <div class="thumbcaption"><div class="magnify"><a href="/wiki/%ED%8C%8C%EC%9D%BC:Thinkpad_701.jpg" class="internal" title="實際 크기로"></a></div></div></div></div>
<p>10.4인치 <ruby class="hanja grade60"><rb>畵面</rb><rp>(</rp><rt>화면</rt><rp>)</rp></ruby>을 가진 <ruby class="hanja grade20"><rb>小型</rb><rp>(</rp><rt>소형</rt><rp>)</rp></ruby>노트북에 두조각으로된 키보드가 노트북을 접었을때에는 뚜껑 안쪽에 <ruby class="hanja grade40"><rb>收納</rb><rp>(</rp><rt>수납</rt><rp>)</rp></ruby>되어있다가 노트북을 열면 <ruby class="hanja grade72"><rb>左右</rb><rp>(</rp><rt>좌우</rt><rp>)</rp></ruby>로 펼쳐져 <a href="/index.php?title=%ED%92%80%EC%82%AC%EC%9D%B4%EC%A6%88_%ED%82%A4%EB%B3%B4%EB%93%9C&amp;action=edit&amp;redlink=1" class="new" title="풀사이즈 키보드 (없는 문서)">풀사이즈 키보드</a>가 되는 <ruby class="hanja grade40"><rb>構造</rb><rp>(</rp><rt>구조</rt><rp>)</rp></ruby>이다.
</p><p>노트북을 펼치면 접혀있던 키보드가 <ruby class="hanja grade72"><rb>左右</rb><rp>(</rp><rt>좌우</rt><rp>)</rp></ruby>로 펼쳐지는 모양이 <a href="/index.php?title=%EB%82%98%EB%B9%84&amp;action=edit&amp;redlink=1" class="new" title="나비 (없는 문서)">나비</a>가 날개를 펼치는 모습같다 하여 붙은 <ruby class="hanja grade60"><rb>別名</rb><rp>(</rp><rt>별명</rt><rp>)</rp></ruby>이다.
</p>
<h2><span id=".EC.95.A0.ED.94.8C_.EB.A0.88.ED.8B.B0.EB.82.98_.EB.A7.A5.EB.B6.81_.E5.A4.96.28.EC.99.B8.29"></span><span class="mw-headline" id="애플_레티나_맥북_外(외)">애플 레티나 맥북 <ruby class="hanja grade80"><rb>外</rb><rp>(</rp><rt>외</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EB%82%98%EB%B9%84_%ED%82%A4%EB%B3%B4%EB%93%9C&amp;action=edit&amp;section=2" title="부분 편집: 애플 레티나 맥북 外(외)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/index.php?title=2015%E5%B9%B4&amp;action=edit&amp;redlink=1" class="new" title="2015年 (없는 문서)">2015<ruby class="hanja grade80 unknown"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby></a> <ruby class="hanja grade70"><rb>出市</rb><rp>(</rp><rt>출시</rt><rp>)</rp></ruby>된 12인치 <a href="/index.php?title=%EB%A7%A5%EB%B6%81_(2015)&amp;action=edit&amp;redlink=1" class="new" title="맥북 (2015) (없는 문서)">맥북</a>에 처음으로 <ruby class="hanja grade40"><rb>適用</rb><rp>(</rp><rt>적용</rt><rp>)</rp></ruby>되어 <a href="/wiki/%EB%A7%A5%EB%B6%81%ED%94%84%EB%A1%9C" class="mw-redirect" title="맥북프로">맥북프로</a>, <a href="/wiki/%EB%A7%A5%EB%B6%81%EC%97%90%EC%96%B4" class="mw-redirect" title="맥북에어">맥북에어</a>로 <ruby class="hanja grade32"><rb>漸次</rb><rp>(</rp><rt>점차</rt><rp>)</rp></ruby> <ruby class="hanja grade30"><rb>擴大</rb><rp>(</rp><rt>확대</rt><rp>)</rp></ruby>되어 <ruby class="hanja grade10"><rb>搭載</rb><rp>(</rp><rt>탑재</rt><rp>)</rp></ruby>된 키보드로, <ruby class="hanja grade40"><rb>普通</rb><rp>(</rp><rt>보통</rt><rp>)</rp></ruby> 노트북에 <ruby class="hanja grade40"><rb>適用</rb><rp>(</rp><rt>적용</rt><rp>)</rp></ruby>되는 <a href="/index.php?title=%EA%B0%80%EC%9C%84%E5%BC%8F_%ED%82%A4%EB%B3%B4%EB%93%9C&amp;action=edit&amp;redlink=1" class="new" title="가위式 키보드 (없는 문서)">가위<ruby class="hanja grade60 unknown"><rb>式</rb><rp>(</rp><rt>식</rt><rp>)</rp></ruby> 키보드</a>보다도 키보드를 얇게 만들어서 노트북을 <ruby class="hanja grade42"><rb>極限</rb><rp>(</rp><rt>극한</rt><rp>)</rp></ruby>으로 <ruby class="hanja grade50"><rb>輕量化</rb><rp>(</rp><rt>경량화</rt><rp>)</rp></ruby>하기<ruby class="hanja grade42"><rb>爲</rb><rp>(</rp><rt>위</rt><rp>)</rp></ruby>해 <ruby class="hanja grade60"><rb>開發</rb><rp>(</rp><rt>개발</rt><rp>)</rp></ruby>되었다.
</p><p><ruby class="hanja grade62"><rb>各</rb><rp>(</rp><rt>각</rt><rp>)</rp></ruby>키의 <ruby class="hanja grade40"><rb>內部構造</rb><rp>(</rp><rt>내부구조</rt><rp>)</rp></ruby>가 <a href="/index.php?title=%EB%82%98%EB%B9%84&amp;action=edit&amp;redlink=1" class="new" title="나비 (없는 문서)">나비</a>처럼 생겼다 하여 <ruby class="hanja grade52"><rb>公式的</rb><rp>(</rp><rt>공식적</rt><rp>)</rp></ruby>으로 '나비 메커니즘'이라 <ruby class="hanja grade70"><rb>命名</rb><rp>(</rp><rt>명명</rt><rp>)</rp></ruby>되었다.
</p>
