</p><p>1874<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby>부터 <a href="/wiki/%E8%8B%B1%E5%9C%8B" title="英國"><ruby class="hanja api grade60"><rb>英國</rb><rp>(</rp><rt>영국</rt><rp>)</rp></ruby></a>의 <a href="/index.php?title=%E6%A4%8D%E6%B0%91%E5%9C%B0&amp;action=edit&amp;redlink=1" class="new" title="植民地 (없는 文書)"><ruby class="hanja api grade70 unknown"><rb>植民地</rb><rp>(</rp><rt>식민지</rt><rp>)</rp></ruby></a>였으며, 1970<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade52"><rb>獨立</rb><rp>(</rp><rt>독립</rt><rp>)</rp></ruby>했다. <a href="/wiki/%E8%8B%B1%E8%81%AF%E9%82%A6" title="英聯邦"><ruby class="hanja api grade30"><rb>英聯邦</rb><rp>(</rp><rt>영연방</rt><rp>)</rp></ruby></a>의 <ruby class="hanja api grade42"><rb>一員</rb><rp>(</rp><rt>일원</rt><rp>)</rp></ruby>이었으나 2009<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> 9<ruby class="hanja api grade80"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby>부터 2013<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> 8<ruby class="hanja api grade80"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby>까지 <ruby class="hanja api grade72"><rb>一時</rb><rp>(</rp><rt>일시</rt><rp>)</rp></ruby><ruby class="hanja api grade40"><rb>資格停止</rb><rp>(</rp><rt>자격정지</rt><rp>)</rp></ruby>. <a href="/wiki/%EC%BF%A0%EB%8D%B0%ED%83%80" title="쿠데타">쿠데타</a>로 <ruby class="hanja api grade42"><rb>政治體制</rb><rp>(</rp><rt>정치체제</rt><rp>)</rp></ruby>가 <ruby class="hanja api grade40"><rb>混亂</rb><rp>(</rp><rt>혼란</rt><rp>)</rp></ruby>을 겪은 적이 있다.
</p><p><ruby class="hanja api grade80"><rb>東</rb><rp>(</rp><rt>동</rt><rp>)</rp></ruby>쪽으로는 <a href="/index.php?title=%E6%B9%AF%E5%8A%A0%E7%8E%8B%E5%9C%8B&amp;action=edit&amp;redlink=1" class="new" title="湯加王國 (없는 文書)"><ruby class="hanja api grade32 unknown"><rb>湯加</rb><rp>(</rp><rt>탕가</rt><rp>)</rp></ruby><ruby class="hanja api grade80 unknown"><rb>王國</rb><rp>(</rp><rt>왕국</rt><rp>)</rp></ruby></a>, <a href="/index.php?title=%EC%99%88%EB%A6%AC%EC%8A%A4_%ED%93%8C%ED%8A%80%EB%82%98&amp;action=edit&amp;redlink=1" class="new" title="왈리스 퓌튀나 (없는 文書)">왈리스 퓌튀나</a>와 <a href="/index.php?title=%EA%B0%80%EA%B9%9D%EA%B3%A0&amp;action=edit&amp;redlink=1" class="new" title="가깝고 (없는 文書)">가깝고</a>, <ruby class="hanja api grade52"><rb>世界的</rb><rp>(</rp><rt>세계적</rt><rp>)</rp></ruby>으로는 작은 나라이지만 <ruby class="hanja api grade32"><rb>大洋洲</rb><rp>(</rp><rt>대양주</rt><rp>)</rp></ruby>에서는 <a href="/wiki/%E6%BF%A0%E6%B4%B2" title="濠洲"><ruby class="hanja api grade20"><rb>濠洲</rb><rp>(</rp><rt>호주</rt><rp>)</rp></ruby></a>, <a href="/index.php?title=%E5%B7%B4%E5%B8%83%E4%BA%9E%E6%96%B0%E5%B9%BE%E5%85%A7%E4%BA%9E%E7%8D%A8%E7%AB%8B%E5%9C%8B&amp;action=edit&amp;redlink=1" class="new" title="巴布亞新幾內亞獨立國 (없는 文書)"><ruby class="hanja api grade10 unknown"><rb>巴布亞新幾內亞</rb><rp>(</rp><rt>파포아신기내아</rt><rp>)</rp></ruby><ruby class="hanja api grade52 unknown"><rb>獨立國</rb><rp>(</rp><rt>독립국</rt><rp>)</rp></ruby></a>, <a href="/wiki/%E6%96%B0%E8%A5%BF%E8%98%AD" title="新西蘭"><ruby class="hanja api grade32"><rb>新西蘭</rb><rp>(</rp><rt>신서란토</rt><rp>)</rp></ruby></a>, <a href="/wiki/%E6%89%80%E7%BE%85%E9%96%80%E8%AB%B8%E5%B3%B6" title="所羅門諸島"><ruby class="hanja api grade32"><rb>所羅門諸島</rb><rp>(</rp><rt>소라문제도</rt><rp>)</rp></ruby></a>에 이어 5<ruby class="hanja api grade60"><rb>番</rb><rp>(</rp><rt>번</rt><rp>)</rp></ruby>째로 큰 나라이다. 
</p><p><ruby class="hanja api grade80"><rb>國土</rb><rp>(</rp><rt>국토</rt><rp>)</rp></ruby><ruby class="hanja api grade40"><rb>面積</rb><rp>(</rp><rt>면적</rt><rp>)</rp></ruby>은 18,376km²로 <a href="/wiki/%E6%BF%9F%E5%B7%9E%E7%89%B9%E5%88%A5%E8%87%AA%E6%B2%BB%E9%81%93" title="濟州特別自治道"><ruby class="hanja api grade42"><rb>濟州</rb><rp>(</rp><rt>제주</rt><rp>)</rp></ruby><ruby class="hanja api grade60"><rb>特別</rb><rp>(</rp><rt>특별</rt><rp>)</rp></ruby><ruby class="hanja api grade42"><rb>自治</rb><rp>(</rp><rt>자치</rt><rp>)</rp></ruby><ruby class="hanja api grade72"><rb>道</rb><rp>(</rp><rt>도</rt><rp>)</rp></ruby></a>의 9<ruby class="hanja api grade50"><rb>倍</rb><rp>(</rp><rt>배</rt><rp>)</rp></ruby> 크기로, <a href="/wiki/%E6%85%B6%E5%B0%99%E5%8C%97%E9%81%93" title="慶尙北道"><ruby class="hanja api grade32"><rb>慶尙北道</rb><rp>(</rp><rt>경상북도</rt><rp>)</rp></ruby></a>보다 <ruby class="hanja api grade32"><rb>若干</rb><rp>(</rp><rt>약간</rt><rp>)</rp></ruby> 작다. <a href="/index.php?title=%E4%BA%BA%E5%8F%A3&amp;action=edit&amp;redlink=1" class="new" title="人口 (없는 文書)"><ruby class="hanja api grade70 unknown"><rb>人口</rb><rp>(</rp><rt>인구</rt><rp>)</rp></ruby></a>는 934,435<ruby class="hanja api grade72"><rb>名</rb><rp>(</rp><rt>명</rt><rp>)</rp></ruby>으로 성남시와 비슷하다.
