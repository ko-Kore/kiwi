<html><head></head><body><div class="mw-parser-output"><p><b><ruby class="hanja api grade60"><rb>英國</rb><rp>(</rp><rt>영국</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade70"><rb>國歌</rb><rp>(</rp><rt>국가</rt><rp>)</rp></ruby></b>는 헨리 캐리가 <ruby class="hanja api grade50"><rb>作曲</rb><rp>(</rp><rt>작곡</rt><rp>)</rp></ruby>한 노래이다.
</p><p>또한 이 노래는 <a href="/wiki/%E5%8A%A0%E5%A5%88%E9%99%80" title="加奈陀"><ruby class="hanja api grade50"><rb>加</rb><rp>(</rp><rt>가</rt><rp>)</rp></ruby><ruby class="hanja api grade30"><rb>奈</rb><rp>(</rp><rt>내</rt><rp>)</rp></ruby><ruby class="hanja api grade10"><rb>陀</rb><rp>(</rp><rt>타</rt><rp>)</rp></ruby></a>, <a href="/index.php?title=%E6%96%B0%E8%A5%BF%E8%98%AD&amp;action=edit&amp;redlink=1" class="new" title="新西蘭 (없는 文書)"><ruby class="hanja api grade32 unknown"><rb>新西蘭</rb><rp>(</rp><rt>신서란토</rt><rp>)</rp></ruby></a> 및 <a href="/wiki/%E6%BF%A0%E6%B4%B2" title="濠洲"><ruby class="hanja api grade20"><rb>濠洲</rb><rp>(</rp><rt>호주</rt><rp>)</rp></ruby></a> <b><ruby class="hanja api grade32"><rb>皇室</rb><rp>(</rp><rt>황실</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade40"><rb>頌歌</rb><rp>(</rp><rt>송가</rt><rp>)</rp></ruby></b>로 <ruby class="hanja api grade60"><rb>使用</rb><rp>(</rp><rt>사용</rt><rp>)</rp></ruby>됩니다
</p>
<h2><span id=".E6.AD.8C.E8.A9.9E.28.EA.B0.80.EC.82.AC.29"></span><span class="mw-headline" id="歌詞(가사)"><ruby class="hanja api grade32"><rb>歌詞</rb><rp>(</rp><rt>가사</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E8%8B%B1%E5%9C%8B%EC%9D%98_%E5%9C%8B%E6%AD%8C&amp;action=edit&amp;section=1" title="部分 編輯: 歌詞(가사)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span id=".E5.9C.8B.E7.8E.8B.28.EA.B5.AD.EC.99.95.29_.E6.99.82.E6.9C.9F.28.EC.8B.9C.EA.B8.B0.29"></span><span class="mw-headline" id="國王(국왕)_時期(시기)"><ruby class="hanja api grade80"><rb>國王</rb><rp>(</rp><rt>국왕</rt><rp>)</rp></ruby> <ruby class="hanja api grade50"><rb>時期</rb><rp>(</rp><rt>시기</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E8%8B%B1%E5%9C%8B%EC%9D%98_%E5%9C%8B%E6%AD%8C&amp;action=edit&amp;section=2" title="部分 編輯: 國王(국왕) 時期(시기)">編輯</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>1.<br>
God save our gracious King,<br>
Long live our noble King,<br>
God save the King.<br>
Send him victories,<br>
Happy and glories,<br>
Long to reign over us:<br>
God save the King!<br>
2.<br>
O Lord, our God, arise,<br>
Scatter thine (him) enemies,<br>
And make them fall:<br>
Confound their politics,<br>
Frustrate their knavish tricks,<br>
On thee our hopes we fix:<br>
God save us all!<br>
3.<br>
Thy choicest gifts in store,<br>
On him be pleased to pour,<br>
Long may he reign.<br>
May he defend our laws,<br>
And give us ever cause,<br>
To sing with heart and voice:<br>
God save the King!
</p>
