<p><b><ruby class="hanja api grade20"><rb>阿弗利加</rb><rp>(</rp><rt>아불리가</rt><rp>)</rp></ruby></b>(아프리카)는 <a href="/wiki/%E4%BA%9E%E7%B4%B0%E4%BA%9E" title="亞細亞"><ruby class="hanja api grade32"><rb>亞細亞</rb><rp>(</rp><rt>아세아</rt><rp>)</rp></ruby></a>에 이어 <a href="/wiki/%E4%B8%96%E7%95%8C" title="世界"><ruby class="hanja api grade62"><rb>世界</rb><rp>(</rp><rt>세계</rt><rp>)</rp></ruby></a>에서 두 <ruby class="hanja api grade60"><rb>番</rb><rp>(</rp><rt>번</rt><rp>)</rp></ruby>째로 큰 <a href="/wiki/%E5%A4%A7%E9%99%B8" title="大陸"><ruby class="hanja api grade52"><rb>大陸</rb><rp>(</rp><rt>대륙</rt><rp>)</rp></ruby></a>이자 두 <ruby class="hanja api grade60"><rb>番</rb><rp>(</rp><rt>번</rt><rp>)</rp></ruby>째로 <ruby class="hanja api grade70"><rb>人口</rb><rp>(</rp><rt>인구</rt><rp>)</rp></ruby>가 많은 <ruby class="hanja api grade52"><rb>大陸</rb><rp>(</rp><rt>대륙</rt><rp>)</rp></ruby>이다. <ruby class="hanja api grade62"><rb>地球</rb><rp>(</rp><rt>지구</rt><rp>)</rp></ruby> <ruby class="hanja api grade52"><rb>陸地</rb><rp>(</rp><rt>육지</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>面積</rb><rp>(</rp><rt>면적</rt><rp>)</rp></ruby>의 20%, <a href="/index.php?title=%E5%9C%B0%E7%90%83&amp;action=edit&amp;redlink=1" class="new" title="地球 (없는 文書)"><ruby class="hanja api grade62 unknown"><rb>地球</rb><rp>(</rp><rt>지구</rt><rp>)</rp></ruby></a> <ruby class="hanja api grade62"><rb>全體</rb><rp>(</rp><rt>전체</rt><rp>)</rp></ruby> <ruby class="hanja api grade52"><rb>表面的</rb><rp>(</rp><rt>표면적</rt><rp>)</rp></ruby>의 6%를 차지한다. <ruby class="hanja api grade32"><rb>阿洲</rb><rp>(</rp><rt>아주</rt><rp>)</rp></ruby>라고도 한다. <b><ruby class="hanja api grade32"><rb>亞</rb><rp>(</rp><rt>아</rt><rp>)</rp></ruby><ruby class="hanja api grade42"><rb>非</rb><rp>(</rp><rt>비</rt><rp>)</rp></ruby><ruby class="hanja api grade62"><rb>利</rb><rp>(</rp><rt>리</rt><rp>)</rp></ruby><ruby class="hanja api grade50"><rb>加</rb><rp>(</rp><rt>가</rt><rp>)</rp></ruby></b>라고도 쓴다.
</p>
