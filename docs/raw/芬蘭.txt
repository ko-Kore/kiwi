<h2><span id=".E6.A7.AA.E8.BF.B0.28.EA.B0.9C.EC.88.A0.29"></span><span class="mw-headline" id="槪述(개술)"><ruby class="hanja api grade32"><rb>槪述</rb><rp>(</rp><rt>개술</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E8%8A%AC%E8%98%AD&amp;action=edit&amp;section=1" title="部分 編輯: 槪述(개술)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a class="mw-selflink selflink">芬蘭</a>는 <a href="/wiki/%EC%9C%A0%EB%9F%BD" class="mw-redirect" title="유럽">유럽</a> <ruby class="hanja api grade62"><rb>北部</rb><rp>(</rp><rt>북부</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade42"><rb>位置</rb><rp>(</rp><rt>위치</rt><rp>)</rp></ruby>해 있다. <a href="/wiki/%E7%91%9E%E5%85%B8" title="瑞典"><ruby class="hanja api grade20"><rb>瑞典</rb><rp>(</rp><rt>서전</rt><rp>)</rp></ruby></a> <a href="/index.php?title=%E8%AB%BE%E5%A8%81&amp;action=edit&amp;redlink=1" class="new" title="諾威 (없는 文書)"><ruby class="hanja api grade32 unknown"><rb>諾威</rb><rp>(</rp><rt>낙위</rt><rp>)</rp></ruby></a> <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" class="mw-redirect" title="러시아">러시아</a>와 <ruby class="hanja api grade42"><rb>國境</rb><rp>(</rp><rt>국경</rt><rp>)</rp></ruby>을 <ruby class="hanja api grade42"><rb>接</rb><rp>(</rp><rt>접</rt><rp>)</rp></ruby>하고 <ruby class="hanja api grade80"><rb>南</rb><rp>(</rp><rt>남</rt><rp>)</rp></ruby>쪽은 <ruby class="hanja api grade12"><rb>芬蘭</rb><rp>(</rp><rt>분란</rt><rp>)</rp></ruby>만 <ruby class="hanja api grade80"><rb>西</rb><rp>(</rp><rt>서</rt><rp>)</rp></ruby>쪽은 <ruby class="hanja api grade20"><rb>抛</rb><rp>(</rp><rt>포</rt><rp>)</rp></ruby>티니아만에 <ruby class="hanja api grade42"><rb>接</rb><rp>(</rp><rt>접</rt><rp>)</rp></ruby>해 있다. <ruby class="hanja api grade32"><rb>海岸線</rb><rp>(</rp><rt>해안선</rt><rp>)</rp></ruby>의 길이는 1100km이다. <ruby class="hanja api grade62"><rb>地形</rb><rp>(</rp><rt>지형</rt><rp>)</rp></ruby>은 <ruby class="hanja api grade80"><rb>北</rb><rp>(</rp><rt>북</rt><rp>)</rp></ruby>쪽이 높고 <ruby class="hanja api grade80"><rb>南</rb><rp>(</rp><rt>남</rt><rp>)</rp></ruby>쪽이 낮다. "<ruby class="hanja api grade52"><rb>內陸</rb><rp>(</rp><rt>내륙</rt><rp>)</rp></ruby> <ruby class="hanja api grade60"><rb>樹林</rb><rp>(</rp><rt>수림</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>面積</rb><rp>(</rp><rt>면적</rt><rp>)</rp></ruby>은 <ruby class="hanja api grade72"><rb>全國</rb><rp>(</rp><rt>전국</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>面積</rb><rp>(</rp><rt>면적</rt><rp>)</rp></ruby>의 10%를 차지하며 <ruby class="hanja api grade72"><rb>全國</rb><rp>(</rp><rt>전국</rt><rp>)</rp></ruby> 땅의 1/3이 <a href="/index.php?title=%E5%8C%97%E6%A5%B5%E5%9C%88&amp;action=edit&amp;redlink=1" class="new" title="北極圈 (없는 文書)"><ruby class="hanja api grade20 unknown"><rb>北極圈</rb><rp>(</rp><rt>북극권</rt><rp>)</rp></ruby></a> 안에 있다. <a href="/index.php?title=%E6%BA%AB%E5%B8%B6%E6%B5%B7%E6%B4%8B%E6%80%A7%E6%B0%A3%E5%80%99&amp;action=edit&amp;redlink=1" class="new" title="溫帶海洋性氣候 (없는 文書)"><ruby class="hanja api grade42 unknown"><rb>溫帶</rb><rp>(</rp><rt>온대</rt><rp>)</rp></ruby><ruby class="hanja api grade40 unknown"><rb>海洋性氣候</rb><rp>(</rp><rt>해양성기후</rt><rp>)</rp></ruby></a>에 <ruby class="hanja api grade40"><rb>屬</rb><rp>(</rp><rt>속</rt><rp>)</rp></ruby>한다. <ruby class="hanja api grade40"><rb>平均氣溫</rb><rp>(</rp><rt>평균기온</rt><rp>)</rp></ruby>은 겨울에는 -14℃~3℃, 여름에는 13℃~17℃이며 <ruby class="hanja api grade40"><rb>年平均</rb><rp>(</rp><rt>연평균</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>降雨量</rb><rp>(</rp><rt>강우량</rt><rp>)</rp></ruby>은 600mm이다.
</p>
<h2><span id=".E6.AD.B7.E5.8F.B2.28.EC.97.AD.EC.82.AC.29"></span><span class="mw-headline" id="歷史(역사)"><ruby class="hanja api grade52"><rb>歷史</rb><rp>(</rp><rt>역사</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E8%8A%AC%E8%98%AD&amp;action=edit&amp;section=2" title="部分 編輯: 歷史(역사)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja api grade52"><rb>約</rb><rp>(</rp><rt>약</rt><rp>)</rp></ruby> 9000<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> <ruby class="hanja api grade72"><rb>前</rb><rp>(</rp><rt>전</rt><rp>)</rp></ruby> <ruby class="hanja api grade50"><rb>氷河期</rb><rp>(</rp><rt>빙하기</rt><rp>)</rp></ruby> <ruby class="hanja api grade50"><rb>末期</rb><rp>(</rp><rt>말기</rt><rp>)</rp></ruby>에 핀란드<ruby class="hanja api grade80"><rb>人</rb><rp>(</rp><rt>인</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade70"><rb>祖上</rb><rp>(</rp><rt>조상</rt><rp>)</rp></ruby>들은 <ruby class="hanja api grade80"><rb>南</rb><rp>(</rp><rt>남</rt><rp>)</rp></ruby>쪽과 <ruby class="hanja api grade80"><rb>南東</rb><rp>(</rp><rt>남동</rt><rp>)</rp></ruby>쪽으로 <ruby class="hanja api grade42"><rb>移住</rb><rp>(</rp><rt>이주</rt><rp>)</rp></ruby>했다.
</p><p>12<ruby class="hanja api grade40"><rb>世紀</rb><rp>(</rp><rt>세기</rt><rp>)</rp></ruby> <ruby class="hanja api grade62"><rb>後半</rb><rp>(</rp><rt>후반</rt><rp>)</rp></ruby>부터 스웨덴에 <ruby class="hanja api grade40"><rb>所屬</rb><rp>(</rp><rt>소속</rt><rp>)</rp></ruby>되어 14<ruby class="hanja api grade40"><rb>世紀</rb><rp>(</rp><rt>세기</rt><rp>)</rp></ruby> <ruby class="hanja api grade32"><rb>中盤</rb><rp>(</rp><rt>중반</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade52"><rb>本格的</rb><rp>(</rp><rt>본격적</rt><rp>)</rp></ruby>으로 스웨덴의 <ruby class="hanja api grade62"><rb>一部</rb><rp>(</rp><rt>일부</rt><rp>)</rp></ruby>가 되었다.
</p><p>1809<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" class="mw-redirect" title="러시아">러시아</a>-<a href="/wiki/%E7%91%9E%E5%85%B8" title="瑞典"><ruby class="hanja api grade20"><rb>瑞典</rb><rp>(</rp><rt>서전</rt><rp>)</rp></ruby></a> <ruby class="hanja api grade50"><rb>戰爭</rb><rp>(</rp><rt>전쟁</rt><rp>)</rp></ruby> <ruby class="hanja api grade52"><rb>以後</rb><rp>(</rp><rt>이후</rt><rp>)</rp></ruby> 러시아의 <ruby class="hanja api grade62"><rb>大公國</rb><rp>(</rp><rt>대공국</rt><rp>)</rp></ruby>이 되었다.
</p><p>1917<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> 12<ruby class="hanja api grade80"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby> 6<ruby class="hanja api grade80"><rb>日</rb><rp>(</rp><rt>일</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade52"><rb>獨立</rb><rp>(</rp><rt>독립</rt><rp>)</rp></ruby>하여 1919<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade62"><rb>共和國</rb><rp>(</rp><rt>공화국</rt><rp>)</rp></ruby>이 <ruby class="hanja api grade60"><rb>樹立</rb><rp>(</rp><rt>수립</rt><rp>)</rp></ruby>되었다.
</p><p>1939<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby>부터 1940<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby>까지 <ruby class="hanja api grade62"><rb>第</rb><rp>(</rp><rt>제</rt><rp>)</rp></ruby>1<ruby class="hanja api grade42"><rb>次</rb><rp>(</rp><rt>차</rt><rp>)</rp></ruby> <ruby class="hanja api grade12"><rb>芬蘭</rb><rp>(</rp><rt>분란</rt><rp>)</rp></ruby>-<ruby class="hanja api grade32"><rb>蘇聯</rb><rp>(</rp><rt>소련</rt><rp>)</rp></ruby> <ruby class="hanja api grade50"><rb>戰爭</rb><rp>(</rp><rt>전쟁</rt><rp>)</rp></ruby> <ruby class="hanja api grade52"><rb>以後</rb><rp>(</rp><rt>이후</rt><rp>)</rp></ruby> 핀란드는 <ruby class="hanja api grade32"><rb>蘇聯</rb><rp>(</rp><rt>소련</rt><rp>)</rp></ruby>과 <ruby class="hanja api grade32"><rb>蘇聯</rb><rp>(</rp><rt>소련</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade50"><rb>領土</rb><rp>(</rp><rt>영토</rt><rp>)</rp></ruby>를 <ruby class="hanja api grade32"><rb>讓渡</rb><rp>(</rp><rt>양도</rt><rp>)</rp></ruby>하는 <ruby class="hanja api grade12"><rb>芬蘭</rb><rp>(</rp><rt>분란</rt><rp>)</rp></ruby>-<ruby class="hanja api grade32"><rb>蘇聯</rb><rp>(</rp><rt>소련</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>條約</rb><rp>(</rp><rt>조약</rt><rp>)</rp></ruby>을 <ruby class="hanja api grade20"><rb>締結</rb><rp>(</rp><rt>체결</rt><rp>)</rp></ruby>할 수밖에 없었다.
</p>
