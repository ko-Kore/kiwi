<h2><span id=".EC.BB.A8.EC.85.89"></span><span class="mw-headline" id="컨셉">컨셉</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EB%8D%94_%EB%B0%94%EC%9D%B4%EB%84%88%EB%A6%AC&amp;action=edit&amp;section=1" title="부분 편집: 컨셉">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<blockquote>
<p>'<ruby class="hanja grade62"><rb>第</rb><rp>(</rp><rt>제</rt><rp>)</rp></ruby>3의스킨'이라불리는 3D아바타로 <a href="/index.php?title=%E7%8F%BE%E5%AF%A6&amp;action=edit&amp;redlink=1" class="new" title="現實 (없는 문서)"><ruby class="hanja grade52 unknown"><rb>現實</rb><rp>(</rp><rt>현실</rt><rp>)</rp></ruby></a>과 <a href="/index.php?title=%E5%81%87%E6%83%B3%E7%A9%BA%E9%96%93&amp;action=edit&amp;redlink=1" class="new" title="假想空間 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>假想空間</rb><rp>(</rp><rt>가상공간</rt><rp>)</rp></ruby></a> 모두에서 <a href="/index.php?title=%EB%9D%BC%EC%9D%B4%EB%B8%8C&amp;action=edit&amp;redlink=1" class="new" title="라이브 (없는 문서)">라이브</a>를 하는 <ruby class="hanja grade42"><rb>次世代</rb><rp>(</rp><rt>차세대</rt><rp>)</rp></ruby> 아티스트.
</p><p><a href="/index.php?title=2019%E5%B9%B44%E6%9C%88&amp;action=edit&amp;redlink=1" class="new" title="2019年4月 (없는 문서)">2019<ruby class="hanja grade80 unknown"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby>4<ruby class="hanja grade80 unknown"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby></a> <a href="/index.php?title=%EC%9C%A0%EB%A6%AC%EC%9D%B4%C2%B7%EC%B9%B4%EB%85%BC&amp;action=edit&amp;redlink=1" class="new" title="유리이·카논 (없는 문서)">유리이·카논</a>의 오리지널<ruby class="hanja grade50"><rb>樂曲</rb><rp>(</rp><rt>악곡</rt><rp>)</rp></ruby> 'Ego도Id도<ruby class="hanja grade42"><rb>單純</rb><rp>(</rp><rt>단순</rt><rp>)</rp></ruby>히'를 첫<ruby class="hanja grade32"><rb>投稿</rb><rp>(</rp><rt>투고</rt><rp>)</rp></ruby>로 데뷔.
</p><p><ruby class="hanja grade52"><rb>現實</rb><rp>(</rp><rt>현실</rt><rp>)</rp></ruby>과 <ruby class="hanja grade42"><rb>假想</rb><rp>(</rp><rt>가상</rt><rp>)</rp></ruby>을 <ruby class="hanja grade20"><rb>融合</rb><rp>(</rp><rt>융합</rt><rp>)</rp></ruby>한 콘텐츠를 여러 <a href="/index.php?title=SNS&amp;action=edit&amp;redlink=1" class="new" title="SNS (없는 문서)">SNS</a>에 <ruby class="hanja grade32"><rb>投稿</rb><rp>(</rp><rt>투고</rt><rp>)</rp></ruby>하며 <ruby class="hanja grade52"><rb>獨自的</rb><rp>(</rp><rt>독자적</rt><rp>)</rp></ruby>인 <ruby class="hanja grade52"><rb>世界觀</rb><rp>(</rp><rt>세계관</rt><rp>)</rp></ruby>을 <ruby class="hanja grade42"><rb>創出</rb><rp>(</rp><rt>창출</rt><rp>)</rp></ruby>해나간다.
</p>
