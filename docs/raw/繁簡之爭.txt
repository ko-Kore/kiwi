<h2><span id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"></span><span class="mw-headline" id="槪要(개요)"><ruby class="hanja api grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E7%B9%81%E7%B0%A1%E4%B9%8B%E7%88%AD&amp;action=edit&amp;section=1" title="部分　編輯：槪要(개요)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja api grade40"><rb>簡體字</rb><rp>(</rp><rt>간체자</rt><rp>)</rp></ruby>와 <ruby class="hanja api grade70"><rb>正字</rb><rp>(</rp><rt>정자</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade32"><rb>贊反</rb><rp>(</rp><rt>찬반</rt><rp>)</rp></ruby><ruby class="hanja api grade42"><rb>論</rb><rp>(</rp><rt>론</rt><rp>)</rp></ruby>이다. <ruby class="hanja api grade32"><rb>卽</rb><rp>(</rp><rt>즉</rt><rp>)</rp></ruby> <ruby class="hanja api grade42"><rb>傳統文化</rb><rp>(</rp><rt>전통문화</rt><rp>)</rp></ruby>가 먼저냐 <ruby class="hanja api grade52"><rb>筆記</rb><rp>(</rp><rt>필기</rt><rp>)</rp></ruby><ruby class="hanja api grade40"><rb>容易</rb><rp>(</rp><rt>용이</rt><rp>)</rp></ruby><ruby class="hanja api grade52"><rb>性</rb><rp>(</rp><rt>성</rt><rp>)</rp></ruby>먼저이다라는 <ruby class="hanja api grade50"><rb>分爭</rb><rp>(</rp><rt>분쟁</rt><rp>)</rp></ruby>이다.
</p>
<h2><span id=".E7.84.A1.E5.BF.83.28.EB.AC.B4.EC.8B.AC.29.E4.B9.8B.28.EC.A7.80.29.E6.84.9B.28.EC.95.A0.29"></span><span class="mw-headline" id="無心(무심)之(지)愛(애)"><ruby class="hanja api grade50"><rb>無心</rb><rp>(</rp><rt>무심</rt><rp>)</rp></ruby><ruby class="hanja api grade32"><rb>之</rb><rp>(</rp><rt>지</rt><rp>)</rp></ruby><ruby class="hanja api grade60"><rb>愛</rb><rp>(</rp><rt>애</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E7%B9%81%E7%B0%A1%E4%B9%8B%E7%88%AD&amp;action=edit&amp;section=2" title="部分　編輯：無心(무심)之(지)愛(애)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja api grade60"><rb>愛</rb><rp>(</rp><rt>애</rt><rp>)</rp></ruby>를 하는데 <ruby class="hanja api grade70"><rb>心</rb><rp>(</rp><rt>심</rt><rp>)</rp></ruby>이 없는데 어찌 <ruby class="hanja api grade60"><rb>愛</rb><rp>(</rp><rt>애</rt><rp>)</rp></ruby>를 하냐 이다. <a href="/wiki/%E7%B0%A1%E9%AB%94%E5%AD%97" title="簡體字"><ruby class="hanja api grade40"><rb>簡體字</rb><rp>(</rp><rt>간체자</rt><rp>)</rp></ruby></a>에서는 <ruby class="hanja api grade60"><rb>愛</rb><rp>(</rp><rt>애</rt><rp>)</rp></ruby>를 <ruby class="hanja api grade70"><rb>心</rb><rp>(</rp><rt>심</rt><rp>)</rp></ruby>을 뺀 <ruby class="hanja api grade0"><rb>爱</rb><rp>(</rp><rt>애</rt><rp>)</rp></ruby>라고 <ruby class="hanja api grade62"><rb>表記</rb><rp>(</rp><rt>표기</rt><rp>)</rp></ruby>한다.
더 있는데 
1. <ruby class="hanja api grade0"><rb>厰</rb><rp>(</rp><rt>창</rt><rp>)</rp></ruby>(<ruby class="hanja api grade72"><rb>工場</rb><rp>(</rp><rt>공장</rt><rp>)</rp></ruby>)인데 <ruby class="hanja api grade12"><rb>敞</rb><rp>(</rp><rt>창</rt><rp>)</rp></ruby>이 빠진 <ruby class="hanja api grade0"><rb>厂</rb><rp>(</rp><rt>엄</rt><rp>)</rp></ruby>인데 <ruby class="hanja api grade72"><rb>工場</rb><rp>(</rp><rt>공장</rt><rp>)</rp></ruby>인데 왜 안에 아무것도 없냐고 한다.
2. <ruby class="hanja api grade0"><rb>鄉</rb><rp>(</rp><rt>향</rt><rp>)</rp></ruby>(시골)인데 <ruby class="hanja api grade0"><rb>郎</rb><rp>(</rp><rt>낭</rt><rp>)</rp></ruby>이 빠진 <ruby class="hanja api grade0"><rb>乡</rb><rp>(</rp><rt>향</rt><rp>)</rp></ruby>인데 시골인데 사내들은 다 없다라고 한다.
3. <ruby class="hanja api grade42"><rb>衛</rb><rp>(</rp><rt>위</rt><rp>)</rp></ruby>는 <ruby class="hanja api grade40"><rb>簡體字</rb><rp>(</rp><rt>간체자</rt><rp>)</rp></ruby>로 <ruby class="hanja api grade0"><rb>卫</rb><rp>(</rp><rt>위</rt><rp>)</rp></ruby>인데 <ruby class="hanja api grade32"><rb>遠征</rb><rp>(</rp><rt>원정</rt><rp>)</rp></ruby>를 <ruby class="hanja api grade60"><rb>行</rb><rp>(</rp><rt>행</rt><rp>)</rp></ruby>(가지)를 못하는데 어떻게 지키냐 라고 한다.
4. <ruby class="hanja api grade42"><rb>應</rb><rp>(</rp><rt>응</rt><rp>)</rp></ruby>는 <ruby class="hanja api grade70"><rb>心</rb><rp>(</rp><rt>심</rt><rp>)</rp></ruby>이 없어진 <ruby class="hanja api grade0"><rb>应</rb><rp>(</rp><rt>응</rt><rp>)</rp></ruby>인데 <ruby class="hanja api grade42"><rb>眞心</rb><rp>(</rp><rt>진심</rt><rp>)</rp></ruby>이 아닌데 어떻게 <ruby class="hanja api grade42"><rb>應</rb><rp>(</rp><rt>응</rt><rp>)</rp></ruby>하냐라고 한다. 
5. <ruby class="hanja api grade10"><rb>麵</rb><rp>(</rp><rt>면</rt><rp>)</rp></ruby>은 <ruby class="hanja api grade0"><rb>麦</rb><rp>(</rp><rt>맥</rt><rp>)</rp></ruby>가 없어진 <ruby class="hanja api grade70"><rb>面</rb><rp>(</rp><rt>면</rt><rp>)</rp></ruby>인데 <ruby class="hanja api grade0"><rb>麦</rb><rp>(</rp><rt>맥</rt><rp>)</rp></ruby>를 없이 어떻게 <ruby class="hanja api grade10"><rb>麵</rb><rp>(</rp><rt>면</rt><rp>)</rp></ruby>을 만드냐 라고 한다.
</p>
