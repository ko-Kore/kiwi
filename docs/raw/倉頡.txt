<h2><span id="槪要(개요)"></span><span class="mw-headline" id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"><ruby class="hanja grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E5%80%89%E9%A0%A1&amp;action=edit&amp;section=1" title="부분 편집: 槪要(개요)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%E6%BC%A2%E5%AD%97" title="漢字"><ruby class="hanja grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby></a>를 <ruby class="hanja grade42"><rb>創製</rb><rp>(</rp><rt>창제</rt><rp>)</rp></ruby>했다고 하는 <a href="/index.php?title=%E5%82%B3%E8%AA%AA&amp;action=edit&amp;redlink=1" class="new" title="傳說 (없는 문서)"><ruby class="hanja grade52 unknown"><rb>傳說</rb><rp>(</rp><rt>전설</rt><rp>)</rp></ruby></a>속의 <ruby class="hanja grade72"><rb>人物</rb><rp>(</rp><rt>인물</rt><rp>)</rp></ruby>이다. <a href="/index.php?title=%E9%BB%83%E5%B8%9D&amp;action=edit&amp;redlink=1" class="new" title="黃帝 (없는 문서)"><ruby class="hanja grade40 unknown"><rb>黃帝</rb><rp>(</rp><rt>황제</rt><rp>)</rp></ruby></a>의 <a href="/index.php?title=%E5%8F%B2%E5%AE%98&amp;action=edit&amp;redlink=1" class="new" title="史官 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>史官</rb><rp>(</rp><rt>사관</rt><rp>)</rp></ruby></a>이었으며 <a href="/index.php?title=%EB%88%88&amp;action=edit&amp;redlink=1" class="new" title="눈 (없는 문서)">눈</a>이 4개 달려있었다 한다.
</p>
<h2><span id="漢字創製(한자창제)"></span><span class="mw-headline" id=".E6.BC.A2.E5.AD.97.E5.89.B5.E8.A3.BD.28.ED.95.9C.EC.9E.90.EC.B0.BD.EC.A0.9C.29"><ruby class="hanja grade42"><rb>漢字創製</rb><rp>(</rp><rt>한자창제</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E5%80%89%E9%A0%A1&amp;action=edit&amp;section=2" title="부분 편집: 漢字創製(한자창제)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja grade52"><rb>元來</rb><rp>(</rp><rt>원래</rt><rp>)</rp></ruby> <a href="/wiki/%E4%B8%AD%E5%9C%8B" title="中國"><ruby class="hanja grade80"><rb>中國</rb><rp>(</rp><rt>중국</rt><rp>)</rp></ruby></a> 사람들은 <ruby class="hanja grade32"><rb>記錄媒體</rb><rp>(</rp><rt>기록매체</rt><rp>)</rp></ruby>로서 <a href="/index.php?title=%EC%9E%89%EC%B9%B4%E5%B8%9D%E5%9C%8B&amp;action=edit&amp;redlink=1" class="new" title="잉카帝國 (없는 문서)">잉카<ruby class="hanja grade40 unknown"><rb>帝國</rb><rp>(</rp><rt>제국</rt><rp>)</rp></ruby></a>의 <a href="/index.php?title=%ED%82%A4%ED%91%B8&amp;action=edit&amp;redlink=1" class="new" title="키푸 (없는 문서)">키푸</a>와 비슷한 <a href="/index.php?title=%E7%B5%90%E7%B9%A9&amp;action=edit&amp;redlink=1" class="new" title="結繩 (없는 문서)"><ruby class="hanja grade12 unknown"><rb>結繩</rb><rp>(</rp><rt>결승</rt><rp>)</rp></ruby></a>이라는 <ruby class="hanja grade60"><rb>方式</rb><rp>(</rp><rt>방식</rt><rp>)</rp></ruby>을 <ruby class="hanja grade60"><rb>使用</rb><rp>(</rp><rt>사용</rt><rp>)</rp></ruby>하고 있었는데, <ruby class="hanja grade0"><rb>倉頡</rb><rp>(</rp><rt>창힐</rt><rp>)</rp></ruby>은 <a href="/index.php?title=%EC%83%88&amp;action=edit&amp;redlink=1" class="new" title="새 (없는 문서)">새</a>와 짐승이 걸어간 <a href="/index.php?title=%EB%B0%9C%EC%9E%90%EA%B5%AD&amp;action=edit&amp;redlink=1" class="new" title="발자국 (없는 문서)">발자국</a>을 보고 그게 무슨 새와 짐승의 발자국인지 알 수 있다는 <ruby class="hanja grade52"><rb>事實</rb><rp>(</rp><rt>사실</rt><rp>)</rp></ruby>과 함께 <a href="/wiki/%E6%96%87%E5%AD%97" title="文字"><ruby class="hanja grade70"><rb>文字</rb><rp>(</rp><rt>문자</rt><rp>)</rp></ruby></a>로써 <ruby class="hanja grade32"><rb>槪念</rb><rp>(</rp><rt>개념</rt><rp>)</rp></ruby>을 나타낼 수 있다는 것을 깨달았다 한다.
</p><p><a href="/index.php?title=%E8%AA%AA%E6%96%87%E8%A7%A3%E5%AD%97&amp;action=edit&amp;redlink=1" class="new" title="說文解字 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>說文解字</rb><rp>(</rp><rt>설문해자</rt><rp>)</rp></ruby></a>에는 <ruby class="hanja grade0"><rb>倉頡</rb><rp>(</rp><rt>창힐</rt><rp>)</rp></ruby>이 처음 만든 <ruby class="hanja grade70"><rb>文字</rb><rp>(</rp><rt>문자</rt><rp>)</rp></ruby>는 <a href="/wiki/%E8%B1%A1%E5%BD%A2%E6%96%87%E5%AD%97" title="象形文字"><ruby class="hanja grade40"><rb>象形文字</rb><rp>(</rp><rt>상형문자</rt><rp>)</rp></ruby></a>로 이를 '<a href="/wiki/%E6%96%87" title="文"><ruby class="hanja grade70"><rb>文</rb><rp>(</rp><rt>문</rt><rp>)</rp></ruby></a>'이라 하고, 나중에 <a href="/wiki/%E5%BD%A2%E8%81%B2%E6%96%87%E5%AD%97" title="形聲文字"><ruby class="hanja grade42"><rb>形聲文字</rb><rp>(</rp><rt>형성문자</rt><rp>)</rp></ruby></a>가 만들어져서 이를 '<a href="/wiki/%E5%AD%97" title="字"><ruby class="hanja grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby></a>'라고 부른다고 되어 있다.
</p><p><ruby class="hanja grade52"><rb>當然</rb><rp>(</rp><rt>당연</rt><rp>)</rp></ruby>히 <ruby class="hanja grade62"><rb>現代</rb><rp>(</rp><rt>현대</rt><rp>)</rp></ruby> <a href="/index.php?title=%E6%AD%B7%E5%8F%B2%E5%AD%B8&amp;action=edit&amp;redlink=1" class="new" title="歷史學 (없는 문서)"><ruby class="hanja grade52 unknown"><rb>歷史學</rb><rp>(</rp><rt>역사학</rt><rp>)</rp></ruby></a>에서는 단 한 사람이 그 <ruby class="hanja grade0"><rb>厖大</rb><rp>(</rp><rt>방대</rt><rp>)</rp></ruby>한 <ruby class="hanja grade42"><rb>文字體係</rb><rp>(</rp><rt>문자체계</rt><rp>)</rp></ruby>를 만들었다고 보지 않고, <ruby class="hanja grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby>의 <ruby class="hanja grade42"><rb>起原</rb><rp>(</rp><rt>기원</rt><rp>)</rp></ruby>은 <a href="/wiki/%E7%94%B2%E9%AA%A8%E6%96%87" class="mw-redirect" title="甲骨文"><ruby class="hanja grade40"><rb>甲骨文</rb><rp>(</rp><rt>갑골문</rt><rp>)</rp></ruby></a>에서 <ruby class="hanja grade62"><rb>始作</rb><rp>(</rp><rt>시작</rt><rp>)</rp></ruby>한 것으로 여긴다.
</p>
