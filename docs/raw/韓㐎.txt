<h2><span id=".E5.90.8D.E7.A8.B1.28.EB.AA.85.EC.B9.AD.29"></span><span class="mw-headline" id="名稱(명칭)"><ruby class="hanja api grade40"><rb>名稱</rb><rp>(</rp><rt>명칭</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E9%9F%93%E3%90%8E&amp;action=edit&amp;section=1" title="部分　編輯：名稱(명칭)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja api grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby><ruby class="hanja api grade62"><rb>表記</rb><rp>(</rp><rt>표기</rt><rp>)</rp></ruby>는 <ruby class="hanja api grade10"><rb>諺文</rb><rp>(</rp><rt>언문</rt><rp>)</rp></ruby>, <ruby class="hanja api grade62"><rb>正音</rb><rp>(</rp><rt>정음</rt><rp>)</rp></ruby> <ruby class="hanja api grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby> 이 있으며, <a href="/wiki/%E9%9F%93" title="韓"><ruby class="hanja api grade80"><rb>韓</rb><rp>(</rp><rt>한</rt><rp>)</rp></ruby></a><a href="/wiki/%E3%90%8E" title="㐎"><ruby class="hanja api grade0"><rb>㐎</rb><rp>(</rp><rt>글</rt><rp>)</rp></ruby></a>과 같이 <ruby class="hanja api grade32"><rb>音借</rb><rp>(</rp><rt>음차</rt><rp>)</rp></ruby>되는 <ruby class="hanja api grade40"><rb>境遇</rb><rp>(</rp><rt>경우</rt><rp>)</rp></ruby>도 있다.
</p><p>'한글'의 <ruby class="hanja api grade42"><rb>意味</rb><rp>(</rp><rt>의미</rt><rp>)</rp></ruby>는 <ruby class="hanja api grade52"><rb>元來</rb><rp>(</rp><rt>원래</rt><rp>)</rp></ruby> 크다는 <ruby class="hanja api grade42"><rb>意味</rb><rp>(</rp><rt>의미</rt><rp>)</rp></ruby>의 ‘하다’와 ‘글’이 <ruby class="hanja api grade60"><rb>合</rb><rp>(</rp><rt>합</rt><rp>)</rp></ruby>쳐진 말이라는<ruby class="hanja api grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby> 여러 <ruby class="hanja api grade52"><rb>說</rb><rp>(</rp><rt>설</rt><rp>)</rp></ruby>이 있다.
</p><p>이 <ruby class="hanja api grade70"><rb>文字</rb><rp>(</rp><rt>문자</rt><rp>)</rp></ruby><ruby class="hanja api grade40"><rb>體系</rb><rp>(</rp><rt>체계</rt><rp>)</rp></ruby>는 <ruby class="hanja api grade60"><rb>本來</rb><rp>(</rp><rt>본래</rt><rp>)</rp></ruby> 이름이 '한글'이 아니었으나, 1912<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> <ruby class="hanja api grade70"><rb>有力</rb><rp>(</rp><rt>유력</rt><rp>)</rp></ruby>한 한글<ruby class="hanja api grade60"><rb>學者</rb><rp>(</rp><rt>학자</rt><rp>)</rp></ruby> <a href="/index.php?title=%E5%91%A8%E6%99%82%E7%B6%93&amp;action=edit&amp;redlink=1" class="new" title="周時經(없는　文書）"><ruby class="hanja api grade40 unknown"><rb>周時經</rb><rp>(</rp><rt>주시경</rt><rp>)</rp></ruby></a>에 의해 ‘한글’로 부르기 <ruby class="hanja api grade62"><rb>始作</rb><rp>(</rp><rt>시작</rt><rp>)</rp></ruby>한 것이라고 알려져 있다.<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">[<ruby class="hanja api grade80"><rb>一</rb><rp>(</rp><rt>일</rt><rp>)</rp></ruby>]</a></sup>
</p>
