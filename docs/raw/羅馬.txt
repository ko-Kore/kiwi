<p><b>Urbs Aeterna<br><ruby class="hanja api grade32"><rb>不滅</rb><rp>(</rp><rt>불멸</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade50"><rb>都市</rb><rp>(</rp><rt>도시</rt><rp>)</rp></ruby></b>
</p>
</blockquote>
<p><b><ruby class="hanja api grade42"><rb>羅馬</rb><rp>(</rp><rt>나마</rt><rp>)</rp></ruby></b>(로마)는 <a href="/wiki/%E4%BC%8A%E5%A4%AA%E5%88%A9" title="伊太利"><ruby class="hanja api grade12"><rb>伊太利</rb><rp>(</rp><rt>이태리</rt><rp>)</rp></ruby></a>의 <ruby class="hanja api grade50"><rb>首都</rb><rp>(</rp><rt>수도</rt><rp>)</rp></ruby>이다. 한때 <ruby class="hanja api grade70"><rb>有名</rb><rp>(</rp><rt>유명</rt><rp>)</rp></ruby>한 <a href="/index.php?title=%E7%BE%85%E9%A6%AC%E5%B8%9D%E5%9C%8B&amp;action=edit&amp;redlink=1" class="new" title="羅馬帝國 (없는 문서)"><ruby class="hanja api grade40 unknown"><rb>羅馬帝國</rb><rp>(</rp><rt>나마제국</rt><rp>)</rp></ruby></a>의 <a href="/wiki/%E9%A6%96%E9%83%BD" title="首都"><ruby class="hanja api grade50"><rb>首都</rb><rp>(</rp><rt>수도</rt><rp>)</rp></ruby></a>였던 이 <ruby class="hanja api grade50"><rb>都市</rb><rp>(</rp><rt>도시</rt><rp>)</rp></ruby>는 <ruby class="hanja api grade40"><rb>帝國</rb><rp>(</rp><rt>제국</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade0"><rb>西쪽</rb><rp>(</rp><rt>서쪽</rt><rp>)</rp></ruby>이 <ruby class="hanja api grade20"><rb>野蠻族</rb><rp>(</rp><rt>야만족</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade40"><rb>侵略</rb><rp>(</rp><rt>침략</rt><rp>)</rp></ruby>으로 <ruby class="hanja api grade32"><rb>陷落</rb><rp>(</rp><rt>함락</rt><rp>)</rp></ruby>된 <ruby class="hanja api grade72"><rb>後</rb><rp>(</rp><rt>후</rt><rp>)</rp></ruby> <ruby class="hanja api grade32"><rb>漸次</rb><rp>(</rp><rt>점차</rt><rp>)</rp></ruby> <ruby class="hanja api grade42"><rb>羅馬主敎</rb><rp>(</rp><rt>나마주교</rt><rp>)</rp></ruby>(<ruby class="hanja api grade72"><rb>後</rb><rp>(</rp><rt>후</rt><rp>)</rp></ruby>날 <a href="/wiki/%E6%95%8E%E7%9A%87" title="敎皇"><ruby class="hanja api grade32"><rb>敎皇</rb><rp>(</rp><rt>교황</rt><rp>)</rp></ruby></a>)의 <ruby class="hanja api grade42"><rb>統制下</rb><rp>(</rp><rt>통제하</rt><rp>)</rp></ruby>에 들어가 <a href="/index.php?title=%E6%95%8E%E7%9A%87%E5%BB%B3&amp;action=edit&amp;redlink=1" class="new" title="敎皇廳 (없는 문서)"><ruby class="hanja api grade32 unknown"><rb>敎皇廳</rb><rp>(</rp><rt>교황청</rt><rp>)</rp></ruby></a>의 <ruby class="hanja api grade50"><rb>首都</rb><rp>(</rp><rt>수도</rt><rp>)</rp></ruby>가 되었고, <a href="/index.php?title=19%E4%B8%96%E7%B4%80&amp;action=edit&amp;redlink=1" class="new" title="19世紀 (없는 문서)">19<ruby class="hanja api grade40 unknown"><rb>世紀</rb><rp>(</rp><rt>세기</rt><rp>)</rp></ruby></a>에는 새로 <ruby class="hanja api grade30"><rb>誕生</rb><rp>(</rp><rt>탄생</rt><rp>)</rp></ruby>한 <ruby class="hanja api grade12"><rb>伊太利</rb><rp>(</rp><rt>이태리</rt><rp>)</rp></ruby> <ruby class="hanja api grade72"><rb>國家</rb><rp>(</rp><rt>국가</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade50"><rb>首都</rb><rp>(</rp><rt>수도</rt><rp>)</rp></ruby>가 되어 오늘날까지 이어져 오고 있다.
</p><p><ruby class="hanja api grade42"><rb>羅馬</rb><rp>(</rp><rt>나마</rt><rp>)</rp></ruby>는 <ruby class="hanja api grade60"><rb>西洋</rb><rp>(</rp><rt>서양</rt><rp>)</rp></ruby> <ruby class="hanja api grade62"><rb>文明</rb><rp>(</rp><rt>문명</rt><rp>)</rp></ruby>과 <a href="/wiki/%E5%9F%BA%E7%9D%A3%E6%95%8E" title="基督敎"><ruby class="hanja api grade42"><rb>基督敎</rb><rp>(</rp><rt>기독교</rt><rp>)</rp></ruby></a> <ruby class="hanja api grade52"><rb>文化</rb><rp>(</rp><rt>문화</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade30"><rb>發祥地</rb><rp>(</rp><rt>발상지</rt><rp>)</rp></ruby>이자 <ruby class="hanja api grade42"><rb>基督敎</rb><rp>(</rp><rt>기독교</rt><rp>)</rp></ruby> 가톨릭<ruby class="hanja api grade62"><rb>敎會</rb><rp>(</rp><rt>교회</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade70"><rb>中心地</rb><rp>(</rp><rt>중심지</rt><rp>)</rp></ruby>로 널리 여겨지고 있다.
</p><p><a href="/index.php?title=%EB%B0%94%ED%8B%B0%EC%B9%B8%E5%B8%82%E5%9C%8B&amp;action=edit&amp;redlink=1" class="new" title="바티칸市國 (없는 문서)">바티칸<ruby class="hanja api grade72 unknown"><rb>市國</rb><rp>(</rp><rt>시국</rt><rp>)</rp></ruby></a>은 <ruby class="hanja api grade42"><rb>羅馬</rb><rp>(</rp><rt>나마</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade50"><rb>都市</rb><rp>(</rp><rt>도시</rt><rp>)</rp></ruby> <ruby class="hanja api grade42"><rb>境界</rb><rp>(</rp><rt>경계</rt><rp>)</rp></ruby> 안에 있는 <ruby class="hanja api grade52"><rb>獨立</rb><rp>(</rp><rt>독립</rt><rp>)</rp></ruby> <ruby class="hanja api grade72"><rb>國家</rb><rp>(</rp><rt>국가</rt><rp>)</rp></ruby>로, <ruby class="hanja api grade50"><rb>都市</rb><rp>(</rp><rt>도시</rt><rp>)</rp></ruby> 안에 <ruby class="hanja api grade72"><rb>國家</rb><rp>(</rp><rt>국가</rt><rp>)</rp></ruby>가 있는 <ruby class="hanja api grade40"><rb>現存</rb><rp>(</rp><rt>현존</rt><rp>)</rp></ruby>하는 <ruby class="hanja api grade30"><rb>唯一</rb><rp>(</rp><rt>유일</rt><rp>)</rp></ruby>한 <ruby class="hanja api grade60"><rb>事例</rb><rp>(</rp><rt>사례</rt><rp>)</rp></ruby>입니다.
</p>
<h2><span id=".EC.9D.B4.EB.A6.84"></span><span class="mw-headline" id="이름">이름</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E7%BE%85%E9%A6%AC&amp;action=edit&amp;section=1" title="부분 편집: 이름">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja api grade42"><rb>古代羅馬</rb><rp>(</rp><rt>고대나마</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade40"><rb>起源說話</rb><rp>(</rp><rt>기원설화</rt><rp>)</rp></ruby>에 따르면, '<ruby class="hanja api grade42"><rb>羅馬</rb><rp>(</rp><rt>나마</rt><rp>)</rp></ruby>'라는 이름은 <ruby class="hanja api grade50"><rb>都市</rb><rp>(</rp><rt>도시</rt><rp>)</rp></ruby>의 첫 <ruby class="hanja api grade0"><rb>番째</rb><rp>(</rp><rt>번째</rt><rp>)</rp></ruby> <ruby class="hanja api grade0"><rb>王이</rb><rp>(</rp><rt>왕이</rt><rp>)</rp></ruby>자 <ruby class="hanja api grade42"><rb>設立者</rb><rp>(</rp><rt>설립자</rt><rp>)</rp></ruby>인 로물루스의 이름을 따 지어졌다고 한다. 하지만 <ruby class="hanja api grade62"><rb>一部</rb><rp>(</rp><rt>일부</rt><rp>)</rp></ruby> <ruby class="hanja api grade52"><rb>學說</rb><rp>(</rp><rt>학설</rt><rp>)</rp></ruby>에 따르면 로물루스라는 이름이 <ruby class="hanja api grade50"><rb>都市</rb><rp>(</rp><rt>도시</rt><rp>)</rp></ruby> <ruby class="hanja api grade42"><rb>羅馬</rb><rp>(</rp><rt>나마</rt><rp>)</rp></ruby>의 이름에서 <ruby class="hanja api grade60"><rb>由來</rb><rp>(</rp><rt>유래</rt><rp>)</rp></ruby>했을 <ruby class="hanja api grade50"><rb>可能性</rb><rp>(</rp><rt>가능성</rt><rp>)</rp></ruby>도 <ruby class="hanja api grade42"><rb>提起</rb><rp>(</rp><rt>제기</rt><rp>)</rp></ruby>되며, <ruby class="hanja api grade52"><rb>基外</rb><rp>(</rp><rt>기외</rt><rp>)</rp></ruby>에도 다른 <ruby class="hanja api grade52"><rb>學說</rb><rp>(</rp><rt>학설</rt><rp>)</rp></ruby>들이 <ruby class="hanja api grade40"><rb>存在</rb><rp>(</rp><rt>존재</rt><rp>)</rp></ruby>한다.
</p>
