<h2><span id=".E6.A7.AA.E8.BF.B0.28.EA.B0.9C.EC.88.A0.29"></span><span class="mw-headline" id="槪述(개술)"><ruby class="hanja api grade32"><rb>槪述</rb><rp>(</rp><rt>개술</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E5%88%A9%E6%AF%94%E4%BA%9E&amp;action=edit&amp;section=1" title="部分 編輯: 槪述(개술)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%E9%98%BF%E5%BC%97%E5%88%A9%E5%8A%A0" title="阿弗利加"><ruby class="hanja api grade20"><rb>阿弗利加</rb><rp>(</rp><rt>아불리가</rt><rp>)</rp></ruby></a> <ruby class="hanja api grade62"><rb>北部</rb><rp>(</rp><rt>북부</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade42"><rb>位置</rb><rp>(</rp><rt>위치</rt><rp>)</rp></ruby>하며 <a href="/index.php?title=%E5%9F%83%E5%8F%8A&amp;action=edit&amp;redlink=1" class="new" title="埃及 (없는 文書)"><ruby class="hanja api grade12 unknown"><rb>埃及</rb><rp>(</rp><rt>애급</rt><rp>)</rp></ruby></a> 수단 <a href="/index.php?title=%E7%AA%81%E5%B0%BC%E6%96%AF&amp;action=edit&amp;redlink=1" class="new" title="突尼斯 (없는 文書)"><ruby class="hanja api grade20 unknown"><rb>突尼斯</rb><rp>(</rp><rt>돌니사</rt><rp>)</rp></ruby></a> <a href="/index.php?title=%E9%98%BF%E7%88%BE%E5%8F%8A&amp;action=edit&amp;redlink=1" class="new" title="阿爾及 (없는 文書)"><ruby class="hanja api grade10 unknown"><rb>阿爾及</rb><rp>(</rp><rt>아이급</rt><rp>)</rp></ruby></a> 니제르 <a href="/index.php?title=%E8%8C%B6%E9%83%BD&amp;action=edit&amp;redlink=1" class="new" title="茶都 (없는 文書)"><ruby class="hanja api grade32 unknown"><rb>茶都</rb><rp>(</rp><rt>차도</rt><rp>)</rp></ruby></a>와 접하고 있다. 
</p><p><ruby class="hanja api grade62"><rb>北部</rb><rp>(</rp><rt>북부</rt><rp>)</rp></ruby>은 <a href="/index.php?title=%E5%9C%B0%E4%B8%AD%E6%B5%B7&amp;action=edit&amp;redlink=1" class="new" title="地中海 (없는 文書)"><ruby class="hanja api grade70 unknown"><rb>地中海</rb><rp>(</rp><rt>지중해</rt><rp>)</rp></ruby></a>에 <ruby class="hanja api grade42"><rb>接</rb><rp>(</rp><rt>접</rt><rp>)</rp></ruby>해 있고 <ruby class="hanja api grade32"><rb>海岸線</rb><rp>(</rp><rt>해안선</rt><rp>)</rp></ruby>은 1,900 <ruby><rb>粁</rb><rp>(</rp><rt>킬로미터</rt><rp>)</rp></ruby> <ruby class="hanja api grade32"><rb>海岸</rb><rp>(</rp><rt>해안</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>地域</rb><rp>(</rp><rt>지역</rt><rp>)</rp></ruby>은 <a href="/index.php?title=%E5%9C%B0%E4%B8%AD%E6%B5%B7%E5%9E%8B%E6%B0%A3%E5%80%99&amp;action=edit&amp;redlink=1" class="new" title="地中海型氣候 (없는 文書)"><ruby class="hanja api grade70 unknown"><rb>地中海</rb><rp>(</rp><rt>지중해</rt><rp>)</rp></ruby><ruby class="hanja api grade20 unknown"><rb>型</rb><rp>(</rp><rt>형</rt><rp>)</rp></ruby><ruby class="hanja api grade40 unknown"><rb>氣候</rb><rp>(</rp><rt>기후</rt><rp>)</rp></ruby></a>이며 <ruby class="hanja api grade52"><rb>內陸</rb><rp>(</rp><rt>내륙</rt><rp>)</rp></ruby>의 넓은 <ruby class="hanja api grade40"><rb>地域</rb><rp>(</rp><rt>지역</rt><rp>)</rp></ruby>은 <a href="/index.php?title=%E7%86%B1%E5%B8%B6%E7%A0%82%E6%BC%A0%E6%B0%A3%E5%80%99&amp;action=edit&amp;redlink=1" class="new" title="熱帶砂漠氣候 (없는 文書)"><ruby class="hanja api grade42 unknown"><rb>熱帶</rb><rp>(</rp><rt>열대</rt><rp>)</rp></ruby><ruby class="hanja api grade0 unknown"><rb>砂漠</rb><rp>(</rp><rt>사막</rt><rp>)</rp></ruby><ruby class="hanja api grade40 unknown"><rb>氣候</rb><rp>(</rp><rt>기후</rt><rp>)</rp></ruby></a>이다.

<h2><span id=".E6.AD.B7.E5.8F.B2.28.EC.97.AD.EC.82.AC.29"></span><span class="mw-headline" id="歷史(역사)"><ruby class="hanja api grade52"><rb>歷史</rb><rp>(</rp><rt>역사</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E5%88%A9%E6%AF%94%E4%BA%9E&amp;action=edit&amp;section=2" title="部分 編輯: 歷史(역사)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja api grade40"><rb>紀元前</rb><rp>(</rp><rt>기원전</rt><rp>)</rp></ruby> 3<ruby class="hanja api grade40"><rb>世紀</rb><rp>(</rp><rt>세기</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade32"><rb>利比亞</rb><rp>(</rp><rt>이비아</rt><rp>)</rp></ruby><ruby class="hanja api grade80"><rb>人</rb><rp>(</rp><rt>인</rt><rp>)</rp></ruby>들은 카르타고 <ruby class="hanja api grade40"><rb>帝國</rb><rp>(</rp><rt>제국</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade42"><rb>支配</rb><rp>(</rp><rt>지배</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade40"><rb>對抗</rb><rp>(</rp><rt>대항</rt><rp>)</rp></ruby>하는 <ruby class="hanja api grade40"><rb>鬪爭</rb><rp>(</rp><rt>투쟁</rt><rp>)</rp></ruby>에서 <ruby class="hanja api grade42"><rb>統一</rb><rp>(</rp><rt>통일</rt><rp>)</rp></ruby>된 <ruby class="hanja api grade42"><rb>努</rb><rp>(</rp><rt>노</rt><rp>)</rp></ruby><ruby class="hanja api grade60"><rb>米</rb><rp>(</rp><rt>미</rt><rp>)</rp></ruby><ruby class="hanja api grade40"><rb>底</rb><rp>(</rp><rt>저</rt><rp>)</rp></ruby> <ruby class="hanja api grade80"><rb>國</rb><rp>(</rp><rt>국</rt><rp>)</rp></ruby>을 세웠다.
</p><p>7<ruby class="hanja api grade40"><rb>世紀</rb><rp>(</rp><rt>세기</rt><rp>)</rp></ruby>에 <b><a href="/index.php?title=%E4%BA%9E%E5%89%8C%E6%AF%94%E4%BA%9E%E4%BA%BA&amp;action=edit&amp;redlink=1" class="new" title="亞剌比亞人 (없는 文書)"><ruby class="hanja api grade32 unknown"><rb>亞</rb><rp>(</rp><rt>아</rt><rp>)</rp></ruby><ruby class="hanja api grade10 unknown"><rb>剌</rb><rp>(</rp><rt>랄</rt><rp>)</rp></ruby><ruby class="hanja api grade50 unknown"><rb>比</rb><rp>(</rp><rt>비</rt><rp>)</rp></ruby><ruby class="hanja api grade32 unknown"><rb>亞</rb><rp>(</rp><rt>아</rt><rp>)</rp></ruby><ruby class="hanja api grade80 unknown"><rb>人</rb><rp>(</rp><rt>인</rt><rp>)</rp></ruby></a></b>들은 비잔틴<ruby class="hanja api grade80"><rb>人</rb><rp>(</rp><rt>인</rt><rp>)</rp></ruby>들을 물리치고 <ruby class="hanja api grade62"><rb>現地</rb><rp>(</rp><rt>현지</rt><rp>)</rp></ruby> 베르베르<ruby class="hanja api grade80"><rb>人</rb><rp>(</rp><rt>인</rt><rp>)</rp></ruby>들을 <ruby class="hanja api grade32"><rb>征服</rb><rp>(</rp><rt>정복</rt><rp>)</rp></ruby>하여 아랍 <ruby class="hanja api grade52"><rb>文化</rb><rp>(</rp><rt>문화</rt><rp>)</rp></ruby>와 <a href="/wiki/%E5%9B%9E%E6%95%8E" title="回敎"><ruby class="hanja api grade42"><rb>回敎</rb><rp>(</rp><rt>회교</rt><rp>)</rp></ruby></a>를 가져왔다. 
</p><p>16<ruby class="hanja api grade40"><rb>世紀</rb><rp>(</rp><rt>세기</rt><rp>)</rp></ruby>에 <b><a href="/wiki/%E5%B4%87%E9%AB%98%ED%95%9C_%EC%98%A4%EC%8A%A4%EB%A7%8C%E5%9C%8B" title="崇高한 오스만國"><ruby class="hanja api grade40"><rb>崇高</rb><rp>(</rp><rt>숭고</rt><rp>)</rp></ruby>한 오스만<ruby class="hanja api grade80"><rb>國</rb><rp>(</rp><rt>국</rt><rp>)</rp></ruby></a></b>은 <b>트리폴리타니아</b>(<ruby class="hanja api grade62"><rb>現</rb><rp>(</rp><rt>현</rt><rp>)</rp></ruby> <ruby class="hanja api grade32"><rb>利比亞</rb><rp>(</rp><rt>이비아</rt><rp>)</rp></ruby> <ruby class="hanja api grade62"><rb>西部</rb><rp>(</rp><rt>서부</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>地域</rb><rp>(</rp><rt>지역</rt><rp>)</rp></ruby>)와 <b>키레니카</b>(<ruby class="hanja api grade62"><rb>現</rb><rp>(</rp><rt>현</rt><rp>)</rp></ruby> <ruby class="hanja api grade32"><rb>利比亞</rb><rp>(</rp><rt>이비아</rt><rp>)</rp></ruby> <ruby class="hanja api grade62"><rb>東部</rb><rp>(</rp><rt>동부</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>地域</rb><rp>(</rp><rt>지역</rt><rp>)</rp></ruby>)를 <ruby class="hanja api grade40"><rb>占領</rb><rp>(</rp><rt>점령</rt><rp>)</rp></ruby>하여 <ruby class="hanja api grade32"><rb>海岸</rb><rp>(</rp><rt>해안</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>地域</rb><rp>(</rp><rt>지역</rt><rp>)</rp></ruby>을 <ruby class="hanja api grade20"><rb>掌握</rb><rp>(</rp><rt>장악</rt><rp>)</rp></ruby>했다. 
</p><p>1912<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> <b><ruby class="hanja api grade32"><rb>利比亞</rb><rp>(</rp><rt>이비아</rt><rp>)</rp></ruby></b>는 <a href="/index.php?title=%E4%BC%8A%E5%9C%9F%E6%88%B0%E7%88%AD&amp;action=edit&amp;redlink=1" class="new" title="伊土戰爭 (없는 文書)"><ruby class="hanja api grade12 unknown"><rb>伊土戰爭</rb><rp>(</rp><rt>이토전쟁</rt><rp>)</rp></ruby></a> <ruby class="hanja api grade52"><rb>以後</rb><rp>(</rp><rt>이후</rt><rp>)</rp></ruby> <a href="/wiki/%E4%BC%8A%E5%A4%AA%E5%88%A9" title="伊太利"><ruby class="hanja api grade12"><rb>伊太利</rb><rp>(</rp><rt>이태리</rt><rp>)</rp></ruby></a> <a href="/index.php?title=%E6%A4%8D%E6%B0%91%E5%9C%B0&amp;action=edit&amp;redlink=1" class="new" title="植民地 (없는 文書)"><ruby class="hanja api grade70 unknown"><rb>植民地</rb><rp>(</rp><rt>식민지</rt><rp>)</rp></ruby></a>가 되었다.
