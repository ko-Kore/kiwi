<h2><span id=".E6.AD.B7.E5.8F.B2.28.EC.97.AD.EC.82.AC.29"></span><span class="mw-headline" id="歷史(역사)"><ruby class="hanja api grade52"><rb>歷史</rb><rp>(</rp><rt>역사</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E9%A6%AC%E6%B5%9C&amp;action=edit&amp;section=1" title="部分 編輯: 歷史(역사)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja api grade0"><rb>馬浜</rb><rp>(</rp><rt>마빈</rt><rp>)</rp></ruby>는 <ruby class="hanja api grade52"><rb>元來</rb><rp>(</rp><rt>원래</rt><rp>)</rp></ruby> <a href="/index.php?title=%E7%BE%8E%E6%B4%B2%E5%8E%9F%E4%BD%8F%E6%B0%91&amp;action=edit&amp;redlink=1" class="new" title="美洲原住民 (없는 文書)"><ruby class="hanja api grade32 unknown"><rb>美洲</rb><rp>(</rp><rt>미주</rt><rp>)</rp></ruby><ruby class="hanja api grade50 unknown"><rb>原住民</rb><rp>(</rp><rt>원주민</rt><rp>)</rp></ruby></a>이 <ruby class="hanja api grade40"><rb>居住</rb><rp>(</rp><rt>거주</rt><rp>)</rp></ruby>했습니다.
