<h2><span id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"></span><span class="mw-headline" id="槪要(개요)"><ruby class="hanja api grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E4%BA%9E%E7%88%BE%E7%84%B6%E4%B8%81&amp;action=edit&amp;section=1" title="部分 編輯: 槪要(개요)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja api grade60"><rb>南美</rb><rp>(</rp><rt>남미</rt><rp>)</rp></ruby> <ruby class="hanja api grade62"><rb>南東部</rb><rp>(</rp><rt>남동부</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade42"><rb>位置</rb><rp>(</rp><rt>위치</rt><rp>)</rp></ruby>하여 <ruby class="hanja api grade80"><rb>東</rb><rp>(</rp><rt>동</rt><rp>)</rp></ruby>쪽은 <ruby class="hanja api grade60"><rb>大西洋</rb><rp>(</rp><rt>대서양</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade42"><rb>接</rb><rp>(</rp><rt>접</rt><rp>)</rp></ruby>해 있으며 <ruby class="hanja api grade80"><rb>南</rb><rp>(</rp><rt>남</rt><rp>)</rp></ruby>쪽은 바다를 건너 <ruby class="hanja api grade42"><rb>南極</rb><rp>(</rp><rt>남극</rt><rp>)</rp></ruby> <ruby class="hanja api grade52"><rb>大陸</rb><rp>(</rp><rt>대륙</rt><rp>)</rp></ruby>, <ruby class="hanja api grade80"><rb>西</rb><rp>(</rp><rt>서</rt><rp>)</rp></ruby>쪽은 칠레, <ruby class="hanja api grade80"><rb>北</rb><rp>(</rp><rt>북</rt><rp>)</rp></ruby>쪽은 볼리비아와 파라과이, <ruby class="hanja api grade80"><rb>北東</rb><rp>(</rp><rt>북동</rt><rp>)</rp></ruby>쪽은 우루과이와 브라질에 <ruby class="hanja api grade42"><rb>接</rb><rp>(</rp><rt>접</rt><rp>)</rp></ruby>해 있습니다. <ruby class="hanja api grade80"><rb>南北</rb><rp>(</rp><rt>남북</rt><rp>)</rp></ruby> 길이 3,694km, <ruby class="hanja api grade80"><rb>東西</rb><rp>(</rp><rt>동서</rt><rp>)</rp></ruby> 너비 1,423km입니다. <ruby class="hanja api grade52"><rb>陸地</rb><rp>(</rp><rt>육지</rt><rp>)</rp></ruby> <ruby class="hanja api grade42"><rb>國境</rb><rp>(</rp><rt>국경</rt><rp>)</rp></ruby>의 길이는 25,728km이고 <ruby class="hanja api grade32"><rb>海岸線</rb><rp>(</rp><rt>해안선</rt><rp>)</rp></ruby>의 길이는 4,725km입니다. <ruby class="hanja api grade62"><rb>北部</rb><rp>(</rp><rt>북부</rt><rp>)</rp></ruby>는 <ruby class="hanja api grade42"><rb>熱帶性</rb><rp>(</rp><rt>열대성</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>氣候</rb><rp>(</rp><rt>기후</rt><rp>)</rp></ruby>, <ruby class="hanja api grade62"><rb>中部</rb><rp>(</rp><rt>중부</rt><rp>)</rp></ruby>는 <ruby class="hanja api grade32"><rb>亞熱帶</rb><rp>(</rp><rt>아열대</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>氣候</rb><rp>(</rp><rt>기후</rt><rp>)</rp></ruby>, <ruby class="hanja api grade62"><rb>南部</rb><rp>(</rp><rt>남부</rt><rp>)</rp></ruby>는 <ruby class="hanja api grade60"><rb>溫和</rb><rp>(</rp><rt>온화</rt><rp>)</rp></ruby>한 <ruby class="hanja api grade40"><rb>氣候</rb><rp>(</rp><rt>기후</rt><rp>)</rp></ruby>입니다. <ruby class="hanja api grade40"><rb>年平均</rb><rp>(</rp><rt>연평균</rt><rp>)</rp></ruby> <ruby class="hanja api grade60"><rb>氣溫</rb><rp>(</rp><rt>기온</rt><rp>)</rp></ruby>은 <ruby class="hanja api grade62"><rb>北部</rb><rp>(</rp><rt>북부</rt><rp>)</rp></ruby> 24°C, <ruby class="hanja api grade62"><rb>南部</rb><rp>(</rp><rt>남부</rt><rp>)</rp></ruby> 5.5°C입니다.

<h2><span id=".E6.AD.B7.E5.8F.B2.28.EC.97.AD.EC.82.AC.29"></span><span class="mw-headline" id="歷史(역사)"><ruby class="hanja api grade52"><rb>歷史</rb><rp>(</rp><rt>역사</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E4%BA%9E%E7%88%BE%E7%84%B6%E4%B8%81&amp;action=edit&amp;section=2" title="部分 編輯: 歷史(역사)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>16<ruby class="hanja api grade40"><rb>世紀</rb><rp>(</rp><rt>세기</rt><rp>)</rp></ruby> <ruby class="hanja api grade52"><rb>以前</rb><rp>(</rp><rt>이전</rt><rp>)</rp></ruby>에는 <ruby class="hanja api grade80"><rb>土</rb><rp>(</rp><rt>토</rt><rp>)</rp></ruby><ruby class="hanja api grade32"><rb>著</rb><rp>(</rp><rt>저</rt><rp>)</rp></ruby> 인디언들이 살았다. 
</p><p>16<ruby class="hanja api grade40"><rb>世紀</rb><rp>(</rp><rt>세기</rt><rp>)</rp></ruby> <ruby class="hanja api grade32"><rb>中盤</rb><rp>(</rp><rt>중반</rt><rp>)</rp></ruby>에 <a href="/wiki/%E8%A5%BF%E7%8F%AD%E7%89%99" title="西班牙"><ruby class="hanja api grade32"><rb>西班牙</rb><rp>(</rp><rt>서반아</rt><rp>)</rp></ruby></a> <ruby class="hanja api grade70"><rb>植民地</rb><rp>(</rp><rt>식민지</rt><rp>)</rp></ruby>가 되었습니다. 
</p><p>1810<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> 5<ruby class="hanja api grade80"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby> 25<ruby class="hanja api grade80"><rb>日</rb><rp>(</rp><rt>일</rt><rp>)</rp></ruby>, <a href="/wiki/%E8%A5%BF%E7%8F%AD%E7%89%99" title="西班牙"><ruby class="hanja api grade32"><rb>西班牙</rb><rp>(</rp><rt>서반아</rt><rp>)</rp></ruby></a> <ruby class="hanja api grade70"><rb>植民</rb><rp>(</rp><rt>식민</rt><rp>)</rp></ruby> <ruby class="hanja api grade42"><rb>統治</rb><rp>(</rp><rt>통치</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade40"><rb>對抗</rb><rp>(</rp><rt>대항</rt><rp>)</rp></ruby>하는 5<ruby class="hanja api grade80"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>革命</rb><rp>(</rp><rt>혁명</rt><rp>)</rp></ruby>이 일어나 <ruby class="hanja api grade50"><rb>最初</rb><rp>(</rp><rt>최초</rt><rp>)</rp></ruby>의 <ruby class="hanja api grade42"><rb>政府</rb><rp>(</rp><rt>정부</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>委員會</rb><rp>(</rp><rt>위원회</rt><rp>)</rp></ruby>가 <ruby class="hanja api grade40"><rb>構成</rb><rp>(</rp><rt>구성</rt><rp>)</rp></ruby>되었다.
