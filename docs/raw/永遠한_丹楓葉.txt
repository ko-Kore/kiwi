<html><head></head><body><div class="mw-parser-output"><p><b>&lt;<ruby class="hanja api grade60"><rb>永遠</rb><rp>(</rp><rt>영원</rt><rp>)</rp></ruby>한 <ruby class="hanja api grade32"><rb>丹楓</rb><rp>(</rp><rt>단풍</rt><rp>)</rp></ruby><ruby class="hanja api grade50"><rb>葉</rb><rp>(</rp><rt>엽</rt><rp>)</rp></ruby>(The Maple Leaf Forever)&gt;</b>은 알렉산더 뮤어(Alexander Muir, 1830-1906)가 1867<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> <a href="/wiki/%E5%8A%A0%E5%A5%88%E9%99%80%E8%81%AF%E9%82%A6" title="加奈陀聯邦"><ruby class="hanja api grade50"><rb>加</rb><rp>(</rp><rt>가</rt><rp>)</rp></ruby><ruby class="hanja api grade30"><rb>奈</rb><rp>(</rp><rt>내</rt><rp>)</rp></ruby><ruby class="hanja api grade10"><rb>陀</rb><rp>(</rp><rt>타</rt><rp>)</rp></ruby><ruby class="hanja api grade30"><rb>聯邦</rb><rp>(</rp><rt>연방</rt><rp>)</rp></ruby></a>이 수립되었을 때 작곡한 캐나다 애국곡이다. 그는 1866<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> 리치웨이 전투에서 <a href="/index.php?title=%E5%A4%9A%E5%80%AB%E5%A4%9A&amp;action=edit&amp;redlink=1" class="new" title="多倫多 (없는 文書)"><ruby class="hanja api grade60 unknown"><rb>多</rb><rp>(</rp><rt>다</rt><rp>)</rp></ruby><ruby class="hanja api grade32 unknown"><rb>倫</rb><rp>(</rp><rt>륜</rt><rp>)</rp></ruby><ruby class="hanja api grade60 unknown"><rb>多</rb><rp>(</rp><rt>다</rt><rp>)</rp></ruby></a> 여왕 개인 소총대와 함께 피니아인에 대항하여 이 작품을 만들었다.
</p><p>이 <ruby class="hanja api grade50"><rb>曲</rb><rp>(</rp><rt>곡</rt><rp>)</rp></ruby>는 <a href="/wiki/%E5%8A%A0%E5%A5%88%E9%99%80%E8%81%AF%E9%82%A6" title="加奈陀聯邦"><ruby class="hanja api grade50"><rb>加</rb><rp>(</rp><rt>가</rt><rp>)</rp></ruby><ruby class="hanja api grade30"><rb>奈</rb><rp>(</rp><rt>내</rt><rp>)</rp></ruby><ruby class="hanja api grade10"><rb>陀</rb><rp>(</rp><rt>타</rt><rp>)</rp></ruby><ruby class="hanja api grade30"><rb>聯邦</rb><rp>(</rp><rt>연방</rt><rp>)</rp></ruby></a>의 <a href="/wiki/%E5%9C%8B%E6%AD%8C" title="國歌"><ruby class="hanja api grade70"><rb>國歌</rb><rp>(</rp><rt>국가</rt><rp>)</rp></ruby></a>.
</p>
<h2><span id=".E6.AD.8C.E8.A9.9E.28.EA.B0.80.EC.82.AC.29"></span><span class="mw-headline" id="歌詞(가사)"><ruby class="hanja api grade32"><rb>歌詞</rb><rp>(</rp><rt>가사</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%B0%B8%E9%81%A0%ED%95%9C_%E4%B8%B9%E6%A5%93%E8%91%89&amp;action=edit&amp;section=1" title="部分 編輯: 歌詞(가사)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<blockquote>
<p>1.
</p><p>In Days of yore, From Britain's shore
</p><p>Wolfe the dauntless hero came
</p><p>And planted firm Britannia's flag
</p><p>On Canada's fair domain.
</p><p>Here may it wave, Our boast, our pride
</p><p>And joined in love together,
</p><p>The thistle, shamrock, rose entwined,
</p><p>The Maple Leaf Forever.
</p><p><br>
2.
</p><p>At Queenston Heights and Lundy's Lane
</p><p>Our brave fathers side by side
</p><p>For freedom's home and loved ones dear,
</p><p>Firmly stood and nobly died.
</p><p>And so their rights which they maintained,
</p><p>We swear to yield them never.
</p><p>Our watchword ever more shall be
</p><p>The Maple Leaf Forever.
</p><p><br>
3.
</p><p>Our fair Dominion now extends
</p><p>From Cape Race to Nootka Sound
</p><p>May peace forever be our lot
</p><p>And plenty a store abound
</p><p>And may those ties of love be ours
</p><p>Which discord cannot sever
</p><p>And flourish green for freedom's home
</p><p>The Maple Leaf Forever
</p><p><br>
<ruby class="hanja api grade42"><rb>副</rb><rp>(</rp><rt>부</rt><rp>)</rp></ruby><ruby class="hanja api grade70"><rb>歌</rb><rp>(</rp><rt>가</rt><rp>)</rp></ruby>.
</p><p>The Maple Leaf our Emblem Dear,
</p><p>The Maple Leaf Forever.
</p><p>God save our Queen<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">[1]</a></sup> and heaven bless,
</p><p>The Maple Leaf Forever.
</p>
</blockquote>
