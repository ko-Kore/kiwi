<h2><span id=".E9.9B.B6.28.EC.98.81.29"></span><span class="mw-headline" id="零(영)"><ruby class="hanja api grade30"><rb>零</rb><rp>(</rp><rt>영</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E3%80%87&amp;action=edit&amp;section=1" title="部分 編輯: 零(영)">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><b><ruby class="hanja api grade0"><rb>〇</rb><rp>(</rp><rt>영</rt><rp>)</rp></ruby></b>은 <a href="/index.php?title=%EC%95%84%EB%9D%BC%EB%B9%84%EC%95%84_%E6%95%B8%E5%AD%97&amp;action=edit&amp;redlink=1" class="new" title="아라비아 數字 (없는 文書)">아라비아 <ruby class="hanja api grade70 unknown"><rb>數字</rb><rp>(</rp><rt>숫자</rt><rp>)</rp></ruby></a> <a href="/index.php?title=0&amp;action=edit&amp;redlink=1" class="new" title="0 (없는 文書)">0</a>의 <a href="/wiki/%E6%BC%A2%E5%AD%97" title="漢字"><ruby class="hanja api grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby></a> 버전이며, 흔히 ‘<a href="/index.php?title=%E9%9B%B6&amp;action=edit&amp;redlink=1" class="new" title="零 (없는 文書)"><ruby class="hanja api grade30 unknown"><rb>零</rb><rp>(</rp><rt>영</rt><rp>)</rp></ruby></a>’ <ruby class="hanja api grade62"><rb>代身</rb><rp>(</rp><rt>대신</rt><rp>)</rp></ruby> 쓰인다. <ruby class="hanja api grade60"><rb>例</rb><rp>(</rp><rt>예</rt><rp>)</rp></ruby>를 들어 1024를 ‘<ruby class="hanja api grade80"><rb>一</rb><rp>(</rp><rt>일</rt><rp>)</rp></ruby><ruby class="hanja api grade0"><rb>〇</rb><rp>(</rp><rt>영</rt><rp>)</rp></ruby><ruby class="hanja api grade80"><rb>二四</rb><rp>(</rp><rt>이사</rt><rp>)</rp></ruby>’로 나타낼 수 있다. 
‘<span lang="ja">漢字源</span>’ 6<ruby class="hanja api grade32"><rb>版</rb><rp>(</rp><rt>판</rt><rp>)</rp></ruby>에 <a rel="nofollow" class="external text" href="https://twitter.com/kanokijuro/status/1078870238802792448">따르면</a>, 놀랍게도 <a href="/index.php?title=%E5%A7%93&amp;action=edit&amp;redlink=1" class="new" title="姓 (없는 文書)"><ruby class="hanja api grade72 unknown"><rb>姓</rb><rp>(</rp><rt>성</rt><rp>)</rp></ruby></a>이 <ruby class="hanja api grade0"><rb>〇</rb><rp>(</rp><rt>영</rt><rp>)</rp></ruby>인 사람도 있는 <ruby class="hanja api grade40"><rb>模樣</rb><rp>(</rp><rt>모양</rt><rp>)</rp></ruby>이다.
</p>
