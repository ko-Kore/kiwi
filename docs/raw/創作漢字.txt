<html><head></head><body><div class="mw-parser-output"><h2><span id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"></span><span class="mw-headline" id="槪要(개요)"><ruby class="hanja grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E5%89%B5%E4%BD%9C%E6%BC%A2%E5%AD%97&amp;action=edit&amp;section=1" title="부분 편집: 槪要(개요)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><b><ruby class="hanja grade42"><rb>創作漢字</rb><rp>(</rp><rt>창작한자</rt><rp>)</rp></ruby></b>는 <ruby class="hanja grade30"><rb>旣存</rb><rp>(</rp><rt>기존</rt><rp>)</rp></ruby>에 있던 <a href="/wiki/%E6%BC%A2%E5%AD%97" title="漢字"><ruby class="hanja grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby></a>가 아닌 <ruby class="hanja grade42"><rb>個人</rb><rp>(</rp><rt>개인</rt><rp>)</rp></ruby>이 <a href="/index.php?title=%E5%89%B5%E4%BD%9C&amp;action=edit&amp;redlink=1" class="new" title="創作 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>創作</rb><rp>(</rp><rt>창작</rt><rp>)</rp></ruby></a>한 <ruby class="hanja grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby>를 <ruby class="hanja grade40"><rb>指稱</rb><rp>(</rp><rt>지칭</rt><rp>)</rp></ruby>한다.
</p>
<h2><span id=".E4.BE.8B.E7.A4.BA.28.EC.98.88.EC.8B.9C.29"></span><span class="mw-headline" id="例示(예시)"><ruby class="hanja grade50"><rb>例示</rb><rp>(</rp><rt>예시</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E5%89%B5%E4%BD%9C%E6%BC%A2%E5%AD%97&amp;action=edit&amp;section=2" title="부분 편집: 例示(예시)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%E6%97%A5%E6%9C%AC" title="日本"><ruby class="hanja grade60"><rb>日本</rb><rp>(</rp><rt>일본</rt><rp>)</rp></ruby></a>에서는 <a href="/index.php?title=2010%E5%B9%B4&amp;action=edit&amp;redlink=1" class="new" title="2010年 (없는 문서)">2010<ruby class="hanja grade80 unknown"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby></a>부터 <ruby class="hanja grade42"><rb>創作漢字</rb><rp>(</rp><rt>창작한자</rt><rp>)</rp></ruby>를 <ruby class="hanja grade30"><rb>公募</rb><rp>(</rp><rt>공모</rt><rp>)</rp></ruby>해 <ruby class="hanja grade42"><rb>施賞</rb><rp>(</rp><rt>시상</rt><rp>)</rp></ruby>하는 <a rel="nofollow" class="external text" href="https://sousaku-kanji.com/">콘테스트</a>도 <ruby class="hanja grade72"><rb>每年</rb><rp>(</rp><rt>매년</rt><rp>)</rp></ruby> <ruby class="hanja grade42"><rb>進行</rb><rp>(</rp><rt>진행</rt><rp>)</rp></ruby>되고있다. <a href="/wiki/2020%E5%B9%B4" title="2020年">2020<ruby class="hanja grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby></a> <ruby class="hanja grade62"><rb>大會</rb><rp>(</rp><rt>대회</rt><rp>)</rp></ruby>부터는 <a href="/wiki/%EC%BD%94%EB%A1%9C%EB%82%9819" class="mw-redirect" title="코로나19">코로나19</a><ruby class="hanja grade32"><rb>關聯</rb><rp>(</rp><rt>관련</rt><rp>)</rp></ruby> <ruby class="hanja grade42"><rb>創意的</rb><rp>(</rp><rt>창의적</rt><rp>)</rp></ruby>인 글<ruby class="hanja grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>들이 많이 올라와 <ruby class="hanja grade60"><rb>注目</rb><rp>(</rp><rt>주목</rt><rp>)</rp></ruby>받고있다.
</p>
