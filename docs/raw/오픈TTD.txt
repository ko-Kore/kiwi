<h2><span id=".EA.B2.8C.EC.9E.84_.ED.94.8C.EB.A0.88.EC.9D.B4"></span><span class="mw-headline" id="게임_플레이">게임 플레이</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EC%98%A4%ED%94%88TTD&amp;action=edit&amp;section=1" title="部分　編輯：게임 플레이">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/index.php?title=%ED%8A%B8%EB%9E%9C%EC%8A%A4%ED%8F%AC%ED%8A%B8_%ED%83%80%EC%9D%B4%EC%BF%A4&amp;action=edit&amp;redlink=1" class="new" title="트랜스포트 타이쿤(없는　文書）">트랜스포트 타이쿤</a>과 마찬가지로 <ruby class="hanja api grade52"><rb>基本的</rb><rp>(</rp><rt>기본적</rt><rp>)</rp></ruby>으로 <ruby class="hanja api grade32"><rb>運輸會社</rb><rp>(</rp><rt>운수회사</rt><rp>)</rp></ruby>를 차려 <ruby class="hanja api grade40"><rb>運營</rb><rp>(</rp><rt>운영</rt><rp>)</rp></ruby>하는 게임으로, <a href="/index.php?title=%EB%B2%84%EC%8A%A4&amp;action=edit&amp;redlink=1" class="new" title="버스(없는　文書）">버스</a>, <a href="/index.php?title=%E5%88%97%E8%BB%8A&amp;action=edit&amp;redlink=1" class="new" title="列車(없는　文書）"><ruby class="hanja api grade42 unknown"><rb>列車</rb><rp>(</rp><rt>열차</rt><rp>)</rp></ruby></a>, <a href="/index.php?title=%E8%88%B9%E8%88%B6&amp;action=edit&amp;redlink=1" class="new" title="船舶(없는　文書）"><ruby class="hanja api grade20 unknown"><rb>船舶</rb><rp>(</rp><rt>선박</rt><rp>)</rp></ruby></a>, <a href="/index.php?title=%E8%88%AA%E7%A9%BA%E6%A9%9F&amp;action=edit&amp;redlink=1" class="new" title="航空機(없는　文書）"><ruby class="hanja api grade40 unknown"><rb>航空機</rb><rp>(</rp><rt>항공기</rt><rp>)</rp></ruby></a>와 같은 <ruby class="hanja api grade40"><rb>交通手段</rb><rp>(</rp><rt>교통수단</rt><rp>)</rp></ruby>으로 <ruby class="hanja api grade60"><rb>路線</rb><rp>(</rp><rt>노선</rt><rp>)</rp></ruby>을 짜서 <ruby class="hanja api grade52"><rb>旅客</rb><rp>(</rp><rt>여객</rt><rp>)</rp></ruby>이나 <ruby class="hanja api grade42"><rb>貨物</rb><rp>(</rp><rt>화물</rt><rp>)</rp></ruby>을 <ruby class="hanja api grade42"><rb>運送</rb><rp>(</rp><rt>운송</rt><rp>)</rp></ruby>해 돈을 버는 것이 <ruby class="hanja api grade52"><rb>目的</rb><rp>(</rp><rt>목적</rt><rp>)</rp></ruby>이다.
</p><p>랜덤으로 <ruby class="hanja api grade62"><rb>生成</rb><rp>(</rp><rt>생성</rt><rp>)</rp></ruby>된 맵에서 처음 게임을 <ruby class="hanja api grade62"><rb>始作</rb><rp>(</rp><rt>시작</rt><rp>)</rp></ruby>하면 <ruby <h3><span id=".E4.B8.AD.E6.AF.92.E6.80.A7.28.EC.A4.91.EB.8F.85.EC.84.B1.29"></span><span class="mw-headline" id="中毒性(중독성)"><ruby class="hanja api grade42"><rb>中毒性</rb><rp>(</rp><rt>중독성</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EC%98%A4%ED%94%88TTD&amp;action=edit&amp;section=2" title="部分　編輯：中毒性(중독성)">編輯</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>이런<ruby class="hanja api grade52"><rb>類</rb><rp>(</rp><rt>류</rt><rp>)</rp></ruby>의 게임이 그렇듯이 <ruby class="hanja api grade42"><rb>中毒性</rb><rp>(</rp><rt>중독성</rt><rp>)</rp></ruby>이 <ruby class="hanja api grade52"><rb>相當</rb><rp>(</rp><rt>상당</rt><rp>)</rp></ruby>하다. <ruby class="hanja api grade32"><rb>初盤</rb><rp>(</rp><rt>초반</rt><rp>)</rp></ruby>에는 얼마 없는 돈을 <ruby class="hanja api grade40"><rb>管理</rb><rp>(</rp><rt>관리</rt><rp>)</rp></ruby>하며 <ruby class="hanja api grade42"><rb>最大限</rb><rp>(</rp><rt>최대한</rt><rp>)</rp></ruby> <ruby class="hanja api grade32"><rb>效率</rb><rp>(</rp><rt>효율</rt><rp>)</rp></ruby>을 뽑아내 <ruby class="hanja api grade32"><rb>貸出金</rb><rp>(</rp><rt>대출금</rt><rp>)</rp></ruby>을 빨리 <ruby class="hanja api grade32"><rb>償還</rb><rp>(</rp><rt>상환</rt><rp>)</rp></ruby>할 <ruby class="hanja api grade32"><rb>計劃</rb><rp>(</rp><rt>계획</rt><rp>)</rp></ruby>을 세우게 되며, <ruby class="hanja api grade80"><rb>中</rb><rp>(</rp><rt>중</rt><rp>)</rp></ruby><ruby class="hanja api grade62"><rb>後半</rb><rp>(</rp><rt>후반</rt><rp>)</rp></ruby>정도 되면 <ruby class="hanja api grade62"><rb>自身</rb><rp>(</rp><rt>자신</rt><rp>)</rp></ruby>이 만든 <ruby class="hanja api grade60"><rb>交通</rb><rp>(</rp><rt>교통</rt><rp>)</rp></ruby><ruby class="hanja api grade40"><rb>帝國</rb><rp>(</rp><rt>제국</rt><rp>)</rp></ruby>(?)을 바라보며 <ruby class="hanja api grade42"><rb>滿足感</rb><rp>(</rp><rt>만족감</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade32"><rb>醉</rb><rp>(</rp><rt>취</rt><rp>)</rp></ruby>할 수 있는데, 이렇게 될 때까지의 <ruby class="hanja api grade42"><rb>過程</rb><rp>(</rp><rt>과정</rt><rp>)</rp></ruby>이 손을 뗄 수 없게 만드는 <ruby class="hanja api grade42"><rb>中毒性</rb><rp>(</rp><rt>중독성</rt><rp>)</rp></ruby>이 있는 것이다.
