<p>1500<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> 4<ruby class="hanja api grade80"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby> 22<ruby class="hanja api grade80"><rb>日</rb><rp>(</rp><rt>일</rt><rp>)</rp></ruby> <a href="/wiki/%E8%91%A1%E8%90%84%E7%89%99" title="葡萄牙"><ruby class="hanja api grade10"><rb>葡萄牙</rb><rp>(</rp><rt>포도아</rt><rp>)</rp></ruby></a>의 <ruby class="hanja api grade42"><rb>航海士</rb><rp>(</rp><rt>항해사</rt><rp>)</rp></ruby> 페드로 카브랄이 브라질에 <ruby class="hanja api grade52"><rb>到着</rb><rp>(</rp><rt>도착</rt><rp>)</rp></ruby>했다. 
</p><p>1630<ruby class="hanja api grade62"><rb>年代</rb><rp>(</rp><rt>연대</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade10"><rb>葡萄牙</rb><rp>(</rp><rt>포도아</rt><rp>)</rp></ruby> <ruby class="hanja api grade32"><rb>遠征隊</rb><rp>(</rp><rt>원정대</rt><rp>)</rp></ruby>가 <a class="mw-selflink selflink">伯剌西爾植民地</a>를 세우고 1549<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade42"><rb>總督</rb><rp>(</rp><rt>총독</rt><rp>)</rp></ruby>을 임명했다. 
</p><p>1808<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> <a href="/index.php?title=%EB%82%98%ED%8F%B4%EB%A0%88%EC%98%B9%EC%9D%B4&amp;action=edit&amp;redlink=1" class="new" title="나폴레옹이 (없는 文書)">나폴레옹이</a> <ruby class="hanja api grade10"><rb>葡萄牙</rb><rp>(</rp><rt>포도아</rt><rp>)</rp></ruby>을 <ruby class="hanja api grade40"><rb>侵攻</rb><rp>(</rp><rt>침공</rt><rp>)</rp></ruby>하자 <ruby class="hanja api grade10"><rb>葡萄牙</rb><rp>(</rp><rt>포도아</rt><rp>)</rp></ruby><ruby class="hanja api grade80"><rb>王室</rb><rp>(</rp><rt>왕실</rt><rp>)</rp></ruby>은 <ruby class="hanja api grade10"><rb>伯剌西爾</rb><rp>(</rp><rt>백랄서이</rt><rp>)</rp></ruby>로 <ruby class="hanja api grade42"><rb>移住</rb><rp>(</rp><rt>이주</rt><rp>)</rp></ruby>했다. 
</p><p>1821<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> <ruby class="hanja api grade10"><rb>葡萄牙</rb><rp>(</rp><rt>포도아</rt><rp>)</rp></ruby><ruby class="hanja api grade80"><rb>王室</rb><rp>(</rp><rt>왕실</rt><rp>)</rp></ruby>이 <ruby class="hanja api grade70"><rb>里</rb><rp>(</rp><rt>이</rt><rp>)</rp></ruby><ruby class="hanja api grade30"><rb>斯</rb><rp>(</rp><rt>사</rt><rp>)</rp></ruby><ruby class="hanja api grade60"><rb>本</rb><rp>(</rp><rt>본</rt><rp>)</rp></ruby>으로 <ruby class="hanja api grade42"><rb>移住</rb><rp>(</rp><rt>이주</rt><rp>)</rp></ruby>하면서 <ruby class="hanja api grade72"><rb>王子</rb><rp>(</rp><rt>왕자</rt><rp>)</rp></ruby> 페드로 리우<ruby class="hanja api grade40"><rb>派</rb><rp>(</rp><rt>파</rt><rp>)</rp></ruby>가 <ruby class="hanja api grade30"><rb>攝政</rb><rp>(</rp><rt>섭정</rt><rp>)</rp></ruby><ruby class="hanja api grade80"><rb>王</rb><rp>(</rp><rt>왕</rt><rp>)</rp></ruby>이 되었다. 
</p><p>1822<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> 9<ruby class="hanja api grade80"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby> 7<ruby class="hanja api grade80"><rb>日</rb><rp>(</rp><rt>일</rt><rp>)</rp></ruby> 페드로 <ruby class="hanja api grade72"><rb>王子</rb><rp>(</rp><rt>왕자</rt><rp>)</rp></ruby>는 <ruby class="hanja api grade52"><rb>獨立</rb><rp>(</rp><rt>독립</rt><rp>)</rp></ruby>을 <ruby class="hanja api grade40"><rb>宣言</rb><rp>(</rp><rt>선언</rt><rp>)</rp></ruby>하고 <a class="mw-selflink selflink">伯剌西爾帝國</a>을 세웠다.
</p><p>1889<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> 11<ruby class="hanja api grade80"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby> 15<ruby class="hanja api grade80"><rb>日</rb><rp>(</rp><rt>일</rt><rp>)</rp></ruby> 폰세카 <ruby class="hanja api grade42"><rb>將軍</rb><rp>(</rp><rt>장군</rt><rp>)</rp></ruby>이 <a href="/index.php?title=%EC%BF%A0%EB%8D%B0%ED%83%80&amp;action=edit&amp;redlink=1" class="new" title="쿠데타 (없는 文書)">쿠데타</a>를 일으켜 <ruby class="hanja api grade32"><rb>皇帝</rb><rp>(</rp><rt>황제</rt><rp>)</rp></ruby><ruby class="hanja api grade42"><rb>制</rb><rp>(</rp><rt>제</rt><rp>)</rp></ruby>를 무너뜨리고 <a class="mw-selflink selflink">伯剌西爾合衆國</a>을 <ruby class="hanja api grade60"><rb>樹立</rb><rp>(</rp><rt>수립</rt><rp>)</rp></ruby>했다. 
</p><p>1964<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> 3<ruby class="hanja api grade80"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby> 31<ruby class="hanja api grade80"><rb>日</rb><rp>(</rp><rt>일</rt><rp>)</rp></ruby> <ruby class="hanja api grade80"><rb>軍人</rb><rp>(</rp><rt>군인</rt><rp>)</rp></ruby> 쿠데타가 <ruby class="hanja api grade32"><rb>執權</rb><rp>(</rp><rt>집권</rt><rp>)</rp></ruby>하여 <ruby class="hanja api grade32"><rb>獨裁</rb><rp>(</rp><rt>독재</rt><rp>)</rp></ruby>를 <ruby class="hanja api grade42"><rb>斷行</rb><rp>(</rp><rt>단행</rt><rp>)</rp></ruby>하여.
</p><p>1967<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> <a class="mw-selflink selflink">伯剌西爾聯邦共和國</a>으로 <ruby class="hanja api grade40"><rb>改稱</rb><rp>(</rp><rt>개칭</rt><rp>)</rp></ruby>되었다.
</p><p>1985<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> 1<ruby class="hanja api grade80"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby> <ruby class="hanja api grade42"><rb>野黨</rb><rp>(</rp><rt>야당</rt><rp>)</rp></ruby>이 <ruby class="hanja api grade42"><rb>大統領</rb><rp>(</rp><rt>대통령</rt><rp>)</rp></ruby> <ruby class="hanja api grade42"><rb>間接</rb><rp>(</rp><rt>간접</rt><rp>)</rp></ruby> <ruby class="hanja api grade50"><rb>選擧</rb><rp>(</rp><rt>선거</rt><rp>)</rp></ruby>에서 <ruby class="hanja api grade60"><rb>勝利</rb><rp>(</rp><rt>승리</rt><rp>)</rp></ruby>하여 <ruby class="hanja api grade80"><rb>軍人</rb><rp>(</rp><rt>군인</rt><rp>)</rp></ruby> <ruby class="hanja api grade32"><rb>執權</rb><rp>(</rp><rt>집권</rt><rp>)</rp></ruby>을 종식시켰다. 1989<ruby class="hanja api grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby> 처음으로 <ruby class="hanja api grade80"><rb>國民</rb><rp>(</rp><rt>국민</rt><rp>)</rp></ruby> <ruby class="hanja api grade42"><rb>直接</rb><rp>(</rp><rt>직접</rt><rp>)</rp></ruby> <ruby class="hanja api grade50"><rb>選擧</rb><rp>(</rp><rt>선거</rt><rp>)</rp></ruby>로 <ruby class="hanja api grade42"><rb>總選</rb><rp>(</rp><rt>총선</rt><rp>)</rp></ruby>이 <ruby class="hanja api grade42"><rb>實施</rb><rp>(</rp><rt>실시</rt><rp>)</rp></ruby>되었다.
