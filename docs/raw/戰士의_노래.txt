<td>Seo dhíbh, a chairde, duan Óglaigh<br>
<p>Caithréimeach bríomhar ceolmhar<br>
Ár dtinte cnámh go buacach táid<br>
’S an spéir go mín réaltógach<br>
Is fonnmhar faobhrach sinn chun gleo<br>
’S go tiúnmhar glé roimh thíocht don ló<br>
Fé chiúnas chaomh na hoíche ar seol<br>
Seo libh, canaídh Amhrán na bhFiann
</p>
</td>
<td>We'll sing a song, a soldier's song<br>
<p>With cheering, rousing chorus<br>
As round our blazing fires we throng<br>
The starry heavens o'er us<br>
Impatient for the coming fight<br>
And as we wait the morning's light<br>
Here in the silence of the night<br>
We'll chant a soldier's song<br>
</p>
</td>
<td>우리는 부르리라, <ruby class="hanja api grade52"><rb>戰士</rb><rp>(</rp><rt>전사</rt><rp>)</rp></ruby>의 노래를,<br>
<p>입을 모아 <ruby class="hanja api grade40"><rb>歡呼</rb><rp>(</rp><rt>환호</rt><rp>)</rp></ruby>하고 <ruby class="hanja api grade32"><rb>激勵</rb><rp>(</rp><rt>격려</rt><rp>)</rp></ruby>하며.<br>
타오르는 불꽃 <ruby class="hanja api grade40"><rb>周圍</rb><rp>(</rp><rt>주위</rt><rp>)</rp></ruby>로 둘러앉아<br>
별이 빛나는 하늘 아래.<br>
어서 <ruby class="hanja api grade40"><rb>戰鬪</rb><rp>(</rp><rt>전투</rt><rp>)</rp></ruby>에 나서길 바라며<br>
<ruby class="hanja api grade70"><rb>來日</rb><rp>(</rp><rt>내일</rt><rp>)</rp></ruby> 아침이 밝아오길 기다리는 가운데<br>
<ruby class="hanja api grade30"><rb>只今</rb><rp>(</rp><rt>지금</rt><rp>)</rp></ruby> 고요한 밤 한가운데서<br>
우리는 <ruby class="hanja api grade52"><rb>戰士</rb><rp>(</rp><rt>전사</rt><rp>)</rp></ruby>의 노래를 부르리라.<br>
</p>

<p>whose lives are pledged to Ireland<br>

<ruby class="hanja api grade30"><rb>只今</rb><rp>(</rp><rt>지금</rt><rp>)</rp></ruby> 자랑스럽게 우리 앞에서 펄럭이는.<br>
