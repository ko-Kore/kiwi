<h2><span id="槪要(개요)"></span><span class="mw-headline" id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"><ruby class="hanja grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E9%AB%94&amp;action=edit&amp;section=1" title="부분 편집: 槪要(개요)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/index.php?title=%EB%AA%B8&amp;action=edit&amp;redlink=1" class="new" title="몸 (없는 문서)">몸</a>이나 <a href="/index.php?title=%E8%BA%AB%E9%AB%94&amp;action=edit&amp;redlink=1" class="new" title="身體 (없는 문서)"><ruby class="hanja grade62 unknown"><rb>身體</rb><rp>(</rp><rt>신체</rt><rp>)</rp></ruby></a>를 뜻하는 글<ruby class="hanja grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>이다. <a href="/wiki/%E6%9C%83%E6%84%8F%E6%96%87%E5%AD%97" title="會意文字"><ruby class="hanja grade62"><rb>會意文字</rb><rp>(</rp><rt>회의문자</rt><rp>)</rp></ruby></a>로 <a href="/index.php?title=%E9%AA%A8&amp;action=edit&amp;redlink=1" class="new" title="骨 (없는 문서)"><ruby class="hanja grade40 unknown"><rb>骨</rb><rp>(</rp><rt>골</rt><rp>)</rp></ruby></a>과 <a href="/index.php?title=%E8%B1%8A&amp;action=edit&amp;redlink=1" class="new" title="豊 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>豊</rb><rp>(</rp><rt>풍</rt><rp>)</rp></ruby></a>이 합쳐져 <a href="/index.php?title=%EB%BC%88&amp;action=edit&amp;redlink=1" class="new" title="뼈 (없는 문서)">뼈</a>를 <ruby class="hanja grade32"><rb>包含</rb><rp>(</rp><rt>포함</rt><rp>)</rp></ruby>한 여러가지가 갖추어진 <ruby class="hanja grade62"><rb>身體</rb><rp>(</rp><rt>신체</rt><rp>)</rp></ruby>를 뜻하게 되었다.
</p>
<h2><span id="用途(용도)"></span><span class="mw-headline" id=".E7.94.A8.E9.80.94.28.EC.9A.A9.EB.8F.84.29"><ruby class="hanja grade32"><rb>用途</rb><rp>(</rp><rt>용도</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E9%AB%94&amp;action=edit&amp;section=2" title="부분 편집: 用途(용도)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja grade62"><rb>身體</rb><rp>(</rp><rt>신체</rt><rp>)</rp></ruby>라는 뜻 말고도 <a href="/index.php?title=%E7%89%A9%E8%B3%AA&amp;action=edit&amp;redlink=1" class="new" title="物質 (없는 문서)"><ruby class="hanja grade52 unknown"><rb>物質</rb><rp>(</rp><rt>물질</rt><rp>)</rp></ruby></a>이나 <a href="/index.php?title=%E7%89%A9%E9%AB%94&amp;action=edit&amp;redlink=1" class="new" title="物體 (없는 문서)"><ruby class="hanja grade62 unknown"><rb>物體</rb><rp>(</rp><rt>물체</rt><rp>)</rp></ruby></a>, <a href="/index.php?title=%E9%AB%94%E5%88%B6&amp;action=edit&amp;redlink=1" class="new" title="體制 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>體制</rb><rp>(</rp><rt>체제</rt><rp>)</rp></ruby></a> 같은 의미로도 쓰이고, <a href="/index.php?title=%E6%96%87%E9%AB%94&amp;action=edit&amp;redlink=1" class="new" title="文體 (없는 문서)"><ruby class="hanja grade62 unknown"><rb>文體</rb><rp>(</rp><rt>문체</rt><rp>)</rp></ruby></a>의 줄임말로 쓰이기도 한다.
</p>
