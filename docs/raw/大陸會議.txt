</p><p>第1次 會議는 1774年 <a href="/index.php?title=%E5%85%B7%E7%90%89%E8%80%B3%E5%B7%9E&amp;action=edit&amp;redlink=1" class="new" title="具琉耳州 (없는 文書)"><ruby class="hanja api grade10 unknown"><rb>具琉耳</rb><rp>(</rp><rt>구류이</rt><rp>)</rp></ruby><ruby class="hanja api grade52 unknown"><rb>州</rb><rp>(</rp><rt>주</rt><rp>)</rp></ruby></a>를 除外한 <a href="/wiki/%E7%BE%8E%E5%9C%8B" class="mw-redirect" title="美國"><ruby class="hanja api grade60"><rb>美國</rb><rp>(</rp><rt>미국</rt><rp>)</rp></ruby></a>植民地의 代表들이 <a href="/index.php?title=%E8%B2%BB%E5%BA%9C&amp;action=edit&amp;redlink=1" class="new" title="費府 (없는 文書)"><ruby class="hanja api grade42 unknown"><rb>費府</rb><rp>(</rp><rt>비부</rt><rp>)</rp></ruby></a>에서 英國의 抑壓에 反對하는 會議를 열었을 때 열렸습니다. 이 會議는 <b>植民地住民들의 權利를 主張하는 權利宣言</b>을 公布하고, 英國에 對抗하는 <b>貿易斷絶同盟</b>의 結成을 採擇했다. 이듬해인 1775年 第2次 會議에서 그는 <a href="/wiki/%EC%9B%8C%EC%8B%B1%ED%84%B4" class="mw-redirect" title="워싱턴">워싱턴</a>을 總司令官으로 임명했고, 1776年에는 <a href="/wiki/%E7%BE%8E%E5%9C%8B%E7%8D%A8%E7%AB%8B%E5%AE%A3%E8%A8%80" title="美國獨立宣言"><ruby class="hanja api grade40"><rb>獨立宣言書</rb><rp>(</rp><rt>독립선언서</rt><rp>)</rp></ruby></a>를 採擇했다. 1781年 美國<a href="/index.php?title=%E8%81%AF%E5%90%88%E8%A6%8F%E7%B4%84&amp;action=edit&amp;redlink=1" class="new" title="聯合規約 (없는 文書)"><ruby class="hanja api grade32 unknown"><rb>聯合規約</rb><rp>(</rp><rt>연합규약</rt><rp>)</rp></ruby></a>이 制定되고 <a href="/index.php?title=%E8%81%AF%E9%82%A6%E6%9C%83%E8%AD%B0&amp;action=edit&amp;redlink=1" class="new" title="聯邦會議 (없는 文書)"><ruby class="hanja api grade30 unknown"><rb>聯邦</rb><rp>(</rp><rt>연방</rt><rp>)</rp></ruby><ruby class="hanja api grade42 unknown"><rb>會議</rb><rp>(</rp><rt>회의</rt><rp>)</rp></ruby></a>로 代替될 때까지 存續했습니다.
