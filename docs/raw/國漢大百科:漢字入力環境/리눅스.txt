<p><a href="/index.php?title=%EB%A6%AC%EB%88%85%EC%8A%A4&amp;action=edit&amp;redlink=1" class="new" title="리눅스 (없는 문서)">리눅스</a>에서 <a href="/wiki/%E6%BC%A2%E5%AD%97" title="漢字"><ruby class="hanja grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby></a>를 <ruby class="hanja grade40"><rb>支援</rb><rp>(</rp><rt>지원</rt><rp>)</rp></ruby>하는 <a href="/wiki/%E9%9F%93%E5%9C%8B%E8%AA%9E_%E5%85%A5%E5%8A%9B%E6%96%B9%E5%BC%8F" title="韓國語 入力方式"><ruby class="hanja grade70"><rb>韓國語</rb><rp>(</rp><rt>한국어</rt><rp>)</rp></ruby> <ruby class="hanja grade40"><rb>入力環境</rb><rp>(</rp><rt>입력환경</rt><rp>)</rp></ruby></a>을 갖추기 <ruby class="hanja grade42"><rb>爲</rb><rp>(</rp><rt>위</rt><rp>)</rp></ruby>한 <ruby class="hanja grade52"><rb>方法</rb><rp>(</rp><rt>방법</rt><rp>)</rp></ruby>에 <ruby class="hanja grade62"><rb>對</rb><rp>(</rp><rt>대</rt><rp>)</rp></ruby>해 <ruby class="hanja grade52"><rb>說明</rb><rp>(</rp><rt>설명</rt><rp>)</rp></ruby>하는 <ruby class="hanja grade62"><rb>文書</rb><rp>(</rp><rt>문서</rt><rp>)</rp></ruby>이다.
</p>
<h2><span class="mw-headline" id="Ubuntu_22.04">Ubuntu 22.04</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E5%9C%8B%E6%BC%A2%E5%A4%A7%E7%99%BE%E7%A7%91:%E6%BC%A2%E5%AD%97%E5%85%A5%E5%8A%9B%E7%92%B0%E5%A2%83/%EB%A6%AC%EB%88%85%EC%8A%A4&amp;action=edit&amp;section=2" title="부분 편집: Ubuntu 22.04">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<h3><span id="設置(설치)_方法(방법)"></span><span class="mw-headline" id=".E8.A8.AD.E7.BD.AE.28.EC.84.A4.EC.B9.98.29_.E6.96.B9.E6.B3.95.28.EB.B0.A9.EB.B2.95.29"><ruby class="hanja grade42"><rb>設置</rb><rp>(</rp><rt>설치</rt><rp>)</rp></ruby> <ruby class="hanja grade52"><rb>方法</rb><rp>(</rp><rt>방법</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E5%9C%8B%E6%BC%A2%E5%A4%A7%E7%99%BE%E7%A7%91:%E6%BC%A2%E5%AD%97%E5%85%A5%E5%8A%9B%E7%92%B0%E5%A2%83/%EB%A6%AC%EB%88%85%EC%8A%A4&amp;action=edit&amp;section=3" title="부분 편집: 設置(설치) 方法(방법)">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><ruby class="hanja grade40"><rb>運營體制</rb><rp>(</rp><rt>운영체제</rt><rp>)</rp></ruby> <ruby class="hanja grade42"><rb>設置</rb><rp>(</rp><rt>설치</rt><rp>)</rp></ruby> <ruby class="hanja grade72"><rb>時</rb><rp>(</rp><rt>시</rt><rp>)</rp></ruby> <ruby class="hanja grade42"><rb>言語設定</rb><rp>(</rp><rt>언어설정</rt><rp>)</rp></ruby>에서 <ruby class="hanja grade70"><rb>韓國語</rb><rp>(</rp><rt>한국어</rt><rp>)</rp></ruby>를 <ruby class="hanja grade40"><rb>選擇</rb><rp>(</rp><rt>선택</rt><rp>)</rp></ruby>한 <ruby class="hanja grade40"><rb>境遇</rb><rp>(</rp><rt>경우</rt><rp>)</rp></ruby> 이미 ibus-hangul이 <ruby class="hanja grade42"><rb>設置</rb><rp>(</rp><rt>설치</rt><rp>)</rp></ruby>되어 있기 때문에 <ruby class="hanja grade52"><rb>以下</rb><rp>(</rp><rt>이하</rt><rp>)</rp></ruby> <ruby class="hanja grade50"><rb>命令語</rb><rp>(</rp><rt>명령어</rt><rp>)</rp></ruby>는 <ruby class="hanja grade52"><rb>實行</rb><rp>(</rp><rt>실행</rt><rp>)</rp></ruby>할 <ruby class="hanja grade52"><rb>必要</rb><rp>(</rp><rt>필요</rt><rp>)</rp></ruby>는 없다.
</p>
<pre>$ sudo apt install ibus-hangul
</pre>
<h3><span id="設定(설정)_方法(방법)"></span><span class="mw-headline" id=".E8.A8.AD.E5.AE.9A.28.EC.84.A4.EC.A0.95.29_.E6.96.B9.E6.B3.95.28.EB.B0.A9.EB.B2.95.29"><ruby class="hanja grade42"><rb>設定</rb><rp>(</rp><rt>설정</rt><rp>)</rp></ruby> <ruby class="hanja grade52"><rb>方法</rb><rp>(</rp><rt>방법</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E5%9C%8B%E6%BC%A2%E5%A4%A7%E7%99%BE%E7%A7%91:%E6%BC%A2%E5%AD%97%E5%85%A5%E5%8A%9B%E7%92%B0%E5%A2%83/%EB%A6%AC%EB%88%85%EC%8A%A4&amp;action=edit&amp;section=4" title="부분 편집: 設定(설정) 方法(방법)">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>ibus <ruby class="hanja grade42"><rb>設定</rb><rp>(</rp><rt>설정</rt><rp>)</rp></ruby> 메뉴에서 ‘한자 전용’을 <ruby class="hanja grade52"><rb>活性化</rb><rp>(</rp><rt>활성화</rt><rp>)</rp></ruby>함으로써 한글을 <ruby class="hanja grade70"><rb>入力</rb><rp>(</rp><rt>입력</rt><rp>)</rp></ruby>하면 <ruby class="hanja grade52"><rb>實時間</rb><rp>(</rp><rt>실시간</rt><rp>)</rp></ruby>으로 <ruby class="hanja grade32"><rb>變換</rb><rp>(</rp><rt>변환</rt><rp>)</rp></ruby> <ruby class="hanja grade32"><rb>候補</rb><rp>(</rp><rt>후보</rt><rp>)</rp></ruby>가 나타난다.
</p>
