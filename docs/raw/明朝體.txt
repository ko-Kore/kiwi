<h2><span id="槪要(개요)"></span><span class="mw-headline" id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"><ruby class="hanja grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%98%8E%E6%9C%9D%E9%AB%94&amp;action=edit&amp;section=1" title="부분 편집: 槪要(개요)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja grade60"><rb>明朝體</rb><rp>(</rp><rt>명조체</rt><rp>)</rp></ruby>는 <ruby class="hanja grade32"><rb>橫劃</rb><rp>(</rp><rt>횡획</rt><rp>)</rp></ruby>보다 <ruby class="hanja grade32"><rb>縱劃</rb><rp>(</rp><rt>종획</rt><rp>)</rp></ruby>이 굵고 <ruby class="hanja grade32"><rb>劃</rb><rp>(</rp><rt>획</rt><rp>)</rp></ruby>의 <ruby class="hanja grade42"><rb>末端</rb><rp>(</rp><rt>말단</rt><rp>)</rp></ruby>에 <a href="/index.php?title=%EC%84%B8%EB%A6%AC%ED%94%84&amp;action=edit&amp;redlink=1" class="new" title="세리프 (없는 문서)">세리프</a>가 있는 <a href="/wiki/%E6%BC%A2%E5%AD%97" title="漢字"><ruby class="hanja grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby></a> 또는 <a href="/wiki/%ED%95%9C%EA%B8%80" title="한글">한글</a> <a href="/index.php?title=%E6%9B%B8%E9%AB%94&amp;action=edit&amp;redlink=1" class="new" title="書體 (없는 문서)"><ruby class="hanja grade62 unknown"><rb>書體</rb><rp>(</rp><rt>서체</rt><rp>)</rp></ruby></a>이다. <a href="/wiki/%EA%B3%A0%EB%94%95%E9%AB%94" title="고딕體">고딕<ruby class="hanja grade62"><rb>體</rb><rp>(</rp><rt>체</rt><rp>)</rp></ruby></a>와 <ruby class="hanja grade50"><rb>對比</rb><rp>(</rp><rt>대비</rt><rp>)</rp></ruby>되며 <a href="/index.php?title=%E8%8B%B1%E6%96%87&amp;action=edit&amp;redlink=1" class="new" title="英文 (없는 문서)"><ruby class="hanja grade60 unknown"><rb>英文</rb><rp>(</rp><rt>영문</rt><rp>)</rp></ruby></a><ruby class="hanja grade70"><rb>活字</rb><rp>(</rp><rt>활자</rt><rp>)</rp></ruby>의 <a href="/index.php?title=%EC%84%B8%EB%A6%AC%ED%94%84%E9%AB%94&amp;action=edit&amp;redlink=1" class="new" title="세리프體 (없는 문서)">세리프<ruby class="hanja grade62 unknown"><rb>體</rb><rp>(</rp><rt>체</rt><rp>)</rp></ruby></a>에 <ruby class="hanja grade30"><rb>該當</rb><rp>(</rp><rt>해당</rt><rp>)</rp></ruby>된다.
</p>
<h2><span id="漢字(한자)_明朝體(명조체)"></span><span class="mw-headline" id=".E6.BC.A2.E5.AD.97.28.ED.95.9C.EC.9E.90.29_.E6.98.8E.E6.9C.9D.E9.AB.94.28.EB.AA.85.EC.A1.B0.EC.B2.B4.29"><ruby class="hanja grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby> <ruby class="hanja grade60"><rb>明朝體</rb><rp>(</rp><rt>명조체</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%98%8E%E6%9C%9D%E9%AB%94&amp;action=edit&amp;section=2" title="부분 편집: 漢字(한자) 明朝體(명조체)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%E4%B8%AD%E5%9C%8B" title="中國"><ruby class="hanja grade80"><rb>中國</rb><rp>(</rp><rt>중국</rt><rp>)</rp></ruby></a> <a href="/index.php?title=%E6%98%8E%EB%82%98%EB%9D%BC&amp;action=edit&amp;redlink=1" class="new" title="明나라 (없는 문서)"><ruby class="hanja grade62 unknown"><rb>明</rb><rp>(</rp><rt>명</rt><rp>)</rp></ruby>나라</a> 때 <ruby class="hanja grade42"><rb>製作</rb><rp>(</rp><rt>제작</rt><rp>)</rp></ruby>되어 <a href="/index.php?title=%E6%98%8E%E6%9C%9D&amp;action=edit&amp;redlink=1" class="new" title="明朝 (없는 문서)"><ruby class="hanja grade60 unknown"><rb>明朝</rb><rp>(</rp><rt>명조</rt><rp>)</rp></ruby></a><ruby class="hanja grade62"><rb>體</rb><rp>(</rp><rt>체</rt><rp>)</rp></ruby>라고 하지만, <ruby class="hanja grade52"><rb>事實</rb><rp>(</rp><rt>사실</rt><rp>)</rp></ruby> <a href="/index.php?title=%E5%AE%8B%EB%82%98%EB%9D%BC&amp;action=edit&amp;redlink=1" class="new" title="宋나라 (없는 문서)"><ruby class="hanja grade12 unknown"><rb>宋</rb><rp>(</rp><rt>송</rt><rp>)</rp></ruby>나라</a>의 <a href="/index.php?title=%E6%9B%B8%E9%AB%94&amp;action=edit&amp;redlink=1" class="new" title="書體 (없는 문서)"><ruby class="hanja grade62 unknown"><rb>書體</rb><rp>(</rp><rt>서체</rt><rp>)</rp></ruby></a>에 더 가까워 <a href="/wiki/%E4%B8%AD%E5%9C%8B" title="中國"><ruby class="hanja grade80"><rb>中國</rb><rp>(</rp><rt>중국</rt><rp>)</rp></ruby></a>에서는 <a href="/index.php?title=%E5%AE%8B%E9%AB%94&amp;action=edit&amp;redlink=1" class="new" title="宋體 (없는 문서)"><ruby class="hanja grade12 unknown"><rb>宋體</rb><rp>(</rp><rt>송체</rt><rp>)</rp></ruby></a>라 <ruby class="hanja grade40"><rb>稱</rb><rp>(</rp><rt>칭</rt><rp>)</rp></ruby>한다.
</p>
<h2><span id="한글_明朝體(명조체)"></span><span class="mw-headline" id=".ED.95.9C.EA.B8.80_.E6.98.8E.E6.9C.9D.E9.AB.94.28.EB.AA.85.EC.A1.B0.EC.B2.B4.29">한글 <ruby class="hanja grade60"><rb>明朝體</rb><rp>(</rp><rt>명조체</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%98%8E%E6%9C%9D%E9%AB%94&amp;action=edit&amp;section=3" title="부분 편집: 한글 明朝體(명조체)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%ED%95%9C%EA%B8%80" title="한글">한글</a> <ruby class="hanja grade60"><rb>明朝體</rb><rp>(</rp><rt>명조체</rt><rp>)</rp></ruby>는 <a href="/index.php?title=%E5%AE%AE%E9%AB%94&amp;action=edit&amp;redlink=1" class="new" title="宮體 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>宮體</rb><rp>(</rp><rt>궁체</rt><rp>)</rp></ruby></a>에서 <ruby class="hanja grade60"><rb>由來</rb><rp>(</rp><rt>유래</rt><rp>)</rp></ruby>되었으며, <a href="/index.php?title=%EB%B0%94%ED%83%95%E9%AB%94&amp;action=edit&amp;redlink=1" class="new" title="바탕體 (없는 문서)">바탕<ruby class="hanja grade62 unknown"><rb>體</rb><rp>(</rp><rt>체</rt><rp>)</rp></ruby></a>라고도 한다. <a href="/wiki/%E6%BC%A2%E5%AD%97" title="漢字"><ruby class="hanja grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby></a>의 <ruby class="hanja grade60"><rb>明朝體</rb><rp>(</rp><rt>명조체</rt><rp>)</rp></ruby>와는 <ruby class="hanja grade40"><rb>差異</rb><rp>(</rp><rt>차이</rt><rp>)</rp></ruby>가 있으며, <a href="/index.php?title=%E5%82%B3%E7%B5%B1&amp;action=edit&amp;redlink=1" class="new" title="傳統 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>傳統</rb><rp>(</rp><rt>전통</rt><rp>)</rp></ruby></a><ruby class="hanja grade60"><rb>明朝體</rb><rp>(</rp><rt>명조체</rt><rp>)</rp></ruby>에 <ruby class="hanja grade10"><rb>恰似</rb><rp>(</rp><rt>흡사</rt><rp>)</rp></ruby>하도록 <ruby class="hanja grade42"><rb>製作</rb><rp>(</rp><rt>제작</rt><rp>)</rp></ruby>된것은 <a href="/index.php?title=%E7%B4%94%E6%98%8E%E6%9C%9D&amp;action=edit&amp;redlink=1" class="new" title="純明朝 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>純明朝</rb><rp>(</rp><rt>순명조</rt><rp>)</rp></ruby></a>라고 한다.
</p>
