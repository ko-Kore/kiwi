<h2><span id="槪要(개요)"></span><span class="mw-headline" id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"><ruby class="hanja grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E9%9F%B3&amp;action=edit&amp;section=1" title="부분 편집: 槪要(개요)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/index.php?title=%EC%86%8C%EB%A6%AC&amp;action=edit&amp;redlink=1" class="new" title="소리 (없는 문서)">소리</a>를 뜻하는 글<ruby class="hanja grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>로, <a href="/wiki/%E8%A8%80" title="言"><ruby class="hanja grade60"><rb>言</rb><rp>(</rp><rt>언</rt><rp>)</rp></ruby></a>의 <a href="/index.php?title=%E5%8F%A3&amp;action=edit&amp;redlink=1" class="new" title="口 (없는 문서)"><ruby class="hanja grade70 unknown"><rb>口</rb><rp>(</rp><rt>구</rt><rp>)</rp></ruby></a> 속에 <a href="/wiki/%E4%B8%80" title="一"><ruby class="hanja grade80"><rb>一</rb><rp>(</rp><rt>일</rt><rp>)</rp></ruby></a>을 그은 <a href="/wiki/%E6%8C%87%E4%BA%8B%E6%96%87%E5%AD%97" title="指事文字"><ruby class="hanja grade42"><rb>指事文字</rb><rp>(</rp><rt>지사문자</rt><rp>)</rp></ruby></a>이다.
</p>
<h2><span id="字形(자형)"></span><span class="mw-headline" id=".E5.AD.97.E5.BD.A2.28.EC.9E.90.ED.98.95.29"><ruby class="hanja grade62"><rb>字形</rb><rp>(</rp><rt>자형</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E9%9F%B3&amp;action=edit&amp;section=2" title="부분 편집: 字形(자형)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/index.php?title=%E6%A5%B7%E6%9B%B8&amp;action=edit&amp;redlink=1" class="new" title="楷書 (없는 문서)"><ruby class="hanja grade10 unknown"><rb>楷書</rb><rp>(</rp><rt>해서</rt><rp>)</rp></ruby></a>에 와서는 <ruby class="hanja grade62"><rb>字形</rb><rp>(</rp><rt>자형</rt><rp>)</rp></ruby>이 <ruby class="hanja grade60"><rb>言</rb><rp>(</rp><rt>언</rt><rp>)</rp></ruby>과는 많이 <ruby class="hanja grade40"><rb>差異</rb><rp>(</rp><rt>차이</rt><rp>)</rp></ruby>나게 되었지만, <a href="/index.php?title=%E7%AF%86%E6%9B%B8%E9%AB%94&amp;action=edit&amp;redlink=1" class="new" title="篆書體 (없는 문서)"><ruby class="hanja grade10 unknown"><rb>篆書體</rb><rp>(</rp><rt>전서체</rt><rp>)</rp></ruby></a>만 봐도 <ruby class="hanja grade30"><rb>類似性</rb><rp>(</rp><rt>유사성</rt><rp>)</rp></ruby>을 <ruby class="hanja grade42"><rb>確認</rb><rp>(</rp><rt>확인</rt><rp>)</rp></ruby>할 수 있는데, <a href="/wiki/%E7%94%B2%E9%AA%A8%E6%96%87" class="mw-redirect" title="甲骨文"><ruby class="hanja grade40"><rb>甲骨文</rb><rp>(</rp><rt>갑골문</rt><rp>)</rp></ruby></a>에서는 ‘말‘과 ‘소리’의 뜻에 <ruby class="hanja grade60"><rb>區分</rb><rp>(</rp><rt>구분</rt><rp>)</rp></ruby>이 없었다가 <a href="/index.php?title=%E9%87%91%E6%96%87&amp;action=edit&amp;redlink=1" class="new" title="金文 (없는 문서)"><ruby class="hanja grade70 unknown"><rb>金文</rb><rp>(</rp><rt>금문</rt><rp>)</rp></ruby></a>에서 <ruby class="hanja grade60"><rb>言</rb><rp>(</rp><rt>언</rt><rp>)</rp></ruby>에 <ruby class="hanja grade32"><rb>劃</rb><rp>(</rp><rt>획</rt><rp>)</rp></ruby>을 하나 더 그어 두 뜻을 <ruby class="hanja grade60"><rb>區別</rb><rp>(</rp><rt>구별</rt><rp>)</rp></ruby>하게 된 것이다.
</p><p><a href="/index.php?title=%E6%84%8F&amp;action=edit&amp;redlink=1" class="new" title="意 (없는 문서)"><ruby class="hanja grade62 unknown"><rb>意</rb><rp>(</rp><rt>의</rt><rp>)</rp></ruby></a>, <a href="/index.php?title=%E9%9F%BF&amp;action=edit&amp;redlink=1" class="new" title="響 (없는 문서)"><ruby class="hanja grade32 unknown"><rb>響</rb><rp>(</rp><rt>향</rt><rp>)</rp></ruby></a> <ruby class="hanja grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby> 다른 글<ruby class="hanja grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>에 <ruby class="hanja grade52"><rb>結合</rb><rp>(</rp><rt>결합</rt><rp>)</rp></ruby>되어 ‘소리’와 <ruby class="hanja grade32"><rb>關聯</rb><rp>(</rp><rt>관련</rt><rp>)</rp></ruby>된 뜻을 나타내기도 한다.
</p>
