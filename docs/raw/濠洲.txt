<h2><span id="槪要(개요)"></span><span class="mw-headline" id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"><ruby class="hanja grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%BF%A0%E6%B4%B2&amp;action=edit&amp;section=1" title="부분 편집: 槪要(개요)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><b><ruby class="hanja grade20"><rb>濠洲聯邦</rb><rp>(</rp><rt>호주련방</rt><rp>)</rp></ruby></b>, <ruby class="hanja grade40"><rb>通稱</rb><rp>(</rp><rt>통칭</rt><rp>)</rp></ruby> “<b><ruby class="hanja grade20"><rb>濠洲</rb><rp>(</rp><rt>호주</rt><rp>)</rp></ruby></b>”는 <a href="/index.php?title=%E5%A4%A7%E6%B4%8B%E6%B4%B2&amp;action=edit&amp;redlink=1" class="new" title="大洋洲 (없는 문서)"><ruby class="hanja grade32 unknown"><rb>大洋洲</rb><rp>(</rp><rt>대양주</rt><rp>)</rp></ruby></a>에 <ruby class="hanja grade42"><rb>位置</rb><rp>(</rp><rt>위치</rt><rp>)</rp></ruby>한 <a href="/index.php?title=%E6%BF%A0%E6%B4%B2%E5%A4%A7%E9%99%B8&amp;action=edit&amp;redlink=1" class="new" title="濠洲大陸 (없는 문서)"><ruby class="hanja grade20 unknown"><rb>濠洲大陸</rb><rp>(</rp><rt>호주대륙</rt><rp>)</rp></ruby></a> <ruby class="hanja grade60"><rb>本土</rb><rp>(</rp><rt>본토</rt><rp>)</rp></ruby>, <a href="/index.php?title=%ED%83%9C%EC%A6%88%EB%A9%94%EC%9D%B4%EB%8B%88%EC%95%84%EC%84%AC&amp;action=edit&amp;redlink=1" class="new" title="태즈메이니아섬 (없는 문서)">태즈메이니아섬</a>과 수많은 작은 섬으로 <ruby class="hanja grade40"><rb>構成</rb><rp>(</rp><rt>구성</rt><rp>)</rp></ruby>된 <a href="/index.php?title=%E7%AB%8B%E6%86%B2%E5%90%9B%E4%B8%BB%E5%88%B6&amp;action=edit&amp;redlink=1" class="new" title="立憲君主制 (없는 문서)"><ruby class="hanja grade40 unknown"><rb>立憲君主制</rb><rp>(</rp><rt>입헌군주제</rt><rp>)</rp></ruby></a> <ruby class="hanja grade72"><rb>國家</rb><rp>(</rp><rt>국가</rt><rp>)</rp></ruby>이다. <a href="/index.php?title=%E8%8B%B1%E8%81%AF%E9%82%A6&amp;action=edit&amp;redlink=1" class="new" title="英聯邦 (없는 문서)"><ruby class="hanja grade30 unknown"><rb>英聯邦</rb><rp>(</rp><rt>영련방</rt><rp>)</rp></ruby></a> <ruby class="hanja grade42"><rb>會員國</rb><rp>(</rp><rt>회원국</rt><rp>)</rp></ruby>이며 <a href="/index.php?title=%E8%8B%B1%E8%81%AF%E9%82%A6%E7%8E%8B%E5%9C%8B&amp;action=edit&amp;redlink=1" class="new" title="英聯邦王國 (없는 문서)"><ruby class="hanja grade30 unknown"><rb>英聯邦王國</rb><rp>(</rp><rt>영련방왕국</rt><rp>)</rp></ruby></a>의 한 나라이다.
</p>
<h2><span id="國家象徵(국가상징)"></span><span class="mw-headline" id=".E5.9C.8B.E5.AE.B6.E8.B1.A1.E5.BE.B5.28.EA.B5.AD.EA.B0.80.EC.83.81.EC.A7.95.29"><ruby class="hanja grade32"><rb>國家象徵</rb><rp>(</rp><rt>국가상징</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%BF%A0%E6%B4%B2&amp;action=edit&amp;section=2" title="부분 편집: 國家象徵(국가상징)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span id="國名(국명)"></span><span class="mw-headline" id=".E5.9C.8B.E5.90.8D.28.EA.B5.AD.EB.AA.85.29"><ruby class="hanja grade72"><rb>國名</rb><rp>(</rp><rt>국명</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%BF%A0%E6%B4%B2&amp;action=edit&amp;section=3" title="부분 편집: 國名(국명)">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><ruby class="hanja grade72"><rb>國名</rb><rp>(</rp><rt>국명</rt><rp>)</rp></ruby>의 <ruby class="hanja grade60"><rb>由來</rb><rp>(</rp><rt>유래</rt><rp>)</rp></ruby>는 <a href="/index.php?title=%EB%9D%BC%ED%8B%B4%E8%AA%9E&amp;action=edit&amp;redlink=1" class="new" title="라틴語 (없는 문서)">라틴<ruby class="hanja grade70 unknown"><rb>語</rb><rp>(</rp><rt>어</rt><rp>)</rp></ruby></a>로 “<ruby class="hanja grade80"><rb>南</rb><rp>(</rp><rt>남</rt><rp>)</rp></ruby>쪽의 땅”을 <ruby class="hanja grade42"><rb>意味</rb><rp>(</rp><rt>의미</rt><rp>)</rp></ruby>하는 terra australis, 이것은 유럽에서 <ruby class="hanja grade52"><rb>傳說</rb><rp>(</rp><rt>전설</rt><rp>)</rp></ruby>의 <ruby class="hanja grade52"><rb>大陸</rb><rp>(</rp><rt>대륙</rt><rp>)</rp></ruby>, Terra Australis Incognita의 것을 가리키고있다.
</p><p><a href="/wiki/%E9%9F%93%E5%9C%8B%E8%AA%9E" title="韓國語"><ruby class="hanja grade70"><rb>韓國語</rb><rp>(</rp><rt>한국어</rt><rp>)</rp></ruby></a> <ruby class="hanja grade72"><rb>國名</rb><rp>(</rp><rt>국명</rt><rp>)</rp></ruby>의 <ruby class="hanja grade60"><rb>由來</rb><rp>(</rp><rt>유래</rt><rp>)</rp></ruby>는 <ruby class="hanja grade10"><rb>濠斯太剌利亞洲</rb><rp>(</rp><rt>호사태랄리아주</rt><rp>)</rp></ruby>의 <ruby class="hanja grade40"><rb>略語</rb><rp>(</rp><rt>약어</rt><rp>)</rp></ruby>. <a href="/wiki/%E6%97%A5%E6%9C%AC%E8%AA%9E" title="日本語"><ruby class="hanja grade60"><rb>日本語</rb><rp>(</rp><rt>일본어</rt><rp>)</rp></ruby></a>의 <a href="/wiki/%E6%BC%A2%E5%AD%97" title="漢字"><ruby class="hanja grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby></a><ruby class="hanja grade62"><rb>表記</rb><rp>(</rp><rt>표기</rt><rp>)</rp></ruby>는 <ruby class="hanja grade10"><rb>濠太剌利</rb><rp>(</rp><rt>호태랄리</rt><rp>)</rp></ruby>이며, 거기에서 따와 <ruby class="hanja grade20"><rb>濠洲</rb><rp>(</rp><rt>호주</rt><rp>)</rp></ruby>라고도 불린다. “<a href="/index.php?title=%E8%81%AF%E9%82%A6&amp;action=edit&amp;redlink=1" class="new" title="聯邦 (없는 문서)"><ruby class="hanja grade30 unknown"><rb>聯邦</rb><rp>(</rp><rt>연방</rt><rp>)</rp></ruby></a>”을 붙여 <ruby class="hanja grade20"><rb>濠洲聯邦</rb><rp>(</rp><rt>호주련방</rt><rp>)</rp></ruby>라고도 한다. <a href="/index.php?title=%E4%B8%AD%E5%9C%8B%E8%AA%9E%E5%9C%88&amp;action=edit&amp;redlink=1" class="new" title="中國語圈 (없는 문서)"><ruby class="hanja grade20 unknown"><rb>中國語圈</rb><rp>(</rp><rt>중국어권</rt><rp>)</rp></ruby></a>의 <ruby class="hanja grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby> <ruby class="hanja grade62"><rb>表記</rb><rp>(</rp><rt>표기</rt><rp>)</rp></ruby>는 <ruby class="hanja grade0"><rb>澳大利亞</rb><rp>(</rp><rt>오대리아</rt><rp>)</rp></ruby>, <ruby class="hanja grade40"><rb>略稱</rb><rp>(</rp><rt>약칭</rt><rp>)</rp></ruby>은 <ruby class="hanja grade0"><rb>澳洲</rb><rp>(</rp><rt>오주</rt><rp>)</rp></ruby>이다.
</p>
<h2><span id="領土(영토)"></span><span class="mw-headline" id=".E9.A0.98.E5.9C.9F.28.EC.98.81.ED.86.A0.29"><ruby class="hanja grade50"><rb>領土</rb><rp>(</rp><rt>영토</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%BF%A0%E6%B4%B2&amp;action=edit&amp;section=4" title="부분 편집: 領土(영토)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
