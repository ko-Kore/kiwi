<h2><span id="槪要(개요)"></span><span class="mw-headline" id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"><ruby class="hanja grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%ED%94%BC%EC%9E%90%ED%8F%AC%ED%85%8C%ED%86%A0&amp;action=edit&amp;section=1" title="부분 편집: 槪要(개요)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>피자포테토(<ruby class="hanja grade80"><rb>日</rb><rp>(</rp><rt>일</rt><rp>)</rp></ruby>: ピザポテト)는 <a href="/index.php?title=1992%E5%B9%B4&amp;action=edit&amp;redlink=1" class="new" title="1992年 (없는 문서)">1992<ruby class="hanja grade80 unknown"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby></a><a href="/index.php?title=9%E6%9C%88&amp;action=edit&amp;redlink=1" class="new" title="9月 (없는 문서)">9<ruby class="hanja grade80 unknown"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby></a>부터 <a href="/index.php?title=%EA%B0%80%EB%A3%A8%EB%B9%84&amp;action=edit&amp;redlink=1" class="new" title="가루비 (없는 문서)">가루비</a><ruby class="hanja grade32"><rb>株式會社</rb><rp>(</rp><rt>주식회사</rt><rp>)</rp></ruby>가 <ruby class="hanja grade42"><rb>製造</rb><rp>(</rp><rt>제조</rt><rp>)</rp></ruby>・<ruby class="hanja grade30"><rb>販賣</rb><rp>(</rp><rt>판매</rt><rp>)</rp></ruby>하고있는 <a href="/index.php?title=%E8%8F%93%E5%AD%90&amp;action=edit&amp;redlink=1" class="new" title="菓子 (없는 문서)"><ruby class="hanja grade20 unknown"><rb>菓子</rb><rp>(</rp><rt>과자</rt><rp>)</rp></ruby></a><ruby class="hanja grade42"><rb>製品</rb><rp>(</rp><rt>제품</rt><rp>)</rp></ruby>이다.
</p>
<h2><span id="特徵(특징)"></span><span class="mw-headline" id=".E7.89.B9.E5.BE.B5.28.ED.8A.B9.EC.A7.95.29"><ruby class="hanja grade32"><rb>特徵</rb><rp>(</rp><rt>특징</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%ED%94%BC%EC%9E%90%ED%8F%AC%ED%85%8C%ED%86%A0&amp;action=edit&amp;section=2" title="부분 편집: 特徵(특징)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>그 이름으로부터 <ruby class="hanja grade40"><rb>推測</rb><rp>(</rp><rt>추측</rt><rp>)</rp></ruby>되는대로, <a href="/index.php?title=%ED%94%BC%EC%9E%90&amp;action=edit&amp;redlink=1" class="new" title="피자 (없는 문서)">피자</a>의 맛을 <ruby class="hanja grade50"><rb>再現</rb><rp>(</rp><rt>재현</rt><rp>)</rp></ruby>하려하는 <ruby class="hanja grade42"><rb>製品</rb><rp>(</rp><rt>제품</rt><rp>)</rp></ruby>이며, <ruby class="hanja grade62"><rb>第一</rb><rp>(</rp><rt>제일</rt><rp>)</rp></ruby>로 <ruby class="hanja grade32"><rb>特徵的</rb><rp>(</rp><rt>특징적</rt><rp>)</rp></ruby>인<ruby class="hanja grade40"><rb>點</rb><rp>(</rp><rt>점</rt><rp>)</rp></ruby>은, <ruby class="hanja grade42"><rb>眞</rb><rp>(</rp><rt>진</rt><rp>)</rp></ruby>짜 <a href="/index.php?title=%EC%97%90%EB%A9%98%ED%83%88_%EC%B9%98%EC%A6%88&amp;action=edit&amp;redlink=1" class="new" title="에멘탈 치즈 (없는 문서)">에멘탈 치즈</a>와 <a href="/index.php?title=%EC%B2%B4%EB%8B%A4_%EC%B9%98%EC%A6%88&amp;action=edit&amp;redlink=1" class="new" title="체다 치즈 (없는 문서)">체다 치즈</a>의 <ruby class="hanja grade60"><rb>使用</rb><rp>(</rp><rt>사용</rt><rp>)</rp></ruby>을<ruby class="hanja grade60"><rb>通</rb><rp>(</rp><rt>통</rt><rp>)</rp></ruby>한 <ruby class="hanja grade20"><rb>濃厚</rb><rp>(</rp><rt>농후</rt><rp>)</rp></ruby>한 <a href="/index.php?title=%EC%B9%98%EC%A6%88&amp;action=edit&amp;redlink=1" class="new" title="치즈 (없는 문서)">치즈</a>의<ruby class="hanja grade42"><rb>風味</rb><rp>(</rp><rt>풍미</rt><rp>)</rp></ruby>라 할수있다. 치즈<ruby class="hanja grade52"><rb>以外</rb><rp>(</rp><rt>이외</rt><rp>)</rp></ruby>에도 피자의 맛을 <ruby class="hanja grade50"><rb>再現</rb><rp>(</rp><rt>재현</rt><rp>)</rp></ruby>하기<ruby class="hanja grade42"><rb>爲</rb><rp>(</rp><rt>위</rt><rp>)</rp></ruby>해 토마토 플레이크, 파슬리<ruby class="hanja grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby>도 올려져있다.
