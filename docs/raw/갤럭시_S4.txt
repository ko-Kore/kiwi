<p><a href="/index.php?title=%E4%B8%89%E6%98%9F%E9%9B%BB%E5%AD%90&amp;action=edit&amp;redlink=1" class="new" title="三星電子 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>三星電子</rb><rp>(</rp><rt>삼성전자</rt><rp>)</rp></ruby></a>가 <a href="/index.php?title=2013%E5%B9%B4&amp;action=edit&amp;redlink=1" class="new" title="2013年 (없는 문서)">2013<ruby class="hanja grade80 unknown"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby></a><a href="/index.php?title=3%E6%9C%88&amp;action=edit&amp;redlink=1" class="new" title="3月 (없는 문서)">3<ruby class="hanja grade80 unknown"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby></a> <ruby class="hanja grade60"><rb>公開</rb><rp>(</rp><rt>공개</rt><rp>)</rp></ruby>한 <a href="/index.php?title=%EC%95%88%EB%93%9C%EB%A1%9C%EC%9D%B4%EB%93%9C_%EC%8A%A4%EB%A7%88%ED%8A%B8%ED%8F%B0&amp;action=edit&amp;redlink=1" class="new" title="안드로이드 스마트폰 (없는 문서)">안드로이드 스마트폰</a>으로 <a href="/wiki/%EA%B0%A4%EB%9F%AD%EC%8B%9C_S_%EC%8B%9C%EB%A6%AC%EC%A6%88" title="갤럭시 S 시리즈">갤럭시 S 시리즈</a>의 네<ruby class="hanja grade60"><rb>番</rb><rp>(</rp><rt>번</rt><rp>)</rp></ruby>째 <ruby class="hanja grade42"><rb>製品</rb><rp>(</rp><rt>제품</rt><rp>)</rp></ruby>이다.
</p>
<h2><span id="特徵(특징)"></span><span class="mw-headline" id=".E7.89.B9.E5.BE.B5.28.ED.8A.B9.EC.A7.95.29"><ruby class="hanja grade32"><rb>特徵</rb><rp>(</rp><rt>특징</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EA%B0%A4%EB%9F%AD%EC%8B%9C_S4&amp;action=edit&amp;section=2" title="부분 편집: 特徵(특징)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>4.99<a href="/index.php?title=%EC%9D%B8%EC%B9%98&amp;action=edit&amp;redlink=1" class="new" title="인치 (없는 문서)">인치</a> <a href="/index.php?title=%ED%92%80_HD&amp;action=edit&amp;redlink=1" class="new" title="풀 HD (없는 문서)">풀 HD</a> <a href="/index.php?title=%EC%8A%88%ED%8D%BC_%EC%95%84%EB%AA%B0%EB%A0%88%EB%93%9C&amp;action=edit&amp;redlink=1" class="new" title="슈퍼 아몰레드 (없는 문서)">슈퍼 아몰레드</a> <ruby class="hanja grade60"><rb>畵面</rb><rp>(</rp><rt>화면</rt><rp>)</rp></ruby>이 <ruby class="hanja grade40"><rb>適用</rb><rp>(</rp><rt>적용</rt><rp>)</rp></ruby>되었다. <a href="/index.php?title=AP&amp;action=edit&amp;redlink=1" class="new" title="AP (없는 문서)">AP</a>는 <a href="/index.php?title=%E4%B8%96%E7%95%8C%E6%9C%80%E5%88%9D&amp;action=edit&amp;redlink=1" class="new" title="世界最初 (없는 문서)"><ruby class="hanja grade50 unknown"><rb>世界最初</rb><rp>(</rp><rt>세계최초</rt><rp>)</rp></ruby></a>로 <a href="/index.php?title=%EB%B9%85.%EB%A6%AC%ED%8B%80&amp;action=edit&amp;redlink=1" class="new" title="빅.리틀 (없는 문서)">빅.리틀</a> <ruby class="hanja grade50"><rb>技術</rb><rp>(</rp><rt>기술</rt><rp>)</rp></ruby>이 <ruby class="hanja grade40"><rb>適用</rb><rp>(</rp><rt>적용</rt><rp>)</rp></ruby>된 CPU를 <ruby class="hanja grade10"><rb>搭載</rb><rp>(</rp><rt>탑재</rt><rp>)</rp></ruby>한 <a href="/index.php?title=%EC%97%91%EC%8B%9C%EB%85%B8%EC%8A%A4&amp;action=edit&amp;redlink=1" class="new" title="엑시노스 (없는 문서)">엑시노스</a> 5 옥타 (5410)이 <ruby class="hanja grade32"><rb>內藏</rb><rp>(</rp><rt>내장</rt><rp>)</rp></ruby>되어 <ruby class="hanja grade72"><rb>電力</rb><rp>(</rp><rt>전력</rt><rp>)</rp></ruby> <ruby class="hanja grade32"><rb>效率</rb><rp>(</rp><rt>효율</rt><rp>)</rp></ruby>이 <ruby class="hanja grade50"><rb>改選</rb><rp>(</rp><rt>개선</rt><rp>)</rp></ruby>되었다.
</p><p><a href="/index.php?title=3G&amp;action=edit&amp;redlink=1" class="new" title="3G (없는 문서)">3G</a> 모델과 함께 <a href="/index.php?title=LTE&amp;action=edit&amp;redlink=1" class="new" title="LTE (없는 문서)">LTE</a> 모델이 먼저 <ruby class="hanja grade50"><rb>發賣</rb><rp>(</rp><rt>발매</rt><rp>)</rp></ruby>되었고, <ruby class="hanja grade72"><rb>後</rb><rp>(</rp><rt>후</rt><rp>)</rp></ruby>에 <ruby class="hanja grade32"><rb>若干</rb><rp>(</rp><rt>약간</rt><rp>)</rp></ruby>의 <ruby class="hanja grade42"><rb>修正</rb><rp>(</rp><rt>수정</rt><rp>)</rp></ruby>을 거친 <a href="/index.php?title=LTE-A&amp;action=edit&amp;redlink=1" class="new" title="LTE-A (없는 문서)">LTE-A</a> 모델이 <ruby class="hanja grade70"><rb>出市</rb><rp>(</rp><rt>출시</rt><rp>)</rp></ruby>되었다.
</p>
