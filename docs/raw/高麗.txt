<p><b><ruby class="hanja grade42"><rb>高麗</rb><rp>(</rp><rt>고려</rt><rp>)</rp></ruby></b>는 <a href="/index.php?title=918%E5%B9%B4&amp;action=edit&amp;redlink=1" class="new" title="918年 (없는 문서)">918<ruby grade52"><rb>以後</rb><rp>(</rp><rt>이후</rt><rp>)</rp></ruby>, <a href="/index.php?title=1392%E5%B9%B4&amp;action=edit&amp;redlink=1" class="new" title="1392年 (없는 문서)">1392<ruby class="hanja grade80 unknown"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby></a> <a href="/index.php?title=%E6%9D%8E%E6%88%90%E6%A1%82&amp;action=edit&amp;redlink=1" class="new" title="李成桂 (없는 문서)"><ruby class="hanja grade32 unknown"><rb>李成桂</rb><rp>(</rp><rt>이성계</rt><rp>)</rp></ruby></a>에 <ruby class="hanja grade40"><rb>依</rb><rp>(</rp><rt>의</rt><rp>)</rp></ruby>해 <ruby class="hanja grade32"><rb>滅亡</rb><rp>(</rp><rt>멸망</rt><rp>)</rp></ruby>하기까지 <a href="/index.php?title=%E9%9F%93%E5%8D%8A%E5%B3%B6&amp;action=edit&amp;redlink=1" class="new" title="韓半島 (없는 문서)"><ruby class="hanja grade50 unknown"><rb>韓半島</rb><rp>(</rp><rt>한반도</rt><rp>)</rp></ruby></a>에 <ruby class="hanja grade40"><rb>存在</rb><rp>(</rp><rt>존재</rt><rp>)</rp></ruby>하던 <a href="/wiki/%E5%9C%8B%E5%AE%B6" unknown"><rb>外王內帝</rb><rp>(</rp><rt>외왕내제</rt><rp>)</rp></ruby></a>를 가장 <ruby class="hanja grade40"><rb>積極的</rb><rp>(</rp><rt>적극적</rt><rp>)</rp></ruby>으로 한 <ruby class="hanja grade72"><rb>國家</rb><rp>(</rp><rt>국가</rt><rp>)</rp></ruby>이며 <ruby class="hanja grade42"><rb>高麗</rb><rp>(</rp><rt>고려</rt><rp>)</rp></ruby>의 <ruby class="hanja grade40"><rb>君主</rb><rp>(</rp><rt>군주</rt><rp>)</rp></ruby>는 <ruby class="hanja grade72"><rb>生前</rb><rp>(</rp><rt>생전</rt><rp>)</rp></ruby>에는 <ruby class="hanja grade32"><rb>皇帝</rb><rp>(</rp><rt>황제</rt><rp>)</rp></ruby>라 <ruby class="hanja grade40"><rb>稱</rb><rp>(</rp><rt>칭</rt><rp>)</rp></ruby>했으나 <ruby class="hanja grade30"><rb>崩御</rb><rp>(</rp><rt>붕어</rt><rp>)</rp></ruby> <ruby class="hanja grade52"><rb>以後</rb><rp>(</rp><rt>이후</rt><rp>)</rp></ruby> <a href="/index.php?title=%E8%AB%A1%E8%99%9F&amp;action=edit&amp;redlink=1" class="new" title="諡號 (없는 문서)"><ruby class="hanja grade10 unknown"><rb>諡號</rb><rp>(</rp><rt>시호</rt><rp>)</rp></ruby></a>는 <ruby
