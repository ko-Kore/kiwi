<p><b><ruby class="hanja api grade60"><rb>新式</rb><rp>(</rp><rt>신식</rt><rp>)</rp></ruby><ruby class="hanja api grade40"><rb>回轉</rb><rp>(</rp><rt>회전</rt><rp>)</rp></ruby><ruby class="hanja api grade10"><rb>交叉路</rb><rp>(</rp><rt>교차로</rt><rp>)</rp></ruby></b>(<span style="font-size: smaller;"><a href="/wiki/%E8%8B%B1%E8%AA%9E" title="英語">英語</a>: </span><span lang="en">Modern roundabout</span>)는 1960<ruby class="hanja api grade62"><rb>年代</rb><rp>(</rp><rt>연대</rt><rp>)</rp></ruby>부터 <a href="/wiki/%E8%8B%B1%E5%9C%8B" title="英國"><ruby class="hanja api grade60"><rb>英國</rb><rp>(</rp><rt>영국</rt><rp>)</rp></ruby></a>에서 <ruby class="hanja api grade50"><rb>改良</rb><rp>(</rp><rt>개량</rt><rp>)</rp></ruby>된 <ruby class="hanja api grade40"><rb>回轉</rb><rp>(</rp><rt>회전</rt><rp>)</rp></ruby><ruby class="hanja api grade10"><rb>交叉路</rb><rp>(</rp><rt>교차로</rt><rp>)</rp></ruby> <ruby class="hanja api grade60"><rb>方式</rb><rp>(</rp><rt>방식</rt><rp>)</rp></ruby>으로 <ruby class="hanja api grade40"><rb>優秀性</rb><rp>(</rp><rt>우수성</rt><rp>)</rp></ruby>이 <ruby class="hanja api grade40"><rb>立證</rb><rp>(</rp><rt>입증</rt><rp>)</rp></ruby>되어 <ruby class="hanja api grade62"><rb>全世界</rb><rp>(</rp><rt>전세계</rt><rp>)</rp></ruby>에 <ruby class="hanja api grade32"><rb>普及</rb><rp>(</rp><rt>보급</rt><rp>)</rp></ruby>되고 있다. 
</p><p><ruby class="hanja api grade32"><rb>特徵</rb><rp>(</rp><rt>특징</rt><rp>)</rp></ruby>은 다음과 같다. 
</p>
<ul><li><a href="/index.php?title=%E4%BF%A1%E8%99%9F%E7%87%88&amp;action=edit&amp;redlink=1" class="new" title="信號燈 (없는 文書)"><ruby class="hanja api grade42 unknown"><rb>信號燈</rb><rp>(</rp><rt>신호등</rt><rp>)</rp></ruby></a>이 없다.</li>
<li><a href="/index.php?title=%E4%B8%80%E6%96%B9%E9%80%9A%E8%A1%8C%E9%81%93%E8%B7%AF&amp;action=edit&amp;redlink=1" class="new" title="一方通行道路 (없는 文書)"><ruby class="hanja api grade60 unknown"><rb>一方通行</rb><rp>(</rp><rt>일방통행</rt><rp>)</rp></ruby><ruby class="hanja api grade60 unknown"><rb>道路</rb><rp>(</rp><rt>도로</rt><rp>)</rp></ruby></a>의 <ruby class="hanja api grade50"><rb>最多</rb><rp>(</rp><rt>최다</rt><rp>)</rp></ruby><ruby class="hanja api grade62"><rb>車線數</rb><rp>(</rp><rt>차선수</rt><rp>)</rp></ruby>는 2<ruby class="hanja api grade62"><rb>車線</rb><rp>(</rp><rt>차선</rt><rp>)</rp></ruby>이다.</li>
<li><ruby class="hanja api grade40"><rb>回轉</rb><rp>(</rp><rt>회전</rt><rp>)</rp></ruby><ruby class="hanja api grade20"><rb>車輛</rb><rp>(</rp><rt>차량</rt><rp>)</rp></ruby>은 <ruby class="hanja api grade42"><rb>進入</rb><rp>(</rp><rt>진입</rt><rp>)</rp></ruby><ruby class="hanja api grade20"><rb>車輛</rb><rp>(</rp><rt>차량</rt><rp>)</rp></ruby>보다 높은 <a href="/index.php?title=%E5%84%AA%E5%85%88%E6%AC%8A&amp;action=edit&amp;redlink=1" class="new" title="優先權 (없는 文書)"><ruby class="hanja api grade40 unknown"><rb>優先權</rb><rp>(</rp><rt>우선권</rt><rp>)</rp></ruby></a>을 가진다.</li>
<li><ruby class="hanja api grade42"><rb>進入</rb><rp>(</rp><rt>진입</rt><rp>)</rp></ruby><ruby class="hanja api grade62"><rb>車線</rb><rp>(</rp><rt>차선</rt><rp>)</rp></ruby> 마다 <ruby class="hanja api grade42"><rb>進出</rb><rp>(</rp><rt>진출</rt><rp>)</rp></ruby>할 수 있는 <ruby class="hanja api grade60"><rb>方向</rb><rp>(</rp><rt>방향</rt><rp>)</rp></ruby>이 <ruby class="hanja api grade40"><rb>嚴格</rb><rp>(</rp><rt>엄격</rt><rp>)</rp></ruby>히 <ruby class="hanja api grade50"><rb>規定</rb><rp>(</rp><rt>규정</rt><rp>)</rp></ruby>되어 있다.</li></ul>
<p><a href="/wiki/%E9%9F%93%E5%9C%8B" title="韓國"><ruby class="hanja api grade80"><rb>韓國</rb><rp>(</rp><rt>한국</rt><rp>)</rp></ruby></a>에서는 2020<ruby class="hanja api grade62"><rb>年代</rb><rp>(</rp><rt>연대</rt><rp>)</rp></ruby>부터 <ruby class="hanja api grade52"><rb>法的</rb><rp>(</rp><rt>법적</rt><rp>)</rp></ruby> <ruby class="hanja api grade40"><rb>根據</rb><rp>(</rp><rt>근거</rt><rp>)</rp></ruby>가 마련되고 있다. 그러나 <ruby class="hanja api grade60"><rb>通行</rb><rp>(</rp><rt>통행</rt><rp>)</rp></ruby><ruby class="hanja api grade52"><rb>方法</rb><rp>(</rp><rt>방법</rt><rp>)</rp></ruby> <ruby class="hanja api grade70"><rb>敎育</rb><rp>(</rp><rt>교육</rt><rp>)</rp></ruby>이 <ruby class="hanja api grade52"><rb>不實</rb><rp>(</rp><rt>부실</rt><rp>)</rp></ruby>하고 <ruby class="hanja api grade42"><rb>事故</rb><rp>(</rp><rt>사고</rt><rp>)</rp></ruby><ruby class="hanja api grade62"><rb>發生</rb><rp>(</rp><rt>발생</rt><rp>)</rp></ruby><ruby class="hanja api grade72"><rb>時</rb><rp>(</rp><rt>시</rt><rp>)</rp></ruby> <ruby class="hanja api grade52"><rb>過失</rb><rp>(</rp><rt>과실</rt><rp>)</rp></ruby><ruby class="hanja api grade32"><rb>比率</rb><rp>(</rp><rt>비율</rt><rp>)</rp></ruby>이 <ruby class="hanja api grade60"><rb>不合理</rb><rp>(</rp><rt>불합리</rt><rp>)</rp></ruby>하게 <ruby class="hanja api grade60"><rb>算定</rb><rp>(</rp><rt>산정</rt><rp>)</rp></ruby>되어 <ruby class="hanja api grade50"><rb>規則</rb><rp>(</rp><rt>규칙</rt><rp>)</rp></ruby>을 지키지 않고 <ruby class="hanja api grade70"><rb>不便</rb><rp>(</rp><rt>불편</rt><rp>)</rp></ruby>함을 <ruby class="hanja api grade32"><rb>呼訴</rb><rp>(</rp><rt>호소</rt><rp>)</rp></ruby>하는 <ruby class="hanja api grade72"><rb>市民</rb><rp>(</rp><rt>시민</rt><rp>)</rp></ruby>들이 많다.
</p>
