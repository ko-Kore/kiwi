<p><b>테오</b>(テオ)는 <a href="/index.php?title=Omoi&amp;action=edit&amp;redlink=1" class="new" title="Omoi (없는 문서)">Omoi</a>에 <ruby class="hanja grade40"><rb>依</rb><rp>(</rp><rt>의</rt><rp>)</rp></ruby>해 <ruby class="hanja grade32"><rb>作詞</rb><rp>(</rp><rt>작사</rt><rp>)</rp></ruby>·<ruby class="hanja grade50"><rb>作曲</rb><rp>(</rp><rt>작곡</rt><rp>)</rp></ruby>되어 <a href="/index.php?title=2017%E5%B9%B4&amp;action=edit&amp;redlink=1" class="new" title="2017年 (없는 문서)">2017<ruby class="hanja grade80 unknown"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby></a><a href="/index.php?title=7%E6%9C%888%E6%97%A5&amp;action=edit&amp;redlink=1" class="new" title="7月8日 (없는 문서)">7<ruby class="hanja grade80 unknown"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby>8<ruby class="hanja grade80 unknown"><rb>日</rb><rp>(</rp><rt>일</rt><rp>)</rp></ruby></a>에 <ruby class="hanja grade50"><rb>發賣</rb><rp>(</rp><rt>발매</rt><rp>)</rp></ruby>된 <a href="/index.php?title=%EB%B3%B4%EC%BB%AC%EB%A1%9C%EC%9D%B4%EB%93%9C_%EC%98%A4%EB%A6%AC%EC%A7%80%EB%84%90_%E6%9B%B2&amp;action=edit&amp;redlink=1" class="new" title="보컬로이드 오리지널 曲 (없는 문서)">보컬로이드 오리지널 <ruby class="hanja grade50 unknown"><rb>曲</rb><rp>(</rp><rt>곡</rt><rp>)</rp></ruby></a>이다.
</p>
<h2><span id="曲名(곡명)"></span><span class="mw-headline" id=".E6.9B.B2.E5.90.8D.28.EA.B3.A1.EB.AA.85.29"><ruby class="hanja grade50"><rb>曲名</rb><rp>(</rp><rt>곡명</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%ED%85%8C%EC%98%A4_(Omoi)&amp;action=edit&amp;section=1" title="부분 편집: 曲名(곡명)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>테오(テオ)라는 <ruby class="hanja grade50"><rb>曲名</rb><rp>(</rp><rt>곡명</rt><rp>)</rp></ruby>은 <ruby class="hanja grade32"><rb>歌詞</rb><rp>(</rp><rt>가사</rt><rp>)</rp></ruby>에도 <ruby class="hanja grade40"><rb>繼續</rb><rp>(</rp><rt>계속</rt><rp>)</rp></ruby> 나오는 ‘手を’(손을)라는 말과 <ruby class="hanja grade62"><rb>發音</rb><rp>(</rp><rt>발음</rt><rp>)</rp></ruby>이 같다.
</p>
<h2><span id="其他(기타)"></span><span class="mw-headline" id=".E5.85.B6.E4.BB.96.28.EA.B8.B0.ED.83.80.29"><ruby class="hanja grade32"><rb>其他</rb><rp>(</rp><rt>기타</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%ED%85%8C%EC%98%A4_(Omoi)&amp;action=edit&amp;section=2" title="부분 편집: 其他(기타)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>보컬에는 <a href="/index.php?title=%E5%88%9D%E9%9F%B3%EB%AF%B8%EC%BF%A0&amp;action=edit&amp;redlink=1" class="new" title="初音미쿠 (없는 문서)"><ruby class="hanja grade50 unknown"><rb>初音</rb><rp>(</rp><rt>초음</rt><rp>)</rp></ruby>미쿠</a>가, 코러스에는 <a href="/index.php?title=GUMI&amp;action=edit&amp;redlink=1" class="new" title="GUMI (없는 문서)">GUMI</a>, <a href="/index.php?title=%E9%8F%A1%E9%9F%B3%EB%A6%B0&amp;action=edit&amp;redlink=1" class="new" title="鏡音린 (없는 문서)"><ruby class="hanja grade40 unknown"><rb>鏡音</rb><rp>(</rp><rt>경음</rt><rp>)</rp></ruby>린</a>, <a href="/index.php?title=IA&amp;action=edit&amp;redlink=1" class="new" title="IA (없는 문서)">IA</a>가 <ruby class="hanja grade60"><rb>使用</rb><rp>(</rp><rt>사용</rt><rp>)</rp></ruby>되었다.
</p>
