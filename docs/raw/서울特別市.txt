<ruby class="hanja api grade60"><rb>使用</rb><rp>(</rp><rt>사용</rt><rp>)</rp></ruby>하다가 <a href="/index.php?title=2005%E5%B9%B4&amp;action=edit&amp;redlink=1" class="new" title="2005年(없는　文書）">2005<ruby grade60"><rb>現在</rb><rp>(</rp><rt>현재</rt><rp>)</rp></ruby> <ruby class="hanja api grade60"><rb>公式</rb><rp>(</rp><rt>공식</rt><rp>)</rp></ruby>으로는 이쪽이 많이 쓰이고 있다.
