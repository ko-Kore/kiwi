<h2><span id="槪要(개요)"></span><span class="mw-headline" id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"><ruby class="hanja grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EC%84%9C%EC%9A%B8%E7%89%B9%E5%88%A5%E5%B8%82&amp;action=edit&amp;section=1" title="부분 편집: 槪要(개요)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>서울<ruby class="hanja grade60"><rb>特別市</rb><rp>(</rp><rt>특별시</rt><rp>)</rp></ruby>는 <a href="/wiki/%E5%A4%A7%E9%9F%93%E6%B0%91%E5%9C%8B" title="大韓民國"><ruby class="hanja grade80"><rb>大韓民國</rb><rp>(</rp><rt>대한민국</rt><rp>)</rp></ruby></a>의 <ruby class="hanja grade30"><rb>唯一</rb><rp>(</rp><rt>유일</rt><rp>)</rp></ruby>한 <a href="/index.php?title=%E7%89%B9%E5%88%A5%E5%B8%82&amp;action=edit&amp;redlink=1" class="new" title="特別市 (없는 문서)"><ruby class="hanja grade60 unknown"><rb>特別市</rb><rp>(</rp><rt>특별시</rt><rp>)</rp></ruby></a>로 <ruby class="hanja grade80"><rb>大韓民國</rb><rp>(</rp><rt>대한민국</rt><rp>)</rp></ruby>의 <a href="/wiki/%E9%A6%96%E9%83%BD" title="首都"><ruby class="hanja grade50"><rb>首都</rb><rp>(</rp><rt>수도</rt><rp>)</rp></ruby></a>이며 <a href="/index.php?title=%E9%A6%96%E9%83%BD%E5%9C%88&amp;action=edit&amp;redlink=1" class="new" title="首都圈 (없는 문서)"><ruby class="hanja grade20 unknown"><rb>首都圈</rb><rp>(</rp><rt>수도권</rt><rp>)</rp></ruby></a> <ruby class="hanja grade50"><rb>最大</rb><rp>(</rp><rt>최대</rt><rp>)</rp></ruby>의 <ruby class="hanja grade50"><rb>都市</rb><rp>(</rp><rt>도시</rt><rp>)</rp></ruby>이다.
</p>
<h2><span id="名稱(명칭)"></span><span class="mw-headline" id=".E5.90.8D.E7.A8.B1.28.EB.AA.85.EC.B9.AD.29"><ruby class="hanja grade40"><rb>名稱</rb><rp>(</rp><rt>명칭</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EC%84%9C%EC%9A%B8%E7%89%B9%E5%88%A5%E5%B8%82&amp;action=edit&amp;section=2" title="부분 편집: 名稱(명칭)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/index.php?title=%E5%85%89%E5%BE%A9&amp;action=edit&amp;redlink=1" class="new" title="光復 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>光復</rb><rp>(</rp><rt>광복</rt><rp>)</rp></ruby></a> <ruby class="hanja grade52"><rb>以後</rb><rp>(</rp><rt>이후</rt><rp>)</rp></ruby> <a href="/index.php?title=1946%E5%B9%B4&amp;action=edit&amp;redlink=1" class="new" title="1946年 (없는 문서)">1946<ruby class="hanja grade80 unknown"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby></a> <a href="/index.php?title=%E4%BA%AC%E5%9F%8E%E5%BA%9C&amp;action=edit&amp;redlink=1" class="new" title="京城府 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>京城府</rb><rp>(</rp><rt>경성부</rt><rp>)</rp></ruby></a>를 <a href="/index.php?title=%EC%84%9C%EC%9A%B8%E5%B8%82&amp;action=edit&amp;redlink=1" class="new" title="서울市 (없는 문서)">서울<ruby class="hanja grade72 unknown"><rb>市</rb><rp>(</rp><rt>시</rt><rp>)</rp></ruby></a>로 <ruby class="hanja grade40"><rb>改稱</rb><rp>(</rp><rt>개칭</rt><rp>)</rp></ruby>하였으며, <a href="/index.php?title=%E4%BA%AC%E7%95%BF%E9%81%93_%EC%84%9C%EC%9A%B8%E5%B8%82&amp;action=edit&amp;redlink=1" class="new" title="京畿道 서울市 (없는 문서)"><ruby class="hanja grade32 unknown"><rb>京畿道</rb><rp>(</rp><rt>경기도</rt><rp>)</rp></ruby> 서울<ruby class="hanja grade72 unknown"><rb>市</rb><rp>(</rp><rt>시</rt><rp>)</rp></ruby></a>를 <a href="/index.php?title=%EC%84%9C%EC%9A%B8%E7%89%B9%E5%88%A5%E8%87%AA%E7%94%B1%E5%B8%82&amp;action=edit&amp;redlink=1" class="new" title="서울特別自由市 (없는 문서)">서울<ruby class="hanja grade60 unknown"><rb>特別自由市</rb><rp>(</rp><rt>특별자유시</rt><rp>)</rp></ruby></a>로 <ruby class="hanja grade32"><rb>昇格</rb><rp>(</rp><rt>승격</rt><rp>)</rp></ruby>하였고, 서울<ruby class="hanja grade60"><rb>特別市</rb><rp>(</rp><rt>특별시</rt><rp>)</rp></ruby>라는 명칭은 <a href="/index.php?title=1949%E5%B9%B4&amp;action=edit&amp;redlink=1" class="new" title="1949年 (없는 문서)">1949<ruby class="hanja grade80 unknown"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby></a> <ruby class="hanja grade40"><rb>改稱</rb><rp>(</rp><rt>개칭</rt><rp>)</rp></ruby>하고 나서부터 쓰이기 <ruby class="hanja grade62"><rb>始作</rb><rp>(</rp><rt>시작</rt><rp>)</rp></ruby>했다.
</p><p>“서울”은 <ruby class="hanja grade50"><rb>原來</rb><rp>(</rp><rt>원래</rt><rp>)</rp></ruby> <a href="/wiki/%E9%A6%96%E9%83%BD" title="首都"><ruby class="hanja grade50"><rb>首都</rb><rp>(</rp><rt>수도</rt><rp>)</rp></ruby></a>를 뜻하는 <a href="/wiki/%E5%9B%BA%E6%9C%89%E8%AA%9E" title="固有語"><ruby class="hanja grade50"><rb>固有語</rb><rp>(</rp><rt>고유어</rt><rp>)</rp></ruby></a>이며, <ruby class="hanja grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby>로는 <ruby class="hanja grade52"><rb>種種</rb><rp>(</rp><rt>종종</rt><rp>)</rp></ruby> “<ruby class="hanja grade60"><rb>京</rb><rp>(</rp><rt>경</rt><rp>)</rp></ruby>”으로 <ruby class="hanja grade62"><rb>表記</rb><rp>(</rp><rt>표기</rt><rp>)</rp></ruby>된다.&nbsp;<a href="/index.php?title=%E4%B8%AD%E5%9C%8B&amp;action=edit&amp;redlink=1" class="new" title="中國 (없는 문서)"><ruby class="hanja grade80 unknown"><rb>中國</rb><rp>(</rp><rt>중국</rt><rp>)</rp></ruby></a>에서는 옛 <ruby class="hanja grade70"><rb>地名</rb><rp>(</rp><rt>지명</rt><rp>)</rp></ruby>인 <a href="/index.php?title=%E6%BC%A2%E5%9F%8E&amp;action=edit&amp;redlink=1" class="new" title="漢城 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>漢城</rb><rp>(</rp><rt>한성</rt><rp>)</rp></ruby></a>을 <ruby class="hanja grade60"><rb>使用</rb><rp>(</rp><rt>사용</rt><rp>)</rp></ruby>하다가 <a href="/index.php?title=2005%E5%B9%B4&amp;action=edit&amp;redlink=1" class="new" title="2005年 (없는 문서)">2005<ruby class="hanja grade80 unknown"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby></a> <ruby class="hanja grade60"><rb>公式</rb><rp>(</rp><rt>공식</rt><rp>)</rp></ruby> <ruby class="hanja grade52"><rb>中國語表記法</rb><rp>(</rp><rt>중국어표기법</rt><rp>)</rp></ruby>이 <ruby class="hanja grade0"><rb><span class="noruby">“<ruby class="hanja grade10"><rb>首爾</rb><rp>(</rp><rt>수이</rt><rp>)</rp></ruby>”</span></rb><rp>(</rp><rt>[ʂoʊ˧˥&nbsp;əɻ˨˩˦]</rt><rp>)</rp></ruby>(“서울”의 <ruby class="hanja grade32"><rb>音譯</rb><rp>(</rp><rt>음역</rt><rp>)</rp></ruby>)로 <ruby class="hanja grade60"><rb>定</rb><rp>(</rp><rt>정</rt><rp>)</rp></ruby>해지면서 <ruby class="hanja grade60"><rb>現在</rb><rp>(</rp><rt>현재</rt><rp>)</rp></ruby> <ruby class="hanja grade60"><rb>公式</rb><rp>(</rp><rt>공식</rt><rp>)</rp></ruby>으로는 이쪽이 많이 쓰이고 있다.
