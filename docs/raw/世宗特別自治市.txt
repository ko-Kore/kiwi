<h2><span id="槪要(개요)"></span><span class="mw-headline" id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"><ruby class="hanja grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E4%B8%96%E5%AE%97%E7%89%B9%E5%88%A5%E8%87%AA%E6%B2%BB%E5%B8%82&amp;action=edit&amp;section=1" title="부분 편집: 槪要(개요)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><b><ruby class="hanja grade42"><rb>世宗特別自治市</rb><rp>(</rp><rt>세종특별자치시</rt><rp>)</rp></ruby></b>는 <a href="/wiki/%E5%A4%A7%E9%9F%93%E6%B0%91%E5%9C%8B" title="大韓民國"><ruby class="hanja grade80"><rb>大韓民國</rb><rp>(</rp><rt>대한민국</rt><rp>)</rp></ruby></a>의 <ruby class="hanja grade30"><rb>唯一</rb><rp>(</rp><rt>유일</rt><rp>)</rp></ruby>한 <a href="/index.php?title=%E7%89%B9%E5%88%A5%E8%87%AA%E6%B2%BB%E5%B8%82&amp;action=edit&amp;redlink=1" class="new" title="特別自治市 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>特別自治市</rb><rp>(</rp><rt>특별자치시</rt><rp>)</rp></ruby></a>이다. <ruby class="hanja grade40"><rb>行政中心複合都市</rb><rp>(</rp><rt>행정중심복합도시</rt><rp>)</rp></ruby>로서 <ruby class="hanja grade32"><rb>計劃</rb><rp>(</rp><rt>계획</rt><rp>)</rp></ruby>되고 <ruby class="hanja grade42"><rb>設置</rb><rp>(</rp><rt>설치</rt><rp>)</rp></ruby>되어 <a href="/wiki/%EC%84%9C%EC%9A%B8%E7%89%B9%E5%88%A5%E5%B8%82" title="서울特別市">서울<ruby class="hanja grade60"><rb>特別市</rb><rp>(</rp><rt>특별시</rt><rp>)</rp></ruby></a>로부터 <ruby class="hanja grade40"><rb>行政機關</rb><rp>(</rp><rt>행정기관</rt><rp>)</rp></ruby>들이 <ruby class="hanja grade40"><rb>移轉</rb><rp>(</rp><rt>이전</rt><rp>)</rp></ruby>되었으며, <a href="/index.php?title=2022%E5%B9%B4&amp;action=edit&amp;redlink=1" class="new" title="2022年 (없는 문서)">2022<ruby class="hanja grade80 unknown"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby></a> <ruby class="hanja grade60"><rb>現在</rb><rp>(</rp><rt>현재</rt><rp>)</rp></ruby> <ruby class="hanja grade80"><rb>大韓民國</rb><rp>(</rp><rt>대한민국</rt><rp>)</rp></ruby>의 <a href="/index.php?title=%E8%A1%8C%E6%94%BF%E9%A6%96%E9%83%BD&amp;action=edit&amp;redlink=1" class="new" title="行政首都 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>行政首都</rb><rp>(</rp><rt>행정수도</rt><rp>)</rp></ruby></a> <ruby class="hanja grade32"><rb>役割</rb><rp>(</rp><rt>역할</rt><rp>)</rp></ruby>을 <ruby class="hanja grade32"><rb>隨行</rb><rp>(</rp><rt>수행</rt><rp>)</rp></ruby>하고 있다.
</p>
