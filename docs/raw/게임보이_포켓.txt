<h2><span id=".ED.95.98.EB.93.9C.EC.9B.A8.EC.96.B4"></span><span class="mw-headline" id="하드웨어">하드웨어</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EA%B2%8C%EC%9E%84%EB%B3%B4%EC%9D%B4_%ED%8F%AC%EC%BC%93&amp;action=edit&amp;section=1" title="部分 編輯: 하드웨어">編輯</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja api grade52"><rb>基本的</rb><rp>(</rp><rt>기본적</rt><rp>)</rp></ruby>으로 <ruby class="hanja api grade50"><rb>初代</rb><rp>(</rp><rt>초대</rt><rp>)</rp></ruby><a href="/wiki/%EA%B2%8C%EC%9E%84%EB%B3%B4%EC%9D%B4" title="게임보이">게임보이</a>를 <ruby class="hanja api grade20"><rb>小型化</rb><rp>(</rp><rt>소형화</rt><rp>)</rp></ruby>·<ruby class="hanja api grade50"><rb>輕量化</rb><rp>(</rp><rt>경량화</rt><rp>)</rp></ruby>한 <ruby class="hanja api grade50"><rb>物件</rb><rp>(</rp><rt>물건</rt><rp>)</rp></ruby>이라 <ruby class="hanja api grade40"><rb>仕樣</rb><rp>(</rp><rt>사양</rt><rp>)</rp></ruby>에 큰 <ruby class="hanja api grade40"><rb>差異</rb><rp>(</rp><rt>차이</rt><rp>)</rp></ruby>는 없다. 다만 훨씬 작아진 크기에 맞춰 몇가지 <ruby class="hanja api grade40"><rb>變更</rb><rp>(</rp><rt>변경</rt><rp>)</rp></ruby><ruby class="hanja api grade32"><rb>事項</rb><rp>(</rp><rt>사항</rt><rp>)</rp></ruby>이 있었다.
</p><p><ruby class="hanja api grade40"><rb>電源</rb><rp>(</rp><rt>전원</rt><rp>)</rp></ruby>은 <a href="/index.php?title=AAA%E9%9B%BB%E6%B1%A0&amp;action=edit&amp;redlink=1" class="new" title="AAA電池 (없는 文書)">AAA<ruby class="hanja api grade32 unknown"><rb>電池</rb><rp>(</rp><rt>전지</rt><rp>)</rp></ruby></a> 두<ruby class="hanja api grade42"><rb>個</rb><rp>(</rp><rt>개</rt><rp>)</rp></ruby>만으로 <ruby class="hanja api grade62"><rb>作動</rb><rp>(</rp><rt>작동</rt><rp>)</rp></ruby>한다. 그런데 <ruby class="hanja api grade42"><rb>容量</rb><rp>(</rp><rt>용량</rt><rp>)</rp></ruby>도 작아졌으면서 <ruby class="hanja api grade50"><rb>原價</rb><rp>(</rp><rt>원가</rt><rp>)</rp></ruby><ruby class="hanja api grade42"><rb>節減</rb><rp>(</rp><rt>절감</rt><rp>)</rp></ruby>을 <ruby class="hanja api grade42"><rb>爲</rb><rp>(</rp><rt>위</rt><rp>)</rp></ruby>해서인지 배터리<ruby class="hanja api grade40"><rb>殘量</rb><rp>(</rp><rt>잔량</rt><rp>)</rp></ruby> <ruby class="hanja api grade42"><rb>確認</rb><rp>(</rp><rt>확인</rt><rp>)</rp></ruby><ruby class="hanja api grade62"><rb>用</rb><rp>(</rp><rt>용</rt><rp>)</rp></ruby> <a href="/index.php?title=LED&amp;action=edit&amp;redlink=1" class="new" title="LED (없는 文書)">LED</a>가 들어가지 않았다. 하도 <ruby class="hanja api grade70"><rb>不便</rb><rp>(</rp><rt>불편</rt><rp>)</rp></ruby>했는지 <ruby class="hanja api grade50"><rb>後期</rb><rp>(</rp><rt>후기</rt><rp>)</rp></ruby><ruby class="hanja api grade20"><rb>型</rb><rp>(</rp><rt>형</rt><rp>)</rp></ruby>에는 다시 <ruby class="hanja api grade32"><rb>追加</rb><rp>(</rp><rt>추가</rt><rp>)</rp></ruby>되었다.
</p><p><ruby class="hanja api grade60"><rb>畵面</rb><rp>(</rp><rt>화면</rt><rp>)</rp></ruby>이 많이 <ruby class="hanja api grade50"><rb>改善</rb><rp>(</rp><rt>개선</rt><rp>)</rp></ruby>되었는데, <ruby class="hanja api grade32"><rb>殘像</rb><rp>(</rp><rt>잔상</rt><rp>)</rp></ruby>이 덜하고 밝으면서 크기도 <ruby class="hanja api grade32"><rb>若干</rb><rp>(</rp><rt>약간</rt><rp>)</rp></ruby> 커져 <ruby class="hanja api (없는 文書)">게임보이 컬러</a>와 <ruby class="hanja api grade32"><rb>比較</rb><rp>(</rp><rt>비교</rt><rp>)</rp></ruby>해도 더욱 크고 밝을 <ruby class="hanja api grade42"><rb>程度</rb><rp>(</rp><rt>정도</rt><rp>)</rp></ruby>. (없는 文書)"><ruby class="hanja api grade10 unknown"><rb>醋酸</rb><rp>(</rp><rt>초산</rt><rp>)</rp></ruby><ruby class="hanja api grade52 unknown"><rb>化</rb><rp>(</rp><rt>화</rt><rp>)</rp></ruby><ruby class="hanja api grade32 unknown"><rb>症候群</rb><rp>(</rp><rt>증후군</rt><rp>)</rp></ruby></a> <ruby class="hanja api grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby>으로 <a
