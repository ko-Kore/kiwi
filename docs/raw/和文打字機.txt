<h2><span id="槪要(개요)"></span><span class="mw-headline" id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"><ruby class="hanja grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E5%92%8C%E6%96%87%E6%89%93%E5%AD%97%E6%A9%9F&amp;action=edit&amp;section=1" title="부분 편집: 槪要(개요)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><b><ruby class="hanja grade40"><rb>和文打字機</rb><rp>(</rp><rt>화문타자기</rt><rp>)</rp></ruby></b>(<ruby class="hanja grade62"><rb>和文</rb><rp>(</rp><rt>화문</rt><rp>)</rp></ruby>타이프라이터, <ruby class="hanja grade62"><rb>和文</rb><rp>(</rp><rt>화문</rt><rp>)</rp></ruby>タイプライター)는 <a href="/index.php?title=%E5%92%8C%E6%96%87&amp;action=edit&amp;redlink=1" class="new" title="和文 (없는 문서)"><ruby class="hanja grade62 unknown"><rb>和文</rb><rp>(</rp><rt>화문</rt><rp>)</rp></ruby></a>(<ruby class="hanja grade70"><rb>日文</rb><rp>(</rp><rt>일문</rt><rp>)</rp></ruby>)을 <ruby class="hanja grade50"><rb>打字</rb><rp>(</rp><rt>타자</rt><rp>)</rp></ruby>할수있는 <a href="/wiki/%E6%89%93%E5%AD%97%E6%A9%9F" title="打字機"><ruby class="hanja grade40"><rb>打字機</rb><rp>(</rp><rt>타자기</rt><rp>)</rp></ruby></a>이다.
</p>
<h2><span id="特徵(특징)"></span><span class="mw-headline" id=".E7.89.B9.E5.BE.B5.28.ED.8A.B9.EC.A7.95.29"><ruby class="hanja grade32"><rb>特徵</rb><rp>(</rp><rt>특징</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E5%92%8C%E6%96%87%E6%89%93%E5%AD%97%E6%A9%9F&amp;action=edit&amp;section=2" title="부분 편집: 特徵(특징)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%E5%B9%B3%E5%81%87%E5%90%8D" title="平假名"><ruby class="hanja grade42"><rb>平假名</rb><rp>(</rp><rt>평가명</rt><rp>)</rp></ruby></a>과 <a href="/wiki/%E7%89%87%E5%81%87%E5%90%8D" title="片假名"><ruby class="hanja grade32"><rb>片假名</rb><rp>(</rp><rt>편가명</rt><rp>)</rp></ruby></a>은 <ruby class="hanja grade32"><rb>勿論</rb><rp>(</rp><rt>물론</rt><rp>)</rp></ruby>이고 <a href="/wiki/%E6%BC%A2%E5%AD%97" title="漢字"><ruby class="hanja grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby></a>와 <a href="/index.php?title=%E8%8B%B1%E6%96%87%E5%AD%97&amp;action=edit&amp;redlink=1" class="new" title="英文字 (없는 문서)"><ruby class="hanja grade60 unknown"><rb>英文字</rb><rp>(</rp><rt>영문자</rt><rp>)</rp></ruby></a>까지 <ruby class="hanja grade50"><rb>打者</rb><rp>(</rp><rt>타자</rt><rp>)</rp></ruby>할 수 있으며, 이를 <ruby class="hanja grade42"><rb>爲</rb><rp>(</rp><rt>위</rt><rp>)</rp></ruby>해 <ruby class="hanja grade32"><rb>一般的</rb><rp>(</rp><rt>일반적</rt><rp>)</rp></ruby>인 <a href="/wiki/%E6%89%93%E5%AD%97%E6%A9%9F" title="打字機"><ruby class="hanja grade40"><rb>打字機</rb><rp>(</rp><rt>타자기</rt><rp>)</rp></ruby></a>와는 많이 다르게 생겼다. <ruby class="hanja grade42"><rb>實際</rb><rp>(</rp><rt>실제</rt><rp>)</rp></ruby>로 글<ruby class="hanja grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>를 찍는 <ruby class="hanja grade40"><rb>機能</rb><rp>(</rp><rt>기능</rt><rp>)</rp></ruby>을 하는 것은 레버 하나뿐이고, 수많은 <a href="/index.php?title=%E6%B4%BB%E5%AD%97&amp;action=edit&amp;redlink=1" class="new" title="活字 (없는 문서)"><ruby class="hanja grade70 unknown"><rb>活字</rb><rp>(</rp><rt>활자</rt><rp>)</rp></ruby></a><ruby class="hanja grade80"><rb>中</rb><rp>(</rp><rt>중</rt><rp>)</rp></ruby>에서 <ruby class="hanja grade50"><rb>願</rb><rp>(</rp><rt>원</rt><rp>)</rp></ruby>하는 글<ruby class="hanja grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>를 한<ruby class="hanja grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>한<ruby class="hanja grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby> 찾아야 한다. 타자기한대로 칠수있는글자는 1000~2000자정도라 완전한문장을치기위한 예비활자박스가 별도로있고, 타자기활자판에도 이를위한예비공간이 존재한다.
</p><p>수동화문타자기는 횡서가기본이지만 활자를90도돌려서 종서도할수있는 경우도있다한다.
