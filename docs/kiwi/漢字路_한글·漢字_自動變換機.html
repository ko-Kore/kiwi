漢字路 한글·漢字 自動變換機

原本 編輯 歷史
分類 : 國漢文混用 웹 사이트
네이버 뉴스 메인 페이지가 變換된 모습

[1] 漢字路(한자로)는 傳統文化硏究會(전통문화연구회)에서 開發(개발)·運營(운영)하는 웹 사이트이다. '한글·漢字(한자) 自動變換機(자동변환기)'라는 道具(도구)가 內藏(내장)되어 웹 사이트를 國漢文混用(국한문혼용)으로 變換(변환)해서 볼 수 있다.

機能(기능)[편집]

한글專用(전용)으로 된 韓國語(한국어) 웹 사이트를 國漢文混用(국한문혼용)으로 變換(변환)해주는 機能(기능)을 提供(제공)한다. 메인 페이지의 住所欄(주소란)에 願(원)하는 사이트의 住所(주소)를 입력하면 該當(해당) 사이트의 漢字語(한자어)를 漢字表記(한자표기)로 변환해 表示(표시)해준다. 텍스트 變換機能(변환기능)도 別途(별도)로 提供(제공)한다.

옵션을 바꿔 變換結果(변환결과)를 한글과 倂記(병기)할지, 漢字(한자)만 表示(표시)할지 選擇(선택)할 수 있으며, 漢字檢定試驗(한자검정시험)의 級數別(급수별)로 變換(변환)할 漢字(한자)의 區分(구분)을 選擇(선택)할 수도 있다.

短點(단점)[편집]

HTTPS를 支援(지원)하지 않는지, https로 로드되는 이미지 等(등) 리소스가 로드·表示(표시)되지 않거나 一部(일부) 사이트는 表示(표시)조차 되지 않기도 한다.

한글專用(전용) 文章(문장)의 漢字語(한자어)를 人間(인간)의 檢修(검수) 따위를 하지 않고 漢字(한자)로 變換(변환)하는 것이기 때문에[1], 誤變換(오변환)의 可能性(가능성)이 있으며 資料(자료) 等(등)으로 使用(사용)하려면 事前(사전)에 確認(확인)을 거쳐야 한다.

其他(기타)[편집]

蔚山大學校(울산대학교) 韓國語處理硏究室(한국어처리연구실)에서 開發(개발)한 ‘한글漢字自動變換機(한자자동변환기)’을 바탕으로 開發(개발)되었다 하며, 該當(해당) 技術(기술)은 UTagger에 基盤(기반)한 듯하다.

  1. ↑ 漢字(한자)를 한글로 變換(변환)하는 것보다 文脈(문맥)에 對(대)한 依存度(의존도)가 높다.
