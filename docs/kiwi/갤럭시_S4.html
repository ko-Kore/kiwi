槪要(개요)[편집]

三星電子(삼성전자)가 2013年(년)3月(월) 公開(공개)한 안드로이드 스마트폰으로 갤럭시 S 시리즈의 네番(번)째 製品(제품)이다.

特徵(특징)[편집]

4.99인치 풀 HD 슈퍼 아몰레드 畵面(화면)이 適用(적용)되었다. AP는 世界最初(세계최초)로 빅.리틀 技術(기술)이 適用(적용)된 CPU를 搭載(탑재)한 엑시노스 5 옥타 (5410)이 內藏(내장)되어 電力(전력) 效率(효율)이 改選(개선)되었다.

3G 모델과 함께 LTE 모델이 먼저 發賣(발매)되었고, 後(후)에 若干(약간)의 修正(수정)을 거친 LTE-A 모델이 出市(출시)되었다.

機能(기능)[편집]

高感度(고감도) 터치 스크린이 搭載(탑재)되어, 平素(평소)에는 別(별) 差異(차이) 없이 使用(사용)하다가 掌匣(장갑)을 낀 손으로도 터치 입력이 認識(인식)되도록 一時的(일시적)으로 感度(감도)를 上向(상향)시키거나, 터치하지 않고 손가락을 아이콘 위에 가져다 대서[1] 미리보기를 表示(표시)하거나 할 수 있는 '에어뷰' 機能(기능)이 追加(추가)되었다.

에어 제스처 센서가 追加(추가)되어 화면에 손을 가져가지도 않고 電話(전화)를 받거나 슬라이드 쇼의 寫眞(사진)을 넘기는 等(등)의 動作(동작)을 空中(공중) 제스처로 入力(입력)하는 것이 可能(가능)해졌다.

色相(색상)[편집]

最初(최초) 公開(공개) 當時(당시)에 '블랙 미스트'와 '화이트 프로스트' 色相(색상)이 公開(공개)되었으며, 後(후)에 出市(출시)된 LTE-A 모델 專用(전용) 色相(색상)으로 '블루 아크틱', '레드 오로라', '로즈골드 화이트', '로즈골드 블랙' 等(등)이 追加(추가)되었다.

派生(파생) 모델[편집]

  • 갤럭시 S4 미니 (小型化(소형화) 모델)
  • 갤럭시 S4 액티브 (防水防塵(방수방진) 特化(특화)모델)
  • 갤럭시 S4 줌 (카메라 特化(특화)모델)

內需用(내수용) 모델만의 特徵(특징)[편집]

通信社(통신사)別(별) 모델 區分(구분)이 사라지지 않고 오히려 LGT가 VoLTE를 導入(도입)하면서 KT와 SKT用(용) 모델 사이는 部品(부품)이나 소프트웨어 等(등)이 互換(호환)되지만, LGT用(용) 모델만 差異點(차이점)이 多數(다수) 存在(존재)해서 互換性(호환성)이 떨어진다.

例(예)를 들어 SKT나 KT用(용) SIM카드로는 LGT용 모델의 通信機能(통신기능)을 使用(사용)할 수 없고, LGT의 SIM으로는 他(타) 通信社(통신사) 機種(기종)들에서 通信(통신)이 안 되거나 했다. 後(후)에 LGT가 他社(타사) 機器(기기)에서 데이터 네트워크만큼은 動作(동작)하도록 措置(조치)하였지만, 音聲(음성)·映像(영상)通話(통화)나 文字(문자) 메시지의 受(수)·發信(발신)은 如前(여전)히 不可能(불가능)했다.
