세리에 C甲

原本 編輯 歷史
分類 : 蹴球리그
伊太利(이태리) 프로 蹴球(축구)리그
세리에 B → 세리에 C甲 → 세리에 C乙(을)

세리에 C甲(갑)는 伊太利(이태리) 蹴球(축구)리그 시스템의 第(제)3級(급)別(별) 리그이다.

C리그는 두 个(개)의 區域(구역)으로 나뉘는데 각 區域(구역)에 18個(개) 팀이 있습니다. 1978年(년) 구 伊太利蹴球丙級리그는 2個(개) 級(급)別(별) 리그인 C甲(갑) 리그와 C乙(을) 리그로 나뉘었다.

每(매) 시즌말, 各(각) 區(구)마다 2個(개) 팀 세리에 B에 進出(진출)하고, 세 팀이 C乙(을) 리그에 降等(강등)되었다.
