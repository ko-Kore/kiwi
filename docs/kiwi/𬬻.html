𬬻

原本 編輯 歷史
分類 :

𬬻는 簡體字(간체자) 中國語(중국어)에서 104번 元素(원소) 러더포듐(rutherfordium)을 나타내는 漢字(한자)이다. 왼쪽이 钅(금)인데, 異常(이상)하게도 이 위키에서는 볼드體(체)가 아닌 境遇(경우)에 왼쪽이 釒(금)으로 뜬다.

  • 유니코드: U+2CB3B
  • 通用規範漢字表(통용규범한자표): 7006番(번)
  • 中國語(중국어) 讀音(독음): lú
  • 繁體字(번체자): 鑪(노)(U+946A)
