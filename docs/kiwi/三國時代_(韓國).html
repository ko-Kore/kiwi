三國時代 (韓國)

原本 編輯 歷史
分類 : 韓國史

三國時代(삼국시대)는 韓國(한국)史(사)의 時代(시대) 區分(구분)으로 韓半島(한반도)에 存在(존재)했던 高句麗(고구려), 百濟(백제), 新羅(신라)가 三韓(삼한)과 古朝鮮(고조선)에서 成長(성장)하고 統一新羅(통일신라)에 依(의)해 滅亡(멸망)과 統合(통합)이 이루어지는 時期(시기)를 말한다. 三國(삼국)은 上記(상기)한 세 國家(국가)를 意味(의미)하지만 一般的(일반적)으로 伽倻(가야)까지 三國時代(삼국시대)에 包含(포함)된다. 各國(각국)의 歷史(역사)는 各各(각각)의 文書(문서)를 參考(참고)하라.

構成(구성)國(국)

  • 高句麗(고구려)
  • 百濟(백제)
  • 新羅(신라)
  • 伽倻(가야)

三國(삼국)의 共通點(공통점)[編輯]

伽倻(가야)를 除外(제외)한 三國(삼국)은 强力(강력)한 部(부)의 代表(대표)가 王(왕)이 되어 外交權(외교권)과 軍事(군사)權(권) 行使(행사)했다. 또한 各(각) 部(부)는 自治的(자치적)으로 運營(운영)되는 封建制的(봉건제적) 性格(성격)이 强(강)했으며 國家(국가)의 重大事(중대사)는 王(왕)과 部(부)의 代表(대표)들이 會議體(회의체)를 通(통)해 合議(합의)해서 決定(결정)했다.

會議體(회의체)

  • 高句麗(고구려) : 諸加(제가) 會議(회의)
  • 百濟(백제) : 政事巖(정사암) 會議(회의)
  • 新羅(신라) : 和白(화백) 會議(회의)
