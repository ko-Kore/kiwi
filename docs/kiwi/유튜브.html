유튜브

原本 編輯 歷史
分類 : 動映像 웹 사이트 구글

使用者(사용자)가 動映像(동영상)을 自由(자유)롭게 올리거나 視聽(시청)할 수 있는 구글의 콘텐츠 호스팅 웹사이트이자, 2023年(년) 現在(현재) 世界(세계) 最大(최대) 規模(규모)의 비디오 플랫폼이다.

유튜브라는 名稱(명칭)은 使用者(사용자)를 가리키는 'You(당신)'와 美國英語(미국영어)에서 텔레비전의 別稱(별칭)으로 使用(사용)되는 'Tube'를 더한 것이다. 過去(과거) 텔레비전이 브라운管(관)(Cathode-Ray 'Tube', CRT)를 使用(사용)했기 때문에 텔레비전을 美國(미국) 英語(영어)에서 다른 말로 '튜브'라 부른다. 即(즉) 'YouTube'라는 名稱(명칭)의 語感(어감)은 '當身(당신)을 爲(위)한 텔레비전', '當身(당신)이 곧 텔레비전' 정도이다. 네모난 아이콘의 끝부분은 둥글게 깎여 있고 각 변은 끝부분보다 볼록하게 나와 있는데 옛 텔레비전에 쓰였던 브라운管(관)의 視覺上(시각상) 特徵(특징)을 아이콘으로 디자인한 것이다. 韓國(한국)에서는 이른바 '너튜브'로도 通用(통용)된다.

유튜브는 美國(미국)의 企業(기업)이지만, '유튜브'라는 發音(발음)은 사실 美國式(미국식)보다는 英國式(영국식)에 가깝다. 美國式(미국식)으로는 'tju'가 아닌 'tu'로 발음하여 '유투브' 정도로 읽는다.
