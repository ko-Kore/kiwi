花園明朝

分類 : 書體

槪要(개요)[편집]

花園明朝(화원명조)(はなぞのみんちょう, Hanazono Mincho)는 明朝體(명조체) 基盤(기반) 韓中日(한중일) 漢字書體(한자서체)이다. 유니코드의 거의 모든 漢字(한자)를 支援(지원)하며, 非漢字(비한자) 文字(문자)도 一部(일부) 包含(포함)한다.

收錄(수록) 漢字(한자) 領域(영역)[편집]

漢字(한자) 總(총) 88,884字(자)와 異體字(이체자) 總(총) 8,828字(자)가 收錄(수록)되어 있다.

  • U+4E00 .. U+9FA5 URO[1] 20,902字(자)
  • U+9FA6 .. U+9FEA URO追加(추가) 69字(자)
  • U+3400 .. U+4DB5 Ext-A 6,582字(자)
  • U+20000 .. U+2A6D6 Ext.B 42,711字(자)
  • U+2A700 .. U+2B734 Ext.C 4,149字(자)
  • U+2B740 .. U+2B81D Ext.D 222字(자)
  • U+2B820 .. U+2CEA1 Ext.E 5,762字(자)
  • U+2CEB0 .. U+2EBE0 Ext.F 7,473字(자)
  • U+F900 .. U+FAD9 互換文字(호환문자) 472字(자)
  • U+2F800 .. U+2FA1D 互換擴張(호환확장) 542字(자)
  • IVD[2] .. 2016年(년)8月(월)15日(일)版(판) 28,770字(자) (其中(기중) 8,828字(자)가 追加(추가)異體字(이체자), IVD重複(중복) 16個(개)는 따로 計數(계수))
