槪要(개요)[편집]

갤럭시 S10e는 2019年(년)2月(월)20日(일)에 公開(공개)되어 同年(동년) 3月(월)8日(일)에 出市(출시)된 三星電子(삼성전자)의 스마트폰이며, 갤럭시 S 시리즈의 열番(번)째 製品(제품)인 갤럭시 S10의 派生(파생) 모델이다.

特徵(특징)[편집]

S 시리즈에서 처음으로 생긴 入門型(입문형) 모델이다. 이름의 알파벳 'e'는 essential을 意味(의미)한다고 한다.[1]

仕樣(사양)[편집]

5.8인치로 S10 시리즈 中(중) 가장 작은 크기의 扁平(편평)한[2] 인피니티 O 디스플레이를 搭載(탑재)했다.

指紋認識(지문인식) 센서는 主(주)모델과 다르게 畵面內(화면내)가 아닌 側面(측면) 電源(전원)버튼에 內藏(내장)되었고, 左側面(좌측면)의 音量(음량)버튼의 밑에는 빅스비 呼出(호출) 버튼이 位置(위치)해 있다. 全體的(전체적)으로 봤을 때 音量(음량)버튼보다 電源(전원)버튼이 上端(상단)으로 올라가 있는데, 機器(기기)를 잡을 時(시) 손의 位置(위치)를 考慮(고려)했을 때 빅스비를 많이 쓰도록 意圖(의도)한 것이 아니냐는 意見(의견)도 있었다.

後面(후면) 카메라는 廣角(광각)과 標準畵角(표준화각)으로 두 가지가, 前面(전면)은 하나가 搭載(탑재)되어 있다.

色相(색상)은 '프리즘 블랙', '프리즘 화이트', '카나리아 옐로우' 세 가지로 出市(출시)되었다.

運營體制(운영체제)[편집]

出市當時(출시당시) 안드로이드 9 (파이)와 三星(삼성) 원 UI 1.1이 搭載(탑재)되었다. 以後(이후) 안드로이드 10과 원 UI 2.0, 안드로이드 11과 원 UI 3.0 順(순)으로 업그레이드되었다.
